# porong-economy — 경제 서버 (구상 단계)

> **[STATUS: 구상 — 개발 미착수]** 경제/거래/생산/시장 중심의 후보 서버. 컨셉을 모으는 단계이며,
> 아직 실제 코드·서버 런타임·빌드 산출물은 없다.

## 한 줄 소개
유저 간 거래·시장·생산·경제 밸런스가 게임플레이의 중심이 되는 경제 특화 후보 서버.

## 현재 상태
- 개발 착수 전. 이 폴더에는 **문서만** 존재한다(런타임/빌드 없음).
- 구상 내용: [`docs/concept.md`](docs/concept.md)
- 미확정 아이디어: [`docs/idea_inbox.md`](docs/idea_inbox.md)

## 방향(요약)
- 거래·시장(경매/상점/유저샵), 생산·가공 체인, 화폐·물가·경제 밸런스, 유저 간 경제 플레이.
- 세부 플랫폼(모드/플러그인 여부)·규모·시즌제 등은 **미확정**.

> 착수가 결정되면 worktree(`porong-work-economy` / `feature/economy-dev`)를 만들고 본격 설계 문서를 추가한다.
> 절차는 루트 [`docs/worktree_policy.md`](../docs/worktree_policy.md) §6 참고.
