# porong-economy 밸런싱 시트 (P1 초기값)

> **성격:** 설계서(`spec.md`)·결정로그가 `P1`로 미뤄둔 **수치·계수의 초기값(seed) 카탈로그.**
> **철학(DL-E084):** 3단 페이싱(바닥 보장·중간 보수적 창발·천장 하드게이트) + **라이브 튜닝.** 여기 값은 **플레이테스트 출발점**이지 확정이 아니다 — *내부 일관성*과 *역산 근거*가 핵심.
> **메타 원칙(DL-E118):** ① **개인 수치(설비비·연구·생산)는 보수적 하향 불필요 — 약간 높게 OK.** 메타 게임플레이(Create 학습·설비 시행착오·뜯어고치기) 시간이 페이싱을 자연 완충하므로, "산수상 시즌 내 도달?"에만 맞춰 타이트하게 잡지 않는다. ② **집단 임계(마을 진화)는 인원·활동 정규화 필수** — 개인이 아니라 인원 변동(예상보다 적거나 많음)에 취약(고정 절대값 회피). ③ 단 **산수상 생존·진입 자체가 불가능한 붕괴**(임금<필수재·day-0 연구비 폭발)는 메타 시간과 무관 → 항상 수정.
> **단위:** 게임 화폐 = **골드(G)**. 시간 = **틱**(아래 §0). 인구 = 집계(명).
> **최종 갱신:** 2026-06-12 (DL-E100 — 기준 앵커 1차)

## §1. 시즌 페이싱 — 진화 16 세부단계 (28일 기준)

> DL-E055 클라이맥스 도달 보장 + DL-E093 진화 임계 = **인원수 정규화**(소규모 영구 정체 방지).

| 진화 단계 | 목표 일자 | 시대 | 핵심 해금 |
|---|---|---|---|
| **마을** 1~4 | D1–4 | 수공업 | 의회 출범·1차 산업·필수재·부트스트랩 |
| **소도시** 1~4 | D4–8 | 증기 | 강철·중공업·2차 가공·B2B·계약·산업/시장물가 안건 |
| **도시** 1~4 | D8–14 | **전기** | **자동차·가전·자본시장 활성·금융 안건 = 클라이맥스·정산**(전기 6일 충분) |
| ~~대도시·국가~~ | **시즌2+** | 자동화·대외무역 | "발전된 미래 도시" — PC·고급차·반도체·M&A·대외무역(E132·E133·E134) |

- **시즌1 = 3단계 × 4서브 = 12 세부단계, ~1.17일/레벨**(14일). **클라이맥스 = 도시(전기·자본시장) Week2.**
- **일자 = 단계 머무는 기간(진입~다음 진입):** 마을 시작 D1 · **소도시 진입 D4** · **도시(전기) 진입 D8** · 도시 6일 체류 → **정산 D14.** (도시 도착이 D14가 아니라 D8 진입·D14 종료)
- **단계 명칭 = SoT(DL-E134):** 마을·소도시·도시(시즌1) / 대도시·국가(시즌2). 〔구 명칭 매핑: 구 "도시"(증기)=소도시 · 구 "대도시"(전기)=도시 · 구 "국가"(자동화)=대도시. 타 문서의 "대도시=전기·자본시장·금융" 등 구 언급은 본 §1 기준으로 점진 정리〕
- **다인자 진화 게이트(DL-E051·E138):** `score = w₁·(인구/기준인구) + w₂·(GDP/기준) + w₃·산업다양성 + w₄·평균학력 + w₅·인프라`. 가중치 **인구 0.1·GDP 0.25·산업 0.2·학력 0.2·인프라 0.25**(인구 비중↓ — 14일 인구 성장은 물리적 상한이 낮아 *보조 지표*로 강등, 활동·생산이 주, E138), 5지표 중 **다수(3+) 충족** 시 상승. **기준값 = 인원·활동 정규화**(DL-E093·E118) → 실제 참여 인원/활동(접속·생산)에 비례 → 사람이 예상보다 적어도 영영 못 가거나 많아도 너무 빨리 가지 않음(집단 임계는 인원 변동 취약). 30명이 평범히 플레이하면 일정 도달 보장.
- **⏱ 클라이맥스 시간 fallback(E138, DL-E055 정렬):** 다인자가 빡빡해도 **D4 소도시·D8 도시(전기) *접근권*은 시간 기반으로 보장 개방**. 진화 *연출·보너스*(추가 수용력·생산성)는 다인자 충족으로, 클라이맥스(전기·자본시장) *도달*은 시간 보장으로 **분리** → "2주 안에 도시·자본시장을 확실히 맛본다"는 시즌1 명제를 산수 리스크와 무관하게 보장. 다인자를 일찍 채우면 **조기 진입**(보너스), 못 채워도 시간 도달 시 개방(바닥).

- **GDP = 1인당 실질 GDP(CPI 디플레이트, DL-E108):** 명목 아님 → **인플레만으로 거저 진화 차단**(§18 인덱싱 일관). 진짜 생산성·산업 고도화만 반영.

> 30명 테스트로 "땅 사서 회사 짓고 팔아 돈 벌고 주민 소비로 루프가 도는" 최소 경제 증명(DL-E074 Phase 1).

| 항목 | 초기값 | 근거 |
|---|---|---|
| 소비성향(MPC) | 저소득 0.9 · 중산 0.7 · 고학력 0.5 | 저소득 고MPC(DL-E044) — 임금이 곧 소비로 환류 |
| 기본 임금(틱당) | 저숙련 100 G · 중숙련 200 G · 고학력 400 G | 계층 격차 ~4배 |
| 최저임금 floor | **90 G/틱** | 90×0.95×0.9 = 76.95 **> 필수재 75 → 생존선 흑자**(바닥 3종 미달 보정 E117). 착취 바닥(DL-E017·E092) |

## §3. 생산·가동률 (spec §4 / DL-E086·E092·E093)

`가동률 = min(1, 충원율) × 학력매칭 × 전력충족률 − 안전 페널티 × (1 − 외생 페널티)`

| 항목 | 초기값 | 비고 |
|---|---|---|
| 설비 요구 인원 | **초소형 1~2** · 소형 5 · 중형 20 · 대형 50명 | 규모 비례(큰 공장=많은 고용) |
| **유저 본인 노동 기여** | 고숙련 노동력 **2~3명분**(무임금, 직접 조작 시) | **초반 자영** — 초소형·소형은 유저+소수로 가동, 대형은 고용 필수(DL-E116) |
| **전력충족률** | 공급/수요 비(전기 시대+), 부족 시 < 1.0 → 가동률↓ | 가동 = 노동 AND 전력 게이트(spec §11-5). 전기시대 전엔 1.0 |
| 학력매칭 | 충족 1.0 · 1단계 미달 0.7 · 2단계+ 0.4 | 고급 산업은 고학력 필수(개별 직무) |
| 고장 확률(틱) | base 5% × (1 − 충원율) | 충원율 1.0이면 ~0 |
| 고장 복구 | 2틱(감독관 시 1틱) | 학력·감독관 단축(DL-E047) |
| 품질(weakest-link) | 각 stage 0~100, 최종 = min(stages) × (0.5 + 0.5·학력) | 미투자 stage = base 30 |
| 외생 페널티 kᵢ | k₁오염 0.20 · k₂범죄 0.15 · k₃산재 0.10 · **k₄파업 0.10~0.30**(강도, DL-E129) (합 cap 0.6) | 가동률 최대 −60%까지. 파업=노동 거부→생산 차질(E131 배선) |

## §18. 인플레이션 처리 — 명목 vs 실질 (DL-E107)

> balance 골드값 = **시즌 시작 명목 기준.** 시간 축에서 통화량(대출=발권, spec §5-2) 누적 → 명목 가격 전반 상승(인플레 = 창발). 고정 명목값이 인플레로 상대가치 왜곡되지 않도록 처리.

| 가격 종류 | 인플레 반영 |
|---|---|
| **시장 형성가**(상품·부동산 거래·임금·주가) | **수급으로 자동 반영**(통화량↑→명목가↑) — 별도 처리 불필요 |
| **CORE 고정값**(부동산 기준가·최저임금·실업급여) | **지표 연동(인덱싱)** 필요 |

- **㉠ 인덱싱(지표 연동 — 인플레 자동 반영, DL-E109 전수 분류):**
  - 부동산 기준가(§14)·**상품 기준가(§4 100/300/5000G)** = 평균 임금/CPI 연동
  - 최저임금·실업급여(§2) = CPI 연동(생존선 유지)
  - **IPO 자기자본 요건(§7 5,000G)·진화 GDP 임계(§1-1)** = 실질(CPI 디플레이트, DL-E108)
  - **§16 전 명목 effect**(건설비·운영비·순찰·국채·긴급유동성·보조금·R&D예산·출산보조 등 일체) = 명목 GDP 연동(후반에도 유효 규모, E117)
  - 주민회사 앵커 = 비용 연동(DL-E088, 자동)
- **㉡ 1회성 명목(시작 시점만, 인플레 무관):** 시드 자본 10,000G·국고 120,000G·저축 200G(시즌 초 소진).
- **㉢ 명목 중립(%·배수, 인플레 무관):** 세율(법인 20%·거래 2%)·금리·MPC·탄력·HHI·부채/세수 배수·레버리지 ×2·쿨다운 일수·서킷 % 임계 · PER·OPEX%·산재 보상배수·배당%·자산효과 k·α/β/γ·학력매칭·고장% **등 기타 모든 %·배수·계수.**
- **㉣ 자동(시장 형성가, 처리 불필요):** 임금·상품/부동산 거래가·주가(수급으로 인플레 자동 반영).
- **적정 인플레 = 거시 재미**(명목 성장감) / 하이퍼인플레 = 서킷(DL-E096) / 디플레 = 저축 재순환 watch(§16). 연동 계수 N·CPI 기준 P1.
