# porong-economy 주민 집계 동학 모델 (구상)

> **[STATUS: 구상 — 스켈레톤]** 주민 집계(하이브리드 §4·DL-E003)의 **행동/동학** — 상태를 굴리는 규칙·함수. 상태(데이터)는 [`data_schema.md`](../05-data/data_schema.md) `population_stratum`.
> 이 모델이 **구현 1순위 토대** — 환경·치안·시장·재정·물가 지표가 모두 여기서 창발. 함수의 구체 계수·임계는 P1 밸런싱.
> 개요 [`concept.md`](../01-vision/concept.md) · 시스템 상세 [`design.md`](../02-design/design.md)(§N) · 결정 [`decision_log.md`](../decision_log.md). 본 문서의 "§N"은 design.md.

## 왜 별도 문서인가
데이터 스키마(상태: 인구·학력·저축)와 동학(행동: 소비·노동·출산·투자 함수)은 다른 관심사다. 동학은 시뮬레이션 엔진의 심장이라 한 곳에 모은다 — 지금 concept에 흩어진 창발 함수들을 여기로 consolidate.

## 모델링 원칙 (2층 · DL-E042)
- **① 인과 메커니즘 (이 문서):** 각 동학은 **단일 원인이 아니라 다인자 함수** — 여러 요인이 +/− 방향으로 겹친다(예: 출산은 "호황" 하나가 아니라 고용안정·주거·양육비·소득·안전의 복합). 현실 경제 원리 반영(예: **소득-출산 역U자** 역설). 단순 "X 좋으면 Y↑"는 지양.
- **② 계수·임계 (P1 밸런싱):** 각 요인의 가중치·함수 모양·임계값. 구현·튜닝 단계.
→ 지금은 ①(요인과 방향)을 제대로 잡고, ②는 P1.

## 채울 항목 (스켈레톤)

1. **계층 정의** — 학력 tier(무학/기초/중등/고학력) × 고용 상태. 계층 간 이동(교육→상향, 실업→하향).
2. **소비 함수** — 소득탄력성·필수품/사치품·MPC(한계소비성향). 고품질=사치 소비(DL-E013). → 시장 수요(DL-E014·E026).
3. **노동 공급·고용·임금** — 학력별 가능 직업, 임금 결정, 실업. **노동=소비 루프**(임금↓·해고→소비력↓→경기 악화, §4) 자기조정.
4. **인구 동학** — 출산·사망·이주 = f(경제·생활수준). 인구 증가↔노동/소비.
5. **저축·투자 풀** — 소득 일부 저축, 주식 투자 압력 집계 함수(재무·주가추세·경기·저축률, DL-E033).
6. **창발 지표 함수** (concept에서 이관·정리):
   - 환경오염 = Σ(생산 × 업종 오염계수) + 발전 − 흡수 (DL-E010)
   - 범죄율 = f(실업률·빈곤율·불평등) (DL-E011)
   - 행복도 = 가중합(욕구충족·실질소득·고용안정 − 노동시간·산재·불평등·범죄) (DL-E009)
   - CPI·인플레 = 수급·통화량 (DL-E014)
   - 빈곤율·생활수준·지니 = 소득·욕구충족 분포
7. **틱 주기** — 집계 갱신 주기(실시간 vs 배치), indicator_snapshot 기록 주기.

## 가계 흐름 모델 (1차 · 엔진 심장) — 계층별·매틱

> 주민 = 살아있는 가계(household). 돈이 **소득 → 소비 → 저축 → 자산배분** 한 방향으로 흐른다. (항목 2·3·5 충족.)

**① 소득**
```
가처분소득 = 임금(고용분) + 자산소득(예금이자·배당·채권이자) + 이전소득(복지·실업급여) − 소득세 − 부채이자 + 신규 차입
```
- **주민 가계부채(DL-E120·E121, 거시 발권 주력):** **실질금리(명목−인플레)·호황·소비자신뢰↑** 시 **차입 소비**(가처분 초과). **부채이자 = 가계부채 × 금리**를 가처분에서 차감 → 과다 차입 시 **디레버리징**(소비↓·상환). 호황 부채↑(소비 거품) ↔ 불황 디레버리징(소비 위축) = 신용 사이클. 부채/소득 임계 초과 = 가계 부실·소비붕괴(agenda §11 규제 트리거). 수치 balance §6.
- **인플레↔가계부채 상호작용(DL-E122):** 인플레 = 기존 명목 부채 실질가치↓(채무자 유리) → 고인플레·저금리(실질금리↓)에 차입 폭증(거품) → **인플레 대응 금리 인상 시 이자 폭증 → 변동금리 가계 직격·일제 부실 → 소비붕괴**(신용 사이클 클라이맥스 = 대공황 방아쇠, 현실 금리 인상기 위기와 동형).

**② 소비 (다인자 · 노동=소비 루프 핵심)**
| 요인 | 방향 |
|---|---|
| 가처분소득 | + (주 동인) |
| 소비자 신뢰(경기 심리·고용 안정) | + (불안하면 **예비적 저축**↑ → 소비↓) |
| 물가(실질소득) | − (인플레→실질소득↓→소비↓) |
| 자산효과(주가·부동산↑) | + (부유감 → 소비↑ — 자본시장 §3↔실물 연결) |
| 신용 접근(대출) | + (소비 당겨씀) / 부채 과다 − (상환 부담) |
| 미래 불확실성(노후·고용) | − (예비적 저축) |
- 구조: 필수품 우선(소득탄력 낮음) → 여유분 사치=고품질(소득탄력 높음, DL-E013). 저소득 = 높은 MPC / 고소득 = 낮은 MPC.
- **노동=소비 루프:** 임금↓·해고 → 가처분↓ **+ 소비자 신뢰↓(예비적 저축)** → 소비 이중 위축 → 기업 매출↓ → 경기 악화. 심리가 루프를 **가속**(불황 공포→소비 붕괴 / 호황 자신감→과소비·거품). 착취 자기조정(§4).

**③ 저축** = 가처분 − 소비 → 계층 `savings` 누적.

**④ 자산 배분** — 저축을 예금/채권/주식에 (살아있는 투자자)
| 자산 | 성격 | 공급처 |
|---|---|---|
| 예금 | 안전·이자(금리) | 중앙은행(§10, 단일·monobank) 예치 — **통화 싱크/저장**(대출=발권이라 예금이 재원 아님, 신용기준이 한도 조절, DL-E078·E093) |
| 채권 | 중위험·이자 | 국채(재정 §10) / 회사채(옵션) |
| 주식 | 고위험·고수익(주가·배당) | 자본시장(DL-E033) |
- **배분 = 다인자:**
  | 요인 | 효과 |
  |---|---|
  | 기대수익 격차 | 높은 쪽으로 자금 이동(주식 기대수익 vs 채권·예금 이자) |
  | 금리(중앙은행) | 금리↑→예금·채권 매력↑(주식 이탈, 안전선호) / 금리↓→주식·위험자산(수익 추구) |
  | 위험 심리(경기·변동성) | 호황→위험선호(주식↑) / 불황·공포→안전선호(예금·채권 도피) |
  | 계층 위험감내 | 저소득=예금(유동성·안전) / 고소득=주식·채권(위험·장기) |
  | 가격 모멘텀 | 주가↑→추격 매수(거품) / 폭락→패닉 매도(붕괴) — 자기강화 |
- **핵심 채널:** ① **금리 = 자금 스위치**(통화정책 거시 §5 → 자산 이동 → 주가·자본조달·저축) ② **모멘텀 = 거품/폭락 엔진**(거시 사이클 §8 자산 측면) ③ **계층 차이 → 불평등 심화**(부유층만 자산소득 → 지니↑ → 범죄·복지 안건, 자산효과 소비도 부유층 위주).
- **의미:** 주민 저축이 **자산 시장 유동성의 원천** — 채권→재정/회사 자금, 주식→자본 조달. (예금은 통화 싱크 — **대출은 중앙은행 발권**, 예금 잔액이 아니라 신용기준이 한도 결정, DL-E078.) 자금이 자산 간 이동하며 금리·주가·신용·불평등이 연동.

**계층 차이 요약:** 저소득(고MPC·소비·예금) ↔ 고소득(저MPC·저축·주식/채권).

## 노동·고용·임금 모델 (1차 · 가계 흐름의 소득원)

- **노동 공급:** 계층별(학력) 인구가 노동력. 학력이 **가능 직업 tier**를 가른다(고학력=연구·고부가가치, 저학력=1·2차). 학력 상향은 교육(§7).
- **노동 수요 = 민간 + 공공:**
  - 민간: 유저 회사 고용(생산 규모·자동화 수준). 자동화↑→인력↓(기술적 실업, 산업 §7), 단 채용 0이면 설비 정지(§2).
  - **공공: 인프라·학교·경찰서/교도소·주민회사·행정** — 재정으로 고용, 의회 안건이 증감.
- **학력 매칭 = 운영 효율·고장·복구 (공공·민간 공통):** 시설/회사의 운영 효율·고장률·복구 속도 = 배치된 주민 학력이 시설 요구 티어를 충족하는가. **고티어 시설(발전소·행정·연구소·고부가가치 공장)은 고학력 필수** — 부족하면 저효율·잦은 고장·느린 복구(예: 발전소 고학력 부족 → 정전 잦음). → 교육(§7) 투자 동기.
- **임금 (다인자):**
  | 요인 | 방향 |
  |---|---|
  | 노동 수급 | 수요>공급 → 임금↑ |
  | 최저임금(주민 안건 §6) | 하한 |
  | 학력 프리미엄 | 고학력 부족 → ↑ |
  | 자동화 압력 | 자동화↑ → 노동 수요↓ → 임금↓·실업 |
  | 공공 고용 | 정부 고용↑ → 노동 수요↑ → 임금 지지·실업↓ |
  | 생산성 | 생산성↑ → 임금 지불 여력↑ |
- **실업:** = 노동 공급 − (민간 고용 + 공공 고용) → 가계 소득↓ + 실업급여 + 범죄율↑(DL-E011). **공공 고용이 완충.**
- **공공 고용 = 경기 부양 레버(승수효과):** 불황 시 의회가 인프라 건설(§6·재정) 가결 → 공공 일자리로 실업 흡수 → 소비 회복(케인스식). 재정 지출이 고용→소비로 환류.
- **루프 완성:** 임금·고용(민간+공공) → 가계 소득(가계 흐름 ①) → 소비 → 매출 → 고용 = **노동=소비 루프 닫힘**(§4).

## 인구 동학 모델 (1차 · 장기 성장 엔진)

- **출산 (다인자 — 단일 "호황→↑" X):** 출산율 = 여러 요인의 복합:
  | 요인 | 방향 |
  |---|---|
  | 고용 안정성 | + (불확실성이 출산 억제 — 안정이 핵심) |
  | 주거 접근성 | + / 주거비 폭등 − (토지 §9) |
  | 양육 비용(생활비·교육비) | − |
  | 소득 수준 | **역U자** (빈곤=못 낳음 / 중산층=최적 / 고소득=기회비용↑로 ↓) |
  | 사회 안정(범죄·오염↓) | + |
  → **마을이 부유해지면 자연 출산율이 떨어지는 역설**(선진국 저출산) → 이주 의존·출산 장려 안건(§6) 필요. 신생아 → **무학 계층** 진입(→ 교육 §7 상향).
- **사망 (다인자):** 자연 사망(고령·인구 비례) + **초과 사망** = f(빈곤[욕구충족↓·기아]·범죄율·산재·오염 − 의료/복지·생활수준). 의료 인프라·복지가 초과 사망을 누른다.
- **이주 (다인자):** 순이주 = **유입** − **유출**.
  - 유입 = f(마을 매력: 저실업·고임금·생활수준↑·범죄/오염↓·저세금) + **난민 수용 안건**(비선별 대량).
  - 유출 = f(불황·고실업·저임금·고세금·범죄/오염↑·과밀).
  - 유입 계층: **난민=저학력 위주 / 자발적 이주=일자리 따라 혼합**. 유입 = 즉시 노동 공급.
  - 안건(§6): 이주 유치(선별·매력) / 난민 수용(비선별 — 인건비↓·인구↑ vs 범죄↑·복지, 저출산 역설 DL-E042의 출구).
- **계층 이동:** 교육 → 학력 상향(계층 상승) / 장기 실업·빈곤 → 하향. 신생아=무학 진입.
- **인구 ↔ 마을 진화:** 인구·경제 규모 임계 넘으면 마을→도시→대도시→국가(§5). 인구 = 노동 공급+소비 수요+진화 게이트.
- **자기조정(과속/정체 방지):** 인구 폭증 → 과밀·생활수준↓ → 출산↓·유출↑ / 인구 부족 → 임금↑·매력↑ → 유입↑. 균형 자동.

## 창발 지표 함수 (주민 집계 → 지표 → 안건 트리거)

> concept에 흩어진 창발 공식을 한 곳에 정리. 모두 주민 집계·시장·생산 상태에서 나와 **indicator_snapshot(시계열) 기록 → 대시보드·웹 차트 + CORE 안건 트리거(§6)**.

| 지표 | 함수 | 출처 |
|---|---|---|
| 환경오염(stock) | 유입[Σ(생산×업종 오염계수)+발전×발전원계수] − 흡수[녹지 정화+정화사업] | DL-E010 |
| 범죄율 | f(실업률↑·빈곤율↑·불평등↑) | DL-E011 |
| 행복도 | 가중합(욕구충족+실질소득+고용안정 − 노동시간−산재−불평등−범죄−오염) | DL-E009 |
| CPI·인플레 | 필수품 바스켓 가격(수급) / 인플레=통화량(소스>싱크)·수요 초과 | DL-E014 |
