## 확정 방향 (이번까지)
- **최종 목표:** 유저가 돈을 많이 버는 것.
- **장르 정체성:** 마인크래프트로 하는 **경제발전 시뮬레이션(계급 구조)** — 유저=자본가 계급(엔진), 주민 집계=노동·소비 계급(몸통), 대표 빌리저=얼굴.
- **주민 구현 = 하이브리드 (P0-① 해소):** 집계 모델(계층 풀)이 진짜 경제, 대표 빌리저 수십 명이 얼굴. 수만 인구(국가)까지 부하 없이 확장. 거시경제는 집계에서 **창발**(별도 엔진 아님).
- **조작 방향성 = MC답게 (확정):** 회사는 땅 사서 직접 짓는 건축물. 자동화 모드/플러그인을 쓰되 **core 모드가 제어** — 채용 주민 없으면 정지, 감독관/안전규칙 미준수 시 **산재**. 운영은 **대시보드**(산출량·평균품질·시장평균가·주민선호도·공급안정성)로 확인.
- **주민회사(시장 안정 닻):** 담합·공급공백→경제 붕괴 방지용 공영 NPC 기업. 중간 품질·기준가로 가격 닻. 유저는 품질로 그 위 경쟁.
- **단일 마을**(4개 마을 분할안 폐기). 마을 → 도시 → 대도시 → 국가로 진화.
- **주민 = 노동·소비·투자 주체**(정치 안 함). 물건·주식 수요자가 주민.
- **노동=소비 루프:** 임금↓·해고 → 소비력↓ → 자기 물건도 안 팔림(착취 자기조정).
- **정부 = 유저 의회.** 게임/DB가 지표 보고 안건 발의 → 유저 토론 → 찬/반 과반 가결 → 정책 발효(수치 트레이드오프).
- **연구·교육:** 재료만으론 기술 진보 불가. 고학력 주민으로 연구해 발전 트리. 학교 등 교육 인프라 필요.
- **토지:** 회사는 마을 토지 구매처에만 설립(마을 내부 한정), 외부 자원은 수입 가능. 발전 칸은 마을 발전 따라 확장.
- **시장:** 수요·필수품/사치품·공급수요로 팔림. 품질(quality) 속성으로 경쟁.
- **인프라:** 판매에 전력·운송 필요. 의회 에너지 정책에 따라 비용 변동.
- **거시경제:** 호황→인플레, 대공황, 은행 파산(리만式) 등 사이클.

## 착수 제작 체크리스트 — 커스텀 아이템 (DL-E073)
> 기존 모드/바닐라로 안 되는 것만 커스텀. 각 = **등록(코드) + 텍스처(16x16) + 레시피(KubeJS 가공 체인)**.

**커스텀 = 완성 소비재 5~6군** (나머지 식품·원료·부품=기존 모드, 배=바닐라·기차=Steam'n'Rails·가구=가구 모드):
| 품목군 | 레시피(가공 체인 개략) |
|---|---|
| 의류·신발 | 원단(섬유)+노동 → 옷·신발 |
| 운송(자동차·자전거) | 강철·엔진·부품 → 차체 → 자동차 |
| 가전(냉장고·세탁기·TV) | 부품·전자회로 → 가전 |
| 전자완성(PC·스마트폰) | 반도체·회로·부품 → 전자제품 |
| 의약·화장품 | 화학원료 → 의약/화장품 |

작업 3종 + 규모:
- [ ] **등록(코드):** 아이템 등록 + 품질 속성 hook (가벼움)
- [ ] **텍스처:** 수십 개 16x16 ⚠️ 주 일거리 → 외부 킷·AI·직접
- [ ] **레시피(제작법):** KubeJS = 가공 체인(B2B 원료→완성품) — 2층 시장·품질 전파·시대 게이팅 구현 지점
- [ ] **CORE 연동:** 산출물 박스 업종-품목 인식·시장 등록·소비유형 태그
- 규모 감각: ~5~6군 × 품질/티어 변주 = 수십 아이템 (텍스처 부담 핵심, 레시피는 KubeJS로 빠름)

## 구현 로드맵 (착수 순서 · 수직 슬라이스 우선 · DL-E074)
> 원칙: Phase 1 수직 슬라이스로 핵심 재미(노동=소비 루프) 먼저 증명 → 정치·UI 폴리시 나중. 단계마다 플레이 가능 유지(1인 개발).

- **Phase 0 — 셋업:** NeoForge 1.21.1 + Create(큐레이션) + KubeJS + DB(SQL) + 자동설치기 골격.
- **Phase 1 — 수직 슬라이스 MVP ⭐:** 주민 집계 엔진(가계·노동·인구 동학, 토대) + 회사 설립·생산(Create)·산출물 박스 + 시장 추상 정산(호가·수요곡선) + 중앙은행 현금·거래세. → "땅 사서 회사 짓고 팔아 돈 벌고 주민 소비로 노동=소비 루프가 도는" 최소 경제. **30명 테스트로 핵심 재미 먼저 검증.**
- **Phase 2 — 경제 확장:** 기술 트리·시대(마을 진화 해금)·분야 특화(연구원) + 자본시장(주식·M&A·지배구조) + 계약(B2B 납품박스) + 대시보드 풀 GUI.
- **Phase 3 — 정치·창발:** 창발 지표(오염·범죄·CPI)·안건 트리거 + 의회(발의·투표·정당·서킷브레이커). (의회는 마을 성장 후 등장 → 자연 후순위)
- **Phase 4 — 폴리시·시즌:** 풀 텍스쳐 UI + 커스텀 완성품 아이템·텍스처 + 디스코드 봇 알림 + 시즌 정산·랭킹 + P1 밸런싱(플레이테스트).
