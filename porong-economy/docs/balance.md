# porong-economy 밸런싱 시트 (P1 초기값)

> **성격:** 설계서(`spec.md`)·결정로그가 `P1`로 미뤄둔 **수치·계수의 초기값(seed) 카탈로그.**
> **철학(DL-E084):** 3단 페이싱(바닥 보장·중간 보수적 창발·천장 하드게이트) + **라이브 튜닝.** 여기 값은 **플레이테스트 출발점**이지 확정이 아니다 — *내부 일관성*과 *역산 근거*가 핵심.
> **단위:** 게임 화폐 = **골드(G)**. 시간 = **틱**(아래 §0). 인구 = 집계(명).
> **최종 갱신:** 2026-06-12 (DL-E100 — 기준 앵커 1차)

## §1. 시즌 페이싱 — 진화 16 세부단계 (28일 기준)

> DL-E055 클라이맥스 도달 보장 + DL-E093 진화 임계 = **인원수 정규화**(소규모 영구 정체 방지).

| 진화 단계 | 목표 일자 | 시대 | 핵심 해금 |
|---|---|---|---|
| 마을 1~4 | D1–7 | 수공업 | 의회 출범·1차 산업·필수재 |
| 도시 1~4 | D7–14 | 증기 | 2차 가공·B2B 심화·산업/시장물가 안건 |
| 대도시 1~4 | D14–21 | 전기 | 자본시장 활성·금융 안건·가전/자동차 |
| 국가 1~4 | D21–28 | 자동화 | 고부가 완성품(PC·고급차)·부 경쟁 클라이맥스·정산 |

- **레벨당 ≈ 1.75일**(28 / 16). 서브레벨 임계 = 직전 대비 점증.
- **다인자 진화 게이트(DL-E051):** `score = w₁·(인구/기준인구) + w₂·(GDP/기준) + w₃·산업다양성 + w₄·평균학력 + w₅·인프라`. **기준값을 인원수에 정규화** → 30명이 평범히 플레이하면 위 일정 도달 보장(절대 임계로 영구 정체 X). 가중치 w·기준값 = 플레이테스트 튜닝.
- **쇠퇴(역진화) 지속 임계:** 임계 미달 **연속 2일**(96틱) 지속 시 강등 → 초반 출렁임에 즉시 강등 방지.

## §2. 노동 = 소비 핵심 루프 ⭐ (가장 먼저 검증할 재미)

> 30명 테스트로 "땅 사서 회사 짓고 팔아 돈 벌고 주민 소비로 루프가 도는" 최소 경제 증명(DL-E074 Phase 1).

| 항목 | 초기값 | 근거 |
|---|---|---|
| 소비성향(MPC) | 저소득 0.9 · 중산 0.7 · 고학력 0.5 | 저소득 고MPC(DL-E044) — 임금이 곧 소비로 환류 |
| 기본 임금(틱당) | 저숙련 100 G · 중숙련 200 G · 고학력 400 G | 계층 격차 ~4배 |
| 최저임금 floor | **80 G/틱** | 착취 바닥(DL-E017·E092) — 무캡이되 절벽 방지 |
| 필수재 1인 소비 | ~150 G/틱(저소득 기준) | 시드 저축 200 G = 1.3틱 버퍼와 정합 |
| 실업급여 | 최저임금의 50% = 40 G/틱 | 자동 안정자 바닥(DL-E096), 위험밴드 시 비례 확대 |

- **자기조정 검증:** 임금↓·해고 → 가처분소득↓ → 소비↓ → 자기 물건도 안 팔림(DL-E071). 이 루프가 30명·600주민에서 *체감되는 속도*로 도는지가 1순위 플레이테스트.
- **착취↔산재 무캡 도박(DL-E092):** 안전 투자 비용 < 기대 산재 비용이 되도록 산재 확률 계수 튜닝(§산재).

## §3. 생산·가동률 (spec §4 / DL-E086·E092·E093)

`가동률 = min(1, 충원율) × 학력매칭 − 안전 페널티 × (1 − 외생 페널티)`

| 항목 | 초기값 | 비고 |
|---|---|---|
| 설비 요구 인원 | 소형 5명 · 중형 20명 · 대형 50명 | 규모 비례(큰 공장=많은 고용) |
| 학력매칭 | 충족 1.0 · 1단계 미달 0.7 · 2단계+ 0.4 | 고급 산업은 고학력 필수 |
| 고장 확률(틱) | base 5% × (1 − 충원율) | 충원율 1.0이면 ~0 |
| 고장 복구 | 2틱(감독관 시 1틱) | 학력·감독관 단축(DL-E047) |
| 품질(weakest-link) | 각 stage 0~100, 최종 = min(stages) × (0.5 + 0.5·학력) | 미투자 stage = base 30 |
| 외생 페널티 kᵢ | k₁오염 0.20 · k₂범죄 0.15 · k₃산재 0.10 (합 cap 0.5) | 가동률 최대 −50%까지 |

## §4. 시장·가격 (spec §4 / DL-E014·E027·E097)

`가격 = 기준가 × (수요 / 공급)^(1/|탄력|)`

| 항목 | 초기값 | 비고 |
|---|---|---|
| 수요 탄력 | 필수재 −0.3 · 내구재 −1.0 · 사치재 −1.8 | 사치재가 경기 민감 |
| CPI 가중 | 필수 0.5 · 내구 0.3 · 사치 0.2 | 물가지수 |
| 기준가(시대별) | 필수재 100 G · 중간재 300 G · 완성품 5,000 G(전기+) | 부의 천장 상승(DL-E068) |
| HHI 임계 | 0.25 → 독점 안건 / 0.40 → **주민회사 자동신설**(DL-E097) | 투표 불요 경쟁 주입 |
| 앵커 발동 | 공급 < 수요의 **70%** → 동적 발동 / 점유율 상한 **30%** | 정상화 시 철수(DL-E097) |

## §5. 인구 동학 (population_model / DL-E041·E042·E048·E096)

| 항목 | 초기값 | 비고 |
|---|---|---|
| 자연 출산(일) | 정점 +2%/일 (욕구충족 중간), 양끝 +0.5% | 역U자(DL-E042) |
| 자연 사망(일) | base 0.5%/일 | 고령·인구 비례 |
| 초과 사망(일) | (빈곤 + 범죄 + 산재 + 오염) × 0.1 계수 | 복지·의료가 완화(DL-E048) |
| 이주 유입(일) | 매력도(실질임금·고용·생활수준) > 임계 → +1%/일 | 자기조정 |

| 강제 이주(인구 앵커) | 인구 붕괴 속도 > 1%/일 지속 → 강제 유입 발동 | 비가역 0 수렴 차단 |
