# 필드보스 패턴 배정 초안

> **[STATUS: REFERENCE]** — 필드보스 패턴 조합 참조. 공식 보스 기준은 `../06_fields_bosses/CANON.md`가 우선.

> 작성일: 2026-05-13  
> 필드보스는 공용 패턴(P-00~P-09, P-11~P-13)만 사용한다. 고유 패턴 없음. (P-10 폐기, DL-048)  
> P-00은 전 보스 상시 발동 기본 공격.  
> 수치는 초안이며 프리 시즌 실측 후 조정 예정.

---

## 원칙

- 필드보스는 공용 패턴 모듈만 조합한다 (`common_patterns.md` 참조).
- 보스별 테마에 맞게 이펙트(파티클 색상, 소리)와 파라미터만 오버라이드한다.
- 공용 패턴 재사용률 목표: 80% 이상.
- Phase 전환(P-08)은 모든 필드보스에 최소 1회 적용한다.

---

## 1. 들판 포식자

**테마**: 맹수. 돌진·강타 중심. 1차 시즌 입문 보스.  
**난이도**: 쉬움 (패턴 수 최소, 예고 시간 넉넉)

| 패턴 | 코드 | 파라미터 오버라이드 | 활성 Phase |
|---|---|---|---|
| 기본 공격 | P-00 | 피해 55%, 쿨 2.0s (맹수 기본 타격) | Phase 1·2 |
| 전방 강타 | P-01 | 피해 100%, 범위 3블럭, 쿨 5s | Phase 1·2 |
| 직선 돌진 | P-02 | 피해 130%, 거리 12블럭, 예고 2.0s | Phase 1·2 |
| 부채꼴 휩쓸기 | P-04 | 각도 100°, 거리 4블럭, 쿨 6s | Phase 1·2 |
| 페이즈 전환 | P-08 | 트리거 50%, 무적 없음 | 50% HP |
| 원형 폭발 (분노) | P-03 | 피해 160%, 반지름 6블럭 | Phase 2 전용 |

**Phase 구조**
- Phase 1 (100%~50%): P-01, P-02, P-04 순환
- Phase 2 (50%~0%): P-01(피해+10%), P-02(속도+20%), P-03 추가

**참고**
- P-02 돌진 후 원위치 복귀에 0.5s 소요. 복귀 중 피해 없음.
- 필드보스 중 가장 단순한 구조. 유저 패턴 학습 목적.

---

## 2. 폐광 골렘

**테마**: 중장갑 구조체. 느리고 강함. 지면 충격·소환 중심.  
**난이도**: 보통 (이동 속도 느림, 범위 넓음)

| 패턴 | 코드 | 파라미터 오버라이드 | 활성 Phase |
|---|---|---|---|
| 기본 공격 | P-00 | 피해 70%, 쿨 2.5s (느린 골렘 타격) | Phase 1·2 |
| 전방 강타 | P-01 | 피해 160%, 범위 4블럭, 쿨 6s, 넉백 2.5 | Phase 1·2 |
| 원형 폭발 | P-03 | 피해 140%, 반지름 5블럭, 딜레이 1.5s | Phase 1·2 |
| 소환체 호출 | P-07 | 수 2, HP 3,000, 방치 시 폭발 (범위 피해) | Phase 1·2 |
| 연속 타격 | P-12 | 피해 145%, 연타 2~3회, 간격 0.7s, 쿨 10s | Phase 2 전용 |
| 페이즈 전환 | P-08 | 트리거 60%, 무적 없음 | 60% HP |

**Phase 구조**
- Phase 1 (100%~60%): P-01, P-03, P-07 순환
- Phase 2 (60%~0%): P-01(피해+15%), P-03(반지름+1), P-12 연속 타격 추가

**참고**
- P-07 소환체는 "파편 골렘"(소형). 처치하지 않으면 10초 후 폭발 (반지름 3 피해).
- 이동 속도 기본값의 70%. 느린 대신 P-01 범위가 넓어 근접 패턴을 강요.
- P-12 연속 타격: Phase 2 고유 압박. 타격 간격 0.7s로 골렘 특유의 묵직한 연타 표현.

---

## 3. 수로 군주

**테마**: 수생 생물. 독·투사체·추적 구체 중심. 원거리 대응 필요.  
**난이도**: 보통 (추적 독 구체+투사체 조합으로 위치 선택 강제)

| 패턴 | 코드 | 파라미터 오버라이드 | 활성 Phase |
|---|---|---|---|
| 기본 공격 | P-00 | 피해 60%, 쿨 2.0s (촉수 기본 타격) | Phase 1·2 |
| 직선 돌진 | P-02 | 피해 120%, 거리 8블럭, 경로 너비 3 | Phase 1·2 |
| 부채꼴 휩쓸기 | P-04 | 각도 140°, 거리 5블럭 (촉수 휩쓸기) | Phase 1·2 |
| 투사체 발사 | P-05 | 피해 100%, 속도 10, 발사 수 2, 독 이펙트 | Phase 1·2 |
| 추적 독 구체 | P-11 | 유도 투사체, 피해 110%, 착탄 시 독 3s, 쿨 8s | Phase 2 전용 |
| 페이즈 전환 | P-08 | 트리거 40%, 무적 없음 | 40% HP |
| 산개탄 (분노) | P-06 | 발사 수 8, 낙하 범위 10, 독 착탄 | Phase 2 전용 |

**Phase 구조**
- Phase 1 (100%~40%): P-02, P-04, P-05 순환
- Phase 2 (40%~0%): P-05(발사 수 3), P-11 추적 독 구체, P-06 추가

**참고**
- P-11 추적 독 구체: Phase 2에서 가장 가까운 대상 유도. 착탄 시 3초 독 상태이상. P-05보다 빠른 추가 위협.
- P-05 독 이펙트: 착탄 후 3초 독 상태이상 (0.5초마다 15 피해).

---

## 4. 타락한 수호기사

**테마**: 타락한 기사. 칼날 공격·어둠 산개 중심. 공격적이고 빠름.  
**난이도**: 어려움 (패턴 속도 빠름, Phase 전환 시 패턴 밀도 증가)

| 패턴 | 코드 | 파라미터 오버라이드 | 활성 Phase |
|---|---|---|---|
| 기본 공격 | P-00 | 피해 65%, 쿨 1.6s (빠른 칼날 타격) | Phase 1·2 |
| 전방 강타 | P-01 | 피해 140%, 범위 4블럭, 쿨 4s, 예고 0.6s | Phase 1·2 |
| 직선 돌진 | P-02 | 피해 170%, 거리 14블럭, 속도 4× | Phase 1·2 |
| 부채꼴 휩쓸기 | P-04 | 각도 90° × 연속 2회 (좌→우), 쿨 7s | Phase 1·2 |
| 산개탄 | P-06 | 발사 수 5, 어둠 파편, 착탄 범위 2, 쿨 14s | Phase 1·2 |
| 페이즈 전환 | P-08 | 트리거 50%, 무적 없음 | 50% HP |
| 전방 강타 (강화) | P-01 | 피해 180%, 예고 0.5s | Phase 2 전용 |

**Phase 구조**
- Phase 1 (100%~50%): P-01, P-02, P-04, P-06 순환
- Phase 2 (50%~0%): 모든 쿨타임 -20%, P-01 피해 증가, P-04 연속 3회로 변경

**참고**
- P-04 연속 발동: Phase 1은 좌→우 2회, Phase 2는 좌→우→좌 3회. 예고 방향 파티클 확인 필요.
- 빠른 예고 시간(0.5~0.6s)이 특징. 반응 요구 수준이 높음.

---

## 5. 균열 파수꾼

**테마**: 균열 에너지. 고난도 종합 보스. 패턴 밀도 최대.  
**난이도**: 어려움 (5번 필드 최종, 시즌보스 입장 전 준비 보스)

| 패턴 | 코드 | 파라미터 오버라이드 | 활성 Phase |
|---|---|---|---|
| 기본 공격 | P-00 | 피해 70%, 쿨 1.8s (균열 에너지 기본 타격) | Phase 1·2·3 |
| 전방 강타 | P-01 | 피해 160%, 범위 4블럭, 쿨 4s | Phase 1·2·3 |
| 원형 폭발 | P-03 | 피해 150%, 반지름 6블럭, 딜레이 1.0s | Phase 1·2·3 |
| 투사체 발사 | P-05 | 피해 130%, 발사 수 3, 관통 있음, 쿨 7s | Phase 2·3 |
| 소환체 호출 | P-07 | 수 2~3, HP 4,000, 방치 시 보스 공격력 +15% | Phase 2·3 |
| 낙하 충격 | P-13 | 피해 160%, 반지름 5블럭, 균열 이펙트, 18s 주기 | Phase 3 전용 |
| 페이즈 전환 | P-08 | 트리거 70%, 40% (2회 전환) | 70%, 40% HP |
| 무력화+딜타임 | P-09 | 무적 시간 30s, 기절 2s, 취약 창 +20% | Phase 3 전용 |

**Phase 구조**
- Phase 1 (100%~70%): P-01, P-03 순환
- Phase 2 (70%~40%): P-01, P-03, P-05, P-07 추가
- Phase 3 (40%~0%): 전 패턴 활성 + P-13 낙하 충격, P-09 추가

**참고**
- P-09 발동 조건: 보스가 40% 전환 후 처음 한 번만. "균열 폭주" 연출 후 30초 무적. 이 구간에 기여도 딜 기록 중지 (보상 분배 기준 고정).
- P-07 소환체 방치 시 보스에게 귀환해 공격력 버프 부여 (처치가 권장이지만 강제 아님).
- P-13 낙하 충격: Phase 3에서 균열 에너지 낙하 연출. 18초 주기, 예고 파티클 후 반지름 5블럭 즉발 충격.
