## §1. 시대 × 분야 매트릭스 (24칸 — 누적 해금)

> 세로 6분야(개인 특화) × 가로 4시대(마을 진화 자동 해금, DL-E062). 각 칸 = 그 시대에 그 분야가 만들 수 있는 산출품군. **누적**(전기 시대 = 수공업~전기 전부 가능).

| 분야 ↓ 시대 → | 수공업 | 증기 | 전기 | 자동화 |
|---|---|---|---|---|
| **농업·식품** | 곡물·채소·축산(원료) | 밀가루·통조림·가공식품 | 음료·기호품(술·커피·과자)·냉장식품 | 합성·정밀영양식품 |
| **광업·소재** | 손채굴 광석·석재·목재 | 강철·합금·주물·벽돌 | 전선·구리·알루미늄·유리 | 반도체 소재·희토류·복합소재 |
| **제조·공산품** | 도구·목가구·수공 직물 | 대량 의류·기계부품·**기차·배** | **자동차·자전거·가전·가구** | **고급차·정밀기계·로봇** |
| **에너지·화학** | 목탄·식물성 기름 | 석탄 코크스·증기 동력 | 석유 정제·연료·플라스틱·화학원료 | 정밀화학·의약원료·배터리 |
| **전자·첨단** | — | (기초 전기 부품) | 전자회로·전구·기본 가전 | **반도체·PC·스마트폰·첨단기기** |
| **서비스·금융** | 운송(짐꾼)·소매 | 철도 물류·창고업 | 금융·전력·통신 서비스 | 데이터·R&D 대행·고급 금융 |

- **부 천장 상승(DL-E068):** 좌상단(필수재·저마진) → 우하단(완성 내구/사치재·고마진). **시즌1 클라이맥스 = Week2 도시(전기) 우하단**(자동차·가전); PC·고급차는 시즌2(자동화). (14일, E133)
- **분업(DL-E067):** 한 칸 특화 → 다른 칸 산출물을 B2B로 구매(소재 유저 강철 → 제조 유저 자동차). 품질 전파(DL-E064).

## §2. 완성 소비재 (B2C — 주민 구매) + 소비 유형 (DL-E066)

| 품목군 | 구체 품목 | 소비 유형 | 시대 |
|---|---|---|---|
| 식료품 | 빵·통조림·고기·채소 요리 | 필수(경기 둔감) | 수공업+ |
| 기호식품 | 술·커피·과자·음료 | 사치(경기 민감) | 전기+ |
| 의류·신발 | 옷·신발·가방·패션 | 필수(기본)~사치(고급) | 수공업+ |
| 운송 | 자전거 · 자동차 · 고급차 | 내구(경기 민감) | 전기+ |
| 가전 | 냉장고·세탁기·TV·에어컨 | 내구(경기 민감) | 전기+ |
| 전자 | PC·스마트폰·태블릿 | 내구·사치(최고 부가) | 자동화 |
| 의약·화장품 | 의약품·화장품·생활화학 | 필수(의약)~사치(화장품) | 전기+ |
| 가구 | 가구·생활용품 | 내구 | 수공업+ |

> **사치품 = 별도 품목 아님(DL-E013):** 같은 빵도 저품질=필수(빈곤층)·고품질=사치(부유층). 품질 상층이 사치 시장.

## §3. 가공 체인 (원료 → 중간재 → 완성품)

> Create 가공 단계마다 산출품(체인 깊을수록 부가가치↑, DL-E061). `[ ]` = 공정.

| 완성품 | 가공 체인 |
|---|---|
| **빵** | 밀 →[제분]→ 밀가루 →[제빵]→ 빵 |
| **옷** | 면화/양모 →[방적]→ 실 →[직조]→ 원단 →[재봉]→ 옷 |
| **자동차** | 광석 →[제련]→ 강철 + 석유 →[정제]→ 플라스틱·연료 + 반도체 →[조립]→ 회로 → (차체 + 엔진 + 부품) →[조립]→ 자동차 |
| **PC** | 반도체 소재 →[정밀]→ 반도체 →[조립]→ 회로 → 부품 →[조립]→ PC |
| **가전** | 강철·플라스틱 + 전자회로 →[조립]→ 가전 |
| **의약품** | 화학원료 + 식물 →[화학]→ 의약원료 →[정제]→ 의약품 |

- **체인 깊이 페이싱(DL-E070 ⓑ):** 깊은 체인은 초반 입력 공급 부족 → B2B 의존·계약 활성. 깊이별 시대 게이팅(P1).

## §5. 커스텀 완성품 5~6군 상세 (DL-E073 — 등록 + 텍스처 + 레시피)

> 기존 모드/바닐라로 안 되는 **최종 완성 소비재만** 커스텀. 각 = ① 아이템 등록(품질 속성 hook) ② 텍스처(16×16) ③ KubeJS 레시피(가공 체인). 티어/품질 변주로 품목당 수 종.

| # | 품목군 | 레시피 체인(개략) | 입력 소스 |
|---|---|---|---|
| 1 | **의류·신발** | 원단(IE/Create) + 노동 → 옷·신발·가방 | 원단=직조 체인 |
| 2 | **운송**(자동차·자전거) | 강철 + 엔진 + 부품 + 플라스틱 → 차체 → 자동차/자전거 | IE 강철·연료 |
| 3 | **가전**(냉장고·세탁기·TV·에어컨) | 강철·플라스틱 + 전자회로 → 가전 | IE 소재 + 전자 |
| 4 | **전자완성**(PC·스마트폰·태블릿) | 반도체 + 회로 + 부품 → 전자제품 | IE 반도체 소재 |
| 5 | **의약·화장품** | 화학원료 → 의약원료 → 의약/화장품 | IE Petroleum 화학 |
| (6) | **고급 변주**(고급차·명품·첨단기기) | 1~5의 고품질/자동화 시대 상층 | 품질 전파 상층 |

- **규모 감각:** 5~6군 × 품질/티어 변주 = **수십 아이템.** 텍스처가 주 일거리(외부 킷·AI·직접, DL-E058) / 레시피는 KubeJS로 빠름.
- **CORE 연동(DL-E073):** 산출물 박스가 업종-품목 인식 → 시장 등록 → 소비유형 태그(필수/내구/사치). 커스텀 아이템에 품질 속성(data component) 부착.

## §6. 서비스·금융 분야 = 무(無)아이템 수익 (DL-E072)

> 6분야 중 서비스·금융은 아이템 산출 없음 → CORE 시스템 수익(DL-E070 ⓒ 해소).

| 트랙 | 수익 모델 |
|---|---|
| 금융·증권 | 주식 중개 수수료·투자 차익·배당(DL-E033·E072) |
| 전력·발전 | 전기세 판매(price-taker, DL-E087) |
| 물류·운송 | 계약 운송 대행 수수료·창고업 |
| 연구·R&D | 연구원 고용 → 기술 우위 → 분야 노드(간접 부가가치, DL-E062) |

→ 제조 없는 트랙으로도 부 축적 가능(무캡, DL-E091). 수수료율·수익 계수 P1.
