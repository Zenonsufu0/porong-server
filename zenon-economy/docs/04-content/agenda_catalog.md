# zenon-economy 안건 카탈로그 (CORE 발의 안건)

> **[STATUS: 구상 — 작성 중]** CORE SYSTEM이 [`indicators.md`](../03-economy/indicators.md)(경제 지표 마스터 리스트)를 감시하다 임계 조건에서 발의하는 안건 목록. 유저는 발의 불가, 찬/반만(`design.md` §6, DL-E006).
> 컬럼: 안건 / 트리거(지표·임계) / 가결 효과 / 부결·방치 / 가치축. **트리거 임계 수치 = [`balance.md`](../03-economy/balance.md) §15 / effect 크기 = §16**(초기값, DL-E103). 전부 플레이테스트 조정.
> **기술 게이팅:** 모든 안건은 서버 공통 기술 트리(`design.md` §7)의 **선행 기술 전제**에 묶인다(예: 친환경 전환 = 친환경 발전 연구 후). 세부 선행조건 매핑은 P1.
> 결정 기록: [`decision_log.md`](../decision_log.md)(`DL-E###`).

## 1. 환경

> 오염 모델(누적 stock = 유입−흡수)·업종 오염계수는 `design.md` §1-A 참조. 세 안건이 각각 다른 레버(유입↓·흡수↑·녹지↑)를 건드린다.

| 안건 | 트리거 | 가결 효과 (트레이드오프) | 부결·방치 | 축 |
|---|---|---|---|---|
| **친환경 에너지 전환** | 환경오염 ≥ 위험 임계 | 발전칸 −30 · 전기세 +50G/kWh · 오염 유입↓(발전원 오염계수↓) | 오염 계속 증가 | 환경 |
| **탄소세(고오염 산업 과세)** | 오염 상승률 가파름 / 고오염 업종 비중↑ | **과표 = 생산량 × 오염계수**(이익 아님 → 이전가격 회피 무력, DL-E099) · 오염 증가 둔화 · 해당 업종 수익성↓ | 오염↑ 지속, 규제 회피 산업 난립 | 환경 |
| **그린벨트 지정** | 녹지 비율 < 임계(난개발) | 특정 토지 구매 불가(가용 토지↓) · 자연 흡수↑(오염↓) · 주변 부동산 가치↑ | 개발 지속·녹지 소실 | 환경 |
| **오염 정화 사업** | 오염 stock 매우 높음(임계 초과 지속) | 재정 지출↑(세수 소모) · 흡수 일시 급증(오염 stock 직접 대폭↓) | 오염 누적 → 행복도·노동생산성↓ | 중립(재정 부담) |

## 2. 치안·범죄

> 범죄율 = 활동 창발 `f(실업률↑·빈곤율↑·불평등↑)` — `indicators.md` §11 참조. 단속(질서) vs 예방(분배) 축이 갈린다.

| 안건 | 트리거 | 가결 효과 (트레이드오프) | 부결·방치 | 축 |
|---|---|---|---|---|
| **경찰서 건설** | 범죄율 ≥ 임계 / 치안 수준 < 임계 | 가용 토지↓ · 세수 소모(건설+운영) · 치안↑(범죄율↓) | 약탈·생산 차질·행복도↓ | 단속(질서) |
| **교도소 건설** | 수감 수요 > 교정 수용(범죄자 적체) | 가용 토지↓ · 세수 소모 · 재범 억제(범죄율 추가↓) | 수용 초과 → 치안 효과 반감 | 단속(질서) |
| **치안 예산 증액(순찰 강화)** | 범죄율 단기 급등(건물 없이 즉응) | 경상 재정 지출↑(지속 비용) · 범죄율↓(즉효·일시) | 범죄 확산 | 중립 |
| **사회 복지 확대(근본 대응)** | 범죄 원인이 빈곤·실업(실업률·빈곤율↑) | 복지 지출↑(세수↓) · 빈곤↓→범죄율 근본↓ · 행복도↑ | 빈곤→범죄 악순환 | 분배·예방 |

## 3. 재정

> 재정 = 모든 안건의 돈줄(`재정수지 = 세수 − 공공지출`, 부채 stock+이자). 모델은 `design.md` §10. 고전 4대 재정 레버. 성장·자유(감세·긴축) vs 분배(증세·복지) 축이 정면 충돌.

| 안건 | 트리거 | 가결 효과 (트레이드오프) | 부결·방치 | 축 |
|---|---|---|---|---|
| **증세**(법인·소득·거래세 인상) | 재정 적자 지속 / 부채-세수 비율 ≥ 임계 | 세율↑ → 세수↑·재정 개선 / 회사 수익성↓·주민 가처분소득↓(소비↓) | 적자·부채 지속 | 분배 |
| **감세** | 재정 흑자 / 경기 침체(부양 필요) | 세율↓ → 기업·소비 활성(성장) / 세수↓ | 현행 유지 | 성장·자유 |
| **국채 발행**(적자 재정) | 대규모 지출 필요 & 세수 부족 | 즉시 자금 확보(사업 가능) / 부채↑·이자 부담↑(미래 세수 잠식) | 사업 보류 | 중립(현재 vs 미래) |
| **긴축**(공공지출 삭감) | 부채 위기(부채-세수 비율 위험) | 지출↓·재정 개선 / 복지·인프라·교육 축소 → 행복도↓·성장 둔화·범죄율↑ | 부채 눈덩이 → 신용 위기(거시 §3) | 성장·자유(작은정부) |
| **누진세 강화/완화** | 불평등(지니) ≥ 임계 | 고소득 세율↑·저소득↓ → 불평등↓·저소득 소비여력↑ / **자본가(유저)·고소득 부담↑**, 투자 위축 가능 | 불평등 심화 → 범죄율↑·사회 불만 | 분배 |
| **산업 보조금/세액공제** | **최근 연구 완료된 기술 존재** / 산업 다양성↓ | 해당 기술 사용 공장에 보조금(세수 소모) → 신기술 채택 가속·산업 부양 | 신기술 보급 더딤 | 성장 |

## 4. 시장·물가

> 가격=수급에서 형성, CPI·인플레·독점도(HHI) 모델은 `design.md` §11. **시장 개입 안건은 전부 DL-E008 "경쟁 O·시장 파괴 X" 준수**(강제분할·국유화 없음). 독점 대응 2종이 성장/분배로 갈린다.

| 안건 | 트리거 | 가결 효과 (트레이드오프) | 부결·방치 | 축 |
|---|---|---|---|---|
| **가격 상한제(필수품)** | 필수품 CPI ≥ 임계(생계 위협) | 필수품 가격 상한 → 빈곤층 보호 / 공급자 마진↓ → 공급 위축·품귀 위험 | 물가 지속↑ → 빈곤·범죄율↑ | 분배 |
| **독점 품목 주민회사 투입** | 독점도(HHI) ≥ 임계 | 국유 주민회사 신설(§11-A) → 경쟁 주입·가격↓ / 재정 지출(운영비)·민간 위축 | 독점 지속 → 가격 횡포 | 중립·분배 |
| **신규 진입 장려(경쟁 보조금)** | 독점도↑ & 신규 회사 적음 | 신규 진입 보조금(재정) → 경쟁사 유도(독점 희석) | 진입 장벽 → 독과점 고착 | 성장 |
| **비축 방출/매입(수급 안정)** | 품귀(공급≪수요) 또는 과잉 | 비축 방출(품귀 완화) 또는 매입 비축(과잉 시 가격 바닥 지지) / 재정 비용 | 가격 급등락·시장 불안 | 중립 |
| **물가 안정(통화 싱크 강화)** | 인플레이션율 ≥ 임계 | 통화 회수(세금·수수료↑ = 화폐 싱크) → 물가 안정 / 경기 위축·소비↓ | 인플레 가속 → 자산 거품(거시 §3) | 중립 |

## 5. 거시·경기 (통화정책 포함)

> 통화정책 = **의회 안건(A안)**이되, 파국 임계에서 **CORE 비상 안전장치**(서킷 브레이커, `design.md` §6) 발동 — ① 경고=거부권 ② 파국 임박=긴급명령. 유저는 단기 부양(금리↓·주가↑)을 선호하므로 ⚠️ 표시 안건이 주 발동 대상.

| 안건 | 트리거 | 가결 효과 (트레이드오프) | 부결·방치 | 축 |
|---|---|---|---|---|
| **기준금리 인상** | 인플레↑ / 경기 과열 / 자산 거품 | 차입 비용↑ → 과열·거품 진정 / 투자·소비 위축·성장 둔화 | 거품 팽창 → 붕괴 위험 | 중립(긴축) |
| **기준금리 인하** ⚠️ | 경기 침체 / 실업↑ | 차입 싸짐 → 투자·소비·주가↑(부양) / 인플레·거품 위험 | 침체 지속 | 성장 |
| **양적완화/긴축** ⚠️ | 심한 침체(완화) / 과열(긴축) | 통화 공급 조절 → 유동성↑↓ / 거품 or 신용경색 위험 | 유동성 위기 or 과열 방치 | 중립 |
| **신용(대출) 규제** | 신용 과팽창 / 경색 | 중앙은행 대출 기준 조절 → 신용 팽창/수축 억제 | 신용 거품 or 경색 | 중립 |
| **신용경색 긴급 유동성** | 대출 급감·신용경색(중앙은행이 최종 대부) | 중앙은행 긴급 유동성 공급 → 신용 회복 / 통화 팽창·도덕적 해이 | 신용경색 → 대공황 | 중립 |

## 6. 노동·고용 (노동=소비 루프의 정치)

> 노동시장을 의회가 직접 조정 — **노동=소비 루프(`design.md` §4) 핵심.** 대부분 자본가(유저)에 직접 비용 → 성장 vs 분배 충돌. DL-E008(시장 파괴 X) 안에서. 〔§6 분리 DL-E130: 인구·복지·사회 → §12〕

| 안건 | 트리거 | 가결 효과 (트레이드오프) | 부결·방치 | 축 |
|---|---|---|---|---|
| **최저임금 인상/인하** | 빈곤율↑·임금 중위값↓(인상) / 실업률↑(인하) | (인상) 주민 소득↑→소비↑(경기 부양) / 기업 인건비↑→고용 위축·수익성↓ | 저임금·빈곤 지속 → 소비 위축 | 분배 |
| **노동시간 상한 규제** | 평균 노동시간 ≥ 임계(과로) | 노동시간↓ → 행복도↑·산재↓ / 생산량↓·기업 부담 | 과로 → 행복도·장기 생산성↓ | 분배 |
| **노동안전 규제 강화** | 산재 발생률 ≥ 임계(DL-E004) | 산재↓·행복도↑ / 기업 안전 비용↑(설비·감독관) | 산재 지속 → 노동력 손실·평판↓ | 분배 |
| **정년 연장/노동 참여** | 생산가능인구 부족(부양비↑·노동력 핍박) | 노년 노동 참여↑ → 노동력 보강·연금 부담↓ / 청년 일자리 경쟁·생산성 논란 | 노동력 부족 지속 → 성장 제약 | 중립·성장 |
| **아동노동 금지/규제** | 유소년 고용률 ≥ 임계(DL-E127) | 유소년 고용 금지·적발 시 평판/벌금 → 교육 기회·미래 학력 보호 / 인건비↑(기업 부담) | 아동노동 만연 → 평판·인적자본 손실·국제 비난 | 분배·윤리 |
| **노동 협상·집회 대응(시위·파업)** | 행복도 < 임계 → 파업·시위(DL-E129) | **달램**(임금·복지 양보 → 생산 복구·행복도↑) **or 진압**(치안 동원 → 질서 회복·불만 잔존) | 파업·시위 지속 → 가동률↓·정치 불안 | 분배(달램) ↔ 질서(진압) |

## 9. 연구·교육 (인적자본)

> 기술 트리 = 서버 공통(개인별 X), 안건·산업을 게이팅(`design.md` §7). 학교·연구소는 건설형 안건. **연구 자체·연구 방향 선택은 의회 밖 별도 연구 시스템**(의회 출범 전부터 작동) — 의회는 R&D 예산으로 가속만.

| 안건 | 트리거 | 가결 효과 (트레이드오프) | 부결·방치 | 축 |
|---|---|---|---|---|
| **학교 건설/증설** | 학교 수용량 < 수요 / 평균 학력 정체 | 학교 건설(건설형·재정·토지) → 고학력 주민 양성↑ | 학력 정체 → 연구·산업 한계 | 중립·분배 |
| **연구 투자(R&D 예산)** | 연구 진척도 정체 / 신기술 필요 | 연구 예산↑(재정) → 기술 트리 진행 가속 | 기술 정체 → 경쟁력↓ | 성장 |
| **무상/보조 교육** | 저소득 학력 격차(불평등) | 저소득 학력↑·계층 이동·불평등↓ / 재정 부담 | 교육 격차 → 불평등 고착 | 분배 |
