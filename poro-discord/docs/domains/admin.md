# 운영/관리자 도메인 — 명세 (스텁, 설계 선행)

> **[도메인: 운영]** **[STATUS: STUB/설계 선행]** — 보안 영향이 큰 영역.
> 실제 구현 전 설계/승인을 선행한다. 코드: `modules/admin/commands.py`(빈 Cog).

## 보안 원칙

- 모든 운영 명령어는 `core/permissions.py` 의 `@requires_permission(...)` 로 보호한다.
- 운영 권한 역할은 **수동 지급 전용**(봇 자동 지급 금지) — [`../roles_and_permissions.md`] §B.
- 상태 변경(역할 부여·골드 지급 등)은 게임 서버 API 경유로만, 사용자가 명시적으로
  요청할 때만 구현한다.
- 운영 명령어는 전용 채널(`#운영로그`)에서만 실행하도록 제한하는 방향을 검토한다.
