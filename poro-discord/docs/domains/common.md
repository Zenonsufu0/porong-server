# 공통 도메인 — 명령어

> **[도메인: 공통]** — 특정 게임(RPG/포로몬)에 종속되지 않는 봇 전역 기능.
> 코드: `modules/common/`.

## 명령어

| 명령어 | 설명 | 상태 |
|---|---|---|
| `/핑` | 봇 응답 지연(latency, ms) 확인 (ephemeral) | 🟢 구현 (`general.py`) |

## 설계 (T18)

| 명령어 | 설명 | 비고 |
|---|---|---|
| `/도움말` | 사용 가능한 명령어 안내(도메인별 그룹) | 호출자가 보유한 접근역할·권한 기준으로 노출 명령만 표시(가시성 정책 §10 정합) |
| `/서버상태` | RPG·포로몬 서버 온라인/접속 인원 통합 조회 | 각 도메인 integration 클라이언트 집계. **active 서버만** 집계(레지스트리). 포로몬 = `poromon_api`(T4) 선행 |

- 둘 다 공통(게이트 없음). `/서버상태`는 종료 서버 제외(레지스트리 `state=active`만).
- 커뮤니티 레벨·출석 등 공통 기능은 [`../community_level.md`](../community_level.md)(T13·T14).
