## 한 줄 컨셉
경계 있는 **영속 월드**에서 **기지를 짓고**, 버려진 시설을 **파밍**하고, 서로 **약탈**하고,
**밤의 습격**을 막아내는 총기 서바이벌. **시즌(와이프) 단위** 운영.
= 마크형 **"러스트 + 타르코프"**.

## 왜 이 방향인가 (핵심 결정)
- **마크 고유 메커니즘(건설·파괴·채집·PvE) × 총** = "다른 총겜 하면 되지"에 대한 답.
  배그·러스트·타르코프 **어느 하나로도 대체 불가**한 조합. 차별점은 총이 아니라 **부서지고 지어지는 세계.**
- **시즌(와이프) 운영에 최적.** 러스트식 1~2주 와이프 주기 = "2주 시즌"과 정합. 시즌 종료=맵 와이프+시즌2 보강.
- **코어 재사용성(중요):** 영속월드 코어(총·무게·부위·손실·기지·파밍 시스템)를 만들어두면,
  훗날 **타르코프/러스트/배틀로얄 등 분리형 모드로 재활용** 가능. → 과거 "멀티모드 셀렉터" 방향은
  폐기가 아니라 **미래 확장 옵션**으로 보존. ("정말 타르코프 하고 싶다" 요청 시 코어 떼어 미니게임화.)

## 코어 게임플레이 루프 — "낮엔 벌고, 밤엔 살아남는다"
마크 고유의 **낮/밤 사이클**을 게임 리듬으로 채택(러스트엔 없는 강점).
레퍼런스 = 〈7 Days to Die〉(낮 파밍·건설 → 습격) + 러스트(기지·레이드) + 타르코프/다크앤다커(파밍·손실·비정한 교전).

```
☀️ 낮 (벌이 시간, 비교적 안전)        🌙 밤 (위협 고조 — 살아남기)
  · POI 파밍·자원 채집                  · 스캐브 활성↑ + 좀비 강화
  · 기지 건설/보강·함정                 · 기지 농성 / 야외 잔류 / 약탈, 다 선택
  · 이동·정찰·선택적 PvP(레이드)        · 사람이 더 무섭다 — 비정한 교전
```

- **상시 야간 위협(매 밤):** 밤이 되면 스캐브·좀비가 강해진다 = **환경 압박**(박자가 아닌 상시 긴장).
  기지 농성·야외 잔류·이동·약탈 모두 자유 선택. 회피 개념 없는 환경 요소.
- **최후의 결전(시즌 마지막날, 단 1회):** ❌정기 대습격 폐기(영속 서버는 게임시간이 24/7 흘러 "게임 7일"=현실 2.3시간이라 박자로 못 씀).
  대신 **시즌 피날레 = 거대 습격이 혼돈을 만드는 무대.** 목표는 *다같이 막기*가 아니라 **그 혼돈 속에서 템 확보하고 살아남기.**
  동맹 방어·솔로 생존·약탈/배신 전부 유효. → **시즌 엔드게임 겸함**: 2주간의 모든 파밍·기지·장비가 *이 밤을 향한 대비*가 된다. (상세 → 「시즌 피날레 — 최후의 결전」.)
- **기지 습격 메커니즘:** 좀비가 **플레이어 타게팅 + 경로 블록 파괴** → 두 전략 모두 허용:
  - **기지 농성:** 벽·함정·고지대 활용, 단 좀비가 벽 부숨 → 기지 손상·수리 비용.
  - **야외 임시 농성:** 기지 보존·기동성, 단 엄폐 없고 죽으면 손실(밤엔 스캐브 협공).
  - 핵심: 블록 파괴가 있어 "벽 치고 무적"이 불가 → 방어가 긴장됨.
- **형평성:** 최후의 결전은 **단 1회 고정 시간**이라 "다같이 모이는 날"로 자연 해결. 상시 야간 위협은 환경이라 회피 개념 없음.
  → 정기 대습격 폐기로 복잡한 스냅샷·오프라인 시뮬레이션 **불필요**(1인 운영 부담 감소).

## 시즌 피날레 — 최후의 결전 (상세)
시즌 마지막날 **단 1회**. 2주간의 모든 파밍·기지·장비·준비가 *이 밤*을 향한 대비. 거대 습격(좀비 호드 + 스캐브)이 몰려오는 혼돈의 클라이맥스이자 시즌 엔드게임.

- **근본 형태 = 농성(기본) + 탈출(특별 옵션) 하이브리드.** 생존(클리어) 경로는 둘이고, 둘 다 유효하다. 약탈·배신·각자도생도 그대로 — 강제 협력은 없다(혼돈 무대 성격 유지).
  - **🛡️ 농성 (누구나 가능):** 결전이 끝날 때까지 버티면 → 칭호 **`[최후의 생존자]`**.
  - **🚁 탈출 (준비한 자만, 최상위 목표):** 시즌 내내 **특별 미션 + 희귀 재료**를 모아 **탑승 티켓**을 제작 → 결전 때 **구조 헬기** 좌석에 탑승 → 칭호 **`[희망을 품고]`** + **배낭(인벤) 전부 다음 시즌 이월**(아래 참조).

- **🚁 탈출 경로 — 구조 헬기 + 탑승 티켓 (시즌 최상위 목표):**
  - 탈출은 "버티기 회피 꼼수"가 아니라 **농성보다 높은 목표.** 시즌 동안 **특별 미션 체인 + 희귀 재료 수집**으로만 **탑승 티켓**을 만들 수 있다.
    - **티켓 = 3겹 게이트:** ①설계도/단서(미션 체인으로 해금) ②핵심 부품 N종(*각각 다른* 고티어 POI·위협에서만 드랍 → 맵 전역 탐험 강제) ③작업대 조립. = 시즌 장기 과제.
  - **1티켓 = 1인 = 1좌석.** 연합이 다같이 탈출하려면 **인원수만큼 티켓을 모아야** 한다(혼자 살 것인가, 동료 몫까지 만들 것인가 = 비정한 톤의 드라마).
  - **좌석 한정 = 탈출 캡.** 결전 때 맵에 **구조 헬기**가 추출 지점에 옴 — 총 좌석 수가 시즌 탈출 정원. 티켓이 있어도 좌석이 차면 못 탐 → **탈출 자체가 경쟁.**
  - → 농성파는 **기지에 투자**, 탈출파는 **맵을 헤집고 다님**(미션·부품) = 시즌 내내 **두 갈래 목표** 공존(플레이 분기).
  - 결전이 시작되면 추출 지점이 열린다 → **혼돈을 뚫고 도달 + 탑승 대기까지 험난**(가는 길·대기 중 무방비). **탈출도 고난도** — 그래서 더 높은 목표.
  - ⚠️ 미정: 미션 체인 내용·개수, 부품 종류·수급 POI 배치, 헬기 대수·좌석 수(탈출 캡)·추출 지점 위치, 탑승 소요시간, 좌석 선점 규칙.

- **🛡️ 농성 경로 — 방벽이 핵심:**
  - **방벽은 결전 중에도 유지된다**(평소 밤처럼 좀비가 못 뚫는 안전벽이 아님). 단 결전 습격은 강력 → **몰려오는 좀비·스캐브의 공격으로 방벽이 깨질 수 있다.**
  - → **기지 레벨을 최대한 올려 방벽을 강화 + 다중 중첩(다중방벽)** 하는 것이 농성 준비의 핵심. 「기지·레이드 시스템」의 다중방벽·내구 업그레이드가 바로 이 밤에 빛을 본다.
  - 다중방벽 = **시간 벌기**: 바깥 겹부터 깨지며 안쪽 코어까지 시간이 걸린다 → 그 사이 방어. 다 깨지면 내부 바닐라 → 코어 노출.
  - **야외 농성**도 선택 가능(기지 보존·기동성, 단 엄폐 없음·스캐브 협공) — 코어 루프의 농성/야외 선택이 결전에서 증폭 적용.

- **규모 / 웨이브 / 스캐브 랭크:**
  - **점증 웨이브:** 초반 Rookie 스캐브·일반 좀비 다수 → 후반 **Expert 스캐브 + 강화 좀비** 비중↑ (시즌 최고 난도).
  - 스캐브 랭크 = **[TACZ] NPCs Rookie→Expert**를 결전 웨이브에 배치(POI 티어 배치와 동일 메커니즘, 결전은 최상위 난도). 진짜 총 쏘는 인간형이 좀비와 함께 압박.
  - ⚠️ 미정: 웨이브 수·간격·규모, 랭크별 비율, 특수/보스 좀비 유무, 결전 총 지속시간.

- **발동 / 형평성:**
  - **고정 시각, 시즌 마지막날 단 1회** → "다같이 모이는 날"로 형평성 자연 해결(스냅샷·오프라인 시뮬 불필요).
  - 시즌 내내 **결전 D-N 카운트다운**이 보여 2주 준비를 이 밤으로 수렴. ⚠️ 정확 시각대 미정(현실 저녁 피크타임 권장).
  - 참여 안 하면(오프라인) 칭호 없음 — 결전은 **참석하는 자의 무대.**

- **보상 — 농성=명예 / 탈출=명예+이월 (비대칭):**
  - **농성 성공 → `[최후의 생존자]`** 칭호(명예) + **보안함·통계·칭호 이월**(배낭 인벤은 리셋). 비정한 톤·공정성에 부합.
  - **결전 중 사망 / 미접속 → 이월 없음(빈손).** 결전 생존이 모든 이월의 관문 → 상세 표는 「시즌 와이프 — 이월 정책」.
  - **탈출 성공 → `[희망을 품고]`** 칭호 + **🔑 탈출 순간의 배낭(인벤) 전부를 다음 시즌으로 보존(이월).** 와이프를 넘어 자산을 가져가는 **유일한 경로** = 탈출이 최상위 목표인 이유.
  - 비대칭의 의미: 농성은 "이 시즌을 끝까지 살아냈다"(명예), 탈출은 "다음을 향해 무언가를 가져갔다"(희망). 칭호 이름과 정합.
  - **이월 범위 = 탈출 시 배낭(인벤) 전부, 상한 없음(잠정·넉넉).** 근거: **시즌2 운영 여부 자체가 미확정**이라 지금 빡빡하게 잡을 이유 없음 + 좌석 한정·고난도로 소수만 가능. (기지 레벨은 와이프 대상이라 이월 불가.)
  - ⚠️ **시즌 간 눈덩이 재검토 여지:** 시즌2 운영이 확정되면 이월이 탈출자에게 다음 시즌 부트스트랩이 됨 → 그때 신규·농성파 균형 보고 **상한/범위 조정 가능**(현재는 넉넉 유지).

## 게임 성격 / 밸런스 철학 (무게중심: 균형)
- **무게중심 = 균형 (PvE 받침 + PvP 스파이스).** 본체는 **"험한 세계에서 살아남기"**(파밍·생존·방어,
  **비제로섬** — 누구나 *혼자서도* 성장 가능)이고, 그 위에 **기지 레이드·약탈을 양념**으로 얹는다.
  MAD(상호확증파괴)는 *유일한 길*이 아니라 *선택적 스파이스*.
- **🎭 게임 톤 = 비정한 자유 서바이벌** (다크앤다커/타르코프/DayZ st).
  > "험한 세계다. 혼자 살아남아도, 믿는 자와 뭉쳐도 된다. 하지만 믿는 자 빼곤 다 적이다."
  - 동반 기지 방어·솔로 생존·총 하나로 파밍러 약탈, **전부 유효 — 강제하는 정답이 없다.**
  - 균형과 모순 아님: PvP의 **빈도는 양념**(시간 대부분은 PvE 생존), 하지만 사람을 만나는 **순간의 톤은 비정.**
- **왜 PvE 받침:** 소규모 지인(캐주얼 다수)에서 순수 제로섬 MAD는 **눈덩이 붕괴**(강자 독주·약자 이탈 → 생태계 붕괴)를
  부른다. 러스트는 대형 서버라 버티지만 우리 규모(5~15명)엔 치명적 → **PvE 비제로섬 축을 받침대로.**
- **야간/습격 = 공동 압박이되 강제 협력은 아님.** 같이 막든 각자 살아남든 자유. "공동의 적"이 협력을 *허용*하지만 *강요*하진 않음.
- **눈덩이 완충 장치 (필수 — 소규모 생존의 관건):**
  - 오프라인 레이드 보호(미접속 시 기지 보호/약화)
  - 손실 상한 + 보안 슬롯(죽어도 일부 보존, 타르코프 secure container) — 약탈이 자유롭되 *다 못 뺏게* 해 PvE 받침 보호
  - 약자 따라잡기(부활 보너스·안전 시작 지역)
  - 레이드 비용(남 기지 털려면 자원 소모 → 무분별한 학살 억제)
- **협력/동맹 = 옵션이되 구속 없음.** 팀(2v2 등) 가능하지만 **배신도 자유**(다크앤다커 파티 톤).
- **시즌 엔드게임:** ✅ **최후의 결전으로 해결**(코어 루프 참조) — 습격 혼돈 속 각자도생·약탈 무대 + 생존자 = 시즌 영웅.

## POI (전리품 거점) — 낮 파밍의 목적지
테마 = **현대 폐허**(버려진 연구소·군사시설·폐도시). 마크 기본 구조물(고대도시·정글사원)은 **판타지라 테마 불일치** → 비추.

- **구현 = 자동 생성 메인 (맵 직접 제작은 안 함):**
  - **자동 생성 모드**(Lost Cities / ChaosZPack / Underground Bunkers) = 세계의 **양·분위기**를 깔아줌(폐도시·벙커 흩뿌림). 노동 0. **이것만으로 충분히 굴러감.**
    ※ Lost Cities는 "전체 도시"가 기본 → **rarity 조정**해 "자연 지형 + 드문 폐도시 POI"로(2000² 자연 월드에 맞게).
  - **🔑 시즌제 시너지:** 와이프 시 **새 시드로 재생성 → POI 자동 재배치.** 매 시즌 새 맵이 공짜. (맵 제작 = 1인 개발 최대 시간 하마라 통째 우회.)
  - **시그니처 POI(선택, 직접 빌드 강제 아님):** 깊이를 더하고 싶을 때만. **직접 빌드 대신** ①**커뮤니티 schem 다운로드** 배치 ②자동 생성 대형 구조물에 **루트 티어링만** → *맵 노동 0 유지*.
- **💡 POI 티어 = 폭약 재료 통제 경로 (계속 미뤄둔 ⚠️ 해결):**
  - **저티어 POI**(흔한 자동 생성 벙커·민가): 기본 루트, 안전.
  - **고티어 POI**(자동 생성 대형/희귀 구조물 — 군사시설·연구소, 빌드 없이 루트 티어링으로 지정): 위험(강한 스캐브)·고급 장비·**폭약 재료 독점 공급**.
  - → 폭약 재료가 위험한 고티어에서만 나오니 **레이드 비용이 자연 통제**(파밍 위험 ↔ 레이드 능력 연결).
- **착수 시 1순위 검증:** ChaosZPack(컨셉 겹침)의 1.21.1 호환·다른 모드와의 충돌. 미확정: POI 티어 수·루트 테이블·스캐브 배치·schem 시그니처 개수.

## 기지 · 레이드 시스템
바닐라 건축 + 철문/레드스톤만으론 보호가 약함(또는 무적). **시스템적 2단계 보호 + 비용 있는 레이드**로 설계.

- **기지 코어(설치 아이템):** 설치 지점 기준 방벽 생성. **보호 범위 = Y30(하한) ~ 빌드 상한(319)까지** — **위로는 끝까지 보호, Y30 아래(깊은 지하)는 무방비**(그 밑으로 터널·굴착 가능). 1.21.1 좌표: 해수면 ~63, 지표 ~63–70이라 Y30은 명확한 지하.
  - 의미: "지하 깊이 숨어도 소용없다" + 밑으로 지나가는 사람 배려. (※ 엔더 차원 봉쇄로 엘리트라 입수 차단 → 공중 무한도주/스카이베이스 악용 방지, 그래서 위로 끝까지 보호해도 안전.)
  - **코어 파괴 = 기지 완전 파괴 + 기지 레벨 초기화** → 코어가 최우선 타겟, **은닉이 곧 방어.** (러스트 Tool Cupboard 레이드를 더 가혹하게.)
  - **코어 재질 = 옵시디언급(커스텀 텍스처):** 즉시 파괴 불가, 깨는 데 시간 소요 → 방어자가 달려와 막을 기회.
  - **위치 전략:** 영역 깊숙이 숨기기 vs 일부러 높은 곳(탑 꼭대기 등) = 심리전. "완벽한 은신처는 없다."
  - 가혹함의 받침: 완파·레벨 초기화는 가혹하지만 옵시디언+은닉+방벽으로 *드물게* 일어나야 톤이 삶.
    **배낭 레벨·보안함은 코어 파괴와 무관하게 유지**되므로 완파당해도 "맨몸 재시작"은 아님(재기 가능).
- **연합(동맹):** 개인당 코어 1개 유지, 연합 = **권한 공유 그룹**(연합원끼리 서로 보호 영역 접근·건축 허용, 약탈·코어 파괴 불가 / 외부인은 적).
  - **연합 규모 상한 = 최대 4명 (확정).** 단일 대연합의 서버 지배(눈덩이 끝판왕) 방지, 여러 세력 경쟁 구도 유도 = 눈덩이 완충.
    - 근거: 동접 ~30명 ÷ 4 = **7~8개 세력** 가능 → 다극 경쟁 확보. 캐주얼 지인 파티의 자연 크기.
    - **헬기 정합:** 연합 4명 ↔ 구조 헬기 4인승이면 "티켓 4장 모아 연합 다같이 탈출" 서사와 맞물림(좌석 수는 밸런싱 때 확정, 4를 기준점). → 「시즌 피날레」 탈출 경로 참조.
  - 코어를 모으면 **연합 마을**(클러스터), 흩어지면 **분산 동맹** — 둘 다 자연 발생.
  - **배신·탈퇴 자유**(비정한 톤): 탈퇴 시 권한 회수, 연합원 배신 가능(다크앤다커 파티 톤). "연합해도 안심 못 한다."
  - 의미: 연합 = 약자의 생존 수단이자 강자의 패권 수단 → 규모 상한이 균형추.
- **2단계 보호:** ①**방벽**(특수 폭약으로만 파괴) → 뚫리면 ②**내부 바닐라 블록**(일반 파괴) → ③코어 수색·파괴 → 완파/약탈.
- **은신처 봉쇄 ("완벽한 은신처는 없다"):** **설치는 어디든 자유, 보호만 하한 Y30**(아래는 무방비).
  지하에 지어도 되지만 보호 안 됨 → **들키면 털릴 각오로 지어라.** 설치 금지 같은 하드 제약 없음 — *자유 + 리스크*가 비정한 톤에 맞음. (Y25 설치금지 폐지.)
  - **Y25 이하 자동 초기화 (유지):** 목적 = 기본 자원 순환 + 중후반 재기 수단(지하 봉쇄 명분은 보호 미적용으로 이미 대체).
    - 구현: 같은 시드로 원본 지형 생성해 **Y25 이하 블록만 덮어쓰기**. 전용 모드 없어 커스텀(resetchunks류 로직).
    - 부하 분산: **"손 탄 청크만" 재생성**(플레이어가 Y25 이하 건드린 청크만 추적) → 월드 2000²(약 15,600청크)라도 감당.
    - 안전장치: 초기화 전 **카운트다운 → Y25 이하 플레이어 지상 강제 텔레포트 → 아이템·건축 소실 경고**. 타이밍 = 현실 새벽 등 고정 시각.
  - ⚠️ 매일 부활하는 자원이 **폭약 재료**면 레이드 비용이 무너짐 → 폭약 희귀 재료는 POI 등 **통제된 경로로만**.
- **구현 분담:**
  - 공격(폭약) = **기존 모드 활용** — C4·RPG·강화 TNT 다수 존재(More Explosives / Even More Explosives / Insane TNT 등).
  - 수비(조건부 보호) = **porongun-core 커스텀** — 기존 클레임 모드는 "완전 보호 on/off"뿐이라
    "평소 무적 + 특수 폭약에만 뚫림"은 직접 만들어야 함. **영리한 경로: 클레임 모드(보호) + 보호를 *관통*하는 커스텀 폭약**.
- **기지 = 업그레이드式(고정 단일 크기 ❌):** 아이템으로 영역 선포, 단계별 확장·강화.
  - 예시: 1단계 전초기지 ~12×12 → 2단계 ~18×18 → 3단계 거점 ~25×25 (수치 미정).
  - 효과: **진행 동기**(파밍→강화, 비제로섬 성장) + **방어 트레이드오프**(크면 자산↑·둘레↑ 방어 불리) + 플레이 스타일 분기.
  - ⚠️ 크기 = 방어 난이도. 소규모 인원이라 **작게 시작** 권장(50×50은 둘레 200이라 방어 불가).
- **⭐ "그냥 안 함" 악순환 방지 (이 장르 근본 딜레마):** 레이드가 너무 쉬우면 기지 건설 무의미("어차피 털림") → 시스템 붕괴.
  **2중 안전판으로 방지:**
  - **폭약 고비용**(제작·희귀자원·POI 부품 → 레이드 **빈도** 억제, "작정한 큰 사건"화) = 눈덩이 완충 "레이드 비용".
  - **보안 슬롯 + 손실 상한 + 오프라인 보호**(레이드 **피해** 제한 → 털려도 다는 안 잃음, 재기 가능).
  - 둘 다 눌러야 함: 빈도만 잡고 한 번에 다 잃게 하면 여전히 의욕 붕괴.
  - **업그레이드가 방어에도 기여:** 강화 방벽=내구↑=폭약 더 필요 → *키운 만큼 단단* → "지을 가치" 발생.
- **결정 필요:** 영역 크기·업그레이드 단계 수치, 폭약 종류/제작 비용, 오프라인 보호 정밀도, 좀비-방벽 상호작용
  (평소 밤=좀비 방벽 못 뚫음=안전지대 / 최후의 결전=방벽 무력화 or 특수 좀비로 긴장).
- **MVP:** 클레임 모드(보호) + 커스텀 폭약 1종(관통) + 고정 소형 영역 1~2단계. / **후순위:** 오프라인 정밀 룰·좀비-방벽·폭약 티어.

## 인벤토리 · 무게 (배낭)
타르코프식 "들고 갈 양 제한"을 마크 슬롯 기반으로 구현. **공간(배낭 칸) + 무게(점유 칸)** 결합.

- **배낭 = 영구 업그레이드 (파밍품 아님, 죽어도 유지):**
  | 배낭 | 슬롯 | 비고 |
  |---|---|---|
  | 기본 | 2줄 = 18칸 | 시작 |
  | LV2 | 3줄 = 27칸 | |
  | LV3 | 4줄 = 36칸 | 풀 인벤 |
  - 죽어도 레벨 유지 = **영구 진행 = 재기 동력**(악순환 방지의 엔진).
- **무게 = 점유 칸 시스템 (= "무게 유리 시각화"의 구현):**
  - 아이템 **lore/태그에 무게(점유 칸수)** 부여 → 배낭에 넣으면 그만큼 칸 점유, **초과분은 유리로 막힘.**
  - 예: 대물저격총 = 3칸 → 자기 1칸 + **유리 2칸** = 배낭 3칸 점유.
  - 효과: 큰 총 몇 자루면 배낭이 참 → **"뭘 들고 나갈까" 선택 긴장**(타르코프st 인벤 관리).
- **이동속도 페널티 (무게→속도, 도입):** 배낭 **적재율에 비례**해 이동속도 감소. 가벼우면 페널티 **미미**, 배낭 **LV3(36칸) 만재 시 최대 ~50% 감소**(잠정 — 밸런싱 때 확정). 큰 캡은 아님(완전 정지는 아님).
  - 효과: 점유 칸 긴장 위에 **"많이 들수록 느려진다"** 트레이드오프 추가 — 파밍 욕심 ↔ 기동성/도주력. 약탈 후 과적 상태로 도망가다 따라잡힘 = 타르코프st 긴장.
  - **다리 부상 속도 감소와 합산**(아래 「손실·사망」 정밀 부위 데미지) — 과적 + 다리 부상이 겹치면 치명적 둔화. 이는 **의도된 압박**: 다리 부상은 치료를 강제하고(의료템 소비), 과적은 짐을 버리게 한다. 합산 둔화 강도·완전정지 여부 수치는 밸런싱 때.
- **보안 도구함:** 타르코프 secure container(죽어도 안 잃는 칸). **기지 레벨로 칸 확장**(기지·레이드 시스템 참조).
  배낭(들고 가는 총 공간)과 별개 — 그중 *죽어도 지키는 일부*.

## 손실 · 사망 처리
타르코프식 "죽으면 잃는다"를 마크에 구현. 단 영구 진행(배낭·보안함)으로 재기 가능.

- **사망 시:** 장착 장비(총·방어구) + 배낭 내용물 → 그 자리에 **데스 박스** 생성. **보안 도구함 내용물·배낭 레벨은 유지.**
- **상자 식별 (타르코프 "검사 시간"):** 약탈자가 데스 박스를 열면 내용물이 **유리로 가려짐** → 아이템에 **마우스오버 N초** 지나야 정체 공개.
  - 효과: "급하게 좋은 것만 빼가기" 불가 → 약탈에도 시간·리스크. 그 사이 원주인 회수·다른 적 난입 가능 = 긴장.
- **상자 접근/지속:** **누구나 열 수 있음**(비정한 톤 — 약탈 자유). 지속 **15분 후 디스폰** → 원주인이 되찾으러 오는 **회수전** 살림 + 서버 부하·맵 정리.
- **부위 데미지 (정밀 단계, 도입):** 타르코프식 **부위별 HP + 골절 + 의료 키트.** "맞은 부위 따라 못 씀"을 깊게 — "깊은 게 플레이 경험에 더 재밌다" 판단으로 보류 해제.
  - **부위별 HP:** 머리·몸통·양팔·양다리 개별 HP. 부위 HP가 0이 되면 **골절/기능 상실**(blackout) → 강한 페널티. 머리·몸통 HP 0 = **사망.**
  - **부위별 페널티:**
    | 부위 | 부상/골절 효과 |
    |---|---|
    | 다리 | **이동속도 대폭 감소(절뚝)**·점프 제한 — 무게 페널티와 합산 |
    | 팔 | 조준 흔들림·**장전 속도 저하**(부상 팔로 못 다룸) |
    | 머리 | 치명타·즉사 위험(최저 HP) |
    | 몸통 | 출혈·HP 직격(최대 HP) |
  - **의료 티어 (타르코프 메디킷st):** **붕대/지혈**(출혈 정지) → **부목(splint)**(골절 임시 고정·페널티 완화) → **수술 키트**(부위 HP 회복·골절 완치). 응급처치에 시간·자원·교전 중 리스크.
  - **🔑 다리 부상 = 치료 강제(설계 의도):** 다리 둔화는 **치료 전까지 지속**(무게 페널티와 합산) → 절뚝이며 못 도망감 → **무조건 치료해야** 정상 기동 복귀. 곧 **의료템이 핵심 소모 자원**(휴대·수급·상점이 생존 필수) = "총만큼 약도 챙겨야 한다"는 타르코프st 긴장. 강한 둔화는 답답함이 아니라 *치료를 강제하는 압박*이다.
  - 효과: 교전이 "HP 0 즉사"가 아니라 **부위 무력화의 누적** = 도주·치료·재교전의 결정. 다리 골절+과적이면 도망 불가 → **그 자리에서 치료냐, 버리고 후퇴냐** → 약탈/회수전 긴장 극대화.
  - **구현 = `Vitality Aids` 활용 (기존 모드, 커스텀 최소):** First Aid의 **Fabric판** — 부위별 8영역 HP·자연재생 제거(의료템 강제 ← 우리 설계와 정합)·커스텀 의료템·출혈(hemorrhage) 설정 가능·다리 부상=둔화. **✅ 1.21.1 Fabric 버전 존재**(`vitality_aids-1.0.4-1.21.1`). 우리 정밀 부위 데미지와 거의 1:1 → 🔴(커스텀 GUI)에서 🟡(기존 모드)로 하향.
    - porongun-core 역할 = **무게→속도 감속과 부위 둔화 합산 + 의료템을 경제(상점·드랍)에 편입**하는 오케스트레이션만.
    - ❌ `TaCZ: Bleeding` 애드온(붕대/출혈)은 **1.20.1 Forge 전용**이라 우리 스택(1.21.1 Fabric)엔 부적합 → Vitality Aids로 대체.
    - ⚠️ "골절"을 별도 메커니즘으로 정밀히 원하면 Vitality Aids는 "부위 HP 0→디버프" 근사 → 정밀 연출은 설정/소량 커스텀 여지.
  - ⚠️ 미정(밸런싱): 부위별 HP 수치, 골절 발생 조건, 페널티 강도, 의료 키트 제작·수급, 합산 둔화 상한. 착수 1순위 검증: Vitality Aids 1.21.1 안정성·TaCZ/타 모드 충돌.
- **보험: 미도입.** 보안 도구함 + 손실 상한으로 완화는 충분 + 비정한 톤("잃으면 잃는 것")에 부합.

## 진행 축 요약 (영구 vs 거점)
| 축 | 성격 | 잃는 조건 |
|---|---|---|
| **배낭 레벨** | 영구(캐릭터) | 안 잃음 |
| **보안 도구함 내용물** | 영구(일부) | 안 잃음 |
| **기지 레벨**(내구·크기·다중방벽·보안칸·상점 해금) | 거점 | **코어 파괴 시 초기화** |
| 일반 인벤·장착 장비 | 휘발 | 사망 시 상자 드랍 |
→ 영구 축(배낭·보안함)이 *재기 동력*, 거점 축(기지)이 *지킬 가치 있는 자산*. 다 잃어도 맨몸 재시작은 아님.
※ 위 "안 잃음"은 **시즌 내** 기준(일반 사망에 안 잃음). **와이프를 넘는 이월은 결전 생존이 전제** → 아래 「시즌 와이프 — 이월 정책」.

## 시즌 와이프 — 이월 정책 (결전 생존이 전제)
와이프 시 월드·기지·일반 진행은 리셋된다. **무엇을 다음 시즌으로 가져가는지는 「시즌 피날레」 결과로 갈린다** — 핵심은 **결전에 참여해 살아남아야 뭐든 이월된다**는 것.

| 시즌 종료 상태 | 다음 시즌 이월 |
|---|---|
| 🚁 **탈출 성공** | **배낭(인벤) 전부 + 배낭 레벨** + 보안함 + 통계 + 칭호 |
| 🛡️ **농성 생존**(결전 끝까지 생존, 탈출 X) | 보안함 + 통계 + 칭호 (배낭 인벤·레벨 리셋) |
| 💀 **결전 중 사망 / 미접속** | **이월 없음 (빈손)** — 통계·칭호·보안함조차 X |

- **결전 생존 = 이월의 관문.** 죽거나 안 오면 보안함·통계·칭호조차 못 가져감 → **"마지막 밤에 살아남아라"가 시즌의 절대 동기.**
- 역할 구분: **명예(통계·칭호)** = 생존자 공통, **보안함** = 생존자가 지켜낸 stash, **배낭(인벤 전부 + 레벨)** = 탈출자 전용 특권(최상위 보상).
- **배낭 레벨(슬롯 18→27→36)은 탈출자만 보존**, 나머지(농성 생존자 포함)는 리셋 → 매 시즌 재성장. 탈출 = 다음 시즌 인벤 용량까지 앞서가는 진짜 특권.
- 기지·기지 레벨·일반 인벤은 누구든 와이프(거점은 영속 월드와 함께 리셋).

## 경제 · 총 획득 · 온보딩
- **총 획득 (TaCZ 제작 시스템 기반):**
  - **작업대 = 바닐라 재료로 제작**(통나무+철+철블록 → 바닐라 크래프팅 테이블). 진입 허들 낮음. 3종: Gunsmith / Attachment / Ammo Table.
  - **기본 총**(티어 0~1): 청사진 없이 제작 → 초반 진입(권총·기본 소총). **PvE 받침·재기.**
  - **고급 총**(티어 2+): **청사진(POI 루트 파밍)으로 해금** → 작업대 제작. 청사진=레시피 영구 학습(죽어도 유지=재기 동력), 총 완성품은 소모(죽으면 데스박스). **청사진을 POI 티어 루트에 배치** = "좋은 총=진행 보상".
  - **시즌 시너지:** 와이프 시 `/gg clearRecipes`로 청사진 리셋 → 매 시즌 무기 해금 재파밍.
  - 탄약 = Ammo Table 제작 + 상점 / 폭약 = 고티어 POI 재료 제작(레이드 비용) / 드랍 = 스캐브·데스박스 약탈.
- **화폐 = 커스텀 화폐**(스캐브·POI 드랍 = 타르코프 루블st). **파밍이 곧 돈.**
- **상점 = 메뉴 아이템 기반 (중심 허브 건물 없음):**
  - **접근 제약 = 자기(또는 연합) 기지 안에서만 거래 활성 (확정).** 허브 없이 *기지가 거래 거점*, 긴장 유지.
  - 등급 해금 = **기지 레벨**(고레벨일수록 고급 무기·탄약 구매).
  - 함의: 신규는 **코어 설치 전엔 거래 불가** → 온보딩 후 첫 목표 = "코어 박고 거점 만들기"(자연스러운 진입 동선). 코어 설치 아이템은 온보딩 지급/쉬운 제작으로 매끄럽게.
- **메뉴 아이템 (핫바 9번 칸):** 슬롯 고정·드랍 불가·무게 0. 기지 관리·상점·정보 GUI. **정보·기지관리는 어디서나, 상점 거래는 기지 안에서만.**
- **온보딩:** 격리 로비 → **서버 설명 + 디스코드 인증** → **랜덤 좌표 스폰**(신규가 기존 세력 옆에 안 떨어지게 분산). 디스코드 인증 자체 구현(봇 연동 방식은 착수 시; porong-gun 독립).
- **중심 허브 없음 → 개인 기지가 각자 허브.**
