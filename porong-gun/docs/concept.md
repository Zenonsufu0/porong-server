## 한 줄 컨셉
경계 있는 **영속 월드**에서 **기지를 짓고**, 버려진 시설을 **파밍**하고, 서로 **약탈**하고,
**밤의 습격**을 막아내는 총기 서바이벌. **시즌(와이프) 단위** 운영.
= 마크형 **"러스트 + 타르코프"**.

## 왜 이 방향인가 (핵심 결정)
- **마크 고유 메커니즘(건설·파괴·채집·PvE) × 총** = "다른 총겜 하면 되지"에 대한 답.
  배그·러스트·타르코프 **어느 하나로도 대체 불가**한 조합. 차별점은 총이 아니라 **부서지고 지어지는 세계.**
- **시즌(와이프) 운영에 최적.** 러스트식 1~2주 와이프 주기 = "2주 시즌"과 정합. 시즌 종료=맵 와이프+시즌2 보강.
- **코어 재사용성(중요):** 영속월드 코어(총·무게·부위·손실·기지·파밍 시스템)를 만들어두면,
  훗날 **타르코프/러스트/배틀로얄 등 분리형 모드로 재활용** 가능. → 과거 "멀티모드 셀렉터" 방향은
  폐기가 아니라 **미래 확장 옵션**으로 보존. ("정말 타르코프 하고 싶다" 요청 시 코어 떼어 미니게임화.)

## 비전 — 7기둥
| # | 기둥 | 영감 |
|---|---|---|
| 1 | **경계 맵 + 플레이어 기지(은신처) 건설** | 러스트 |
| 2 | **POI 파밍** — 버려진 연구소·군사시설 전리품 | 타르코프 |
| 3 | **기지 레이드·약탈** — 남의 기지 파괴/약탈 | 러스트 |
| 4 | **스캐브(적대 NPC)** — POI·기지에서 스폰 | 타르코프 |
| 5 | **밤 좀비 습격 이벤트** — 은신처 단위 방어 | 좀비 서바이벌 |
| 6 | **사망 손실 = 상자 드랍 + 식별** (유리→마우스오버 시간 경과→정체 공개) | 타르코프 검사시간 |
| 7 | **무게 시각화** — 남은 무게를 인벤에 유리로 채워 표현 | 타르코프 |

### 난이도 메모 (1인 기준)
| 기둥 | 난이도 | 비고 |
|---|---|---|
| 기지 건설 / 사망 상자 드랍 | 🟢 낮음 | 마크 기본 블록 / death-chest류 |
| POI 파밍 / 스캐브 | 🟡 중간 | 구조물+루트테이블 / 몹 커스텀 |
| 기지 레이드 / 밤 좀비 습격 | 🟠 중상 | 레이드 룰 밸런스 / 좀비는 블록 못 부숨→커스텀 AI 필요 |
| 상자 식별 UI / 무게 유리 시각화 | 🔴 높음 | 커스텀 GUI |

## 플랫폼 / 기술 스택 (확정 — 영속월드에도 그대로 유효)
- **로더: Fabric 1.21.1 / Java 21** (porong-mon 환경 재사용 — fabric-loom·빌드설정·스크립트·노하우).
  장기 정출 의지 생기면 그때 NeoForge 신규 시작.
- **총기 모드: TaCZ: Refabricated(메인) / Vic's Point Blank(백업).** GPLv3·experimental →
  내부 Mixin 지양(공개 API만), 착수 전 테스트 검증 필수. 비공개 내부 사용이라 배포 의무는 실무상 무관.
- **중심 모드: `porongun-core`** (group `kr.porong.gun` 예정). 총은 TaCZ가 담당, 우리는
  **서바이벌 시스템(기지·파밍·손실·무게·부위·이벤트) + 룰**만 만든다.

## 운영 전제
- **시즌(와이프) 운영.** 기본 2주, 유저 원하면 연장. 시즌 종료=와이프+시즌2 보강.
- 지인 전용 비공개(공개성 X), 동접 소규모, **1인 개발 + 1인 운영**.
- 다른 Porong 서버와 완전 독립. 착수는 **porong-mon → porong-rpg 이후**(시간 여유 확보).

## 게임 성격 / 밸런스 철학 (무게중심: 균형)
- **무게중심 = 균형 (PvE 받침 + PvP 스파이스).** 본체는 **"험한 세계에서 같이 살아남기"**(파밍·생존·방어,
  **비제로섬** — 누구나 성장 가능)이고, 그 위에 **기지 레이드·약탈을 양념**으로 얹는다.
  MAD(상호확증파괴)는 *유일한 길*이 아니라 *선택적 스파이스*.
- **왜:** 소규모 지인(캐주얼 다수)에서 순수 제로섬 MAD는 **눈덩이 붕괴**(강자 독주·약자 이탈 → 생태계 붕괴)를
  부른다. 러스트는 대형 서버라 버티지만 우리 규모(5~15명)엔 치명적 → **PvE 비제로섬 축을 받침대로.**
- **밤 좀비 습격 = 공동의 적.** PvP로 으르렁대다 밤엔 같이 막기 → 긴장 환기·일시 협력·PvP 피로 완화.
  지인 보이스와 궁합 최고.
- **눈덩이 완충 장치 (필수 — 소규모 생존의 관건):**
  - 오프라인 레이드 보호(미접속 시 기지 보호/약화)
  - 손실 상한 + 보안 슬롯(죽어도 일부 보존, 타르코프 secure container)
  - 약자 따라잡기(부활 보너스·안전 시작 지역)
  - 레이드 비용(남 기지 털려면 자원 소모 → 무분별한 학살 억제)
- **협력/동맹 옵션:** 1인 제로섬보다 팀(2v2 등)이 사회적으로 재밌고 덜 잔혹 → 소규모에 적합.
- **시즌 엔드게임(열린질문):** 군비경쟁에 방향을 주는 공동 목표(예: 시즌 말 보스 POI 합동 공략)
  + 개인 경쟁(누적 부·생존 기록) 혼합 검토.

## 스코프 원칙 (생명)
- 풀 7기둥은 1인 단기 **불가** → **"북극성 비전"으로 두고 MVP를 역산.**
- 차별화는 **거의 공짜**(마크 기본 건설/파괴 룰 허용)부터 시작.
- 비싼 것(상자 식별 UI, 무게 유리 시각화, 기지 레이드 밸런스, 밤좀비 커스텀 AI)은 **시즌2+**.
- **MVP 범위는 의도적 미확정** — 컨셉 다듬으며 천천히(시간 여유 있음).

## 열린 질문 (미확정)
- MVP 범위(보류 중).
- 기지 소유권/레이드 룰(오프라인 보호·점령 조건 등).
- POI 맵 제작 방식(수작업 건축 vs 구조물 생성 데이터팩).
- 스캐브/밤좀비 구현 수준(바닐라 몹 커스텀 vs 기존 봇/레이드 모드).
- 손실 강도(상자 회수 가능성·보험 유무·보안 슬롯).
- 시즌 와이프 시 보존/초기화 범위(스킨·통계만 보존 등).
