## 2. 🛠️ porongun-core 커스텀 구현 범위 (모드가 못 하는 것)
기존 모드 = 받침(총·부위데미지·스캐브·POI·좀비·클레임). **porongun-core = 그 위의 우리 룰.** 포트폴리오 성격이라 묵직함=실력 증명 자산, 리스크는 "무겁다"가 아니라 "MVP 순서 필요".

| 모듈 | 내용 | 의존 모드 |
|---|---|---|
| **기지·레이드** | 코어·기지 LV1~5·보안칸·밀폐 외피 방벽 판정·코어 자물쇠 | OPaC(권한·파티) |
| **돌파** | 관통 폭약·돌파 게이지·총격 보조·좀비-방벽 | (커스텀 폭발) |

| **화폐·GUI** | 물리 화폐(스택 한도)·상점/배낭/기지/정보 GUI(코어 우클릭·Shift+F) | — |

| **POI 루트** | 상자 후보 풀·리필 재롤·드랍 테이블·청사진 배치 | Lost Cities |

## 3. 🪜 MVP 단계 — 역산 마일스톤 (잠정, 사용자 확정 필요)
풀 7기둥은 1인 단기 불가 → **북극성 비전으로 두고 역산.** 의존 순서로 단계화(각 단계가 "플레이 가능"한 증분):

| 단계 | 범위 | 플레이 경험 | 비고 |
|---|---|---|---|
| **M0 검증** | §1 게이트 7항 실측 + 인벤 GUI 프로토타입 | (개발) | 통과해야 착수 |
| **M1 코어 루프** | 총(TaCZ)·부위데미지(First Aid)·스캐브(POI)·파밍·**화폐·상점**·사망 데스박스·무게 | **"POI 돌고 쏘고 죽고 다시"** = 타르코프 코어 | 기지·레이드·결전 제외 |
| **M2 기지·레이드** | 코어·기지 레벨(LV1~5)·밀폐 외피 방벽·**관통 폭약 T1/T2/T3 전부**·**고급총 전부(배틀·저격·대물)+빈도 분리**·코어 자물쇠·보안칸·**Y25 리셋**(레이드 동기 노브) | **"기지 짓고 털고"** = 러스트 축 | 개발 착수는 폭약 T1·저레벨부터 점진(base_raid MVP) |
| **M3 결전·탈출** | 결전 웨이브·좀비-방벽·헬리패드 추출·탈출 부품/티켓·와이프 이월 | **시즌 클라이맥스** | 시즌 운영 성립 |
| **M4 튜닝·선택** | armor_ignore 곡선·오프라인 정밀 룰·유리 검사시간 차등·**Y25 주기/차등 튜닝**·차량 등 | 정밀화·추가 콘텐츠 | 라이브 보정·시즌2+ |
- **🎯 1차 출시 범위 = M3 (결전까지, 확정 2026-06-10):** 시즌(와이프)이 성립하려면 결전·탈출·이월이 있어야 완결 → 풀 비전을 1차에 담는다(메모리 "MVP 칼질 자제, 풀 비전" 성향 정합).
  - **🔑 와이프 구조 = 시즌1도 풀 콘텐츠 (06-10 정정):** 매 시즌 0→LV5 압축이라 **시즌1에서도 2주 안에 LV5 요새가 나옴** → 그 요새 레이드엔 **T3 폭약**, 강화 방벽엔 T2 필요. 고급총(대물·저격)도 시즌1에 다 등장. ⇒ **폭약 3티어·고급총 전부가 M2(시즌1 콘텐츠)에 포함**("시즌 거듭하며 고레벨 흔해짐"은 영속 서버 논리라 폐기). **M4 = 콘텐츠가 아니라 정밀 튜닝(M1~M3은 단순값 시작) + 선택 콘텐츠(차량).**
  - **개발 착수 vs 출시 범위 구분:** base_raid "MVP=폭약 T1 1종"은 *개발 착수 최소 단위*(T1부터 만들고 T2/T3 붙임), **출시(M3) 범위는 폭약 3티어·고급총 전부.**
  - **⚠️ 정직한 리스크(착수 시점 스코프 경고):** M3 풀 시즌 = **1인 단기 구현 부담 최대 + 출시 지연 리스크.** 완화 = **개발은 M1→M2→M3 순차**(각 단계가 플레이 가능한 증분 → 내부 알파/클로즈드 테스트로 검증하며 전진), **출시는 M3 완성 시.** 한 번에 다 만들지 말고 의존 순서대로.
  - **🪜 순차 개발 = 위험 분산:** M1(모드 받침 위주=비교적 빠름)에서 코어 루프 검증 → M2(밀폐 감지·폭약=난도↑) → M3(결전 오케스트레이션=최난도). 각 단계 끝에서 "여기까지도 재미있나" 점검 = 중도 피벗 여지.
- ⚠️ **단, porong-gun 착수 자체가 porong-mon → porong-rpg 이후**(시간 여유 확보, [`world.md`](world.md) 운영 전제).

- **🟠 1인 운영 부하:** 큰 맵·Y25 리셋·결전 오케스트레이션 = 서버 부하 → 작은 맵 시작·단계적 확장([`world.md`](world.md)).
