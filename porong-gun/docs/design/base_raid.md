# 설계 — 기지 · 레이드 · 연합

> 비전 SoT: [`../concept.md`](../concept.md) · 관련: [`finale.md`](finale.md)(농성·방벽), [`survival.md`](survival.md)(보안함), [`../platform.md`](../platform.md)(OPaC·관통 폭약).

바닐라 건축 + 철문/레드스톤만으론 보호가 약함(또는 무적). **시스템적 2단계 보호 + 비용 있는 레이드**로 설계.

## 기지 코어 (설치 아이템)
설치 지점 기준 **보호 영역(~100×100 고정)**을 선포 — **v2(2026-06-10): 영역 안 플레이어 블록이 코어 레벨 방어 내구를 가짐**(밀폐 외피 폐기 → 아래 「방벽 모델 v2」). **보호 범위 = Y30(하한) ~ 빌드 상한(319)까지** — **위로는 끝까지 보호, Y30 아래(깊은 지하)는 무방비**(그 밑으로 터널·굴착 가능). 좌표 기준: 해수면 ~63, 지표 ~63–70이라 Y30은 명확한 지하.
- 의미: "지하 깊이 숨어도 소용없다" + 밑으로 지나가는 사람 배려. (※ 엔더 차원 봉쇄로 엘리트라 입수 차단 → 공중 무한도주/스카이베이스 악용 방지, 그래서 위로 끝까지 보호해도 안전.)
- **코어 파괴 = 기지 완전 파괴 + 기지 레벨 초기화** → 코어가 최우선 타겟, **은닉이 곧 방어.** (러스트 Tool Cupboard 레이드를 더 가혹하게.)
- **코어 재질 = 옵시디언급(커스텀 텍스처):** 즉시 파괴 불가, 깨는 데 시간 소요 → 방어자가 달려와 막을 기회.
- **위치 전략:** 영역 깊숙이 숨기기 vs 일부러 높은 곳(탑 꼭대기 등) = 심리전. "완벽한 은신처는 없다."
- 가혹함의 받침: 완파·레벨 초기화는 가혹하지만 **은닉 + 두꺼운 벽(블록 내구)**으로 *드물게* 일어나야 톤이 삶. **배낭 레벨·보안함은 코어 파괴와 무관하게 유지**되므로 완파당해도 "맨몸 재시작"은 아님(재기 가능).

### 코어 제작 · 레벨 — v2 (재설계 2026-06-10, 블록 내구 모델)
> **🔄 v2 전환:** 밀폐 외피·밴드·영역 크기 캡 모델 폐기 → **"코어 레벨이 영역 내 블록에 방어 내구 부여 + 공격 도구 등급 해금"**(러스트st 단단한 블록, 재료 무관, MC 자유 건축). 사유: MC 샌드박스 보존 + 종심 캡 답답 해소 + 밀폐 감지(flood-fill) 착수 리스크 제거. **base_raid 전체 v2 재매핑 완료(2026-06-10), economy #2 재조정 남음.**
- **🔑 영역 = 코어 중심 ~100×100 고정 (레벨 무관):** 레벨업마다 영역 안 바뀜 = **리빌드 불필요**(새로 짓기 귀찮음 해소). 그 안에서 **아무 블록으로 자유 건축**. 영역끼리 **겹침 금지**.
- **LV1 코어 제작(잠정):** 철 블록 ×2 + 레드스톤 블록 ×1 + 철 ×4. 온보딩 지급, 재제작 쉬움(이미 코어/연합 코어 있으면 제작 불가=1코어).
- **🧱 레벨 = ① 영역 내 블록 방어 내구↑ ② 공격 도구 등급 해금(상점) ③ 보안칸.** (재료 무관 = 나무든 옵시디언이든 코어 레벨이 내구 결정, 미관만 차이.)

| 레벨 | 영역 | 블록 방어 내구 | 공격 도구 해금(상점) | 보안칸 | 상점 | 업글 재료(잠정) |
|---|---|---|---|---|---|---|
| **LV1 전초기지** | 100×100 고정 | 낮음 | 효율5(바닐라) | 2 | 기본 | — |
| **LV2** | 〃 | ↑ | **효율6** | 3 | 중급 | 철블록8 + 텅스텐16 |
| **LV3 거점** | 〃 | ↑↑ | **효율7** | 4 | 중급 | 텅스텐32 + 다이아8 |
| **LV4** | 〃 | ↑↑↑ | **효율8** | 5 | 고급 | 텅스텐48 + 다이아12 |
| **LV5 요새** | 〃 | 최고 | 효율8 + 고급 폭약 등급 | 6 | 최고급 | 텅스텐64 + 🪖군용16 + 🔌전자8 |

- **🔑 레벨업 이유 = ① 내 블록 단단해짐(방어) + ② 상대 벽 부술 도구 등급↑(공격).** 영역·구조 그대로(리빌드 X). 바닐라 최대 효율5로는 고레벨 내구 못 깨니 **효율6/7/8 도구를 레벨별 상점에서 판매**.
- **보안칸 2/3/4/5/6**: 티켓 1칸 점유라 실여유 1~5칸 = 칸 경쟁.
- **크기 고정**이라 "크게 지어 캡 회피"가 아니라 **레벨로 단단해짐** = 레이드 비용은 블록 내구·도구로(↓「방벽 v2」·돌파).
- **💰 업글·수리 = 재료(게이트) + 화폐(부가):** LV2~5 업글은 위 재료 + **화폐 부가**(일회성 진행 싱크), 레이드·좀비로 깨진 벽 **수리 = 화폐 주도**(지속 싱크, 큰 기지=수리비↑=영구 화폐 흡수). → 경제 중반 화폐 용도 = [`economy.md`](economy.md) #2-B 「경제 페이즈」.
- ⚠️ 수치 전부 잠정 — #2 밸런싱 때 보정.

### 🧱 방벽 모델 — v2 (재설계 2026-06-10, 영역 내 블록 내구)
> **🔄 밀폐 외피·밴드·flood-fill 폐기.** 방벽이라는 별도 시스템 없음 — **보호 영역(100×100) 안에 플레이어가 쌓은 모든 블록이 코어 레벨에 따른 방어 내구를 가짐**(재료 무관). 블록이 곧 물리 벽 = MC 직관.
- **🧱 블록 내구 = 코어 레벨 부여:** 영역 내 설치 블록은 **부수는 데 시간/도구가 듦**(코어 레벨↑=내구↑). **곡괭이(효율5~8)·관통 폭약·RPG·총격으로 부숨**(내구만큼). 재료 무관(나무·옵시디언 동일 — 코어 레벨이 결정). → **떡칠(벽 두껍게/여러 겹) = 자유**(그만큼 부술 양↑ = 비용으로 자연 통제, 종심 캡 불필요).
- **🔑 코어 자물쇠 + 은닉이 진짜 방어 (유지):** 외부인은 영역에 **걸어 들어올 수 있으나**(트인 곳) **영역 내 컨테이너는 코어 살아있으면 잠김**(코어 자물쇠, 100×100 기준). → **벽(블록 내구)은 코어를 숨기고 지키는 물리 장벽**, 외부인은 **벽을 부숴 코어에 접근·파괴해야** 자물쇠 해제 → 약탈. 은닉(코어 깊이·분산 stash)이 핵심 방어. = 밀폐 판정 없이도 "코어 파괴=약탈 전제" 유지.
- **🚪 문 = OPaC 권한 잠금 (유지):** 코어 방·내부 문은 소유자·연합만 여닫음, **외부인은 못 열고 부숴야**(블록 내구). tailgating(따라 들어가기)은 새 모델에선 무의미 — 어차피 외부인이 영역 진입해도 상자 못 열고 코어는 벽 안 은닉.
- **🔍 식별·피드백:** 블록 내구는 **타격 시 진행 표시**(곡괭이/폭약/총격으로 깎이는 게이지·균열·사운드). 보호 영역 경계 = **은은한 표시**(누구에게나, 내 땅 식별). 코어 위치는 은닉. (밴드별 색은 폐기 — 밴드 없음.)
- **구현:** OPaC(권한·파티) + porongun-core **블록 내구·코어 자물쇠**(영역 100×100 기준). **밀폐 감지 flood-fill 불필요** = 🔵 착수 리스크(launch_plan §1-7) **제거**. 블록 내구 = 좌표별 내구 데이터(또는 영역 일괄 배율).
- **✅ base_raid 재매핑 완료(2026-06-10):** 레이드(블록 부수기)·블록 내구표·폭약·총격·농성·결전 좀비·코어 설치·영역(100×100)·2단계 보호·구현 전부 v2 정합. **🔜 남은 연쇄:** economy #2(공급·비파밍·탄·검산·총/청사진 게이트·폭약 과잉) 재조정 + finale 결전 농성 용어 + launch_plan 밀폐 리스크 제거 + 블록 내구·도구 정밀 수치.

## 연합 = 2단 구조 (정식 / 구두)
진짜 안심은 정식 연합뿐, 그 밖 모든 동맹은 구두라 등 뒤가 위험.
- **정식 연합(기지 공유, 권한 그룹):** **연합 = 코어 1개를 공유(단일 거점을 함께 운영) — "기지를 공유하는 사이" (확정 2026-06-10).** 연합원끼리 **보호 영역 접근·건축 허용, 서로 약탈·코어 파괴·PvP 불가**(시스템상 진짜 안전) / 외부인은 적. **가입 시 자기 코어는 연합 코어로 통합**(개인 코어 포기 = 한 거점에 합류) → 연합당 코어 1개 유지. **코어 파괴 = 연합 전체 거점 파괴·레벨 초기화**(공유라 타격 큼). **배신하려면 탈퇴(권한 즉시 회수) → 그 후 적대** = 배신이 *예고됨*(탈퇴하면 상대도 즉시 앎, 기습 백스탭 불가). 탈퇴자는 코어 0이라 자기 코어 재제작 가능(독립). 구현 = **OPaC 파티 시스템**에 매핑.
- **구두 연합(시스템 밖 합의):** 보호 0. "우리끼리 공격 ㄴㄴ" 해놓고 **언제든 배신 자유** → 다크앤다커식 기습 백스탭·비정한 톤은 *여기* 산다. ("어제는 친구, 오늘은 적.")
- **연합 규모 상한 = 최대 4명 (확정).** 단일 대연합의 서버 지배(눈덩이 끝판왕) 방지, 여러 세력 경쟁 구도 유도 = 눈덩이 완충.
  - 근거: 동접 ~30명 ÷ 4 = **7~8개 세력** 가능 → 다극 경쟁 확보. 캐주얼 지인 파티의 자연 크기.
  - **헬기 정합:** 연합 상한 4명(확정) ↔ 헬기 좌석 **6~8(finale 확정 2026-06-10, 헬기 1~2대)** = 연합 1.5~2개분 정원 → "티켓 모아 연합 다같이 탈출 + 막판 좌석 경쟁" 양립. → [`finale.md`](finale.md) 탈출 경로.
- **정식 연합 = 단일 공유 거점**(코어 1개). 여러 세력(비연합·구두 연합)이 가까이 모이면 **"마을"**(각자 코어, 영역 배타라 인접), 흩어지면 **분산** — 둘 다 자연 발생. (정식 연합 자체는 한 거점이지 클러스터 아님.)
- 의미: 정식 연합 = 약자의 생존 수단이자 강자의 패권 수단 → 규모 상한(4)이 균형추. 구두 연합 = 자유롭되 불안정 = 정치·배신 드라마의 무대.

## 2단계 보호 + 은신처 봉쇄
- **2단계 보호 (v2):** ①**벽**(영역 내 블록 = 코어 레벨 내구, 곡괭이 효율6/7/8·관통 폭약·RPG·총격으로 부숨) → 뚫고 진입 → ②**코어 수색**(은닉이 방어, 못 찾으면 빈손) → ③**코어 파괴**(블록 내구 최고 = 곡괭이 채굴 시간 = 방어자 막을 마지막 기회) → 자물쇠 해제·약탈. 상세 = 위 「방벽 모델 v2」.
- **🔑 코어 = 보호 영역 상자의 자물쇠 (확정 2026-06-09 — 코어 파괴 동기):** **코어가 살아있는 한 보호 영역(100×100) 안 모든 컨테이너(상자·통)는 외부인에게 잠김.** 벽 뚫고(또는 트인 곳으로) 들어가도 상자가 안 열림 → **코어를 찾아 부숴야 보호 해제 → 그제서야 약탈 가능.**
  - **문제 해결:** 이게 없으면 공격자는 "벽 뚫고 상자만 털고 퇴장"이 최적이라 **코어 부술 이유가 없음**(코어 시스템 무의미). 코어=자물쇠로 **코어 파괴 = 약탈의 전제 = 레이드의 목적**이 됨.
  - **은닉이 진짜 방어:** 코어를 깊이 숨기면 공격자가 못 찾아 **벽·폭약·시간만 날리고 빈손 퇴장** → "은닉이 곧 방어" 강화. 일상 약탈은 밖(POI·데스박스), 기지 약탈은 작정한 레이드(코어 파괴)로 분리.

- **은신처 봉쇄 ("완벽한 은신처는 없다"):** 보호는 하한 Y30(아래는 무방비). 지하에 지어도 되지만 보호 안 됨 → **들키면 털릴 각오로 지어라.** *자유 + 리스크*가 비정한 톤에 맞음. (Y25 설치금지 폐지.)

  1. **POI 내 설치 불가** — POI는 파밍 공유 공간 → 점유 독점·리필 충돌 방지(점유 충돌 처리보다 단순). POI 옆은 가능(입지 트레이드오프 유지).

  - 그 외(지하·지표·POI 옆 등)는 자유.

- **공격 = porongun-core 커스텀 (블록 내구 부수기)** — 영역 내 블록은 **코어 레벨 내구**(바닐라 blast resistance·hardness 대신 우리 내구). 곡괭이(효율6/7/8)·관통 폭약·RPG·총격이 블록 내구를 깎음 → porongun-core가 블록별 내구·부수기 판정. **폭약 바닐라 폭발은 파티클·사운드 연출용만**(실제 파괴는 내구 판정).
- **수비(블록 내구·자물쇠) = porongun-core 커스텀** — 영역 내 블록에 코어 레벨 내구 부여 + 코어 자물쇠(컨테이너 잠금)는 직접 만들어야 함. **영리한 경로: OPaC(클레임·파티·API, 권한·문) + porongun-core 블록 내구·코어 자물쇠.** (밀폐 외피 감지 불필요 = v2.)

## ⭐ "그냥 안 함" 악순환 방지 (이 장르 근본 딜레마)
레이드가 너무 쉬우면 기지 건설 무의미("어차피 털림") → 시스템 붕괴. **2중 안전판으로 방지:**
- **폭약 고비용**(제작·희귀자원·POI 부품 → 레이드 **빈도** 억제, "작정한 큰 사건"화) = 눈덩이 완충 "레이드 비용".
- **보안 슬롯 + 손실 상한 + 오프라인 보호**(레이드 **피해** 제한 → 털려도 다는 안 잃음, 재기 가능).
- 둘 다 눌러야 함: 빈도만 잡고 한 번에 다 잃게 하면 여전히 의욕 붕괴.
- **업그레이드가 방어에도 기여:** 코어 레벨↑=블록 내구↑ + 두껍게 떡칠 → 폭약·곡괭이 시간 더 필요 → *키운 만큼 단단* → "지을 가치" 발생.

## ⭐ "왜 레이드를 하는가" (레이드 동기 — 위 딜레마의 짝, 설계 2026-06-09)
**반대 함정:** 파밍이 무한 가능하면 "위험한 레이드 대신 그냥 더 쉬운 POI 돌면 됨" → **레이드 동기 死 = 시스템 장식.** 풀어야 함.
- **🔑 레이드의 본질 = "남이 들인 시간을 강탈하는 지름길":** 파밍은 무한이어도 **즉시는 아님**(시간+위험+소모: 탄약·의료·말). 부자 기지 = 큰 비축(특히 **비파밍 재료·완성 고급 장비** = POI 수십 번 분량) → 터는 게 "내가 직접 파밍할 시간을 한 방에 절약". → 부자일수록 표적, **가난한 기지는 안 털림**(POI가 나음 = 자연스러움).
- **레이드 동기 3축:** ①**비축 강탈**(시간 지름길) ②**코어 파괴=견제**(레벨 0=강자 영구 후퇴=약자→강자 눈덩이 역전) ③복수·정치(비정한 톤). → **강자일수록 표적 = 레이드가 눈덩이 완충**("레이드 비용" 빈도 억제와 짝).
- **성립 3조건:** ①**보안칸 한정**(2~6) → 강자도 다 못 지킴, 초과 비축=일반 상자(코어 자물쇠)=취약(✅) ②**비파밍 재료 희소**(POI만, 느림)(✅) ③**🔑 파밍이 "비용"이어야**(즉시 무한 ❌) — **Y25 초기화·POI 리필 주기가 노브.**
- **⚠️ Y25·POI 리필 = 핵심 균형 노브(#2):** 너무 빠름=파밍 즉시 무한=레이드 동기 死 / 너무 느림=자원 고갈·진행 막힘 / **적정=파밍은 꾸준하되 시간 들고, 큰 비축은 강탈할 가치 유지.** → [`world.md`](world.md) POI 리필·Y25.
- **레이드(누적 비축) ≠ POI PvP(즉발 약탈):** POI에서 만난 적 급습=방금 먹은 템 강탈(즉발, 늘 가치). 레이드=쌓인 비축(누적, 부자 표적). 둘 다 약탈이되 규모·동기 다름.
