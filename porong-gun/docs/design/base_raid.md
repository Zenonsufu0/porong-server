# 설계 — 기지 · 레이드 · 연합

> 비전 SoT: [`../concept.md`](../concept.md) · 관련: [`finale.md`](finale.md)(농성·방벽), [`survival.md`](survival.md)(보안함), [`../platform.md`](../platform.md)(OPaC·관통 폭약).

바닐라 건축 + 철문/레드스톤만으론 보호가 약함(또는 무적). **시스템적 2단계 보호 + 비용 있는 레이드**로 설계.

## 기지 코어 (설치 아이템)
설치 지점 기준 **보호 영역**을 선포(방벽은 자동 생성이 아니라 *플레이어가 직접 쌓은 닫힌 외피*가 방벽이 됨 → 아래 「방벽 모델」). **보호 범위 = Y30(하한) ~ 빌드 상한(319)까지** — **위로는 끝까지 보호, Y30 아래(깊은 지하)는 무방비**(그 밑으로 터널·굴착 가능). 좌표 기준: 해수면 ~63, 지표 ~63–70이라 Y30은 명확한 지하.
- 의미: "지하 깊이 숨어도 소용없다" + 밑으로 지나가는 사람 배려. (※ 엔더 차원 봉쇄로 엘리트라 입수 차단 → 공중 무한도주/스카이베이스 악용 방지, 그래서 위로 끝까지 보호해도 안전.)
- **코어 파괴 = 기지 완전 파괴 + 기지 레벨 초기화** → 코어가 최우선 타겟, **은닉이 곧 방어.** (러스트 Tool Cupboard 레이드를 더 가혹하게.)
- **코어 재질 = 옵시디언급(커스텀 텍스처):** 즉시 파괴 불가, 깨는 데 시간 소요 → 방어자가 달려와 막을 기회.
- **위치 전략:** 영역 깊숙이 숨기기 vs 일부러 높은 곳(탑 꼭대기 등) = 심리전. "완벽한 은신처는 없다."
- 가혹함의 받침: 완파·레벨 초기화는 가혹하지만 옵시디언+은닉+방벽으로 *드물게* 일어나야 톤이 삶. **배낭 레벨·보안함은 코어 파괴와 무관하게 유지**되므로 완파당해도 "맨몸 재시작"은 아님(재기 가능).

### 코어 제작 · 레벨 (설계 2026-06-09, porongun-core 커스텀)
- **블록 자체가 "옵시디언급" 단단함**(파괴 시간 소요 = 방어자 막을 기회) = *블록 속성*이지 재료 아님 → LV1 제작은 접근 쉽게(철 기반, 옵시디언 불요).
- **LV1 코어 제작(잠정):** 철 블록 ×2 + 레드스톤 블록 ×1 + 철 ×4 → 기지 코어 LV1. 온보딩 기본 보급에 1개 지급, **재제작 쉬움**(분실·이전 대비).
- **업그레이드 = 코어 GUI에서 재료 소모(기지 레벨↑). 최종(LV5)은 POI 드랍(군용 합금·전자 부품) 필수 = 최종 기지=POI 게이트.**

- **📐 영역 = 반경(radius) 기준** (코어 = 중심 블록): 반경 R → **(2R+1)² 홀수 영역**(중심 블록 + 양쪽 R칸, 대칭). 짝수(16×16)는 코어 중심 시 비대칭이라 **홀수로 정의.** 레벨은 **건축 가능 보호 영역 크기 + 동심 밴드(방벽 인정 겹) 개수**를 늘림.

| 레벨 | 보호 영역(반경) | 방벽 겹(밴드) | 단계 성격 | 보안칸 | 상점 | 업그레이드 재료(잠정) |
|---|---|---|---|---|---|---|
| **LV1 전초기지** | 17×17 (R8) | 밴드1 = **1겹** | 시작(제작) | 2 | 기본 | — |
| **LV2** | 17×17 | 1겹 강화(돌파 내구↑) | **강화·상점** | 3 | 중급 | 철블록8 + 텅스텐 합금16 |
| **LV3 거점** | 25×25 (R12) | +밴드2 = **2겹** | **확장**(밴드 추가) | 4 | 중급 | 텅스텐32 + 다이아8 |
| **LV4** | 25×25 | 2겹 강화 | **강화·상점** | 5 | 고급(고급 총·청사진) | 텅스텐48 + 다이아12 |
| **LV5 요새** | 33×33 (R16) | +밴드3 = **3겹** | **확장**(밴드 추가) | 6 | 최고급 | 텅스텐64 + 🪖군용 합금16 + 🔌전자 부품8 |

- **교대 패턴:** LV2·4 = *강화·상점*(영역·밴드 고정, **기존 겹 돌파 내구↑** = "키운 만큼 단단"), LV3·5 = *영역 확장(반경 +4) + 동심 밴드 1개 추가*. → 확장 시 리빌드 불필요(기존 외피·건물 유지), 확장 링(폭 4칸)은 새 밴드를 두를 보너스 공간.
- **보안칸 2/3/4/5/6**: 티켓이 1칸 점유(A2)라 실여유 1~5칸 = "칸 경쟁" 긴장 유지.
- **크기 17→25→33(반경 8→12→16)**: 큰 기지 = 밴드 더 많음(겹↑) = 레이드 비용↑. 단 결전 좀비 공격면↑·수리비↑ = 트레이드오프.
- **💰 업글·수리 = 재료(게이트) + 화폐(부가):** LV2~5 업글은 위 재료 + **화폐 부가**(일회성 진행 싱크), 레이드·좀비로 깨진 벽 **수리 = 화폐 주도**(지속 싱크, 큰 기지=수리비↑=영구 화폐 흡수). → 경제 중반 화폐 용도 = [`economy.md`](economy.md) #2-B 「경제 페이즈」.
- ⚠️ 수치 전부 잠정 — #2 밸런싱 때 보정.

### 🧱 방벽 모델 (확정 2026-06-09 — 밀폐 외피 + 밴드당 최외곽 1겹)
방벽은 자동 생성 물리 링이 아니라 **플레이어가 직접 쌓은 닫힌 외피의 최외곽 1겹**이다. 모양 자유(네모 강제 X, 반지름 들쭉날쭉 OK).
- **밀폐 감지(flood-fill):** 보호 영역 바깥 공기에서 코어로 *블록을 안 부수고* 도달하는 경로가 있는지 검사. **문을 제외하고 경로가 전부 막히면 = 닫힌 외피 = 그 최외곽 면이 방벽.** 두께 여러 칸이면 외부에서 처음 막는 면만 방벽, 그 뒤는 내부 바닐라.
- **🔑 밴드당 최외곽 1겹 (무한겹 방지):** 레벨이 추가하는 **동심 밴드(거리 구간)**마다 방벽은 **최외곽 닫힌 외피 1겹만** 인정. *같은 밴드에 두 겹을 쌓아도 최외곽만 방벽*, 안쪽은 일반 바닐라. → 종심 = 밴드 수 = 레벨(LV1/3/5 = 1/2/3겹)로 **고정**, "떡칠 무한겹" 꼼수 무효. 종심을 늘리려면 레벨업(새 밴드)뿐.
- **문·다락문·펜스게이트 = 외피 정당 구성원:** 닫힌 상태 = 밀폐 판정상 "막는 블록"으로 카운트(외피 닫힘 인정). 외피 구성 문 = **방벽 속성(관통 폭약으로만 파괴) + OPaC 권한 잠금**(소유자·연합만 여닫음, 외부인은 못 열고 부수려면 폭약). → 정상 출입구이자 벽과 동급 방어 지점. "문으로만 둘러 우회" 꼼수 차단.
- **🔍 식별 (상시 미묘 표시 + 즉시 피드백):** 방벽은 평범한 블록에 속성만 얹은 것이라 식별 장치 필수.
  - **상시 표시:** 방벽 외피는 **은은한 룬/발광 테두리**(누구에게나 상시). **겹(밴드)별 색 구분** → 공격자가 "몇 겹"을 색으로 파악(레이드 계획). **문은 별색(권한 잠금 출입구=폭약 약점)**. 거리 LOD(가까이서만 선명). ※ 보이는 건 외피지 **코어 위치는 여전히 은닉** → 계획성 ↔ 은닉 양립. 방어자에겐 "외피 닫힘 셀프 점검"으로도 유용.
  - **즉시 피드백:** **곡괭이·TNT·일반 폭발**로 방벽 타격 시 **균열(파괴 진행) 없음 + 금속 튕김음 + 불꽃 파티클 + 액션바 "방벽 — 관통 폭약/총격 필요"**. (관통 폭약·총격은 돌파 게이지 감소로 표시 — ↓관통 폭약 「보조 돌파」.)
- **구현:** OPaC(클레임·파티 = 권한/연합 사회 레이어) + **porongun-core 밀폐 감지·방벽 판정**. 방벽 마킹은 **데이터(좌표 셋)** + **클라이언트 오버레이 렌더**(블록 치환 X, 바닐라 블록 그대로). 재계산 = 영역 내 블록 변경 시 + 폭약 폭발 시(R16=33×33이라 flood-fill 가볍게 한정).
- **미정(#2):** 밴드 폭(±N칸), 밴드에 외피 안 두르면 그 겹 공백("제대로 둘러야 산다"), 한 외피가 두 밴드 걸칠 때 귀속, 1×1 구멍(투사체 통과) 밀폐 처리, 식별 색·강도.

## 연합 = 2단 구조 (정식 / 구두)
진짜 안심은 정식 연합뿐, 그 밖 모든 동맹은 구두라 등 뒤가 위험.
- **정식 연합(기지 공유, 권한 그룹):** 개인당 코어 1개 유지, 연합원끼리 **보호 영역 접근·건축 허용, 서로 약탈·코어 파괴·PvP 불가**(시스템상 진짜 안전) / 외부인은 적. **배신하려면 탈퇴(권한 즉시 회수) → 그 후 적대** = 배신이 *예고됨*(탈퇴하면 상대도 즉시 앎, 기습 백스탭 불가). 구현 = **OPaC 파티 시스템**에 매핑.
- **구두 연합(시스템 밖 합의):** 보호 0. "우리끼리 공격 ㄴㄴ" 해놓고 **언제든 배신 자유** → 다크앤다커식 기습 백스탭·비정한 톤은 *여기* 산다. ("어제는 친구, 오늘은 적.")
- **연합 규모 상한 = 최대 4명 (확정).** 단일 대연합의 서버 지배(눈덩이 끝판왕) 방지, 여러 세력 경쟁 구도 유도 = 눈덩이 완충.
  - 근거: 동접 ~30명 ÷ 4 = **7~8개 세력** 가능 → 다극 경쟁 확보. 캐주얼 지인 파티의 자연 크기.
  - **헬기 정합:** 연합 4명 ↔ 구조 헬기 4인승이면 "티켓 4장 모아 연합 다같이 탈출" 서사와 맞물림(좌석 수는 밸런싱 때 확정, 4를 기준점). → [`finale.md`](finale.md) 탈출 경로.
- 코어를 모으면 **연합 마을**(클러스터), 흩어지면 **분산 동맹** — 둘 다 자연 발생.
- 의미: 정식 연합 = 약자의 생존 수단이자 강자의 패권 수단 → 규모 상한(4)이 균형추. 구두 연합 = 자유롭되 불안정 = 정치·배신 드라마의 무대.

## 2단계 보호 + 은신처 봉쇄
- **2단계 보호:** ①**방벽**(닫힌 외피 최외곽 + 권한 잠금 문 = 관통 폭약 주력 / 총격 극저효율 보조로만 돌파, 곡괭이·TNT 무효) → 뚫리면 ②**내부 바닐라 블록**(외피 두께·안쪽 건물 = 일반 파괴) → ③코어 수색·파괴(옵시디언급 곡괭이) → 완파/약탈. 자세한 방벽 정의 = 위 「방벽 모델」.
- **🔑 코어 = 보호 영역 상자의 자물쇠 (확정 2026-06-09 — 코어 파괴 동기):** **코어가 살아있는 한 보호 영역 안 모든 컨테이너(상자·통)는 외부인에게 잠김.** 방벽 뚫고 들어가도 상자가 안 열림 → **코어를 찾아 부숴야 보호 해제 → 그제서야 약탈 가능.**
  - **문제 해결:** 이게 없으면 공격자는 "방벽 뚫고 상자만 털고 퇴장"이 최적이라 **코어 부술 이유가 없음**(코어 시스템 무의미). 코어=자물쇠로 **코어 파괴 = 약탈의 전제 = 레이드의 목적**이 됨.
  - **은닉이 진짜 방어:** 코어를 깊이 숨기면 공격자가 못 찾아 **방벽·폭약·시간만 날리고 빈손 퇴장** → "은닉이 곧 방어" 강화. 일상 약탈은 밖(POI·데스박스), 기지 약탈은 작정한 레이드(코어 파괴)로 분리.
  - 구현 = OPaC 권한(영역 내 컨테이너 외부인 잠금) + 코어 생존 여부 hook. **보안 칸(인벤)은 무관하게 항상 안전**(약탈 불가). 러스트 Tool Cupboard보다 강하게 "코어 하나가 기지 전체 stash의 열쇠".
- **은신처 봉쇄 ("완벽한 은신처는 없다"):** **설치는 어디든 자유, 보호만 하한 Y30**(아래는 무방비). 지하에 지어도 되지만 보호 안 됨 → **들키면 털릴 각오로 지어라.** 설치 금지 같은 하드 제약 없음 — *자유 + 리스크*가 비정한 톤에 맞음. (Y25 설치금지 폐지.)
  - **Y25 이하 자동 초기화 (유지):** 목적 = 기본 자원 순환 + 중후반 재기 수단(지하 봉쇄 명분은 보호 미적용으로 이미 대체).
    - 구현: 같은 시드로 원본 지형 생성해 **Y25 이하 블록만 덮어쓰기**. 전용 모드 없어 커스텀(resetchunks류 로직).
    - 부하 분산: **"손 탄 청크만" 재생성**(플레이어가 Y25 이하 건드린 청크만 추적) → 월드 2000²(약 15,600청크)라도 감당.
    - 안전장치: 초기화 전 **카운트다운 → Y25 이하 플레이어 지상 강제 텔레포트 → 아이템·건축 소실 경고**. 타이밍 = 현실 새벽 등 고정 시각.
  - ⚠️ 매일 부활하는 자원이 **폭약 재료**면 레이드 비용이 무너짐 → 폭약 희귀 재료는 POI 등 **통제된 경로로만**.

## 💥 관통 폭약 (Breaching Charge — 설계 2026-06-09, porongun-core 커스텀)
방벽을 뚫는 **주력 수단**. 평소 방벽은 **곡괭이·TNT·일반 폭발 전부 무효(데미지 0)** → 관통 폭약(주력) + 총격(극저효율 보조, ↓「보조 돌파」)만 돌파 게이지를 깎음. 폭약 비용 = **비파밍 게이트 재료**라 레이드 **빈도**를 통제 = "작정한 큰 사건"화 = 눈덩이 완충 "레이드 비용".

### 설계 원칙
- **시끄럽고 느림**(도화선·광역 알림) → 기습 레이드 불가, 방어자 대응 기회 보장(러스트식, 다크앤다커 기습과 구분).
- **코어 자체는 폭약 무효** — 옵시디언급 **곡괭이 채굴(시간 소요)**. 의도 = 방벽 다 뚫려도 최종 단계에서 방어자가 달려와 막을 진짜 시간 확보. (폭약은 *겹을 여는* 도구, 코어 파괴는 아님.)

### 동작 (설치형 장약)
1. **방벽(외피 최외곽 면 또는 권한 잠금 문)에 부착 설치**(아이템 우클릭). — 폭발 시 부착 블록이 *그 밴드의 최외곽 외피인지* 판정(아니면=내부 바닐라라 폭약 무효/약효).
2. **도화선 N초**(잠정 10초) — 그동안 광역 알림(폭음·연기 파티클 + 기지 소유자 메뉴 경보). 방어자 출동 창.
3. 폭발 → 부착 지점 방벽 겹에 **돌파 데미지**. 누적 돌파 내구를 넘기면 그 자리에 **구멍(breach, 잠정 3×3)** 발생.
4. 구멍으로 진입 → 다음 밴드 겹 / 내부 바닐라 블록(일반 파괴) → 코어 수색.

### 수치 모델 — 방벽 돌파 내구 ↔ 폭약 위력
**각 방벽 겹(밴드) = 돌파 내구(breach HP). 강화 시 ↑. 관통 폭약 = 1발 돌파 위력. 한 겹 돌파 필요 발수 = ⌈돌파 내구 ÷ 위력⌉.**
→ **강화 레벨(LV2·4)이 모든 기존 겹의 돌파 내구를 올림** = "키운 만큼 단단"(지을 가치 발생). 확장 레벨(LV3·5)은 새 밴드 겹 추가 = 종심방어(겹마다 따로 뚫어야).

**방벽 겹별 돌파 내구(잠정):**
| 겹 | 생성 | 강화 후 |
|---|---|---|
| **밴드1**(R8 내) | LV1 100 | LV2 200 → LV4 300 |
| **밴드2**(R8~R12) | LV3 200 | LV4 300 |
| **밴드3**(R12~R16) | LV5 400 | — |

### 폭약 3티어 (재료 트리 정합 — [`economy.md`](economy.md) 3층 트리)
| 등급 | 1발 돌파 위력 | 핵심 재료(게이트) | 게이트 성격 |
|---|---|---|---|
| **T1 파쇄 장약** | 50 | ⚙️텅스텐 합금 + 화약 + 철 | 작업대 제작(텅스텐=파밍가능 페이싱) — 저티어 기지용 |
| **T2 성형작약** | 120 | ☢️열화우라늄 + 🔌전자 부품 | 전자 부품=POI drop 게이트 — 강화 방벽용 |
| **T3 관통 작약** | 300 | 🪖군용 합금 + 🔌전자 부품 + ☢️열화우라늄 | POI 이중 게이트 — 요새 외벽용 "작정한 연합 레이드" |

**레이드 비용 예(잠정, 한 지점 집중 돌파 기준):**
- **LV1 전초기지**(R8 100): T1 ×2. → 초반 약탈 활발(저비용).
- **LV3 거점**(R8강화 200 + R12 200 = 400): T1 ×8 또는 T2 ×4(전자 부품 필요) → POI 파밍 강제.
- **LV5 요새**(R8 300 + R12 300 + R16 400 = 1000): T3 ×4(군용 합금) → "작정한 대형 사건". 종심 3겹이라 한 지점만 뚫어도 깊이 들어가야 함.

→ 큰 기지일수록 비파밍 재료 소모 폭증 = 레이드 빈도 자연 억제. 단 큰 기지 = 둘레↑·결전 수비면↑ = 트레이드오프(공격 비용 ↔ 수비 부담 균형).

### 🔫 보조 돌파 — 총격 (극저효율, 설계 2026-06-09)
방벽 본체·문에 **총격으로도 돌파 게이지를 깎을 수 있다**(폭약은 빠른 주력, 총은 느린 보조). 총겜 정합 + 탄약 화폐 싱크 + 진입로 다양화. **1발 ≈ 돌파 위력 0.3(잠정)** → LV1 방벽 100 = **~300발**. 폭약 대비 ~100배 비효율.
- **역할 분리:** 폭약 = 빠름(도화선)·기습·**온라인 기지도 타격**. 총 = 느림·**소음·시간** = 쏘는 동안 방어자 출동 → **오프라인/빈 기지 약탈 니치.** 문은 본체보다 총에 잘 뚫림(약한 부재) → "총으로 문 따고 진입" = 도어브리칭.
- **⚠️ 게이트 약화 방지 3중 완화** (탄약=구리+화약=파밍 가능이라, 안 막으면 레이드 빈도 통제 붕괴):
  1. **종심·강화가 총을 자연 차단** — LV5 요새(3겹×강화 = 총 수천 발·수십 분) = 비현실 → **고레벨 기지는 폭약 강제.** 총 실용 구간 = LV1~2 저레벨(원래 가볍게 털림 = 약화 허용 범위).
  2. **소음·시간이 온라인 차단** — 수백 발 사격 = 방어자 출동 → 온라인 방어 기지엔 비현실.
  3. **오프라인 보호 시 총 효율 추가 감소(잠정)** — 한 세션에 못 뚫게 → 여러 세션 강제 = 방어자 로그인해 발견·복구 기회.
- **곁가지(미정 #2):** 열화우라늄 **AP탄 = 방벽 보너스 데미지** 옵션(열화우라늄 탄 용도 확장·"전용 레이드 탄") — 단 열화우라늄=파밍 페이싱이라 게이트 약화 주의. 총 종류별 방벽 데미지 차등(대물저격↑?), 방벽 자가 수리 여부.
- **식별 정합:** 총격 시에도 돌파 게이지 감소 표시(즉시 피드백) — 단 곡괭이·TNT는 여전히 균열 없음 + 금속 튕김(무효).

### 미정 (#2 밸런싱)
- 도화선 시간(10초?), 구멍 크기(3×3?), 광역 알림 반경, 정확 돌파 내구·위력·재료 수량.
- **오프라인 보호 상호작용:** 오프라인 시 돌파 불가 or 내구↑ or 무관(러스트는 무관) → 별도 결정.
- **결전 좀비-방벽:** Undead Nights `demolition_zombie`(TNT)는 방벽 무효라 안 통함 → 결전엔 방벽 일시 무력화 or 전용 관통 능력 부여 필요(좀비-방벽 항목, [`finale.md`](finale.md)).
- **내부 바닐라 구간** 일반 폭발 허용 여부(방벽 안쪽은 TNT 됨? — 톤상 허용 쪽).

## 구현 분담
- **공격(폭약) = porongun-core 커스텀 관통 폭약** — "방벽만 특수 폭약에 뚫림"은 우리 룰. 폭발은 **바닐라 Explosion API 안 씀**(blast resistance만 봐서 겹 돌파 내구 개념 불가) → 커스텀 BlockEntity(부착·도화선) + porongun-core 돌파 판정, **바닐라 폭발은 파티클·사운드 연출용만** 재생. (제3자 폭약 모드 More Explosives 등은 일반 폭발물 flavor 선택.)
- **수비(조건부 보호) = porongun-core 커스텀** — 기존 클레임 모드는 "완전 보호 on/off"뿐이라 "평소 무적 + 특수 폭약에만 뚫림"은 직접 만들어야 함. **영리한 경로: OPaC(클레임·파티·API) + 밀폐 외피 감지 + 보호를 *관통*하는 커스텀 폭약.**
  - ⚠️ 보호 *기하*(코어 반경·Y30 하한·동심 밴드·밀폐 외피·레벨)는 청크 클레임과 안 맞아 porongun-core 커스텀. OPaC는 **연합/권한 사회 레이어 + 문 권한 잠금**으로 활용, 밀폐 감지·방벽 판정·관통 룰·식별 표시는 우리가 hook. 방벽 마킹 = 데이터 셋 + 클라 오버레이(블록 치환 X). 상세 = 위 「방벽 모델」.

## 기지 = 업그레이드式 (고정 단일 크기 ❌)
아이템으로 영역 선포, 단계별 확장·강화.
- 5단계, 강화/확장 교대: LV1 17×17(R8) → LV2 강화 → LV3 25×25(R12,2겹) → LV4 강화 → LV5 33×33(R16,3겹). 다중방벽=동심 밴드(밴드당 최외곽 외피 1겹). 상세 = 위 「코어 제작·레벨」·「방벽 모델」.
- 효과: **진행 동기**(파밍→강화, 비제로섬 성장) + **방어 트레이드오프**(크면 자산↑·둘레↑ 방어 불리) + 플레이 스타일 분기.
- ⚠️ 크기 = 방어 난이도. 소규모 인원이라 **작게 시작** 권장(50×50은 둘레 200이라 방어 불가).

## ⭐ "그냥 안 함" 악순환 방지 (이 장르 근본 딜레마)
레이드가 너무 쉬우면 기지 건설 무의미("어차피 털림") → 시스템 붕괴. **2중 안전판으로 방지:**
- **폭약 고비용**(제작·희귀자원·POI 부품 → 레이드 **빈도** 억제, "작정한 큰 사건"화) = 눈덩이 완충 "레이드 비용".
- **보안 슬롯 + 손실 상한 + 오프라인 보호**(레이드 **피해** 제한 → 털려도 다는 안 잃음, 재기 가능).
- 둘 다 눌러야 함: 빈도만 잡고 한 번에 다 잃게 하면 여전히 의욕 붕괴.
- **업그레이드가 방어에도 기여:** 강화 방벽=내구↑=폭약 더 필요 → *키운 만큼 단단* → "지을 가치" 발생.
