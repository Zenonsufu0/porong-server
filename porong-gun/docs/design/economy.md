# 설계 — 경제 · 장비 획득 · 제작 · 상점 · 온보딩

> 비전 SoT: [`../concept.md`](../concept.md) · 관련: [`survival.md`](survival.md)(의료·무게), [`base_raid.md`](base_raid.md)(상점=기지 안), [`../platform.md`](../platform.md)(TaCZ·First Aid·Tactical 3D Armor).
> **이 문서가 설계 작업(#1 제작법 + #2 경제 모델)의 주 작업 공간.** 제작 정책·가격표는 모드 실레시피 확인 후 채운다.

## 총 획득 (TaCZ 제작 시스템 기반)
- **작업대 2종(jar 확인 2026-06-09):** `gun_smith_table`(총·부착·텅스텐 등 제작, 통나무3+철4+철블록1) + `ammo_workbench`(탄약 전용, 구리2+철1+탄약상자). 탄약 레시피 type은 둘이 공유(`gun_smith_table_crafting`). 둘 다 바닐라 재료로 제작 = 진입 허들 낮음. (옛 "단일 1종"·"3종" 표기 정정.)
- **기본 총**(티어 0~1): 청사진 없이 제작 → 초반 진입(권총·기본 소총). **PvE 받침·재기.**
- **고급 총**(티어 2+): **청사진(POI 루트 파밍)으로 해금** → 작업대 제작. 청사진=레시피 영구 학습(죽어도 유지=재기 동력), 총 완성품은 소모(죽으면 데스박스). **청사진을 POI 티어 루트에 배치** = "좋은 총=진행 보상". 구현 = **TaCZ Weapon Blueprints**(1.20.1 Forge 확인).
- **시즌 시너지:** 와이프 시 `/gg clearRecipes`로 무기 청사진 리셋 → 매 시즌 무기 해금 재파밍. (탈출 부품·티켓 청사진은 porongun-core 자체 리셋.)
- 탄약 = ammo_workbench 제작(구경별 레시피) + 상점 / 폭약 = 고티어 POI 재료 제작(레이드 비용) / 드랍 = 스캐브·데스박스 약탈.

## 🦺 방어구 = 전술 갑옷 (`[TaCZ] Tactical 3D Armor`)
바닐라 갑옷 아님(테마·총겜 정합). **3티어 = 방탄 등급:** Light(권총·SMG) → Medium(라이플·DMR) → Heavy(중화기). 바닐라 갑옷은 초반 저티어 fallback.
- **스캐브 약탈 루프:** 스캐브 랭크별 갑옷(Rookie→Expert) → 처치해서 **약탈** = 타르코프st 진행. 상점/POI/제작 보조.
- **무게 트레이드오프:** 중갑일수록 무거움 → **속도 페널티↑** → 방어 ↔ 기동. 무게=점유칸·다리 부상과 같은 둔화 축.
- **First Aid 부위별 보호 연동:** 헬멧=머리·베스트=몸통 부위 HP 보호. ⚠️ 착수 검증: Tactical 3D Armor × First Aid 연동.

## 의료템 (First Aid 약품 — 핵심 소모재)
**상점 구매 + 드랍(스캐브·데스박스) + 제작**(붕대=종이+양털st 저티어 / 부목·모르핀·수술키트=상위). 부상 치료 강제(다리=둔화)라 **총·탄약과 함께 출격 전 필수 구비** = 꾸준한 화폐 싱크 → 경제 순환의 엔진.

## 화폐 · 수급
- **화폐 = 커스텀 화폐 = 물리 아이템 (확정 2026-06-09)**(스캐브·POI 드랍 = 타르코프 루블st). **파밍이 곧 돈.** 단일 화폐 아이템(스택), 고액은 묶음 압축 아이템으로 칸 절약(잠정).
  - **약탈 가능:** 인벤에 들면 데스박스로 떨어짐(강탈) → **보안 칸에 넣으면 안전 = 칸 경쟁에 화폐도 참여**("큰돈 들고 다니면 위험"). 상점 거래 = 인벤 화폐 차감. 무게=소량(큰돈일수록 칸 부담, #2).
- **💰 화폐 수급 다변화 — 농사·채굴 환금:** 파밍(POI·스캐브) 외에 **바닐라 작물·광물을 상점에 판매**해 화폐화. 비전투 자급 플레이어도 경제 참여(PvE 받침).
  - **자급 vs 잉여 (안전 ↔ 리스크 트레이드오프):**
    - **자급용 소량 농사**(먹을 만큼) = **은신처(기지 보호 영역) 안에서 안전.** 누구나 연명 가능.
    - **환금용 잉여 생산**(팔 만큼 대량) = **기지 밖에서 리스크 감수** — 기지 영역은 작게 시작이라 대농장 공간 부족, 광물은 밖/지하 채굴 필요 → **노출된 농장·광산 = PvP·스캐브 표적.**
  - 함의: 비정한 톤과 정합 — 돈을 더 벌려면 더 노출된다. 밖에 펼친 농장·광산이 약탈·습격의 자연 표적 → PvP 스파이스 유발.

## 상점 · GUI 접근 (중심 허브 건물 없음, 메뉴 아이템 폐기)
- **접근 제약 = 자기(또는 연합) 기지 안에서만 거래 활성 (확정).** 허브 없이 *기지가 거래 거점*, 긴장 유지.
- 등급 해금 = **기지 레벨**(고레벨일수록 고급 무기·탄약 구매).
- 함의: 신규는 **코어 설치 전엔 거래 불가** → 온보딩 후 첫 목표 = "코어 박고 거점 만들기". 코어 설치 아이템은 온보딩 지급/쉬운 제작으로 매끄럽게.
- **🚫 메뉴 아이템(핫바 9번) 폐기 (확정 2026-06-09):** 핫바 슬롯을 먹어 비효율 → 제거. **GUI 접근 = ①코어 블록 우클릭 ②기지 영역 내 Shift+F 단축키.**
  - **GUI 탭(서비스):** 상점(구매·판매) · 배낭 업글 · 기지 관리(코어·방벽·권한) · 정보(통계·진행). **전부 기지 안에서만**(코어/Shift+F). ⚠️ 정보·기지관리도 기지 안만 — 불편 시 정보 탭만 어디서나로 분리 여지.
  - **보안함은 "물건"이라 별개** → 인벤 특수 칸이라 어디서나(↓survival 「보안 도구함」). 메뉴=서비스(거점)·보안함=물건(휴대) 분리.

## 온보딩 · 신규 완충
- **온보딩:** 격리 로비 → **서버 설명 + 디스코드 인증** → **랜덤 좌표 스폰**(신규가 기존 세력 옆에 안 떨어지게 분산). 디스코드 인증 자체 구현(봇 연동 방식은 착수 시; porong-gun 독립).
- **🛡️ 신규 보호막 (PvP 면역 — 진입 완충):** 최초 진입 후 **실시간 12시간**(잠정) 동안 **타 플레이어의 대상 지정·피격 면역**(약탈 표적도 안 됨).
  - **악용 방지:** **최초 진입 1회만**(매 사망 리스폰엔 적용 X). **선공 시 즉시 해제.**
  - **PvE엔 적용 X:** 스캐브·좀비·굶주림·환경엔 첫날부터 노출 → 환경 가혹함 유지.
  - ⚠️ 미정: 12h 기준(실시간 vs 플레이타임), 결전 직전 진입자 처리.
- **🎒 기본 보급 (진입 완충):** 최초 스폰 시 **코어 설치 아이템 + 기본 총·탄약 + 의료템·식량 소량** 지급 → 거래 불가(코어 전)·어려움 난이도 상태에서 첫 생존 가능.
- **중심 허브 없음 → 개인 기지가 각자 허브.**

### #1-B. 커스텀 아이템 제작법 (porongun-core)
> 모든 게이트 재료는 porongun-core 커스텀(모드에 없음, 2026-06-09 스캔 확인).

#### 🧱 재료 3층 트리 (확정 2026-06-09, 소재/발사약 분리)
**소재(탄두 금속) ↔ 화약(발사약) 분리** — 텅스텐/열화우라늄은 순수 금속 소재, 화약은 *탄약 조립 단계*에서 결합(전 구경 공통). 직관적이고, 소재 게이트와 화약을 독립 조절 가능.
```
바닐라(파밍 가능) ─┬─ 철·구리 ───────→ ⚙️ 텅스텐 합금(작업대)        → 소총탄 탄두·총 프레임
                  └─ 텅스텐+다이아 ──→ ☢️ 열화우라늄(AP 라인)        → 저격·배틀라이플·대물 AP탄·고급 총
        + 화약(발사약) ─────────────→ 탄약 조립 단계에서 소재와 결합
POI·스캐브 드랍(비파밍, 레시피 없음) → 🪖 군용 합금 · 🔌 전자 부품  → 최상위 총·방어·탈출·AP탄 (네더라이트/blaze 치환)
```
- **텅스텐 합금·열화우라늄 = 작업대 직접 제작(청사진 불요).** 페이싱 층("탄 공짜 아님"), 결국 파밍 가능. **화약 미포함**(소재일 뿐).
- **🪖 군용 합금 · 🔌 전자 부품 = drop-only (확정).** POI/스캐브 드랍만, 제작 불가 = 유일한 진짜 게이트. 드랍률=진행 노브(#2).
  - 군용 합금=금속(방어·총 프레임), 전자 부품=정밀(고급 탄·신호기·탈출 데이터). → 금속/정밀 라인 분리로 POI 쏠림 완화.

#### ⚖️ 재료 배분 원칙 — 소모성 vs 영속성 ("밑 빠진 독" 방지, 확정 2026-06-09)
**비파밍 재료를 소모템(탄약)에 묶으면 밑 빠진 독** — 탄약은 쏘면 증발(가치 X), 티켓·기지·장비는 보존(가치 O). 인당 시즌 비파밍 공급 ~18개인데 대물탄 7회 출격(전자 2.5×7)이면 다 증발→기지·부품 못 만듦.
| 용도 | 성격 | 재료 정책 |
|---|---|---|
| **영속**(기지 LV5·고급 총·방어구·**티켓·탈출 부품**) | 가치 보존(죽어도 청사진·보안칸·기지) | **비파밍(군용합금·전자) 집중** |
| **소모**(탄약·폭약·의료) | 증발 | **파밍 위주**(화약·텅스텐·구리), 비파밍은 **고비용·저빈도 사치** |
- **🔑 원칙: 비파밍의 주 용도 = 영속템**(필수 진행). 소모템은 ①일상 화력·기본 탄=파밍(화약·텅스텐=무한 받침) ②고급 탄·폭약=비파밍이되 **"가끔 큰 한 방"**(대물탄·T3 폭약=선택적 사치, 없어도 기본 생존, 아껴 써 증발 느림).
- **효과:** 비파밍 공급(빡셈)이 영속 진행에 온전 투입→결전 도달 최소선 확보 / 대물탄·폭약=아껴 쓰는 결정적 한 방(타르코프st "이 탄 쏠 가치 있나") / 일상 교전=파밍 화력=누구나.
- ⚠️ #2 조정: 고급 탄·폭약의 비파밍 비중을 "저빈도 사치"로 낮춰 영속 진행 압박 완화.

### #2-B. 경제 모델 (P2P 없음 → 단일 PvE 루프 + 약탈 재분배)
**플레이어 직거래 없음** → 경제가 **상점(PvE) ↔ 플레이어** 한 축 + **데스박스 약탈(강제 재분배)** 로 단순. 통제 포인트 딱 둘 = **인플레(화폐 총량)** + **진행 곡선(장비 게이트)**. 단순함이 1인 운영엔 장점.

**🎯 핵심 공식 — 파밍 기대값(EV):** 드랍률·위험·장비·적 강함이 전부 이 한 식의 항. POI 티어 노브를 이걸로 맞춤.
```
파밍 EV(티어) = 기대 보상 − 기대 비용
  기대 보상 = 재료 드랍률×가치 + 화폐 드랍 + 청사진 확률
  기대 비용 = 소모(탄약·의료·식량) + P(사망)×장비 손실
  P(사망) ∝ (적 강함 − 내 장비) + PvP 조우 확률
```
- **설계 목표:** 각 티어가 **"적정 장비를 갖춘 자에겐 EV 양수, 맨몸엔 음수"** → 장비 업글이 `P(사망)↓`로 EV를 양수화 = **장비 진행이 자연 강제**("업글하고 가는 게 도움 되는가?"의 답=YES). 고티어 적은 확실히 강함(스캐브 Rookie→Expert). → POI 티어 매트릭스 = [`world.md`](world.md) 「POI 티어」.

**📒 수급(Source) / 싱크(Sink) 장부:**
| 화폐 수급(Source) | 화폐 싱크(Sink) |
|---|---|
| 스캐브 처치 · POI 화폐 상자 | **🧨 화약**(LV1~2 핵심) · 의료템 · 탄약 |
| 작물 · 광물 판매 | 청사진 보조 · 기지 수리 · 상점 총 |
| **고티어 재료 저가 판매**(잉여만) | (고티어 재료 **구매 불가**=게이트 보존) |

**🔩 고티어 재료(군용 합금·전자 부품) 상점 정책 (확정):** **구매 불가**(돈으로 게이트 우회 차단) + **저가 판매 가능**(판매가 ≪ 사용 가치 → "팔 바엔 쓴다", 잉여만 환금). → 게이트 보존 + 잉여 죽은자산 방지 + 화폐 수급 소액 보조.
- **🧨 화약 = LV1~2 상점 주요 판매:** 초반 핵심 화폐 싱크(돈→화약→탄약). 중반 크리퍼 팜 자급되며 비중↓. (↑#2-A 화약 항목.) ⚠️ 상점가 = 크리퍼 스폰율 연동(스폰 낮으면 가격↓ 완충).

#### 🔄 경제 페이즈 — 화폐의 역할 이동 (초반→중반→후반)
초반 싱크(화약·탄·의료)가 중반 **자급**되며 사라짐 → 화폐 인플레·진행 동기 증발 위험. **화폐를 "진행·유지의 부가 비용"으로 이동**해 해결.
| 페이즈 | 경제 성격 | 화폐 주 용도 | 무게중심 |
|---|---|---|---|
| **초반** | 소모재 생존 경제 | 화약·기본탄·붕대 | **화폐**(윤활유) |
| **중반** | 진행·경쟁 경제 | **배낭·기지 업글·기지 수리·상위 의료** | **POI 비파밍 재료**(화폐는 조연) |
| **후반** | 결전 경제(모든 축의 최종 싱크) | 탈출 부품·농성 물자·기지 수리·약탈 손실 보충 | 결전 준비 |

**🚁 후반 = 결전이 모든 경제 축을 빨아들임:** 화폐(이월X)→농성 물자·수리로 막판 소진 / 비파밍 재료→탈출 부품 제작(수요 정점) / 화약·탄약→농성 대량 소비. **🔑 엔진 = "화폐는 이월 안 되니 다 써라"** — 비축이 와이프로 증발하기 전 소비 러시 = **인플레 자연 정산** + "쓸 거면 지금" 긴장. **분기:** 탈출파=비파밍 재료→부품(POI 집중) / 농성파=화폐·화약→물자·기지(거점 강화) / 약탈파=결전 약탈로 부품 강탈(재분배 정점). 탈출 부품 레시피 = ↑#1-B, 결전 상세 = [`finale.md`](finale.md).

**💰 화폐 싱크 = 일회성(진행) + 지속(유지) 조합 (둘 다 필요):**
| 유형 | 항목 | 특성 |
|---|---|---|
| **일회성**(진행) | 배낭 업글 · 기지 업글 · 보안함 확장 | 한 번 사면 끝 → 중반 화폐 흡수, 다 키우면 멈춤 |
| **지속**(유지) | **기지 수리** · 화약 · 의료 · 탄약 | 계속 빠짐 → 후반까지 인플레 억제. **큰 기지=수리비↑=영구 싱크** |

**🛒 화폐 용도 매핑 (재료 게이트 + 화폐 부가):**
| 시스템 | 재료(게이트) | 화폐 | 비파밍 재료 |
|---|---|---|---|
| **배낭 업글** 18→27→36 | 가죽 등 소량 | **주도**(모두의 영구 목표) | — |
| **기지 수리** | 깨진 블록 재질 소량 | **주도**(지속 싱크) | — |
| **기지 업글** LV2~5 | 텅스텐·다이아(게이트) | 부가(추가 비용) | LV5만 군용합금·전자 |
| **보안함 확장** | 철·다이아 | 부가 | — |
- **🔑 화폐의 천장 (인플레·톤 동시 해결):** 화폐로 사는 건 **기지·배낭 같은 "그릇"**, 채울 **장비·실력은 POI 파밍**(비파밍 재료·고급 총·고급 탄 = **돈으로 못 삼**, 구매 불가 확정). → 돈만 모은 자 ≠ 강한 자 = 인플레 천장 + 비정한 톤.
- ⚠️ 화폐 단위·잠정 가격표·드랍 절대 수치 = 플레이테스트 보정. 인플레 통제 = 화폐 총량(드랍률) vs 싱크(수리·업글·화약·의료) 균형.
