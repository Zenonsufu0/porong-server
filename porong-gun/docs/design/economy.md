## 총 획득 (TaCZ 제작 시스템 기반)
- **작업대 2종(jar 확인 2026-06-09):** `gun_smith_table`(총·부착·텅스텐 등 제작, 통나무3+철4+철블록1) + `ammo_workbench`(탄약 전용, 구리2+철1+탄약상자). 탄약 레시피 type은 둘이 공유(`gun_smith_table_crafting`). 둘 다 바닐라 재료로 제작 = 진입 허들 낮음. (옛 "단일 1종"·"3종" 표기 정정.)
- **기본 총**(티어 0~1): 청사진 없이 제작 → 초반 진입(권총·기본 소총). **PvE 받침·재기.**
- **고급 총**(티어 2+): **청사진(POI 루트 파밍)으로 해금** → 작업대 제작. 청사진=레시피 영구 학습(죽어도 유지=재기 동력), 총 완성품은 소모(죽으면 데스박스). **청사진을 POI 티어 루트에 배치** = "좋은 총=진행 보상". 구현 = **TaCZ Weapon Blueprints**(1.20.1 Forge 확인).
- **시즌 시너지:** 와이프 시 `/gg clearRecipes`로 무기 청사진 리셋 → 매 시즌 무기 해금 재파밍. (탈출 부품·티켓 청사진은 porongun-core 자체 리셋.)
- 탄약 = ammo_workbench 제작(구경별 레시피) + 상점 / 폭약 = 고티어 POI 재료 제작(레이드 비용) / 드랍 = 스캐브·데스박스 약탈.

## 🦺 방어구 = 전술 갑옷 (`[TaCZ] Tactical 3D Armor`)
바닐라 갑옷 아님(테마·총겜 정합). **3티어 = 방탄 등급:** Light(권총·SMG) → Medium(라이플·DMR) → Heavy(중화기). 바닐라 갑옷은 초반 저티어 fallback.
- **스캐브 약탈 루프:** 스캐브 랭크별 갑옷(Rookie→Expert) → 처치해서 **약탈** = 타르코프st 진행. 상점/POI/제작 보조.
- **무게 트레이드오프:** 중갑일수록 무거움 → **속도 페널티↑** → 방어 ↔ 기동. 무게=점유칸·다리 부상과 같은 둔화 축.
- **First Aid 부위별 보호 연동:** 헬멧=머리·베스트=몸통 부위 HP 보호. ⚠️ 착수 검증: Tactical 3D Armor × First Aid 연동.

## 의료템 (First Aid 약품 — 핵심 소모재)
**상점 구매 + 드랍(스캐브·데스박스) + 제작**(붕대=종이+양털st 저티어 / 부목·모르핀·수술키트=상위). 부상 치료 강제(다리=둔화)라 **총·탄약과 함께 출격 전 필수 구비** = 꾸준한 화폐 싱크 → 경제 순환의 엔진.

## 화폐 · 수급
- **화폐 = 커스텀 화폐 = 물리 아이템 (확정 2026-06-09)**(스캐브·POI 드랍 = 타르코프 루블st). **파밍이 곧 돈.** **단일 화폐 아이템 = 🪙 코인 1종(커스텀 스택 한도 ~6,000)**, 거액은 자연히 여러 칸 점유=칸 부담 노브(다액면 폐기, ↓#2 화폐 단위·가격).
  - **약탈 가능:** 인벤에 들면 데스박스로 떨어짐(강탈) → **보안 칸에 넣으면 안전 = 칸 경쟁에 화폐도 참여**("큰돈 들고 다니면 위험"). 상점 거래 = 인벤 화폐 차감. 무게=소량(큰돈일수록 칸 부담, #2).
- **💰 화폐 수급 다변화 — 농사·채굴 환금:** 파밍(POI·스캐브) 외에 **바닐라 작물·광물을 상점에 판매**해 화폐화. 비전투 자급 플레이어도 경제 참여(PvE 받침).
  - **자급 vs 잉여 (안전 ↔ 리스크 트레이드오프):**
    - **자급용 소량 농사**(먹을 만큼) = **은신처(기지 보호 영역) 안에서 안전.** 누구나 연명 가능.
    - **환금용 잉여 생산**(팔 만큼 대량) = **기지 밖에서 리스크 감수** — 기지 영역은 작게 시작이라 대농장 공간 부족, 광물은 밖/지하 채굴 필요 → **노출된 농장·광산 = PvP·스캐브 표적.**
  - 함의: 비정한 톤과 정합 — 돈을 더 벌려면 더 노출된다. 밖에 펼친 농장·광산이 약탈·습격의 자연 표적 → PvP 스파이스 유발.

## 상점 · GUI 접근 (중심 허브 건물 없음, 메뉴 아이템 폐기)
- **접근 제약 = 자기(또는 연합) 기지 안에서만 거래 활성 (확정).** 허브 없이 *기지가 거래 거점*, 긴장 유지.
- 등급 해금 = **기지 레벨**(고레벨일수록 고급 무기·탄약 구매).
- 함의: 신규는 **코어 설치 전엔 거래 불가** → 온보딩 후 첫 목표 = "코어 박고 거점 만들기". 코어 설치 아이템은 온보딩 지급/쉬운 제작으로 매끄럽게.
- **🚫 메뉴 아이템(핫바 9번) 폐기 (확정 2026-06-09):** 핫바 슬롯을 먹어 비효율 → 제거. **GUI 접근 = ①코어 블록 우클릭 ②기지 영역 내 Shift+F 단축키.**
  - **GUI 탭(서비스):** 상점(구매·판매) · 배낭 업글 · 기지 관리(코어·방벽·권한) · **연합 관리(초대·멤버·권한·탈퇴)** = **기지 안에서만**(코어/Shift+F=거점 서비스). **📊 정보 탭(통계·진행·결전 D-N 카운트다운) = 어디서나 분리 (확정 2026-06-10):** 조회 전용이라 거점에 묶을 이유 약함 + 결전 카운트다운 등 휴대 확인 편의 ≫ 일관성 약화 → **별도 단축키로 어디서나 열람**(거래·관리는 여전히 기지 안). 정보=읽기 전용이라 게이트 의미 없음(돈·진행에 영향 X).
  - **🤝 연합 초대=기지 안 GUI / 수락=어디서나 (확정 2026-06-10):** 초대(거점 관리 행위)는 기지 안 「연합 관리」 탭, **수락은 어디서나 알림 팝업으로**(밖에서 만나 즉시 합류=가벼운 응답). → [`base_raid.md`](base_raid.md) 「연합 가입·권한」.
  - **보안함은 "물건"이라 별개** → 인벤 특수 칸이라 어디서나(↓survival 「보안 도구함」). 메뉴=서비스(거점)·보안함=물건(휴대) 분리.

## 온보딩 · 신규 완충
- **온보딩 (인게임 코드 인증 = 강한 양방향 증명, 확정 2026-06-10):** **①신규 접속** → porongun-core가 **미인증자를 격리 로비에 가둠**(화이트리스트가 아니라 모드 게이트 — 로비 입장은 허용해야 코드 수령) → **②로비**(서버 설명·규칙·톤 + **인게임 코드 발행**) → **③디스코드 서버에서 봇 `/verify <코드>`**(디스코드 멤버·역할 = **지인 비공개 게이트**) → **④봇이 코드 매칭 → porongun-core에 인증 신호 → 본섭 텔포 + 랜덤 스폰 + 계정 연결**(기존 세력 옆 안 떨어지게 분산).
  - **🔐 강한 양방향 증명 (IP 불필요):** *게임 접속 = 그 정품 계정으로 로그인 = 계정 소유 증명* / *코드 입력 = 디스코드 본인* → 둘이 **코드로 자동 묶임**(신뢰 기반 X). **IP 매칭 검토·기각**: 디스코드가 사용자 IP 비공개(봇이 못 봄) + 설사 얻어도 NAT(같은 집)·모바일↔PC·CGNAT·VPN으로 부정확.
  - **🤖 봇 = `porong-discord` 확장(전 서버 공통)** + **마크 측 = porongun-core 인증 모듈**(코드 발행·로비 격리·인증 플래그). 봇↔모드 통신 = RCON/공유 DB/REST.
  - **🧩 모듈형 설계:** 인증을 porongun-core 내 **독립 모듈**로 분리 → 같은 **Forge 1.20.1** 서버끼리 떼어 재사용 여지(봇은 전 서버 공통). 단 porong-mon(Fabric)·porong-rpg(Paper)는 플랫폼 달라 단일 jar 공유 불가(봇만 공유).
  - **영구:** 한 번 인증=유지(재접속·시즌 와이프에도 재인증 불요, 계정 연결 보존) / 디스코드 추방 시 인증 해제 연동(운영·밴). 코드=시간 만료(보안).
  - ⚠️ **착수 검증:** porongun-core 인증 모듈 + porong-discord 봇 연동(다른 폴더라 착수 시 별도) + 로비 월드 구성.
- **🛡️ 신규 보호막 (PvP 면역 — 진입 완충):** 최초 진입 후 **실시간 12시간**(잠정) 동안 **타 플레이어의 대상 지정·피격 면역**(약탈 표적도 안 됨).
  - **악용 방지:** **최초 진입 1회만**(매 사망 리스폰엔 적용 X). **선공 시 즉시 해제.**
  - **PvE엔 적용 X:** 스캐브·좀비·굶주림·환경엔 첫날부터 노출 → 환경 가혹함 유지.
  - **🪟 시각 = 은은한 보호막 파티클 + 본인 UI 타이머 (확정 2026-06-10):** 미묘한 발광 막(본인+타인 식별, 방벽 룬과 톤 일관). **근거:** ①선공 해제 규칙상 본인이 보호 상태를 알아야 실수 선공 방지 ②타인 헛공격·낚임 분쟁 방지. **풀리기 직전 파티클이 옅어지며 자연 전환**(약자 표적 광고 우려 완화 — 보호 중엔 어차피 피격 X). 본인=화면 비네트+타이머.
  - **12h = 실시간 (확정 2026-06-10):** 첫 진입 후 현실 12시간(접속 안 해도 흐름, 러스트 표준·단순, 하루 적응 후 자연 노출). **결전 직전 진입자 = 동일 적용**(보호막=PvP 면역인데 결전은 PvE라 무관 → 막판 진입자도 좀비·스캐브엔 노출, 농성만 가능).
- **🎒 기본 보급 = "첫 출격 1회분 생존 키트" (최소, 확정 2026-06-10):** 어려움+총겜이라 맨손=즉사 / 거래는 코어 후라 코어 없으면 진입 막힘 → 최소 지급. **과하면 파밍 동기 약화라 딱 첫 30분치.**

  | 🏠 코어 설치 아이템 | ×1 | 거점·거래 첫 관문 |

  - **🔑 최초 1회만(리스폰엔 X)** — 신규 보호막과 같은 규칙, "자살→보급→자살" 파밍 악용 차단. → 거래 불가(코어 전)·어려움 상태에서 첫 생존 가능.
  - **🏠 코어템 = 연합 가입 시 회수**(연합=코어 1개 공유 거점이라 자기 코어 불필요, 설치한 코어도 연합 코어로 통합) / 재제작은 **이미 코어(또는 연합 코어) 있으면 불가**(비연합 개인 1개/연합 공유 1개). → [`base_raid.md`](base_raid.md) 「연합 가입·권한」.

**파밍 가능한 바닐라 재료(철·구리·금·다이아·화약·몹 드랍)는 진짜 게이트가 될 수 없다.** 몹팜·철골렘 공장 등으로 무한 생산 가능(막지 않음 — 자정작용 판단). 따라서:
- **바닐라 재료 비용 = 초반 페이싱에만 유효**(팜 짓기 전). 몹팜 나오는 순간 기본 재료 캡 해제.
- **진짜 게이트 = 비파밍 자원:** ①**청사진**(POI 루트 드랍) ②**커스텀 재료 "군용 합금"**(POI·스캐브 전용 드랍) ③**커스텀 화폐**(POI·스캐브) ④POI 위험 그 자체.
- → **고급 총·탄·방어구는 POI/커스텀 재료를 끼워야** 의미 있는 게이트. 마침 TaCZ 고급 레시피가 **네더라이트·blaze를 요구**(네더 봉쇄로 불가) → 그 자리를 **군용 합금/POI 재료로 치환** = 네더 충돌 해결 + 비파밍 게이트 동시 달성(일석이조).

### #1-B. 커스텀 아이템 제작법 (porongun-core)
> 모든 게이트 재료는 porongun-core 커스텀(모드에 없음, 2026-06-09 스캔 확인).

#### 🧱 재료 3층 트리 (확정 2026-06-09, 소재/발사약 분리)
**소재(탄두 금속) ↔ 화약(발사약) 분리** — 텅스텐/열화우라늄은 순수 금속 소재, 화약은 *탄약 조립 단계*에서 결합(전 구경 공통). 직관적이고, 소재 게이트와 화약을 독립 조절 가능.
```
바닐라(파밍 가능) ─┬─ 철·구리 ───────→ ⚙️ 텅스텐 합금(작업대)        → 소총탄 탄두·총 프레임
                  └─ 텅스텐+다이아 ──→ ☢️ 열화우라늄(AP 라인)        → 저격·대물 AP탄·고급 총 (※배틀라이플 .308=텅스텐 라인)
        + 화약(발사약) ─────────────→ 탄약 조립 단계에서 소재와 결합
POI·스캐브 드랍(비파밍, 레시피 없음) → 🪖 군용 합금 · 🔌 전자 부품  → 최상위 총·방어·탈출·AP탄 (네더라이트/blaze 치환)
```
- **텅스텐 합금·열화우라늄 = 작업대 직접 제작(청사진 불요).** 페이싱 층("탄 공짜 아님"), 결국 파밍 가능. **화약 미포함**(소재일 뿐).
- **🪖 군용 합금 · 🔌 전자 부품 = drop-only (확정).** POI/스캐브 드랍만, 제작 불가 = 유일한 진짜 게이트. 드랍률=진행 노브(#2).
  - 군용 합금=금속(방어·총 프레임), 전자 부품=정밀(고급 탄·신호기·탈출 데이터). → 금속/정밀 라인 분리로 POI 쏠림 완화.

#### ⚖️ 재료 배분 원칙 — 소모성 vs 영속성 ("밑 빠진 독" 방지, 확정 2026-06-09)
**비파밍 재료를 소모템(탄약)에 묶으면 밑 빠진 독** — 탄약은 쏘면 증발(가치 X), 티켓·기지·장비는 보존(가치 O). 인당 시즌 비파밍 공급 ~48개라도 대물탄을 매번 쓰면(전자 2.5/출격) 영속 진행(기지·부품)에 쓸 게 증발 → 그래서 대물탄=아껴 쓰는 사치(↓#2-A AP 빈도).
| 용도 | 성격 | 재료 정책 |
|---|---|---|
| **영속**(기지 LV5·고급 총·방어구·**티켓·탈출 부품**) | 가치 보존(죽어도 청사진·보안칸·기지) | **비파밍(군용합금·전자) 집중** |
| **소모**(탄약·폭약·의료) | 증발 | **파밍 위주**(화약·텅스텐·구리), 비파밍은 **고비용·저빈도 사치** |
- **🔑 원칙: 비파밍의 주 용도 = 영속템**(필수 진행). 소모템은 ①일상 화력·기본 탄=파밍(화약·텅스텐=무한 받침) ②고급 탄·폭약=비파밍이되 **"가끔 큰 한 방"**(대물탄·T3 폭약=선택적 사치, 없어도 기본 생존, 아껴 써 증발 느림).
- **효과:** 비파밍 공급(빡셈)이 영속 진행에 온전 투입→결전 도달 최소선 확보 / 대물탄·폭약=아껴 쓰는 결정적 한 방(타르코프st "이 탄 쏠 가치 있나") / 일상 교전=파밍 화력=누구나.
- ⚠️ #2 조정: 고급 탄·폭약의 비파밍 비중을 "저빈도 사치"로 낮춰 영속 진행 압박 완화.

**제작 재료 레시피 (잠정 수량 — 밸런싱):**
- ⚙️ 텅스텐 합금 = 철 ×9 + 구리 ×9 → 1 (TaCZ 작업대 `materials` 리스트, count 자유 = 슬롯 제약 X. **화약 제외** — 화약은 탄약 조립에서)

### #2-B. 경제 모델 (P2P 없음 → 단일 PvE 루프 + 약탈 재분배)
**플레이어 직거래 없음** → 경제가 **상점(PvE) ↔ 플레이어** 한 축 + **데스박스 약탈(강제 재분배)** 로 단순. 통제 포인트 딱 둘 = **인플레(화폐 총량)** + **진행 곡선(장비 게이트)**. 단순함이 1인 운영엔 장점.

**🎯 핵심 공식 — 파밍 기대값(EV):** 드랍률·위험·장비·적 강함이 전부 이 한 식의 항. POI 티어 노브를 이걸로 맞춤.
```
파밍 EV(티어) = 기대 보상 − 기대 비용
  기대 보상 = 재료 드랍률×가치 + 화폐 드랍 + 청사진 확률
  기대 비용 = 소모(탄약·의료·식량) + P(사망)×장비 손실
  P(사망) ∝ (적 강함 − 내 장비) + PvP 조우 확률
```
- **설계 목표:** 각 티어가 **"적정 장비를 갖춘 자에겐 EV 양수, 맨몸엔 음수"** → 장비 업글이 `P(사망)↓`로 EV를 양수화 = **장비 진행이 자연 강제**("업글하고 가는 게 도움 되는가?"의 답=YES). 고티어 적은 확실히 강함(스캐브 Rookie→Expert). → POI 티어 매트릭스 = [`world.md`](world.md) 「POI 티어」.

**📒 수급(Source) / 싱크(Sink) 장부:**
| 화폐 수급(Source) | 화폐 싱크(Sink) |
|---|---|
| 스캐브 처치 · POI 화폐 상자 | **🧨 화약**(LV1~2 핵심) · 의료템 · 탄약 |
| 작물 · 광물 판매 | 청사진 보조 · 기지 수리 · 상점 총 |
| **고티어 재료 저가 판매**(잉여만) | (고티어 재료 **구매 불가**=게이트 보존) |

**🔩 고티어 재료(군용 합금·전자 부품) 상점 정책 (확정):** **구매 불가**(돈으로 게이트 우회 차단) + **저가 판매 가능**(판매가 ≪ 사용 가치 → "팔 바엔 쓴다", 잉여만 환금). → 게이트 보존 + 잉여 죽은자산 방지 + 화폐 수급 소액 보조.
- **🧨 화약 = LV1~2 상점 주요 판매:** 초반 핵심 화폐 싱크(돈→화약→탄약). 중반 크리퍼 팜 자급되며 비중↓. (↑#2-A 화약 항목.) ⚠️ 상점가 = 크리퍼 스폰율 연동(스폰 낮으면 가격↓ 완충).

#### 🔄 경제 페이즈 — 화폐의 역할 이동 (초반→중반→후반)

레이드 수치 모델·돌파 내구·폭약 위력·대물 관통·RPG = [`base_raid.md`](base_raid.md)「#2 4단계 레이드 수치」. 여기선 **공격자 레이드 지출이 3단계 공급(인당 비파밍 48)과 정합하는지**만 검산.

**레이드 1회 공격자 비파밍 지출(한 지점 종심 돌파):**

|---|---|---|---|---|---|
| **LV1 전초기지**(종심 100) | T1 ×2 | 0 | 0 | (텅스텐=파밍) | **비파밍 0 = 초반 약탈 활발** |
| **LV3 거점**(종심 400) | T2 ×4 | 0 | 4 | 열화 4 | 전자 4 (공급 8%) |

- **🔑 레이드 비용 = 3단계 빌드 위에 얹히는 추가 지출** → 풀 빌드(60)+요새 레이드(16+)는 공급 48 크게 초과. **충당 = 레이드 전리품(피해자 비축=비파밍·완성 장비)으로 회수** → "남의 비축 강탈" 동기(↑base_raid「왜 레이드를 하는가」)와 **비용·보상이 같은 방향** = 순환 정합.
- **🔑 함의 = 플레이 스타일 분기:** **평화 파밍파**(레이드 안 함)는 공급 48을 영속 진행에 온전 투입(탈출·농성) / **레이드파**는 비파밍을 폭약에 태우고 **약탈로 회수**(고위험 고보상). 둘 다 성립 = 강제 PvP 아님(받침+스파이스).

#### 📊 #2 5단계 — 통합 검산 (한 시즌 시뮬 + 모순 점검, 2026-06-10)
> **🔄 v2 우선:** 아래는 **v1 검산**(공급 48 빠듯·비파밍 게이트 전제). v2(탄 싸게·비파밍 과잉·레이드 장려)로 **전면 재계산 필요**(↑「v2 정밀 수치」) — 정밀 통합 검산은 착수 플레이테스트. v1 검산은 *방법론 참고*로 보존.
**1~4단계를 곱해 한 시즌(2주)·가상 플레이어로 돌려 모순을 찾는다.** 단계별 따로 잡은 잠정치가 결합 시 안 맞는 지점 = 노브.

**🎮 가상 플레이어 전제:** 시즌 14일 / 일 1세션(캐주얼 기준, 하드코어는 더) / POI 리필 24h×14회 / 서버 ~30명 / 출격 1회 ≈ 소총탄 150발.

**① 시간 ↔ 공급 정합 (공급 48 근거 재구성):**
- ⚠️ **단위 혼동 주의(검산이 잡아냄):** "후보 N곳"은 **한 POI 내부의 상자 후보 자리**(POI 크기 비례, T1 ~8 ~ T4 ~40, 매 리필 50% 활성)지 **POI 개수가 아님.** 전체 POI는 45~65개(T2~T4 ~20~30이 비파밍 산출) = 별개 단위. → 후보 수 티어 차등 = [`world.md`](world.md)「POI 상자 리필」.
- 하루 서버 비파밍 산출 ≈ Σ(T2~T4 POI × POI당 채움 상자 **티어 차등 T2~7/T3~10/T4~13**, ↓상세 테이블 × 비파밍 확률) ≈ **~100**(3단계 상향 후: 드랍률↑+T4 6개+채움 50%) ÷ 30명 = **인당 ~3.4/일 × 14일 ≈ 48** = **3단계 공급 48과 재구성 일치** ✅. (2단계 ~88은 상향 전 값.) 인당 하루 1.5~2 POI 경쟁 = world.md 분포 정합.

**② 비파밍 수지 (자급 + 레이드 통합):**
| 빌드 | 자급 수요 | 레이드 비용 | 총 | 공급 48 | 충당 |
|---|---|---|---|---|---|
| 평화 탈출 | 15 | 0 | 15 | +33 | ✅✅ |
| 농성 | ~40 | 0~RPG | ~40 | +8 | ✅ |
| 풀 빌드 | 60 | 0 | 60 | −12 | 레이드·약탈 |
| **레이드파**(풀+LV5 요새 레이드) | 60 | 16 | **76** | **−28** | **약탈 회수** |
- **🔑 약탈 회수가 격차를 메움:** LV5 요새 비축 = 비파밍 ~20~40(기지 24+고급총+탈출 부품) → **성공 레이드 1~2회 = 비파밍 20~40 회수** ⇒ −28 충당 ✅. 비용(16)·보상(20~40)이 양수 EV = "강탈 지름길" 동기 정량 성립(↑base_raid). ⚠️ **약탈 회수량 = 노브**(요새 비축 규모 = 너무 크면 레이드 과보상, 작으면 동기 死).

**③ 화폐 수지 (인플레, 약식 재확인):** 누적 수입 ~12~15k vs 지출 ~16.8k → **부족 ~2~5k = 잉여 환금(작물·광물·재료 판매)이 밸브.** 군용합금 80×잉여 + 작물 대량 = 수백~천 단위로 메꿈 → 인플레 억제 ✅. "더 벌면 더 노출"(비정 톤) 정합.

**④ 탄·화약 수지:**
- **기본탄 3,000발/시즌**(150×20출격) = 구리·텅스텐·화약(전부 파밍) → 비파밍 0. 화약 ~200개 = 자급/상점 1,000코인(화폐 지출에 포함) ✅.
- **AP탄(열화+전자)** = **7정**(저격·대물, .308 배틀라이플 제외 후) 쓰는 사람만, 소수·아껴. **🔑 "출격" 구분 (혼동 해소):** 일상 POI 파밍 출격(시즌 ~20회)은 **기본탄·배틀라이플탄(텅스텐=파밍)으로 충분** = 비파밍 0 — *POI 가는 것 자체엔 AP 안 씀*(고급총·AP=대장갑 전용 원칙↑#2-A). 특히 **배틀라이플(.308)이 POI 일상 주력이라 AP 없이 고화력** 운용. **AP를 실제 소비하는 건 저격·대물 대장갑 교전**(Heavy 갑 적·엘리트·PvP 중장갑·방벽)뿐.
  - ⚠️ **AP탄 소비 교전 = 시즌 ~5회라야 정합**(교전당 AP 60발≈전자2 → 5회=전자10, 전자 공급 ~24 중 영속[탈출·기지] 쓰고 남는 여유분). 10회+면 전자 적자 → **비싸서 자연 억제**(영속 우선 원칙). POI 파밍 빈도와 무관 = POI는 매일 가도 됨. 단 AP 교전 빈도·전자 여유 = 플레이테스트 모니터(빡빡하면 전자 공급↑ or AP탄 전자 비중↓ 노브).

**🔍 모순 점검 결과:**
| # | 점검 | 결과 |
|---|---|---|
| 1 | 공급 48 ↔ 시간축 | ✅ 재구성 일치(하루 ~100/30명×14, 상향 후) |
| 2 | 풀+레이드 76 > 공급 48 | ⚠️ 약탈 회수 20~40으로 충당 — **회수량 노브** |
| 3 | "후보 20곳" 단위 | ✅ **검산이 혼동 잡음**: POI당 상자 자리지 POI 개수 아님(전체 45~65) — 문서 정정 |
| 4 | AP탄 빈도 ↔ 공급 | ⚠️ **AP=대장갑 교전 한정**(일상 POI 파밍은 기본탄=별개). AP 소비 교전 ~5회/시즌(비싸서 자연 억제) — 빈도 모니터 |
| 5 | 화폐 지출 ≳ 수입 | ✅ 잉여 환금 밸브로 인플레 억제 |

**✅ 결론:** **1~4단계 앵커 체인 자기정합** — 전투 TTK → 공급 48 → 비파밍 비용 → 화폐 → 레이드가 모순 없이 맞물림. 남은 3노브(**약탈 회수량·POI 후보 정의·AP 출격 빈도**)는 수치 보정 영역(설계 충돌 아님) → 플레이테스트로 확정. **#2 밸런싱 앵커 체인 완료.** 정식 착수 시 datapack 수치·드랍 테이블로 구현 + 인게임 실측 보정(특히 First Aid TTK·방어 포인트 오버라이드 = 1단계 0순위 검증).
