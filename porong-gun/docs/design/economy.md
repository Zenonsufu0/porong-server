## 총 획득 (TaCZ 제작 시스템 기반)
- **작업대 2종(jar 확인 2026-06-09):** `gun_smith_table`(총·부착·텅스텐 등 제작, 통나무3+철4+철블록1) + `ammo_workbench`(탄약 전용, 구리2+철1+탄약상자). 탄약 레시피 type은 둘이 공유(`gun_smith_table_crafting`). 둘 다 바닐라 재료로 제작 = 진입 허들 낮음. (옛 "단일 1종"·"3종" 표기 정정.)
- **기본 총**(티어 0~1): 청사진 없이 제작 → 초반 진입(권총·기본 소총). **PvE 받침·재기.**
- **고급 총**(티어 2+): **청사진(POI 루트 파밍)으로 해금** → 작업대 제작. 청사진=레시피 영구 학습(죽어도 유지=재기 동력), 총 완성품은 소모(죽으면 데스박스). **청사진을 POI 티어 루트에 배치** = "좋은 총=진행 보상". 구현 = **TaCZ Weapon Blueprints**(1.20.1 Forge 확인).
- **시즌 시너지:** 와이프 시 `/gg clearRecipes`로 무기 청사진 리셋 → 매 시즌 무기 해금 재파밍. (탈출 부품·티켓 청사진은 porongun-core 자체 리셋.)
- 탄약 = ammo_workbench 제작(구경별 레시피) + 상점 / 폭약 = 고티어 POI 재료 제작(레이드 비용) / 드랍 = 스캐브·데스박스 약탈.

## 🦺 방어구 = 전술 갑옷 (`[TaCZ] Tactical 3D Armor`)
바닐라 갑옷 아님(테마·총겜 정합). **3티어 = 방탄 등급:** Light(권총·SMG) → Medium(라이플·DMR) → Heavy(중화기). 바닐라 갑옷은 초반 저티어 fallback.
- **스캐브 약탈 루프:** 스캐브 랭크별 갑옷(Rookie→Expert) → 처치해서 **약탈** = 타르코프st 진행. 상점/POI/제작 보조.
- **무게 트레이드오프:** 중갑일수록 무거움 → **속도 페널티↑** → 방어 ↔ 기동. 무게=점유칸·다리 부상과 같은 둔화 축.
- **First Aid 부위별 보호 연동:** 헬멧=머리·베스트=몸통 부위 HP 보호. ⚠️ 착수 검증: Tactical 3D Armor × First Aid 연동.

## 의료템 (First Aid 약품 — 핵심 소모재)
**상점 구매 + 드랍(스캐브·데스박스) + 제작**(붕대=종이+양털st 저티어 / 부목·모르핀·수술키트=상위). 부상 치료 강제(다리=둔화)라 **총·탄약과 함께 출격 전 필수 구비** = 꾸준한 화폐 싱크 → 경제 순환의 엔진.

## 화폐 · 수급
- **화폐 = 커스텀 화폐**(스캐브·POI 드랍 = 타르코프 루블st). **파밍이 곧 돈.**
- **💰 화폐 수급 다변화 — 농사·채굴 환금:** 파밍(POI·스캐브) 외에 **바닐라 작물·광물을 상점에 판매**해 화폐화. 비전투 자급 플레이어도 경제 참여(PvE 받침).
  - **자급 vs 잉여 (안전 ↔ 리스크 트레이드오프):**
    - **자급용 소량 농사**(먹을 만큼) = **은신처(기지 보호 영역) 안에서 안전.** 누구나 연명 가능.
    - **환금용 잉여 생산**(팔 만큼 대량) = **기지 밖에서 리스크 감수** — 기지 영역은 작게 시작이라 대농장 공간 부족, 광물은 밖/지하 채굴 필요 → **노출된 농장·광산 = PvP·스캐브 표적.**
  - 함의: 비정한 톤과 정합 — 돈을 더 벌려면 더 노출된다. 밖에 펼친 농장·광산이 약탈·습격의 자연 표적 → PvP 스파이스 유발.

## 상점 (메뉴 아이템 기반, 중심 허브 건물 없음)
- **접근 제약 = 자기(또는 연합) 기지 안에서만 거래 활성 (확정).** 허브 없이 *기지가 거래 거점*, 긴장 유지.
- 등급 해금 = **기지 레벨**(고레벨일수록 고급 무기·탄약 구매).
- 함의: 신규는 **코어 설치 전엔 거래 불가** → 온보딩 후 첫 목표 = "코어 박고 거점 만들기". 코어 설치 아이템은 온보딩 지급/쉬운 제작으로 매끄럽게.
- **메뉴 아이템 (핫바 9번 칸):** 슬롯 고정·드랍 불가·무게 0. 기지 관리·상점·정보 GUI. **정보·기지관리는 어디서나, 상점 거래는 기지 안에서만.**

## 온보딩 · 신규 완충
- **온보딩:** 격리 로비 → **서버 설명 + 디스코드 인증** → **랜덤 좌표 스폰**(신규가 기존 세력 옆에 안 떨어지게 분산). 디스코드 인증 자체 구현(봇 연동 방식은 착수 시; porong-gun 독립).
- **🛡️ 신규 보호막 (PvP 면역 — 진입 완충):** 최초 진입 후 **실시간 12시간**(잠정) 동안 **타 플레이어의 대상 지정·피격 면역**(약탈 표적도 안 됨).
  - **악용 방지:** **최초 진입 1회만**(매 사망 리스폰엔 적용 X). **선공 시 즉시 해제.**
  - **PvE엔 적용 X:** 스캐브·좀비·굶주림·환경엔 첫날부터 노출 → 환경 가혹함 유지.
  - ⚠️ 미정: 12h 기준(실시간 vs 플레이타임), 결전 직전 진입자 처리.
- **🎒 기본 보급 (진입 완충):** 최초 스폰 시 **코어 설치 아이템 + 기본 총·탄약 + 의료템·식량 소량** 지급 → 거래 불가(코어 전)·어려움 난이도 상태에서 첫 생존 가능.
- **중심 허브 없음 → 개인 기지가 각자 허브.**

### 🔑 경제 근본 원칙 (2026-06-09 — 핵심 결정)
**파밍 가능한 바닐라 재료(철·구리·금·다이아·화약·몹 드랍)는 진짜 게이트가 될 수 없다.** 몹팜·철골렘 공장 등으로 무한 생산 가능(막지 않음 — 자정작용 판단). 따라서:
- **바닐라 재료 비용 = 초반 페이싱에만 유효**(팜 짓기 전). 몹팜 나오는 순간 기본 재료 캡 해제.
- **진짜 게이트 = 비파밍 자원:** ①**청사진**(POI 루트 드랍) ②**커스텀 재료 "군용 합금"**(POI·스캐브 전용 드랍) ③**커스텀 화폐**(POI·스캐브) ④POI 위험 그 자체.
- → **고급 총·탄·방어구는 POI/커스텀 재료를 끼워야** 의미 있는 게이트. 마침 TaCZ 고급 레시피가 **네더라이트·blaze를 요구**(네더 봉쇄로 불가) → 그 자리를 **군용 합금/POI 재료로 치환** = 네더 충돌 해결 + 비파밍 게이트 동시 달성(일석이조).

### #1-A. 제작 정책 (유지 / 오버라이드 / 청사진 게이트)

### #1-B. 커스텀 아이템 제작법 (porongun-core)
> 모든 게이트 재료는 porongun-core 커스텀(모드에 없음, 2026-06-09 스캔 확인).

#### 🧱 재료 3층 트리 (확정 2026-06-09, 소재/발사약 분리)
**소재(탄두 금속) ↔ 화약(발사약) 분리** — 텅스텐/열화우라늄은 순수 금속 소재, 화약은 *탄약 조립 단계*에서 결합(전 구경 공통). 직관적이고, 소재 게이트와 화약을 독립 조절 가능.
```
바닐라(파밍 가능) ─┬─ 철·구리 ───────→ ⚙️ 텅스텐 합금(작업대)        → 소총탄 탄두·총 프레임
                  └─ 텅스텐+다이아 ──→ ☢️ 열화우라늄                 → 대물탄(AP) 탄두·고급 총
        + 화약(발사약) ─────────────→ 탄약 조립 단계에서 소재와 결합
POI·스캐브 드랍(비파밍, 레시피 없음) → 🪖 군용 합금 · 🔌 전자 부품  → 최상위 총·방어·탈출·AP탄 (네더라이트/blaze 치환)
```
- **텅스텐 합금·열화우라늄 = 작업대 직접 제작(청사진 불요).** 페이싱 층("탄 공짜 아님"), 결국 파밍 가능. **화약 미포함**(소재일 뿐).
- **🪖 군용 합금 · 🔌 전자 부품 = drop-only (확정).** POI/스캐브 드랍만, 제작 불가 = 유일한 진짜 게이트. 드랍률=진행 노브(#2).
  - 군용 합금=금속(방어·총 프레임), 전자 부품=정밀(고급 탄·신호기·탈출 데이터). → 금속/정밀 라인 분리로 POI 쏠림 완화.

**제작 재료 레시피 (잠정 수량 — 밸런싱):**
- ⚙️ 텅스텐 합금 = 철 ×9 + 구리 ×9 → 1 (TaCZ 작업대 `materials` 리스트, count 자유 = 슬롯 제약 X. **화약 제외** — 화약은 탄약 조립에서)
  - 단가 ↑(철3구리3 대비 3배) = **팜 전 페이싱 강화**(철구리는 팜으로 무한이라 비싸도 자정작용 OK). 연쇄: 열화우라늄·T1 폭약·소총탄 비용 동반 상승 → 소총탄 텅스텐 수량은 낮춰 균형(↓#2 탄약표).
- ☢️ 열화우라늄 = 텅스텐 합금 ×4 + 다이아 ×1 → 1 (잠정 = 철36구리36+다이아1 등가)

#### 🔫 탄약 = 구경(caliber)별 — 총마다 전용 아님 (TaCZ 실구조)
TaCZ는 **구경별 탄약을 총끼리 공유**(같은 구경 총은 탄 돌려씀). → 53정 개별 아니라 **구경 ~10종에만** 소재 치환하면 됨. 구경이 곧 티어:
| 구경 티어 | 구경(예) / 쓰는 총 | 소재 결합(잠정) | 게이트 |
|---|---|---|---|
| **권총탄** | 9mm·45acp·50ae / glock·mp5·m1911·deagle | 구리 + 화약 (유지, 싸게) | 초반 받침 |
| **소총탄** | 5.56·7.62×39·7.62×51 / m4·ak47·m249·DMR | + ⚙️텅스텐 ×1 + 화약 | 파밍 페이싱 |
| **저격탄** | .30-06·.338·.308 / m700·awp | + ⚙️텅스텐 ×2 + 🔌전자 부품 + 화약 | POI 약게이트 |
| **대물탄** | **.50 BMG** / **m107·m95** | + ☢️열화우라늄 + 🔌전자 부품 + 화약 (blaze 제거) | POI 게이트 (AP탄) |
| **특수** | 12GA·40mm·로켓 / 산탄총·유탄·rpg | (밸런싱 별도) | — |
- **화약 = 전 구경 공통 발사약**(탄약 조립 시). 텅스텐/열화우라늄은 탄두 금속으로만 추가.
- ⚠️ 정확 수량·발당 산출 = #2. base_raid 「보조 돌파」의 **AP탄 방벽 보너스**도 이 대물탄(열화우라늄) 라인과 연결.

#### 💥 관통 폭약 3티어 (방벽 돌파 — [`base_raid.md`](base_raid.md) 수치 모델)
방벽 = 평소 곡괭이·TNT·일반 폭발 무효, **관통 폭약(주력) + 총격(극저효율 보조)**만 돌파 게이지 차감. 폭약 비용=비파밍 게이트 → 레이드 **빈도** 통제(눈덩이 완충 "레이드 비용"). 1발 돌파 위력 vs 방벽 겹 돌파 내구 = 필요 발수. **총격=1발≈0.3(잠정)** 극저효율=오프라인/저레벨 갉기 니치(↓AP탄 시너지 미정).
| 등급 | 돌파 위력 | 제작 재료(잠정) | 게이트 성격 |
|---|---|---|---|
| **T1 파쇄 장약** | 50 | ⚙️텅스텐 합금 ×2 + 화약 ×4 + 철 ×2 → 1 | 작업대 제작(텅스텐=파밍가능 페이싱). MVP 1종 |
| **T2 성형작약** | 120 | ☢️열화우라늄 ×1 + 🔌전자 부품 ×1 + 화약 ×4 → 1 | 전자 부품=POI drop 게이트 |
| **T3 관통 작약** | 300 | 🪖군용 합금 ×2 + 🔌전자 부품 ×2 + ☢️열화우라늄 ×2 → 1 | POI 이중 게이트(요새 외벽용) |
- **방벽 = 닫힌 외피의 최외곽 1겹**(밴드당 1겹, 모양 자유) + 권한 잠금 문 — 폭약은 그 외피/문만 돌파. **코어는 폭약 무효**(옵시디언급 곡괭이 채굴 시간). 방벽 모델·돌파 내구표·식별 = [`base_raid.md`](base_raid.md)「방벽 모델」·관통 폭약 절.
- ⚠️ 화약=바닐라(파밍 가능)지만 게이트는 텅스텐/열화우라늄/군용 합금·전자 부품이 잡음. 정확 수량·드랍률 = #2.

### #2-A. 탄약 밸런스 (잠정 — 기존 배치 유지 + 소재 게이트만 얹기)
**3원칙:** ①기존 TaCZ 배치·산출·구리·화약 **그대로 보존**(밸런스 검증됨), 우리 소재만 항목 추가. ②발당 비용 곡선 = 위력 비례(기존이 이미 9mm 0.2구리/발 → 50bmg 4.6구리/발=23배). ③**"얼마나" = 출격 1회 소비량 환산**(발당 비용 × 출격 소비 = 출격 비용이 티어 적정해야). 게이트 강도 = drop 재료(전자 부품)를 어느 티어부터 넣느냐.

| 구경 티어 | 대표 구경 | 기존 배치 → 산출(실측) | + 우리 소재(잠정) | 출격 환산 / 게이트 |
|---|---|---|---|---|
| **권총** | 9mm·45acp·50ae | 구리10+화약2 → 50발 | (없음) | 0 = 받침(누구나) |
| **소총** | 545·556·762×39 | 구리13+화약3 → 45발 | **+텅스텐 1** | 출격 150발 ≈ 텅스텐 3~4(철30구리30) = 팜 전 페이싱 |
| **저격** | 30-06·338·308 | ~구리30+화약5 → ~30발 | **+텅스텐 2 +전자 1** | 출격 60발 ≈ 전자 2 |
| **대물** | **50bmg** | 구리110+화약20+청금석12+**blaze1** → 24발 | blaze→**전자 2 +열화우라늄 4**(청금석 유지) | 출격 30발 ≈ 전자 2.5 = **POI 2~3회 게이트** |
| **특수** | 12g·40mm·rpg | (별도) | 밸런싱 별도 | — |
- **출격당 소비량이 티어별로 다른 게 핵심:** 대물저격은 고데미지라 출격 ~30발이면 충분 → 발당 비싸도 총량 적어 "큰맘" 게이트로 적정. 소총은 ~150발 대량 → 발당 싸야(텅스텐 1) 감당.
- **텅스텐 단가 = 철9구리9**(↑3층 트리)라 소총탄 텅스텐은 1로 억제. **대물탄부터 전자 부품 필수** = 대물저격이 POI 게이트(AP탄, base_raid 「보조 돌파」 방벽 보너스와 연결).
- 오버라이드 대상 = 게이트 거는 **소총·저격·대물 구경만**(~8종). 권총탄·특수는 유지. ⚠️ 정확 발수·청금석 유지 여부 = 플레이테스트 보정.

### #2-B. 화폐 · 상점 가격 모델
*(화폐 단위 → 수급(스캐브/POI 드랍, 작물·광물·탄약 판매가) ↔ 싱크(의료·탄약·청사진·폭약·수리) 균형 → 잠정 가격표. 가격은 플레이테스트 보정.)*
