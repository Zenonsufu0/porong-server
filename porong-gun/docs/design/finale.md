# 설계 — 시즌 피날레: 최후의 결전

> 비전 SoT: [`../concept.md`](../concept.md) · 관련: [`base_raid.md`](base_raid.md)(농성·방벽), [`world.md`](world.md)(와이프 이월), [`../platform.md`](../platform.md).

시즌 마지막날 **단 1회**. 2주간의 모든 파밍·기지·장비·준비가 *이 밤*을 향한 대비. 거대 습격(좀비 호드 + 스캐브)이 몰려오는 혼돈의 클라이맥스이자 시즌 엔드게임.

## 근본 형태 = 농성(기본) + 탈출(특별 옵션) 하이브리드
생존(클리어) 경로는 둘이고, 둘 다 유효하다. 약탈·배신·각자도생도 그대로 — 강제 협력은 없다(혼돈 무대 성격 유지).
- **🛡️ 농성 (누구나 가능):** 결전이 끝날 때까지 버티면 → 칭호 **`[최후의 생존자]`**.
- **🚁 탈출 (준비한 자만, 최상위 목표):** 시즌 내내 **미션으로 부품·티켓 청사진 해금 + 희귀 재료로 제작**해 **탑승 티켓(구조 신호기)**을 완성 → 결전 때 고정 추출 지점에서 **구조 헬기** 호출·탑승 → 칭호 **`[희망을 품고]`** + **배낭(인벤) 전부 다음 시즌 이월**.

## 🚁 탈출 경로 — 구조 헬기 + 탑승 티켓 (시즌 최상위 목표)
- 탈출은 "버티기 회피 꼼수"가 아니라 **농성보다 높은 목표.** 시즌 동안 **특별 미션 체인 + 희귀 재료 수집**으로만 **탑승 티켓**을 만들 수 있다.
  - **티켓 = 청사진 테크 트리 (기존 청사진 시스템과 정합):**
    - **① 부품 청사진 (병렬 미션):** 고티어 POI에 흩어진 **단서/미션을 아무 순서로** 수행 → 각 **부품 청사진** 해금 → *맵 전역 탐험 강제.* 부품·POI 매핑(ChaosZPack 타입):
      | 부품 | 수급 POI | 의미 |
      |---|---|---|
      | 📡 무선 송신기 | 통신탑/라디오 타워 | 신호 송출 |
      | ⛽ 항공 연료 | 군사기지/추락 헬기 잔해 | 헬기 가동 |
      | 💾 좌표 데이터 | 연구소/병원 서버 | 추출 좌표 해독 |
    - **② 부품 제작 = 고난도:** 청사진을 얻어도 **희귀 재료(고티어 POI 드랍)** 가 있어야 제작 → *만들기 어려움*(최상위 목표에 맞는 난도).
    - **③ 최종 티켓 청사진 (마지막 미션):** 부품 청사진을 다 모은 뒤 **마지막 미션(보스 스캐브 처치 등 클라이맥스 게이트)** → **구조 신호기(=티켓) 청사진** 해금.
    - **④ 조립:** 완성 부품 + 신호기 청사진 → **구조 신호기(티켓)** 제작 → 보안함 보관.
    - **청사진=영구 학습(죽어도 유지)** 이라 진행은 보호되되, 부품·티켓 **완성품은 소모/귀속.** 매 시즌 재파밍.
    - **⚠️ 구현 구분:** TaCZ Weapon Blueprints는 *총* 레시피만 해금(`/gg clearRecipes`로 리셋) → **탈출 부품·티켓 청사진은 총이 아니므로 porongun-core 커스텀 recipe-unlock.** 와이프 시 porongun-core가 *자체 리셋.* 무기 청사진과 **같은 패턴**(파밍→영구 학습→와이프 리셋)이되 **같은 구현체는 아님.**
- **1티켓 = 1인 = 1좌석.** 연합이 다같이 탈출하려면 **인원수만큼 티켓을 모아야** 한다(혼자 살 것인가, 동료 몫까지 만들 것인가 = 비정한 톤의 드라마).
- **🔑 티켓 귀속 = 보안 도구함 전용 보관:** **완성 티켓은 보안 도구함(죽어도 안 잃는 칸)에만 넣을 수 있다** → 데스박스에 안 떨어짐 = **강탈 불가**(2주 노력이 막판에 안 날아감). 단 **조립 전 부품·재료는 일반 인벤** → 사망 시 데스박스 드랍 = **강탈 가능**(견제·약탈 동기 유지).
  - **전략 트레이드오프(보안함 칸 경쟁):** ①*지금 티켓 만들어 보안함에 안전 보관* → 안심이지만 그 **칸이 시즌 내내 묶임**(다른 귀중품·이월 stash 못 넣음). ②*보안함 아껴두고 재료를 들고 다님* → 칸은 자유롭지만 **재료 분실·강탈 위험** 감수. → 보안함 칸(기지 레벨로 확장, 와이프 이월 stash와 공유)이 시즌 내내 살아있는 결정이 됨.
- **좌석 한정 = 탈출 캡.** 총 좌석 수가 시즌 탈출 정원 → 티켓이 있어도 좌석이 차면 못 탐 = **탈출 자체가 경쟁.**
- → 농성파는 **기지에 투자**, 탈출파는 **맵을 헤집고 다님**(미션·부품) = 시즌 내내 **두 갈래 목표** 공존(플레이 분기).
- **추출 = 고정 지점 + 신호기 활성화 (절충, 2-c):** 결전 때 맵에 **고정 추출 지점**(1~2곳, 위험지) 공개 → 거기 도달해 **티켓(신호기)으로 헬기 호출·활성화 → 대기 → 탑승.** 가는 길·대기 중 **무방비**(고정 지점이라 집결=교전 핫스팟, 좌석 경쟁 명확). **탈출도 고난도** — 그래서 더 높은 목표.
- ⚠️ 미정(수치): 미션 개수·내용, 부품 제작 희귀 재료 종류·양, 헬기 대수·좌석 수(탈출 캡), 추출 지점 수·위치, 호출~탑승 대기시간, 좌석 선점 규칙.

## 🛡️ 농성 경로 — 방벽이 핵심
- **방벽은 결전 중에도 유지된다**(평소 밤처럼 좀비가 못 뚫는 안전벽이 아님). 단 결전 습격은 강력 → **몰려오는 좀비·스캐브의 공격으로 방벽이 깨질 수 있다.** (구현 = Undead Nights Demolition Zombies + Zombies Break & Build.)
- → **기지 레벨을 최대한 올려 방벽을 강화 + 다중 중첩(다중방벽)** 하는 것이 농성 준비의 핵심. [`base_raid.md`](base_raid.md)의 다중방벽·내구 업그레이드가 바로 이 밤에 빛을 본다.
- 다중방벽 = **시간 벌기**: 바깥 겹부터 깨지며 안쪽 코어까지 시간이 걸린다 → 그 사이 방어. 다 깨지면 내부 바닐라 → 코어 노출.
- **야외 농성**도 선택 가능(기지 보존·기동성, 단 엄폐 없음·스캐브 협공) — 코어 루프의 농성/야외 선택이 결전에서 증폭 적용.

## 규모 / 웨이브 / 스캐브 랭크
- **점증 웨이브:** 초반 Rookie 스캐브·일반 좀비 다수 → 후반 **Expert 스캐브 + 강화 좀비** 비중↑ (시즌 최고 난도).
- 스캐브 랭크 = **[TACZ] NPCs Rookie→Expert**를 결전 웨이브에 배치(POI 티어 배치와 동일 메커니즘, 결전은 최상위 난도). 진짜 총 쏘는 인간형이 좀비와 함께 압박.
- ⚠️ 미정: 웨이브 수·간격·규모, 랭크별 비율, 특수/보스 좀비 유무, 결전 총 지속시간.

## 발동 / 형평성
- **고정 시각, 시즌 마지막날 단 1회** → "다같이 모이는 날"로 형평성 자연 해결(스냅샷·오프라인 시뮬 불필요).
- 시즌 내내 **결전 D-N 카운트다운**이 보여 2주 준비를 이 밤으로 수렴. ⚠️ 정확 시각대 미정(현실 저녁 피크타임 권장).
- 참여 안 하면(오프라인) 칭호 없음 — 결전은 **참석하는 자의 무대.**

## 보상 — 농성=명예 / 탈출=명예+이월 (비대칭)
> 전체 와이프 이월 정책은 [`world.md`](world.md) 「시즌 와이프 — 이월 정책」 참조.
- **농성 성공 → `[최후의 생존자]`** 칭호(명예) + **보안함·통계·칭호 이월**(배낭 인벤은 리셋). 비정한 톤·공정성에 부합.
- **결전 중 사망 / 미접속 → 이월 없음(빈손).** 결전 생존이 모든 이월의 관문.
- **탈출 성공 → `[희망을 품고]`** 칭호 + **🔑 탈출 순간의 배낭(인벤) 전부를 다음 시즌으로 보존(이월).** 와이프를 넘어 자산을 가져가는 **유일한 경로** = 탈출이 최상위 목표인 이유.
- 비대칭의 의미: 농성은 "이 시즌을 끝까지 살아냈다"(명예), 탈출은 "다음을 향해 무언가를 가져갔다"(희망). 칭호 이름과 정합.
- **이월 범위 = 탈출 시 배낭(인벤) 전부, 상한 없음(잠정·넉넉).** 근거: **시즌2 운영 여부 자체가 미확정**이라 지금 빡빡하게 잡을 이유 없음 + 좌석 한정·고난도로 소수만 가능. (기지 레벨은 와이프 대상이라 이월 불가.)
- ⚠️ **시즌 간 눈덩이 재검토 여지:** 시즌2 운영이 확정되면 이월이 탈출자에게 다음 시즌 부트스트랩이 됨 → 그때 신규·농성파 균형 보고 **상한/범위 조정 가능**(현재는 넉넉 유지).

## 결전 중 약탈·데스박스 (평소와 다름)
시즌 마지막날이라 **일반 템은 곧 와이프 = 가치 약함** → 결전 약탈 동기는 ①**티켓 부품 강탈**(완성 티켓은 보안함이라 못 뺏고, 조립 전 부품만 — 탈출 가로채기/방해) ②**이월용**(내가 탈출할 거면 뺏은 템도 다음 시즌으로) ③**좌석 경쟁자 제거**(탈출 캡 경쟁)로 수렴.
- 데스박스: 결전 중엔 페이스·서버 부하 위해 **식별 단축/생략 + 짧은 지속**으로 조정 여지(평소 15분·유리 식별과 별도). ⚠️ 수치 미정.
