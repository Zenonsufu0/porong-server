## POI (전리품 거점) — 낮 파밍의 목적지
테마 = **현대 폐허**(버려진 연구소·군사시설·폐도시). 마크 기본 구조물(고대도시·정글사원)은 **판타지라 테마 불일치** → 비추.

- **📐 Lost Cities 실측 (jar 2026-06-09):** 빌딩 **34종**(1동=16×16(1청크)×다층, building1=9층+ 고층) + **대형 복합빌딩 13종**(멀티청크) + 흩어진 구조 3 + 거리·하이웨이·**지하철(raildungeon)**. 상자 = 층(part)당 **~1~2개** → 고층 1동 **~10개+**(상자당 2~4 아이템). → **POI 후보 풀에 충분**(빌딩 상자 자리 흡수). 도시 밀도 = rarity 조정(드문 폐도시).
- **🎯 테마 빌딩 보유 → 컨셉 분류·부품 POI 매핑 가능(다른 모드 거의 불필요):** generic(building1~8) 외에 **테마 빌딩 존재** — `radiotower`(통신탑)·`oilrig`(시추선)·`library`(도서관)·`shopping`·`townhall`·`center` + Underground Bunkers(군사). **탈출 부품 POI가 실제 테마에 자연 매핑:** 📡무선=**radiotower** / ⛽연료=**oilrig** / 💾데이터=**library** / 🪖군사=**Bunkers** ([`finale.md`](finale.md)).
  - **POI 티어링:** T1=building1~8·town·cabin / T2=shopping·library(저층) / T3=radiotower·oilrig·library·center(테마=부품) / T4=multibuilding(huge1/2)·Bunkers·지하철·townhall.
  - **충분성:** 구조 양·분위기·테마 다 커버 → **추가 구조물 모드 비추**(통합 게이트 재부담 + Create 의존류 위험). 더 깊게 원하면 Lost Cities **datapack**(citystyle로 생성 비율 조정·커스텀 빌딩 추가)이나 시그니처 schem — 단 part 블록맵 제작=맵 노동이라 후순위·선택.
- **구현 = 자동 생성 메인 (맵 직접 제작은 안 함):**
  - **자동 생성 모드 = The Lost Cities(본체) + Underground Bunkers** = 세계의 **양·분위기**를 깔아줌(폐도시·건물·벙커 흩뿌림). 노동 0.
    - ❌ **ChaosZPack 드랍**(통합 검증서 Create 등 8모드 강제 의존 → 경제·톤 충돌, [`../platform.md`](../platform.md)). 테마 구조물은 잃지만 Lost Cities 건물에 **우리 루트 티어링**으로 대체.
    ※ Lost Cities는 "전체 도시"가 기본 → **rarity 조정**해 "자연 지형 + 드문 폐도시 POI"로(2000² 자연 월드에 맞게).
  - **🔑 시즌제 시너지:** 와이프 시 **새 시드로 재생성 → POI 자동 재배치.** 매 시즌 새 맵이 공짜. (맵 제작 = 1인 개발 최대 시간 하마라 통째 우회.)
  - **시그니처 POI(선택, 직접 빌드 강제 아님):** 깊이를 더하고 싶을 때만. **직접 빌드 대신** ①**커뮤니티 schem 다운로드** 배치 ②자동 생성 대형 구조물에 **루트 티어링만** → *맵 노동 0 유지*.
- **💡 POI 티어 = 리스크-리워드 매트릭스 + 폭약 재료 통제 경로:** 경제의 뼈대(파밍 EV 공식 = [`economy.md`](economy.md) #2-B). 티어별 **(적 강함 × 드랍 × 위험 × 권장 장비)** 를 "적정 장비엔 EV 양수, 맨몸엔 음수"로 맞춰 = 장비 진행 자연 강제.

  | 티어 | POI 예 | 적(스캐브) | 위험 | 화폐 | 재료 드랍 | 청사진 | 권장 장비 | EV 성격 |
  |---|---|---|---|---|---|---|---|---|
  | **T1 흔함** | 민가·소형 벙커 | 없음~Rookie | 낮음 | 소액 | (없음) | 기본 총 | 권총·바닐라갑 | 안정·저보상, **맨몸 OK** |
  | **T2 보통** | 중형 건물 | Rookie~Regular | 중(+PvP) | 중 | 군용합금 **저확률** | 기본 AR | 소총·Light | 받침 |
  | **T3 고급** | 연구소·대형 | Regular~Veteran | 고 | 고 | 군용합금·전자 **중확률** | 고급 총 | 고급총·Medi | 고위험고보상, **장비 필수** |
  | **T4 군사** | 군사시설(희귀) | Veteran~Expert | 최고(PvP 핫스팟) | 최고 | 둘 다 **고확률** | 최상위·탈출 부품 | 대물·Heavy·**연합** | 최상위, 연합 권장 |

  - **드랍 = "낮은 스택 × 높은 확률"** (T4도 군용합금을 한 번에 쏟지 않고 여러 번 돌게 = 트레드밀, 단발 잭팟 방지). **고티어 적은 확실히 강함**(스캐브 랭크 Rookie→Expert 상승).
  - **폭약·고급 재료가 위험한 고티어에서만** 나오니 **레이드 비용이 자연 통제**(파밍 위험 ↔ 레이드 능력 연결). 고티어 재료=구매 불가·저가 판매([`economy.md`](economy.md) #2-B).
  - ⚠️ 잠정 매트릭스 — 티어 수·정확 드랍률·스캐브 배치 = #2/플레이테스트.
- **📍 POI 개수·분포 (잠정, 2000²=작은 맵):** 총 **~45~65개**(인당 ~1.5~2개=경쟁·겹침). T1 ~25~35 / T2 ~10~15 / T3 테마 ~6~10(**부품 radiotower·oilrig·library 각 2~3개**) / T4 ~3~5. **분포=흩어짐**(중앙=고티어 동심원 ❌ — 중앙 쟁탈 단순화·전역 탐험과 충돌, 흩어짐이 다극 경쟁·탐험 정합). **부품 POI 복수 배치**=접근성+경쟁 분산(1개뿐이면 탈출 독점·과열·한쪽 끝 유저 차단). rarity profile 조정, 정확=플레이테스트.
- **🐎 이동수단 = 말·보트(바닐라, 모드 0 — 확정 2026-06-09):** 2000²=끝-끝 말 **~2.5분**이라 충분. **텔레포트류 빠른 이동 비추**(이동=노출=리스크가 비정한 톤 핵심, 순간이동은 깸). 무게(과적)·다리 부상 둔화가 이동에 긴장. 말은 데려가다 죽음=리스크 유지. (차량 모드=현대 테마 보강 **후순위 옵션**, 통합 게이트 재부담·연료/수리 시스템이라 일단 보류.)
- **🏠 기지 입지 트레이드오프 (이동 부담은 의도된 선택):** 고티어/부품 POI **옆 기지**=파밍 편리 + 습격 표적·경쟁 노출 / **외곽 기지**=안전·은신 + 이동 부담. "어디 코어 박을까"=**안전 ↔ 편의 전략 선택** → 이동 부담은 버그가 아니라 입지 트레이드오프. (말·무게 둔화로 완충.)
- **🧟 POI 적 생태계 (결전 갑툭튀 방지의 기반 — 확정 2026-06-09):** POI 빌딩은 **실내가 어두워** 낮에도 **바닐라 몹(좀비·스켈레톤·거미) 자연 스폰** + 스캐브 = 폐도시 생태계. POI 파밍 = 스캐브(총격) + 바닐라 몹(실내)과 일상 교전. → **결전 적 = 이 생태계의 떼·강화 버전**([`finale.md`](finale.md))이라 "안 보이던 적 갑툭튀"가 없음. 위더·피글린 등 이질 몹은 톤·생태계 불일치라 미사용.
- **📦 POI 상자 리필 + 식별 (확정 2026-06-09) — 후보 풀 랜덤(≈위치 랜덤):** POI마다 **상자 후보 위치를 넉넉히**(예: 20곳) 잡고, **리필마다 그중 일부만**(예: 8곳) 랜덤으로 채움 → 플레이어 체감 = "매번 다른 자리" = **위치 랜덤과 사실상 동일** + 자연스러움 보장.
  - **왜 후보 풀:** 완전 절차적 랜덤(영역 스캔→아무 바닥)은 **벽 속·공중·방 밖 사고** + 스캔 비용. 후보 위치는 미리 검수돼 안전.
  - **🔑 후보 자동 추출:** **Lost Cities가 이미 건물에 바닐라 상자를 루트와 함께 배치** → 그 **기존 상자 위치 = 검증된 자연스러운 자리**를 후보 풀로 자동 흡수(수작업 0). 부족하면 런타임 실내 스캔(바닥+천장 휴리스틱) 추가. 우리 루트 테이블로 내용·랜덤 채움만 덮어씀.
  - **유리 가림 식별**([`survival.md`](survival.md) 「상자 식별」)과 결합 = 찼는지/비었는지도 열어 조사해야 앎(20곳 뒤져 8곳 찾기) → **파밍 = 탐색 + 도박**.
  - **리필 = 현실 24시간(매일 1회, 잠정 확정 2026-06-09)**, 채움 40%(20곳 중 8곳). 게임 일수 X 현실 시간(24/7 서버 정합). 데스박스 15분과 별개. 시즌 단위 = **새 시드**. 구현 = porongun-core 후보 풀 + 루트 테이블 재롤. ⚠️ 정확 비율·주기 = #2(economy #2-B 공급량 추정).
  - **🔑 리필 주기 = 레이드 동기 노브:** 너무 빠르면 **파밍 즉시 무한 → 레이드 동기 死**("POI가 항상 나음"), 너무 느리면 자원 고갈. 적정=파밍은 꾸준하되 시간 들어 **큰 비축은 레이드로 강탈할 가치 유지**. Y25 초기화 주기도 같은 축. → [`base_raid.md`](base_raid.md)「왜 레이드를 하는가」.
- **⚠️ 고티어 POI 루트 과부하 주의(🟠):** 고급 총 청사진 + 폭약 재료 + 탈출 부품 청사진 + 탈출 부품 희귀 재료가 *모두* 고티어 POI에 몰림 → 루트 테이블 설계 부담 + "고티어만 돌면 다 됨" 쏠림. 티어/POI 타입별 분산 배치로 완화 필요.
- **POI 티어 지정 = Lost Cities 건물 타입/희귀도 + 루트 테이블(porongun-core).** ChaosZPack의 라벨된 군사/병원/연구소 구조물이 없으므로, **건물 유형·층수·희귀도로 고티어 POI를 우리가 designate.** 미확정: POI 티어 수·루트 테이블·스캐브 배치·시그니처 schem 개수.

## 운영 전제
- **시즌(와이프) 운영.** 기본 2주, 유저 원하면 연장. 시즌 종료=와이프+시즌2 보강.
- 지인 전용 비공개(공개성 X), **최대 동접 ~30명 예측**, **1인 개발 + 1인 운영.**
- **월드 크기: 2000² (월드보더) = 30명 "초기값"(고정 아님, 인원 따라 유연 조정 — 확정 2026-06-09):** 30명 기준 인당 ~13만 블록²(러스트 체감 중간값). 기지·POI 분산 + 적당한 만남 빈도(끝-끝 말 ~2.5분, 기지~가까운 POI ~1분).
  - **왜 작게 시작:** 인원 적을 때 큰 맵 = **유령 서버**(아무도 안 마주침). 비공개 지인 서버 초반엔 **작은 맵이 만남·활기·PvP 긴장**에 유리. 처음부터 크게 잡는 것보다 작게 시작해 키우는 게 안전.
  - **🔧 유연 조정 2경로:** ①**월드보더 런타임 확장**(`/worldborder set`, 시즌 중에도 기존 청크 유지하고 바깥 확장 = 즉시 완충) ②**시즌제(2주)가 조정 주기**(다음 시즌 새 시드로 크기 변경). → 40~50명 정착 시 **2500~3000으로**. ⚠️ 큰 맵=서버 부하↑(Y25 리셋 청크·엔티티·용량) → 1인 운영 사양 대비 **단계적 확장**.
- 다른 Porong 서버와 완전 독립. 착수는 **porong-mon → porong-rpg 이후**(시간 여유 확보).
- **🌐 차원 정책 = 오버월드 단일 월드 (네더·엔더 전부 봉쇄):** **단일 경계 월드** 정체성(월드보더 — 다른 차원=경계 밖 도피처라 긴장 희석 방지) + **현대 폐허 테마**(판타지 지옥/엔드 이질) + 1인 운영 단순. **엔더 봉쇄**=엘리트라 차단(공중 악용 방지). **네더 봉쇄**=네더 루프 빠른 이동·네더라이트 갑옷 악용 차단. 네더 전용 자원(블레이즈 파우더 등)이 필요한 레시피는 **상점/POI/대체 제작**으로 공급. ⚠️ 미정: 봉쇄로 막히는 바닐라 진행(포션 등) 대체 경로 목록.
- **난이도 = 어려움(Hard) 고정.** 배고픔이 빨리 닳아 **식량 관리도 생존 압박**의 일부 → 은신처에 처박혀만 있을 수 없다(식량 소비) → **활동 강제 = 정체 방지.**
  - **굶주림 효과(바닐라 정합):** 배고픔 부족 시 **스프린트(달리기) 불가**, 0이면 **데미지**(어려움=사망까지) → **농사·사냥의 확실한 이유.** 기동 압박이 무게→속도·다리 부상과 함께 *세 번째 둔화 축*.
  - 식량 = 농사·사냥·상점 수급([`economy.md`](economy.md) 자급/잉여 트레이드오프). 굶주림 = 또 하나의 *밖으로 나갈 이유.*
  - **🍖 음식 vs 의료 역할 분담:** First Aid가 자연재생을 끄므로 **음식 = 굶주림 방지(생존), 의료템 = 부위 HP 회복(부상)** 로 분리 — 음식은 부위 HP 직접 회복 안 함. *기존 모드 정합(커스텀 0).* ⚠️ 굶주림 강도는 밸런싱.
  - 어려움이라 평시 PvE 압박(스캐브·좀비·체력 회복 제약)도 강화 = 비정한 톤·생존 무게 강화.

| 🚁 **탈출 성공** | **배낭(인벤) 전부 + 배낭 레벨** + 보안함 + 통계 + 칭호 |
| 🛡️ **농성 생존**(결전 끝까지 생존, 탈출 X) | 보안함 + 통계 + 칭호 (배낭 인벤·레벨 리셋) |
| 💀 **결전 중 사망 / 미접속** | **이월 없음 (빈손)** — 통계·칭호·보안함조차 X |

- **결전 생존 = 이월의 관문.** 죽거나 안 오면 보안함·통계·칭호조차 못 가져감 → **"마지막 밤에 살아남아라"가 시즌의 절대 동기.**
- 역할 구분: **명예(통계·칭호)** = 생존자 공통, **보안함** = 생존자가 지켜낸 stash, **배낭(인벤 전부 + 레벨)** = 탈출자 전용 특권(최상위 보상).
- **배낭 레벨(슬롯 18→27→36)은 탈출자만 보존**, 나머지(농성 생존자 포함)는 리셋 → 매 시즌 재성장.
- 기지·기지 레벨·일반 인벤은 누구든 와이프(거점은 영속 월드와 함께 리셋).
- POI도 새 시드로 재생성(매 시즌 새 맵). 무기 청사진은 `/gg clearRecipes`, 탈출 청사진은 porongun-core 자체 리셋.
