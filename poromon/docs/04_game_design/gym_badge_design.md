# Gym / Badge Design (체육관 / 배지 설계)  — 초안(DRAFT)

> 허브 체육관 구역(`hub_design.md` §3)의 게임 설계. 기록·보상은 PoroMonCore가 담당.
> 연동: `GymBadgeManager` / `PlayerProgress.badges`(`../03_poromoncore/database_schema.md`) / `gyms.json`(`config_structure.md`).
> ⚠️ 초안. `TBD`/`예시`는 검토·확정 대상.

## 1. 목표 / 원칙
- 8개 체육관 = 서버의 **핵심 진행선**(탐험·포획 후 도전하는 중간 목표).
- 짐 클리어 = 단순 보상이 아니라 **해금 게이트**(메가/테라·티켓 구매권·리그 자격).
- 자연 마을은 자유롭게 두고, **짐/배지는 허브에 중앙집중**(결정 010).
- 밸런스 값(레벨캡·보상량)은 하드코딩 금지 → `gyms.json`.

## 2. 8개 체육관 구성 (타입/순서 — 타입·이름은 예시, 레벨캡 곡선은 확정 방향)

| # | 타입(예시) | 레벨캡 | 배지(예시) |
|---|---|---|---|
| 1 | 벌레 Bug | 18 | 코쿤 배지 |
| 2 | 바위 Rock | 28 | 스톤 배지 |
| 3 | 전기 Electric | 38 | 볼트 배지 |
| 4 | 풀 Grass | 50 | 리프 배지 |
| 5 | 물 Water | 62 | 타이드 배지 |
| 6 | 불꽃 Fire | 75 | 플레임 배지 |
| 7 | 에스퍼 Psychic | 88 | 마인드 배지 |
| 8 | 드래곤 Dragon | **100** | 드래곤 배지 |

- **마지막 관장(8번) 레벨캡 = 100 확정.** 1→8 점증 곡선(18~100). 중간 캡 수치는 미세조정 가능하나 8번=100 고정.
- 타입 8종·순서·배지 이름은 **확정 필요**(예시). 관장별 타입 문양은 입장구에 표시(`hub_design.md` §3).
- 레벨캡 = "도전 시 파티 최대 레벨" 제한(§4).

## 3. 체육관 구조
- **입장구(8개)**: 타입 문양 표시. 클릭/상호작용 시 도전 시작.
- **관장(NPC)**: 고정 파티로 Cobblemon 배틀. 파티 구성은 `gyms.json`.
- **배지 보상 NPC**: 클리어 후 배지/보상 수령.
- **재도전 NPC**: 이미 깬 짐 재도전(§6).

## 4. 배틀 규칙
- 형식: **Cobblemon 싱글 배틀**(트레이너 vs 관장). 인간 PvP 아님.
- **레벨캡**: 도전 시 파티 최고 레벨 ≤ 해당 짐 캡. 적용 방식 TBD(소프트 경고 vs 하드 차단 vs 자동 스케일).
- 파티 수: 6 이하(기본). 회복 아이템 사용 허용 여부 TBD.
- **순차 강제(확정).** 이전 짐을 클리어해야 다음 짐 도전 가능(`requirePrevious=true`). 레벨 곡선 유지 목적. 1번부터 순서대로.
- 재시도: 패배 시 즉시 재도전 가능(쿨다운 없음, 보상 없으므로 — §6).

### 4-1. 동시 도전 / 인스턴스 (확정: 방식 A)
- **플레이어별 독립 인스턴스 배틀.** 여러 명이 같은 관장을 **동시에 각자** 도전 가능 — Cobblemon 트레이너 배틀이 도전자별 개별 세션이라 충돌 없음.
- 점유 락·대기열 **불필요**. (전설 제단의 1인 전용 룸과 다름 — 관장은 반복 가능한 배틀이고 보상이 단일 개체가 아니라 기록이므로 횡취 문제 없음.)
- 단, 진행 게이트(순차 강제)는 **플레이어별 진행 상태**로 각자 판정.

## 5. 클리어 조건 / 기록 (PoroMonCore)
- 클리어 판정: 관장 배틀 승리 → `GymBadgeManager`가 수신.
- 저장: `PlayerProgress.badges[]`에 배지 id 추가, `stats.gymCleared++`.
- 멱등성: 이미 보유한 배지는 중복 지급 안 함(최초 1회 보상). 재도전 보상은 별도(§6).
- 로깅: 클리어/보상 지급 = `AuditLog`.
- 검증: 서버 권위(클라 신뢰 금지). 배틀 결과는 Cobblemon 이벤트 기반.

## 6. 보상 — **초회 클리어 보상만 (확정)**

모든 짐 보상은 **최초 클리어 1회만** 지급한다. 재도전은 보상 없음(연습/재미용).

| 보상 | 최초 클리어 | 재도전 |
|---|---|---|
| 배지 | ✅ (영구) | ❌ |
| 골드 | ✅ | ❌ |
| 메가/테라 진행 | 일부 짐에서 해금 조건 충족에 기여 | ❌ |
| 티켓 구매권 | 특정 배지 수 도달 시 개방 | ❌ |
| 리그 참가 자격 | 8배지 시 개방 | ❌ |

- **재도전: 보상 전혀 없음, 쿨다운도 불필요**(어차피 보상이 없으므로 자유 재도전). 순수 연습.
- 멱등성: `GymBadgeManager`가 배지 보유 여부로 초회 판정 → 이미 보유 시 보상 스킵.
- 보상량은 `gyms.json`·`rewards.json`(향후)로 설정. → `RewardManager`.
