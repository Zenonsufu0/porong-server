## 1.5 몹 → 플레이어 ATK 정본 (DL-116, 2026-06-01)

§1 HP 100 스케일을 기준으로 몹/보스가 플레이어에게 가하는 ATK를 정본화. 상세·도출·적용 매핑은 **`../06_fields_bosses/mob_attack_stats_v1.md`**(field_boss_stats의 공격 측 대칭 문서).

**핵심 룰:** 몹 ATK는 티어별로 **추천 강화 풀세트 최대 HP의 고정 비율(유효 피해)**로 정의한다. 설계 철학: **필드 콘텐츠(몹+필드보스)는 가볍게**(포션 없이 버팀 — 회복약은 시즌보스용), **시즌보스는 무겁게**(회피·포션 관리 도전). 같은 P-코드라도 필드보스와 시즌보스는 배율을 분리한다.

| 티어 | 유효 피해(추천 HP 대비) | 명목 피격 횟수 |
|---|---:|---:|
| 일반몹 평타 | 2.5% | ~40타 |
| 정예몹 평타 | 6% | ~17타 |
| 필드보스 기본공격 | 8% | ~12타 |
| 필드보스 일반 패턴 | 16% | ~6타 |
| 필드보스 강 패턴 | 24% | ~4타 |
| 시즌보스 기본공격 | 12% | ~8타 |
| 시즌보스 일반 패턴 | 24% | ~4타 |
| 시즌보스 강 패턴 | 40% | ~2.5타 |

```
몹 ATK(raw) = (티어 비율 × 추천 강화 HP) / 관통율(= 200/(DEF+200))
```

`raw`는 DEF 경감 전 값 = MythicMobs `Damage:` 속성 / 스킬 `damage{a}`에 그대로 입력. `PlayerDefenseListener`가 받는 쪽에서 `×(1−mit)` 적용.

필드보스 ATK 요약(기본공격/일반/강): F1 12/24/37 · F2 14/29/43 · F3 16/32/48 · F4 17/34/50 · F5 18/37/55. 일반몹 4~6, 정예몹 9~14. 시즌보스(무겁게, 12/24/40%)는 기본 22~30 / 일반 43~61 / 강 72~101, 잠정값(상세 문서 §5).

> 현행 MythicMobs 설정의 내부 모순(바닐라 평타 35~120 과대 / 스킬 패턴 8~22 과소)을 이 룰로 통일. 적용은 별도 배포 단계.

## 2. 피해 계산 공식 (2026-05-16 확장)

### 피해 증폭 풀 (가산 통합)

스킬피해%, 태그피해%, 자원스택피해%는 **동일 가산 풀**로 합산.

```
보너스 합계 = 스킬피해% + 태그피해% + 자원스택피해%
```

| 옵션 | 설명 | 적용 조건 |
|---|---|---|
| 스킬피해% | 잠재능력 / 특화 스탯 트리 | 모든 스킬 |
| 태그피해% | 잠재능력 / 보조스탯 (핵심기/이동기/기본기/특수기) | 해당 입력 타입 스킬만 |
| 자원스택피해% | 무기 고유 자원 패시브 | 무기별 조건 |

태그 분류:
- LMB → 기본기 피해%
- RMB → 이동기 피해%
- Shift+RMB → 특수기 피해%
- F → 핵심기 피해%

### 최종 피해 공식

```
최종 피해 = ATK
          × 스킬계수
          × (1 + 스킬피해% + 태그피해% + 자원스택피해%)
          × (1 − DEF방어율)
          × 크리배율

DEF방어율 = DEF / (DEF + 200)
크리배율  = 발생 시 치명피해배율 (기본 1.50 + 치명피해% 증가분)
기대 크리 = 1 + 치명확률 × (치명피해배율 − 1)
```

피격자 기준 실제 피해 (§7 확정):
```
실제 피해 = 최종 피해 × (1 − DEF방어율) × (1 − 받는피해감소%)
```
- DEF방어율과 받는피해감소%는 **독립 곱산** (level_stat_system_v1.md §7 참조)

T1 풀세트 DEF 52 기준 유효 방어율 약 20.6%. DEF 200 기준 50.0%.

## 3. 스킬 쿨타임 표 (v2 신규 결정)

### 설계 기준

- 전투 루프: 자원 생성(좌클릭) → 축적 → 소모(특수기/이동기) → 핵심기 발동이 한 사이클로 완결
- 목표 사이클: 20~35초 (무기별 자원 최대 단계 수에 비례)
- 좌클릭(기본기): 자원 생성 주축. 리듬감이 전투 체감 핵심 → 0.5~1.2초
- 우클릭(이동기): 이동/회피. 남용 방지를 위한 중간 쿨타임 → 4~8초
- Shift+우클릭(특수기): 강한 제어 또는 자기 강화. 긴 쿨타임 → 8~15초
- F키(핵심기): 자원 전체 소모 + 최고 피해. 가장 긴 쿨타임 → 12~20초

자원 최대 단계가 많은 무기(창 5단계, 스태프 5단계)는 핵심기 쿨타임이 길어도
충전 사이클이 이미 25초 이상이므로 실전 체감 순환에 영향이 적다.
자원 최대 단계가 적은 무기(석궁 ON/OFF, 도끼 3단계)는 사이클이 짧으므로
핵심기/특수기 쿨타임을 통해 사이클 길이를 조정한다.

### 쿨타임 표

> v2.1 변경: 좌클릭(기본기) 쿨타임 연장 — 슬롯1 스킬과 기본 공격(타격 속도)을 분리.

### 무기별 쿨타임 설계 근거

**검 (검세 3단계)**
- 좌클릭 3회 명중으로 검세 만충. 0.8초 기준 약 2.4초 충전 완료.
- 이동기는 포지셔닝용 → 6초로 남용 방지.
- 핵심기 16초: 만충 후 즉시 발동할 경우 총 사이클 약 18~20초가 되므로
  이동기/특수기 사용 여부에 따라 22~28초로 늘어남.

**도끼 (충격 3단계)**
- 좌클릭 명중 1단계, 특수기 명중 2단계. 최소 2회 행동으로 만충 가능.
- 특수기(제어기) 쿨타임 12초: 기절 제어 남용 방지.
- 핵심기 18초: 충격 만충 + 소모 루프가 한 번 발생하면 18초 뒤 재발동 가능.
  보스전 기절 → 딜타임 구조가 일정한 간격으로 반복되도록 설계.

**창 (압박 5단계)**
- 좌클릭 5회 명중 필요. 0.7초 기준 3.5초 충전. 이동기 사용 시 1단계 소모.
- 이동기 쿨타임 4초: 짧은 이동기로 전진/후퇴를 자주 활용하는 플레이 스타일 유지.
- F키 15초: 5단계 충전 후 발동. 충전 시간 + 쿨타임 합계 약 18~20초.
- 전체 사이클: 좌클릭 반복 → 압박 축적 → 이동기 → 핵심기 패턴이 25~32초.

**석궁 (조준 ON/OFF)**
- 좌클릭 쿨타임 0.6초로 6무기 중 가장 빠름.
  발사 간격이 전투 속도감 핵심.
- 이동기 후 조준 ON 상태 연계가 중요 → 이동기 5초, 특수기 8초.
- 핵심기 14초: 조준 ON 상태에서 발동하면 최대 효과.
  조준 ON 여부에 따라 핵심기 피해 편차가 있으므로 쿨타임은 중간 수준으로 설정.

**낫 (그림자 흐름 3단계)**
- W-2 조정 반영: 이동기(회피) 사용 후 2초 내 명중 시 흐름 1단계 충전.
  이동기 쿨타임 5초 → 이동기 1회 후 공격 → 이동기 쿨타임 해제까지 약 5초.
  3단계 충전에 이동기 3회 사용이 필요하므로 실질 충전 시간 약 15초.
- 좌클릭 0.8초로 명중 타이밍이 충전 창(2초)에 여유 있게 들어옴.
- 핵심기 16초: 만충 폭발 후 재충전 사이클과 겹치지 않도록 설정.
  암영무희 각인 시 회피 쿨타임 -20% 적용 → 이동기 4초로 체감 단축.

**스태프 (마력 충전 5단계)**
- 좌클릭 5회 명중 필요. 0.9초 기준 4.5초 충전.
- 핵심기 20초: 6무기 중 가장 길다.
  이유: 5단계 만충 시 +45% 피해 + 범위 확장으로 최고 단일 폭발력을 가짐.
  20초 쿨타임이 밸런스 축. 충전 4.5초 + 쿨타임 20초 = 총 사이클 24.5초.
  이동기/특수기 사용 포함 시 28~35초.
- 비전술사 각인 시 5단계 유지 중 +20%가 지속이므로 핵심기를 늦게 쓰는 선택지도 생김.

## 4. 고유 자원 충전/소모 조건 최종안 (W-2~W-4 반영)

## 8. 기여도 산정 방식 결정안 (T-9)

### 결정: 옵션 C 채택

딜 기여도 70% + 참여 보너스 30%

### 옵션 비교

| 옵션 | 내용 | 장점 | 단점 |
|---|---|---|---|
| A | 딜 기여도 100% | 구현 단순 | 저강화 유저 보상 없음, 참여 유인 없음 |
| B | 딜+힐+버프 복합 | 역할 다양성 반영 | 1차 힐러 무기 없음, 버프 수혜량 계산 복잡, 버그/불공정 논란 |
| C | 딜 70% + 참여 30% | 저강화 참여 유도, 구현 용이 | 딜 기여 극소 유저도 참여 보너스 수령 |

### 옵션 C 상세 구조

**딜 기여도 (70%)**
- 보스 최대 HP 대비 해당 플레이어가 가한 유효 피해 비율로 산정
- 유효 피해 = 실제 피해량(오버킬 제외)
- 공식: 개인 유효 피해 / 보스 최대 HP

예시:
```
보스 HP 100,000
플레이어 A 유효 피해 40,000 → 딜 기여율 40%
플레이어 B 유효 피해 30,000 → 딜 기여율 30%
플레이어 C 유효 피해 30,000 → 딜 기여율 30%
```

딜 기여 보상 = 기준 보상 × 0.70 × 개인 딜 기여율

**참여 보너스 (30%)**
- 보스 처치 시 해당 인스턴스에 참여한 모든 플레이어가 균등 분배
- 최소 참여 기준: 보스 전투 시작 후 입장하여 최소 1회 이상 피해를 가한 플레이어
- 공식: 기준 보상 × 0.30 / 참여 인원 수

예시 (파티 3인, 기준 보상 300G):
```
딜 기여 보상 풀: 300G × 0.70 = 210G
A: 210G × 0.40 = 84G
B: 210G × 0.30 = 63G
C: 210G × 0.30 = 63G

참여 보너스 풀: 300G × 0.30 = 90G
A: 90G / 3 = 30G
B: 90G / 3 = 30G
C: 90G / 3 = 30G

최종:
A = 84G + 30G = 114G
B = 63G + 30G = 93G
C = 63G + 30G = 93G
```

**보상 종류별 적용 범위**

| 보상 종류 | 기여도 적용 |
|---|---|
| 골드 보상 | 옵션 C 적용 (딜 70% + 참여 30%) |
| 강화석/재료 | 딜 기여도 상위 N인 기준 드랍 (별도 확정 필요) |
| 필드보스 고유 재료 | 딜 기여도 상위 3인 (필드보스) / 전원 (시즌보스) |
| 치장/희귀 보상 | 딜 기여도 상위 1인 (필드보스) / 파티원 랜덤 (시즌보스) |

필드보스는 다수 유저가 참여하므로 골드도 딜 기여도 100%로 적용하는 것을 별도 검토 권장.
시즌보스는 파티 인원이 제한적이므로 옵션 C가 적합.

### 힐러/버프 역할 부재 보완

1차 시즌에 힐러 전용 무기가 없고 버프는 각인 일부에 국한(성벽수호자 DEF, 암영무희 이속)됨.
버프 수혜량을 기여도에 반영하면 측정 복잡도 대비 실효 차이가 미미하다.
2차 시즌 이후 힐러/서포터 역할이 추가될 경우 옵션 B로 전환을 고려한다.
