## 확정 사항

| 항목 | 내용 |
|---|---|
| 소스 | 흔적 (이전할 장비 — 소모됨) |
| 대상 | 전승받을 장비 (인벤토리에서 드래그) |
| 전승권 | 3종 중 하나 슬롯에 배치 (없으면 기본 전승) |

| 전승 종류 | 이전 내용 | 비용 |
|---|---|---|
| 기본 전승 | 등급 + 서브스탯 옵션 전체 이전 | **0G (무료)** |
| 등급전승권 | 등급(커먼~레전더리)만 이전 | **100,000G** (상점 구매) |
| 세부스탯전승권 | 서브스탯 옵션만 이전 | **100,000G** (상점 구매) |

> 전승권은 상점에서만 구매 가능. 경매소 거래 불가. (DL-015)

---

## 흐름

```
오른쪽 col8 장비 클릭 → [대상] 슬롯에 자동 배치
[흔적] 슬롯에 인벤토리 장비 드래그 (소모됨)
[전승권 슬롯]에 3종 중 하나 배치 (기본 전승은 전승권 불필요)
    │
    ▼
미리보기 자동 갱신 (흔적 현재 옵션 / 전승 후 대상 옵션)
    │
    │  [전승!] 클릭
    ▼
흔적 장비 소멸 + 대상 장비에 옵션 이전
```

## 슬롯 배치 (0-indexed, 45슬롯)

```
     col0  col1  col2  col3  col4  col5  col6  col7  col8
row0 [유리][유리][유리][유리][유리][유리][유리][유리][무기]
row1 [유리][유리][흔적][유리][ → ][유리][대상][유리][투구]
row2 [유리][유리][유리][유리][전승권][유리][유리][유리][상의]
row3 [유리][유리][전승전][유리][유리][유리][전승후][유리][하의]
row4 [뒤로][유리][유리][유리][전승!][유리][유리][유리][신발]
```

| 슬롯 | 역할 | 타입 |

| 29 | 흔적 장비 현재 옵션 미리보기 | 읽기 전용 |
| 33 | 대상 장비 전승 후 옵션 미리보기 | 읽기 전용 |
| 40 | [전승!] 버튼 | 클릭 |
| 8 | 무기 선택 → 대상 슬롯으로 | 클릭 |

| 26 | 상의 선택 → 대상 슬롯으로 | 클릭 |
