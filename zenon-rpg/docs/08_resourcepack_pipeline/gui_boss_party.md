# 보스 파티 GUI 설계

> **[STATUS: REFERENCE]** — 파티 생성·설정·목록·방 내부 GUI 상세 참조. 공식 리소스팩 기준은 `index.md`가 우선.

> 작성일: 2026-05-23
> 접근 조건: 비전투 (보스 서브 허브에서 진입)
> 대상 보스: 시즌보스 1~6 + 시즌최종보스 3종 (총 9종)

---

## 흐름 요약

```
[파티생성] 클릭 (보스 서브 허브 슬롯 9~12, 18~21)
  → §1 Anvil: 방 이름 입력
  → §2 보스 방 설정 GUI: 보스선택·최소강화·최대인원 설정 → 개설
  ↘
   §4 보스 방 내부 GUI  ←──────────────────────────────┐
  ↗                                                      │
[파티목록] 클릭 (보스 서브 허브 슬롯 14~16, 23~25)      │
  → §3 보스 방 목록 GUI: 열린 방 목록 → 입장 클릭 ─────┘
```

## 1. 방 만들기 (Anvil GUI)

입력칸 기본값: `파티 이름`

**좌측 슬롯 아이템**: `BEACON`

```
§e파티 이름 입력
§7──────────
§7방 이름을 입력 후
§7확인을 눌러주세요.
§7──────────
§7최대 16자
```

출력 슬롯 클릭 → 방 이름 저장 → 보스 방 설정 GUI 열림.

### ESC 이탈 처리

방 내부 GUI에서 ESC(GUI 닫기)를 누르면 **즉시 탈퇴하지 않고** 채팅으로 확인을 요청한다.

**채팅 메시지**:
```
§e[보스파티] §f방에서 나왔습니다. 계속 탈퇴하시겠습니까?
§7  §a[복귀]  §7|  §c[탈퇴]
§8  30초 내 응답 없으면 자동 탈퇴됩니다.
```

| 선택 | 동작 |
|---|---|
| `[복귀]` 클릭 | 방 내부 GUI 재오픈, 타이머 취소 |
| `[탈퇴]` 클릭 | 즉시 파티 탈퇴 처리 |
| 30초 무응답 | 자동 탈퇴 처리 |

> 탈퇴 처리 내용: `boss_room_members` 행 삭제 + 방장에게 `§7[보스파티] §f[닉네임]§7이 퇴장했습니다.` 채팅 알림.  
> 방장이 ESC로 이탈하면 탈퇴가 아닌 **방 해산** 처리 (방장 부재 시 방 유지 불가).

## 5. 구현 참조

| 항목 | 내용 |
|---|---|
| 파티생성 트리거 | 보스 서브 허브 슬롯 9~12, 18~21 클릭 |
| 파티목록 트리거 | 보스 서브 허브 슬롯 14~16, 23~25 클릭 |
| 방 이름 입력 | `AnvilInventory#getRenameText()` — 출력 슬롯 클릭 시 저장 |
| 최소강화 범위 | 0 ~ 22강 (1강 단위) |
| 최대인원 기본값 | 3인 |
| 방 목록 | DB 테이블 `boss_rooms` (room_id, title, boss_type, min_enh, max_players, current_players, owner_uuid, status, created_at) |
| 멤버 정보 | DB 테이블 `boss_room_members` (room_id, player_uuid, is_ready, joined_at) |
| 강퇴 처리 | 방장 우클릭 → `boss_room_members` 행 삭제 + 해당 플레이어 GUI 닫기 |
| 방 해산 | 방장 해산 클릭 → 방 status='closed' + 멤버 전원 GUI 닫기 |
| 입장 처리 | 방장 입장 클릭 → 멤버 전원 `world_boss` 워프 + 보스 스폰 트리거 |
| 새로고침 | 슬롯 53 클릭 → GUI 재오픈 (목록 재조회) |
| ESC 이탈 감지 | `InventoryCloseEvent` — 방 내부 GUI 닫힐 때 30초 타이머 스케줄 등록 |
| 복귀/탈퇴 채팅 | Adventure API `ClickEvent.RUN_COMMAND` — `/rpgboss rejoin` / `/rpgboss leave` |
| 30초 타이머 | `BukkitRunnable` — 복귀·탈퇴 클릭 시 `cancel()`, 만료 시 자동 탈퇴 실행 |
| 방장 ESC 이탈 | 30초 타이머 후 자동 탈퇴 대신 방 해산 처리 |
