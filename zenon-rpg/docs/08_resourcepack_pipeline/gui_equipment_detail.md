## 1. 개별 세부 GUI (장비 슬롯 클릭 시)

### 방어구 (투구·상의·하의·신발)

```
     col0  col1  col2  col3  col4  col5  col6  col7  col8
row0 [유리] [유리] [유리] [유리] [현재] [유리] [유리] [유리] [유리]
row1 [유리] [가죽] [사슬] [구리] [철  ] [금  ] [다이아][네더] [유리]
row2 [뒤로🔴][유리] [유리] [유리] [유리] [유리] [숨김] [유리] [보이기]
```

| 슬롯 | 역할 |
|---|---|
| 4 | 현재 선택 외형 표시 — 선택된 재질 `LIME_STAINED_GLASS_PANE` 하이라이트 |
| 10 | 가죽 | 11 | 사슬 | 12 | 구리 |
| 13 | 철 | 14 | 금 | 15 | 다이아 | 16 | 네더라이트 |
| 24 | 숨김 — **미구현** (치장템 시스템 2차 확장, 항상 회색 비활성) |
| 26 | 보이기 — **미구현** (치장템 시스템 2차 확장, 항상 회색 비활성) |
| 18 | 뒤로 (`ARROW`) |

### 검·도끼·낫 (바닐라 재질 6종)

```
row0 [유리] [유리] [유리] [유리] [현재] [유리] [유리] [유리] [무기변경]
row1 [유리] [나무] [돌  ] [철  ] [금  ] [다이아][네더] [유리] [유리]
row2 [뒤로🔴][유리] [유리] [유리] [유리] [유리] [숨김*] [유리] [보이기*]
```

| 상태 | 숨김(슬롯 24) | 보이기(슬롯 26) |
|---|---|---|
| 1차 시즌 현재 | `GRAY_STAINED_GLASS_PANE` (항상 비활성) | `GRAY_STAINED_GLASS_PANE` (항상 비활성) |
| 2차 확장 예정 | 치장템 있음 시 활성 | 치장템 있음 시 활성 |

> **미구현**: 치장템(ModelEngine/커스텀 무기) 시스템이 2차 확장 예정이므로 1차 시즌에는 항상 비활성.

| 슬롯 | 역할 |
|---|---|
| 8 | 무기 변경 (`IRON_SWORD`) — 클릭 → 무기 목록 GUI (`§2`) |

## 2. 무기 변경 GUI (45슬롯)

> 진입: 무기 세부 GUI 슬롯 8 클릭  
> 무기 6종(검·도끼·낫·석궁·창·스태프) 중 하나를 선택. 인벤에는 항상 1종만 존재.

```
     col0  col1  col2  col3  col4  col5  col6  col7  col8
row0 [유리] [유리] [유리] [유리] [유리] [유리] [유리] [유리] [유리]
row1 [유리] [ 검 ] [유리] [유리] [도끼] [유리] [유리] [ 낫 ] [유리]
row2 [유리] [유리] [유리] [유리] [유리] [유리] [유리] [유리] [유리]
row3 [유리] [석궁] [유리] [유리] [ 창 ] [유리] [유리] [스태프][유리]
row4 [뒤로🔴][유리] [유리] [유리] [유리] [유리] [유리] [유리] [유리]
```

| 슬롯 | 역할 |
|---|---|
| 10 | 검 | 13 | 도끼 | 16 | 낫 |
| 28 | 석궁 | 31 | 창 | 34 | 스태프 |
| 36 | 뒤로 (`ARROW`) |

> 현재 장착 중인 무기 슬롯: `LIME_STAINED_GLASS_PANE` 테두리 하이라이트.

**무기 슬롯 lore 예시 (비장착)**

```
§e검
§7──────────────────
§7 강화   : §e+15강
§7 등급   : §d레전더리
§7 잠재   : §a치명타 확률 +25% / 스킬 피해 +15% / 핵심기 +20%
§7 각인   : §a집중 강타 A
§7 재질   : §7다이아몬드
§7 스킬   : §7[LC]회전베기 [RC]돌진 [SRC]폭풍 [F]처형
§7──────────────────
§7 [클릭] 이 무기로 교체
§7 (슬롯 0 무기 아이템 교체, 별도 회수 없음)
```

**장착 중 lore 예시**

```
§e검  §a[장착 중]
§7──────────────────
§7 강화   : §e+15강
§7 등급   : §d레전더리
§7 잠재   : §a치명타 확률 +25% / 스킬 피해 +15% / 핵심기 +20%
§7 각인   : §a집중 강타 A
§7 재질   : §7다이아몬드
§7 스킬   : §7[LC]회전베기 [RC]돌진 [SRC]폭풍 [F]처형
```

### 무기 변경 규칙

| 항목 | 내용 |
|---|---|
| 강화·등급·잠재·각인·세부스탯 | **단일 공유 state** — 모든 무기 종류가 같은 PlayerGrowthState 참조. 교체해도 데이터 유지(단, 종류별 독립 저장 아님). GUI에 "(공유)" 표기. |
| 재질 선택 | **무기 종류별 기억** — `weapon_{TYPE}` 키로 PlayerGrowthState에 저장, 복귀 시 이전 선택 재질 복원 |
| 스킬셋 | **무기 종류에 따라 자동 전환** |
| 교체 처리 | 슬롯 0의 무기 아이템을 교체(PDC 태그·display name 포함) — 별도 회수 없음 |

## 3. 재질 일괄선택 GUI (캐릭터 GUI 슬롯 16) — ⚠️ 미구현 (2차 확장 예정)

> **현재 상태**: 캐릭터 GUI 슬롯 16에 버튼은 노출되나 클릭 핸들러 없음. 이 섹션은 설계 명세만 존재하며 1차 시즌에 미구현입니다.

선택한 재질을 모든 장비에 한 번에 적용. **사슬 제외** (총 6종).

```
     col0  col1  col2  col3  col4  col5  col6  col7  col8
row0 [유리] [유리] [유리] [유리] [안내] [유리] [유리] [유리] [유리]
row1 [유리] [가죽] [구리] [철  ] [금  ] [다이아][네더] [유리] [유리]
row2 [뒤로🔴][유리] [유리] [유리] [유리] [유리] [유리] [유리] [유리]
```

| 슬롯 | 역할 |
|---|---|
| 4 | 안내 아이콘 — 무기 타입에 따라 lore 변경 |
| 10 | 가죽 | 11 | 구리 | 12 | 철 |
| 13 | 금 | 14 | 다이아 | 15 | 네더라이트 |
| 18 | 뒤로 (`ARROW`) |

### 적용 범위

| 무기 타입 | 방어구 재질 적용 | 무기 재질 적용 |
|---|---|---|
| 검·도끼·낫 | ✅ | ✅ |
| 창·스태프·석궁 | ✅ | ❌ |

### 안내 슬롯(4) lore 예시

```
§e재질 일괄 선택
§7──────────────────
§7 선택한 재질로 방어구 + 무기 외형을 한 번에 변경합니다.
§7 (창·스태프·석궁은 방어구만 적용)
§7──────────────────
§7 현재 무기: §e{무기타입}
```
