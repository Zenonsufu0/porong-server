## 2. lore 포맷

### 잔여 포인트 (슬롯 4)

아이템: `EXPERIENCE_BOTTLE`

```
§e잔여 스탯 포인트
§7──────────────────
§7 잔여  : §e{REMAINING}pt
§7 사용  : §7{USED}pt
§7 총계  : §7{TOTAL}pt  (레벨 {LV} × 3pt)
§7──────────────────
§7 비전투 중에만 재배분 가능.
```

### 트리 포인트 현황 (슬롯 13 / 22 / 31)

아이템: `PAPER`

```
§e{트리명} 트리
§7──────────────────
§7 투자 포인트: §e{PT}pt
§7──────────────────
§7 §a주 효과 §7: {주 효과명} §e+{주값}
§7 §7부 효과 §7: {부 효과명} §7+{부값}
```

### 효과값 슬롯 (슬롯 16·17·25·26·34·35)

아이템: `BOOK`

```
§7{효과명}: §e{현재값}
§7 (스탯: +{스탯기여} / 잠재+서브스탯: +{외부기여})
```

### -10 / -1 버튼 (비활성 조건)

- 해당 트리 포인트가 10 미만이면 -10은 `GRAY_STAINED_GLASS_PANE`(비활성)
- 해당 트리 포인트가 0이면 -1도 `GRAY_STAINED_GLASS_PANE`(비활성)
- 전투 중이면 모든 버튼 `GRAY_STAINED_GLASS_PANE`(비활성)

### +10 / +1 버튼 (비활성 조건)

- 잔여 포인트 < 10이면 +10은 `GRAY_STAINED_GLASS_PANE`(비활성)
- 잔여 포인트 = 0이면 +1도 `GRAY_STAINED_GLASS_PANE`(비활성)
- 전투 중이면 모든 버튼 `GRAY_STAINED_GLASS_PANE`(비활성)

### 전체 초기화 (슬롯 50)

아이템: `BARRIER`

```
§c전체 초기화
§7──────────────────
§7 투자한 모든 포인트를 초기화합니다.
§7 클릭 후 확인 클릭으로 적용됩니다.
§c 전투 중에는 사용할 수 없습니다.
```

> 초기화 클릭 시 슬롯 50이 `FIRE_CHARGE`(확인)로 교체 — 재클릭 시 실행, 다른 슬롯 클릭 시 취소.

## 4. 재배분 규칙

- **비전투 중**에만 포인트 변경 가능 (전투 중 모든 버튼 비활성)
- 버튼 클릭 즉시 반영 (별도 확인 없음 — 초기화 제외)
- GUI를 닫으면 변경사항 유지 (임시 상태 없음)
