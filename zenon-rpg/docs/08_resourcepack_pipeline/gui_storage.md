## 1. 창고 개요

- DB 기반 가상 저장고 — 실제 Minecraft 아이템이 아닌 DB 수치로 관리

### 입금 방식

| 방식 | 대상 아이템 | 조건 |
|---|---|---|
| **자동 입금** | 커스텀 생산기 생산물 (약초 재배지·광물 채굴기) | 항상 자동 |
| **자동 입금** | 채굴·수확 산출물 | 영주가 상점 5만G 구매 + 영지설정 자동입금 ON |
| **직접 입금** | 모든 등록 아이템 | 인벤 아이템 들고 창고 슬롯 클릭 |

### 창고 연동 시스템

창고에 보관된 아이템은 아래 시스템에서 직접 사용 가능 — 별도 출금 불필요:

| 시스템 | 사용 방식 |
|---|---|
| **공방 가공기 대기열** | 레시피 등록 시 창고에서 재료 자동 차감 |
| **상점 판매** | 상점 GUI에서 판매 등록 시 창고 수량 직접 사용 |
| **경매장 등록** | 경매 등록 시 창고 수량 직접 차감 |

## 2. 슬롯 레이아웃 (54슬롯)

```
행\열   1     2     3     4     5     6     7     8     9
  1   [아이템 슬롯 × 9]
  2   [아이템 슬롯 × 9]
  3   [아이템 슬롯 × 9]
  4   [아이템 슬롯 × 9]
  5   [아이템 슬롯 × 9]
  6   [뒤로]  ░     ░    [◀]  [페이지]  [▶]   ░     ░     ░
```

| 슬롯 | 역할 |
|---|---|
| 0~44 | 아이템 표시 (페이지당 최대 45종) |
| 45 | 뒤로 (영지 서브 허브) |
| 48 | 이전 페이지 |
| 49 | 현재 페이지 표시 |
| 50 | 다음 페이지 |
| 46~47, 51~53 | GRAY_STAINED_GLASS_PANE |

빈 아이템 슬롯(0~44): GRAY_STAINED_GLASS_PANE.

## 3. 아이템 슬롯 lore

아이템 종류별 1슬롯. 수량은 lore에 표시.

```
§f[아이템 이름]
§7보유량: §f1,234개
§7──────────
§8공방·상점·경매장에서 직접 사용 가능
§7──────────
§7좌클릭        §f1개 출금
§7우클릭        §f전량 출금 §8(최대 500개)
§7──────────
§7아이템 들고 좌클릭  §f1개 입금
§7아이템 들고 우클릭  §f전량 입금
```

수량 0인 아이템은 표시하지 않음 (슬롯 비움).

## 4. 버튼 lore

### 이전 / 다음 페이지 (슬롯 48 / 50)

아이템: `ARROW`

```
§f◀ 이전 페이지      §f다음 페이지 ▶
§71페이지 / 3페이지   §71페이지 / 3페이지
```

1페이지일 때 이전 버튼, 마지막 페이지일 때 다음 버튼 = GRAY_STAINED_GLASS_PANE으로 교체 (비활성).

### 현재 페이지 표시 (슬롯 49)

아이템: `PAPER`

```
§f1 / 3 페이지
§7총 §f87§7종 보관 중
```

### 뒤로 (슬롯 45)

아이템: `ARROW`

```
§7◀ 뒤로
§7영지 메뉴
```

## 5. 출금 규칙

| 클릭 | 출금량 | 비고 |
|---|---|---|
| 좌클릭 | 1개 | |
| 우클릭 | 500개 | 보유량 < 500이면 전량 |

출금량 결정 우선순위:
1. 요청량(1 또는 500) vs 보유량 → 더 작은 값
2. 위 값 vs 인벤토리 여유 공간 → 더 작은 값으로 실제 지급

인벤토리 여유 슬롯이 0이면:
```
§c인벤토리가 가득 찼습니다.
```
채팅으로 알림 후 출금 취소.

## 6. 입금 규칙

### 직접 입금 (커서 아이템 클릭)
- 인벤토리에서 아이템을 집어(커서에 올린) 상태로 창고 슬롯 클릭
  - 좌클릭: 1개 입금
  - 우클릭: 인벤 보유 전량 입금
- 핫바 포함 (커서에 올린 아이템 기준)
- 입금 완료 메시지:
```
§a창고에 입금했습니다: §f철광석 ×1
```

### 자동 입금 (생산기)
- GUI 없이 DB에 직접 적재
- 플레이어 오프라인 상태에서도 동작

---

## 7. 아이템 정렬 순서

페이지 내 아이템 정렬:

1. 광물류 (철, 금, 다이아, 에메랄드 등)
2. 농작물 (밀, 감자, 당근 등)
3. 목재류
4. 커스텀 소재 (영지 생산 전용)
5. 기타

같은 카테고리 내: 수량 많은 순 정렬.
