## 필드보스

필드 리젠형. 매칭으로 빼지 않는다.

| 항목 | 설계값 |
|---|---|
| 스폰 주기 | 30분 고정 |
| 스폰 조건 | 필드 유저 유무 관계없이 생성 |
| 자동 소멸 | 스폰 후 5분 이내 미처치 시 소멸 |
| 월드챗 알람 | 소환 5분 / 3분 / 1분 전 전서버 공지 |

보상 구조: 기여도 충족 여부로만 구분 (이진 판정).
- 충족 시: 기본 보상 (확정, 동일 지급) + 희귀 보상 (개인 독립 확률, 미지급 가능)
- 미충족 시: 보상 없음
- 상위 필드 보스일수록 기본 보상 수량 및 희귀 보상 풀 상향

기본 기여도 기준: **총 피해량 3% 이상** (확정 2026-05-14). (`drop_tables_v1.md §3` 상세)

## 필드보스 원샷 방지 시스템 (확정 2026-05-14)

고IL 유저가 필드보스를 단독 처치해 다른 유저들의 기여도가 0이 되는 상황을 방지하기 위한 피해 제한 메커니즘.

### 동작 방식

```
단일 타격 피해량 ≥ 필드보스 최대체력 × 85%
  → 해당 타격을 최대체력 × 85%로 클램프 (초과분 무효)
  → 이후 해당 플레이어의 모든 공격 피해 = 0 (해당 전투 종료까지)
```

| 항목 | 내용 |
|---|---|
| 트리거 조건 | 단일 타격으로 필드보스 최대체력의 **85% 이상** 피해 |
| 트리거 시 효과 | 해당 타격을 85% HP로 클램프 |
| 이후 처리 | 트리거한 플레이어의 후속 공격 전부 **0 피해** |
| 지속 범위 | 해당 필드보스 전투 종료 (처치 또는 소멸) 시까지 |
| 기여도 판정 | 클램프된 85%는 기여도 산정에 정상 반영 (보상 자격 유지) |

### 설계 의도

- 고IL 유저도 **첫 타격 기여**로 기본 보상 수령 가능 (기여도 ≥ 3% 충족)
- 보스는 항상 **최소 15% HP** 이상을 유지한 채 다른 플레이어에게 딜 기회 전달
- 원샷 방지 + 기여도 분산을 하나의 메커니즘으로 해결

### 구현 시 고려사항

- 0 피해 상태임을 해당 플레이어에게 인게임 메시지로 안내 필요 (예: "이 전투에서의 공격이 제한되었습니다")
- 범위기(AoE) 스킬의 경우 해당 보스에 한해 0 피해 적용 (다른 몹은 정상 적용)

## 아이템 레벨 경계 시스템 (확정 2026-05-14)

각 필드에 "적정 아이템 레벨(강화 단계)" 범위를 설정한다. 범위를 크게 벗어난 유저(주로 고강화 유저의 저필드 보스 독점 방지)가 필드보스 사냥에 참여하면 경고를 발송하고, 반복 위반 시 임시 화이트리스트에서 제외한다.

### 처리 흐름

```
필드보스 처치 기여 감지
  → 참여 유저 아이템 레벨 확인 (무기 강화 단계 기준)
    → 해당 필드 적정 범위 초과 여부 판정
      → 초과 시: 인게임 경고 + 디스코드 운영자 알람
        → 동일 유저 경고 3회 누적 → 임시 화이트리스트 제외
```

### 아이템 레벨 공식 (확정)

- **강화 단계 1당 IL +5** / 평균 IL = 5슬롯 평균 강화 단계 × 5
- 최솟값 IL 0 (0강) / 최댓값 IL 125 (25강 전 슬롯)

### 필드별 적정 평균 IL 및 경고 임계값 (확정)

| 필드 | 적정 평균 IL | 경고 발동 | 해당 강화 기준 |
|---|---|---|---|
| 수도 외곽 평원 | IL 0~35 | **IL 65 이상** | 평균 13강+ |
| 폐광 지대 | IL 30~55 | **IL 75 이상** | 평균 15강+ |
| 오염된 수로 | IL 50~75 | **IL 85 이상** | 평균 17강+ |
| 무너진 초소 | IL 65~85 | **IL 90 이상** | 평균 18강+ |
| 고대 성벽 잔해 | IL 80~100 | 없음 | 최고 필드 |

- 경고 초기화: **누적 유지** (운영자 수동 초기화만)
- 경고 3회 → 임시 화이트리스트 제외 (**관리자 확인·수동 해제 시까지**)
- 알람: 인게임 개인 채팅 문구 + 디스코드 운영로그 자동 기록

### 일반몹 IL 경고 (확정 2026-05-14)

필드보스 외 일반몹·정예몹에도 IL 초과 경고 적용. 필드 **입장 자체는 제한 없음**.

| 항목 | 내용 |
|---|---|
| 트리거 | IL 초과 상태에서 해당 필드 일반몹·정예몹 **누적 30마리마다** 경고 1회 |
| 경고 카운터 | **필드보스 경고와 동일 카운터 공유** (합산 3회 → 화이트리스트 제외) |
| 알람 | 인게임 개인 채팅 안내 + 디스코드 운영로그 |

> 예시: IL 초과 유저가 필드1 일반몹 30마리 → 경고 1회. 이후 필드1 필드보스 참여 → 경고 2회. 합산 3회 → 화이트리스트 제외.

## 잠수 방지 시스템 (확정 2026-05-14)

고IL 유저가 저레벨 필드에 방치되어 동적 스폰 계산을 왜곡하는 행위를 차단한다.

### 킬레이트 기반 스폰 제외

| 항목 | 내용 |
|---|---|
| 판정 기준 | 필드 내 **분당 10마리 미만** 처치 |
| 처리 | 해당 유저를 필드 전체 몬스터 스폰 계산 인원에서 **제외** |
| 리셋 조건 | 필드 퇴장 후 **재입장 시 정상화** |
| 목적 | 잠수 유저가 필드 몬스터 수를 불필요하게 증가시키는 것 방지 |

### AFK 강제 텔포

| 항목 | 내용 |
|---|---|
| 판정 기준 | 필드에서 **5분 이상** 위치 변화 없음 |
| 처리 | 메인 스폰 지점으로 **즉시 강제 텔포** |
| 알람 | 텔포 직전 인게임 개인 채팅 안내 |

> 두 시스템은 독립적으로 작동한다. 잠수 중 킬레이트 제외가 먼저 적용되고, 5분 후 AFK 텔포로 필드에서 완전 퇴장된다.

## 훈련장

제국 수도 내 구역. 필드 GUI의 [훈련장] 클릭으로 진입한다.

### 훈련장 GUI 구조

```
[훈련장 메인 GUI]
  ├─ 훈련장 이동    → 훈련장 구역으로 텔레포트
  ├─ 훈련 시작      → 현재 설정 기준 딜 측정 시작
  ├─ 훈련 설정      → 측정 조건 설정 서브 화면
  └─ 훈련 기록      → 과거 측정 결과 열람
```

#### 훈련장 메인 아이콘

| 버튼 | 바닐라 아이템 | 이유 |
|---|---|---|
| 훈련장 이동 | `COMPASS` | 이동/위치 직관 |
| 훈련 시작 | `CLOCK` | 타이머 시작 연상 |
| 훈련 설정 | `COMPARATOR` | 설정/조정 연상 |
| 훈련 기록 | `WRITTEN_BOOK` | 기록 보관 직관 |

#### 훈련 설정 서브 화면

| 설정 항목 | 바닐라 아이템 | 내용 |
|---|---|---|
| 측정 시간 | `CLOCK` | 10초 / 30초 / 60초 선택 |
| 허수아비 방어력 | `IRON_CHESTPLATE` | 수치 지정 (필드/보스 프리셋 또는 직접 입력) |
| 뒤로 가기 | `ARROW` | 메인으로 복귀 |

기타 몬스터 설정(저항, 상태이상 면역 등)은 추후 확장 항목으로 예약.

#### 훈련 기록

기록 1건당 저장 항목:

| 항목 | 내용 |
|---|---|
| 무기 | 클래스 + 각인 |
| 스탯 | 주요 전투 스탯 스냅샷 (ATK, 치명 등) |
| 측정 시간 | 설정된 측정 시간 |
| 몬스터 설정 | 기록 시점의 허수아비 방어력 등 설정값 |
| 딜량 | 총 피해량 + 평균 DPS |

- 기록은 유저별 최근 **10건** 보관 (초과 시 오래된 순 삭제)
- 기록 열람은 훈련장 밖에서도 가능
- 기록 슬롯 아이콘: `PAPER` (lore에 기록 내용 표시)

### 딜미터기용 허수아비

| 항목 | 내용 |
|---|---|
| 이름 | 훈련용 허수아비 |
| HP | 사실상 무한 (죽지 않음 — 처치 직후 즉시 리셋) |
| 공격 | 없음 |
| 드랍/경험치 | 없음 |

- 타격마다 피해량 홀로그램 표시
- 측정 종료 시 총 피해량·평균 DPS 채팅 출력

### 제약

- 훈련장 내 아이템 레벨 경고 / 잠수 방지 시스템 비적용
- 필드보스 기여도 산정에서 훈련장 피해량 제외
