## 1. 설계 전제 조건

### 1.1 피해 공식 (변경 없음)

```
유효 방어율 = DEF / (DEF + 200)
기본 피해   = ATK × (1 − 유효 방어율) = ATK × 200 / (DEF + 200)
최종 피해   = 기본 피해 × 스킬계수 × (1 + 보너스합계) × 크리배율
```

### 1.2 필드보스 전용 DPS 보정

시즌보스 DPS 모델(분당 히트 57회, 사이클 평균 스킬배율 1.35)에서 **업타임 60%**를 추가 적용한다.

- 시즌보스 (파티 입장형): 집중 딜, 조율된 파티 → 업타임 ~100%
- 필드보스 (리젠형 오픈월드): 다수 유저 난전, 보스 이동, 위치 조정, 넉백 → 업타임 ~60%

```
필드보스 유효 DPS = ATK × (200/(DEF+200)) × 1.35 × 57 × (1+각인%) × 0.60

각인 평균 기여율: +11% (낫/창 기준, `zenon-rpg/docs/_archive/atk_dps_baseline_v1.md` §2 요약 — archive 이동됨)
```

### 1.3 필드별 기준 ATK (v2 비선형, combat_balance_v2.md §1)

## 3. 단인 유효 DPS 역산

```
단인 DPS = ATK × 관통율 × 1.35 × 57 × 1.11 × 0.60
```

| 필드보스 | ATK | 관통율 | 단인 DPS/분 |
|---|---:|---:|---:|
| F1 들판 포식자 | 85 | 87.0% | **3,790** |
| F2 폐광 골렘 | 103 | 76.9% | **4,059** |
| F3 수로 군주 | 110 | 66.7% | **3,760** |
| F4 타락한 수호기사 | 116 | 57.1% | **3,395** |
| F5 균열 파수꾼 | 144 | 50.0% | **3,690** |

> F2 단인 DPS가 F1보다 높은 이유: 10강 마일스톤 점프(ATK +10) + DEF 60은 관통율 여전히 높음.  
> F5 DPS가 F4보다 높은 이유: 18강 마일스톤 점프(ATK +20) 효과가 DEF 50% 방어 손실을 상쇄.  
> (v2 비선형 강화의 의도된 설계 — 마일스톤 강화 시 체감 화력 점프 경험 제공)

## 4. HP 역산 및 확정

### 4.1 설계 기준

- **최소 가능 인원**: 해당 필드 인원이 이 숫자 이상이면 15분 타임아웃 내 처치 가능
- **기준 그룹**: 전형적인 참여 인원 기준, 전체 서버 활성도 감안
- **목표 TTK**: 기준 그룹 기준 8~13분 (15분 타임아웃에 여유를 두고)

다인 효율 계수:

| 인원 | 효율 | 비고 |
|---:|---:|---|
| 3명 | 80% | |
| 4명 | 77% | |
| 5명 | 72% | |
| 6명 | 70% | |
| 7명 | 65% | |

### 4.2 HP 확정 계산

**F1 들판 포식자 (기준: 최소 3명 → 13분 이내)**

```
3명×80%: 3 × 3,790 × 0.80 = 9,096/분
HP 120,000 → TTK = 13.2분 ✓

검증 (5명×72%): 5 × 3,790 × 0.72 = 13,644/분 → TTK = 8.8분 ✓
```

**F2 폐광 골렘 (기준: 최소 4명 → 13분 이내)**

```
4명×77%: 4 × 4,059 × 0.77 = 12,502/분
HP 160,000 → TTK = 12.8분 ✓

검증 (6명×70%): 6 × 4,059 × 0.70 = 17,047/분 → TTK = 9.4분 ✓
검증 (3명×80%): 3 × 4,059 × 0.80 = 9,742/분 → TTK = 16.4분 ✗ (3명 단독 불가)
```

**F3 수로 군주 (기준: 최소 5명 → 13분 이내)**

```
5명×72%: 5 × 3,760 × 0.72 = 13,536/분
HP 175,000 → TTK = 12.9분 ✓

검증 (7명×65%): 7 × 3,760 × 0.65 = 17,108/분 → TTK = 10.2분 ✓
검증 (4명×77%): 4 × 3,760 × 0.77 = 11,589/분 → TTK = 15.1분 ✗ (4명 단독 불가)
```

**F4 타락한 수호기사 (기준: 최소 6명 → 13분 이내)**

```
6명×70%: 6 × 3,395 × 0.70 = 14,259/분
HP 185,000 → TTK = 13.0분 ✓

검증 (8명×63%): 8 × 3,395 × 0.63 = 17,115/분 → TTK = 10.8분 ✓
검증 (5명×72%): 5 × 3,395 × 0.72 = 12,222/분 → TTK = 15.1분 ✗ (5명 단독 불가)
```

**F5 균열 파수꾼 (기준: 최소 6명 → 13분 이내)**

```
6명×70%: 6 × 3,690 × 0.70 = 15,498/분
HP 200,000 → TTK = 12.9분 ✓

검증 (8명×63%): 8 × 3,690 × 0.63 = 18,598/분 → TTK = 10.8분 ✓
검증 (5명×72%): 5 × 3,690 × 0.72 = 13,284/분 → TTK = 15.1분 ✗ (5명 단독 불가)
```

## 5. 확정 스탯표

| 필드보스 | DEF | HP | 유효 HP* | 최소 인원 | 기준 그룹 TTK |
|---|---:|---:|---:|---:|---:|
| 들판 포식자 (F1) | **30** | **120,000** | 138,000 | 3명 | 5명 약 9분 |
| 폐광 골렘 (F2) | **60** | **160,000** | 208,000 | 4명 | 5명 약 11분 |
| 수로 군주 (F3) | **100** | **175,000** | 262,500 | 5명 | 6명 약 11분 |
| 타락한 수호기사 (F4) | **150** | **185,000** | 323,750 | 6명 | 6명 약 13분 |
| 균열 파수꾼 (F5) | **200** | **200,000** | 400,000 | 6명 | 7명 약 11분 |

*유효 HP = HP × (DEF+200)/200. 방어율을 포함한 실질 난이도 지표.  
유효 HP는 단조 증가: 138,000 → 208,000 → 262,500 → 323,750 → 400,000

> **F5 HP(200,000)가 F4 HP(185,000)보다 높은 이유**: 18강 ATK 마일스톤 점프로 F5 플레이어 DPS가 F4보다 높아져, HP를 올려도 최소 인원(6명)이 동일하게 유지된다.  
> **체감 난이도**: 유효 HP 기준으로 F5는 F1의 2.9배. DEF 200 환경에서는 낮은 강화 단계 유저가 참여해도 피해가 절반으로 깎여 실질 기여도가 낮다.

## 7. 원샷 방지 (85% 클램프) 적용 검토

최고 스펙 유저(18강, 최대 각인+잠재)의 단일 스킬 최대 피해:

```
최대 단타 ≈ ATK × 관통율 × 스킬계수(2.0) × 버프합산(2.0)
F5 기준: 144 × 0.50 × 2.0 × 2.0 = 288 (단위: 피해량)
85% 클램프 임계치: 200,000 × 0.85 = 170,000

단타 288 << 170,000 → 1차 시즌 장비 범위에서 클램프 사실상 미발동
```

클램프는 기술적 안전장치 역할. 실질 발동은 버그 또는 GM 지급 아이템 상황에서만.

기존 §9에 기재된 필드보스 플레이스홀더:
