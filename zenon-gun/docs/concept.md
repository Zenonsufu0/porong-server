## 한 줄 컨셉
경계 있는 **영속 월드**에서 **기지를 짓고**, 버려진 시설을 **파밍**하고, 서로 **약탈**하고,
**밤의 위협** 속에서 살아남는 총기 서바이벌. **시즌(와이프) 단위** 운영.
= 마크형 **"러스트 + 타르코프"**.

## 왜 이 방향인가 (핵심 결정)
- **마크 고유 메커니즘(건설·파괴·채집·PvE) × 총** = "다른 총겜 하면 되지"에 대한 답.
  배그·러스트·타르코프 **어느 하나로도 대체 불가**한 조합. 차별점은 총이 아니라 **부서지고 지어지는 세계.**
- **시즌(와이프) 운영에 최적.** 러스트식 1~2주 와이프 주기 = "2주 시즌"과 정합. 시즌 종료=맵 와이프+시즌2 보강.
- **코어 재사용성(중요):** 영속월드 코어(총·무게·부위·손실·기지·파밍 시스템)를 만들어두면,
  훗날 **타르코프/러스트/배틀로얄 등 분리형 모드로 재활용** 가능. → 과거 "멀티모드 셀렉터" 방향은
  폐기가 아니라 **미래 확장 옵션**으로 보존.

## 비전 — 7기둥
| # | 기둥 | 영감 |
|---|---|---|
| 1 | **경계 맵 + 플레이어 기지(은신처) 건설** | 러스트 |

| 4 | **스캐브(적대 NPC)** — POI·야외 스폰, 진짜 총 쏘는 인간형 | 타르코프 |
| 5 | **밤 좀비 습격 이벤트** — 은신처 단위 방어 | 좀비 서바이벌 |

## 코어 게임플레이 루프 — "낮엔 벌고, 밤엔 살아남는다"
마크 고유의 **낮/밤 사이클**을 게임 리듬으로 채택(러스트엔 없는 강점).
레퍼런스 = 〈7 Days to Die〉(낮 파밍·건설 → 습격 — **단 7일 정기 혈월은 폐기**, 낮/밤 분업만 차용) + 러스트(기지·레이드) + 타르코프/다크앤다커(파밍·손실·비정한 교전).

```
☀️ 낮 (벌이 시간, 비교적 안전)        🌙 밤 (위협 고조 — 살아남기)
  · POI 파밍·자원 채집                  · 스캐브 활성↑ + 좀비 강화
  · 기지 건설/보강·함정                 · 기지 농성 / 야외 잔류 / 약탈, 다 선택
  · 이동·정찰·선택적 PvP(레이드)        · 사람이 더 무섭다 — 비정한 교전
```

- **상시 야간 위협(매 밤):** 밤이 되면 스캐브·좀비가 강해진다 = **환경 압박**(박자가 아닌 상시 긴장). 기지 농성·야외 잔류·이동·약탈 모두 자유 선택.
- **최후의 결전(시즌 마지막날, 단 1회):** 정기 대습격은 폐기(영속 서버는 게임시간이 24/7 흘러 박자로 못 씀). 대신 **시즌 피날레 = 거대 습격이 혼돈을 만드는 무대.** 목표는 *다같이 막기*가 아니라 **그 혼돈 속에서 살아남기·탈출.** → 시즌 엔드게임 겸함. (상세 → [`design/finale.md`](design/finale.md).)
- **기지 습격 메커니즘:** **평소 밤 = 방벽(옵시디언급 외피)이 좀비 무효 = 안전지대** / **결전(시즌 마지막날)에만 좀비 근접 공격이 방벽 돌파 게이지를 깎음** → "평소 벽 안에선 안전, 결전엔 못 버팀". 기지/야외 농성 둘 다 허용. (좀비-방벽 모델 상세 → [`design/base_raid.md`](design/base_raid.md)·[`design/finale.md`](design/finale.md).)
- **형평성:** 최후의 결전은 단 1회 고정 시간이라 "다같이 모이는 날"로 자연 해결 → 복잡한 오프라인 시뮬 불필요(1인 운영 부담 감소).

## 게임 성격 / 밸런스 철학 (무게중심: 균형)
- **무게중심 = 균형 (PvE 받침 + PvP 스파이스).** 본체는 **"험한 세계에서 살아남기"**(파밍·생존·방어, **비제로섬** — 누구나 *혼자서도* 성장 가능)이고, 그 위에 **기지 레이드·약탈을 양념**으로 얹는다. MAD는 *유일한 길*이 아니라 *선택적 스파이스*.
- **🎭 게임 톤 = 비정한 자유 서바이벌** (다크앤다커/타르코프/DayZ st).
  > "험한 세계다. 혼자 살아남아도, 믿는 자와 뭉쳐도 된다. 하지만 믿는 자 빼곤 다 적이다."
  - 동반 기지 방어·솔로 생존·총 하나로 파밍러 약탈, **전부 유효 — 강제하는 정답이 없다.**
  - 균형과 모순 아님: PvP의 **빈도는 양념**(시간 대부분은 PvE 생존), 하지만 사람을 만나는 **순간의 톤은 비정.**
- **왜 PvE 받침:** 소규모 지인(캐주얼 다수)에서 순수 제로섬 MAD는 **눈덩이 붕괴**(강자 독주·약자 이탈)를 부른다 → **PvE 비제로섬 축을 받침대로.**
- **🔑 환경 가혹함 ≠ 눈덩이(중요):** PvE 받침의 본질은 "쉬움"이 아니라 **비제로섬·반눈덩이.** 난이도 어려움·자연재생 제거·부위 데미지·굶주림은 **모두가 똑같이 맞는 환경 압박**이고 자산을 *영구 강탈* 안 하므로(죽어도 보안함·배낭레벨 유지=시즌 내 재기) **비제로섬과 양립** — "공동의 적"으로 협력 유도. → **환경 가혹함은 유지**, 완충은 *난이도*가 아니라 **①PvP 눈덩이 ②신규 진입 곡선** 두 군데에만.
- **눈덩이 완충 장치 (PvP·신규 진입에만 집중):**
  - 오프라인 레이드 보호(미접속 시 기지 보호/약화)
  - 손실 상한 + 보안 슬롯(죽어도 일부 보존) — 약탈 자유롭되 *다 못 뺏게*
  - **신규 진입 완충:** 신규 보호막 + 기본 보급([`design/economy.md`](design/economy.md)) — *초반만* 부드럽게, 중후반은 비정.
  - 레이드 비용(남 기지 털려면 자원 소모 → 무분별한 학살 억제)
- **협력/동맹 = 옵션이되 구속 없음.** 정식 연합(안전) / 구두 연합(배신 자유) 2단 — [`design/base_raid.md`](design/base_raid.md).
- **시즌 엔드게임:** ✅ 최후의 결전으로 해결 — 습격 혼돈 속 각자도생·약탈 무대 + 생존자 = 시즌 영웅.

## 스코프 원칙 (생명)
- 풀 7기둥은 1인 단기 **불가** → **"북극성 비전"으로 두고 MVP를 역산.**
- 차별화는 **거의 공짜**(마크 기본 건설/파괴 룰 허용)부터 시작.
- **무게 시각화(유리) + 무게→속도 = 도입 확정**(코어 긴장 요소라 시즌2+로 안 미룸).
- 그 외 비싼 것(상자 식별 UI, 기지 레이드 정밀 밸런스, 좀비-방벽 상호작용)은 **시즌2+ 여지**(착수 시 MVP 칼질).
- **MVP 범위는 의도적 미확정** — 스코프 칼질은 **착수 시점**(구상 단계엔 풀 비전 구체화).
- **porongun-core는 묵직한 커스텀** — 포트폴리오 자산이라 OK, 단 단계적 출시 필요([`platform.md`](platform.md)).
