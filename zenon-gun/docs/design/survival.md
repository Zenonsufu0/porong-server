## 인벤토리 · 무게 (배낭)
타르코프식 "들고 갈 양 제한"을 마크 슬롯 기반으로 구현. **공간(배낭 칸) + 무게(점유 칸)** 결합.

- **배낭 = 영구 업그레이드 (파밍품 아님, 죽어도 유지):**
  | 배낭 | 슬롯 | 비고 |
  |---|---|---|
  | 기본 | 2줄 = 18칸 | 시작 |
  | LV2 | 3줄 = 27칸 | |
  | LV3 | 4줄 = 36칸 | 풀 인벤 |
  - 죽어도 레벨 유지 = **영구 진행 = 재기 동력**(악순환 방지의 엔진). (단 와이프 시 탈출자만 보존 — [`world.md`](world.md).)
  - **⭐ 인벤 = 배낭 (별도 컨테이너 아님, 2026-06-11 명확화):** 별도 배낭 GUI가 따로 열리는 게 아니라 **플레이어 바닐라 인벤토리 그 자체가 배낭**이다. 처음엔 **메인 인벤 일부만(2줄=18칸) 개방**, 업그레이드로 27→36(=바닐라 풀 인벤)까지 줄 단위로 열림(잠긴 줄=막힌 표시·배치 불가). 무게 점유칸·보안칸·검사 가림 유리 레이어는 **바닐라 인벤 슬롯에 직접 얹는다**(E키 인벤이 곧 배낭). ⚠️ **구현 함의:** 별도 컨테이너보다 까다로움 — 바닐라 `InventoryMenu` 슬롯 잠금·무게 배치 거부는 **Mixin/이벤트 후킹** 영역(M-Inv 정식 구현의 핵심 검증거리). M0 프로토타입(2026-06-11)은 **별도 컨테이너로 유리 렌더 기법만** 검증했으므로, "바닐라 인벤 슬롯 제어"는 별도 검증 필요. ⚠️ 미확정: 2줄=18이 **핫바 포함**(핫바9+메인1줄9)인지 메인 2줄(18)인지 + 핫바도 무게 점유 대상인지.
- **무게 = 점유 칸 시스템 (= "무게 유리 시각화"의 구현):**
  - 아이템 **lore/태그에 무게(점유 칸수)** 부여 → 무게 합만큼 칸 점유, **초과분(각 아이템 무게-1의 총합)은 유리로 막힘.**
  - 예: 무게3 커스텀 총 = 자기 1칸 + **유리 2칸** = 배낭 3칸 점유.
  - **⭐ 바닐라 아이템 = 한 세트(스택)당 무게 1 (2026-06-11):** 바닐라템은 들고 있어도 다른 칸을 잠그지 않는다(자기 칸만 차지). **무게 부담은 커스텀 총·청사진·고가치 비파밍템에 집중** = "무거운 건 우리가 정한 핵심템뿐"(바닐라 잡템 관리 스트레스 X). **핫바 9칸도 무게 점유 대상**(총 들면 핫바도 칸 소모).
  - **⭐ 보안칸·무게 잠김 = 양 끝에서 채움 (2026-06-11 확정, "인접 점유"·"맨끝" 폐기):** 보안칸과 무게 잠김을 인벤 **양 끝에서** 채워 가운데를 일반 가용칸으로 비운다(아이템 바로 옆에 잠금 표시 X). **보안칸 = 윗줄 오른쪽부터** 채움(회색 봉인, 코어 레벨로 2→6 늘면 왼쪽으로 확장) / **무게 잠김 = 왼쪽부터** 채움(푸른 유리, 보안칸 만나면 건너뛰어 **다음 줄 왼쪽부터**). 보안칸은 **무게·과적 페널티 제외**(항상 안전). 잠김 칸 수 = Σ(일반칸 아이템 무게-1). 효과=인벤 안 지저분·아이템 옮겨도 잠김 위치 안 흔들림. 예: 무게3 총+무게2 청사진=왼쪽 3칸 잠김.
  - 효과: 큰 총 몇 자루면 배낭이 참 → **"뭘 들고 나갈까" 선택 긴장**(타르코프st 인벤 관리).
  - ✅ **렌더 기법 검증 완료(2026-06-11, runClient):** 슬롯 위 유리 오버레이·무게 잠김·보안 봉인·검사 가림 전부 인게임 동작 확인. "인접 N칸 점유" → "맨 끝부터 잠김"으로 모델 변경(위). 단 정식 구현은 바닐라 인벤 슬롯에 얹어야 함(↑「인벤=배낭」 항목, 바닐라 `InventoryMenu` 슬롯 후킹=잔여 검증). (🔴 GUI 항목, 포트폴리오 가치 큼.)
- **이동속도 페널티 (무게→속도, 도입):** 배낭 **적재율에 비례**해 이동속도 감소. 가벼우면 페널티 **미미**, 배낭 **LV3(36칸) 만재 시 최대 ~50% 감소**(잠정 — 밸런싱 때 확정). 큰 캡은 아님(완전 정지는 아님).
  - 효과: 점유 칸 긴장 위에 **"많이 들수록 느려진다"** 트레이드오프 추가 — 파밍 욕심 ↔ 기동성/도주력.
  - **다리 부상·갑옷 무게와 합산** — 과적 + 다리 부상 + 중갑이 겹치면 치명적 둔화. **의도된 압박**: 다리 부상은 치료를 강제, 과적은 짐을 버리게, 중갑은 방어↔기동 선택. 합산 둔화 강도·완전정지 여부 수치는 밸런싱 때.
- **보안 도구함 = 인벤 속 "잃지 않는 특수 칸" (재설계 2026-06-09):** 별도 인벤이 추가되는 게 아니라 **인벤 안에 죽어도 안 잃는 칸이 추가**된다(타르코프 secure container). **기지 레벨로 보안 칸 수 확장**(2/3/4/5/6, 코어 레벨에 통합 — 별도 확장 아이템 없음). 죽으면 일반 칸=데스박스, **보안 칸=유지.**
  - **🪟 시각 = 회색 유리 봉인 + 금색 테두리 + 자물쇠 (2026-06-11 강화):** 보안 칸은 **금색 테두리**로 묶여 한눈에 식별되고, 빈 보안칸엔 **자물쇠 아이콘**(에셋 없이 픽셀 렌더)이 표시돼 비어 있어도 "여긴 보안칸"이 명확하다. 아이템을 넣으면 **아이템 위에 반투명 회색 유리 오버레이가 겹쳐 렌더**("유리 안에 봉인된 아이템" = 보호 느낌). 구현 = **무게 점유칸 유리 시각화와 같은 커스텀 인벤 GUI 레이어**(유리 테마 통일: 무게 초과=막힌 푸른 유리 / 보안=봉인 회색+금테). 표준 슬롯 렌더라 가능(✅ 프로토타입 검증 2026-06-11).
  - **접근 = 어디서나**(인벤의 일부라 휴대) — 메뉴 GUI(상점·업글, 기지 거점)와 달리 보안 칸은 "물건"이라 밖에서도 넣고 뺌. **결전 추출 지점에서 티켓 바로 사용** 가능.
  - **칸이 곧 전략 자원:** 귀중품·**화폐**(물리 아이템) 보관 + **탈출 티켓 전용 보관소**([`finale.md`](finale.md)) + 와이프 시 **이월 stash**(생존자 보존) 용도가 한정된 칸을 두고 경쟁 → "지금 뭘 안전하게 묶어둘까"가 시즌 내내 결정거리.

## 부위 데미지 · 의료 (정밀 단계, 도입)
타르코프식 **부위별 HP + 골절 + 의료 키트.** "맞은 부위 따라 못 씀"을 깊게.
- **부위별 HP:** 머리·몸통·양팔·양다리 개별 HP. 부위 HP가 0이 되면 **골절/기능 상실**(blackout) → 강한 페널티. 머리·몸통 HP 0 = **사망.**
- **부위별 페널티:**
  | 부위 | 부상/골절 효과 |
  |---|---|
  | 다리 | **이동속도 대폭 감소(절뚝)**·점프 제한 — 무게·갑옷 페널티와 합산 |
  | 팔 | 조준 흔들림·**장전 속도 저하**(부상 팔로 못 다룸) |
  | 머리 | 치명타·즉사 위험(최저 HP) |
  | 몸통 | 출혈·HP 직격(최대 HP) |
- **의료 티어 (타르코프 메디킷st):** **붕대/지혈**(출혈 정지) → **부목(splint)**(골절 임시 고정·페널티 완화) → **수술 키트/모르핀**(부위 HP 회복·골절 완치·디버프 억제). 응급처치에 시간·자원·교전 중 리스크.
- **🔑 다리 부상 = 치료 강제(설계 의도):** 다리 둔화는 **치료 전까지 지속** → 절뚝이며 못 도망감 → **무조건 치료해야** 정상 기동 복귀. 곧 **의료템이 핵심 소모 자원**(휴대·수급·상점이 생존 필수) = "총만큼 약도 챙겨야 한다." 강한 둔화는 답답함이 아니라 *치료를 강제하는 압박*.
- 효과: 교전이 "HP 0 즉사"가 아니라 **부위 무력화의 누적** = 도주·치료·재교전의 결정. 다리 골절+과적이면 도망 불가 → **그 자리에서 치료냐, 버리고 후퇴냐.**
- **구현 = `First Aid`(원조, Forge):** 부위별 8영역 HP·자연재생 제거(의료템 강제 ← 우리 설계와 정합)·붕대/부목/모르핀·다리 둔화. 우리 정밀 부위 데미지와 거의 1:1 → 🔴(커스텀 GUI)에서 🟡(기존 모드)로 하향. porongun-core 역할 = **무게→속도 감속·갑옷·부위 둔화 합산 + 의료템 경제 편입.**
  - 보조: `TaCZ: Bleeding`도 로더 정합(단 First Aid가 이미 출혈 포함, 중복 주의).
  - ⚠️ "골절" 정밀 메커니즘은 First Aid "부위 HP 0→디버프" 근사 → 정밀 연출은 설정/소량 커스텀 여지.
- **🔍 방어구 연동 = 자동(jar 확인 2026-06-09):** `tacticalvests`=순수 바닐라 armor(헬멧+베스트) → First Aid가 **바닐라 슬롯을 자동 부위 매핑**(헬멧=머리·베스트=몸통, 별도 연동 불필요). **팔=무방어**(방어 슬롯 없음), 다리=바닐라 레깅스.
- **부위 HP 잠정(2026-06-09):** 머리 12·몸통 24·양팔 16·양다리 16(머리/몸통 0=사망). 무방어 TTK=권총 4발·AR 3~4발·저격 1발·헤드샷 저격 즉사. 방어구 차등(L12/M18/H24)=AR 5/7/12발, 고화력 관통. → economy.md 추출 스탯.
### 📊 #2 의료·부상 수치 (2026-06-10)
**앵커 = 전투 TTK(1단계 부위 HP) + 화폐 싱크(의료 가격 ↑economy) + 다리 둔화 강제(치료 압박).** 부위 HP=머리12/몸통24/팔16/다리16(1단계 확정 방향 유지).

**🩸 blackout(기능 상실):** 부위 HP 0 → 기능 상실 + 골절. **머리/몸통 0 = 사망** / 팔·다리 0 = 무력화(부목 임시 복구, 수술키트 완치).

**⚙️ 부위별 페널티 강도(잠정):**
| 부위 | 부상(HP 낮음) | blackout(HP 0) |
|---|---|---|
| **다리 1개** | 둔화 ~20% | 절뚝 ~45% |
| **양다리** | — | ~70%(거의 못 감 = 강한 치료 압박) |
| **팔 1개** | 조준 흔들림 + 장전 +30% | 장전 +100%(거의 불가) |
| 머리 | 치명타 위험 | 사망 |
| 몸통 | 출혈 | 사망 |

**💊 의료 효능표(First Aid config + economy 화폐 정합):**
| 아이템 | 효과 | 부위 HP | 사용 | 화폐 |
|---|---|---|---|---|
| 🩹 반창고 | 경상·소량 출혈 | +2 | ~2초 | 5 |
| 🩹 붕대 | 출혈 정지 + 소량 회복 | +4 | ~3초 | 10 |
| 🦴 부목 | blackout 임시 고정(페널티 완화) | 기능 ~50% 복구 | ~5초 | 30 |
| 💉 모르핀 | 통증 디버프 억제 + 일시 기동 회복 | 일시(미회복) | ~2초 | 30 |
| 🧰 수술키트 | 부위 HP 대량 회복 + 골절 완치 | +풀 | ~8초 | 120 |
- 자연재생 없음(First Aid) → 의료템이 유일 회복 = 핵심 소모재(화폐 싱크 엔진). 붕대=접근성(저가), 수술키트=상위 게이트.

**🐌 합산 둔화 상한:** 무게(LV3 만재 ~50%) + 다리 부상(~70%) + 중갑이 겹치면 단순 합산 시 완전 정지 → **상한 ~80%**(기어서라도 이동 가능 = "그 자리 치료냐 버리고 후퇴냐" 선택 유지, 완전 정지 X).

- ⚠️ 미정(밸런싱): 정확 페널티 강도·골절 조건·의료 회복량/시간·합산 둔화 상한 = 플레이테스트. **착수 0순위 검증: tacticalvests 방어 포인트 datapack 오버라이드 가능 여부 + First Aid 데미지 감소·의료 효능 config 인게임 실측**(정확 TTK·회복량).

## 손실 · 사망 처리
타르코프식 "죽으면 잃는다"를 마크에 구현. 단 영구 진행(배낭·보안함)으로 재기 가능.
- **사망 시:** 장착 장비(총·방어구) + 배낭 내용물 → 그 자리에 **데스 박스** 생성. **보안 도구함 내용물·배낭 레벨은 유지.**
- **🛏️ 리스폰 = 침대(부활점) / 없으면 랜덤 (확정 2026-06-10, 러스트 슬리핑백 + 마크 침대):**
  - **부활점 = 침대**(바닐라 침대 활용=구현 0, **스폰 설정 전용**: "자기=시간 스킵"은 비활성[밤 20분 고정·상시 위협 유지]). 죽으면 침대방에서 부활 → 데스박스 회수전.
  - **침대 없음/파괴 = 랜덤 스폰**(신규 첫 스폰과 동일 = **"거점 잃으면 떠돌이"**). **레이드로 코어 파괴 → 침대 파괴 → 다음 부활 랜덤** = 거점 상실 긴장. 단 배낭 레벨·보안함 유지라 맨몸 재시작은 아님(재기).
  - **🔑 침대 위치 = 전략:** 보호 영역 안(코어 자물쇠로 안전) vs 밖(노출, 외부인이 부수면 랜덤 부활). **연합 = 연합원 각자 침대**(공유 기지 안 각자 부활점). **부활 쿨다운 없음**(즉시 — 죽으면 장비 데스박스 손실이 자살 비용이라 악용 억제).
- **상자 식별 (타르코프 "검사 시간") — 데스박스 + POI 상자 공통:** 데스박스/POI 상자를 열면 내용물이 **유리로 가려짐** → 슬롯에 **마우스오버**로 검사해야 정체(또는 빈칸) 공개. "급하게 좋은 것만 빼가기" 불가 → 약탈에도 시간·리스크.
  - **🔑 검사 시간 = 가치별 차등(잠정):** 일반템(탄·의료·식량) **~0.5초** / 귀중품(고급총·청사진·🪖군용·🔌전자) **~1.5~2초** → "좋은 것일수록 빼가는 데 시간" = 교전 중 약탈 리스크↑. 한 번 검사한 아이템은 그 상자 세션 동안 식별 유지.
  - **🛠️ 구현:** POI 상자=**바닐라 chest 재활용**(Lost Cities 흡수)에 루트 테이블+유리 오버레이 / 데스박스=커스텀 엔티티 → **공통 유리 가림 GUI 레이어**(무게 시각화·보안칸 봉인과 같은 커스텀 인벤 레이어, ↓「무게」 유리 테마 통일).
  - **🪟 빈 상자도 조사해야 빈 줄 안다:** POI 상자는 찼는지/비었는지도 유리로 가려짐 → 누가 먼저 털었어도 **열어 조사해봐야 빈칸 확인** = "헛수고 리스크" = 약탈 경쟁 긴장. 데스박스·POI·보안칸이 **같은 유리 가림 GUI 레이어**(무게 시각화 포함 유리 테마 통일).
- **상자 접근/지속:** **누구나 열 수 있음**(비정한 톤 — 약탈 자유). 지속 **15분 후 디스폰** → 원주인 **회수전** 살림 + 서버 부하·맵 정리.
- **보험: 미도입.** 보안 도구함 + 손실 상한으로 완화 충분 + 비정한 톤("잃으면 잃는 것")에 부합.

## 진행 축 요약 (영구 vs 거점)
| 축 | 성격 | 잃는 조건 |
|---|---|---|
| **배낭 레벨** | 영구(캐릭터) | 안 잃음 |
| **보안 도구함 내용물** | 영구(일부) | 안 잃음 |
| **기지 레벨**(내구·크기·다중방벽·보안칸·상점 해금) | 거점 | **코어 파괴 시 초기화** |
| 일반 인벤·장착 장비 | 휘발 | 사망 시 상자 드랍 |
→ 영구 축(배낭·보안함)이 *재기 동력*, 거점 축(기지)이 *지킬 가치 있는 자산*. 다 잃어도 맨몸 재시작은 아님.
※ 위 "안 잃음"은 **시즌 내** 기준(일반 사망에 안 잃음). **와이프를 넘는 이월은 결전 생존이 전제** → [`world.md`](world.md) 「시즌 와이프 — 이월 정책」.
