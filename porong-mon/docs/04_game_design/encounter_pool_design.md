# 전설 조우권 풀 설계 (Encounter Pool)  — 초안(DRAFT)

> 전설 제단 조우권 등급별 + 필드 이벤트 + 컨셉별 특수권의 **소환 포켓몬 풀**. 흐름: `legendary_encounter.md`. 상점: `shop_design.md` §3.7.
> 전제: **전설 자연 랜덤 스폰 off(결정 013)** → 전설은 조우권/통제된 필드 이벤트로만. 조우권 = **PoroMonCore 커스텀 아이템**.
> 모드팩: Legendary Monuments 추가(결정 017) — 실제 추가 전설/구조물 확인 전까지 **후보로만**.
> ✅ **species ID 검증 완료(2026-06-05)**: 실 jar 추출로 전설 71(restricted 27=apex 후보 / 준전설 44)·환상 23·UB 11·패러독스 20 **실 ID 확정** → **`01_modpack/jar_registry_reference.md` §1 인용**. 남은 건 **등급/풀 배치(설계 결정)**뿐. 한국어 표시명은 정식 명칭. **종족값/타입 미수정.**

## 0. 등급 체계 (결정 018~022)
**일반 조우권 5등급** + **컨셉별 특수 조우권 10종**.
| 일반 등급 | 영문 | 조우 방식 | 가격대 |
|---|---|---|---|
| 희귀 조우권 | Rare | 개인방 (비전설 600족/고급) | 초중반 고가 |
| 하급 전설 조우권 | Basic | 개인방 + **2시간 필드 이벤트** | 고가 |
| 중급 전설 조우권 | Intermediate | 개인방 + **2시간 필드 이벤트** | 더 고가 |
| 상급 전설 조우권 | Advanced | 개인방 중심 | 매우 고가 |
| **컨셉별 특수권 10종** | Theme | 개인방(테마 혼합) | **상급 < 컨셉권 < 최상급** |
| 최상급 전설 조우권 | Apex | 개인방 중심 | 최고가 |

**가격 순서(확정): 희귀 < 하급 < 중급 < 상급 < 컨셉별 특수권 < 최상급.**

### 핵심 규칙 (결정 022)
- **컨셉별 특수권**: 중급/상급/최상급 혼합 풀. 추천 가중 **중급 55% / 상급 35% / 최상급 10%**. "이 컨셉 안에서 대박(최상급)을 노린다"는 느낌.
- **컨셉권에서 나오는 최상급 라인은 반드시 최상급 조우권 풀(§7)에도 포함**한다.
- **최상급 전설 조우권은 중급/상급이 나오지 않고 최상급 풀에서만** 등장 — "내 덱의 핵심 전설 하나를 뽑는" 고가 상품.

## 1. 희귀 조우권 풀 (전설 아님 / 결정 021)
비전설 고급 포켓몬. 개인방. **전설/환상 미포함.** 초중반~중반 고가.
**진화 단계 가중(추천): 기본형 70% / 중간진화 20% / 최종진화체 10%** — 기본형을 많이, 최종체를 드물게.
> ✅ **검증 완료(2026-06-05)**: 아래 73종 전수 실 `cobblemon:<id>` 매핑 + **전원 비전설/환상 라벨 아님 확인** → `../03_poromoncore/legendary_pools.draft.yml` `rare_encounter_pool`. (jangmoo/hakamoo/kommoo 등 ID 검증.)

## 2. 하급 전설 풀 (조우권 + 필드 이벤트)
**후보:** 프리져 · 썬더 · 파이어 · 라이코 · 앤테이 · 스이쿤 · 레지락 · 레지아이스 · 레지스틸 · 유크시 · 엠라이트 · 아그놈
**정책:** 하급 조우권(개인방) + **2시간 필드 이벤트 포함**(필드에서 중급보다 **높은 가중치**). species 확인 필요.

## 3. 중급 전설 풀 (조우권 + 필드 이벤트)
**후보:** 라티아스 · 라티오스 · 히드런 · 크레세리아 · 레지기가스 · 레지에레키 · 레지드래고 · 코바르온 · 테라키온 · 비리디온 · 케르디오 · 코스모그(후보) · 후파(후보) · **조타구 · 이야후 · 기로치(로열3 — 신규)** · **타입:널 · 실버디(신규)**
**정책:** 중급 조우권(개인방) + **2시간 필드 이벤트 포함**(필드에서 하급보다 **낮은 가중치**). 하급권보다 고가. **환상 포함 시 별도 정책 검토.** (로열3=중급 독성 3종, 타입:널/실버디=합성 전설.)

## 4. 상급 전설 풀 (조우권 개인방 중심)
**후보:** 루기아 · 칠색조 · 다크라이 · 세레비 · 후파 · 자시안 · 자마젠타 · 총지엔 · 파오젠 · 딩루 · 위유이 · 토네로스(후보) · 볼트로스(후보) · 랜드로스(후보) · **러브로스(후보 — 영물 정합)** · 카푸꼬꼬꼭(후보) · 카푸나비나(후보) · 카푸브루루(후보) · 카푸느지느(후보) · **오거폰(신규) · 치고마 · 우라오스(신규) · 블리자포스 · 레이스포스(신규, 버드렉스 기수)**
**정책:** 상급 조우권 **전용**. **필드 이벤트 미포함.** 고가. 배지 8 이후/후반 후보. (다크라이/세레비/후파 등 환상 포함 — 별도 정책 검토. 러브로스=토네/볼트/랜드로스 영물 4번째 정합. 블리자포스/레이스포스=버드렉스(§5) 기수 라인.)

## 5. 최상급 전설 풀 (최고가 / 최상급권 전용)
**후보:** 뮤츠 · 뮤 · 그란돈 · 가이오가 · 레쿠쟈 · 디아루가 · 펄기아 · 기라티나 · 레시라무 · 제크로무 · 큐레무 · 제르네아스 · 이벨타르 · 지가르데 · 솔가레오 · 루나아라 · 네크로즈마 · 자시안(후보) · 자마젠타(후보) · 무한다이노 · 아르세우스 · **코라이돈 · 미라이돈 · 테라파고스 · 버드렉스(신규 박스전설 — restricted)**
**정책:** 최상급 조우권 **전용**(중급/상급 미등장). 매우 고가. 후반/이벤트/특수 조건. **컨셉권에서 나오는 최상급 라인은 이 풀에도 포함.** 레쿠쟈=하늘, 아르세우스=영원, **코라이돈/미라이돈=용왕, 테라파고스=빛, 버드렉스=수호자** 컨셉권에도 포함. **아르세우스/신규 박스전설 미존재 시 각 `enabled: false`.** (전 종 jar 검증 완료 — `jar_registry_reference.md` §1.)

## 6. 필드 이벤트 통합 풀 (하급 + 중급)
- 풀 = **하급 + 중급 합집합**. 기본 가중 **하급 70% / 중급 30%**.
- 운영: `legendary_encounter.md` 필드 이벤트 정책(2시간 1회·1마리·힌트 공지·디스폰·전투 유예·동시 1·재시작 차단·최소 인원).
- 상급/최상급은 **미포함**.

## 7~8. 컨셉별 특수 조우권 10종
공통: **가격 = 상급 < 컨셉권 < 최상급.** 가중 **중급 55 / 상급 35 / 최상급 10**(후보). 각 테마의 **최상급 후보는 최상급 풀(§5)에도 포함**. (수치·enabled = `legendary_pools.yml`)

- 최상급 후보: **레쿠쟈**
- 정책: **레쿠쟈 중심.** 레쿠쟈는 일반 하급/중급/상급 풀에 넣지 않음(하늘/최상급 전용).
