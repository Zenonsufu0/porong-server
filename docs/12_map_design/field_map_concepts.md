## 공통 전제

- 모든 필드는 독립 구역. 텔레포트로 입장/퇴장한다.
- 바닐라 블록과 바닐라 강화형 몹만 사용한다 (ModelEngine/BetterModel 없음).
- 블록 파괴는 기본 차단 (WorldGuard 보호).
- AFK 강제 텔포: 5분 미이동 시 메인 스폰으로 퇴장. 구역 설계 시 막힌 구석/좁은 골목이 AFK 온상이 되지 않도록 유의.
- 필드보스 아레나는 필드 내부에 포함되며, 보스 리젠 위치는 아레나 중앙 고정이다.
- 각 필드는 입장 포탈과 퇴장 포탈을 구역 경계에 배치한다.

## 필드 1. 수도 외곽 평원 (Plains)

### 1. 컨셉 한 줄 요약

제국 수도를 나서면 처음 마주치는 열린 초원. 낮에는 평화롭게 보이지만 들짐승이 점거한 위험 지대.

### 2. 권장 바이옴 / 블록 팔레트

| 용도 | 블록 |
|---|---|
| 바닥 (기본) | Grass Block, Dirt, Coarse Dirt |
| 바닥 (길/통로) | Gravel, Stone |
| 언덕/지형 변화 | Stone, Andesite, Calcite |
| 식생 | Oak Leaves, Short Grass, Tall Grass, Fern, Dandelion, Poppy, Oxeye Daisy |
| 나무 | Oak Log, Oak Leaves |
| 울타리/장식 | Oak Fence, Hay Bale, Barrel, Lantern |
| 수로 | Water, Mossy Cobblestone |
| 하늘 (느낌) | 개방형, 지붕 없음 |

- 전체 톤: 밝은 초록 + 흙빛. 무너진 농가 흔적을 드문드문 배치해 "버려진 들판" 분위기.

### 3. 구역 구성

- 형태: 오픈 필드형 (개방 공간 중심)
- 권장 크기: **220 x 220** (입장 포탈 포함)
- 세부 구역:

| 구역 | 위치 | 설명 |
|---|---|---|
| 입장 구역 | 남쪽 경계 | 포탈 + 안내 홀로그램. 제국 수도 방향 느낌. |
| 개방 초원 A | 중앙 서쪽 | 일반몹 주 서식지. 시야가 트여 있어 초보자가 적응하기 쉬움. |
| 개방 초원 B | 중앙 동쪽 | 일반몹 주 서식지 2. 낮은 언덕 지형으로 약간의 고저차 있음. |
| 폐농가 군락 | 북동쪽 | 무너진 오두막 2~3채. 정예몹 서식지. 지붕 없는 반파 건물. |
| 보스 아레나 | 북쪽 중앙 | 넓은 원형 공터. 주변을 낮은 돌담으로 둘러싼 형태. |
| 퇴장 구역 | 북서쪽 경계 | 다음 필드(폐광 지대) 방향 포탈 위치 (장식용, 실제 이동은 GUI). |

### 4. 몹 배치 아이디어

- **Plains_Soldier (일반몹)**: 개방 초원 A/B 전역에 산발 배치. 2~4마리 소규모 그룹.
- **Plains_Wildling (일반몹)**: 개방 초원 B 언덕 주변. 단독 또는 2마리. 조금 더 공격적인 느낌.
- **들판의 노식자 (정예몹)**: 폐농가 군락 내부 또는 주변. 1~2마리. 반파 건물 사이에서 순찰.
- **들판 포식자 / Plains_Predator (필드보스)**: 보스 아레나 중앙. 리젠 시 아레나 내 고정 스폰.

### 5. 보스 아레나

- 형태: 지름 약 30칸의 원형 공터
- 재질: 바닥은 Gravel + Coarse Dirt 혼합. 가장자리에 낮은 Cobblestone Wall 1~2단 두름.
- 장식: 쓰러진 Oak Log, 부서진 Lantern 기둥 2~3개.
- 목적: 시야가 완전히 열린 구조. 초보자도 보스의 움직임을 읽기 쉬운 환경.
- 특이사항: 아레나 진입 후 자동으로 보스 전투 범위 진입 처리 없음 — MythicMobs 감지 범위로 처리.

### 6. 조명 / 분위기

- 전체: 자연광 위주. 별도 횃불 없이 낮은 밝기 제한 없이 개방.
- 폐농가 구역: 내부에 Lantern 1~2개만 배치해 "방치된 곳" 느낌.
- 보스 아레나: 횃불 없음. 낮/밤 모두 싸울 수 있는 개방형.
- 분위기 키워드: 밝음, 개방, 약간 쓸쓸함.

### 7. 이동 동선

```
입장 포탈 (남쪽)
  → 개방 초원 A (서쪽) 또는 초원 B (동쪽) 중 선택
    → 폐농가 군락 (북동쪽) — 정예몹 목적
      → 보스 아레나 (북쪽 중앙) — 보스 목적
        → 퇴장 포탈 (북서쪽)
```

- 자연스럽게 남에서 북으로 흐름. 강제 유도는 없고 탁 트인 시야로 목적지를 스스로 인식.

## 필드 2. 폐광 지대 (Mine)

### 1. 컨셉 한 줄 요약

제국이 버린 낡은 광산 지대. 갱도와 노천 채굴 흔적이 뒤엉킨 위험한 지하 미로.

### 2. 권장 바이옴 / 블록 팔레트

| 용도 | 블록 |
|---|---|
| 바닥 (야외) | Stone, Gravel, Cobblestone |
| 바닥 (갱도 내부) | Smooth Stone, Deepslate, Cobbled Deepslate |
| 벽/천장 (갱도) | Stone Bricks, Cracked Stone Bricks, Mossy Stone Bricks, Cobblestone |
| 광맥 장식 | Iron Ore, Coal Ore, Deepslate Iron Ore |
| 목재 지지대 | Spruce Log, Stripped Spruce Log, Oak Planks |
| 비계 흔적 | Ladder, Chain, Lantern |
| 무너진 부분 | Gravel, Coarse Dirt, Cobblestone Slab (반파 느낌) |
| 물웅덩이 | Water, Mossy Cobblestone |

- 전체 톤: 회색 + 갈색. 지하와 반지하가 혼재하는 구조.

### 3. 구역 구성

- 형태: 혼합형 (야외 노천 채굴 구역 + 반지하 갱도 구역)
- 권장 크기: **200 x 200** (지상 기준), 갱도는 지하 4~8칸 깊이
- 세부 구역:

| 구역 | 위치 | 설명 |
|---|---|---|
| 입장 구역 | 남쪽 | 포탈 + 홀로그램. 낡은 채광 마차 장식. |
| 노천 채굴 구역 | 중앙 서쪽 | 지상 절개지 형태. 계단식 구조. 일반몹 배치. |
| 메인 갱도 입구 | 중앙 | 지하로 내려가는 주 입구 터널. |
| 갱도 내부 통로 A | 지하 북쪽 | 좌우로 갈리는 미로형 갱도. 일반몹 배치. |
| 갱도 내부 통로 B | 지하 동쪽 | 더 어둡고 좁은 막장 갱도. 정예몹 서식지. |
| 갱도 중앙 공동 | 지하 중앙 | 넓게 뚫린 자연 동굴 느낌. 보스 아레나. |
| 퇴장 구역 | 지상 북쪽 | 갱도에서 나오는 출구 포탈. |

### 4. 몹 배치 아이디어

- **Mine_Crawler (일반몹)**: 노천 구역 + 갱도 통로 A. 천장이 낮은 좁은 구간 위주.
- **Mine_Husk (일반몹)**: 갱도 통로 A 전역. 밝기가 낮은 구간에 집중.
- **폐광 감시병 (정예몹)**: 갱도 통로 B 막장 지역. 단독 1마리 또는 일반몹 1마리와 함께.
- **폐광 골렘 / Mine_Golem (필드보스)**: 갱도 중앙 공동 고정 스폰.

### 5. 보스 아레나

- 형태: 지하 자연 공동 스타일. 약 25 x 25칸의 불규칙 원형.
- 재질: 바닥은 Stone + Gravel 혼합. 벽면은 Smooth Stone Bricks + 부분적 Deepslate.
- 장식: 무너진 Spruce Log 지지대, 부서진 Lantern, 주변 벽에 Iron Ore 장식.
- 높이: 최소 10칸 이상 확보 (보스 돌진 패턴 P-02 고려).
- 조명: Lantern 벽 부착으로 약한 조명. 어둡지만 플레이 불가 수준은 아님.

### 6. 조명 / 분위기

- 야외 노천 구역: 자연광. 먼지 날리는 척박한 느낌.
- 갱도 입구~통로 A: Lantern + Torch 간격 배치. 밝기 7 이상 유지 (몹 자연 스폰 방지, MythicMobs 지정 스폰만 사용).
- 갱도 통로 B + 아레나: Lantern만 드문드문. 어둡고 폐쇄된 느낌.
- 분위기 키워드: 폐쇄, 불안, 오래된 산업 유적.

### 7. 이동 동선

```
입장 포탈 (남쪽 지상)
  → 노천 채굴 구역 (서쪽) — 지상 일반몹
    → 메인 갱도 입구 (중앙)
      → 갱도 통로 A (북쪽 지하) 또는 통로 B (동쪽 지하, 정예몹 목적)
        → 갱도 중앙 공동 (보스 아레나)
          → 퇴장 포탈 (북쪽 지상 출구)
```

- 지상과 지하의 공간 전환이 자연스럽게 이뤄지도록 갱도 입구를 크고 눈에 띄게 설계.

## 필드 3. 오염된 수로 (Sewer)

### 1. 컨셉 한 줄 요약

제국의 낡은 하수도와 배수 수로가 오염 물질로 뒤덮인 지하 네트워크. 악취와 독기가 가득하다.

### 2. 권장 바이옴 / 블록 팔레트

| 용도 | 블록 |
|---|---|
| 바닥 (수로 바닥) | Mossy Stone Bricks, Mossy Cobblestone, Smooth Stone |
| 벽 (통로 벽) | Stone Bricks, Mossy Stone Bricks, Cracked Stone Bricks |
| 천장 (터널) | Stone Bricks, Deepslate Bricks |
| 수로 물 | Water (초록빛 포션 파티클로 오염 표현) |
| 오염된 웅덩이 | Slime Block (장식, 실제 이동은 제한), Light Blue Concrete (오염수 느낌 장식) |
| 배관 흔적 | Iron Bars, Chain, Chiseled Stone Bricks |
| 이끼/균류 | Moss Block, Mossy Stone Bricks, Brown Mushroom, Red Mushroom |
| 수위 변화 구역 | Stairs + Slab 조합으로 단차 표현 |

- 전체 톤: 초록빛이 도는 어두운 회색. 물과 이끼가 곳곳에.

### 3. 구역 구성

- 형태: 복도형 + 부분 오픈 공간
- 권장 크기: **180 x 180** (지하 구조 기준)
- 세부 구역:

| 구역 | 위치 | 설명 |
|---|---|---|
| 입장 구역 | 지하 입구 | 포탈 + 낮은 아치형 석조 입구. 오염된 공기 파티클 연출. |
| 상류 수로 | 서쪽 | 수위가 낮은 구역. 비교적 넓은 통로. 일반몹 배치. |
| 배수 분기점 | 중앙 | 여러 통로가 갈라지는 교차로. 이정표 역할의 구조물 배치. |
| 하류 수로 | 동쪽 | 수위가 높고 통로가 좁아짐. 일반몹 + 이동 제한 요소. |
| 침수 지하실 | 북동쪽 | 무릎 높이 물이 차 있는 막힌 공간. 정예몹 서식지. |
| 핵심 오염 웅덩이 | 북쪽 중앙 | 가장 오염이 심한 넓은 공간. 보스 아레나. |
| 퇴장 구역 | 북서쪽 | 외부로 이어지는 출구 포탈. |

### 4. 몹 배치 아이디어

- **Waterway_Drowned (일반몹)**: 상류/하류 수로 물가. 수중 또는 수면 근처에 배치.
- **Waterway_Guardian (일반몹)**: 배수 분기점 주변. 통로 막고 서 있는 경비 느낌.
- **오염 군주의 수하 (정예몹)**: 침수 지하실 내부. 1마리 단독 또는 일반몹 2마리와 패키지.
- **수로 군주 / Waterway_Lord (필드보스)**: 핵심 오염 웅덩이 중앙 고정 스폰.

### 5. 보스 아레나

- 형태: 지름 약 28칸의 반원형 오염 웅덩이 공간
- 재질: 바닥은 Mossy Stone Bricks + Slime Block 부분 혼합 (미끄러운 느낌 연출). 사방은 Stone Bricks 벽.
- 물: 아레나 가장자리를 따라 Water 얕게 흐름 (이동 제한 목적이 아닌 연출).
- 높이: 6~8칸 (반지하 공간, 약간 개방감 있게).
- 특이사항: 바닥 Slime Block은 시각 장치 전용. 실제 튕김 효과가 전투 방해가 되면 Mossy Stone Bricks로 대체.

### 6. 조명 / 분위기

- 전체: Soul Torch (청록빛) + 일반 Lantern 혼합. 어둡고 불균일한 조명.
- 상류 수로: Lantern 간격 배치. 밝기 조금 있음.
- 하류/침수 지하실: Soul Torch 주로 사용. 파란빛이 도는 어두운 분위기.
- 보스 아레나: Soul Lantern + Glow Lichen 벽면 배치. 오염된 발광 느낌.
- 분위기 키워드: 습함, 독기, 폐쇄, 청록빛 오염.

### 7. 이동 동선

```
입장 포탈 (지하 입구)
  → 상류 수로 (서쪽) — 일반몹 Drowned 위주
    → 배수 분기점 (중앙) — 갈림길
      → 하류 수로 (동쪽) — 일반몹 Guardian
        → 침수 지하실 (북동쪽) — 정예몹 목적
          → 핵심 오염 웅덩이 (북쪽 중앙) — 보스 아레나
            → 퇴장 포탈 (북서쪽)
```

- 분기점 교차로를 통해 플레이어가 정예몹 노선과 보스 직행 노선을 선택할 수 있도록 설계.

## 필드 4. 무너진 초소 (Outpost)

### 1. 컨셉 한 줄 요약

제국의 최전방 방어선이었던 군사 초소 잔해. 타락한 기사단이 점거한 반파 요새.

### 2. 권장 바이옴 / 블록 팔레트

| 용도 | 블록 |
|---|---|
| 바닥 (야외) | Cobblestone, Stone Bricks, Gravel |
| 바닥 (실내) | Smooth Stone, Stone Bricks |
| 벽 (건물) | Stone Bricks, Cracked Stone Bricks, Chiseled Stone Bricks |
| 벽 (무너진 부분) | Mossy Cobblestone, Cobblestone Wall |
| 지붕/천장 | Stone Slab, Stone Bricks Slab (일부 무너짐) |
| 군사 장식 | Barrel, Chest, Banner (Dark Gray), Chain |
| 방어 시설 흔적 | Cobblestone Wall, Iron Bars, Iron Trapdoor |
| 불/화재 흔적 | Campfire, Soul Campfire, Scorched-feel: Blackstone, Basalt |
| 식생 침투 | Vines, Mossy Stone Bricks, Moss Block |

- 전체 톤: 어두운 회색 + 검은빛. 오래된 전쟁터의 황폐함.

### 3. 구역 구성

- 형태: 혼합형 (야외 광장 + 건물 실내 복수)
- 권장 크기: **190 x 190**
- 세부 구역:

| 구역 | 위치 | 설명 |
|---|---|---|
| 입장 구역 | 남쪽 | 포탈 + 무너진 성문 잔해. 초소 분위기 첫인상. |
| 외곽 야영지 | 남서쪽 | 반파 천막과 모닥불 흔적. 일반몹 배치. |
| 주 광장 | 중앙 | 넓은 돌바닥 광장. 일반몹 순찰. |
| 본관 건물 잔해 | 북서쪽 | 2층 구조 반파 건물. 내부 정예몹 서식. 창문/계단 구조. |
| 무기고 폐허 | 북동쪽 | 무기 보관소 흔적. Barrel/Chest 장식. 정예몹 서식. |
| 사령부 잔해 | 북쪽 중앙 | 초소 내 가장 큰 건물 잔해. 보스 아레나. |
| 퇴장 구역 | 동쪽 경계 | 포탈. |

### 4. 몹 배치 아이디어

- **Outpost_Pillager (일반몹)**: 외곽 야영지 + 주 광장 순찰. 2~3마리 그룹.
- **Outpost_Vindicator (일반몹)**: 주 광장 + 본관 건물 입구 보초 느낌. 단독 또는 2마리.
- **타락 기사 부관 (정예몹)**: 본관 건물 2층 또는 무기고 폐허 내부. 1마리.
- **타락한 수호기사 / Fallen_Knight (필드보스)**: 사령부 잔해 내부 고정 스폰.

### 5. 보스 아레나

- 형태: 지붕이 반쯤 무너진 사령부 홀. 약 22 x 22칸.
- 재질: 바닥은 Smooth Stone + Stone Bricks 혼합. 벽은 Cracked Stone Bricks. 일부 천장 슬라브로 불완전한 지붕 표현.
- 장식: 쓰러진 Banner (Dark Gray), 부서진 Chest, 벽에 Iron Bars 창문 흔적.
- 높이: 8~10칸 (반옥외 느낌. 일부 하늘 보임).
- 특이사항: 건물 내부이지만 지붕이 뚫려 있어 완전히 폐쇄되지 않음 — 보스 패턴 중 돌진(P-02) 공간 확보.

### 6. 조명 / 분위기

- 야외 구역: Soul Campfire + 일반 Campfire 혼합. 꺼진 횃불 장식.
- 본관 내부: Lantern 드문드문. 어둡고 음산함.
- 보스 아레나: Soul Lantern 4~5개 벽면 배치. 창문 틈으로 자연광 일부 유입 느낌.
- 분위기 키워드: 황폐, 전쟁터 유적, 음산함, 타락의 흔적.

### 7. 이동 동선

```
입장 포탈 (남쪽 성문 잔해)
  → 외곽 야영지 (남서쪽) — 일반몹 Pillager
    → 주 광장 (중앙) — 일반몹 순찰
      → 본관 건물 (북서쪽) 또는 무기고 (북동쪽) — 정예몹 노선 선택
        → 사령부 잔해 (북쪽 중앙) — 보스 아레나
          → 퇴장 포탈 (동쪽)
```

- 광장을 중심으로 좌우로 선택지가 갈려 반복 플레이 시 루트 다양성 확보.

## 필드 5. 고대 성벽 잔해 (Ruins)

### 1. 컨셉 한 줄 요약

시대를 앞선 문명이 남긴 성벽 잔해. 균열 에너지가 흘러들어 공간 자체가 뒤틀리기 시작한 곳.

### 2. 권장 바이옴 / 블록 팔레트

| 용도 | 블록 |
|---|---|
| 바닥 (야외) | Deepslate Bricks, Polished Deepslate, Cracked Deepslate Bricks |
| 바닥 (잔해 구역) | End Stone Bricks, Purpur Block (균열 에너지 흔적) |
| 벽 (성벽 잔해) | Deepslate Bricks, Cracked Deepslate Bricks, Chiseled Deepslate |
| 균열 장식 | Crying Obsidian, Amethyst Block, Amethyst Cluster, Budding Amethyst |
| 에너지 흔적 | End Rod, Glow Lichen, Sea Lantern |
| 파괴된 구조물 | Basalt, Blackstone, Polished Blackstone Bricks |
| 고대 조각/문양 | Chiseled Deepslate, Chiseled Stone Bricks |
| 식생 없음 | (식물/초목 거의 없음 — 오염된 땅) |

- 전체 톤: 진한 회색 + 검은빛 + 보라/에메랄드빛 균열 에너지. 가장 어둡고 이질적인 분위기.

### 3. 구역 구성

- 형태: 반복 레이어형 (성벽 단계를 통과하며 깊숙이 진입하는 구조)
- 권장 크기: **200 x 200**
- 세부 구역:

| 구역 | 위치 | 설명 |
|---|---|---|
| 입장 구역 | 남쪽 | 포탈 + 거대한 성벽 잔해 첫 문. 압도적인 규모로 시작. |
| 외벽 구역 | 남쪽 ~ 중앙 외곽 | 성벽 바깥. 넓은 공간, 일반몹 배치. 균열 에너지 조금. |
| 성벽 틈 통로 | 중앙 | 성벽이 쪼개진 좁은 통로. 이동 시 압박감 있는 병목 구조. |
| 내벽 구역 | 중앙 내부 | 성벽 안쪽. 일반몹 + 정예몹 혼재. 균열 에너지 강해짐. |
| 균열 발원지 | 북쪽 중앙 | 가장 안쪽. 균열 에너지가 폭발적으로 흘러나오는 공간. 보스 아레나. |
| 퇴장 구역 | 동쪽 경계 | 포탈. |

### 4. 몹 배치 아이디어

- **Wall_Enderman (일반몹)**: 외벽 구역 + 내벽 구역 전역. 단독 또는 2마리. 순간이동 특성 활용.
- **Wall_Shade (일반몹)**: 성벽 틈 통로 + 내벽 구역. 좁은 공간에서 출현하는 느낌.
- **균열 봉인병 (정예몹)**: 내벽 구역 깊숙한 곳. 균열 발원지 입구 직전. 1마리 보초 역할.
- **균열 파수꾼 / Rift_Watcher (필드보스)**: 균열 발원지 중앙 고정 스폰.

### 5. 보스 아레나

- 형태: 성벽에 둘러싸인 원형 공터. 지름 약 35칸. 가장 큰 보스 아레나.
- 재질: 바닥은 Polished Deepslate + Purpur Block 체크무늬 또는 문양. 벽은 Deepslate Bricks 높이 15칸 이상.
- 균열 장식: 바닥 중앙에 Crying Obsidian + Amethyst Cluster 방사형 배치. 보스 스폰 위치를 시각적으로 표시.
- 높이: 완전 개방 (지붕 없음). 성벽이 사방을 막는 구조 — "경기장" 느낌.
- 조명: End Rod 기둥 8개를 아레나 외곽에 배치. 청백빛 조명.
- 특이사항: 보스 아레나 입구에 Crying Obsidian 문 장식으로 진입 순간 분위기 전환 유도.

### 6. 조명 / 분위기

- 외벽 구역: 거의 어두움. Glow Lichen 바닥/벽 드문드문.
- 성벽 틈 통로: 매우 어두움. Sea Lantern 1~2개로 최소 가시성만 확보.
- 내벽 구역: End Rod + Amethyst Cluster 발광. 보라/청백빛.
- 보스 아레나: End Rod 기둥 + Crying Obsidian 발광. 독특하고 이질적인 조명.
- 분위기 키워드: 균열, 이질감, 압도, 최종 관문.

### 7. 이동 동선

```
입장 포탈 (남쪽 성문 잔해)
  → 외벽 구역 (넓음, 일반몹 Wall_Enderman/Shade)
    → 성벽 틈 통로 (병목, 좁고 어두움)
      → 내벽 구역 (일반몹 + 정예몹 균열 봉인병)
        → 균열 발원지 / 보스 아레나 (최종 목적지)
          → 퇴장 포탈 (동쪽)
```

- 성벽 틈 통로를 지날 때 공간이 좁아지면서 자연스럽게 긴장감이 고조됨.
- 외벽 → 내벽 단계를 거치며 분위기가 점점 이질적으로 변화.

## 빌드 주의사항

### 공통

- 모든 필드 내부는 WorldGuard로 블록 파괴/설치 차단한다.
- MythicMobs 몹 스폰 포인트는 자연 스폰을 사용하지 않고 지정 위치 스폰으로 처리한다. 따라서 필드 전체를 완전히 어둡게 만들어도 무방하다.
- 입장/퇴장 포탈은 EmpireRPG 텔레포트와 연동된다. 실제 Nether Portal 블록을 쓰지 않아도 되며, 장식용 문 구조물로 대체할 수 있다.
- 보스 아레나는 보스 스폰 반경(MythicMobs 설정) 외부에서 접근 가능한 관전 구역을 두지 않는다 — 기여도 판정 경계와 아레나 경계를 일치시킬 것.

### 필드별

- **필드 2 폐광**: 갱도 높이를 최소 5칸 이상으로 유지해 보스 돌진 패턴이 막히지 않도록 주의.
- **필드 3 수로**: 바닥의 Water 블록이 플레이어 이동 속도를 크게 저하시키지 않도록 깊이를 1칸으로 제한. Boat 이동은 WorldGuard로 차단 권장.
- **필드 4 초소**: 본관 건물 2층 구조에서 몹이 1층으로 낙하해 배치가 무너지지 않도록 MythicMobs 스폰 위치를 층별로 고정.
- **필드 5 성벽**: Enderman 기반 Wall_Enderman 몹이 텔레포트로 아레나 밖으로 나가지 않도록 MythicMobs 설정에서 이동 범위를 제한하거나 텔레포트 능력을 비활성화.
