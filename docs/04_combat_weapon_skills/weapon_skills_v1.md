# 04. 전투 / 무기 스킬 설계안

> **[STATUS: DRAFT]** — 무기명 "도끼" 수정 완료 (DL-003). `CANON.md` 참조.

> 최종 동기화: 2026-05-16
> v2.2 변경: 자원 효과 선형 스케일링 도입, 유지형 6스택 상한 통일, 스킬 계수 스택 연동

## 기본 방향

포로 서버의 전투는 쿨타임 기반 액션 전투를 기준으로 한다.

기본 구조는 다음과 같다.

- 무기 6종: 검 / 도끼 / 창 / 석궁 / 낫 / 스태프
- 각 무기 구성
  - 기본 공격 (LMB, 슬롯 1 쿨 중일 때 자동 발동. 자원 생성 없음, 단순 피해)
  - 액티브 스킬 4개 (슬롯 1~4)

## 키 매핑

| 입력 | 역할 | 동작 |
|---|---|---|
| LMB | 기본기 (자원 생성) | 슬롯 1 스킬. 쿨 중이면 기본 공격 자동 발동. |
| RMB | 이동기 / 보조기 | 슬롯 2 스킬 |
| Shift+RMB | 특수기 / 제어기 / 자기 강화 | 슬롯 3 스킬 |
| F | 핵심기 (자원 소모) | 슬롯 4 스킬 |

## 각인 구조

각 무기의 두 각인은 **소모형(A)** 과 **유지형(B)** 으로 분기한다.

| 타입 | 특징 |
|---|---|
| 소모형 (각인 A) | F 발동 시 자원 전체 소모. 만충 상태에서 최대 스케일링 한 방을 노린다. 소모 직후 즉각 리턴으로 다음 사이클 가속. |
| 유지형 (각인 B) | 자원 최대 **6스택** (무기 공통). F 소모 없음. 스택당 스킬 피해 선형 증가. 6스택 달성 시 무기 고유 특수 효과 활성. |

> **자원 기본 패시브 효과는 소모형/유지형 구분 없이 두 각인 공통 적용.**
> 유지형은 스택 상한이 6으로 늘어나 소모형이 도달할 수 없는 스케일링 영역에 접근한다.

## 무기별 기본 공격

| 무기 | 기본 공격 동작 | 타격 속도 |
|---|---|---:|
| 검 | 단타 베기 | 0.8초 |
| 도끼 | 단타 내려치기 | 1.0초 |
| 창 | 단타 찌르기 | 0.7초 |
| 석궁 | 단발 사격 | 0.6초 |
| 낫 | 단타 베기 | 0.8초 |
| 스태프 | 마력탄 1발 | 0.9초 |

전투 설계 원칙:
- 복잡한 마나 관리보다 쿨타임 중심
- 무기별 조작감 차이를 확실히 구분
- 플레이어가 설치하는 지속 장판류는 최소화
- 표식 / 받피증 / 방깎 같은 디버프 난립 금지
- 판정은 가능한 한 단순하고 명확하게
- 스킬은 데이터화 가능한 구조로 설계

## 공통 스킬 데이터 필드 예시

```yaml
skill_id: string
weapon_type: sword | axe | spear | crossbow | scythe | staff
name: string
cooldown: number
cast_time: number
damage_multiplier: number        # 기본 계수
stack_damage_bonus_per: number   # 스택당 추가 피해% (0이면 미적용)
shape_type: melee_arc | line | cone | projectile | self | burst
range: number          # line/projectile/cone: 전방 사거리(블럭)
width: number          # line: 판정 너비(블럭)
radius: number         # melee_arc / burst / projectile AoE: 반경(블럭)
angle: number          # melee_arc / cone: 좌우 총 각도(°). 전방 중심 기준.
hit_count: number
movement: none | dash_forward | dash_back | short_step | side_step
movement_distance: number   # 이동기 이동 거리(블럭). movement: none이면 0.
movement_duration: number   # 이동 소요 시간(초)
effect_key: string     # mm:xxx | dp:xxx | pt:xxx | 빈 값(이펙트 없음)
description: string
```

## 스킬 이펙트 시스템

> **확정 (DL-031)** — 방안 4 혼합 구조. 판정·피해는 EmpireRPG 단독 책임.
> **1차 시즌 추가 확정 (DL-070)** — 모든 플레이어 스킬 이펙트는 Bukkit Particle 직접 구현(`pt:`)으로 통일. MythicMobs `effect_key` 위임은 2차 확장으로 보류.

### 핵심 원칙

1. 스킬 판정, 피해, 쿨타임, 상태 적용은 EmpireRPG가 전부 담당한다.
2. **1차 시즌**: 이펙트는 `BaseWeaponSkill` 헬퍼(`spawnParticleArc`, `spawnParticleCircle` 등) + Bukkit Particle 직접 구현. MythicMobs 이펙트 위임 없음 (DL-070).
3. Display Entity는 0.5초 이상 시각적으로 유지되는 투사체·검기·마법탄에만 사용한다.
4. MythicMobs나 Display Entity 이펙트 실패가 스킬 판정 실패로 이어지면 안 된다.
5. ModelEngine/BetterModel은 2차 확장으로 보류한다.

### 이펙트 설계 원칙

- `shape_type`과 이펙트 모양이 일치해야 한다.

| shape_type | 이펙트 형태 |
|---|---|
| `line` | 직선 궤적 또는 빔 |
| `arc` | 베기 호 (부채꼴 파티클 호) |
| `cone` | 부채꼴 방사형 이펙트 |
| `burst` | 원형 충격파 (중심→외곽) |
| `projectile` | 투사체 본체 또는 궤적 파티클 |
| `self` | 시전자 중심 간단한 이펙트 |

- 플레이어 스킬 이펙트는 보스 패턴 이펙트보다 덜 화려하게 유지한다.
- 월드보스 다인전에서 시야를 가리지 않아야 한다 — 파티클 밀도와 지속 시간을 보수적으로 설정.

### effect_key prefix 규칙

**1차 시즌 기준 (DL-070)**

| prefix | 처리 방식 | 1차 시즌 |
|---|---|---|
| `pt:xxx` | Bukkit Particle 직접 구현 (`BaseWeaponSkill` 헬퍼) | **표준** |
| `mm:xxx` | MythicMobs castSkill 위임 | 2차 확장 예약 — 1차 시즌 미사용 |
| `dp:xxx` | Display Entity 기반 시각 이펙트 | 2차 확장 예약 — 1차 시즌 미사용 |
| (빈 값) | 이펙트 없음 | — |

> 1차 시즌에서 `effect_key`는 식별자 역할만 한다. 실제 이펙트는 각 스킬 클래스가 `BaseWeaponSkill` 헬퍼를 직접 호출한다.

### 1차 이펙트 구현 우선순위 (DL-070 기준)

1. **`BaseWeaponSkill` 헬퍼** (`spawnParticleArc`, `spawnParticleCircle`, `spawnParticleLine`, `playSound`) — 공통 인프라 확정
2. **낫 4스킬** — ✅ 완료: 사신베기·월영회전(기존) + 그믐참(보라/검붉은 cone+마녀입자)·처형낫(보라 직선+영혼불) (2026-05-30)
3. **검 4스킬** — ✅ 완료 (2026-05-30): 섬광베기(흰빛 호+END_ROD)·연속참(강철 3중 호+CRIT)·수호반격(청색 방어 고리+방패음)·결전일섬(금빛 직선 빔+끝점 임팩트)
4. **창 4스킬** — ✅ 완료 (2026-05-30): 관통찌르기(청록 직선+CRIT)·반월창(청록 2겹 호+sweep)·돌파창(청록 궤적+구름)·천뢰일창(전격 직선+ELECTRIC_SPARK+천둥음)
5. **도끼 4스킬** — ✅ 완료 (2026-05-30): 철퇴강타(호박 호+CRIT)·파쇄돌진(호박 궤적+구름)·불굴자세(황금 방어 오라)·거신추락(지면 충격파 2중 고리+중앙 폭발)
6. **석궁·스태프** — ✅ 완료 (2026-05-30): 석궁 4종(녹색 볼트 빔)·스태프 4종(비전 보라 마법). 투사체 궤적은 신규 `BaseWeaponSkill.spawnBeam` 헬퍼(시선 방향 eye+pitch 빔, 솔리드 블록 정지)로 구현.

> **DL-070 전 스킬 이펙트 구현 완료 (2026-05-30): 24/24종.** 모든 무기 스킬이 `BaseWeaponSkill` 헬퍼 기반 `pt:` 파티클 + 사운드 보유. F 스킬은 전부 기존대로 논타겟(시선 기반) 유지.

> `mm:` / `dp:` 기반 아키텍처(`EffectDispatcher`, `MythicMobsEffectHandler`)는 2차 확장 시 설계한다.

# 1. 검

## 무기 컨셉

검은 가장 기본적이고 안정적인 근접 무기다.
공격, 기동, 방어, 마무리가 전부 있는 균형형 무기로 잡는다.

* 역할: 근접 균형 딜러
* 장점: 안정성, 쉬운 조작, 빠른 대응
* 단점: 특화 화력은 도끼나 낫보다 낮음
* 추천 베이스 아이템: `minecraft:netherite_sword`

## 자원: 검세 (소모형 최대 3스택 / 유지형 최대 6스택)

| 조건 | 효과 |
|---|---|
| 스택당 (공통) | 기본 공격 피해 +3% |
| 6스택 달성 (유지형 한정) | 수호반격 반격 계수 2배 (440%), 검세 유지 중 피해 감소 +15% |

> 소모형 만충(3스택): 기본공격 +9%, 섬광베기 160+24=184%, 결전일섬 320+45=365%.
> 유지형 만충(6스택): 기본공격 +18%, 섬광베기 208%, 결전일섬 392%, + 6스택 특수 효과.

복잡한 마나 관리 없이 쿨타임으로 굴리는 마법 무기로 잡는다.

* 역할: 마법 원거리 딜러
* 장점: 원거리, 속성 연출, 순간 폭발
* 단점: 몸 약함, 근접 대응 약함
* 추천 베이스 아이템: `minecraft:blaze_rod`

## 자원: 마력 충전 (소모형 최대 5스택 / 유지형 최대 6스택)

| 조건 | 효과 |
|---|---|
| 스택당 (공통) | 전체 스킬 피해 +4% |
| 6스택 달성 (유지형 한정) | 별빛쇄도(F) 즉발 시전 (캐스팅 없음), 범위 최대 (+3블럭) |

> 소모형 만충(5스택): 전체 스킬 +20%, 별빛쇄도 380+50=430%.
> 유지형 만충(6스택): 전체 스킬 +24%, 별빛쇄도 440%, + 즉발 시전·범위 최대.

## 스킬 구성

| 슬롯 | 키 | 스킬명 | 설명 | 쿨타임 | 계수 |
| -- | -- | ---- | ---- | --: | --: |
| 1 | LMB | 마력탄 | 마력 투사체를 발사한다. 명중 시 마력 충전 1스택. | 3초 | 150% |
| 2 | RMB | 속성폭발 | 착탄 지점에 즉발 폭발을 일으킨다. 명중 시 마력 충전 1스택. | 6초 | 240% |
| 3 | Shift+RMB | 마력쇄도 | 전방 반경 4블럭 즉발 광역 폭발. 발동 시 미세 후방 이동. 명중 시 마력 충전 1스택. | 11초 | 200% |
| 4 | F | 별빛쇄도 | 마력 소모(소모형) / 유지(유지형). 전방 강력한 마법 탄환. | 20초 | 380% (+10%/스택) |

## 각인 요약

| 각인 | 타입 | 핵심 효과 |
|---|---|---|
| 화염술사 | 소모형 | 마력 3스택+ F 시 추가 폭발 피해 +25% / 5스택 시 범위 +2블럭. F 소모 후 마력 1스택 즉시 복구. |
| 비전술사 | 유지형 | 마력 최대 6스택. F 소모 없음. 6스택 달성 시: 별빛쇄도 즉발 시전, 범위 최대. 마력쇄도 2명 이상 적중 시 마력 +1 추가 충전. |

> 수치 상세: `combat_balance_v2.md §5`

## 구현 메모

* 마력탄 3초 쿨타임 사이에 기본 공격(마력탄 자동, 0.9초) 발동.
* 비전술사(유지형)는 6스택 즉발 별빛쇄도 + 마력쇄도 광역 적중 보너스로 충전 가속.
* `속성폭발`은 착탄 후 즉발 폭발.
* `마력쇄도`는 burst hitbox 판정. 미세 후방 이동은 velocity 벡터로 처리 (대시 애니메이션 불필요).
* `별빛쇄도`는 큰 한 방 마법기로 사용한다.

# 무기별 한 줄 요약

* 검: 표준 근접 (방어/연속 분기)
* 도끼: 느린 한 방 (폭발/탱커 분기)
* 창: 거리 유지 + 직선 관통 (폭발/전 스킬 관통 분기)
* 석궁: 정밀 원거리 (소모 저격/유지 누적 분기)
* 낫: 넓은 베기 + 흡혈/처형 (폭발/유지 흡혈 분기)
* 스태프: 쿨타임 마법 원거리 (폭발/즉발 핵심기 분기)
