# 04. 전투 / 무기 / 스킬 — CANON

> **[STATUS: CANON]** — 6무기, 스킬 입력, 전투 수치, 직업각인, 레벨/스탯 성장의 공식 기준.

## 공식 기준

| 항목 | 기준 |
|---|---|
| 무기 6종 | 검 / 도끼 / 창 / 석궁 / 낫 / 스태프 |
| 스킬 입력 | LC / RC / Shift+RC / F |
| 공용각인 | 1차 시즌 제외 |
| 직업각인 | 무기별 A/B 2종, 고유 각인만 사용 |
| 유지형 자원 | 무기 무관 최대 6스택 |
| 금지 설계 | 지속 장판, 적 표식, 방어 감소, 받는 피해 증가, 파티 전체 피해 증가, 복잡한 스택 폭발 |
| 장비 슬롯 | 무기 / 투구 / 상의 / 하의 / 신발. 악세서리 제외 |
