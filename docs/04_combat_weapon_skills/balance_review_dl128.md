# 밸런스 종합 검토 (DL-128 시리즈 후속, 2026-06-03)

서브에이전트 3종(economy-reviewer, combat-balance ×2) 분석 종합. 현재 확정 수치(DL-128#11~13) 기준.

## 0. 최우선 발견 — 보스 DEF가 실효 0 (구조 문제)

- 플레이어→보스 **스킬 피해에 DEF 경감(`200/(200+DEF)`)이 코드 어디에도 적용 안 됨**. `BaseWeaponSkill.dealDamage`는 raw×general×boss×crit만 곱하고 `target.damage()` 호출.
- `boss_master.csv`에 **DEF 컬럼 자체가 없음**. `season_bosses.yml`에 Armor 속성 없음(바닐라 armor=0).
- `CombatFormulaResolver`의 step6 DEF 경감은 **참조 경로**(통계/웹)일 뿐 실데미지 미사용. `PlayerDefenseListener`의 DEF는 **몹→플레이어(방어)에만** 적용.
- **결론: 보스는 순수 HP 스펀지.** 문서상 "DEF 100/150…"은 무효. DL-128#3의 "DEF로 실효 HP 확보" 의도가 깨진 상태. → **모든 보스 HP 권장치의 전제 결정 필요**: ① dealDamage에 보스DEF 경감 추가 + boss_master DEF 컬럼 신설, ② 또는 DEF 폐기·HP만으로 밸런싱.

## 1. 강화 vs 큐브 골드당 효율 (economy)

- 강화 비용 테이블은 **무기·방어구 동일**. 골드당 스펙은 무기≈방어구. "무기만 올리면 된다"는 **ATK 마일스톤 점프(10·18·25강) 시각 집중** 때문이지 골드 효율 차이는 아님.
- 골드당 ATK 효율: 0~10강 ~315(×10⁻⁶) ≫ 10~18강 ~77 ≫ 18~25강 ~2.5(천장 반영 ~18). 저강화가 4배+ 효율(정상).
- **11강 절벽**: 10강 65%→11강 25%(기대골드 30.8k→86k). 첫 큰 벽.
- 큐브: 1회 500골드 고정(+큐브1). 골드로는 압도적으로 쌈 → **큐브는 골드가 아니라 큐브 재화로 게이트**. 재화 분리는 잘 됨.
- ★ **보스 보상에 골드 직접 드랍 없음**(강화석·큐브조각·흔적만). 골드 인입 경로 미확인 = **최대 리스크**(18강+ 전 골드 병목 가능).

## 2. DPS · 보스 HP (combat-balance)

석궁/검 솔로 DPS(무손실, 잠재 0): +0 ~155-170/s, +10 ~234-257/s, +18 ~377-415/s, +25 ~545-600/s.

| 보스 | 현 HP | +8 솔로 TTK | 판정 |
|---|---:|---:|---|
| fallen_knight | 8,000 | 30~47초 | **과소** — 패턴 페이즈 발동 전에 죽음 |
| 시즌보스 | 150k~170k | 10~17분 | 솔로 타임아웃(15분) 빠듯, **파티3인 3~5분 = 과소** |
| 필드보스 | 100k~660k | 솔로 10~65분 | 66만은 솔로 과함 |
| 최종보스 | 890k | 솔로 불가(+25도 25분>10분) | 의도(DL-038) |

- 시즌보스 HP가 150~170k로 **거의 평탄**한데 권장강화는 6→22강(ATK 1.5→3배) → **상위 보스가 상대적으로 더 쉬운 역전**. HP를 권장강화 ATK에 비례시켜야 함.
- 권장(DEF 무효 가정): fallen_knight 30k~35k(솔로 2~3분). 시즌1 90~110k → 시즌6 450~520k 곡선. 필드보스66만 솔로용이면 150~250k.
- 미반영 변수: 잠재/태그/만찬 피해증가(곱연산) → 상위권 DPS 1.5~2배, 일반↔상위 격차 최대 7배 가능. 별도 검증 필요.

## 3. 필드 45마리/분 (combat-balance)

`FieldSpawnService` 실값: 주기 15초, 웨이브 12마리(리필형), 플레이어캡 40, 필드캡 250, 스폰반경 **5~10블록**(주석 20~30과 불일치).

| 필드 | 지속공급 상한 | +8 킬레이트 | 병목 | 45/분 |
|---|---:|---:|---|---|
| F1(300HP) | ~48/분 | ~40/분 | 공급·킬 균형 | 거의 불가(40 부근) |
| F2(500) | ~48 | ~24 | **킬레이트** | 불가 |
| F3(800) | ~48 | ~15 | 킬레이트 | 불가 |
| F4(1200) | ~48 | ~10 | 킬레이트 | 불가 |
| F5(1800) | ~48 | ~7 | 킬레이트 | 불가 |

- **F1만 스폰율 병목**(고화력 시 12마리 8초컷 후 7초 대기 = 노는 시간). F2~F5는 **스폰 대기가 아니라 화력 부족** — 몹이 없어서 노는 게 아님.
- 즉 "45/분 사냥 = 스폰 대기?"의 답: **F1만 잠재적 스폰 대기, F2+는 애초에 킬레이트가 45 미달**이라 대기 자체가 안 생김.
- 제안: F1 주기 15→10초(또는 BATCH 12→16)로 리필 공백 제거. F2~F5는 45/분 목표를 필드별 차등(F1 45 / F3 20 / F5 10)이 정직. 스폰반경 주석/실값 불일치 확인. stray 정리(1초) 유예 확대.
