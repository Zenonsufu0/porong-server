# 04. 전투 / 무기 / 스킬

> **[STATUS: DRAFT]** — 망치→도끼 수정 완료 (DL-003, DL-004), 마력 과부하 메시지 제거 완료 (DL-002). `CANON.md` 참조.

## 입력 방식

무기를 들고 있을 때만 스킬 입력이 활성화된다.

| 입력 | 역할 |
|---|---|
| 좌클릭 | 기본기/주력기 |
| 우클릭 | 이동기/보조기 |
| Shift+우클릭 | 특수기/제어기/자기 강화 |
| F | 핵심기 |

## 금지 항목

- 플레이어 지속 장판
- 적 표식
- 받는 피해 증가
- 방어 감소
- 파티 전체 피해 증가
- 복잡한 표식 폭발

|---|---|---|---|

### 튜토리얼 무기 선택

최초 접속 시 튜토리얼 중 무기 선택창 GUI를 통해 클래스를 고른다.

- 튜토리얼 단계에서 GUI로 6종 무기 중 하나 선택
- 선택 즉시 무기 클래스 확정 + 기본 무기 지급 (네더라이트 외형)

### 캐릭터 GUI 무기 변경

장비 서브 허브 → 캐릭터(장비 패널) → 무기 장착 슬롯 클릭으로 변경.

- 변경 제한: **쿨타임 3분**, **전투 중 변경 불가** (골드/재료 비용 없음)

### 강화 해금 조건

강화는 별도 해금 없이 처음부터 사용 가능하다.

## 스킬 조회

각인 GUI는 `docs/08_resourcepack_pipeline/`에 현행 기준으로 재작성 필요하다. 슬롯에서 각인 A / B를 선택한다.

각 각인 아이콘 hover lore에 LC / RC / SRC / F 스킬 설명이 표시된다.

| 입력 | 역할 | lore 표시 내용 |
|---|---|---|
| LC (좌클릭) | 기본기/주력기 | 스킬명 + 효과 + 쿨타임 |
| RC (우클릭) | 이동기/보조기 | 스킬명 + 효과 + 쿨타임 |
| SRC (Shift+우클릭) | 특수기/자기강화 | 스킬명 + 효과 + 쿨타임 |
| F | 핵심기 | 스킬명 + 효과 + 쿨타임 |

## 무기 선택창 GUI

36칸(4행×9열). 튜토리얼 최초 선택과 캐릭터 GUI 무기 변경 두 경우 모두 재사용.  
컨텍스트에 따라 타이틀 문구와 뒤로가기 버튼 표시 여부만 분기.

```
행  | 0      | 1      | 2    | 3    | 4      | 5    | 6    | 7        | 8
----|--------|--------|------|------|--------|------|------|----------|------
 0  | DECO   | DECO   | DECO | DECO | DECO   | DECO | DECO | DECO     | DECO
 1  | DECO   | [검]   | DECO | DECO | [도끼] | DECO | DECO | [창]     | DECO
 2  | DECO   | [석궁] | DECO | DECO | [낫]   | DECO | DECO | [스태프] | DECO
 3  | [뒤로] | DECO   | DECO | DECO | DECO   | DECO | DECO | DECO     | DECO
```

- 6종 무기: 2행 × 3열(col 1·4·7) 균등 배치
- 빈 칸: `GRAY_STAINED_GLASS_PANE` (이름 없음)
- 뒤로가기: `ARROW` — 튜토리얼 컨텍스트에서는 표시 안 함

## 핫바 기본 구성

로그인 시 슬롯 7·8에 영구 아이템 자동 지급. 슬롯 0이 무기 기본 위치.

| 슬롯(코드) | 키 | 아이템 | 설명 |
|---|---|---|---|
| 0 | 1 | 장착 무기 | 기본 배치. 장비 패널 핫바 슬롯 선택으로 0~8 중 변경 가능 |
| 1~6 | 2~7 | 자유 슬롯 | 포션·소비 아이템 등 |
| 7 | 8 | 나침반 아이콘 (CMD) | 우클릭 → 탐험 서브 허브 직통 (필드 빠른 이동) |
| 8 | 9 | 메뉴 아이콘 (CMD) | 우클릭 → 메인 허브 54슬롯 PNG |

- 슬롯 7·8 아이템: persistent (버릴 수 없음). 로그인마다 재지급 체크.
- 무기 슬롯 잠금: 지정된 핫바 슬롯에 다른 아이템 이동 불가 (`InventoryClickEvent` 차단).
- 탐험 서브 허브 직통(나침반): 전투 중 차단, 그 외 어디서든 사용 가능.
- 메인 허브(메뉴): 전투 중 차단. 필드 내에서 열면 장비·영지 서브 버튼 비활성화.

## 튜토리얼 최초 접속 플로우

최초 접속 플래그(`tutorial_complete = false`) 확인 후 진행.

```
[첫 접속]
    │
    ▼
튜토리얼 스폰 지점 강제 텔포 (영지 생성 전 임시 구역)
    │
    ▼
채팅 안내: "포로 서버에 오신 것을 환영합니다. 무기를 선택해 여정을 시작하세요."
    │
    ▼
무기 선택창 GUI 자동 오픈 (36칸, 뒤로가기 없음)
    │  무기 선택 → 재확인 클릭
    ▼
클래스 확정 + 기본 장비 일괄 지급:
  - 0강 네더라이트 무기 (선택 클래스)
  - 0강 네더라이트 투구·흉갑·바지·신발
  - 메뉴 아이콘 (핫바 슬롯 8 고정)
  - 나침반 아이콘 (핫바 슬롯 7 고정)
  - 시작 골드 500G (DB)
    │
    ▼
채팅 안내:
  "무기를 선택했습니다."
  "9번 슬롯(메뉴)을 우클릭해 영지를 만들어 보세요."
  "8번 슬롯(나침반)으로 필드에 이동할 수 있습니다."
    │
    ▼
IridiumSkyblock 영지 자동 생성 트리거
    │
    ▼
영지 스폰 지점 텔포
    │
    ▼
tutorial_complete = true 저장
```

- 무기 선택 완료 전: GUI 강제 닫기 시 재오픈, 필드 텔포 차단.
- 재접속 시 `tutorial_complete = true` → 일반 접속 (영지 또는 마지막 위치 스폰).
