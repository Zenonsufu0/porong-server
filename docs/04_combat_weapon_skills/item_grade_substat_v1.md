## 1. 장비의 흔적 (드롭 아이템)

### 개념 정의

전투 중 몹·보스에서 드롭되는 장비형 아이템. 완제품이 아닌 "흔적(remnant)" 형태이며,
드롭 시 등급과 보조스텟이 자동으로 결정된 채 획득된다.

- **장착 가능:** 주스텟 + 보조스텟이 적용된 상태로 착용 가능.
- **강화 불가:** 강화는 시작 5슬롯 메인 장비 전용. 흔적은 강화 대상 아님.
- **전승 소스 사용 가능:** 흔적(A)의 등급·보조스텟을 강화된 메인 장비(B)로 이전 가능.
- **경매소 거래 가능:** 고등급 흔적은 경매소에서 고가로 판매 가능.

### 경제 루프 설계 의도

| 유저 유형 | 흔적 활용 방식 |
|---|---|
| 일반 유저 | 커먼~레어 흔적 → 착용 또는 저가 판매 |
| 선발대 하드 | 경매소에서 에픽~레전더리 흔적 구매 → 전승 소스로 메인 장비 최적화 |
| 운 좋은 파머 | 희귀 등급 흔적 드롭 → 경매소 고가 판매 → 골드로 강화 투자 |

레전더리 흔적은 드랍 확률이 극도로 낮아 경매소 시세가 높게 형성됨.
강화석 경매 루프(초반 유저 판매 ↔ 선발대 구매)와 독립적으로 작동하는 별도 아이템 경매 루프 형성.

### 슬롯 종류 (5종)

| 흔적 종류 | 연결 부위 |
|---|---|
| 무기의 흔적 | 무기 슬롯 |
| 투구의 흔적 | 투구 슬롯 |
| 상의의 흔적 | 상의 슬롯 |
| 하의의 흔적 | 하의 슬롯 |
| 신발의 흔적 | 신발 슬롯 |

- 드롭 시 슬롯은 **랜덤** 결정 (어떤 몹에서도 5종 중 하나가 랜덤 드롭).
- 같은 슬롯끼리만 전승 가능 (무기의 흔적 → 무기 메인, 투구의 흔적 → 투구 메인).

## 2. 아이템 등급 시스템

### 등급 체계

잠재능력 등급과 동일한 5단계를 사용한다.

| 등급 | 주스텟 보정 | 보조스텟 보정 |
|---|---:|---:|
| 커먼 | 기준 (0%) | 기준 (0%) |
| 레어 | +2% | +2% |
| 에픽 | +4% | +4% |
| 유니크 | +6% | +6% |
| 레전더리 | +8% | +8% |

- 등급 간 차이: 2% per tier.
- 보정은 해당 슬롯 주스텟 기본 수치에 곱한다.
  - 예: 투구 기본 HP 400, 레전더리 흔적 → 400 x 1.08 = 432 HP.
- 보조스텟도 동일 보정률 적용.

### 등급이 영향을 미치는 항목

| 항목 | 영향 여부 | 비고 |
|---|---|---|
| 슬롯별 주스텟 수치 | O | 기본 수치 x 등급 보정 |
| 보조스텟 수치 | O | 동일 보정률 적용 |
| 강화 스탯 증가량 | O | 기본 수치가 높으면 강화 증가량도 높아짐 (메인 장비 기준) |
| 잠재능력 옵션 품질 | X | 잠재는 큐브 롤로 독립 결정 |
| 강화 성공률 | X | 등급 무관 동일 확률 |

### 필드별 최대 등급 제한 (확정 — 2026-05-14)

- **레전더리 흔적:** 필드4 (무너진 초소) 이상에서만 드롭. 필드1~3 정예몹·필드보스에서는 레전더리 불가.
- **드롭 확률 순서:** 필드4 정예 < 필드4 필드보스 = 필드5 정예 < 필드5 필드보스 < 시즌보스 < 시즌 최종보스

### 장비의 흔적 드롭 등급 확률 (기본값 — 운영 중 조정 가능)

| 드롭 출처 | 커먼 | 레어 | 에픽 | 유니크 | 레전더리 |
|---|---:|---:|---:|---:|---:|
| 일반몹 | — | — | — | — | — |
| 정예몹 — 필드1~3 | 60% | 35% | 5% | 0% | 0% |
| 정예몹 — 필드4 | 52% | 35% | 10% | 2.5% | **0.5%** |
| 정예몹 — 필드5 | 40% | 35% | 18% | 5% | **2%** |
| 필드보스 S/A — 필드1~3 | 25% | 45% | 25% | 5% | 0% |
| 필드보스 S/A — 필드4 | 20% | 35% | 30% | 13% | **2%** |
| 필드보스 S/A — 필드5 | 10% | 25% | 35% | 25% | **5%** |
| 필드보스 B/C — 전 필드 | 50% | 40% | 10% | 0% | 0% |
| 시즌보스 1~3 | 5% | 25% | 45% | 23% | 2% |
| 시즌 최종보스 (균열왕) | 0% | 10% | 35% | 40% | 15% |

일반몹: 장비의 흔적 드롭 없음. 골드와 필드 재료만 드롭.
레전더리 흔적: 필드4+ 정예/필드보스 및 시즌보스에서만 드롭. 45일 시즌 내 자연 드롭은 희귀한 성취.

## 3. 보조스텟 시스템

### 부여 방식

- **부여 시점:** 장비의 흔적 드롭 시 자동으로 랜덤 부여. 큐브 불필요.
- **옵션 풀:** 해당 슬롯의 잠재능력 전체 풀을 따른다.
  - 무기의 흔적 드롭 → 무기 잠재능력 풀에서 랜덤 보조스텟 부여
  - 투구의 흔적 드롭 → 투구 잠재능력 풀에서 랜덤 보조스텟 부여
  - 나머지 슬롯 동일
- **수치:** 잠재능력 3라인 수치 대비 낮게 설정 (보조스텟이 잠재를 대체하면 안 됨).
  - 보조스텟 1라인 수치 = 잠재 1라인 수치의 약 50~70% 수준.
- **라인 수:** 드롭 시 0~3라인 랜덤. 등급 높을수록 평균 라인 수 높음.

### 드롭 등급별 보조스텟 라인 수 분포

| 등급 | 0라인 | 1라인 | 2라인 | 3라인 |
|---|---:|---:|---:|---:|
| 커먼 | 60% | 30% | 10% | 0% |
| 레어 | 30% | 40% | 25% | 5% |
| 에픽 | 10% | 30% | 40% | 20% |
| 유니크 | 0% | 15% | 40% | 45% |
| 레전더리 | 0% | 5% | 30% | 65% |

### 파밍 전략 연계

보조스텟은 강화 수치를 유지한 채 전승으로 이전 가능하므로, 선발대 하드 유저의 목표가 생긴다:
- "높은 강화 수치" + "좋은 잠재 라인" + "좋은 보조스텟"의 조합이 최적 아이템.
- 보스 딜 체크나 최종보스 공략을 위해 치명타 특화 조합(치명 100% + 보조스텟 치명타) 같은 극최적화가 가능.
- 일반 유저는 보조스텟 신경 안 써도 진행에 문제 없음.

## 4. 전승 시스템

### 기능 정의

장비의 흔적(A)의 등급·보조스텟을 현재 강화된 메인 장비(B)로 이전한다.

| 항목 | 기본 전승 | 보조스텟만 전승권 | 등급만 전승권 |
|---|---|---|---|
| 보조스텟 (라인·수치) | O 이전 | O 이전 | X 유지 |
| 아이템 등급 | O 이전 | X 유지 | O 이전 |
| 강화 단계 | X 유지 | X 유지 | X 유지 |
| 잠재능력 라인 | X 유지 | X 유지 | X 유지 |

- **기본 전승:** 흔적(A) + 골드 소모 → 등급 + 보조스텟 모두 이전. A 소멸.
- **보조스텟만 전승권:** 별도 구매 아이템. 등급 이전 없이 보조스텟만 이전.
- **등급만 전승권:** 별도 구매 아이템. 보조스텟 이전 없이 등급만 이전.

### 전승 시나리오 예시

```
[시나리오 1 - 기본 전승]
흔적 A : 투구의 흔적, 커먼 등급, 보조스텟 치명타 +8% (3라인)
메인 B : 투구 18강, 커먼 등급, 보조스텟 없음

기본 전승 실행 후:
메인 B : 투구 18강, 커먼 등급, 보조스텟 치명타 +8% (3라인)
(흔적 A 소멸)

[시나리오 2 - 등급만 전승권 사용]
흔적 C : 투구의 흔적, 레전더리 등급, 보조스텟 치명타 +12% (3라인)
메인 B : 투구 18강, 커먼 등급, 보조스텟 치명타 +8% (3라인) [시나리오 1 결과]

등급만 전승권 사용 후:
메인 B : 투구 18강, 레전더리 등급, 보조스텟 치명타 +8% (3라인) 유지
(흔적 C 소멸)
```

### 전승권 비용 (미확정)

| 종류 | 이전 항목 | 비용 |
|---|---|---|
| 기본 전승 | 등급 + 보조스텟 모두 | 골드 소모 (흔적 소멸) |
| 보조스텟만 전승권 | 보조스텟만 | TBD G (별도 구매) |
| 등급만 전승권 | 등급만 | TBD G (별도 구매) |

- 전승권은 기본 전승에 추가하는 선택 아이템 (선택적 부분 이전 원할 때 사용).
- 전승 시 흔적(A)은 항상 소멸. 전승 횟수 제한 없음.
- 비용은 오픈 후 경매소 흔적 시세 확인 후 확정.

## 5. 시스템 간 관계 요약

```
장비의 흔적 드롭 (정예몹 5% / 필드보스 S등 확률 / 시즌보스 확정)
  ├─ 아이템 등급 (커먼~레전더리, 2% 보정) — 랜덤 결정
  ├─ 보조스텟 (잠재능력 전체 풀, 0~3라인 랜덤)
  └─ 용도 선택
       ├─ 직접 장착 (강화 불가, 등급·보조스텟 스탯 즉시 적용)
       ├─ 기본 전승 → 메인 장비에 등급+보조스텟 이전 (흔적 소멸)
       ├─ 전승권 사용 → 선택적 이전 (보조스텟만 OR 등급만)
       └─ 경매소 판매 (에픽+ 흔적은 고가)

메인 장비 5슬롯 (시작 시 지급)
  └─ 강화 (0강~20강) → 주스텟 증가
       └─ 흔적 전승으로 등급·보조스텟 이식 가능 (강화 단계 유지)

큐브 사용 (별도 시스템)
  └─ 잠재능력 3라인 롤 (잠재 등급 커먼~레전더리)
       └─ 흔적 등급·보조스텟과 독립 — 세 가지 모두 높으면 최적 아이템
```
