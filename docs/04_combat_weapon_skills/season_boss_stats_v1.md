## 1. 확정 원칙 및 설계 방향

### 1.1 핵심 확정 사항

- 전 시즌보스 최대 파티 3인. 1인·2인 입장도 가능.
- 난이도는 HP 덩어리가 아닌 패턴 구조로 만든다. 무적 구간 / 쉴드 구간 / 공략 메카닉이 전투 시간을 채운다.
- TTK 기준은 3인 파티 기준 실 딜 가능 시간 × DPS로 역산한다.
  - 시즌보스 1~6: 전투 시간 15분. 무적/공략 구간 합계 3~7분. 실딜 시간 8~12분.
  - 균열왕: 타임아웃 10분. HP 890,000. 무적 구간 없음 — 패턴 실패 시 HP 회복 메카닉으로 난이도 구현 (2026-05-16 확정)
- **방어력 무시 잠재 메타 (재확정 2026-05-24)**: HP와 DEF 모두 우상향 설계. 보스4부터 방어력 무시 17%(유니크) 없으면 타임아웃 실패. 보스6은 레전더리 수준(30%) 방무가 사실상 필요. 취약 창 활용만으로는 보정 불가 — 방무가 실질 필수 슬롯.

### 1.2 피해 공식 (변경 없음)

```
유효 방어율 = DEF / (DEF + 200)
기본 피해   = ATK × (1 - 유효 방어율)
           = ATK × 200 / (DEF + 200)
스킬 피해   = 기본 피해 × 스킬 배율
```

## 2. 인원별 스케일링 공식

### 2.1 설계 원칙

인원 감소 시 HP와 DEF를 낮춰 클리어 가능성을 열되, 1인·2인은 패턴 부담이 높아 실질 난이도는 떨어지지 않는다.
드랍 확률은 총 보상량을 3인=1.0×로 기준 삼되, 인원 감소 시 개인 보상은 늘려 솔로 도전에 인센티브를 부여한다.

### 2.2 스케일링 표

| 인원 | HP 배율 | DEF 조정 | 드랍 배율(총량) | 드랍 배율(개인) |
|---:|---:|---|---:|---:|
| 1인 | 0.45× | 기준 DEF에서 -15% (정수 반내림) | 0.55× | 0.55× (전량 1인 수령) |
| 2인 | 0.72× | 기준 DEF에서 -8% (정수 반내림) | 0.78× | 0.39×/인 |
| 3인 | 1.00× | 기준 DEF (변동 없음) | 1.00× | 0.33×/인 |

- HP 배율 설계 근거: 1인 효율 65%, 3인 효율 85%. 1인 솔로 유효 DPS는 3인 파티의 (65%)/(85%×3) = 25.5%. HP 0.45×면 TTK가 파티보다 약 1.75배 늘어나는 수준으로, 도주 타이머 압박이 실질 난이도를 만든다.
- DEF 조정 근거: 솔로 플레이어는 회피에 집중해야 하므로 유효 피해가 아주 낮으면 플레이 자체가 불쾌해진다. 소폭 DEF 감소로 "피해가 들어가는 체감"을 유지한다. DEF 격차가 크지 않아 고강화 유저의 스케일 혜택을 과도하게 주지 않는다.
- 드랍 총량 1인 < 3인: 솔로로 여러 번 공략해도 파티 3인 총 보상을 초과하지 않도록 설계한다. 파밍 효율은 3인 파티가 우위.

### 3.2 단인 유효 DPS 역산

> v2 비선형 ATK 기준으로 전면 수정 (2026-05-17). DPS는 v1 수치에 ATK 변화 비율을 곱해 산출. 이 수식 기반 DPS는 실제 스킬 사이클 DPS(`docs/_archive/dps_analysis_v1.md`: 단인 ~17,000~21,000/분, archive 이동됨)보다 크게 낮으므로 절대값이 아닌 **보스간 상대적 난이도 비교용**으로만 참고할 것.

스킬 포함 분당 유효 DPS (3인 기준 DEF 환경, ATK는 v2 비선형 기준값):

```
기본 피해 = ATK × 관통율 (= 200 / (DEF + 200))
스킬 가중 = 기본 피해 × 1.35 (스킬 배율)
DPS/분    = 스킬 가중 × 57 (분당 히트) × (1 + 각인%) × (1 + 잠재%)
```

잠재%는 무기 슬롯 단독 기대 기여율 (1차 시즌 장갑 슬롯 없음).

## 5. 보스별 패턴 구조 설계

패턴 구조가 전투 시간을 만든다. 시즌보스 1~6은 무적 구간 + 공략 메카닉으로 15분의 전투가 완성된다. 균열왕은 무적 구간 없이 HP 자체가 10분을 채우며, 패턴 실패 시 HP 회복으로 타임아웃을 초과하게 만드는 구조다.  
패턴 상세(공용 P-0X 조합, 고유 SP-XX 파라미터): `../07_boss_pattern_modules/season_boss_patterns.md` 참조.

### 5.1 시즌보스 1 — 타락 기사장 (입문형)

**설계 방향**: 패턴 회피를 배우는 첫 보스. 공략 메카닉 실패 패널티가 있지만 치명적이지 않다. 솔로 20강 유저가 도전 가능한 마지노선.

**Phase 구조**

| Phase | HP 기준 | 지속 시간 | 특징 |
|---:|---|---:|---|
| 1 | 100% ~ 60% | 약 4~5분 | 기본 패턴 (광역 스탬프, 돌진) |
| 전환 1 | 60% 도달 시 | 2분 | 무적 + 쉴드 구간 #1 |
| 2 | 60% ~ 30% | 약 3~4분 | 분노 패턴 추가 (연속 스탬프, 방향 가시) |
| 전환 2 | 30% 도달 시 | 2분 | 무적 + 쉴드 구간 #2 |
| 3 | 30% ~ 0% | 약 2~3분 | 전방위 폭발 추가, 이동 속도 증가 |

**무적 구간 (Phase 전환 시 2회, 각 2분)**

- 보스가 공중으로 부상하며 무적 상태 진입
- 전환 #1: 지면에 에너지 석판 4개 등장. 파티원이 석판을 밟아 기를 흡수하면 무적 해제. 3인 파티는 석판 위치를 나눠 밟아야 효율적. 1인 솔로는 석판 4개를 순서대로 밟아야 하므로 시간이 더 걸린다 (설계상 1분 30초~2분 소요).
- 전환 #2: 보스 주위에 석판 대신 유도 에너지 구체 4개 등장. 구체를 근접 공격으로 격파해야 무적 해제. 격파하지 않으면 구체가 파티원을 추적하며 피해를 준다.
- 무적 해제 성공 시: 보스가 3초간 기절 + 받는 피해 +25% (취약 창)
- 무적 해제 실패 (2분 초과): 전환 패턴이 강제 종료되고 보스가 복귀, 분노 상태로 다음 Phase 진입

**도주/분노 타이머**

- 전투 시간 15분 초과: 도주 타이머 발동. 보스가 점차 이동속도 증가 + 도주 준비 애니메이션. 16분 시 도주 완료, 전투 강제 종료, 보상 없음.
- 14분 이후 생존 시 보스 분노 강화: 기본 공격 속도 +20%, 스탬프 범위 +30%.

**핵심 메카닉**
- 방향 가시: Phase 2에서 보스 주변 6방향으로 가시 방출. 가시 패턴 직전 보스 몸에서 빛 방향 예고 표시. 예고 반대쪽으로 이동하면 회피.
- 연속 스탬프: 보스가 2블럭 이상 이동 후 착지 시 착지 지점 범위 피해. 착지 직전 그림자 경고 표시.

- 무적 해제 실패: 보스가 전기 폭발로 복귀 (범위 피해), 분노 상태 진입

- 전투 시간 15분 초과: 도주 타이머. 16분 시 도주 완료.
- 12분 이후: 보스 전기장 상시 활성화 (전체 필드에 주기적 피해 발생).

**핵심 메카닉**
- 직선 관통 충격파: 보스 시선 방향으로 선형 경고 3초 후 관통 충격파 발사. 세로 방향으로 피신.

**Phase 전환 이벤트 (무적 없음, 각 약 5~8초)**

- 70% 전환: 보스가 전장 전체에 에너지 폭발 발동. 5초 내 보스로부터 6블럭 이상 거리 확보 필수. 실패 시 즉사 수준 피해. 이동기(RMB) 활용이 사실상 필수.
- 40% 전환: 보스가 분신 2개 소환. 분신이 살아있는 동안 보스 받는 피해 -70% 적용. 분신과 보스에게 동시 딜 가능 (무적 아님).

**분신 소환 메카닉 (Phase 3 진입 시)**

- 분신 HP: 각 25,000 (합계 50,000). 보스 HP와 별개.
- 분신이 살아있는 동안 보스 피해 감소 -70% 적용.

- 분신 처치 완료 시: 보스 기절 3초 + 받는 피해 +40% (취약 창).

**도주/분노 타이머**

- 전투 시간 10분: 강제 종료. 전장 붕괴 폭발로 파티 전멸, 보상 없음.

**핵심 메카닉**

- 추적 충격파 (Phase 1~3): 보스가 가장 가까운 유저를 향해 충격파 발사 (쿨타임 3초). 이동기(RMB) 타이밍 회피 필수.
