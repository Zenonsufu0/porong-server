## 0. 설계 전제

- 레벨 상한 **100레벨**. 표면상 제한 없음, 도달은 사실상 불가 설계.
- 레벨당 스탯 포인트 **3pt** 자유 배분. 3가지 트리 중 어디든 원하는 만큼.
- **자유 재배분**: **비전투 중**에 포인트를 재분배할 수 있다. (전투 중 변경 불가)
- 레벨링 주 수급: **필드 몹 사냥**. 필드 단계가 높을수록 경험치 수급량 증가.
- 레벨 자체는 콘텐츠 진입 조건이 아님. 오직 스탯 투자 보상만 존재.
- **레벨별 고정 스탯 자동 증가 없음** (HP/ATK/DEF는 장비 강화로만 성장). 확정.

## 3. 치명타 100% 달성 설계 기준

> 2026-05-16 재설계. 스탯 +0.30%/pt, 잠재 수치 확정 반영.

치명 빌드의 핵심 목표치. **스탯 + 잠재능력 + 보조스탯** 세 축을 모두 활용해야 달성.

### 치명타 확률% 수급원 (확정)

| 수급원 | 레어 | 레전더리 |
|---|---:|---:|
| 기본값 | 5% | 5% |
| 스탯 치명 (레벨 50 all-in, 150pt) | +45% | +45% |
| 잠재능력 치명확률% 1라인 | +10% | +25% |
| 보조스탯 치명확률% 1라인 (잠재 50~70%) | +5~7% | +12~18% |

### 100% 달성 시나리오

| 시나리오 | 조건 | 합산 |
|---|---|---|
| **레벨 80 + 레전더리 잠재** | 치명 240pt (+72%) + 레전더리 치명확률 (+25%) | 5 + 72 + 25 = **102%** ✓ |
| **레벨 65 + 레전더리 잠재 + 보조스탯 1슬롯** | 치명 195pt (+58.5%) + 레전더리 (+25%) + 보조스탯 (+15%) | 5 + 58.5 + 25 + 15 = **103.5%** ✓ |
| **레벨 50 + 레전더리 잠재 + 보조스탯 2슬롯** | 치명 150pt (+45%) + 레전더리 (+25%) + 보조스탯 (+30%) | 5 + 45 + 25 + 30 = **105%** ✓ |

**해석:**
- **레벨 50**: 레전더리 잠재 + 보조스탯 2슬롯 모두 치명확률% 확보 시 100% 달성. 경직된 조합.
- **레벨 65**: 레전더리 잠재 + 보조스탯 1슬롯으로 달성 가능. 빌드 유연성 확보.
- **레벨 80**: 잠재 1개만으로 100% 달성. 보조스탯은 치명피해량% 등 다른 옵션에 투자 가능.
- **레어 잠재만으로는 100% 불가** — 레벨 80 all-in + 레어 (+10%): 5+72+10 = 87%. 의도적 설계.

### 레벨별 치명 all-in 시 치명타 확률

| 레벨 | 총 포인트 | 스탯만 | 레전더리 잠재 포함 | 레어 잠재 포함 |
|---:|---:|---:|---:|---:|
| 30 | 90pt | 5 + 27 = **32%** | +25 = 57% | +10 = 42% |
| 50 | 150pt | 5 + 45 = **50%** | +25 = 75% | +10 = 60% |
| 65 | 195pt | 5 + 58.5 = **63.5%** | +25 = 88.5% | +10 = 73.5% |

치명 100% 달성은 레전더리 잠재 없이 불가. 레전더리가 "잠금 해제" 역할을 함.

## 5. 레벨링 경험치 구조

> 2026-05-16 확정

- **주 수급원**: 필드 몹 사냥 (골드/강화석 파밍 시 자동 수급 병행)
- 필드 단계가 높을수록 경험치 증가
- 보스 처치 시 보너스 경험치
- 경험치 요구량은 후반으로 갈수록 급증

스탯 트리 효과는 잠재능력과 동일 항목이 있으면 합산 적용. 강제 캡 없음 (치명타 확률 100% 초과분은 자동 무효).

### 잠재능력 수치 (확정)

> 전체 수치 확정 2026-05-16. 상세 슬롯별 풀 → `item_grade_substat_v1.md` 참조.

| 옵션 | 레어 1라인 | 레전더리 1라인 | 적용 슬롯 |
|---|---:|---:|---|
| 치명타 확률% | **+10%** | **+25%** | 무기 / 머리 |
| 치명타 피해량% | **+10%** | **+20%** | 무기 |
| 스킬 피해% | **+8%** | **+15%** | 무기 / 상의 |
| 핵심기 피해% (F) | **+8%** | **+20%** | 전 슬롯 |
| 이동기 피해% (RMB) | **+8%** | **+20%** | 전 슬롯 |
| 기본기 피해% (LMB) | **+8%** | **+20%** | 전 슬롯 |
| 특수기 피해% (Shift) | **+8%** | **+20%** | 전 슬롯 |
| HP% | **+4%** | **+10%** | 머리 / 상의 / 신발 |
| DEF% | **+4%** | **+10%** | 상의 / 하의 / 신발 |
| 받는 피해 감소% | — | **+6%** | 머리 / 상의 / 하의 (유니크+) |

> "태그 피해%"는 적 종류 태그가 아닌 **스킬 입력 타입 4종**으로 분리. 잠재 라인 부여 시 4종 중 1종이 랜덤 배정.

### 최강 치명 조합

잠재 3라인 최적 조합: **치명타 확률% + 치명타 피해량% + 스킬 피해%**

이 세 옵션이 서로 다른 배율 층(크리 확률 / 크리 배율 / 스킬 배율)에서 곱산되어 단일 옵션 중복 대비 효율이 가장 높음.

| 스탯 트리 항목 | 잠재능력 동일 항목 존재 여부 |
|---|---|
| 치명타 확률% | ✓ 무기 / 머리 |
| 치명타 피해량% | ✓ 무기 |
| 스킬 피해% | ✓ 무기 / 상의 |
| 핵심기/이동기/기본기/특수기 피해% | ✓ 전 슬롯 (4종 독립 옵션) |
| 받는 피해 감소% | ✓ 머리 / 상의 / 하의 (유니크+) |
| 방어력 | — |

치명 100% 빌드는 **스탯 치명 + 레전더리 잠재 + 보조스탯** 세 축 동시 투자 필수. 레전더리 없이는 레벨 80+ all-in으로도 87% 상한.

## 7. 피해 감소 스택 공식 (확정 2026-05-16)

받는 피해 감소%와 DEF 방어율은 **곱산(독립 감소)** 적용.

```
실제 받는 피해 = 공격자 피해 × (1 - DEF 방어율) × (1 - 받는 피해 감소%)
DEF 방어율 = DEF / (DEF + 200)
```

예시 (인내 all-in 150pt + 0강 DEF 52):
- DEF 방어율: 112 / 312 = 35.9%
- 받는 피해 감소: 22.5%
- 실질 받는 피해: (1 - 0.359) × (1 - 0.225) = 0.641 × 0.775 = **49.7%** (총 50.3% 감소)
