## 3. 피해 계산 공식

### 기본 공식

```
유효 방어율 = DEF / (DEF + 200)
기본 피해   = ATK × (1 − 유효 방어율)
스킬 피해   = 기본 피해 × 스킬 배율
```

**수치 예시:**

| ATK | DEF | 유효 방어율 | 기본 피해 |
|---:|---:|---:|---:|
| 80 | 0 | 0% | 80 |
| 80 | 60 | 23.1% | 61.5 |
| 80 | 100 | 33.3% | 53.3 |
| 80 | 200 | 50.0% | 40.0 |
| 80 | 400 | 66.7% | 26.7 |
| 120 | 60 | 23.1% | 92.3 |
| 200 | 60 | 23.1% | 153.8 |

> DEF + 200 공식 → 방어 100% 불가능. T1 풀세트 DEF 60 기준 감소율 약 23%.  
> 스킬 배율은 정수 또는 소수 1자리 이하로 제한.

### 강화 스탯 증가 (선형)

```
강화 ATK 증가 = 기본 ATK × 0.04 × 강화 단계
강화 DEF 증가 = 기본 DEF × 0.04 × 강화 단계
강화 HP  증가 = 기본 HP  × 0.04 × 강화 단계
```

**무기 ATK 증가 예시 (기본 80):**

| 강화 단계 | 총 ATK |
|---:|---:|
| 0강 | 80 |
| 5강 | 96 |
| 10강 | 112 |
| 15강 | 128 |
| 18강 | 138 |
| 20강 | 144 |

> ⚠️ 20강 ATK 144 = 0강 대비 +80%. 필드보스 DEF도 함께 스케일링하지 않으면 후반 보스 체감이 무너짐 (위험요소 1 참조).

## 6. 필드보스 HP 기준선

**DPS 가정:** 0강 풀세트 기준 스킬 포함 분당 유효 피해 약 2,500~3,500. 10강 기준 약 3,500~5,000.

| 필드 | 보스 이름 | 권장 강화 구간 | 보스 DEF | HP 범위 | 파티 3인 예상 TTK |
|---|---|---|---:|---|---|
| 수도 외곽 평원 | 들판 포식자 | 0강~5강 | 30 | 90,000~110,000 | 10~14분 |
| 폐광 지대 | 폐광 골렘 | 5강~10강 | 80 | 180,000~220,000 | 11~15분 |
| 오염된 수로 | 수로 군주 | 8강~13강 | 120 | 280,000~340,000 | 11~15분 |
| 무너진 초소 | 타락 기사장 | 12강~16강 | 160 | 420,000~500,000 | 12~16분 |
| 고대 성벽 잔해 | 균열 파수꾼 | 15강~18강 | 200 | 600,000~720,000 | 12~16분 |

> 보스별 DEF 스케일링 적용 → 강화 ATK 증가에도 보스 DEF가 따라오므로 후반 필드 체감이 유지됨.  
> 오염된 수로: HP보다 패턴 복잡도로 난이도 조절. 수치는 폐광 골렘 대비 약 1.4배.

## 7. 위험요소 및 조정 제안

| # | 위험요소 | 조정 제안 |
|---|---|---|
| W-1 | 강화 선형 증가 시 보스 DEF 의미 희석 | 필드단계별 보스 DEF 스케일링 적용 (위 표 기준) |
| W-2 | 낫 배후 판정 불안정 | "회피기 후 2초 내 공격 시 충전"으로 단순화 |

| W-4 | 기계궁수 — 강제 이동 패턴 보스에서 사실상 무력화 | "정적 상태에서 DEF +10% 추가" 조건 병기 → 탱킹+화력 복합 스타일 |
| W-5 | 강화 재료 수급과 45일 사이클 미스매치 | 일 2~3h 기준 강화석 자연 획득량 → 18강 도달 기대 30~35일 역산 후 드랍률 설계 |
