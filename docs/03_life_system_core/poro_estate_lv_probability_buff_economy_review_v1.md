# 영지 Lv 연동 확률형 버프 경제 영향 검토 v1

> **상태**: 2026-04-19 사용자 **축소 도입 + 보너스 귀속 확정**. 가공·연금·요리 한정, Lv3=1% / Lv4=2% / Lv5=3%, T1 장비·T2 전 라인·강화석 제작대 제외. **보너스 수량 = 귀속 O(경매 등록 불가)**. 후속: 성장 재료 개별 제외 / 시즌1 45일차 경매 모니터링 대시보드(data-schema).

## 문서 목적
`poro_allinone_workbench_draft.md` 오픈 질문 #7 — **영지 Lv 연동 확률형 버프**("소량 추가 생산 5% 확률 +1") 도입 여부에 대한 economy-reviewer 1차 검토. 2026-04-19 사용자 결정("경제 영향 검토 후 문제 없으면 도입")에 따른 후속 자문. CLAUDE.md `장점 → 위험요소 → 조정 제안` 흐름을 따른다.

## 배경 요약
- **검토 대상**: 올인원 조합대에서 제작 시 영지 Lv이 높을수록 확률로 추가 산출물을 주는 메커니즘.
- **제안 수치 기본값**: Lv당 +1% 씩 증가, Lv5에서 5% 확률 +1개.
- **전제 1**: 강화 25강·파괴 없음·결정적 파밍 구조.
- **전제 2**: NPC 효과는 이미 "생산속도 증가 단일"로 통일(2026-04-19). 희귀 확률 부스트는 폐기된 상태.
- **전제 3**: 배치 제작 상한도 Lv별 차등으로 Lv5가 이미 Lv1 대비 큰 배수를 가진다.

## 1. 장점 (도입 시 얻는 것)

- **고 Lv 영지의 투자 회수감 보강**: Lv4→Lv5 업그레이드 비용(400만 골드 + T2 상위 코어 50 + 아르카논 심층 콘텐츠)은 매우 무거운데, 이를 체감할 포인트가 "공간·전력 번들 + 배치 상한" 외에 하나 더 필요하다는 근거는 있다.
- **"영지에 들러 제작한다"는 동선 강화**: 오프라인 누적이 아닌 온라인 수동 제작(특히 T2 수동 확정 구간)에서 작은 양의 기대치 보상이 생기면 "지금 접속해서 돌리자"는 인센티브로 작용한다.
- **확률이 낮고 단위가 작으면 체감 노이즈 이하**: Lv5에서도 5%/+1 수량은 평균적으로 배치 100개당 5개 추가 수준이라, 도파민 장치로는 약하되 경제 교란 요소로도 약한 구간이다.
- **기획 서사적 일관성**: "Lv이 높은 영지 = 숙련된 장인 라인"이라는 플레이버가 자연스럽다.

## 2. 위험 요소

### 2-1. 결정적 파밍 구조와의 충돌
- **NPC 효과를 속도 단일로 통일한 바로 그 결정과 역행**한다. 2026-04-19 회의에서 "희귀 확률 부스트·연료 절감·실패율 감소 등 종류별 특수 효과 폐기"가 명문화됐다. 확률형 추가 생산은 명목만 "수량"이지 메커니즘은 "희귀 확률 부스트"와 같은 클래스다.
- **확률 요소가 이미 여러 레이어에 분포**: 드랍, 잠재 옵션, 마력 채집기의 심층 결정 랜덤 1회/일 기믹, 일부 채집기 NPC(폐기된 구 설계이지만 흔적 남음) 등. 여기서 한 축을 더 얹으면 "어디까지가 결정적이냐"는 경계가 흐려진다.

### 2-2. 인플레이션 리스크 (시즌 누적 기준)
- Lv5 영지 기준 5% +1은 **재료 공급량 ≈ +5%** 이동.
- 문제는 **Lv5 자체가 시즌 후반 상위권의 선점 프리미엄**이라는 점. 상위권 소수 인원이 경매·시세를 주도하는 구조에서, 상위권만 누리는 +5% 공급은 T1 강화석·가공 주괴·T2 중간재 시세에 **지속적인 하방 압력**을 만든다.
- 시즌 내내 누적되면 평균 보유량 기준으로는 미미해도, **중간재 경매 최저가**는 누가 가장 싸게 내놓느냐로 결정되므로 체감 하락폭이 산술치(+5%)보다 커진다.

### 2-3. Lv1~Lv5 격차 확대 (이중 보상)
- 배치 제작 상한이 이미 Lv별 차등이다. 가상 배수를 Lv1=1x / Lv3=2x / Lv5=3x 수준으로 가정해도, 여기에 확률 버프를 얹으면 **Lv5 시간당 산출 / Lv1 시간당 산출 ≈ 3x × 1.05 ≈ 3.15x**.
- 개별 승수는 작지만 "공간 + 전력 + 배치 상한 + 확률" 4중 보상이 되면 **Lv4→Lv5 업그레이드 유인 과열 → 일반 유저의 최종선인 Lv3~4 근처 유저가 쫓기는 구조**가 된다.

### 2-4. 상업화 철학 충돌 여지
- 성장 재료 직접 판매는 지양하는 게 기본 기조다. 시즌 후반 "영지 Lv5만 빨리 올리면 무료로 재료 +5%" 구도는 **Lv5로 가는 업그레이드 비용 자체가 간접적 성장 재료 게이트**로 기능할 수 있다. 치장·편의 중심 기조를 해치지는 않지만, "Lv5 도달이 결국 영속 자원 차이"가 되는 서사는 부담스럽다.

### 2-5. T2 파밍 의미 훼손
- T2는 최종 파밍 장비이자 재료 제작형이다. 여기에 "운 좋으면 재료 +1" 구조는 T2 수동 확정 UX("클릭 = 확정")로 정리한 철학과 정면 충돌한다. T2 한정 적용은 특히 위험하다.

## 3. 수치 시나리오 (저/중/고 영지 Lv 분포)

가정: 시즌 90일, 시즌 후반 30일 동안 적용. 시간당 기준 생산은 상대 단위로 "Lv1 영지 기본 라인 100"을 1로 둔다. 배치 상한 배수는 Lv1=1.0 / Lv2=1.3 / Lv3=1.8 / Lv4=2.4 / Lv5=3.0 가정. 확률 버프는 Lv당 +1%, Lv5=5%.

| Lv | 배치 상한 배수 | 확률 버프 | 시간당 산출(가중) | Lv1 대비 배수 |
|---|---|---|---|---|
| Lv1 | 1.0x | 0% | 1.00 | 1.00x |
| Lv2 | 1.3x | 2% | 1.326 | 1.33x |
| Lv3 | 1.8x | 3% | 1.854 | 1.85x |
| Lv4 | 2.4x | 4% | 2.496 | 2.50x |
| Lv5 | 3.0x | 5% | 3.15 | 3.15x |

### 시나리오 A — 저 분포 (시즌 초반 2주차)
- 분포 가정: Lv1 60% / Lv2 30% / Lv3 10% / Lv4~5 0%.
- 전체 플레이어 평균 산출 배수 ≈ 0.6×1.00 + 0.3×1.326 + 0.1×1.854 ≈ **1.18x**.
- 버프 기여 분: 약 **+1.2%**. 경매·시세 영향 무시 가능.

### 시나리오 B — 중 분포 (시즌 중반, 약 45일차)
- 분포 가정: Lv1 20% / Lv2 30% / Lv3 30% / Lv4 15% / Lv5 5%.
- 평균 산출 배수 ≈ 0.2×1.00 + 0.3×1.326 + 0.3×1.854 + 0.15×2.496 + 0.05×3.15 ≈ **1.68x**.
- 버프 기여 분: 약 **+2.6%**. 여기서부터 중간재 경매 최저가에 체감 압력.

### 시나리오 C — 고 분포 (시즌 후반, 70일차 이후)
- 분포 가정: Lv1 5% / Lv2 15% / Lv3 30% / Lv4 30% / Lv5 20%.
- 평균 산출 배수 ≈ 0.05×1.00 + 0.15×1.326 + 0.3×1.854 + 0.3×2.496 + 0.2×3.15 ≈ **2.18x**.
- 버프 기여 분: 약 **+3.7%**. Lv5 20% 풀이 경매 공급의 다수를 쥐게 되는 구간 → **T1 강화석 시세 눈에 띄게 하락 가능성**.

> 순수 버프 기여만 보면 연중 최대 +3.7% 수준이고 배치 상한 차등 효과에 비하면 작다. **단 "상위권 쏠림"이 경매 최저가에 주는 압력**은 산술치보다 체감이 크다는 점이 핵심 리스크.

## 4. 조정 제안

### 4-1. 수치 하향 (도입 시)
- Lv당 +1%는 체감 대비 "희귀 확률 부스트 부활"이라는 상징적 무게가 크다. 도입한다면 **Lv5=3% / Lv4=2% / Lv3=1% / Lv1~2=0%** 수준으로 내리고, Lv1~2에는 아예 버프가 없도록 시작 구간에서 차이를 없애는 게 안전하다.
- 또는 **"확률 +1개"가 아니라 "낮은 확률로 제작 시간 단축"** 으로 변환. 이미 영지 Lv5 -15% 제작 시간 버프가 있으므로 같은 축에 흡수하면 결정적 구조를 해치지 않는다.

### 4-2. 적용 범위 제한
- **T2 라인 전면 제외 필수**. T1 강화석·T1 장비·가공재만 대상. T2는 "클릭 = 확정" 철학 유지.
- **장비 제작 제외 권장** (장비는 이미 1회 = 1개 고정이므로 확률 +1이 붙으면 희소성 정의가 흔들린다). 가공재·연금·요리만 대상이 되도록 카테고리 차단.

### 4-3. 가시성·예측 가능성
- 확률이 적용되면 **우측 상세 패널에 명시**("영지 Lv5 보너스: 5% 확률로 +1 생산"). "운으로 더 나왔다"가 아니라 "영지 Lv 효과로 나왔다"로 인지되도록.
- 배치 제작 종료 시 배치 UI 하단에 "보너스 산출: N개" 요약 표기 → 실제 체감이 과대 평가되지 않게.

### 4-4. 운영 지표 수집
- 시즌 누적 "버프로 생성된 보너스 수량 총계"를 카테고리별 로깅. 시즌 중간(45일차) 경매 최저가 대비 상관관계 확인 후 **시즌2 수치 재튜닝 근거**로 사용.

## 5. 도입 권장 여부 판정

**권장: 변경 도입 (조건부)** — 아래 수정안으로 축소 도입.

### 권장안 (단일)
- **적용 범위**: 가공·연금·요리 카테고리 한정. T1 장비 제작·T2 전 라인·강화석 제작대 제외.
- **수치**: Lv1~2 = 0%, Lv3 = 1%, Lv4 = 2%, Lv5 = 3%.
- **표시 방식**: 올인원 조합대 우측 패널에 명시. 배치 결과 UI에 보너스 요약.
- **시즌1은 실험 단계**: 45일차에 경매 시세·누적 수량 로그 검토, 문제 있으면 시즌2 이전에 수치 하향 또는 폐기.

### 리스크 2개 (명시)
1. **상위권 경매 주도 리스크**: Lv5 도달 플레이어가 경매 최저가를 끌어내려 일반 유저의 "팔아서 골드 번다" 경로가 약해질 수 있다. 모니터링 필수.
2. **결정적 파밍 철학 원칙 희석**: 수치를 아무리 낮춰도 "확률 요소 추가" 자체가 NPC 효과 단일화 결정에 역행하는 상징 문제가 남는다. 유저 커뮤니티 반응이 부정적이면 철회 경로를 항상 열어둘 것.

## 오픈 질문
1. 권장안의 카테고리 범위(가공·연금·요리)에서 "연금 강화 보조 물약"은 성장 재료 성격이 강한데, 이 일부만 제외할지 여부.
2. 버프로 추가 생산된 수량을 **경매 등록 가능 여부**로 구분할지(예: 보너스 분은 "귀속" 표식). 경매 교란 리스크를 직접 제거하는 가장 강한 옵션이나 UX 복잡성 증가.
3. 길드 영지(보류 중) 도입 시 확률 버프가 공유 영지에 어떻게 분배되는지. 현 시점에는 개인 영지만 가정.
4. 시즌2 이후 Lv6+ 프레스티지 구간이 열릴 때 본 버프가 계단식으로 확장되는지, Lv5에서 소프트캡되는지.
