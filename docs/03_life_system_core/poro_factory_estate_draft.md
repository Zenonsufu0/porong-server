# 포로 서버 공장형 영지 시스템 초안 (엔드필드 참고)

## 문서 목적
기존 영지 기준(영지 전용 생활 / 올인원 조합대 / 채집기 Lv5 / 월렛 통합 / 바닐라 UI 차단 / 제련로·양조대 커스텀 대체)을 **유지한 채** 엔드필드식 자동화 공장 라인을 해석 레이어로 덧씌운 초안. content-designer 1차 기반.

핵심 차별점 3가지:
- 설비를 단일 블록이 아닌 **"라인 구성원"** 으로 재해석
- 플레이어는 **설계자**, NPC 일꾼/무인 드로이드는 실행자
- **오프라인 누적 생산 기본** + 고급 라인(T2)만 플레이어 개입 지점

## 핵심 규칙
- 생활은 영지 안에서만, 필드 채집 없음
- 라인 4단계: **채집 → 가공 → 제작 → 보관**. 역순 배치 불가 또는 효율 페널티
- 설비는 **입력 포트 / 출력 포트**를 가짐. 포트 간 연결은 커스텀 경로 블록(운송 레일·마법 송출선)
- 올인원 조합대 = 3단계 "제작" 허브
- 제련로·양조대 상당 = 2단계 "가공" 전용 커스텀 블록 (바닐라 차단 정책과 일관)
- 영지 창고(월렛 통합) = 4단계 "보관" 엔드포인트, 1차 생산품 자동 송출
- 오프라인 생산 기본 허용, **제작 단계 일부와 T2 제작재 라인**만 플레이어 개입
- 수도 반경 밖 이동은 메인 퀘스트 진행도로 게이트

## 공장 라인 4단계

### 1단계: 원재료 채집 (Extractor Layer)
- 광석 채집기 (철/구리/은/금/미스릴, 바이옴 영향 없음)
- 목재 채집기 (영지 바이옴 프리셋 기반)
- 약초/곡물 채집기 (T1 소비품 원료)
- 마력 채집기 (T2 원료, 메인 6 이후 해금)
- 연결: 출력 포트 1개, 가공 입력 직결 또는 임시 버퍼 상자 경유

### 2단계: 가공 (Processor Layer)
- **커스텀 제련로** (광석 → 주괴, 바닐라 화로 대체)
- **커스텀 양조대** (약초 → 추출물, 바닐라 양조대 대체)
- 목재 가공대 (원목 → 판재·가공목)
- 마력 정제기 (마력 원석 → 정제 마력, T2 전용)
- 연결: 입력 1·출력 1. 복수 재료는 "합류 블록"

### 3단계: 제작 (Assembler Layer)
- **올인원 조합대** (허브, 수동 조작 가능한 유일 설비)
- 강화석 제작대 (T1/T2 강화재 전용)
- 장비 제작대 (T1 장비 전용)
- 소비품 제작대 (포션·요리)
- 연결: 입력 최대 3·출력 1

### 4단계: 보관 (Storage Layer)
- 영지 중앙 창고 (월렛 연동 엔드포인트)
- 카테고리 분류기 (강화재/소비품/제작재 자동 분류)
- 오버플로우 보관함 (창고 풀 시 임시, 일정 시간 후 소멸 경고)
- 연결: 입력 무제한, 출력 없음 (월렛 직행)

### 기존 설비 재배치
- **채집기** → 1단계 Extractor. Lv3+부터 **출력 포트 2개로 분기 생산**
- **올인원 조합대** → 3단계 허브. 메인 4 청사진 = "공장 라인 중심 허브 설치"로 서사 재해석
- **영지 창고** → 4단계 Storage 엔드포인트, 월렛 통합 그대로

## NPC 일꾼 — 무인 기본 + 추가 버프형 (사용자 확정, 2026-04-19 갱신)
**기본 구조는 완전 무인 자동화**. NPC 일꾼은 **선택적 부스트**로만 존재하며 영지 정상 작동의 필수 조건이 아니다.

- **고용**: 수도 중앙 광장 "인력 상인 NPC", 계약 단위 1주 실시간
- **급여: 골드 단일 (2026-04-19 확정)**. 명성·평판 혼합 폐기. 미지급 시 자동 해고.
- **배치**: 설비 1기당 NPC 1명. **효과 = 생산 속도 증가 단일 (2026-04-19 확정)**. 희귀 중간재 확률 부스트 등 종류별 특수 효과 폐기. 등급별 속도 차등(일반 +15% / 전문 +25% / 장인 +30%). 설비 기본 생산은 NPC 없이도 동일.
- **종류**: 채집꾼(1단계) / 장인(2단계) / 명장(3단계) / 창고지기(4단계) — 계통별 배치 제약만 남고 효과 자체는 속도 증가 단일
- **오프라인**: NPC 있어도 오프라인 누적 동일, 부스트만 유지
- **무인 기본**: 설비 자체의 "마력 자동화" 세계관으로 서사 보강. NPC는 운영·경제 측 선택 레이어.

## 오프라인 진행 정책
- **1단계 채집**: 100% 오프라인 누적, 영지 Lv별 저장 상한
- **2단계 가공**: 100% 오프라인, 원재료 버퍼 비면 정지
- **3단계 제작**:
  - T1 강화재·소비품: 100% 오프라인
  - **T2 제작재**: **플레이어 개입 필수 — 단순 클릭 확정 (2026-04-19)**. 미니게임·판정 요소 폐기. 최종 파밍 체감은 "클릭 = 확정" 단일 UX로 보장.
- **4단계 보관**: 자동
- **상한**: 오프라인 누적 최대 24시간 임시 — **경제 영향 우려로 economy-reviewer 재검토 중 (2026-04-19)**. 확정 전까지 24h 수치는 잠정.

## 레이아웃
- 영지 1칸 기본: **32×32** (바닥, 높이 16). 16×16은 좁음, 64×64는 부하 과다
- 설비 개수 상한:
  - Lv1 8기 (채집 3 / 가공 2 / 제작 2 / 보관 1)
  - Lv2 14기
  - Lv3 20기 + 48×48 확장
  - Lv4 28기
  - Lv5 36기 + 64×64 확장 (T2 풀 라인)
- 동선 L자: 입구 → 채집 → 가공 → 올인원 조합대 → 창고 → 출구
- 시각: 구조재(돌벽돌·심층암·무른 현무암) / 라인 경로(레일·반블록·커스텀 송출선) / 설비 외곽(모루·주괴·구리) / 마력 라인 T2(에메랄드·앰베서스·엔드 로드)

## 수도 반경 동선 검토 (콘텐츠 관점)
### 포함 범위 (권장)
- **안(허용)**: 중앙 광장·상업 지구·빈민가·외곽 성문·경계 농경지·**외곽 필드보스 체험 구역**
- **바깥(차단)**: 동/서/남/북 지역 연결 도로·사하르 교역로·아르카논 입구

### 제한 해제 시점
- **분기 A**: **메인 7 `갈라진 징조` 완료(파견 열차 탑승 준비) 시점** (메인 8까지 늦추면 "탔는데 못 내림" 모순)
- **분기 B**: **히든 메인 8 `수습 추천` 완료 시점** (열차 미탑승이므로 별도 해제 플래그)

### 외곽 필드보스 호환성
메인 2 외곽 필드보스 구역이 수도 반경 "안"에 포함되어야 모순 없음 → 해당 구역을 **"수도 외곽 수비 훈련장"** 컨셉으로 서사 재해석 (수도 관할 명시). "신입 모험가의 첫 실전이 수도 관할 내" 그림과도 맞물림.

### 제한 UX
- 경계에 도달 시 **자연스러운 지형·수비대 바리케이드·외교 검문소**
- 추가 이동 시도 → **액션바** "아직 수도 관할을 벗어날 수 없습니다"
- 반복 시도(3회+) → **채팅**으로 현재 메인 단계·해제 조건 안내
- **투명벽은 최후 수단**, 마지막 1블록에만 적용 (몰입 저해 최소화)

## 구현 포인트
- **설비 포트 시스템 (2026-04-19 갱신)**: 블록 NBT로 입력/출력 포트 좌표 저장. **포트 연결 = 플레이어 수동 배치 확정** — 자동 탐색 폐기. 경로 블록(운송 레일·마법 송출선)을 유저가 직접 배치해야 설비 간 물류 연결. 엔드필드식 설계 재미를 설치 단계의 핵심 개입 구간으로 둠.
- **설비 청사진 블록 (2026-04-19 신설)**: 모든 설비는 **"ooo청사진" 이름의 블록** 형태로 소지·배치. 이동·철거 시 청사진 블록으로 회수, 재설치 가능. **이동·철거 비용 없음**. 커스텀 2x2 블록(청사진 비주얼 식별성)은 **구현 가능성 후속 검토**(Paper CustomModelData + 멀티블록 엔티티 방식 대안 포함).
- **라인 시뮬레이션 틱**: 오프라인 로그인 시 일괄 계산, 온라인 1~5초 단위 틱
- **월렛 연동**: 4단계 창고 송출은 기존 월렛 API 그대로
- **커스텀 제련로/양조대**: 바닐라 블록 위 CustomModelData 표현, GUI는 커스텀 인벤토리
- **NPC 일꾼**: Citizens 플러그인 + 설비 좌표 바인딩, 급여는 스케줄러 주기 차감
- **이동 제한**: WorldGuard 리전 + 메인 퀘스트 플래그 연동 (기술 세부는 implementation-reviewer 후속)
- **레시피 해금**: 영지 Lv × 메인 퀘스트 진행도 이중 게이트
