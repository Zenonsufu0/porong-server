# 포로 서버 올인원 조합대 설계 초안

## 문서 목적
`poro_master_planning.md` 생활 시스템 섹션 + 메인 4 `살아남는 방식` 확장.
**바닐라 2×2 / 워크벤치 / 모루 / 제련로 / 양조대 / 조합대 전면 차단 + 영지 내 올인원 조합대 단일 창구** 결정에 따른 **UI·차단 정책·조합법·숙련도 체계** 1차 초안. content-designer 1차 기반이며 수치는 economy-reviewer / combat-balance 패스 전 가안.

## 배경 (기존 확정)
- 생활은 **영지 전용**, 필드 채집 제거
- 메인 4에서 올인원 조합대 청사진·설치
- T2 장비는 재료 제작형 (보스 재료 4~5 + 결투 재료 12~15 + 골드 30~40k + 보조 2)
- **1차 생산품 월렛 통합 저장** (골드·강화재·영지 재료)
- E 키 인벤 가로채기 + 바닐라 인벤 보조 진입점(커스텀 메뉴 좌측 상단 고정 버튼)

## 설계 기조
- **"영지에 들러야 할 이유"를 제작에 걸어둔다** — 필드 생활 제거와 맞물려 영지 귀환 동선을 자연스럽게.
- **재료 보관은 월렛, 결과물만 실물** — 창고 관리 피로도 최소화.
- **제작은 T1 성장 루트의 필수 경로, T2는 보스·결투 파밍과 맞물린 고단계 루트**로 분리.
- **하나의 UI지만 카테고리는 분리** — 장비·연금·요리·가공·특수가 각자 숙련도·해금 동선을 가짐.

## 핵심 규칙 (2026-04-19 갱신)
- **설치 위치 제한**: 영지 내부에만 배치 가능. 영지 미보유 신규 플레이어는 **서버 튜토리얼 시 T1 기본 장비 일괄 지급**으로 구제(메인 4 이전 본편 진입 가능).
- **바닐라 조합 UI·블록 배치 전면 금지 확정**: 워크벤치·모루·제련로·양조대·조합대·레드스톤 소자 모두 UI 오픈 차단 + **영지 내 블록 배치 불가**(예외 없음). 치장용 건축 블록은 **수도 상점 NPC 판매로 분리** — 영지 외부에서만 사용 가능, 영지 내 반입·설치는 여전히 차단(영지 내부 블록 화이트리스트 기반).
- **배치 제작 상한 = 영지 Lv별 차등**: Lv1~5 단계가 올라갈수록 배치 슬롯·1회 배치 크기 증가(구체 수치 economy-reviewer 후속).
- **리소스팩 도입 확정**: 치장용 블록·커스텀 UI·무기·치장 아이템 전반 CustomModelData 기반. 별도 리소스팩 설계 초안 신설 예정.
- **재료 합산 조회**: 월렛(1차) + 플레이어 인벤 + 영지 창고 **합산**. 소비 우선순위 **월렛 → 창고 → 인벤** 고정(UX 단순화).
- **제작 결과물 지급 규칙**:
  - 장비·잠재석·단서·치장류 → **실물 인벤 지급**
  - 가공재료·강화재·소비재(요리/연금) → **월렛 지급**
- **숙련도는 카테고리별 Lv 20**. 카테고리 간 경험치 공유 없음.
- **해금 게이트 3축**: 메인 퀘스트 진척 / 카테고리 숙련도 Lv / 세력 평판.

## UI 초안

### 메뉴 카테고리 (5개 권고)
1. **장비 제작** — T1/T2 무기·방어구·장신구
2. **연금술** — 회복·상태 대응·전투 보조 물약
3. **요리** — 일반 요리·만찬·지역 특산
4. **재료 가공** — 원재료 → 가공재료, 합금·정제
5. **특수 제작** — 영지 도구, 이벤트·테마 한정 레시피 (장기적으로 "치장/염색" 분리 여지)

### 와이어프레임 방향
```
┌──────────────────────────────────────────────────────────┐
│ [E 인벤 복귀]  ★ 올인원 조합대 — 영지 Lv N             │
├──────────┬───────────────────────────────┬───────────────┤
│ 좌측 탭  │ 중앙 레시피 그리드 (6×N)      │ 우측 상세 패널│
│ 장비     │ [아이콘][아이콘][아이콘]...    │ 결과물 미리보기│
│ 연금술   │ [아이콘][아이콘][아이콘]...    │ 필요 재료 목록 │
│ 요리     │                                │  - 재료 A 3/5  │
│ 가공     │ (회색/컬러/경고 3상태 배지)    │  - 재료 B 0/2  │
│ 특수     │                                │ 요구 숙련도    │
│ ───────  │                                │ 제작 시간      │
│ 숙련도   │                                │ [제작] [배치]  │
│ 장비 12  │                                │                │
│ 연금 07  │                                │                │
│ 요리 05  │                                │                │
│ 가공 09  │                                │                │
│ 특수 02  │                                │                │
└──────────┴───────────────────────────────┴───────────────┘
```

- **좌측 탭**: 5 카테고리 + 숙련도 요약 상시 표시
- **중앙 그리드**: 정렬 기본값 "해금 → 숙련도 순". 상단 검색/필터(재료명·결과물 태그·테마)
- **우측 패널**: 결과물 프리뷰·재료 보유/필요·요구 숙련도·제작 시간·제작/배치 버튼
- **좌측 상단 고정**: "바닐라 인벤 보조 진입점" 버튼 (E 차단 상태에서도 인벤 접근)

### 레시피 상태 표기 (3상태)
- **미해금 (회색)**: 아이콘 그레이스케일 + 자물쇠 오버레이, 툴팁에 힌트만 ("메인 퀘스트 X 완료 후 해금")
- **해금 (컬러)**: 기본 컬러, 제작 가능 시 테두리 강조
- **숙련도 미달 (경고)**: 컬러이되 노란 느낌표 배지, 툴팁에 "장비 숙련도 Lv 8 필요 (현재 Lv 5)"

> 재료 부족은 4번째 상태가 아니라 **우측 패널에서만 색 강조**. 그리드 자체 상태는 "해금/구조적 잠금"에만 사용.

### 재료 조회 합산
- 우측 패널 재료 줄: `재료명 [보유/필요] (세부: 월렛 N / 창고 N / 인벤 N)`
- 마우스 오버 시 세부 출처 툴팁
- 소비 우선순위 월렛 → 창고 → 인벤 고정(플레이어 선택 금지, UX 단순화)
- 월렛 저장 대상이 아닌 실물 재료는 창고/인벤만 조회

### 배치 제작 · 일괄 큐
- **Shift+클릭**: 보유 재료 기준 최대 수량 즉시 제작 (상한 99)
- **일괄 큐**: 카테고리별 1개 큐(동시 최대 5건 예약). 오프라인 진행 허용, 완료 시 월렛/우편 자동 수령
- 동일 레시피 중복 큐 허용, 서로 다른 레시피 순차 진행
- 요리·연금·가공은 배치 친화, **장비 제작은 1회 = 1개 고정**(희소성 유지)

### 제작 진행 UI
- **즉시 완료형**: 요리·가공·하위 연금 (0.5초 연출)
- **실시간 시간형**: 장비 제작·고급 연금·특수 제작 (T1 30초~2분, T2 5~30분). 오프라인 진행, 영지 Lv 상승 시 시간 감소 버프(예: Lv5 -15%)
- **배치 제작은 0.7n 로그형 가산** (배치 인센티브)

## 바닐라 조합 UI 전면 차단 정책

### 서버 이벤트 훅 조합
- `PrepareItemCraftEvent`: 결과 슬롯을 항상 AIR로 치환 → 2×2/3×3 조합 결과 자체 불가
- `CraftItemEvent`: 이중 안전망으로 `setCancelled(true)` + 액션바 경고
- `InventoryOpenEvent`: `InventoryType`이 `WORKBENCH / ANVIL / FURNACE / BLAST_FURNACE / SMOKER / BREWING / CARTOGRAPHY / GRINDSTONE / LOOM / STONECUTTER / SMITHING`이면 취소 + 안내
- `PlayerInteractEvent`: 위 블록군 우클릭 시 `setUseInteractedBlock(DENY)` — 오픈 단계 이전 차단
- `BlockPlaceEvent`: 바닐라 조합 블록군 배치 자체 금지 (권고)
- `InventoryClickEvent`: 플레이어 인벤 내 2×2 크래프팅 슬롯(raw slot 1~4) 클릭 전면 차단

### 블록 배치 금지 vs UI 오픈만 차단 — **배치 자체 금지 권고**
- 영지 꾸미기 용도로도 바닐라 워크벤치/모루가 놓여 있으면 플레이어는 자연히 우클릭 시도 → UX 혼선
- 치장용으로 허용하면 "왜 이건 안 열리지?" 버그 문의 지속
- 대안: **치장용 더미 블록** (모양만 같은 커스텀 아이템)을 특수 제작에서 제공
- 예외적으로 **제련로·양조대**는 레드스톤 자동화·기술적 활용 논쟁 여지 → 오픈 질문

### 플레이어 인벤 2×2 슬롯 강제 비활성
- `InventoryClickEvent`에서 raw slot 1~4 클릭 이벤트 무조건 cancel
- 해당 슬롯에 아이템이 드래그·셔플 진입 차단
- 좌측 상단 "바닐라 인벤 보조 진입점" 버튼으로 들어와도 **동일 규칙 유지**
- 로그인 시 2×2 슬롯에 아이템이 남아있다면(구버전 유저) 자동으로 인벤 빈 슬롯/우편 이전

### 안내 메시지 UX
- **액션바 권고** (채팅 스팸 방지, 토스트보다 반응 빠름)
- 예: 워크벤치 오픈 시도 → `[조합] 바닐라 조합대는 사용할 수 없습니다. 영지 올인원 조합대를 이용하세요.`
- 2×2 슬롯 클릭 → `[조합] 2×2 조합은 비활성화되어 있습니다.`
- **최초 1회는 토스트** (튜토리얼 진입 유도), 이후 반복은 액션바
- 채팅 쿨타임 60초 1회 보조 출력 (튜토리얼 퀘스트 링크)

### 바닐라 구이·제련·양조 대체
- **전부 올인원 조합대로 일원화** 권고
- 근거: 제작 동선 분산 방지 / 레시피 관리·밸런스 패치 일원화 / 월렛·실물 지급 일관성
- 연료 개념 제거 (제련로·양조대 대체 시 연료 UI 불필요) → 제작 시간·골드 코스트로 환산

## 숙련도·해금 체계

### 카테고리별 Lv 20
- 카테고리마다 독립 경험치 테이블
- 경험치는 **결과물 레어도 × 재료 복합도**로 산출. 반복 제작 시 하위 레시피는 경험치 체감(반감)
- Lv 상승 시 해금 레시피 + 카테고리별 패시브(예: 요리 Lv10 = 배치 제작 슬롯 +1)

### 직업·세력 특화
- **직업 특화**: T2 테마 장비는 누구나 제작 가능. 무기 마스터리 퀘스트 완료는 향후 특수 레시피(이벤트·테마 한정) 해금 게이트로 사용 예정. (이전 "각인 시약 무기 특화형" 항목은 각인 시약 제도 삭제로 제거.)
- **세력 특화**:
  - 사하르 상인연합 평판 3 이상 → 사하르 연회 만찬, 사막 향신료 가공
  - 아르카논 평판 3 이상 → 마력 결정 가공 상위 티어
  - 수도 평판은 기본값이라 특화 레시피 없음(공용 루트 유지)
- 세력 특화 레시피는 **카테고리 숙련도와 별개로 평판 게이트** (숙련도만으로는 해금 불가)
