# 올인원 조합대 잔여 4건 일괄 자문 (economy-reviewer v1)

> **상태**: 2026-04-19 economy-reviewer 1차 자문. 본 문서는 `poro_allinone_workbench_draft.md` 오픈 질문 #3·#4·#5·#6 잔여 4건을 일괄 검토한다. CLAUDE.md `장점 → 위험요소 → 조정 제안` 흐름과 경제 철학(결정적 파밍·T2 재료 제작형·상위권 Lv5 프리미엄 관리)을 동시 적용한다.

## 문서 목적

올인원 조합대 오픈 질문 중 앞서 해결된 #1(바닐라 블록 배치 금지)·#2(신규 플레이어 구제)·#7(확률형 버프 축소 도입)·#8(리소스팩)·#9(각인 시약 폐기) 이후 남은 4건에 대한 권장안을 일괄 정리한다. 4건 모두 경제 관점과 UX 관점이 동시에 관여하며, 특히 #4 배치 제작 상한 Lv별 수치는 이미 #7 확률 버프 축소 도입(+3.7% 최대) 결정과 연동되어 "Lv별 산출 배수가 얼마까지 벌어지는가"의 핵심 축이 된다.

## 배경·전제 요약

- **시즌1 파밍 버짓**: 캐주얼 10~18h / 코어 6~10h(1막 기준).
- **확률 버프 축소 도입 확정**: 가공·연금·요리만 Lv3/4/5 = 1/2/3% +1 산출. T1 장비·T2 전 라인·강화석 제작대 제외.
- **영지 Lv 영속 보존**: 시즌 초기화 시에도 Lv·배치·월렛 유지. 배치 상한은 한 번 풀리면 시즌 넘어서도 유효.
- **T2 = 재료 제작형 최종 파밍 장비**: 보스 재료 4~5 + 결투 재료 12~15 + 골드 30~40k 구조. "클릭 = 확정" 철학.
- **배치 제작 친화 카테고리 = 가공·연금·요리**. 장비 제작은 1회 = 1개 고정(희소성 유지).

## 질문 #3 — 파티 공유 요리(만찬) 지급 방식

### 옵션 비교

| 옵션 | 장점 | 단점 |
|---|---|---|
| (A) 실물 상차림 1개 지급 + 우클릭/배치 시 파티 광역 버프 | 인벤 아이콘으로 "함께 먹는" 의례성 살아남, 선물·교환·경매 가능(파티 단위 상품화), 초안 §요리 본문과 일치 | 인벤 슬롯 1칸 점유, 오프라인 큐 결과물은 우편으로 가는데 상차림 1개만 나오면 파티 모임 시점과 제작 시점 불일치 | 
| (B) 월렛 차감 + 파티 버프 즉시 부여(트리거 UI에서 "만찬 사용" 클릭) | 인벤 슬롯 점유 제로, 월렛 UX 통일성, 즉시 발동 | 의례성 소멸, 선물·거래 불가(월렛 = 귀속 1차 생산품), "만찬 이벤트"의 물리 상차림 연출 사라짐 |
| (C) 하이브리드: 실물 상차림 지급(거래·선물 가능) + 상차림 배치 시 광역 버프, 단 영지 내에서는 즉시 발동 간소화 | 두 UX 장점 결합, 영지 외부 파티 사냥터에서도 의례성 유지 | 구현 분기 2개, 규칙 설명 길어짐 |

### 권장안: **(A) 실물 상차림 1개 지급**

**근거**:
- 초안 §요리 본문에 이미 "만찬은 실물 '상차림' 아이템으로 지급해 파티 공유 UX 유지" 방침이 잠정 기재됨. 기존 방향 존중.
- 만찬은 파티 광역 30분, 고비용(300~350골드 + 고급 재료 6~8)으로 **제작 빈도 자체가 낮은 카테고리** → 인벤 슬롯 점유 부담이 실질적으로 작음.
- 선물·거래 가능 구조는 **파티·소규모 길드 내부 경제**(만찬 셔틀 플레이어) 생성 여지, 건강한 협력 요소.
- 경매 교란 리스크는 만찬 자체가 소비재이므로 확률 버프 보너스 귀속 정책과 달리 장기 인플레이션 위험 낮음.

**구체 사양 제안**:
- 만찬 실물 아이템 `meal_dish_royal_feast` / `meal_dish_sahar_banquet`: 스택 제한 5개(만찬끼리만, 일반 요리와 분리).
- 사용 시: 파티원 전체(같은 청크 반경 32블록 내) 30분 광역 버프 부여. 영지 밖에서도 작동.
- 우편 수령 시 상차림 아이템으로 정상 지급. 배치 큐가 동일 만찬 중복이면 스택 누적.
- 거래·경매 등록 허용(파티 셔틀 경제 인정). 단 **확률 버프로 추가 산출된 보너스 1개는 귀속**(귀속 정책 일관).

## 질문 #4 — T2 직업 특화 강도

### 옵션 비교

| 옵션 | 장점 | 단점 |
|---|---|---|
| (A) 무기 계열 불일치 시 세트 효과 **전면 무효** | 5 테마 × 3 무기 직업 정체성 최강, T2 수집 동기 명확 | 신규 유저 진입 장벽 급상승, 서브 무기 시도 좌절, 멀티클래스 재미 소멸 |
| (B) 세트 효과 **일부 유지**(기본 스탯 100%, 세트 보너스만 불일치 시 50~70%) | 직업 정체성 유지하면서도 시행착오 허용, "테마 = 방향성" 유지 | 일부 유지 비율 튜닝 복잡, 불일치 조합 메타 파생 우려 |
| (C) 무기 계열 무관 세트 효과 100% | 유연성 최대, 세트 수집 단순 | 테마 정체성 훼손, T2 3무기 각 튜닝이 무의미해짐 |

### 권장안: **(B) 일부 유지 — 불일치 시 세트 보너스 50%**

**근거 (경제 관점)**:
- T2는 **보스 재료 4~5 + 결투 재료 12~15 + 골드 30~40k + 영지 고유재 2**로 1벌 제작에 **상당한 파밍 버짓**(캐주얼 기준 1~2주치 1벌)을 소모한다. 불일치 시 전면 무효라면 유저는 **"맞는 무기 계열 확정 후 제작"만 선택** → 실험 불가, 상위권만 교체 가능, T2 중간재 경매 수요 축소.
- 일부 유지(50%) 구조는 **서브 무기 전환 실험 허용 + 확정 주력은 일치 유지** 이중 구조를 만든다. 결투 재료·보스 재료 소비 경로를 다양화해 중간재 경매 건강성 유지.
- 시즌 후반 일반 유저 18강 최종선 기준에서, T2 세트 불일치도 **"일단 입어볼 수 있음"** 이라는 재진입 경로가 필요.

**근거 (balance 관점 — 참조)**:
- combat-balance 영역과 겹치므로 세부 비율(50%)은 combat-balance 후속 자문 필요. economy-reviewer 관점에서는 "전면 무효 ≠ 건강한 경제"만 확정적으로 권고.

**구체 사양 제안**:
- 세트 기본 스탯(공격력·방어력 등): 100% 유지(무기 계열 무관).
- 세트 2피스/4피스 보너스 효과: **일치 시 100% / 불일치 시 50%**.
- 직업 공용 부위(장신구·로브류 일부): 불일치 페널티 없음(이미 초안 §T2 표에서 로브 = 마법계열 공용 방향).
- 표시: 장비 UI에 "세트 일치: 정확/부분" 상태 배지.

## 질문 #5 — 숙련도 경험치 반감 적용 지점

### 옵션 비교

| 옵션 | 장점 | 단점 |
|---|---|---|
| (A) Lv 차 **3 이상** 반감 (예: Lv8 유저가 Lv1~5 레시피 제작 시 반감) | 하위 레시피 무한 반복 억제 강함, 상위 레시피 도전 유인 강함 | 가공·연금은 T1 제작 시 하위 재료를 대량 소모하므로 "재료 가공"에서 경험치가 급속 감소, 체감 성장 정체 |
| (B) Lv 차 **5 이상** 반감 (예: Lv8 유저가 Lv1~3 레시피 제작 시 반감) | 완충 구간 여유, 하위 재료 가공도 어느 정도 경험치 수급, 체감 성장 자연스러움 | 하위 레시피 반복 파밍 구간이 더 길게 유효 → 최적화 유저의 "최저가 레시피 반복" 전략 |

### 권장안: **(B) Lv 차 5 이상 반감**

**근거**:
- 가공 카테고리는 **T1 장비 제작 → 가공 강괴 대량 소모** 구조라 하위 레시피 의존이 설계 의도에 가깝다. 3 이상 반감은 **가공 숙련도 성장 구간(Lv3~10)에서 강한 체증**을 만들어 실질 Lv10 도달에 시간 과다 소비.
- 시즌1 파밍 버짓 **캐주얼 10~18h / 코어 6~10h**(1막 기준) 내에서 카테고리 숙련도 Lv10~14(T2 제작 게이트)까지 자연 도달해야 함. 5 이상 반감이 이 버짓에 더 적합.
- 최적화 유저의 "최저가 레시피 반복" 전략은 **월렛 골드 소모·제작 시간 소모**가 그 자체로 자연 브레이크로 작동(즉시 완료형 = 0.5초이지만 골드·재료 소모는 있음).

**구체 사양 제안**:
- 경험치 배수 테이블:
  - Lv 차 0~4: **100%** (정상 경험치)
  - Lv 차 5~9: **50%** (반감)
  - Lv 차 10+: **20%** (재반감)
- 장비 카테고리만 예외로 **Lv 차 3 이상 반감** 유지(장비는 1회 = 1개 고정 + 높은 골드 코스트로 이미 반복 억제 강함, 숙련도 기준은 더 엄격해도 체감 부담 낮음).
- 숙련도 레시피 카드 우측에 "현재 획득 경험치: 정상/반감/재반감" 표기로 투명성 확보.

## 질문 #6 — 배치 제작 상한 영지 Lv별 차등 구체 수치

### 설계 원칙

- **확률 버프 축소 도입(최대 +3% @ Lv5) 결정과 연동**: 배치 상한 Lv1→Lv5 배수는 확률 버프 기여 분(+3.7% 상위권 경매 압력)과 합산된다. 배치 상한 배수가 과도하면 **Lv1 대비 Lv5 시간당 산출 3.15x 이상**으로 벌어져 Lv4→Lv5 업그레이드 과열 우려(#7 검토 본문 §2-3 경고).
- **캐주얼 일반 유저 기준**: Lv3~4가 "일반 유저 최종선"에 해당(영지 매트릭스 매핑). Lv5는 상위권 도전 구간.
- **카테고리별 차등**: 장비 제작은 1회 = 1개 고정(초안 §배치 제작 본문). 배치 상한은 가공·연금·요리·특수에만 적용.
- **초안 §배치 제작 본문 기준**: Shift+클릭 최대 99개 즉시 제작, 일괄 큐 동시 예약 5건.

### 권장안 수치표

| 영지 Lv | 배치 슬롯 (동시 예약 큐 수) | 1회 배치 크기 상한 (Shift+클릭 최대 수량) | Lv1 대비 산출 배수 (배치 상한 단독) | 확률 버프 합산 배수 |
|---|---|---|---|---|
| Lv1 | **3 슬롯** | **16개** | 1.00x | 1.00x |
| Lv2 | **4 슬롯** | **24개** | 1.33x | 1.33x (버프 0%) |
| Lv3 | **5 슬롯** | **36개** | 1.81x | 1.83x (+1%) |
| Lv4 | **6 슬롯** | **56개** | 2.47x | 2.52x (+2%) |
| Lv5 | **7 슬롯** | **80개** | 3.02x | 3.11x (+3%) |

### 수치 근거

- **슬롯 수 3→7**: 초안 "일괄 큐 동시 최대 5건"은 **Lv3 기본값**으로 재해석. Lv1 신규는 3 슬롯으로 학습 곡선 완화, Lv5는 7 슬롯으로 상위권 체감(+40%) 보장.
- **1회 배치 크기 16→80**: 초안 "Shift+클릭 최대 99개" 상한은 **Lv5 상한 근접치(80)** 로 설정해 "99개 제작"이 상위권 전용 체감이 되도록 배치. Lv1 16개는 T1 초기 요리·회복 물약 수급에 충분.
- **Lv1 대비 Lv5 단독 배수 ≈ 3.02x**: #7 검토 본문 §3 시나리오표에서 가정한 "Lv1=1.0 / Lv5=3.0" 기준 일치. 확률 버프와 합산해도 3.11x로 §3 시나리오 C 범위(2.18x 평균 ≈ Lv5 개인 3.15x) 이내.
- **Lv4→Lv5 증가폭(+24개 / +1 슬롯)**: Lv3→Lv4(+20개 / +1 슬롯) 대비 약간 크지만, 업그레이드 비용(100만 골드 → 400만 골드 + T2 상위 코어 50)에 비례하는 체감 보상으로 타당.

### 카테고리별 세부 규칙

- **가공·연금·요리**: 상기 표 그대로 적용.
- **특수 제작**: 상기 표의 **1회 배치 크기만 50% 적용** (Lv5 = 40개). 특수는 이벤트·테마 한정 성격이므로 대량 생산 인센티브 낮춤.
- **장비 제작**: 배치 상한 적용 대상 외. 1회 = 1개, 일괄 큐만 슬롯 공유(Lv5 = 7 슬롯 내 장비 큐도 자리 차지).

### 오프라인 큐 연동 주의

- 일괄 큐 슬롯은 **오프라인 진행 허용**(초안 §배치 제작 본문). 시즌 후반 Lv5 유저가 7 슬롯 × 80개 = **580개/큐 1사이클 오프라인 누적** 가능 → 하루 로그인 한 번에 월렛 대량 유입.
- **완화**: 각 큐 완료 후 다음 큐 자동 시작 허용하되, **"오프라인 중 큐 자동 연쇄"는 최대 2회**로 제한. 즉 로그아웃 시 7 슬롯 × 2 사이클 = 최대 14 사이클 = 1,120개까지. 이 이상은 로그인해서 재큐잉 필요.

   - T2 세트 불일치 착용 유저 비율·평균 전투력 차이(#4).
   - 카테고리별 숙련도 분포 히스토그램(반감 구간 체증 여부 확인, #5).
   - Lv별 평균 배치 큐 가동률·오프라인 누적 산출량(#6).
2. **시즌1 중반 재검토 지점**: 3가지 지표 중 2개 이상이 임계치 초과 시 시즌2 이전 수치 재튜닝.
   - 만찬 거래 최저가 < 300골드 (#3 기본 골드 코스트 미만) → 경매 등록 금지 전환 검토.
   - T2 세트 불일치 비율 > 40% → combat-balance에 세트 보너스 비율 재검토 요청.
   - 가공 Lv10 도달 캐주얼 유저 비율 < 50% @ 45일차 → 반감 구간 완화(Lv 차 5 → 7).
   - Lv5 유저 월렛 가공 강괴 유입량이 Lv3 유저 대비 3.5x 초과 → 배치 상한 Lv5 1회 크기 80 → 64 하향.
3. **질문 #4는 combat-balance 후속 자문 필수**: 세트 보너스 50% 비율은 economy 근거로 지지되지만 구체 세트 효과(피해 증가 카테고리·태그 적용 등)는 combat-balance 영역.
