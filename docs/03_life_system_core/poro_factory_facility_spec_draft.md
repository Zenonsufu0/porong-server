# 포로 공장형 영지 설비별 세부 스펙 (12기)

## 문서 목적
`poro_factory_estate_draft.md`의 4단계 라인(채집·가공·제작·보관) 각 설비의 Lv1~Lv5 세부 스펙 + 포트·레시피·NPC 부스트·오프라인 규칙. content-designer 1차 드래프트. 수치는 상대값 기준, 밸런싱 전 검토 단위.

## 참조 문서

## 공통 원칙 (2026-04-19 갱신)
- 1블록 = 영지 그리드 1칸 (2×2×2) — **커스텀 2x2 블록 구현 가능성 후속 검토**(청사진 블록 비주얼 식별성)
- 포트: 입력(파랑) / 출력(주황) / 에너지(보라) — **플레이어 수동 배치 확정**(자동 탐색 폐기, 엔드필드식 설계 재미 유지)
- 경로 블록: 운송 레일(고체) / 마법 송출선(액체·마력)
- 오프라인 누적: **T1 100% / T2 수동 확정** — T2 수동 = **단순 클릭**(미니게임·판정 요소 폐기)
- **설비 = "ooo청사진" 이름의 블록 형태로 존재 (2026-04-19 확정)**. 이동·철거 시 청사진 블록으로 회수, 재설치 가능. **이동·철거 비용 없음**(이전 쿨다운 1h·환급률 70% 제도 폐기).
- **NPC 부스트 공통: 무인 100% / 일반 115% / 전문 125% / 장인 130% — 효과는 모두 "생산속도 증가" 단일로 통일 (2026-04-19 확정)**. 희귀 확률·연료 절감·실패율 감소 등 종류별 특수 효과 폐기. 아래 개별 설비 항목의 "NPC 특수 효과" 라인은 후속 재정리 대상.
- **NPC 급여: 골드 단일 (2026-04-19 확정)**. 명성·평판 혼합 폐기.
