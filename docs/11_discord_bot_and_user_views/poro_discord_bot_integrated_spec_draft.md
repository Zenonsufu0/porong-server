# 포로 서버 디스코드 봇 통합 명세 초안

## 문서 목적
디스코드 봇 기획·설계 문서 5종(응답 포맷 / 슬래시 명령 / payload·response / 커맨드 마스터 / 임베드 예시)을 단일 통합 문서로 재편. 원본 5파일은 병합 직후 삭제.

핵심 원칙:
- 디코 봇은 짧고 빠르게. 긴 통계표가 아닌 카드형 요약·최근 기록·핵심 수치 중심.
- 웹과 역할 분리: 디코 봇은 **유저용 조회 허브**, 웹은 **운영자용 상세 분석**.
- 모든 명령은 카드/임베드 응답으로 통일, 추후 embed 메시지 구조로 바로 이관 가능.

## 1. 기본 철학

### 1-1. 봇은 짧고 빠르게
- 핵심 요약 / 최근 기록 / 현재 장비 상태를 **한 화면 카드**로 끝낸다.
- 세부 정보는 하위 명령(`/내잠재`, `/내강화`)으로 분리.

### 1-2. 웹 ↔ 디코 역할 분리
- 디코 봇: 유저용 조회, 일상 사용 빈도 높음.
- 웹: 운영자·통계·상세 필터링, 장기 분석용.

## 3. 핵심 명령 슬래시 명세

### `/내정보`
- 옵션: 없음
- 응답: 직업, 무기, 총 강화, 대표 세트, 최근 클리어

### `/내장비`
- 옵션: `상세여부(optional)`
- 응답: 장비 부위 요약, 강화, 세트, 잠재 대표값

### `/보스기록`
- 옵션: `보스명(optional)`
- 응답: 최근 기록 또는 특정 보스 기록

### `/시세`
- 옵션: `아이템명(required)`
- 응답: 평균 시세, 거래량, 최근 변화

### `/내영지`
- 옵션: 없음
- 응답: 시설 목록, 레벨, 최근 수확, 대기 생산량

### `/명예의전당`
- 옵션: `카테고리(optional)`
- 응답: 서버 최초 / 극상위 / 시즌 기록 요약

## 4. 응답 포맷 감각

### 4-1. `/내장비` 응답 표시 항목
- 직업 / 무기
- 장착 장비 요약
- 총 강화 합
- 대표 잠재 2~3개
- 세트효과 적용 상태

한 화면에 핵심만 보이고, 세부는 `/내잠재`·`/내강화`로 분리.

### 4-2. `/보스기록` 응답 표시 항목
- 최근 클리어한 보스 5개
- 클리어 시간 / 최고 기록
- 무사망 여부
- 극상위 클리어 여부

### 4-3. `/시세` 응답 표시 항목
- 아이템 이름
- 최근 평균 시세
- 최근 거래량
- 어제 대비 상승/하락
- 추천 매수/매도 감각(선택)

## 5. Payload / Response 예시

### 5-1. `/내장비`
**payload**
```json
{
  "command": "내장비",
  "userId": "u_1001",
  "options": {
    "상세여부": false
  }
}
```

**response**
```json
{
  "title": "현도의 장비 요약",
  "classId": "warrior",
  "weaponId": "greatsword",
  "totalEnhancement": 87,
  "setSummary": "황성의 위광 4세트",
  "topPotentials": ["핵심 태그 피해 +8%", "최대 체력 +6%"]
}
```

### 5-2. `/시세`
**payload**
```json
{
  "command": "시세",
  "userId": "u_1001",
  "options": {
    "아이템명": "용암 결정핵"
  }
}
```

**response**
```json
{
  "title": "시세 정보: 용암 결정핵",
  "itemId": "lava_core",
  "avgPrice": 1980,
  "tradeVolume": 37,
  "changeRate": 4.2,
  "relatedTheme": "남부 T2 장비"
}
```

## 6. 카드 응답 예시 (유저 뷰)

### 6-1. `/내장비`
```
**현도의 장비 요약**
- 직업: 전사 / 대검
- 총 강화 합: 87
- 세트: 황성의 위광 4세트
- 대표 잠재: 핵심 태그 피해 +8%, 최대 체력 +6%
- 대표 각인: 파쇄의 권위 / 핵심 증폭 Lv5 / 회복 강화 Lv4
```

### 6-2. `/보스기록`
```
**최근 보스 기록**
- 라그네스: 클리어 4회 / 최고 8분 42초
- 모르바인: 클리어 2회 / 최고 10분 11초
- 아그네르: 미클리어 / 최고 4페이즈 진입
```

### 6-3. `/내생활`
```
**생활 요약**
- 채집 11 / 채광 9 / 벌목 7 / 낚시 8
- 요리 10 / 연금 8
- 영지 시설: 약초 재배지 Lv3 / 광물 추출장 Lv2
- 최근 수확: 설혼초 12, 정제 광석 8
```

### 6-4. `/시세 용암 결정핵`
```
**시세 정보: 용암 결정핵**
- 최근 평균가: 1,980 골드
- 최근 거래량: 37
- 어제 대비: +4.2%
- 연결 제작: 남부 T2 장비
```

## 7. 한 줄 요약
**디코 봇은 내 캐릭터·보스 기록·생활/영지·시세·업적/명예를 카드형 요약으로 빠르게 보여주는 조회 허브로 운영하고, 모든 슬래시 명령은 payload + response DTO + 임베드 구조로 통일한다.**
