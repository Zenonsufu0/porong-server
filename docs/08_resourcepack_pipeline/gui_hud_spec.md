## 2. Action Bar 5레이어 구조

Action Bar 문자열 하나에 커스텀 폰트 글리프 5개 행을 negative ascent로 위치 분리.
`sendActionBar()` 한 번 호출로 전체 HUD 갱신.

| 행 | 내용 | ascent | 렌더 범위 |
|---|---|---:|---|
| 1 (최상단) | 스택 자원게이지 | +4 | baseline -4 ~ +5 |
| 2 | 쿨타임 Row1 (LC / RC) | -6 | baseline +6 ~ +15 |
| 3 | 쿨타임 Row2 (SRC / F) | -16 | baseline +16 ~ +25 |
| 4 | XP바 + Lv.숫자 | -26 | baseline +26 ~ +35 |
| 5 (최하단) | HP바 + 현재/최대 | -36 | baseline +36 ~ +45 |

> ascent 값은 인게임 테스트 후 2~4씩 조정 필요. GUI 스케일에 따라 달라질 수 있음.

## 3. 행별 표시 형식

### 3-1. HP바 (행 5)

```
[hp_glyph] 850/1190
```

- `hp_glyph`: `hp_000.png` ~ `hp_100.png` (5% 단위, 136×9), U+E100~E114
- 현재/최대 HP 숫자를 글리프 우측에 텍스트로 표시
- 무기 미착용 시에도 항상 표시

### 3-2. XP바 (행 4)

```
[xp_glyph] Lv.42
```

- `xp_glyph`: `xp_000.png` ~ `xp_100.png` (5% 단위, 136×9), U+E150~E164
- 현재 레벨을 글리프 우측에 `Lv.숫자` 텍스트로 표시
- 레벨업 시 즉시 갱신

### 3-3. 쿨타임 (행 2~3)

```
Row1:  LC [cd_glyph] 1.2s   RC [cd_glyph] 2.3s
Row2:  SRC [cd_glyph] Xs    F [cd_glyph] Xs
```

- `cd_glyph`: `cd_010.png` ~ `cd_100.png` (10% 단위, 55×9)
  - Row1 (LC/RC): U+E120~E129 (ascent=-6)
  - Row2 (SRC/F): U+E130~E139 (ascent=-16, 같은 PNG 재사용)
- 쿨타임 수치 표시 규칙:
  - 즉시 사용 가능: `§a-` (초록 대시)
  - 쿨타임 진행 중: `§e3.2s` (노랑)
  - 5초 이하: `§c1.2s` (빨강)
- 무기 미착용 시 쿨타임 행 미표시

### 3-4. 스택 자원게이지 (행 1)

```
[각인이름] [filled_glyph × 현재스택][empty_glyph × 잔여스택] 현재/최대
```

예시:
```
검세 ●●○ 2/3
충격 ●●●○ 3/4   (유지형 각인 장착 시)
압박 ●●●○○ 3/5
```

- 무기별 전용 colored 글리프 사용 (무기 교체 시 자동 변경)
- 유지형 각인 장착 시 최대 스택 +1 (글리프 개수 자동 반영)
- 무기 미착용 시 스택 행 미표시

## 4. 스택 글리프 맵 (무기별 색상)

| 무기 | filled 색상 | filled | empty | PNG 크기 |
|---|---|---|---|---|
| 검 | 빨강 | U+E140 | U+E141 | 9×9 |
| 망치 | 주황 | U+E142 | U+E143 | 9×9 |
| 스태프 | 하늘 | U+E144 | U+E145 | 9×9 |
| 석궁 | 초록 | U+E146 | U+E147 | 9×9 |
| 낫 | 보라 | U+E148 | U+E149 | 9×9 |
| 창 | 파랑 | U+E14A | U+E14B | 9×9 |

플러그인 글리프 계산:
```java
// weaponIdx: 검=0, 망치=1, 스태프=2, 석궁=3, 낫=4, 창=5
char filled = (char)(0xE140 + weaponIdx * 2);
char empty  = (char)(0xE141 + weaponIdx * 2);
```

## 6. 특수 상태 알림 (Action Bar 임시 오버라이드)

이벤트 발생 시 Action Bar를 2~3초간 교체 후 원래 HUD 복귀.

| 이벤트 | 메시지 |
|---|---|
| 강화 성공 | `§a✔ 강화 성공!  §e18강` |
| 강화 실패 | `§c✘ 강화 실패` |
| 레벨업 | `§6★ 레벨 업!  §eLv. 43` |
| 마력 과부하 | `§c⚠ 마력 과부하! 기계 정지` |
| 필드보스 등장 | `§c⚔ 들판 포식자 출현!` |

## 7. Scoreboard Sidebar

항상 우측 표시. 비전투·전투 중 동일.

```
§6포로 서버
§7──────────
§e검사  §7수호검
§7──────────
 §e12,450§7G
 §b84§7개
 §55§7개
§7──────────
§7Lv.§f42  §e73%  §a+3§7포인트
§7──────────
§7수도 외곽 평원
```

| 항목 | 내용 | 갱신 주기 |
|---|---|---|
| 무기 클래스 | 클래스명 + 현재 각인명 (예: `검사  수호검`) | 무기/각인 변경 시 즉시 |
| 골드 | `` + 보유량 | 변경 시 즉시 |
| 강화석 | `` + 보유량 | 변경 시 즉시 |
| 큐브 | `` + 보유량 | 변경 시 즉시 |
| 레벨 + 경험치% | `Lv.숫자  X%` | 경험치 획득 시 즉시 |
| 사용 가능 스탯 | `§a+N§7포인트` (0이면 미표시) | 레벨업 / 배분 시 |
| 현재 위치 | 필드명 또는 영지명 | 구역 이동 시 |

> 골드·강화석·큐브는 DB 가상 수치 (실물 아이템 카운트 아님).
> 사용 가능 스탯 포인트가 0이면 해당 줄 생략.

## 8. Boss Bar

바닐라 Boss Bar 활용. 보스 인스턴스 진입 시 표시, 종료 시 제거.

| 항목 | 내용 |
|---|---|
| 바 색상 | 필드보스 `RED` / 시즌보스 `PURPLE` / 최종보스 `WHITE` |
| 텍스트 | 보스 이름 + 현재HP / 최대HP |
| 단계 표시 | 페이즈 전환 시 `[페이즈 2]` 보스 이름에 추가 |

## 10. 구현 방식 요약

| HUD 요소 | 구현 수단 | 갱신 방식 |
|---|---|---|
| HP바 + 수치 | Action Bar 커스텀 폰트 글리프 | 피해/회복 시 즉시 |
| XP바 + Lv | Action Bar 커스텀 폰트 글리프 | 경험치 변경 시 즉시 |
| 스택 자원게이지 | Action Bar 커스텀 폰트 글리프 | 자원 변경 시 즉시 |
| 쿨타임 2행 | Action Bar 커스텀 폰트 글리프 | 1틱(0.05초) 갱신 |
| 임시 알림 | Action Bar 오버라이드 | 이벤트 발생 시 2~3초 |
| 골드/강화석/레벨/위치 | Scoreboard Sidebar | 변경 시 즉시 |
| 보스 HP | Boss Bar (바닐라) | 피해 시 즉시 |
| 바닐라 하트·허기·XP | icons.png 완전 투명 | — |
