# 장비 세부 GUI 설계 (27슬롯)

> **[STATUS: REFERENCE]** — 장비 개별·일괄 외형 설정 GUI.  
> 진입: 캐릭터 GUI(gui_equipment_panel.md) 장비 슬롯 클릭 또는 슬롯 16(일괄선택)

## 1. 개별 세부 GUI (장비 슬롯 클릭 시)

### 방어구 (투구·상의·하의·신발)

```
     col0  col1  col2  col3  col4  col5  col6  col7  col8
row0 [유리] [유리] [유리] [유리] [현재] [유리] [유리] [유리] [유리]
row1 [유리] [가죽] [사슬] [구리] [철  ] [금  ] [다이아][네더] [유리]
row2 [뒤로🔴][유리] [유리] [유리] [유리] [유리] [숨김] [유리] [보이기]
```

| 슬롯 | 역할 |
|---|---|
| 4 | 현재 선택 외형 표시 — 선택된 재질 `LIME_STAINED_GLASS_PANE` 하이라이트 |
| 10 | 가죽 | 11 | 사슬 | 12 | 구리 |
| 13 | 철 | 14 | 금 | 15 | 다이아 | 16 | 네더라이트 |
| 24 | 숨김 — 외형 없음 (스킨만 보임) |
| 26 | 보이기 — 외형 표시 |
| 18 | 뒤로 (`ARROW`) |

### 검·도끼·낫 (바닐라 재질 6종)

```
row0 [유리] [유리] [유리] [유리] [현재] [유리] [유리] [유리] [무기변경]
row1 [유리] [나무] [돌  ] [철  ] [금  ] [다이아][네더] [유리] [유리]
row2 [뒤로🔴][유리] [유리] [유리] [유리] [유리] [숨김*] [유리] [보이기*]
```

| 상태 | 숨김(슬롯 24) | 보이기(슬롯 26) |
|---|---|---|
| 치장템 없음 | `GRAY_STAINED_GLASS_PANE` (비활성) | `GRAY_STAINED_GLASS_PANE` (비활성) |
| 치장템 있음 | 활성 — 치장템 표시로 복귀 | 활성 — 바닐라 무기(선택 재질) 표시 |

> 치장템이 있을 때 기본값 = 치장템 표시 상태.  
> 보이기 = 바닐라 무기, 숨김 = 다시 치장템으로.

| 슬롯 | 역할 |
|---|---|
| 8 | 무기 변경 (`IRON_SWORD`) — 클릭 → 무기 목록 GUI (`§2`) |

### 창·석궁 (1종)

```
row0 [유리] [유리] [유리] [유리] [현재] [유리] [유리] [유리] [무기변경]
row1 [유리] [유리] [유리] [유리] [창/석궁][유리] [유리] [유리] [유리]
row2 [뒤로🔴][유리] [유리] [유리] [유리] [유리] [숨김*] [유리] [보이기*]
```

> 숨김/보이기: 무기와 동일 규칙 적용. 슬롯 8 무기 변경 버튼 포함.

### 스태프 (2종)

```
row0 [유리] [유리] [유리] [유리] [현재] [유리] [유리] [유리] [무기변경]
row1 [유리] [유리] [유리] [막대기][블막대] [유리] [유리] [유리] [유리]
row2 [뒤로🔴][유리] [유리] [유리] [유리] [유리] [숨김*] [유리] [보이기*]
```

> 숨김/보이기: 무기와 동일 규칙 적용. 슬롯 8 무기 변경 버튼 포함.

## 2. 무기 변경 GUI (45슬롯)

> 진입: 무기 세부 GUI 슬롯 8 클릭  
> 무기 6종(검·도끼·낫·석궁·창·스태프) 중 하나를 선택. 인벤에는 항상 1종만 존재.

```
     col0  col1  col2  col3  col4  col5  col6  col7  col8
row0 [유리] [유리] [유리] [유리] [유리] [유리] [유리] [유리] [유리]
row1 [유리] [ 검 ] [유리] [유리] [도끼] [유리] [유리] [ 낫 ] [유리]
row2 [유리] [유리] [유리] [유리] [유리] [유리] [유리] [유리] [유리]
row3 [유리] [석궁] [유리] [유리] [ 창 ] [유리] [유리] [스태프][유리]
row4 [뒤로🔴][유리] [유리] [유리] [유리] [유리] [유리] [유리] [유리]
```

| 슬롯 | 역할 |
|---|---|
| 10 | 검 | 13 | 도끼 | 16 | 낫 |
| 28 | 석궁 | 31 | 창 | 34 | 스태프 |
| 36 | 뒤로 (`ARROW`) |

> 현재 장착 중인 무기 슬롯: `LIME_STAINED_GLASS_PANE` 테두리 하이라이트.

**무기 슬롯 lore 예시 (비장착)**

```
§e검
§7──────────────────
§7 강화   : §e+15강
§7 등급   : §d레전더리
§7 잠재   : §a치명타 확률 +25% / 스킬 피해 +15% / 핵심기 +20%
§7 각인   : §a집중 강타 A
§7 재질   : §7다이아몬드
§7 스킬   : §7[LC]회전베기 [RC]돌진 [SRC]폭풍 [F]처형
§7──────────────────
§7 [클릭] 이 무기로 교체
§c 기존 무기는 회수됩니다.
```

**장착 중 lore 예시**

```
§e검  §a[장착 중]
§7──────────────────
§7 강화   : §e+15강
§7 등급   : §d레전더리
§7 잠재   : §a치명타 확률 +25% / 스킬 피해 +15% / 핵심기 +20%
§7 각인   : §a집중 강타 A
§7 재질   : §7다이아몬드
§7 스킬   : §7[LC]회전베기 [RC]돌진 [SRC]폭풍 [F]처형
```

### 무기 변경 규칙

| 항목 | 내용 |
|---|---|
| 강화·등급·잠재·각인·세부스탯 | **무기 종류별 독립 저장** — 교체해도 각 무기 데이터 유지 |
| 재질 선택 | **무기 종류별 기억** — 복귀 시 이전 선택 재질 그대로 복원 |
| 스킬셋 | **무기 종류에 따라 자동 전환** |
| 교체 처리 | 기존 인벤 무기 서버 회수 → 선택 무기 인벤 지급 |

## 3. 재질 일괄선택 GUI (캐릭터 GUI 슬롯 16)

선택한 재질을 모든 장비에 한 번에 적용. **사슬 제외** (총 6종).

```
     col0  col1  col2  col3  col4  col5  col6  col7  col8
row0 [유리] [유리] [유리] [유리] [안내] [유리] [유리] [유리] [유리]
row1 [유리] [가죽] [구리] [철  ] [금  ] [다이아][네더] [유리] [유리]
row2 [뒤로🔴][유리] [유리] [유리] [유리] [유리] [유리] [유리] [유리]
```

| 슬롯 | 역할 |
|---|---|
| 4 | 안내 아이콘 — 무기 타입에 따라 lore 변경 |
| 10 | 가죽 | 11 | 구리 | 12 | 철 |
| 13 | 금 | 14 | 다이아 | 15 | 네더라이트 |
| 18 | 뒤로 (`ARROW`) |

### 적용 범위

| 무기 타입 | 방어구 재질 적용 | 무기 재질 적용 |
|---|---|---|
| 검·도끼·낫 | ✅ | ✅ |
| 창·스태프·석궁 | ✅ | ❌ |

### 안내 슬롯(4) lore 예시

```
§e재질 일괄 선택
§7──────────────────
§7 선택한 재질로 방어구 + 무기 외형을 한 번에 변경합니다.
§7 (창·스태프·석궁은 방어구만 적용)
§7──────────────────
§7 현재 무기: §e{무기타입}
```
