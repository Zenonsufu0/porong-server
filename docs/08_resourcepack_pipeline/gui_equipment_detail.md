# 장비 세부 GUI 설계 (27슬롯)

> **[STATUS: REFERENCE]** — 장비 개별·일괄 외형 설정 GUI.  
> 진입: 캐릭터 GUI(gui_equipment_panel.md) 장비 슬롯 클릭 또는 슬롯 16(일괄선택)

---

## 1. 개별 세부 GUI (장비 슬롯 클릭 시)

### 방어구 (투구·상의·하의·신발)

```
     col0  col1  col2  col3  col4  col5  col6  col7  col8
row0 [유리] [유리] [유리] [유리] [현재] [유리] [유리] [유리] [유리]
row1 [유리] [가죽] [사슬] [구리] [철  ] [금  ] [다이아][네더] [유리]
row2 [뒤로🔴][유리] [유리] [유리] [유리] [유리] [숨김] [유리] [보이기]
```

| 슬롯 | 역할 |
|---|---|
| 4 | 현재 선택 외형 표시 — 선택된 재질 `LIME_STAINED_GLASS_PANE` 하이라이트 |
| 10 | 가죽 | 11 | 사슬 | 12 | 구리 |
| 13 | 철 | 14 | 금 | 15 | 다이아 | 16 | 네더라이트 |
| 24 | 숨김 — 외형 없음 (스킨만 보임) |
| 26 | 보이기 — 외형 표시 |
| 18 | 뒤로 (`RED_STAINED_GLASS_PANE`) |

### 검·도끼·낫 (바닐라 재질 6종)

```
row0 [유리] [유리] [유리] [유리] [현재] [유리] [유리] [유리] [유리]
row1 [유리] [나무] [돌  ] [철  ] [금  ] [다이아][네더] [유리] [유리]
row2 [뒤로🔴][유리] [유리] [유리] [유리] [유리] [숨김*] [유리] [보이기*]
```

| 상태 | 숨김(슬롯 24) | 보이기(슬롯 26) |
|---|---|---|
| 치장템 없음 | `GRAY_STAINED_GLASS_PANE` (비활성) | `GRAY_STAINED_GLASS_PANE` (비활성) |
| 치장템 있음 | 활성 — 치장템 표시로 복귀 | 활성 — 바닐라 무기(선택 재질) 표시 |

> 치장템이 있을 때 기본값 = 치장템 표시 상태.  
> 보이기 = 바닐라 무기, 숨김 = 다시 치장템으로.

### 창·석궁 (1종)

```
row1 [유리] [유리] [유리] [유리] [창/석궁][유리] [유리] [유리] [유리]
```

> 숨김/보이기: 무기와 동일 규칙 적용.

### 스태프 (2종)

```
row1 [유리] [유리] [유리] [막대기][블막대] [유리] [유리] [유리] [유리]
```

> 숨김/보이기: 무기와 동일 규칙 적용.

## 2. 재질 일괄선택 GUI (캐릭터 GUI 슬롯 16)

선택한 재질을 모든 장비에 한 번에 적용. **사슬 제외** (총 6종).

```
     col0  col1  col2  col3  col4  col5  col6  col7  col8
row0 [유리] [유리] [유리] [유리] [안내] [유리] [유리] [유리] [유리]
row1 [유리] [가죽] [구리] [철  ] [금  ] [다이아][네더] [유리] [유리]
row2 [뒤로🔴][유리] [유리] [유리] [유리] [유리] [유리] [유리] [유리]
```

| 슬롯 | 역할 |
|---|---|
| 4 | 안내 아이콘 — 무기 타입에 따라 lore 변경 |
| 10 | 가죽 | 11 | 구리 | 12 | 철 |
| 13 | 금 | 14 | 다이아 | 15 | 네더라이트 |
| 18 | 뒤로 (`RED_STAINED_GLASS_PANE`) |

### 적용 범위

| 무기 타입 | 방어구 재질 적용 | 무기 재질 적용 |
|---|---|---|
| 검·도끼·낫 | ✅ | ✅ |
| 창·스태프·석궁 | ✅ | ❌ |

### 안내 슬롯(4) lore 예시

```
§e재질 일괄 선택
§7──────────────────
§7 선택한 재질로 방어구 + 무기 외형을 한 번에 변경합니다.
§7 (창·스태프·석궁은 방어구만 적용)
§7──────────────────
§7 현재 무기: §e{무기타입}
```
