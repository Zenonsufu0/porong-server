  2    ░  [재료]    ░   [블럭]    ░   [치장]    ░   [특수]    ░

|---|---|---|

## 2. 개별 상점 GUI (54슬롯)

모든 카테고리 동일 레이아웃. 아이템 종류만 다름.

```
행\열   1     2     3     4     5     6     7     8     9
  1   [아이템 슬롯 × 9]
  2   [아이템 슬롯 × 9]
  3   [아이템 슬롯 × 9]
  4   [아이템 슬롯 × 9]
  5   [아이템 슬롯 × 9]
  6  [판매] ░     ░    [◀]  [페이지]  [▶]   ░   [뒤로] [닫기]
```

| 슬롯 | 역할 |
|---|---|
| 0~44 | 상점 아이템 (페이지당 최대 45종) |
| 45 | 판매 버튼 |
| 48 | 이전 페이지 |
| 49 | 현재 페이지 |
| 50 | 다음 페이지 |
| 52 | 뒤로 (카테고리 허브) |
| 53 | 닫기 |

## 3. 아이템 슬롯 lore

```
§f[아이템 이름]
§7──────────
§e구매 §f500G §71개
§7좌클릭  §f1개 구매
§7Shift+클릭  §f64개 구매
§7──────────
§2판매 §f120G §71개
§7우클릭  §f판매 메뉴
```

- 판매가 없는 아이템(치장 등)은 판매 줄 생략
- 재고 없는 상점 아이템: `§c품절` 표시, 클릭 비활성

---

## 4. 구매 처리

| 클릭 | 구매량 | 차감 |
|---|---|---|
| 좌클릭 | 1개 | 골드 |
| Shift+클릭 | 64개 | 골드 × 64 |

골드 부족 시:
```
§c골드가 부족합니다. (보유: §f1,200G §c/ 필요: §f5,000G§c)
```

구매 아이템은 **인벤토리 → 자리 없으면 창고**로 자동 적재.

## 5. 판매 처리

### 판매 버튼 (슬롯 45)
아이템: `GOLD_INGOT`

```
§e판매
§7──────────
§7인벤토리 또는 창고의 아이템을
§7상점에 판매합니다.
§7
§7클릭  §f판매 목록 열기
```

클릭 시 **판매 서브 GUI (54슬롯)** 열기.

### 아이템 슬롯 우클릭 → 해당 아이템 바로 판매 메뉴로 이동

## 6. 판매 서브 GUI (54슬롯)

현재 상점 카테고리에서 판매 가능한 아이템만 표시.
인벤토리·창고 보유량 합산 표시.

```
행\열   1     2     3     4     5     6     7     8     9
  1   [판매가능 아이템 슬롯 × 9]
  2   [판매가능 아이템 슬롯 × 9]
  3   [판매가능 아이템 슬롯 × 9]
  4   [판매가능 아이템 슬롯 × 9]
  5   [판매가능 아이템 슬롯 × 9]
  6   [전량판매]  ░   ░   [◀]  [페이지]  [▶]   ░   [뒤로]  [닫기]
```

### 판매 아이템 슬롯 lore
```
§f[아이템 이름]
§2판매가: §f120G §71개
§7──────────
§7인벤토리: §f32개
§7창고: §f480개
§7합계: §f512개  →  §e61,440G
§7──────────
§7좌클릭  §f1개 판매
§7우클릭  §f64개 판매
§7Shift+클릭  §f전량 판매
```

판매 우선순위: 인벤토리 → 창고 순으로 차감.

### 전량 판매 버튼 (슬롯 45)
아이템: `GOLD_BLOCK`

```
§e전량 판매
§7──────────
§7이 카테고리의 판매 가능한
§7아이템을 전부 판매합니다.
§7예상 수익: §e123,400G
§7
§c§l[주의] 취소 불가
§7§f[클릭]
```
