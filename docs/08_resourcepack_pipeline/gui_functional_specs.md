## 1. 강화 GUI (54슬롯)

### 1-1. 흐름

```
[장비 서브 허브] → [강화] 클릭
    │
    ▼
[강화 GUI] 열림 — 오른쪽에 현재 장착 장비 5종 표시
    │
    │  오른쪽 장비 클릭
    ▼
중앙에 해당 장비 아이콘 이동 + 강화 정보(단계·성공률·가호) 자동 업데이트
    │
    │  [강화] 버튼 클릭
    ▼
강화석 자동 차감 (DB 수치 — 실물 아이템 아님) → 성공/실패 결과 표시
```

> 강화석은 DB에서 관리하는 가상 자원. 실물 아이템 드랍 아님.  
> 강화 성공률·가호 시스템: `numbers_and_open_decisions.md M-5 참조`.

### 1-2. 슬롯 배치 (0-indexed)

```
행\열   1    2    3    4    5    6    7    8    9
 1     ░    ░    ░    ░    ░    ░    ░    ░    ░
 2     ░  [선택장비]  ░  [강화단계·성공률]  ░   ░  [무기] [투구] ░
 3     ░  [강화후수치]  ░  [가호 N/천장]  ░   ░  [상의] [하의] ░
 4     ░    ░    ░   [강화버튼]  ░   ░  [신발]  ░    ░
 5     ░  [강화석잔량]  ░    ░    ░    ░    ░    ░    ░
 6    [뒤로]  ░    ░    ░    ░    ░    ░    ░  [닫기]
```

| 슬롯 | 역할 | 타입 |
|---|---|---|
| 10 | 선택된 장비 표시 | 읽기 전용 |
| 13 | 강화 단계·성공률 표시 | 읽기 전용 |
| 19 | 강화 후 ATK/DEF/HP 예측 | 읽기 전용 |
| 22 | 가호 카운터 (현재/천장) | 읽기 전용 |
| 30 | [강화] 버튼 | 클릭 |
| 37 | 강화석 잔량 표시 | 읽기 전용 |
| 15 | [무기] 선택 | 클릭 — 슬롯 10으로 이동 |
| 16 | [투구] 선택 | 클릭 |
| 24 | [상의] 선택 | 클릭 |
| 25 | [하의] 선택 | 클릭 |
| 33 | [신발] 선택 | 클릭 |

## 2. 잠재능력 큐브 GUI (54슬롯)

### 2-1. 흐름

```
[장비 서브 허브] → [잠재] 클릭
    │
    ▼
[큐브 GUI] — 상단에 장비 5종 선택 버튼
    │
    │  장비 클릭 → 왼쪽 패널에 현재 장비 + 현재 잠재 옵션 표시
    ▼
[사용] 버튼 클릭 — 큐브 1개 소모
    │
    ▼
오른쪽 패널에 새 옵션 표시 (결과 장비 아이콘 + 새 3개 잠재 라인)
    │
    ├─ [오른쪽 장비 클릭] → 새 옵션 확정 → 오른쪽 패널 초기화
    └─ [왼쪽 장비 클릭]  → 기존 옵션 유지 → [사용] 버튼 다시 활성화
                            (반복 사용 가능 — 별도 자동 반복 버튼 없음)
```

> 버튼은 [사용] 하나뿐. 왼쪽 클릭(기존 유지) 후 [사용]을 다시 누르는 것이 자연스러운 반복 흐름.

### 2-2. 슬롯 배치 (0-indexed)

```
행\열   1    2    3    4    5    6    7    8    9
 1     ░  [무기] [투구] [상의] [하의] [신발]  ░    ░    ░
 2    [현재장비]  ░    ░    ░    ░    ░    ░  [결과장비]  ░
 3    [현옵션1]  ░    ░    ░    ░    ░    ░  [새옵션1]  ░
 4    [현옵션2]  ░    ░   [사용]  ░    ░    ░  [새옵션2]  ░
 5    [현옵션3]  ░    ░    ░    ░    ░    ░  [새옵션3]  ░
 6    [뒤로]  ░    ░    ░    ░    ░    ░    ░  [닫기]
```

| 슬롯 | 역할 | 타입 |
|---|---|---|
| 1~5 | 무기·투구·상의·하의·신발 선택 | 클릭 |
| 9 | 현재 장비 (클릭 = 기존 옵션 유지, [사용] 다시 활성화) | 클릭 가능 |
| 17 | 결과 장비 (클릭 = 새 옵션 확정) | 클릭 가능 (사용 후 활성화) |
| 18 | 현 옵션1 | 읽기 전용 |
| 27 | 현 옵션2 | 읽기 전용 |
| 36 | 현 옵션3 | 읽기 전용 |
| 26 | 새 옵션1 | 읽기 전용 (사용 후 표시) |
| 35 | 새 옵션2 | 읽기 전용 |
| 44 | 새 옵션3 | 읽기 전용 |
| 31 | [사용] 버튼 — 큐브 1개 소모 | 클릭 |
| 45 | [뒤로] | |
| 53 | [닫기] | |

## 3. 각인 GUI (27슬롯)

### 3-1. 확정 사항

- 각인 슬롯: **무기당 1개**, 동시 착용 불가
- 교체 비용: **없음**
- 각인 목록: 해당 무기 전용 각인 2종 중 선택 (보유한 각인만 표시)

### 3-2. 슬롯 배치 (0-indexed)

```
행\열   1    2    3    4    5    6    7    8    9
 1     ░    ░    ░    ░    ░    ░    ░    ░    ░
 2     ░  [현재각인]  ░    ░    ░  [각인 A]  ░  [각인 B]  ░
 3    [뒤로]  ░    ░    ░    ░    ░    ░    ░  [닫기]
```

| 슬롯 | 역할 |
|---|---|
| 10 | 현재 장착 각인 (비어있으면 "없음" 표시) |
| 14 | 각인 A (무기 전용 각인 1) 클릭 → 즉시 교체 |
| 16 | 각인 B (무기 전용 각인 2) 클릭 → 즉시 교체 |
| 18 | [뒤로] |
| 26 | [닫기] |

> 현재 장착 중인 각인을 다시 클릭하면 해제.  
> 각인 미보유 시 해당 슬롯에 자물쇠 아이콘 표시.

## 4. 전승 GUI (54슬롯)

### 4-1. 확정 사항

| 항목 | 내용 |
|---|---|
| 소스 | 흔적 (이전할 장비 — 소모됨) |
| 대상 | 전승받을 장비 (인벤토리에서 드래그) |
| 전승권 | 기본 전승권·등급 전승권·옵션 전승권 3종 중 하나 슬롯에 배치 |
| 전승권 비용 | 각 5,000G (상점 구매) |
| M-4 확정 참조 | `numbers_and_open_decisions.md §M-4` |

| 전승권 종류 | 이전 내용 |
|---|---|
| **기본 전승권** | 등급 + 서브스탯 옵션 전체 이전 |
| **등급 전승권** | 등급(커먼~레전더리)만 이전 |
| **옵션 전승권** | 서브스탯 옵션만 이전 |

### 4-2. 흐름

```
[흔적 슬롯]에 이전할 장비 배치
[대상 슬롯]에 전승받을 장비 배치
[전승권 슬롯]에 3종 중 하나 배치
    │
    ▼
미리보기 자동 갱신 (대상 장비 + 전승 후 결과)
    │
    │  [전승] 클릭
    ▼
흔적 장비 소멸 + 대상 장비에 옵션 이전
```

### 4-3. 슬롯 배치 (0-indexed)

```
행\열   1    2    3    4    5    6    7    8    9
 1     ░    ░    ░    ░    ░    ░    ░    ░    ░
 2     ░  [흔적]  ░   [→]  ░  [대상]  ░    ░    ░
 3     ░    ░    ░  [전승권] ░    ░    ░    ░    ░
 4     ░  [전승 전 옵션]  ░   ░   ░  [전승 후 옵션]  ░   ░    ░
 5     ░    ░    ░    ░  [전승버튼]  ░    ░    ░    ░
 6    [뒤로]  ░    ░    ░    ░    ░    ░    ░  [닫기]
```

| 슬롯 | 역할 | 타입 |
|---|---|---|
| 10 | [흔적] 배치 슬롯 — 소스 장비 드래그 | 입력 가능 |
| 13 | [→] 장식 아이콘 | 읽기 전용 |
| 14 | [대상] 배치 슬롯 — 전승받을 장비 드래그 | 입력 가능 |
| 21 | [전승권] 슬롯 — 기본·등급·옵션 중 1개 | 입력 가능 |
| 19 | 현재 옵션 미리보기 | 읽기 전용 |
| 23 | 전승 후 옵션 미리보기 | 읽기 전용 |
| 31 | [전승] 버튼 | 클릭 |
| 45 | [뒤로] | |
| 53 | [닫기] | |

### 5-2. 슬롯 배치 (0-indexed, 27슬롯)

```
행\열   1    2    3    4    5    6    7    8    9
 1     ░    ░    ░    ░    ░    ░    ░    ░    ░

```

| 슬롯 | 역할 |
|---|---|
| 9~13 | 대기열 슬롯 1~5 (작위 올라갈수록 14·15·16… 활성화) |
| 15, 16 | 레시피 선택 버튼 (카테고리: 큐브·흔적 승급) |

| 26 | [닫기] |

> 대기열 슬롯 클릭 → 54슬롯 레시피 선택 팝업 열림.  
> 제작 중인 슬롯 클릭 → 남은 시간 표시, 취소 가능 (재료 반환).

### 5-3. 레시피 목록 (확정)

| 레시피 | 재료 | 결과 | 제작 시간 | 비고 |
|---|---|---|:---:|---|
| 잠재능력 큐브 | 큐브 조각 ×10 | 잠재능력 큐브 ×1 | 60분 | 기존 확정 |
| 언커먼 흔적 합성 | 커먼 흔적 ×5 | 언커먼 흔적 ×1 | 15분 | |
| 레어 흔적 합성 | 언커먼 흔적 ×3 | 레어 흔적 ×1 | 25분 | |
| 에픽 흔적 합성 | 레어 흔적 ×3 | 에픽 흔적 ×1 | 50분 | |
