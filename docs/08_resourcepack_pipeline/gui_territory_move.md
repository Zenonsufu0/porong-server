# 영지 이동 GUI 설계

> **[STATUS: REFERENCE]** — 영지 이동 GUI 상세 참조. 공식 영지 기준은 `../05_island_farm_system/CANON.md`가 우선.

> 작성일: 2026-05-23
> 인벤토리: 27슬롯 (3×9), 바닐라 아이템 아이콘 전용
> 접근 조건: 전투 중 불가 / 필드 내 허용 / 영지 내 허용

---

## 1. 진입 슬롯 (영지 서브 허브 슬롯 9)

아이템: `COMPASS`

```
§f영지 이동
§7──────────
§7§e[영지명]  §8([작위])
§7방문자 §f[N]명
§7──────────
§7클릭 → 내 영지 / 공개 영지 목록
```

클릭 시 영지 이동 GUI 열기.

---

## 2. 슬롯 레이아웃

```
행\열   1          2~9
  1   [내 영지]    ░  ░  ░  ░  ░  ░  ░  ░
  2   [공개 1]   [공개 2] [공개 3] [공개 4] [공개 5] [공개 6] [공개 7] [공개 8] [공개 9]
  3   [뒤로]      ░        ░     [◀ 이전] [현재/전체] [다음 ▶]  ░        ░        ░
```

| 슬롯 | 역할 |
|---|---|
| 0 | 내 영지 (항상 고정) |
| 1~8 | 빈 구분 공간 (GRAY_STAINED_GLASS_PANE) |
| 9~17 | 공개 영지 목록 (9개/페이지) |
| 18 | 뒤로 (영지 서브 허브) |
| 21 | ◀ 이전 페이지 |
| 22 | 현재 페이지 표시 |
| 23 | 다음 페이지 ▶ |

---

## 3. 내 영지 슬롯 (슬롯 0)

아이템: `PLAYER_HEAD` (본인 스킨)

```
§6[내 영지]
§7──────────
§7§e[영지명]
§7작위: §f[작위명]
§7방문자: §f[N]명
§7──────────
§7클릭 → 내 영지 스폰으로 이동
```

클릭 시 GUI 닫고 즉시 영지 스폰 텔레포트.

---

## 4. 공개 영지 슬롯 (슬롯 9~17)

### 전체 공개 영지

아이템: 해당 영지 소유자 `PLAYER_HEAD`

```
§f[영지명]
§7소유자: §f[플레이어 이름]
§7작위: §f[작위명]
§7──────────
§7방문자 §f[N]명
§7클릭 → 방문
```

> 비공개(방문설정: 비공개) 영지는 목록에 표시되지 않음.

### 빈 슬롯 (공개 영지 없음 또는 페이지 초과)

아이템: `GRAY_STAINED_GLASS_PANE`

```
§8공개된 영지가 없습니다.
```

---

## 5. 목록 정렬 기준

방문자 수 많은 순 → 작위 높은 순 → 이름 가나다 순 (3중 정렬).

---

## 6. 페이지 네비게이션 (슬롯 21~23)

### 이전 페이지 (슬롯 21)

아이템: `ARROW`

```
§7◀ 이전 페이지
§8([현재 페이지] / [전체 페이지])
```

1페이지에서 클릭 시 → 반응 없음 (아이템: `GRAY_DYE`, lore `§8첫 페이지입니다.`)

### 현재 페이지 표시 (슬롯 22)

아이템: `PAPER`

```
§f[현재 페이지] / [전체 페이지]
```

클릭 무반응.

### 다음 페이지 (슬롯 23)

아이템: `ARROW`

```
§7다음 페이지 ▶
§8([현재 페이지] / [전체 페이지])
```

마지막 페이지에서 클릭 시 → 반응 없음 (아이템: `GRAY_DYE`, lore `§8마지막 페이지입니다.`)

---

## 7. 방문 처리 흐름

```
공개 영지 슬롯 클릭
  │
  ├─ 전투 태그 확인 → 있으면 §c[영지] 전투 중에는 이동할 수 없습니다. 중단
  │
  ├─ 방문 설정 재확인 (전체공개 / 친구만 여부)
  │    └─ 변경됐으면 §c[영지] 해당 영지에 입장할 수 없습니다. 중단
  │
  └─ GUI 닫고 해당 영지 스폰 텔레포트
```

## 8. 구현 참조

| 항목 | 내용 |
|---|---|
| 열기 트리거 | 영지 서브 허브 슬롯 9 클릭 |
| 접근 제한 | 전투 중 차단 / 필드 내 허용 |
| 목록 조회 | `island_settings` WHERE `visit_type = 'public'` |
| 정렬 | 방문자 수 DESC → 작위 DESC → 이름 ASC |
| 페이지 크기 | 9개/페이지 |
| 텔레포트 | IridiumSkyblock API `IslandManager#teleportHome(player, island)` |
