# 훈련장 GUI 설계

> **[STATUS: REFERENCE]** — 훈련장 GUI 상세 참조. 공식 리소스팩 기준은 `index.md`가 우선.
>
> 트리거: 탐험 서브 허브 슬롯 16 [훈련장] 클릭

---

## 1. 훈련장 서브 메뉴 (27슬롯)

```
행\열  1      2      3         4      5         6    7    8    9
 1    ░      ░      ░         ░      ░         ░    ░    ░    ░
 2   [이동]   ░   [방어력설정]  ░   [훈련기록]   ░    ░    ░    ░
 3   [뒤로]   ░      ░         ░      ░         ░    ░    ░    ░
```

| 슬롯 | 내용 | 아이콘 | 비고 |
|---|---|---|---|
| 9 | 훈련장 이동 | `TARGET` | 전투 중 차단 |
| 11 | 방어력 설정 | `SHIELD` | 훈련장 구역 내에서만 활성 |
| 13 | 훈련 기록 | `WRITTEN_BOOK` | 어디서나 열람 가능 |
| 18 | 뒤로 | `ARROW` | 탐험 서브 허브로 |
| 나머지 | 장식 | `BLACK_STAINED_GLASS_PANE` | |

**슬롯 11 — 방어력 설정 (비활성 시 lore)**
```
§7방어력 설정
§7─────────────────
§c훈련장에 있을 때만 설정할 수 있습니다.
```

## 2. 방어력 설정 GUI (27슬롯)

접근 조건: 훈련장 구역 내에서만. 타 구역 클릭 시 채팅 안내 후 GUI 닫힘.

```
행\열  1        2        3        4        5        6         7    8    9
 1    ░        ░        ░        ░        ░        ░         ░    ░    ░
 2   [DEF 0] [DEF 100] [DEF 200] [DEF 300] [DEF 400] [커스텀]  ░    ░    ░
 3   [뒤로]   ░        ░        ░        ░        ░         ░    ░    ░
```

| 슬롯 | DEF | 유효 방어율 | 아이콘 |
|---|---|---|---|
| 9 | 0 | 0% | `GLASS` |
| 10 | 100 | 약 33% | `WHITE_STAINED_GLASS_PANE` |
| 11 | 200 | 약 50% | `YELLOW_STAINED_GLASS_PANE` |
| 12 | 300 | 약 60% | `ORANGE_STAINED_GLASS_PANE` |
| 13 | 400 | 약 67% | `RED_STAINED_GLASS_PANE` |
| 14 | 커스텀 | 직접 입력 | `ANVIL` |
| 18 | 뒤로 | — | `ARROW` |
| 나머지 | 장식 | `BLACK_STAINED_GLASS_PANE` | |

> 현재 설정된 슬롯: 아이콘 재질을 `LIME_STAINED_GLASS_PANE`으로 하이라이트.

**DEF 프리셋 버튼 lore (예: DEF 200)**
```
§e방어력 200
§7─────────────────
§7 유효 방어율 : §e약 50%
§7─────────────────
§7 [클릭] 이 방어력으로 설정
```

**커스텀 입력 버튼 lore**
```
§e커스텀 방어력
§7─────────────────
§7 0 ~ 999 범위로 직접 입력합니다.
§7─────────────────
§7 [클릭] Anvil 입력창 열기
```

> Anvil 입력창 기본값: 현재 설정된 DEF 수치. 유효 범위 0~999; 범위 초과 시 채팅으로 안내.

## 3. 훈련 기록 GUI (54슬롯)

트리거: 훈련장 서브 메뉴 슬롯 13 클릭. 어디서나 열람 가능.  
54개 슬롯 전체가 기록 슬롯. 닫기(ESC)로 이전 GUI로 복귀.

```
슬롯 0  : 최신 기록
슬롯 53 : 가장 오래된 기록
55번째 기록 추가 시 슬롯 53 자동 삭제
```

**기록 슬롯 lore (아이콘: `PAPER`)**
```
§e훈련 #1  §8(최신)
§7─────────────────
§7 날짜       : §f2026-05-23
§7 방어력     : §e200  §8(유효 약 50%)
§7 총 피해량  : §f1,248,300
§7 DPS       : §e20,805
§7─────────────────
§7 장비 세팅
§7 무기  : §6균열의 검  §e+18강
§7 평균 IL: §e90
§7 각인   : §b파괴의 인장 Lv3 · §b속도의 각인 Lv2
```

**기록 없음 (슬롯 0에 안내 아이템)**

아이콘: `GRAY_STAINED_GLASS_PANE`
```
§7훈련 기록 없음
§7─────────────────
§7 훈련장에서 허수아비를 공격하면
§7 1분간의 DPS 기록이 저장됩니다.
```

## 4. 훈련 메커니즘

| 항목 | 내용 |
|---|---|
| 허수아비 HP | 무한 (피격 시 HP 변화 없음) |
| 타이머 시작 | 첫 데미지 인식 순간 |
| 측정 시간 | 1분 (60초) |
| 종료 조건 | 1분 경과 후 10초간 추가 데미지 없음 |
| 종료 알림 | Title: `§e훈련 종료` / Subtitle: `§7결과가 기록되었습니다.` |
| 타이머 UI | Action Bar에 남은 시간 표시 (측정 중에만) |
| 기록 저장 | 종료 확정 시 자동 저장 |

**측정 중 Action Bar**
```
§e측정 중  §f42초 남음  §7│  §e총 피해: §f524,000  §7│  §e현재 DPS: §f18,600
```

**종료 후 10초 대기 중 Action Bar**
```
§7대기 중  §c10초 안에 공격 없으면 종료  §7│  §e총 피해: §f1,248,300
```

## 5. 구현 참조

| 항목 | 내용 |
|---|---|
| 트리거 | 탐험 서브 허브 슬롯 16 클릭 |
| 허수아비 구현 | MythicMobs 커스텀 몹, HP=2,147,483,647, 공격 없음 |
| 방어력 적용 | 데미지 계산 시 설정된 DEF 값으로 override |
| 타이머 | `BukkitRunnable` 매초 실행, 60초 후 10초 비딜 체크 |
| 기록 DB | `training_records (player_uuid, date, def_value, total_dmg, dps, equipment_snapshot)` |
| 기록 한도 | 신규 저장 전 `COUNT`가 54 이상이면 oldest 1건 삭제 후 저장 |
| 방어력 설정 비활성 | 훈련장 구역 외 클릭 시 채팅 메시지 후 처리 중단 |
