# 아이템 lore 표시 포맷

> **[STATUS: REFERENCE]** — 무기·방어구 아이템 hover lore 표준 포맷.  
> 수치 기준: `combat_balance_v2.md` (고정 스탯) / `item_grade_substat_v1.md` (등급·보조스탯·잠재)

---

## 1. 메인 장비 (시작 5슬롯 — 강화 대상)

### 무기

**아이템 이름 (displayName)**

```
§r{무기종류} {이름} §7+{n}강
```

예) `§r검 용사의 검 §7+15강`

**lore**

```
§7등급: {등급색}{등급명}
§7──────────────────
§7 공격력: §e{ATK}
§7──────────────────
§7보조스탯:
{보조스탯 라인 × 0~3}
§7──────────────────
§7잠재능력 {잠재등급색}[{잠재등급명}]
{잠재 라인 × 3}
```

예시 (레전더리 장비 / 레전더리 잠재 / 보조스탯 2라인)

```
§7등급: §d레전더리
§7──────────────────
§7 공격력: §e312
§7──────────────────
§7보조스탯:
§a 치명타 확률 +15%
§a 스킬 피해 +8%
§7──────────────────
§7잠재능력 §d[레전더리]
§a 치명타 확률 +25%
§a 스킬 피해 +15%
§a 핵심기 피해 +20%
```

### 방어구 (투구·상의·하의·신발)

**아이템 이름**

```
§r{부위} {이름}
```

예) `§r투구 용사의 투구`  (방어구는 강화 단계 없음)

**고정 스탯 — 부위별**

| 부위 | 표시 항목 | 0강 기본값 |
|---|---|---|
| 투구 | 체력 | 400 |
| 상의 | 체력 + 방어력 | HP 150 / DEF 15 |
| 하의 | 방어력 | 30 |
| 신발 | 체력 + 방어력 | HP 100 / DEF 7 |

> 강화 시 보유 스탯만 증가. 투구는 HP만, 하의는 DEF만 증가.

**lore**

```
§7등급: {등급색}{등급명}
§7──────────────────
§7 체력  : §e{HP}     ← HP 있는 부위만 (투구·상의·신발)
§7 방어력: §e{DEF}    ← DEF 있는 부위만 (상의·하의·신발)
§7──────────────────
§7보조스탯:
{보조스탯 라인 × 0~3}
§7──────────────────
§7잠재능력 {잠재등급색}[{잠재등급명}]
{잠재 라인 × 3}
```

**예시 — 투구 (HP만)**

```
§7등급: §5에픽
§7──────────────────
§7 체력  : §e432
§7──────────────────
§7보조스탯:
§a 치명타 확률 +7%
§7──────────────────
§7잠재능력 §9[레어]
§a 치명타 확률 +10%
§a HP% +4%
§a 이동기 피해 +6%
```

**예시 — 상의 (HP + DEF)**

```
§7등급: §d레전더리
§7──────────────────
§7 체력  : §e162
§7 방어력: §e16
§7──────────────────
§7보조스탯:
§a 스킬 피해 +8%
§7──────────────────
§7잠재능력 §d[레전더리]
§a 스킬 피해 +15%
§a HP% +10%
§a 특수기 피해 +20%
```

**예시 — 하의 (DEF만)**

```
§7등급: §9레어
§7──────────────────
§7 방어력: §e31
§7──────────────────
§7잠재능력 §9[레어]
§a DEF% +3%
§a 쿨타임 감소 +2%
§a 보스 피해 +2%
```

## 2. 장비의 흔적 (드롭 아이템 — 강화 불가)

**아이템 이름**

```
§r{부위}의 흔적
```

예) `§r무기의 흔적` / `§r투구의 흔적`

**lore**

```
§7등급: {등급색}{등급명}
§7──────────────────
§7 공격력: §e{ATK}          ← 무기의 흔적만
§7 체력  : §e{HP}           ← HP 있는 부위만 (투구·상의·신발)
§7 방어력: §e{DEF}          ← DEF 있는 부위만 (상의·하의·신발)
§7──────────────────
§7보조스탯:
{보조스탯 라인 × 0~3}
§7──────────────────
§c 강화 불가 · 전승 소스로 사용 가능
```

> 흔적에는 잠재능력 없음. 보조스탯만 존재.  
> 고정 스탯은 부위별로 동일 규칙 적용 (투구=HP만, 하의=DEF만 등).

## 3. 등급 색상 코드

| 등급 | 색상 코드 | 표시 |
|---|---|---|
| 커먼 | `§7` | §7커먼 |
| 레어 | `§9` | §9레어 |
| 에픽 | `§5` | §5에픽 |
| 유니크 | `§6` | §6유니크 |
| 레전더리 | `§d` | §d레전더리 |

## 4. 보조스탯 없을 때

보조스탯 0라인이면 해당 블록 전체 생략 (구분선 포함).

```
§7등급: §7커먼
§7──────────────────
§7 공격력: §e80
§7──────────────────
§7잠재능력 §7[커먼]
§7 ATK% +1%
§7 방어력 무시% +1%
§7 쿨타임 감소% +1%
```

## 6. 무기 변경 GUI — 무기 선택 슬롯 lore

> `gui_equipment_detail.md §2` 무기 변경 GUI의 각 무기 종류 슬롯에 표시.  
> 인게임 아이템이 아닌 GUI 슬롯 전용 lore.

**비장착 무기**

```
§e{무기종류}
§7──────────────────
§7 강화   : §e+{n}강
§7 등급   : {등급색}{등급명}
§7 공격력 : §e{ATK}
§7──────────────────
§7 잠재   : {잠재색}{옵션1} / {옵션2} / {옵션3}
§7 각인   : §a{각인명}
§7 재질   : §7{재질명}
§7 스킬   : §7[LC]{스킬명} [RC]{스킬명} [SRC]{스킬명} [F]{스킬명}
§7──────────────────
§7 [클릭] 이 무기로 교체
§c 기존 무기는 회수됩니다.
```

**장착 중인 무기**

```
§e{무기종류}  §a[장착 중]
§7──────────────────
§7 강화   : §e+{n}강
§7 등급   : {등급색}{등급명}
§7 공격력 : §e{ATK}
§7──────────────────
§7 잠재   : {잠재색}{옵션1} / {옵션2} / {옵션3}
§7 각인   : §a{각인명}
§7 재질   : §7{재질명}
§7 스킬   : §7[LC]{스킬명} [RC]{스킬명} [SRC]{스킬명} [F]{스킬명}
```

예시 (검, 장착 중)

```
§e검  §a[장착 중]
§7──────────────────
§7 강화   : §e+15강
§7 등급   : §d레전더리
§7 공격력 : §e312
§7──────────────────
§7 잠재   : §d치명타 확률 +25% / 스킬 피해 +15% / 핵심기 +20%
§7 각인   : §a집중 강타 A
§7 재질   : §7다이아몬드
§7 스킬   : §7[LC]회전베기 [RC]돌진 [SRC]폭풍 [F]처형
```

> 잠재 미부여 시: `§8미부여` 표시.  
> 각인 미선택 시: `§8없음` 표시.
