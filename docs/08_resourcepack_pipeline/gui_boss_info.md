# 보스 정보 GUI 설계 (54슬롯)

> **[STATUS: DRAFT]** — 보스 정보 GUI 재작성 초안. 영지·보스·필드 GUI는 세부 사항 재작성 필요 상태다.

> 기반: `gui_hub_structure.md`  
> 복원: `_archive/gui_functional_specs.md` §7 (2026-05-23) — 드랍 lore 현행 기준 수정

---

## 목적

보스 서브 허브 슬롯 13 [보스 정보] 클릭 시 열림.  
접근 제한 없음 — 전투 중·필드 내·어디서든 열람 가능.  
보스 선택 → 우측에 HP/DEF/패턴 개요/드랍 미리보기 표시.

## 아이템 lore 예시

**좌측 보스 선택 버튼 (예: 균열 파수꾼)**
```
§e균열 파수꾼
§7─────────────────
§7 위치  : §e고대 성벽 잔해 (필드5)
§7 권장  : §e15강 ~ 18강
§7─────────────────
§7 [클릭] 상세 정보 보기
```

**슬롯 4 — 보스명·분류 (필드보스)**
```
§c[필드보스] 균열 파수꾼
§7 고대 성벽 잔해
§7─────────────────
§7 리젠 방식  : §e필드 리젠형 (30~60분)
§7 보상 방식  : §e기여도 기반
```

**슬롯 4 — 보스명·분류 (시즌보스)**
```
§5[시즌보스 2]
§7 오염된 수로의 지배자
§7─────────────────
§7 입장 방식  : §e파티형 (최대 3인)
§7 목표 클리어율: §e45%
```

**슬롯 13 — 기본 정보 (필드보스 예시: 균열 파수꾼)**
```
§e기본 정보
§7─────────────────
§7 HP    : §a600,000 ~ 720,000
§7 DEF   : §e200  §7(유효 방어율 약 50%)
§7 제한 시간: §c15분
§7 인원   : §e제한 없음
```

**슬롯 13 — 기본 정보 (시즌보스 예시: 시즌보스 2, 3인 기준)**
```
§e기본 정보 §7(3인 기준)
§7─────────────────
§7 HP    : §a확인 필요 §7(인원 스케일링 적용)
§7   1인 : §70.45×  2인 : 0.72×  3인 : 1.00×
§7 DEF   : §e확인 필요
§7 제한 시간: §c15분
§7 인원   : §e1 ~ 3인
```

**슬롯 14 — 권장 강화 (필드보스)**
```
§e권장 강화 구간
§7─────────────────
§7 입문  : §e15강
§7 적정  : §e16 ~ 17강
§7 여유  : §e18강 이상
```

**슬롯 14 — 난이도 (균열왕)**
```
§c난이도 정보 §4[최종보스]
§7─────────────────
§7 목표 클리어율 : §c10%
§7 최소 인원    : §c2인 필수
§7 제한 시간    : §c10분
§7 입장 권장    : §c22강 + 레전더리 잠재
```

**슬롯 22~25 — 패턴 개요**
```
§e⚔ 전방 강타
§7─────────────────
§7 정면 근거리를 내리칩니다.
§a 대응: 옆으로 빠르게 이동.
```
```
§e★ 시그니처 패턴
§7─────────────────
§7 [보스별 고유 패턴 설명]
§c 실패 시 즉사 또는 클리어 불가.
```
```
§e⟳ 페이즈 전환
§7─────────────────
§7 HP 특정 구간 도달 시 전환.
§7 무력화 딜타임 발생.
§a 대응: 전환 전 피해 최대화.
```

**슬롯 31~34 — 드랍 미리보기 (필드보스 예시)**
```
§6[확정] 전장의 파편 (필드 공통 재료)
§7─────────────────
§7 드랍 조건: 기여도 참여 전원
```
```
§6[확정] 균열 파수꾼의 결정 (보스 핵심 재료)
§7─────────────────
§7 드랍 조건: 기여도 상위 3인
```
```
§e[확률] 강화석
§7─────────────────
§7 드랍 조건: 기여도 비례
```
```
§e[확률] 큐브 조각
§7─────────────────
§7 드랍 조건: 기여도 상위 1인
```

**슬롯 31~34 — 드랍 미리보기 (시즌보스)**
```
§d[확정] 장비의 흔적 (에픽~레전더리)
§7─────────────────
§7 드랍 조건: 파티원 전원
§7 강화석 드랍 없음
```

## 플러그인 처리

```
InventoryClickEvent (보스 정보 GUI)
  └─ 좌측 보스 버튼 (슬롯 0·1·2·9·10·18·19·20·27) 클릭
      ├─ 선택 보스 하이라이트 갱신 (LIME_STAINED_GLASS_PANE)
      └─ 우측 슬롯 (4·13·14·22~25·31~34) 해당 보스 데이터로 갱신

GUI 열릴 때 (onOpen):
  - 기본 선택 보스 없음 → 우측에 안내 아이템 표시
```
