## 1. 슬롯 레이아웃

```
행\열   1      2      3      4        5      6    7    8       9
  1    ░      ░      ░      ░         ░      ░    ░    ░       ░
  2   [보스1] [보스2] [보스3]  ░      [최종보스]  ░    ░    ░       ░
  3   [뒤로]   ░      ░      ░         ░      ░    ░    ░   [닫기]
```

| 슬롯 | 역할 |
|---|---|
| 9 | 시즌보스 1 — 대지의 폭군 |
| 10 | 시즌보스 2 — 강철의 심판자 |
| 11 | 시즌보스 3 — 심연의 군주 |
| 13 | 시즌 최종보스 — 균열의 군주 |
| 18 | 뒤로 (보스 서브 허브) |
| 26 | 닫기 |

빈 슬롯: `GRAY_STAINED_GLASS_PANE`.

## 2. 보스 슬롯 lore 형식

## 3. 보스 상세 뷰 (클릭 후 갱신)

선택한 보스 1개를 중앙에 확대 표시하는 뷰로 전환.

```
행\열   1      2      3      4      5      6      7      8      9
  1   [드랍1] [드랍2] [드랍3] [드랍4] [드랍5]  ░      ░      ░      ░
  2    ░      ░   [보스 아이콘]  ░      ░    [권장IL] [타이머] [인원]   ░
  3   [뒤로]   ░      ░      ░      ░      ░      ░      ░   [닫기]
```

| 슬롯 | 역할 |
|---|---|
| 0~4 | 보스 주요 드랍 아이템 (lore에 확률 표기) |
| 11 | 보스 메인 아이콘 (HP/DEF/패턴 요약) |
| 14 | 권장 IL 정보 |
| 15 | 전투 타이머 |
| 16 | 최대 인원 |
| 18 | 뒤로 (목록 복귀) |
| 26 | 닫기 |

### 드랍 슬롯 lore 예시 (슬롯 0~4)

```
§e강화석 파편
§7──────────
§7확정 드랍: §f×50개
§8(클리어 인원 무관 고정 지급)

§b장비의 흔적 (A급)
§7──────────
§7드랍률: §f100%  §7수량: §f1개
§7등급: §fS 20% / §bA 60% / §7레어 20%

§d레전더리 흔적
§7──────────
§7드랍률: §f0.5%
§7최고 등급 — 시즌 내 수량 희귀
```
