## 1. 슬롯 레이아웃

```
행\열   1          2       3       4       5       6       7       8       9
  1   [자동심기]  [씨앗1] [씨앗2] [씨앗3] [씨앗4] [씨앗5] [씨앗6]   ░       ░
  2      ░      [작물1] [작물2] [작물3] [작물4] [작물5] [작물6]   ░       ░
  3   [뒤로]      ░       ░       ░       ░       ░       ░       ░    [닫기]
```

| 슬롯 | 역할 |
|---|---|
| 0 | 자동심기 ON/OFF 토글 |
| 1~6 | 씨앗 재고 (종류별 1슬롯) |
| 10~15 | 수확 현황 — 창고 보관 작물 수량 |
| 18 | 뒤로 (영지 서브 허브) |
| 26 | 닫기 |

빈 슬롯: GRAY_STAINED_GLASS_PANE.

## 3. 씨앗 슬롯 (슬롯 1~6)

씨앗 종류별 1슬롯. 재고 수량 lore 표시.

```
§f[씨앗 이름]
§7재고: §f128개
§7──────────
§7자동심기 ON 시 수확 후 자동 소모
§7재고 0개 → 자동심기 중단
§7
§7좌클릭  §f64개 보충 (창고 → 씨앗 재고)
§7우클릭  §f전량 반환 (씨앗 재고 → 창고)
```

재고 0개일 때:
```
§f[씨앗 이름]
§c재고: §f0개 §c(심기 중단)
§7──────────
§7좌클릭  §f64개 보충 (창고에서 가져오기)
```

씨앗 종류 기본 6종:

| 슬롯 | 아이템 | 씨앗 |
|---|---|---|
| 1 | `WHEAT_SEEDS` | 밀 씨앗 |
| 2 | `POTATO` | 감자 |
| 3 | `CARROT` | 당근 |
| 4 | `BEETROOT_SEEDS` | 비트루트 씨앗 |
| 5 | `MELON_SEEDS` | 수박 씨앗 |
| 6 | `PUMPKIN_SEEDS` | 호박 씨앗 |

## 4. 수확 현황 슬롯 (슬롯 10~15)

창고에 보관 중인 작물 수량 요약 표시. 창고 GUI와 동기화.

```
§f[작물 이름]
§7창고 보관: §f1,234개
§7──────────
§7§8(창고에서 직접 출금하려면
§8 영지 창고를 이용하세요)
```

수량 0이면 슬롯 비움.

---

## 5. 자동심기 동작 규칙

1. 자동심기 ON + 씨앗 재고 ≥ 1 → 수확 후 즉시 재심기
2. 씨앗 재고 0 → 자동심기 중단, 플레이어에게 알림:
   ```
   §e[영지] §f[씨앗 이름] 재고가 소진되어 자동 심기가 중단됐습니다.
   ```
3. 자동 재배기에서 수확된 작물은 창고로 직접 입금
4. 플레이어 오프라인 상태에서도 동작
