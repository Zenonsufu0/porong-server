## 외형 표시 규칙

- 치장 슬롯에 아이템이 있으면 → 치장 아이템 외형으로 표시
- 치장 슬롯이 비어있으면 → 실제 장착 장비 외형으로 표시
- 숨김 상태면 → 해당 부위 외형 없음

## 장비 지급 정책

- 무기·방어구 최초 지급 시 인벤토리가 아닌 **캐릭터 슬롯에 직접 배치**
- 캐릭터 슬롯 배치 후 자동 반영:
  - 방어구(투구·상의·하의·신발) → 바닐라 방어구 슬롯에 자동 장착
  - 무기 → 인벤토리(핫바)에 자동 지급
- **버리기 불가** — 드롭 이벤트 cancel
- **방어구 빼기 불가** — 장착 슬롯에서 꺼내기 불가 (무기는 강화·전승 목적으로 교체 가능)
