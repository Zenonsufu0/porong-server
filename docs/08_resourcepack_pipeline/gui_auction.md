# 경매장 GUI 설계

> **[STATUS: REFERENCE]** — 경매장 GUI 상세 참조. 공식 리소스팩 기준은 `index.md`가 우선.

> 작성일: 2026-05-17 / 수정: 2026-05-23  
> 방식: 즉시구매 전용  
> 수수료: 판매가의 5% (판매 완료 시 차감)  
> 만료: 등록 후 3일, 미판매 시 창고 반환  
> 접근 조건: 영지 내 또는 필드 / 전투 중 불가

## 1. 경매장 메인 GUI (54슬롯)

```
행\열   1        2       3       4       5       6       7       8       9
row0  [등록하기] [내목록] [전체] [흔적]  [재료]  [치장]  [기타]   ░    [정렬]
row1  [아이템 슬롯 × 9]
row2  [아이템 슬롯 × 9]
row3  [아이템 슬롯 × 9]
row4  [아이템 슬롯 × 9]
row5  [뒤로]    ░      ░    [◀]  [페이지]  [▶]    ░      ░      ░
```

| 슬롯 | 역할 |
|---|---|
| 0 | 등록하기 (`GOLD_INGOT`) |
| 1 | 내 등록 목록 (`PLAYER_HEAD`) |
| 2~6 | 카테고리 필터 (전체/흔적/재료/치장/기타) |
| 7 | `BLACK_STAINED_GLASS_PANE` |
| 8 | 정렬 토글 (`HOPPER`) |
| 9~44 | 등록된 아이템 (페이지당 최대 36개) |
| 45 | 뒤로 (`ARROW`) |
| 48 | 이전 페이지 (`ARROW`) |
| 49 | 현재 페이지 (`PAPER`) |
| 50 | 다음 페이지 (`ARROW`) |
| 46~47, 51~53 | `GRAY_STAINED_GLASS_PANE` |

빈 아이템 슬롯(9~44): `GRAY_STAINED_GLASS_PANE`.

### 카테고리 필터

| 슬롯 | 카테고리 | 아이템 | 포함 품목 |
|---|---|---|---|
| 2 | 전체 | `COMPASS` | 모든 등록 아이템 |
| 3 | 흔적 | `PRISMARINE_CRYSTALS` | 장비의 흔적 5종 + 고대흔적 4종 |
| 4 | 재료 | `CRAFTING_TABLE` | 정수류·약초·합금 등 소모 재료 |
| 5 | 치장 | `DIAMOND` | 치장 무기 |
| 6 | 기타 | `CHEST` | 포션·만찬·기타 |

현재 선택된 필터: `LIME_STAINED_GLASS_PANE`.

## 5. 등록 GUI (27슬롯)

```
행\열   1        2    3    4      5      6    7    8    9
row0  [아이템]   ░    ░    ░      ░      ░    ░    ░    ░
row1     ░       ░    ░  [가격]   ░   [확인]  ░    ░    ░
row2  [뒤로]     ░    ░    ░      ░      ░    ░    ░    ░
```

| 슬롯 | 역할 |
|---|---|
| 0 | 등록 아이템 슬롯 |
| 12 | 가격 설정 |
| 14 | 등록 확인 |
| 18 | 뒤로 (`ARROW`) |
| 나머지 | `GRAY_STAINED_GLASS_PANE` |

### 등록 흐름

1. 등록 GUI 열림 — 인벤토리가 GUI 아래에 함께 표시됨
2. 인벤토리에서 아이템을 집어 **슬롯 0에 올림** → 아이템 배치
3. 슬롯 12 클릭 → Anvil GUI에서 가격 숫자 입력 후 확인
4. 아이템 + 가격 모두 설정 시 슬롯 14 활성화 → 클릭 시 등록
5. GUI 닫으면 슬롯 0 아이템 인벤토리 자동 반환

### 등록 아이템 슬롯 (슬롯 0)

비어있을 때 — 아이템: `WHITE_STAINED_GLASS_PANE`
```
§7등록할 아이템
§7──────────
§7인벤에서 아이템을 집어
§7이 슬롯에 올려주세요.
```

아이템 배치 후:
```
§f[아이템 이름]
§7──────────
§83일 평균시세: §74,820G
§8(참고용)
§7──────────
§7클릭 → 아이템 회수
```

### 가격 설정 (슬롯 12)

미설정 — 아이템: `GRAY_STAINED_GLASS_PANE`
```
§f가격 설정
§7현재: §c미설정
§7클릭 → 가격 입력
```

설정 후 — 아이템: `GOLD_NUGGET`
```
§f가격 설정
§7현재: §e5,000G
§7수수료(5%): §f250G
§7실수령: §f4,750G
§7──────────
§7클릭 → 가격 변경
```

### 등록 확인 (슬롯 14)

아이템·가격 미설정: `BLACK_STAINED_GLASS_PANE` (비활성)

설정 완료: `LIME_DYE`
```

§7──────────

§7수수료: §f250G §8(판매 완료 시)

```

등록 완료 채팅:

## 8. 제한 및 규칙

| 항목 | 내용 |
|---|---|
| 계정당 최대 등록 | 5개 |
| 등록 만료 | 3일 |
| 수수료 | 판매가의 5%, 판매 완료 시 차감 |
| 최소 등록가 | 100G |
| 거래 가능 품목 | 재료·흔적·치장·포션·만찬 등 (무기·방어구 거래 불가) |
| 3일 평균시세 기준 | 최근 3일 거래 완료 건의 평균 가격 |
| 판매 골드 | 오프라인 시 로그인 때 자동 지급 |
| 만료 아이템 | 창고로 자동 반환 |

## 9. 구현 참조

| 항목 | 내용 |
|---|---|
| 열기 트리거 | 영지 서브 허브 슬롯 15 클릭 |
| 접근 제한 | 비전투 |
| DB 테이블 | `auction_listings` (item_data, price, seller_uuid, listed_at, expires_at, status) |
| 평균시세 쿼리 | `SELECT AVG(price) FROM auction_listings WHERE item_type=? AND status='sold' AND sold_at > NOW()-INTERVAL 3 DAY` |
| 등록 아이템 슬롯 | `InventoryClickEvent` — cursor 아이템을 슬롯 0에 올릴 때 item_data 직렬화 후 DB 임시 저장 |
| 만료 처리 | 스케줄러 매 1분 체크 |
| 판매 완료 | 구매 즉시 DB 상태 변경, 판매자 골드 지급 |
