# 경매장 GUI 설계

> **[STATUS: REFERENCE]** — 경매장 GUI 상세 참조. 공식 리소스팩 기준은 `index.md`가 우선.

> 작성일: 2026-05-17
> 방식: 즉시구매 전용 — 등록 순서대로 54칸에 나열
> 수수료: 판매가의 5% (판매 완료 시 차감)
> 만료: 등록 후 3일, 미판매 시 아이템 반환
> 접근 조건: 영지 내 또는 필드 / 전투 중 불가

## 2. 아이템 슬롯 lore

등록된 아이템 1개 = 슬롯 1칸. 등록 순서대로 채워짐.
같은 종류 아이템이 여러 개 등록돼 있어도 각각 개별 슬롯으로 표시.

```
§f[아이템 이름]
§7강화: §f+15   등급: §e레어
§7──────────
§e가격: §f5,000G
§7판매자: §f포로전사
§7남은 시간: §f2일 14시간
§7──────────
§83일 평균시세: §74,820G
§7──────────
§7좌클릭  §f구매
```

- **3일 평균시세**: 최근 3일간 해당 아이템 종류의 실제 거래 완료가 기준
- 거래 데이터 없으면: `§83일 평균시세: §7데이터 없음`
- 같은 아이템이 여러 슬롯에 나올 때 평균시세가 가격 판단 기준이 됨

## 5. 등록 GUI (27슬롯)

```
행\열   1        2       3       4       5       6       7       8       9
  1   [아이템]   ░      ░      ░      ░      ░      ░      ░      ░
  2    ░        ░      ░    [가격]    ░    [확인]    ░      ░      ░
  3   [뒤로]    ░      ░      ░      ░      ░      ░      ░    [닫기]
```

| 슬롯 | 역할 |
|---|---|
| 0 | 등록 아이템 슬롯 |
| 12 | 가격 설정 (클릭 → Anvil 입력) |
| 14 | 등록 확인 |
| 18 | 뒤로 |
| 26 | 닫기 |

### 등록 아이템 슬롯 (슬롯 0)

비어있을 때:
```
§7등록할 아이템
§7──────────
§7좌클릭  §f인벤토리에서 선택
§7우클릭  §f창고에서 선택
```

아이템 선택 후 — 3일 평균시세 바로 참고 표시:
```
§f[아이템 이름]
§7강화: §f+15
§7──────────
§83일 평균시세: §74,820G
§8(참고용 — 가격은 자유 설정)
§7──────────
§7클릭  §f아이템 변경
```

### 가격 설정 (슬롯 12)

설정 전:
```
§f가격 설정
§7현재: §c미설정
§7클릭  §f가격 입력
```

설정 후:
```
§f가격 설정
§7현재: §e5,000G
§7수수료(5%): §f250G
§7실수령: §f4,750G
§7──────────
§7클릭  §f가격 변경
```

### 등록 확인 (슬롯 14)

아이템·가격 미설정: `GRAY_STAINED_GLASS_PANE` (비활성)

설정 완료: `LIME_DYE`
```
§a등록 확인
§7──────────
§f[아이템 이름]
§e가격: §f5,000G
§7수수료: §f250G (판매 완료 시)
§7만료: §f3일 후
§7──────────
§7클릭  §f등록
```

등록 완료 채팅:
```
§a[경매장] §f[아이템 이름]§7을 §e5,000G§7에 등록했습니다.
```

## 8. 제한 및 규칙

| 항목 | 내용 |
|---|---|
| 계정당 최대 등록 | 5개 |
| 등록 만료 | 3일 |
| 수수료 | 판매가의 5%, 판매 완료 시 차감 |
| 최소 등록가 | 100G |
| 3일 평균시세 기준 | 최근 3일 거래 완료 건의 평균 가격 |
| 판매 골드 | 오프라인 시 로그인 때 자동 지급 |
| 만료 아이템 | 창고로 자동 반환 |

## 9. 구현 참조

| 항목 | 내용 |
|---|---|
| 열기 트리거 | 영지 서브 허브 슬롯 15 클릭 |
| 접근 제한 | 비전투 |
| DB 테이블 | `auction_listings` (아이템·가격·판매자·등록시각·만료시각·상태) |
| 평균시세 쿼리 | `SELECT AVG(price) FROM auction_listings WHERE item_type=? AND status='sold' AND sold_at > NOW()-3days` |
| 만료 처리 | 스케줄러 매 1분 체크 |
| 판매 완료 | 구매 즉시 DB 상태 변경, 판매자 골드 지급 |
