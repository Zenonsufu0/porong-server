## 1. 슬롯 레이아웃

```

  1   [영지명]  [방문설정]  [채굴허용]  [농사허용]  [날씨설정]  [시간설정]  [농작물보호]  [물보호]  [권한설정]
  2   [자동입금]  ░          ░          ░          ░          ░          ░          ░          ░
  3   [멤버 1]  [멤버 2]   [멤버 3]   [멤버 4]   [멤버 5]   [멤버 6]   [멤버 7]   [멤버 8]    ░
  4   [뒤로]     ░          ░          ░          ░          ░          ░          ░          ░
```

| 슬롯 | 역할 |
|---|---|
| 0 | 영지명 변경 |
| 1 | 방문자 공개 설정 |
| 2 | 방문자 채굴 허용 ON/OFF |
| 3 | 방문자 농사 허용 ON/OFF |
| 4 | 날씨 설정 (서버기본/맑음고정/비고정 순환) |
| 5 | 시간 설정 (서버기본/낮고정/밤고정 순환) |
| 6 | 농작물 보호 ON/OFF |
| 7 | 물 파괴 보호 ON/OFF |
| 8 | 권한 등급 설정 (부영주/영지민/방문자 등급별 권한 편집) |
| 9 | 자동입금 ON/OFF (상점 5만G 구매 필요) |
| 18~25 | 영지 멤버 슬롯 (최대 8명) |
| 27 | 뒤로 (영지 서브 허브) |

빈 슬롯: GRAY_STAINED_GLASS_PANE.

## 2. 영지명 변경 (슬롯 0)

아이템: `NAME_TAG`

```
§f영지명 변경
§7현재: §e포로의 집
§7──────────
§7클릭하면 이름 입력창이 열립니다.
§8(Anvil GUI)
```

클릭 시 Anvil GUI 열기. 기존 영지명 pre-fill. 확인 시 DB 반영.
영지명 제한: 최대 16자, 특수문자 금지 (`[a-zA-Z0-9가-힣 ]`).

## 8. 농작물 보호 (슬롯 6)

아이템: `WHEAT`  
아직 다 자라지 않은 작물(최대 성장 단계 미만)은 **모든 방식**으로 파괴 불가.  
차단 대상: 플레이어 좌클릭 파괴, 밟기(트램플링), 몹 트램플링, 폭발, 피스톤.  
완전히 자란 작물은 정상 수확 가능.

### ON
```
§a농작물 보호: §2[활성화]
§7──────────
§7다 자라지 않은 작물은
§7좌클릭·밟기·몹 등 모든 방식으로
§7파괴되지 않습니다.
§8(완전히 자란 작물은 수확 가능)
§7클릭 → 비활성화
```

### OFF
```
§7농작물 보호: §8[비활성화]
§7──────────
§7작물이 보호되지 않습니다.
§7클릭 → 활성화
```

## 10. 자동입금 (슬롯 9)

아이템: `HOPPER`

영주가 상점에서 **5만 골드**에 구매하면 영지 전체에 적용.  
ON/OFF는 영지 내 전원(영지민·방문자 포함) 일괄 적용.

§8(영지 내 전원에게 적용)

## 11. 영지 멤버 슬롯 (슬롯 18~25)

다른 플레이어를 영지에 초대. 초대 수락 시 기본 등급 **영지민**으로 추가. 최대 8명.

### 초대된 멤버 슬롯
아이템: 해당 플레이어 `PLAYER_HEAD`

```
§f[플레이어 이름]
§7등급: §e영지민
§7──────────
§7좌클릭  §f등급 변경
§7우클릭  §c강퇴
```

등급에 따라 색상 변경: 영주 §6, 부영주 §b, 영지민 §e

좌클릭 시 → **멤버 등급 변경 서브 GUI** (§12 참조)  
우클릭 시 → 즉시 강퇴 확인 메시지 표시

### 빈 슬롯 (초대 가능)
아이템: `WHITE_STAINED_GLASS_PANE`

```
§7플레이어 초대
§7──────────
§7클릭 후 플레이어 이름 입력
§8(Anvil GUI)
```

클릭 시 Anvil GUI로 이름 입력. 처리 흐름:

| 상황 | 결과 |
|---|---|
| 온라인 플레이어 | 대상에게 초대 알림 전송, 수락/거절 대기 |
| 오프라인 플레이어 | `§c현재 접속 중인 플레이어가 아닙니다.` |
| 이미 멤버 | `§c이미 초대된 플레이어입니다.` |
| 존재하지 않는 이름 | `§c해당 플레이어를 찾을 수 없습니다.` |

### 초대 알림 (대상 플레이어 채팅)
```
§6[영지 초대] §f[소유자 이름]§7님이 영지에 초대했습니다.
§a[수락] §c[거절]  §8(60초 내 응답)
```

60초 내 응답 없으면 자동 만료.

## 12. 권한 등급 설정 (슬롯 8 → 서브 GUI)

영지 내 권한은 등급(Role) 단위로 관리한다.

### 등급 구조

| 등급 | 대상 | 변경 가능 여부 |
|---|---|---|
| 영주 | 영지 소유자 1명 | 고정 (변경 불가) |
| 부영주 | 영주가 지정한 멤버 | 영주만 설정 가능 |
| 영지민 | 초대된 일반 멤버 (기본값) | 영주·부영주 설정 가능 |
| 방문자 | 초대 없이 방문하는 플레이어 | 방문설정(슬롯 1)으로 접근 제어 |

### 권한 항목 기본값

| 권한 항목 | 영주 | 부영주 | 영지민 | 방문자 |
|---|:---:|:---:|:---:|:---:|
| 창고 입출금 | ✔ | ✔ | ✔ | ✘ |
| 공방 가공기 사용 | ✔ | ✔ | ✔ | ✘ |
| 채굴 (광물 채굴기) | ✔ | ✔ | ✘ | ✘ |
| 농사 (약초 재배지) | ✔ | ✔ | ✔ | ✘ |
| 물 제거·배치 | ✔ | ✔ | ✘ | ✘ |
| 멤버 초대 | ✔ | ✔ | ✘ | ✘ |
| 멤버 강퇴 | ✔ | ✔(영지민만) | ✘ | ✘ |

## 14. 구현 참조

| 항목 | 내용 |
|---|---|
| 열기 트리거 | 영지 서브 허브 슬롯 16 클릭 |
| 접근 제한 | 영지 소유자 + 부영주 / 비전투 |
| DB 테이블 | `island_settings` (영지명·공개설정·방문자채굴허용·방문자농사허용·날씨설정·시간설정·농작물보호·물파괴보호·자동입금활성화) |
| DB 테이블 | `island_members` (island_id, player_uuid, role ENUM(부영주/영지민)) |
| DB 테이블 | `island_role_permissions` (island_id, role, permission_key, enabled) |
| 농작물 보호 훅 | `BlockBreakEvent` + `EntityChangeBlockEvent` + `EntityExplodeEvent` — `Ageable` 블록이고 `age < maxAge` 이면 cancel |
| 자동입금 훅 | `BlockDropItemEvent` + `PlayerHarvestBlockEvent` — 플레이어가 영지 리전 내, 해당 영지 자동입금 ON이면 드롭 cancel 후 DB 적재 |
| 자동입금 구매 | `island_settings.auto_deposit_purchased = true` (영주 구매 시 설정) |
| 갱신 주기 | 설정 변경 시 즉시 반영 |
