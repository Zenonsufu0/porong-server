## 1. 슬롯 레이아웃

```
행\열   1         2         3         4         5        6    7    8       9
  1   [영지명]  [방문설정]  [채굴허용]  [농사허용]   ░       ░    ░    ░       ░
  2   [관리자1] [관리자2]  [관리자3]  [관리자4]  [관리자5]  ░    ░    ░       ░
  3   [뒤로]     ░         ░         ░          ░       ░    ░    ░    [닫기]
```

| 슬롯 | 역할 |
|---|---|
| 0 | 영지명 변경 |
| 1 | 방문자 공개 설정 |
| 2 | 방문자 채굴 허용 ON/OFF |
| 3 | 방문자 농사 허용 ON/OFF |
| 9~13 | 영지 초대 멤버 슬롯 (최대 5명) |
| 18 | 뒤로 (영지 서브 허브) |
| 26 | 닫기 |

빈 슬롯: GRAY_STAINED_GLASS_PANE.

## 2. 영지명 변경 (슬롯 0)

아이템: `NAME_TAG`

```
§f영지명 변경
§7현재: §e포로의 집
§7──────────
§7클릭하면 이름 입력창이 열립니다.
§8(Anvil GUI)
```

클릭 시 Anvil GUI 열기. 기존 영지명 pre-fill. 확인 시 DB 반영.
영지명 제한: 최대 16자, 특수문자 금지 (`[a-zA-Z0-9가-힣 ]`).

## 3. 방문자 공개 설정 (슬롯 1)

3가지 상태를 클릭 시 순환.

### 전체 공개
아이템: `LIME_STAINED_GLASS_PANE`

```
§7모든 플레이어가 방문 가능합니다.

```

### 친구만
아이템: `YELLOW_STAINED_GLASS_PANE`
```
§e방문 설정: §f친구만

§7친구 목록에 있는 플레이어만
§7방문 가능합니다.
§7클릭 → §c비공개
```

### 비공개
아이템: `RED_STAINED_GLASS_PANE`
```
§c방문 설정: §f비공개

§7소유자와 공동 관리자만 입장 가능합니다.

```

## 6. 영지 초대 슬롯 (슬롯 9~13)

다른 플레이어를 영지에 초대. 초대 수락 시 영지 입장 가능. 최대 5명.

### 초대된 멤버 슬롯
아이템: 해당 플레이어 PLAYER_HEAD

```
§f[플레이어 이름]
§7영지 멤버
§7──────────
§a✔ §7방문    §a✔ §7창고
§a✔ §7공방    §c✘ §7채굴
§7──────────
§7좌클릭  §f권한 설정
§7우클릭  §c강퇴
```

현재 권한 상태를 lore에 요약 표시. 좌클릭 시 멤버 권한 서브 GUI 열기.

### 빈 슬롯 (초대 가능)
아이템: `LIME_STAINED_GLASS_PANE`

```
§7플레이어 초대
§7──────────
§7클릭 후 플레이어 이름 입력
§8(Anvil GUI)
```

클릭 시 Anvil GUI로 이름 입력. 처리 흐름:

| 상황 | 결과 |
|---|---|
| 온라인 플레이어 | 대상에게 초대 알림 전송, 수락/거절 대기 |
| 오프라인 플레이어 | `§c현재 접속 중인 플레이어가 아닙니다.` |
| 이미 멤버 | `§c이미 초대된 플레이어입니다.` |
| 존재하지 않는 이름 | `§c해당 플레이어를 찾을 수 없습니다.` |

### 초대 알림 (대상 플레이어 채팅)
```
§6[영지 초대] §f[소유자 이름]§7님이 영지에 초대했습니다.
§a[수락] §c[거절]  §8(60초 내 응답)
```

60초 내 응답 없으면 자동 만료.

## 7. 멤버 권한 서브 GUI (9슬롯)

멤버 슬롯 좌클릭 시 열리는 권한 설정 화면.

```
행\열   1       2       3       4       5    6    7    8      9
  1   [방문]  [창고]  [공방]  [채굴]   ░    ░    ░    ░   [뒤로]
```

| 슬롯 | 권한 | 기본값 |
|---|---|---|
| 0 | 방문 (영지 입장) | ON |
| 1 | 창고 입출금 | ON |
| 2 | 공방 가공기 사용 | ON |
| 3 | 채굴·농사 | OFF |
| 8 | 뒤로 (설정 GUI) | — |

### 권한 슬롯 — ON 상태
아이템: `LIME_DYE`
```
§a[권한명]: §2[허용]
§7클릭 → 비허용
```

### 권한 슬롯 — OFF 상태
아이템: `GRAY_DYE`
```
§7[권한명]: §8[비허용]
§7클릭 → 허용
```

> 방문 권한을 OFF로 설정하면 해당 멤버는 영지 입장 자체가 차단됨 (강퇴와 동일 효과, 단 멤버 목록에는 유지).
