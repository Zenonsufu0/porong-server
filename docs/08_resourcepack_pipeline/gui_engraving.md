## 확정 사항

- 각인 슬롯: **무기당 1개**, 동시 착용 불가
- 교체 비용: **없음**
- 각인 목록: 해당 무기 전용 각인 2종 중 선택 (보유한 각인만 표시)

---

## 슬롯 배치 (0-indexed)

```
행\열   1    2    3    4    5    6    7    8    9
 1     ░    ░    ░    ░    ░    ░    ░    ░    ░
 2     ░  [현재각인]  ░    ░    ░  [각인 A]  ░  [각인 B]  ░
 3    [뒤로]  ░    ░    ░    ░    ░    ░    ░  [닫기]
```

| 슬롯 | 역할 |
|---|---|
| 10 | 현재 장착 각인 (비어있으면 "없음" 표시) |
| 14 | 각인 A (무기 전용 각인 1) 클릭 → 즉시 교체 |
| 16 | 각인 B (무기 전용 각인 2) 클릭 → 즉시 교체 |
| 18 | [뒤로] |
| 26 | [닫기] |

> 현재 장착 중인 각인을 다시 클릭하면 해제.  
> 각인 미보유 시 해당 슬롯에 자물쇠 아이콘 표시.
