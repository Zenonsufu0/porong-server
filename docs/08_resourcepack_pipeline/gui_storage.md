> **[STATUS: REFERENCE]** — 영지 창고 GUI 상세 참조. 공식 영지 기준은 `../05_island_farm_system/CANON.md`가 우선.

> 인벤토리: 54슬롯 (6×9), 바닐라 아이템 아이콘 전용 (커스텀 PNG 배경 없음)
> 접근 조건: 영지 내부에서만 열기 가능

## 1. 창고 개요

- DB 기반 가상 저장고 — 실제 Minecraft 아이템이 아닌 DB 수치로 관리

### 입금 방식

| 방식 | 대상 아이템 | 조건 |
|---|---|---|
| **자동 입금** | 커스텀 생산기 생산물 (자동 재배기 등) | 항상 자동 |
| **자동 입금** | 바닐라 농사·채광 아이템 | 편의 권한 해금 후 |
| **수동 입금** | 바닐라 농사·채광 아이템 | 권한 해금 전 — 인벤토리 → "전체 입금" 버튼 |

### 창고 연동 시스템

창고에 보관된 아이템은 아래 시스템에서 직접 사용 가능 — 별도 출금 불필요:

| 시스템 | 사용 방식 |
|---|---|
| **공방 가공기 대기열** | 레시피 등록 시 창고에서 재료 자동 차감 |
| **상점 판매** | 상점 GUI에서 판매 등록 시 창고 수량 직접 사용 |
| **경매장 등록** | 경매 등록 시 창고 수량 직접 차감 |

## 2. 슬롯 레이아웃 (54슬롯)

```
행\열   1     2     3     4     5     6     7     8     9
  1   [아이템 슬롯 × 9]
  2   [아이템 슬롯 × 9]
  3   [아이템 슬롯 × 9]
  4   [아이템 슬롯 × 9]
  5   [아이템 슬롯 × 9]
  6   [입금] ░     ░    [◀]  [페이지]  [▶]   ░   [뒤로] [닫기]
```

| 슬롯 | 역할 |
|---|---|
| 0~44 | 아이템 표시 (페이지당 최대 45종) |
| 45 | 전체 입금 버튼 |
| 48 | 이전 페이지 |
| 49 | 현재 페이지 표시 |
| 50 | 다음 페이지 |
| 52 | 뒤로 (영지 서브 허브) |
| 53 | 닫기 |

빈 슬롯은 GRAY_STAINED_GLASS_PANE (장식용).

## 3. 아이템 슬롯 lore

아이템 종류별 1슬롯. 수량은 lore에 표시.

```
§f[아이템 이름]
§7보유량: §f1,234개
§7──────────
§8공방·상점·경매장에서 직접 사용 가능
§7──────────
§7좌클릭  §f64개 출금
§7우클릭  §f1개 출금
§7Shift+클릭  §f256개 출금
```

수량 0인 아이템은 표시하지 않음 (슬롯 비움).

## 4. 버튼 lore

### 전체 입금 (슬롯 45)

아이템: `CHEST` (노랑 이름)

```
§e전체 입금
§7──────────
§7인벤토리의 모든 아이템을
§7창고에 입금합니다.
§7
§8편의 권한 해금 시 채굴·수확 아이템은
§8자동으로 입금됩니다.
§7§f[클릭]
```

### 이전 / 다음 페이지 (슬롯 48 / 50)

아이템: `ARROW`

```
§f◀ 이전 페이지      §f다음 페이지 ▶
§71페이지 / 3페이지   §71페이지 / 3페이지
```

1페이지일 때 이전 버튼, 마지막 페이지일 때 다음 버튼 = GRAY_STAINED_GLASS_PANE으로 교체 (비활성).

### 현재 페이지 표시 (슬롯 49)

아이템: `PAPER`

```
§f1 / 3 페이지
§7총 §f87§7종 보관 중
```

### 뒤로 (슬롯 52)

아이템: `DARK_OAK_DOOR`

```
§7◀ 뒤로
§7영지 메뉴
```

## 5. 출금 규칙

| 클릭 | 출금량 | 비고 |
|---|---|---|
| 좌클릭 | 64개 | 보유량 < 64이면 전량 |
| 우클릭 | 1개 | |
| Shift+클릭 | 256개 | 보유량 < 256이면 전량 |

출금 시 플레이어 인벤토리 여유 슬롯이 없으면:
```
§c인벤토리가 가득 찼습니다.
```
채팅으로 알림 후 출금 취소.

---

## 6. 입금 규칙

### 수동 입금 (전체 입금 버튼)
- 플레이어 인벤토리의 창고 등록 아이템 전량 입금
- 핫바 포함
- 입금 완료 메시지:
```
§a창고에 입금했습니다: §f철광석 ×128, 금광석 ×32 ...
```

### 자동 입금 (생산기)
- GUI 없이 DB에 직접 적재
- 플레이어 오프라인 상태에서도 동작

---

## 7. 아이템 정렬 순서

페이지 내 아이템 정렬:

1. 광물류 (철, 금, 다이아, 에메랄드 등)
2. 농작물 (밀, 감자, 당근 등)
3. 목재류
4. 커스텀 소재 (영지 생산 전용)
5. 기타

같은 카테고리 내: 수량 많은 순 정렬.
