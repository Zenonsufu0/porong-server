## 3. 자동심기 허용 (슬롯 1)

아이템: `WHEAT_SEEDS`

영지 전체에서 자동심기 기능 사용 여부를 제어하는 마스터 스위치.  
권한(자동심기 구매 여부)과 무관하게 ON/OFF 설정 가능.  
OFF 시 영지민·방문자 포함 영지 내 전원 자동심기 비활성화.  
기본값: **ON**

### ON
```
§a자동심기: §2[허용]
§7──────────
§7이 영지에서 자동심기 기능이
§7활성화되어 있습니다.
§8(권한 보유자에 한해 작동)
§7클릭 → 비허용
```

### OFF
```
§7자동심기: §8[비허용]
§7──────────
§7이 영지에서 자동심기 기능이
§7비활성화되어 있습니다.
§8(권한 보유 여부 무관, 전원 비활성화)
§7클릭 → 허용
```

## 4. 플라이 허용 (슬롯 2)

아이템: `FEATHER`

영지 전체에서 플라이(비행) 가능 여부를 제어하는 마스터 스위치.  
권한(플라이권 구매 여부)과 무관하게 ON/OFF 설정 가능.  
OFF 시 영지민·방문자 포함 영지 내 전원 플라이 비활성화.  
기본값: **ON**

### ON
```
§a플라이: §2[허용]
§7──────────
§7이 영지에서 비행이
§7허용되어 있습니다.
§8(플라이권 보유자에 한해 작동)
§7클릭 → 비허용
```

### OFF
```
§7플라이: §8[비허용]
§7──────────
§7이 영지에서 비행이
§7허용되지 않습니다.
§8(플라이권 보유 여부 무관, 전원 비활성화)
§7클릭 → 허용
```

## 5-1. 작위 목록 서브 GUI (27슬롯)

```
행\열  1        2        3        4        5        6        7        8        9
 1  [개척지]  [기사]  [준남작]  [남작]  [자작]  [백작]  [후작]  [공작]   ░
 2     ░        ░        ░        ░        ░        ░        ░        ░        ░
 3  [뒤로]     ░        ░        ░        ░        ░        ░        ░        ░
```

### 슬롯 상태 컨벤션 (전역 공통)

| 상태 | 아이템 |
|---|---|
| 현재 작위 | `LIME_STAINED_GLASS_PANE` (초록 유리) |
| 이미 달성한 작위 | `GRAY_STAINED_GLASS_PANE` |
| 다음 승급 가능 작위 | `GOLD_INGOT` (비용 표시, 클릭 가능) |
| 잠긴 미래 작위 | `BLACK_STAINED_GLASS_PANE` (검은 유리) |

### 현재 작위 슬롯 (초록 유리)

```
§a[작위명]  §8← 현재
§7──────────

§7저장고: §f[N]
```

### 달성 완료 작위 슬롯 (회색 유리)

```

```

## 7. 시설 배정 서브 GUI (9슬롯)

빈 시설 슬롯 클릭 시 열리는 시설 선택 화면.

```
[뒤로] [ ░ ] [ ░ ] [약초재배지] [광물채굴기] [공방가공기] [ ░ ] [ ░ ] [ ░ ]
  0      1     2        3            4             5          6     7     8
```

| 슬롯 | 아이템 | 역할 |
|---|---|---|
| 0 | `ARROW` | 뒤로 (영지 상태) |
| 1~2 | `WHITE_STAINED_GLASS_PANE` | 빈 칸 |
| 3 | `WHEAT` | 약초 재배지 배정 |
| 4 | `IRON_PICKAXE` | 광물 채굴기 배정 |
| 5 | `CRAFTING_TABLE` | 공방 가공기 배정 |
| 6~8 | `WHITE_STAINED_GLASS_PANE` | 빈 칸 |

### 시설 선택 lore

현재 슬롯에 없는 시설:
```
§f약초 재배지
§7──────────
§7생산: §f제국 약초 2~3개 §8(20분)
§7현재 레벨: §fLv.1
§7──────────
§7클릭 → 이 슬롯에 배정
```

현재 슬롯에 배정된 시설 (LIME_STAINED_GLASS_PANE으로 하이라이트):
```
§a약초 재배지 §8← 현재 배정됨
§7──────────
§7생산: §f제국 약초 2~3개 §8(20분)
§7현재 레벨: §fLv.1
§7──────────
§7클릭 → 제거
```

## 8. 구현 참조

| 항목 | 내용 |
|---|---|
| 열기 트리거 | 영지 서브 허브 슬롯 10 클릭 |
| 접근 제한 | 영지 내부 / 비전투 |
| DB 테이블 | `island_territory` (작위·저장고 용량·자동심기허용·플라이허용), `island_facility_slots` (시설 배정·레벨·생산 타이머) |
| 자동심기 OFF | 영지 내 전원에게 자동심기 이벤트 훅 비활성화 (권한 보유 무관) |
| 플라이 OFF | 영지 진입 시 `player.setAllowFlight(false)` 강제 적용, 진행 중 비행도 즉시 해제 |
| 승급 재료 소모 | 창고 → 인벤토리 합산 차감 (전역 재화 소모 규칙 적용) |
| 생산 타이머 | `last_produced_ts` 기준 현재 시각과의 차이로 남은 시간 계산 |
| 공방 슬롯 클릭 | 공방 GUI 직접 진입 (별도 서브 GUI 없음) |
