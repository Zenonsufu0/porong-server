# 영지 상태 GUI 설계

> 작성일: 2026-05-17  
> 인벤토리: 27슬롯 (3×9), 바닐라 아이템 아이콘 전용  
> 접근 조건: 영지 내부 / 전투 중 불가  
> 열기 트리거: 영지 서브 허브 슬롯 10 클릭

---

## 1. 슬롯 레이아웃

```
행\열   1         2         3         4         5      6    7    8       9
  1   [영지명]   [마력]    [발전기]   [가공기]    ░      ░    ░    ░       ░
  2   [자동심기] [자동입금] [가공기연동] [경매연동]  [편의5]  ░    ░    ░       ░
  3   [뒤로]     ░         ░         ░          ░      ░    ░    ░   [닫기]
```

| 슬롯 | 역할 |
|---|---|
| 0 | 영지명 + 현재 작위 |
| 1 | 마력 현황 |
| 2 | 발전기 수 |
| 3 | 공방 가공기 현황 |
| 9 | 자동 심기 해금 상태 |
| 10 | 자동 입금(채굴·수확) 해금 상태 |
| 11 | 가공기 창고 자동 연동 해금 상태 |
| 12 | 경매장 창고 자동 연동 해금 상태 |
| 13 | 편의 해금 추가분 (economy_numbers_v2.md §10 참조) |
| 18 | 뒤로 (영지 서브 허브) |
| 26 | 닫기 |

빈 슬롯: `GRAY_STAINED_GLASS_PANE`.

## 3. 마력 현황 (슬롯 1)

아이템:
- 정상: `BLAZE_ROD` (노랑 이름)
- 과부하: `BLAZE_ROD` + `§c[과부하]` 표기, GLINT 효과

### 정상 상태

```
§e마력 현황
§7──────────
§7보유: §f1,240 / 2,000 MP  §862.0%
§7생산: §a+30 MP/h §8(발전기 1기)
§7소모: §c−22 MP/h §8(가동 기계 합산)
§7순 증감: §a+8 MP/h
§7──────────
§8완충까지: 94시간
```

### 마력 부족 (MP < 20%)

```
§c마력 현황 §8[주의]
§7──────────
§7보유: §c280 / 2,000 MP  §c14.0%
§7생산: §a+30 MP/h
§7소모: §c−22 MP/h
§7──────────
§c기계가 곧 정지합니다.
§7발전기를 추가하거나 기계를 줄이세요.
```

### 마력 과부하

```
§c§l마력 과부하!
§7──────────
§7보유: §c2,000 / 2,000 MP
§7소모 기계: §c전부 정지
§7──────────
§c마력 소비 기계를 가동하거나
§c창고에서 마력을 방출하세요.
```

마력 과부하 조건: 현재 MP ≥ 최대 MP (기계 생산물 저장 불가)

## 6. 편의 해금 슬롯 (슬롯 9~13)

편의 해금 전체 목록 및 가격은 `economy_numbers_v2.md §10` 참조.

### 해금 완료 슬롯

아이템: `LIME_DYE`

```
§a[편의명]: §2[활성]
§7──────────
§7[기능 설명 1줄]
§7구매가: §f50,000G §8(구매 완료)
```

### 미해금 슬롯

아이템: `GRAY_DYE`

```
§7[편의명]: §8[미해금]
§7──────────
§7[기능 설명 1줄]
§7구매가: §e50,000G
§7──────────
§7클릭  §f구매 (골드 차감)
```

클릭 시 확인 메시지:
```
§e[영지] §f'자동 심기'를 §e50,000G§f에 구매합니다.
§7[§a수락§7] §8/§7 [§c취소§7]  §8(채팅으로 응답)
```

골드 부족 시:
```
§c골드가 부족합니다. (보유: §f30,000G §c/ 필요: §f50,000G§c)
```

## 7. 구현 참조

| 항목 | 내용 |
|---|---|
| 열기 트리거 | 영지 서브 허브 슬롯 10 클릭 |
| 접근 제한 | 영지 내부 / 비전투 |
| DB 테이블 | `island_settings` (작위·편의해금 비트마스크), `island_machines` (기계 목록·가동 상태) |
| 마력 계산 | `island_power` 테이블 현재 MP + 발전기/소비 기계 합산 |
| 갱신 주기 | GUI 열기 시 1회 조회 (실시간 아님) |
