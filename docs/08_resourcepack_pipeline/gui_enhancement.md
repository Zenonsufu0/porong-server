# 강화 GUI 설계 (45슬롯)

> **[STATUS: REFERENCE]** — 강화 GUI 레이아웃·흐름 기준. 수치는 `economy_numbers_v2.md` 강화 비용표가 우선.

> 기반: `gui_hub_structure.md`  
> 소스: `_archive/implementation_reference_legacy.md` §9.6 (2026-05-23)

## 흐름

```
[장비 서브 허브] → [강화] 클릭
    │
    ▼
[강화 GUI] 열림 — 오른쪽 col8에 현재 장착 장비 5종 표시
    │
    │  오른쪽 장비 클릭
    ▼
중앙에 해당 장비 미리보기 + 성공률·골드비용·강화석 자동 업데이트
    │
    │  [강화!] 버튼 클릭
    ▼
골드 + 강화석 자동 차감 (DB) → 성공/실패 결과 표시
```

> 강화석은 DB에서 관리하는 가상 자원. 실물 아이템 드랍 아님.

## 강화 정보 lore 예시

**슬롯 10 — 성공률**
```
§e강화 성공률
§7─────────────────
§7 현재 단계: §e18강
§7 성공률   : §c5%
§7 가호     : §a23 / 40회
```

**슬롯 22 — 미리보기**
```
§e⚔ 무기   현재 18강
§7─────────────────
§7 강화 후 ATK: 86 → §a+2 = 88
§7─────────────────
§7 [클릭] 장비 선택
```

**슬롯 28 — 골드 비용**
```
§6골드 비용
§7─────────────────
§7 차감 골드: §6{N}G
§7 보유 골드: §e{현재잔액}G
```

**슬롯 33 — 강화석**
```
§e강화석
§7─────────────────
§7 필요 강화석: §e{N}개
§7 보유 강화석: §e{현재}개
```

**슬롯 38 — 강화 흔적 (DL-089)**
```
§a강화 흔적: §f태양의 흔적
§7─────────────────
§7 성공률 보정: §a+50%p
§7 강화 시 §e1개 소모 §7(천장 보정 시 미소모)
§7─────────────────
§7 별 §e3 §7· 달 §e1 §7· 태양 §e2
§e 클릭 → 다음 흔적 / 미사용
```
> 영지 customItems에서 소모. **현재 +10강 이상**에서만 선택·적용(`equipment_growth_spec §3.4`). 별 +20 / 달 +30 / 태양 +50 %p 가산. 가호 천장 강제 성공 시에는 소모하지 않는다.

**슬롯 40 — 강화! 버튼**
```
§7 [클릭] 강화 시도
§7 골드 + 강화석이 자동 차감됩니다.
```

> 강화 성공률·가호·골드·강화석 수치: `economy_numbers_v2.md` §0 강화 비용표 기준.
