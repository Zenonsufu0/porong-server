# 메인 허브 구조 설계

> **[STATUS: REFERENCE]** — 메인/서브 허브 GUI 상세 참조. 공식 리소스팩 기준은 `index.md`가 우선.

> 작성일: 2026-05-16
> 구조: 4분할 메인 허브 → 카테고리별 서브 허브 → 기능 GUI
> 접근 제한: RPG 수도 또는 영지 내부에서만 오픈 가능. 전투 중 / 필드 내 불가.

## 1. 전체 계층 구조

```
[메인 허브]  ── 54슬롯, 4분할 배경
   │
   ├─ [장비]  ── 54슬롯, `menu_equipment.png`
   │    ├─ 강화
   │    ├─ 잠재능력
   │    ├─ 각인
   │    ├─ 캐릭터   (장착·치장·스탯 배분 통합 창)
   │    └─ 전승
   │
   ├─ [영지]  ── 54슬롯, 6구역 3×3  ← 홈 베이스 허브 (DL-073)
   │    ├─ 영지 이동     (준비 중)
   │    ├─ 영지 상태     (작위·저장고·시설 3종)
   │    ├─ 창고
   │    ├─ 공방          (재료 가공·강화 흔적 제작)
   │    ├─ 상점          (NPC 구매·판매, 준비 중)
   │    └─ 영지 설정     (시설 현황 포함 — 시설 관리 흡수)
   │    ※ 경매장은 메인 허브로 이동 (DL-073)
   │
   ├─ [보스]  ── 54슬롯, `menu_boss.png`
   │    ├─ 보스 매칭     (방 생성·보스 선택)
   │    ├─ 방 이동       (기존 방 참가)
   │    ├─ 보스 정보     (HP·패턴 개요·드랍 미리보기)
   │    └─ 클리어 기록   (개인 클리어 횟수·시즌 정복자 조건·최고 기록)
   │
   └─ [탐험]  ── 27슬롯
        ├─ 수도 외곽 평원  (0~5강, 접속자 수·보스 타이머)
        ├─ 폐광 지대
        ├─ 오염된 수로
        ├─ 무너진 초소
        └─ 고대 성벽 잔해  (15~18강)
```

### 보스 방 시스템 상세 (2026-05-16 확정)

**방 만들기 흐름:**
```
[방 만들기] 클릭
    │
    ▼
[Anvil GUI] — 방 제목 입력 (PrepareAnvilEvent로 텍스트 수신)
    │
    ▼
[방 설정 GUI] 54슬롯
    ├─ 보스 선택: [시즌보스1~6] [시즌최종보스1~3]
    ├─ 최소 무기 강화 단계: [-] [현재값] [+]  (0~25, 기본 0)
    ├─ 최대 인원: [1인] [2인] [3인]  (기본 3인)
    └─ [방 개설] 버튼
```

**방 목록 GUI:**
```
[방 목록] — 54슬롯, 현재 대기 중인 모든 방 표시
    각 방 아이템 lore:
      - 방 제목
      - 보스: 시즌보스 X
      - 인원: 1/3  (현재/최대)
      - 최소 강화 단계: XX강 이상
    클릭 → 조건 충족 시 참가 / 미충족 시 거부 메시지
```

**입장까지 전체 흐름:**
```
방 개설 or 방 참가
    │
    ▼
[방 내부 GUI] — 방장: 강퇴 버튼 / 전원: 준비 상태 확인
    │  파티원 슬롯: 이름 + 무기 강화 단계 표시
    │
    │  방장이 [입장] 클릭
    ▼
[보스 로비 공간] — 모든 보스 파티 공유 단일 로비
    │  파티원끼리 확인·소통 가능
    │  보스 정보 열람 가능
    │
    │  방장이 [보스 입장] 클릭  ← 전투 태그 재확인 후 입장
    ▼
[보스 인스턴스] — 파티 전용 공간 생성
```

**방장 권한:**
- 강퇴 (방 내부 GUI에서 파티원 슬롯 클릭)
- 보스 변경 (방 개설 후 로비 이동 전까지)
- 입장 결정 (로비 → 보스 인스턴스 이동)

**인원 규칙:**
- 시즌보스 1~6 및 시즌최종보스 3종 모두 최소 1인 입장 가능. 최소 입장 제한은 두지 않는다.
- 방 최대 인원은 방 개설 시 1~3인으로 설정.

**클리어 기록 표시 항목:**
- 보스별 총 클리어 횟수
- 균열왕 심장 획득 여부 (시즌 정복자 조건)
- 파티 최고 기록 (클리어 시간)

## 7. 메뉴 접근 제한

> 수도 제거 결정 (2026-05-16) 반영.

```
[장비 관련] 강화·잠재·각인·캐릭터·전승
    └─ 전투 중 차단 + 필드 내 차단. 영지에서만.

[영지 관련] 영지 상태·창고·공방·시설·설정
    └─ 전투 중 차단 + 필드 내 차단. 영지에서만.

[영지 이동]
    └─ 전투 중만 차단. 필드 내 허용. (2026-05-16 확정)

[경매장 · 상점]
    └─ 전투 중만 차단. 필드 내 허용. (2026-05-16 확정)

[탐험 — 필드 이동]
    └─ 전투 중만 차단. 필드 내 허용. (2026-05-16 확정)

[보스 매칭 · 방 이동]
    └─ 전투 중만 차단. 필드 내 허용. (2026-05-16 확정)
       단, 보스 입장 순간 전투 태그 잔존 시 → 입장 취소 + 안내 메시지.
       메시지: "전투 상태에서는 보스에 입장할 수 없습니다. 잠시 후 다시 시도하세요."

[보스 정보 · 클리어 기록]
    └─ 제한 없음. 항상 열람 가능.
```

허용 구역 판정: `WorldGuard` 리전 태그 `poro:menu_allowed`.  
각 플레이어 영지 내부에 적용. (수도 제거로 스폰 존 별도 지정 불필요)
