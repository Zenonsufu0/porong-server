# 잠재능력 옵션 시스템 v2 (3라인 방식)

> **[STATUS: REFERENCE]** — 잠재능력 옵션 풀 상세 참조. 공식 적용 기준은 `CANON.md`가 우선.

> 기준일: 2026-05-13
> 대상: 1차 시즌 T1 장비 5슬롯 (무기 / 머리 / 상의 / 하의 / 신발) — 장갑 슬롯 없음
> 등급: 커먼 / 레어 / 에픽 / 유니크 / 레전더리
> 구조: **슬롯당 3개 옵션 동시 부여 (메이플스토리 방식)**

---

## 0. 설계 전제

### 0-1. 핵심 구조 확정

- 슬롯당 잠재 옵션 수: **3개**. 등급이 3개 라인의 품질 기준을 결정한다.
- **라인 구조**:
  - **1라인**: 현재 등급 옵션 풀 + 현재 등급 수치
  - **2·3라인 (기본)**: 바로 아래 등급 옵션 풀 + 바로 아래 등급 수치
  - **이탈**: 2·3라인이 바로 아래 등급이 아닌 현재 등급으로 나오는 경우. 라인당 독립 발생.
- 이탈 확률: **2라인 10%, 3라인 5%** (라인마다 독립, 설계 파라미터).
- 커먼 등급은 바로 아래 등급이 없으므로 3라인 모두 커먼 수치를 사용한다.
- 큐브 사용 시: **3개 라인 전부 동시 재롤**.
- 중복 허용: 동일 옵션이 3라인 모두 나올 수 있다.
- 1차 시즌 최고 등급은 레전더리. 45일 시즌 내 레전더리 달성은 무기 단일 슬롯 기준 최상위 유저 일부만 가능한 극소수 목표.
- 강화는 최대 25강 (파괴/하락 없음). 잠재는 강화의 보조 성장축이며, 대체 성장축이 아니다.
- 레전더리 달성은 시즌 내 전체 유저의 0.5~1%에 해당하는 극소수 목표다 (1슬롯 기준).
- 유니크 달성이 활성 유저 상위 5~10%의 현실적 상위권 목표다.
- 에픽 달성이 활성 유저 10~20%의 현실적 중상위권 목표다.
- 조건부 옵션, 스택 누적, 발동 확률형 옵션은 45일 단기 시즌 취지상 제외한다.
- 강제 합산 캡 없음. 극단 조합은 확률적 희귀성으로 통제.

### 0-3. 기본 스탯 및 스탯 트리 연계

잠재능력은 기본 스탯 + 스탯 트리 투자와 합산된다 (`level_stat_system_v1.md` 참조).

| 스탯 | 기본값 | 주 수급원 |
|---|---|---|
| 치명타 확률 | 5% | 스탯 치명 (**0.30%/pt**) + 무기/머리 잠재 |
| 치명타 피해량 | 150% 배율 | 스탯 치명 (0.2%/pt) + 무기 잠재 |

> 치명 확률 = **0.30%/pt** 가 정본 (`level_stat_system_v1 §3`, 2026-05-16 재설계 하향). 구 0.45%/pt 표기 폐기.

**치명타 100% 설계 기준** (`level_stat_system_v1 §3` 정합 — 스탯 0.30%/pt + base 5%):
- **레벨 80 + 레전더리 잠재 1개**: 치명 240pt(+72%) + 레전(+25%) + 5% = **102%** → 잠재 1개로 달성
- **레벨 65 + 레전더리 잠재 + 보조스탯 1슬롯**: 58.5% + 25% + 15% + 5% = **103.5%**
- **레벨 50 + 레전더리 잠재 + 보조스탯 2슬롯**: 45% + 25% + 30% + 5% = **105%** (경직된 조합)

> 치명 100%는 레전더리 잠재 없이 불가(레어 80강 all-in = 87%) — 레전더리가 "잠금 해제" 역할.
> 레전더리 치명타 확률% 수치 (10~14%, 평균 12%)는 위 기준에 부합한다.

### 0-2. 기대 기여율 산출 공식

1라인은 현재 등급, 2·3라인은 기본적으로 바로 아래 등급이며 이탈 확률(2라인 P2=10%, 3라인 P3=5%)로 현재 등급이 나온다.

**무기 슬롯 ATK% 기대 기여율 공식 (옵션 풀 N종, 2라인 이탈 P2, 3라인 이탈 P3):**

```
기대 ATK% = (1/N)×E
           + (1/N)×[(1-P2)×R + P2×E]
           + (1/N)×[(1-P3)×R + P3×E]

E  = 현재 등급 ATK% 평균값
R  = 바로 아래 등급 ATK% 평균값
N  = 옵션 풀 종수 (무기 = 11)
P2 = 2라인 이탈 확률 (0.10)
P3 = 3라인 이탈 확률 (0.05)
```

**에픽 기준 계산 (E=10%, R=4%, N=11, P2=0.10, P3=0.05):**
```
= (1/11)×10% + (1/11)×[0.90×4% + 0.10×10%] + (1/11)×[0.95×4% + 0.05×10%]
= (10 + 4.60 + 4.30) / 11
= 18.90 / 11
= ~1.7%
```

**유니크 기준 계산 (E=13.5%, R=10%, N=11, P2=0.10, P3=0.05):**
```
= (1/11)×13.5% + (1/11)×[0.90×10% + 0.10×13.5%] + (1/11)×[0.95×10% + 0.05×13.5%]
= (13.5 + 10.35 + 10.175) / 11
= 34.025 / 11
= ~3.1%
```

비고:
- 옵션 풀이 11종으로 확장되어 ATK% 단독 기대치는 낮아지나, 치명타·스킬타입 피해·보스 피해·방어력 무시 등 **전체 공격 옵션 합산 DPS 기여는 유지**된다.
- `season_boss_stats_v1.md`의 잠재% 수치는 ATK% 단독 기준이며, 11종 풀 기준 전체 공격 옵션 기여 기준으로 재산출이 필요하다.

## 2. 등급별 라인당 수치 범위

### 2-0. 수치 적용 방식

- **1라인**: 현재 잠재 등급의 수치 범위에서 결정.
- **2·3라인 (기본)**: 바로 아래 등급의 수치 범위에서 결정.
- **2·3라인 (이탈, 2라인 10% / 3라인 5%)**: 현재 등급 수치 범위에서 결정 (1라인과 동일 조건).
- 커먼 등급은 바로 아래 등급 없음 → 3라인 모두 커먼 수치.
- 수치는 정수 단위. 소수점 없음. 라인별 독립 결정.

---

### 2-1. 무기 슬롯 수치 범위

수치 테이블은 **등급별 해당 수치 범위**를 정의한다. 라인별 적용:
- **1라인**: 현재 잠재 등급 열 수치 사용
- **2·3라인 (기본)**: 한 칸 왼쪽 등급 열 수치 사용
- **2·3라인 (이탈)**: 현재 등급 열 수치 사용

| 옵션 | 커먼 | 레어 | 에픽 | 유니크 | 레전더리 |
|---|---|---|---|---|---|
| ATK% | 1~2% | 3~5% | 8~12% | 12~15% | 16~21% |
| 방어력 무시% | 1~2% | 2~4% | 5~8% | 8~12% | 13~18% |
| 스킬 피해% | 1~2% | 3~5% | 7~10% | 11~14% | 15~20% |
| 보스 피해% | 1~2% | 3~4% | 6~9% | 10~13% | 14~18% |
| 기본기 피해% | 2~4% | 5~8% | 10~14% | 15~19% | 20~26% |
| 이동기 피해% | 2~4% | 5~8% | 10~14% | 15~19% | 20~26% |
| 특수기 피해% | 2~4% | 5~8% | 10~14% | 15~19% | 20~26% |
| 핵심기 피해% | 2~4% | 5~8% | 10~14% | 15~19% | 20~26% |

## 3. 등급별 기대 ATK 기여율

이탈 확률 2라인 10% / 3라인 5% 적용. 무기 슬롯 옵션 풀 9종.

### 3-1. 무기 슬롯 ATK% 기대 기여율

```
기대 ATK% = (1/11)×E + (1/11)×[(1-P2)×R + P2×E] + (1/11)×[(1-P3)×R + P3×E]
```

옵션 풀 11종 (무기), 이탈 확률 2라인 P2=10% / 3라인 P3=5% 기준.

| 등급 | 1라인 수치(E) | 2·3라인 수치(R) | ATK% 단독 기대 기여율 |
|---|---|---|---|
| 커먼 | 1~2% (avg 1.5%) | — (3라인 모두 커먼) | 3×(1/11)×1.5% = **~0.4%** |
| 레어 | 3~5% (avg 4%) | 커먼 avg 1.5% | **~0.66%** |
| 에픽 | 8~12% (avg 10%) | 레어 avg 4% | **~1.7%** |
| 유니크 | 12~15% (avg 13.5%) | 에픽 avg 10% | **~3.1%** |
| 레전더리 | 16~21% (avg 18.5%) | 유니크 avg 13.5% | **~4.2%** |

레전더리 계산 (E=18.5%, R=13.5%, N=11, P2=0.10, P3=0.05):
```
= (1/11)×18.5% + (1/11)×[0.90×13.5%+0.10×18.5%] + (1/11)×[0.95×13.5%+0.05×18.5%]
= (18.5 + 14.0 + 13.75) / 11
= 46.25 / 11
= ~4.2%
```

### 3-2. 무기 슬롯 ATK 기여율 해석

옵션 풀이 11종으로 확장되어 ATK% 단독 기대치는 낮아진다. 그러나 다른 공격 옵션(방어력 무시·치명타·보스 피해·스킬타입 피해% 4종)의 기대 기여가 병렬로 추가되므로 **전체 공격력 기여는 분산 풀 설계에서도 유지**된다.

ATK% 이외 공격 옵션의 라인당 기대 기여 (에픽 기준, 차등 확률 적용):
- 방어력 무시% 기대 라인 ≈ 0.27라인 → 에픽 avg 6.5% → DEF 200 보스 기준 유효 DEF -1.8
- 치명타 확률% 기대 라인 ≈ 0.27라인 → 에픽 평균 5.5% × 0.27 = **~1.5%**
- 치명타 데미지% 기대 라인 ≈ 0.27라인 → 에픽 평균 9% × 0.27 = **~2.4%**
- 기본기/이동기/특수기/핵심기 피해% 각 ~0.37라인씩 → 에픽 avg 12% × 0.37 = **~4.4% (4종 합산 기대, 스킬 집중도에 따라 편차)**
- 스킬 피해%·쿨타임 감소%·보스 피해% 각 ~0.37라인씩 추가

레전더리 기대 ATK% ~4.2%는 설계 목표(강화 보조 성장축) 달성 범위. 1차 시즌 ATK% 잠재 기여는 무기 슬롯 단독.

| 슬롯 | 일반 유저 (1,260회) | 하드 유저 (2,970회) | 선발대 유저 (5,040회) | 선발대 하드 (6,750회) |

| 무기 | **레전더리** (무기 집중 시) | **레전더리** | **레전더리** | **레전더리** + 옵션 재롤 |
| 머리 | **유니크** | **레전더리** | **레전더리** | **레전더리** + 옵션 재롤 |
| 상의 | **유니크** | **레전더리** | **레전더리** | **레전더리** + 옵션 재롤 |

- 선발대 하드: 5슬롯 레전더리 + 약 3,000큐브 이상 옵션 최적화 여유.

| 일반 유저 (1~2h) | **레전더리** (무기 집중) | **~4.2%** | 큐브 무기 집중 전략 시. 나머지 슬롯은 유니크 |
| 하드 유저 (3~4h) | **레전더리** | **~4.2%** | 무기 레전더리 달성. 방어구 슬롯 레전더리 3~4슬롯 추가 |
| 선발대 유저 (6h+) | **레전더리** | **~4.2%** | 전 슬롯 레전더리 도전. 옵션 최적화 여유 있음 |
| 선발대 하드 (8h+) | **레전더리** | **~4.2%** | 전 슬롯 레전더리 보장 + 약 3,000큐브 옵션 재롤 |

비고:
- 새 옵션 풀(11종) 기준 레전더리 ATK% 기대치는 ~4.2% (기존 N=9 기준 ~5.1%에서 하향).
- 유저 간 차별화 지점은 ATK% 수치보다 **레전더리 달성 슬롯 수**와 **옵션 조합 품질(재롤 횟수)**이다.
- 스킬타입 피해%, 치명타, 보스 피해% 등 전체 공격력 기여는 ATK% 단독보다 높다.

## 5. 큐브 사용 확률 분포 상세

### 5-1. 단일 큐브 사용 결과 확률

| 결과 등급 | 확률 | 변경 전 |
|---|---|---|
| 커먼 | **75%** | 45% |
| 레어 | **20%** | 33% |
| 에픽 | **4%** | 15% |
| 유니크 | **0.9%** | 6% |
| 레전더리 | **0.1%** | 1% |

### 5-2. 등급 보장 구조 하 전환 기대 횟수

등급 보장 전제: 재롤 결과가 현재 등급보다 낮으면 적용하지 않고 큐브만 소모.

| 전환 구간 | 유효 승급 확률/큐브 | 기대 큐브 소모 | 천장 반영 평균 |
|---|---|---:|---:|
| 시작 → 레어 이상 | **25%** | **~4회** | ~3.8회 |
| 레어 → 에픽 이상 | **5%** | **~20회** | ~17회 |
| 에픽 → 유니크 이상 | **1%** | **~100회** | ~87회 |
| 유니크 → 레전더리 | **0.1%** | **~1,000회** | **~632회** |
| 시작 → 유니크 (합산) | — | **~124회** | ~108회 |
| 시작 → 레전더리 (합산) | — | **~1,124회** | **~740회** (천장 반영) |

비고:
- 천장 반영 평균: 각 구간 천장 도달 시 확정 상승 → 실제 평균 소모 기댓값보다 낮음.
- 유니크→레전더리: 기댓값 1,000회, 천장 1,000회 적용 시 평균 실소모 **~632회**. (자연 달성 63%, 천장 37%).
- 천장 시스템으로 최악의 경우를 제한하며 기대치 자체를 크게 단축하지 않음.

### 5-3. 차등 확률 적용 시 주요 조합 기대치

차등 확률 도입 (섹션 6) 이후 무기 슬롯 ATK% 선택 확률: 균등 9.1%(1/11) → **~7.1%** (희귀 등급, 가중치 1/14).

| 조합 | 균등 확률 (N=11 기준) | 차등 확률 (희귀 적용) |
|---|---|---|
| 무기 ATK% 1라인 선택 확률 | 9.1% | ~7.1% |
| 3라인 모두 ATK%: | (1/11)³ = 0.075% | 0.071³ = **0.036%** |
| 유니크 도달(~124회) + 트리플 ATK% | ~1/3,440회 큐브 | **~1/7,220회 큐브** |

레전더리 핵심기 피해% 트리플 (무기, 차등 기준, 보통 가중치):
- 라인당 핵심기 피해% 선택 확률: ~10.7%
- 3라인 모두: 0.107³ = **0.122%** (레전더리 큐브 1회당)
- 레전더리 도달(~1,124회) + 트리플 핵심기 피해%: **~1/820,000회 총 큐브 소모** (사실상 불가)

비고:
- 차등 확률로 최상위 조합 달성은 균등 대비 더 어려워짐.
- 레전더리 도달 이후도 원하는 옵션 조합을 위해 지속 큐브 소모 동기 유지됨.

### 5-4. 확정 승급 천장 시스템 (Pity / 천장)

기댓값 기준으로 각 구간별 확정 승급 횟수를 설정한다. 천장 초과 시 다음 큐브 사용에서 상위 등급 확정.

| 구간 | 기댓값 | 천장 (확정 횟수) | 천장 도달 확률 | 천장 반영 평균 소모 |
|---|---|---|---|---|
| 시작 → 레어 | ~4회 | **10회** | ~5.6% | ~3.8회 |
| 레어 → 에픽 | ~20회 | **40회** | ~12.9% | ~17회 |
| 에픽 → 유니크 | ~100회 | **200회** | ~13.4% | ~87회 |
| 유니크 → 레전더리 | ~1,000회 | **1,000회** | **~36.8%** | **~632회** |

**천장 동작 방식:**
- 각 구간 천장 카운터는 해당 등급 달성 시점부터 독립 카운트.
- 큐브 1회 사용마다 카운터 +1. 해당 등급 이상 결과가 나오면 카운터 초기화.
- 카운터가 천장 수치에 도달하면 다음 큐브 사용 시 해당 등급으로 **확정 승급** (옵션은 정상 롤).
- 천장 도달 후 상위 등급으로 이동 시 카운터는 새 구간 기준으로 초기화.
- 천장 카운터는 서버 재시작·로그아웃 후에도 유지 (DB 저장 필수).

**설계 의도:**
- 유니크→레전더리 천장(1,000회): 기댓값과 동일. 자연 달성 63%, 천장 37%.
  - 천장 반영 평균 실소모 **~632회**. 분산 제거 목적.
  - 선발대 하드 최악의 경우(전 슬롯 천장): 5×1,000 + 620 = 5,620 < 6,750 → **반드시 완성 보장**.
- 에픽→유니크 천장(200회): 기댓값 100회. 천장 도달 ~13%만 영향. 유니크 달성 하한선 제공.
- 중간 등급 천장은 기댓값의 2배로 설정해 "비운이 겹치는 경우"만 보호. 기댓값 자체는 유지.

### 6-6. 차등 확률 설계 근거

- **목적**: ATK%, 치명타 확률%, 보스 피해% 등 강력한 % 옵션의 3라인 동시 발생 확률을 균등 대비 낮춤.
- **+ 옵션 제거**: HP+/DEF+/ATK+ 전 슬롯 제거. 일반(2.5) 가중치 구간이 없어짐. 모든 옵션이 희귀(1) 또는 보통(1.5)으로 단순화.
- **슬롯별 희귀 옵션**: 무기는 공격/크리 5종이 희귀. 방어구(머리/상의/하의)는 스킬타입 피해% 4종 + 슬롯 핵심 옵션이 희귀. 신발은 ATK%·방어력무시%·스킬타입 피해% 4종이 희귀.
- **스킬타입 피해% 설계**: 스킬 피해%(범용)보다 수치가 높고 희귀~보통 사이 가중치를 부여. 특화 빌드에서 더 강력, 범용 빌드에서는 스킬 피해%가 효율적.
- **구현**: 슬롯·등급 조합별 가중치 테이블을 EmpireRPG 플러그인 설정 파일(YAML)로 외부 관리 권장.

## 7. 재롤 정책 및 메모리얼 시스템

### 7-1. 확정 정책

- 큐브 1회 사용 시 **3개 라인 전부 동시 재롤**.
- 재롤 결과가 현재 등급보다 낮은 등급이면 **자동 등급 보호** — 결과를 버리고 큐브만 소모.
- 결과가 현재 등급과 같거나 높으면 **메모리얼 선택 화면** 진입.

### 7-2. 메모리얼 기능 (Memorial)

큐브를 사용하면 결과를 즉시 적용하지 않고, **현재 옵션 vs 신규 롤 결과를 동시에 보여주며 플레이어가 선택**할 수 있다.

| 상황 | 처리 |
|---|---|
| 신규 결과 등급 < 현재 | 등급 보호 자동 적용, 선택 화면 없음 (큐브만 소모) |
| 신규 결과 등급 = 현재 | **메모리얼 선택 화면** → 현재 유지 또는 신규 적용 중 택일 |
| 신규 결과 등급 > 현재 | **등급 상승 확인 화면** → 자동 적용 권장 (항상 상위 등급이므로) |

**메모리얼 선택 화면 동작:**
- 좌측: 현재 3개 옵션/수치 (보관된 결과)
- 우측: 신규 롤 3개 옵션/수치
- 이탈 발생 라인은 별도 표시 (예: ★이탈)
- 선택 후 큐브 소모 확정. 선택하지 않고 닫으면 현재 유지 (큐브 소모 없음 — 단, 이 경우 재롤 결과는 폐기).

**메모리얼 설계 의도:**
- 플레이어가 "좋은 조합"을 직접 비교하며 선택하는 재미 제공.
- 큐브 소모는 선택 확정 시에만 발생하므로 "롤 미리보기" 어뷰징을 방지하기 위해 **선택 화면을 열면 큐브는 이미 소모된 것으로 처리**한다 (선택 결과만 남음).
- 큐브를 아끼려고 선택창을 닫는 플레이어가 나오지 않도록 주의. 닫으면 현재 유지지만 큐브는 소모됨.

### 7-3. 재롤 정책 UX 요약

- 큐브 사용 전: 현재 등급·3개 옵션·수치를 GUI에 명확히 표시. 보유 큐브 수량 표시.
- 큐브 사용 후: 메모리얼 선택 화면 진입 (등급 하락 시 자동 보호 메시지).
- 레전더리 달성 후: 등급 보호 없음. 이후 큐브는 레전더리 내 옵션 재조합 목적. 이 구간이 레전더리 이후 지속적 큐브 소모 동기.

## 9. 리스크 및 운영 체크포인트

### 9-1. 리스크

**R1. 큐브 수급 과잉 시**
시즌 중반 이후 큐브가 예상보다 쉽게 모이면 에픽 이상 도달 비율이 높아진다. 필드 큐브 조각 드랍률을 운영 중 하향 조정하는 것이 주 완충 수단.

**R2. 에픽 조기 달성 + 수치 재조합 목적 큐브 소모 집중**
에픽 달성 후 유니크를 노리는 구간에서 큐브 소모가 집중될 수 있다. 이는 의도된 구조다. 단, 큐브 수급량이 이 구간 소모를 감당하는지 시즌 2주차 기준으로 로그를 점검한다.

**R3. 보스 피해% 3라인 몰림**
옵션 풀 11종 기준 보스 피해% 3라인이 나올 확률은 균등 기준 (1/11)^3 ≒ 0.075% (차등 확률 적용 시 더 낮음). 발생 시 보스 피해% 합산 최대 레전더리 기준 3×18% = 54%. 보스 밸런스 설계 시 잠재 보스 피해%를 반영한 기준값 사용 필요.

**R4. 치명타 데미지% 몰림**
치명타 데미지%는 라인당 수치가 크다 (레전더리 18~25%). 트리플 레전더리 치명타 데미지% = 최대 75%. 치명타 확률% 기반 실 발동 확률과 곱하면 DPS 배율이 매우 높아질 수 있다. 수치 조정은 시즌 운영 중 로그 기반으로 대응한다.

**R5. 메모리얼 오용**
메모리얼 선택 화면 중 서버 재시작/연결 끊김 시 현재 유지 fallback이 필요하다. 미구현 시 큐브만 소모되고 결과가 소실되는 문제 발생.

## 10. 요약

### 핵심 확정 사항

1. 슬롯당 잠재 옵션 3개 동시 부여 (3라인 방식). 큐브 사용 시 3라인 전부 재롤.
2. 등급이 3개 라인 전체 품질을 결정. 중복 옵션 허용.
3. 라인당 수치는 1슬롯 1옵션 시대 대비 하향 조정. 전체 ATK 기여율 목표 범위는 유지.
4. **3라인 구조**: 1라인=현재 등급, 2·3라인=바로 아래 등급. 이탈(2라인 10% / 3라인 5%)로만 현재 등급 가능.
5. ATK% 단독 기대 기여율: 에픽 **~1.7%**, 유니크 **~3.1%**, 레전더리 **~4.2%** (옵션 풀 11종, 이탈 2라인 10%/3라인 5% 기준). 치명타·보스·스킬타입 피해·방어력 무시 포함 전체 공격 기여는 이보다 높다.
6. **등급 확률**: 커먼 **75%** / 레어 **20%** / 에픽 **4%** / 유니크 **0.9%** / 레전더리 **0.1%**. 레전더리 도달 기대 ~1,124회. 수급 증가(10조각=큐브1개, kills/min 40~50)에 맞춰 전 등급 확률 하향. 에픽→유니크 기대 100회가 중간 체감 장벽.
7. **확정 승급 천장**: 유니크→레전더리 **1,000회** / 에픽→유니크 **200회** / 레어→에픽 **40회** / 시작→레어 **10회**. 천장 반영 슬롯당 평균 실소모 **~740회**. 선발대 하드(6,750큐브) 최악 케이스에도 전 슬롯 완성 보장.
8. **옵션 차등 확률**: 슬롯별 가중치 시스템. 희귀(1):보통(1.5) 2단계 구조 (+ 옵션 제거로 일반 등급 소멸). 공격/크리 % 옵션(희귀 ~7~12.5%)이 유틸리티 % 옵션(보통 ~11~19%)보다 드물게 등장. 스킬타입 피해%는 슬롯별로 희귀 또는 보통 적용.
9. 이탈이 희귀하므로 2·3라인은 체감상 아래 등급 보너스. 이탈 발생 시 강한 보상 체감 설계.
10. **메모리얼**: 큐브 사용 후 현재 옵션 vs 신규 롤 결과를 비교해 선택 가능. 등급 하락 시 자동 보호. 등급 상승 시 자동 적용.
11. **GUI**: 메모리얼 선택 화면 (좌=현재/우=신규), 이탈 라인 ★ 표시, 등급별 색상 (보라/주황/초록 등).
12. 재롤 정책: 등급 보장 (하락 없음), 수치 단독 재롤 미제공. 강제 합산 캡 없음 — 극단 조합은 확률적 희귀성으로 통제.

| 목표 | 결과 |

| 유니크 ATK 기여율 상위권 목표 | ~8.5% → 목표 범위 달성 |
