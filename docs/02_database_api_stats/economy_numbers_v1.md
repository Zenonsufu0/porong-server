## 5. 병목·과잉 및 조정 제안

### 병목 구간

| 구간 | 문제 | 권장 조치 |
|---|---|---|
| 15~18강 보조재 C 수급 | 후반 필드 강제 우려 | 하급 재료 다량으로 대체 가능한 레시피 추가 |
| 17~18강 연속 실패 | 재화 소모 집중, 이탈 리스크 | 연속 실패 N회 시 성공률 가호 +5%/실패 (최대 +20%) 고려 |
| 큐브 조각 필드보스 단일 의존 | 타이밍 놓치면 당일 수급 0 | 정예몹 저확률 드랍 (0.5~1%) 또는 일일 퀘스트 보조 수급 추가 |

### 조정 제안

| 항목 | 방향 |
|---|---|
| 일 2h 유저 18강 도달 | 17~18강 골드 비용 약 10% 감소 또는 정예몹 골드 +10% |
| 큐브 수급 다양화 | 정예몹 0.5~1% 조각 드랍 또는 일일 필드 퀘스트 보상 추가 |
| 보조재 C 수급 부담 완화 | 하급 재료 다량 대체 레시피 열기 |
