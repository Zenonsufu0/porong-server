## 1. 영지 작위 구매 비용

### 설계 기준

- 기사령: 하드 유저 (3~4h) 기준 5~7일이면 구매 가능한 수준
  - 하드 유저 혼합 기대 골드: 3,300~4,500G/일 (3h 기준)
  - 5~7일 누적: 약 16,500~31,500G → 기사령 비용 목표: 약 18,000~22,000G
- 공작령: 시즌 내 선발대 유저 중 일부만 도달 가능 (하드 유저 기준 강화 병행 시 사실상 불가)
  - 하드 유저 (4h) 45일 총 수입: 288,000~360,000G
  - 선발대 유저 (6h+) 45일 총 수입: 약 430,000~540,000G
  - 강화에도 골드를 쓴 뒤 공작령 도달하려면 공작령까지 누적 비용이 약 120,000~180,000G 수준이어야 현실적
- 구매 재화는 골드 + 해당 구간 필드 공통 재료 조합
- 작위 칭호는 골드 sink로 기능 — 골드 과잉 축적 억제

검토:
- 하드 유저 (3h 기준) 45일 기대 총 골드 약 150,000~202,000G 기준으로 남작령(110,000G)은 도달 가능권, 자작령(185,000G)은 타이트한 목표선이다.
- 강화 비용과 병행하면 자작령은 골드를 아끼거나 필드 효율을 높인 하드 유저만 도달 가능하다 → 적절한 긴장감.

운영 조정 옵션:
- 공작령 진입장벽이 너무 높다면 후작령 이하 비용을 10~15% 하향 조정하고 공작령은 유지한다.
- 영지 재료 의존도 조정 — 균열 파편은 후반 필드 강제 유발 우려가 있으므로 정예몹 누적 드랍으로도 충당 가능하게 드랍률을 열어둔다.

## 2. 영지 시설 슬롯 및 기계 구성

> **2026-05-19 설계 변경: 발전기(마력 시스템) 폐지**
> 마력 발전기 + 마력 소비 구조는 슬롯 한도와 이중 제약이 되어 복잡도 대비 재미가 없음 → 완전 제거.
> 슬롯 한도만으로 생산량을 조절한다.

### 시설 슬롯 구조

- 영지 작위별로 최대 설치 가능한 시설 슬롯 수가 결정된다.
- 슬롯마다 유저가 자유롭게 시설을 선택 설치한다: 자동 재배기 / 공명 추출기 / 공방 가공기.
- 공방 가공기 1대 설치 = 제작 대기열 슬롯 3칸 해금.
- 공방 가공기를 설치하지 않으면 제작 대기열 슬롯이 없다.

### 시설 구성 예시

초보 영지(개척지 4슬롯) 예시:
```
자동 재배기 ×2 + 공명 추출기 ×1 + 공방 가공기 ×1
→ 제작 대기열 3칸 확보
```

중급 영지(남작령 9슬롯) 예시:
```
자동 재배기 ×3 + 공명 추출기 ×2 + 공방 가공기 ×2 + 잉여 슬롯 2
→ 제작 대기열 6칸 확보, 남은 2슬롯 자유 배치
```

후반 영지(공작령 18슬롯) 예시:
```
자동 재배기 ×5 + 공명 추출기 ×4 + 공방 가공기 ×3 + 잉여 슬롯 6
→ 제작 대기열 9칸 확보
```

설계 원칙:
- 마력 개념 없음. 설치 가능 슬롯 수가 유일한 생산 제약이다.
- 같은 슬롯 수에서 무엇을 우선 설치할지가 전략 선택이 된다.
- 공방 가공기를 많이 설치할수록 제작 효율이 높아지나 채집 시설이 줄어든다.

## 4. 공방 가공기 대기열 한도

> **2026-05-19 설계 변경: 작위 기반 슬롯 → 공방 가공기 설치 기반 슬롯으로 전환**

### 결정안: 공방 가공기 설치 수 × 3 = 제작 대기열 슬롯

- 공방 가공기를 설치하지 않으면 제작 대기열 슬롯 0.
- 공방 가공기 1대 설치 → 슬롯 3칸.
- 공방 가공기 2대 설치 → 슬롯 6칸.
- 최대 설치 가능 수는 영지 시설 슬롯 총량에 의해 자연스럽게 제한됨.

운영 원칙:

- 제작 완료 알림은 인게임 액션바 + 디스코드 봇 선택 알림으로 제공한다.
- 공방 가공기를 많이 설치할수록 대기열이 늘지만 채집/채굴 시설 슬롯이 줄어든다. (트레이드오프)

## 6. 연속 실패 가호 시스템 — 천장 방식 (확정 2026-05-15)

### 설계 (확정)

- 확률 보너스 없음. 시도 횟수 천장 도달 시 무조건 성공.
- **천장 = `ceil(2 / 성공률)`** (기댓값의 2배, 올림).
- 성공 시 해당 슬롯 카운터 초기화.
- 슬롯별 독립 카운터.
- 적용 구간: 11강 이상.
- 강화 GUI에서 현재 카운터 / 천장 표시.

| 경매 방식 | 고정가 즉시 거래 (입찰 방식 없음) |
| 등록 만료 | 72시간 미거래 시 아이템 반환, 수수료 없음 |
| 접근 조건 | 전투 중만 차단, 필드 내 허용 (영지 서브 허브 → [경매]) |
| 전승권 | 경매소 거래 불가 (상점 고정가 전용) |

## 15. 경매소 경제 의도 및 모니터링 지표

> 추가일: 2026-05-14.

### 강화석 파편 경매 루프

**설계 의도:** 17강 이후 강화석 시도당 소모량이 줄어도 기대 총소모는 급증한다 (16강 피크 → 17~20강 재차 증가). 이 구조는 의도적이다.

| 유저 유형 | 강화석 상태 | 골드 상태 | 경매 행동 |
|---|---|---|---|
| 초반 / 부족 유저 | 잉여 (0→10강 구간 쌓임) | 부족 (강화 못 함) | **강화석 판매** |
| 선발대 (16강+) | 부족 (17~20강 소모 급증) | 잉여 (하드 파밍) | **강화석 구매** |

강화석 파편 경매 시세는 골드 경제의 핵심 지표다. 시세가 낮으면 선발대 강화석 소모가 원활하지 않은 것, 시세가 너무 높으면 초반 유저 골드 수급이 지나치게 유리한 것이다.

### 잠재 큐브 경매 루프

**설계 의도:** 큐브를 모았지만 유니크에서 만족하고 "주차"를 선택한 유저가 여분 큐브를 경매소에 내놓는다. 레전더리를 노리는 선발대가 이 큐브를 골드로 구매한다.

| 유저 유형 | 큐브 상태 | 행동 |
|---|---|---|
| 유니크 주차 유저 | 여분 큐브 보유 | **큐브 경매 판매** → 골드 수입 추가 |
| 레전더리 지향 선발대 | 큐브 부족 | **큐브 경매 구매** → 잠재 롤 추가 |

레전더리 확률 1% 기준 기댓값 100회 → 큐브 수요가 지속된다. 큐브 시세는 레전더리 가치(아이템 주스텟 +8% 보정)와 수렴해야 한다.
