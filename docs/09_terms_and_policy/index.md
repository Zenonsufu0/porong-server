# 09. 약관 / 운영정책 핵심

## 약관 범위

현재 초안은 유지한다. 세부 법적 문구는 정식 공개 전 한 번 더 검토한다.

## 필수 고지

- 45일 프로젝트 서버
- 데이터 보존/초기화 가능성
- 디스코드-마크 계정 연동
- 디스코드 ID, 마크 UUID/닉네임, 플레이 기록 저장
- 핵/매크로/버그 악용/현금거래 금지
- 경제 안정화를 위한 운영 개입 가능
- 리소스팩 무단 추출/재배포 금지
- 성장형 유료 상품 없음
- 자발적 커피값 후원은 보상 없는 후원으로만 가능

## 인증 문구

약관 동의 후 닉네임 입력, 임시 화이트리스트, 최종 코드 인증 구조로 간다.
