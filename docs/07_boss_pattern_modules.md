# 07. 보스 패턴 모듈

## 원칙

보스 패턴은 개별 하드코딩하지 않는다. 패턴 모듈, 보스별 조합, 테마 연출 스킨, 수치 파라미터를 분리한다.

## 공용 패턴 후보

- 전방 강타
- 직선 돌진
- 원형 폭발
- 부채꼴 휩쓸기
- 투사체 발사
- 산개탄
- 소환체 호출
- 페이즈 전환
- 무력화/딜타임
- 안전지대 패턴

## 계층별 재사용

- 필드보스: 공용 80%, 고유 20%
- 시즌보스: 공용 60~70%, 고유 30~40%
- 시즌 최종보스: 공용 40%, 고유 60%

## 2차 확장

1차에서는 바닐라 강화형으로 만들고, 2차에서 외형만 ModelEngine/BetterModel/FMM 계열로 교체할 수 있게 보스 ID와 패턴 로직을 분리한다.
