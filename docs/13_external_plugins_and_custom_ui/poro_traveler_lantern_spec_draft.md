# 포로 서버 여행자의 등불 규격서 (소모품형 커스텀 아이템)

## 문서 목적
영지 외 블록 설치 금지 정책 하에서 야외 탐험 안전 확보 수단인 **여행자의 등불** 1차 MVP 규격. feature-planner 초안 기반.

## 배경
- 바닐라 횃불과 다른 **커스텀 아이템** (인벤 보유 시 주변 광원)
- **소모품(연료)형 확정** — 시간당 연료 소모, 고갈 시 재충전
- 월드 사운드 제거 a안 준수 (획득·UI 사운드는 유지)

## 1. 연료 3종 (1차 MVP)

| 연료 | 지속시간 | 광원 반경·광도 | 서브 효과 | 수급 |
|---|---|---|---|---|
| **기름 (Lamp Oil)** | 30분/개 | 반경 6 / 광도 12 | — | 상점·올인원 조합대 제작·메인 퀘스트 보상 |
| **마력 결정 조각** | 60분/개 | 반경 7 / 광도 13 | 반경 +1 | 던전·필드 드랍·파편 가공 |
| **성화 향** | 20분/개 | 반경 6 / 광도 12 | 언데드·어둠 속성 피해 감소 보정 | 퀘스트·지역 보상 |

> 바닐라 횃불(반경 14·광도 14)보다 **의도적으로 약하게** 설정 — 거점(영지) 광원과 차별화, 야외에서 "아주 밝지는 않지만 안전 최저선 확보" 톤.

## 2. 등불 본체 규격
- 최대 연료 슬롯: **기름 환산 240분(=8개)**
- 차감 틱: **30초 단위** (부하 절감, 1초 틱 아님)
- **로그아웃 중 소모 없음** (접속 시간 기준만)
- PDC(PersistentDataContainer)에 연료 잔량(분 단위) 저장
- 인벤 이동 시 광원 위치 재계산 주기 최소 0.5초

## 3. 재충전 UX
- **1차 MVP**: 인벤에서 등불 **우클릭** → 연료 선택 GUI → 보유 연료 중 선택 주입
- 올인원 조합대 "재충전" 카테고리는 2차로 미룸 (UX 분산 방지)
- 영지 내 연료 공급 설비는 3차 확장 오픈 질문

## 4. 수급 경로
- **상점 구매**: 기름 1개 소액 골드 (T1 소비재 가격대)
- **메인 퀘스트 초반 보상**: 첫 야간 탐험 트리거 퀘스트 = 등불 본체 1개 + 기름 8개
- **올인원 조합대 특수 제작**: 기름 = 동물성 지방·식물성 기름 계열 채집물 / 마력 결정 조각 = 파편 가공
- **채집기 부산물**: 수렵·양초 계열 채집기 저확률 부산물 (오픈 질문)

## 5. 바닐라 횃불과 시각 차별화
- **모델**: 랜턴·등불 형태 CustomModelData
- **색조**: 청색~청록 톤 (마법 광원 연출, 바닐라 주황과 구분)
- **소리**: 낮은 마력 웅웅거림 loop SFX 저볼륨

## 6. 파티 광원 공유
- **1차 MVP: 공유 없음** — 각자 등불 휴대 원칙
- 근거: 파티원 주변 광원 연산 비용 / 무임승차 방지 / 연료 경제 일관성
- 2차에서 "파티 리더 등불 강화 연료" 형태 공유 옵션 고려 (오픈 질문)

## 7. 구현 포인트
- 광원 구현: **FakeBlockLight(패킷 기반)** 또는 **LightAPI** — Paper 호환성 검증 필요
- 연료 선택 GUI는 공통 인벤 UI 프레임워크 재사용
- 30초 차감 틱은 플레이어별 독립 스케줄러
- 연료 잔량 UI: 등불 아이콘에 잔량 바 오버레이 (인벤 툴팁)

## 8. 오픈 질문
- 채집기 부산물로 기름 확보 가능 여부 — 채집기 설계팀 확인
- 영지 내 연료 공급 설비 도입 여부·시점
- 파티 공유 기능 2차 도입 시 경제 영향
- 성화 향 서브 효과가 전투 밸런스(피해 증가 카테고리)와 충돌 — combat-balance 검토
- 로그아웃 중 연료 소모 정책 최종 확정
- "마력 결정 조각" 네이밍이 기존 파편·조각 체계와 혼동 가능성 — 재검토 필요
- 광원 구현(LightAPI vs FakeBlockLight) Paper 최종 호환성
