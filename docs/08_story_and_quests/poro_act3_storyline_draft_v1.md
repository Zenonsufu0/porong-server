## 문서 목적
2막 씬 카드 v1(`poro_act2_capital_and_themes_scene_cards_v1.md`) 완성 후 이어지는 **3막 전체 구조 초안**. content-designer 작성, 2026-04-19. 메인 21~30 + 사하르·아르카논 전용 씬 + 수도 씬 + 세력별 결실 씬을 1장에 묶은 구조 초안이며, 세부 대사·플래그 최종값·보상 수치는 후속 v2에서 결정. 씬 카드는 1~2문단 압축 버전으로 작성한다.

## 개요
- 3막은 **"덮개가 완전히 들춰진 이후, 이름 없는 자들이 제 이름을 찾는 시간"**이다.
- 2막 메인 20 결전 직후 레오니드가 "꼬리의 꼬리"를 추적하기 위한 **두 개의 새 도시(사하르·아르카논) 공식 접촉**을 승인한다.
- 2막에서 결정한 세력(기사단 / 그림자)이 본격적으로 **재분기 서사**로 열매를 맺는다. 메인 15 선택은 3막 메인 27·28·29에서 완전히 다른 보상·연출·관계로 드러난다.
- 4막 복선 아이템 7종(수도·동·서·남·북 + 사하르·아르카논)이 모두 **3막 결전 보스 부산물**로 회수된다. 1막 심기 → 2막 저강도 재관측 → 3막 회수 3단 구조의 마지막 단계.
- 카엘은 3막 종료 시점에 **진짜 소속**이 드러난다. 관찰자 NPC는 3막 중반 처음으로 플레이어와 동일 프레임에 선다.

## 목표
- 3막 메인 21~30 (10개) 각 단계를 한 줄 설명 + 분기 A/B/공용 구분으로 제시한다.
- 사하르 3막 개방 씬 3~5종, 아르카논 3막 개방 씬 3~5종을 압축 카드로 정리한다.
- 세력별 결실 씬 2종(기사단 / 그림자)을 대칭 구조로 설계한다.
- 3막 결전(메인 30 근처) 보스를 **세력별 2종**으로 나누고, 4막 결전 `황성 심핵의 재앙체`와의 관계를 정리한다.
- 수도 3막 씬 5~7종으로 사하르·아르카논 개방 계기 + 카엘·관찰자 재등장 타이밍을 배치한다.
- 4막 복선 아이템 회수표를 테마별 1표로 정리한다.

## 핵심 규칙
- 2막 메인 15 세력 선택은 **3막에서 완전히 다른 보상·연출로 분기**. 3막 메인 27~29가 세력별 코드 분리 지점.
- 사하르·아르카논은 **3막 개방이 확정된 새 지역**이며, 강제 방문이 아니다. 플레이어는 사하르·아르카논·세력 결실 중 원하는 순서로 진행 가능(파견지 강제성 X 원칙 계승).
- **T2 완제품 직드랍 금지**는 3막에서도 유지. T2는 재료 제작형 구조.
- 각인서 드랍 상한을 3막에서 해제한다. 메인 결전·세력 결실에서 **레전드리 각인서 드랍 허용**. 시즌1 45일차 상한 기준선은 `poro_boss_drop_grade_matrix_v1.md` 기준을 따른다.
- 4막 복선 회수는 **모두 3막 결전 보스 부산물**로 처리. 회수 시 UI 토스트만, 특별한 NPC 반응 없음(저강도 원칙 유지).
- 파티 3명 상한은 3막 결전 기믹에서도 전제.
- 관찰자 NPC의 "공식 노출"은 3막 후반 단 1회. 이전까지는 스쳐가는 실루엣 수준.

# 1. 3막 한 줄 감각과 감정 그래프

## 한 줄 감각
> **"덮개가 들춰진 뒤, 이름 없는 자들이 제 이름을 찾아 걷는다."**

1막이 "황성의 얼굴"을, 2막이 "황성의 숨결"을 드러냈다면, 3막은 **"황성 바깥에서 황성에게 이름을 요구하는 시간"**이다. 사하르는 계약에 이름을 새기려 하고, 아르카논은 울리지 않던 공명에 이름을 부여하려 한다. 기사단 수습은 정식 기사로, 그림자의 무명 증인은 제 이름을 되찾는 의식으로 결실을 본다. 카엘조차 더 이상 "뒤에서 따르는 사람"이 아니라 **제 소속을 밝히는 이름**으로 선다.

## 감정 그래프 5단

1. **외곽 도시 접촉 (메인 21~23)** — 호기심·환대. 사하르·아르카논이 공식 개방되고 플레이어는 두 도시 중 한 곳에서 먼저 접촉한다. 환영·계약·공명이라는 새 언어를 배우는 시간. 2막의 무거운 톤이 한 번 환기된다.
2. **세력별 핵심 임무 심화 (메인 24~26)** — 의심·결속. 사하르와 아르카논 각각에서 "두 번째 접촉"이 벌어진다. 기만과 공명 이면의 비공식 신호가 플레이어 세력 증표와 반응한다. 카엘이 반대 세력 쪽에서 모습을 드러낸다.
3. **세력별 재분기 결실 (메인 27·28)** — 무게·결단. 기사단 루트는 정식 기사 승격 서사, 그림자 루트는 이름을 되찾는 의식 서사. 양쪽 모두 **그동안 쌓인 평판·플래그·선택 이력이 형태를 갖추는** 지점이다. 2막 메인 15의 선택이 여기서 처음으로 "그 선택을 하지 않았다면 달랐을 것"이라는 무게로 돌아온다.
4. **3막 결전 (메인 29·30)** — 폭발·정적. 세력별 결전 보스 2종 중 자기 세력 루트에 해당하는 보스를 격퇴한다. 4막 복선 아이템 7종이 모두 회수된다. 보스가 소멸하는 순간 카엘의 진짜 소속이 한 번의 대사로 드러난다. 관찰자가 처음으로 플레이어와 같은 화면에 선다.
5. **4막 예고 (메인 30 직후)** — 침묵. 레오니드·세리안·잿빛이 한자리에 모인다. 관찰자는 그 방의 벽에 그림자로 서 있다. 메인 20에서 기록된 "심핵 공명 +1"이 이 시점 **+7까지 누적**되어 있다는 사실이 로그에만 조용히 찍힌다. 4막 진입 판정은 이 누적값이 연다.

# 2. 메인 퀘스트 단계 (메인 21~30)

각 단계는 한 줄 설명 + 분기 구조 + 핵심 플래그 키를 포함한다. 분기 A=기사단(`act2_faction.knights`), 분기 B=그림자(`act2_faction.shadow`), C=공용.

## 메인 21 — 「두 개의 공식 승인」 (C)
- **한 줄**: 레오니드가 사하르·아르카논 접촉을 동시에 공식 승인한다. 플레이어는 **어느 도시를 먼저 방문할지 자유 선택**한다(강제성 X).
- **분기 영향**: 없음. 단, 세력 증표가 도시 입성 첫 대사에 다르게 읽힌다.
- **4막 떡밥**: 레오니드가 "꼬리의 꼬리" 너머 "몸통"에 대해 처음으로 한 줄을 흘린다. 로그에만 남는다.
- **플래그**: `quest.act3_main.m21_dual_city_authorized`, `quest.act3_main.m21_first_city_choice` (sahar|arcanon)

## 메인 22 — 「사하르 상인연합 접촉」 (C / 사하르 선행 시)
- **한 줄**: 사하르 상인연합 중개인이 플레이어에게 "견본 계약"을 건넨다. 환영 판별 초급 체험.
- **분기 영향**: 1막 서부 서브 `사하르 상단 신뢰` 플래그 보유 시 중개인의 태도가 정중해진다(대사 1줄 변화).
- **4막 떡밥**: 없음.
- **플래그**: `quest.act3_main.m22_sahar_contact_made`, `quest.theme.sahar.sample_contract_signed`

## 메인 23 — 「아르카논 공명 관문」 (C / 아르카논 선행 시)
- **한 줄**: 아르카논 공명 관문 관리관이 플레이어에게 "구획 이동권"을 발급한다. 공명 표식 초급 체험.
- **분기 영향**: 1막 결정 `마력 채집기·정제기 해금` 보유 시 관리관이 "이미 준비된 분"이라고 읽는다(대사 1줄 변화).
- **4막 떡밥**: 없음.
- **플래그**: `quest.act3_main.m23_arcanon_contact_made`, `quest.theme.arcanon.gate_pass_issued`

## 메인 24 — 「사하르 환영 판별 고난이도」 (C)
- **한 줄**: 사하르 환영 시장 심층에서 **계약한 적 없는 계약**의 실체를 추적한다. 환영 판별 상급 + 정예 `환영 사냥군주`.
- **분기 영향**: 기사단 = 계약서 정본 증거 확보 경로 / 그림자 = 유출본 추적 경로. 보상 플래그 공통.
- **4막 떡밥**: 사하르 계약의 증인이 "위쪽에 더 큰 계약이 있다"고 흘린다. 로그에만.
- **플래그**: `quest.act3_main.m24_sahar_illusion_resolved`, `flag.foreshadow.m24_greater_contract_above`

## 메인 25 — 「아르카논 봉인층 첨탑」 (C)
- **한 줄**: 아르카논 공명 시계탑의 **봉인된 한 층**에 진입한다. 공명 표식 상급 + 정예 `균열 드론 군집장 Ⅱ`.
- **분기 영향**: 기사단 = 관문 관리관 공식 동행 / 그림자 = 연구원 망명자 동행. 동행자 대사만 분기.
- **4막 떡밥**: 봉인층 벽에 적힌 수식의 마지막 항이 "Ψ"와 같은 기호. UI 명시 없음.
- **플래그**: `quest.act3_main.m25_arcanon_sealed_floor_cleared`, `flag.foreshadow.m25_psi_glyph_on_wall`

## 메인 26 — 「거울의 도시, 같은 방에 다시」 (C / 4막 떡밥 재관측 1지점)
- **한 줄**: 사하르·아르카논 양 도시가 동일 시각 동일 주기로 "같은 공명 현상"을 기록했다는 사실이 잿빛의 관측소에 합쳐진다. 수도 귀환 후 3자(레오니드·세리안·잿빛) 브리핑.
- **분기 영향**: 없음. 합류 씬.
- **4막 떡밥**: 잿빛이 Ψ 좌표 옆에 **새 좌표 1개**를 추가한다. 공식 설명 없음(저강도).

## 메인 27 — 「수습의 마지막 증명」 (A 전용) / 「이름 없는 자의 첫 서명」 (B 전용)
- **한 줄 (A)**: 기사단 원로회가 수습 기간 종료를 선언하고 **정식 기사 서임 전 마지막 증명**을 요구한다.
- **한 줄 (B)**: 그림자 회중이 "이름을 되찾는 의식" 전 **첫 서명 절차**를 준비한다.
- **분기 영향**: 상호 배타. 선택 세력에 따라 완전히 다른 씬으로 진입. 양쪽 모두 2막까지의 세력 평판·서브 이력이 대사에 반영된다.
- **4막 떡밥**: 없음. 다만 양쪽 서사 모두 "당신이 다음에 서야 할 자리는 여기가 아니다"로 끝난다.
- **플래그**: `quest.act3_main.m27_knights_final_proof` (A) / `quest.act3_main.m27_shadow_first_signature` (B)

## 메인 28 — 「원로 기사 합류」 (A) / 「은둔 연락원 복귀」 (B)
- **한 줄 (A)**: 2막 메인 15에서 수습 서약을 받았던 시험관 B의 스승, **원로 기사 텔레온**이 플레이어 파티에 일시 합류한다.
- **한 줄 (B)**: 회중이 오랜 기간 접촉을 끊었던 **은둔 연락원 "이름 없는 12번"**이 복귀해 플레이어 파티에 일시 합류한다.
- **분기 영향**: 합류 NPC는 3막 결전에서 파티 3명 중 1자리를 대체할 수 있다(플레이어 선택). NPC 합류 시 파티 2명 + NPC 1명 구성.
- **4막 떡밥**: 텔레온은 "첨탑 아래"라는 단어를 2막 이후 처음으로 구체화한다. 12번은 "이름을 잊은 사람이 가장 먼저 알아본다"는 대사를 남긴다.
- **플래그**: `quest.act3_main.m28_knights_elder_joined` (A) / `quest.act3_main.m28_shadow_witness_12_returned` (B)

## 메인 29 — 「결전 직전, 카엘의 편지」 (C / 4막 떡밥 재관측 2지점)
- **한 줄**: 결전 직전 플레이어에게 카엘의 편지가 도착한다. **편지에 처음으로 카엘의 소속이 명시되어 있다** — 그러나 수신자 이름 칸은 비어 있다.

- **플래그**: `quest.act3_sub_capital.c2_seriani_first_question`, 값 responsibility|curiosity|unknown

## C3 — 「카엘, 2막 이후 첫 재등장」 (공식 3회차)
카엘이 수도 행정동 4층 대기실(메인 12와 같은 장소)에 앉아 있다. 플레이어가 도착하자 카엘은 **반대 세력 증표**를 테이블 위에 올려둔 상태다. 대사는 짧고 서로 인사만 나눈다. 이 씬은 **2막 오픈 질문 "카엘 등장 3회 vs 5회"의 3막 해답** — 3막 추가 등장 2회(C3·메인 29)로 최종 5회 완성.
- **보상**: 없음(서사 씬).
- **플래그**: `quest.act3_sub_capital.c3_kael_third_encounter`

## C4 — 「잿빛의 벽, 새 좌표」 (메인 26 직후 허브 씬)
잿빛이 관측소 벽 지도에 Ψ 옆 두 번째 좌표를 추가하는 순간을 플레이어가 직접 목격한다. 플레이어가 묻지 않는 한 잿빛은 아무 설명도 하지 않는다. 선택지: "설명해 주시겠습니까?" (저강도 떡밥 로그 1건) / 침묵(잿빛 호감 +소).
- **보상**: 잿빛 호감 변동, 관측 로그 1부.
- **플래그**: `quest.act3_sub_capital.c4_psi_secondary_revealed`, `flag.foreshadow.c4_psi_second_point_inquiry`

## C5 — 「관찰자 공식 노출」 (3막 후반 1회 한정)
메인 29 이후 수도 뒷골목 한 지점에서 관찰자 NPC가 **처음으로 플레이어와 같은 화면 안에** 선다. 말은 없다. 관찰자는 플레이어 쪽을 **정면으로 보고**, 플레이어가 다가가면 사라진다. 이 씬은 4막 진입 판정의 **저강도 재관측 마지막 지점**이며, 플래그는 UI 미표시.
- **보상**: 없음(서사 씬). 로그에만 "관찰자 접촉 기록 +1" 찍힌다.
- **플래그**: `flag.foreshadow.c5_observer_public_exposure`

## C6 — 「3막 결전 후, 네 사람의 방」 (메인 30 직후)
3막 결전 클리어 후 레오니드·세리안·잿빛·로웬이 수도 집무실에 모인다. 관찰자는 벽 그림자로 서 있다. 레오니드가 "이제부터는 기록되지 않는 회의를 시작합니다"라고 말하며 회의실 문을 닫는다. 플레이어는 문밖에 남는다. 4막 예고의 단 한 장면.
- **보상**: 4막 예고 로그 1건, 3막 종결 칭호 `봉인층 상위를 본 자` 진입 가능.
- **플래그**: `quest.act3_sub_capital.c6_four_room_meeting_closed`

3막 결전 보스(세력 A / B 공통)에서 플레이어 인벤 보유분 기준으로 자동 회수. 회수는 UI 토스트만, NPC 반응 없음(저강도).

# 9. 공통 구현 포인트

- **플래그 저장소**: `quest.act3_main.*`, `quest.theme.sahar.*`, `quest.theme.arcanon.*`, `quest.act3_sub_capital.*`, `flag.foreshadow.*`, `item.foreshadow.*_planted|granted` 계열. 복합 JSON 구조가 필요한 경우(예: 3막 결전 7종 회수 상태 동시 기록)는 **v0.2 JSON 허용** 대기 가능.
- **세력 상호 배타 검증**: 메인 27·28·30과 세력 결실 K/D는 반대 세력 수락자에게 **접근 자체 차단**. 세력별 2종 결전 보스는 동일 인스턴스 공간이 아닌 별도 공간으로 분리.
- **파티 3명 상한**: 메인 28 NPC 합류 시 "파티 2 + NPC 1" 구성. 솔로 진입 시 NPC 동반 강제 or 난이도 스케일 다운(combat-balance 후속).
- **4막 떡밥 저강도 원칙**: 3막에서도 `flag.foreshadow.*` UI 표시 금지. 관찰자 C5 씬, 잿빛 C4 씬, 메인 25 수식 글리프, 메인 26 두 번째 좌표 모두 UI 토스트 없이 로그만.
- **복선 회수 자동 스캔**: data-schema 후속 작업 필요. 인벤·창고·차원도서관 3탭 동시 스캔 로직, 회수됨 상태 영구 저장.
- **레전드리 각인서 드랍 허용**: 메인 30 세력별 보스 + 세력 결실 K/D에서 허용. 드랍 확률은 등급 드랍 매트릭스 시즌1 45일차 상한 기준선(`poro_boss_drop_grade_matrix_v1.md`) 준수. combat-balance 후속 검토.
- **BetonQuest 미사용**: 대화 UI는 EmpireRPG 커스텀 UI 또는 click-handler. 선택지·플래그 분기 로직은 기존 2막과 동일 방식 계승.
