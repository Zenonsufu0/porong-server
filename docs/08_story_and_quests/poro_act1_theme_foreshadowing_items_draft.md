# 포로 서버 테마별 4막 복선 아이템 7종 설계

## 문서 목적
기존 수도 `식어버린 황성 파편`(1종) 외에 **3막 테마별 결전 보스 6개 각각의 복선 아이템**을 추가 설계해 7종 체계로 확장. content-designer 1차 초안.

## 설계 원칙 — 기능·서사 분리 2축
- **수도 1종 (기존 유지)**: `식어버린 황성 파편` — **게임플레이 키 아이템** (4막 시즌1 최종 보스 `황성 심핵의 재앙체` 진입 키)
- **나머지 6종 (본 문서)**: **컬렉션·서사 전용** (3막 테마별 결전 회수, 게임플레이 기능 없음)
- 6종 수집 시 숨겨진 칭호 **`[일곱 결전의 증인]`**, 7종 전체 수집 시 상위 칭호 **`[봉인된 이름을 본 자]`**(오픈)

## 공통 규칙
- **귀속 / 파괴 불가 / 창고 전용 복선 칸**(`memoria` 또는 `echoes` 탭 가칭)
- 1~3막 획득 시점엔 **효과·용도 전무**
- 해당 테마 3막 결전 보스 킬 판정 → 인벤/창고 자동 스캔 → `미완` → `회수됨` 전환
- 획득 시 UI 토스트만, 특별한 NPC 반응 없음 (저강도 원칙)

- **대응 결전**: **3막 수도 결전보스 `황성 심핵의 재앙체`** (사용자 확정 — 기존 "4막 진입 키" 철회)
- 심기: 1막 분기 B 메인 13 (황성 외곽 잠식 기사 보스 부산물)
- 회수: **3막 수도 결전보스 `황성 심핵의 재앙체` 클리어 시점**
- 기능: **서사·컬렉션 전용** (나머지 6종과 동일 취급, 게임플레이 키 기능 제거)
- **7종 완전 대칭 구조 완성**: 수도·동·서·남·북·사하르·아르카논 모두 3막 결전 회수

## 동부 침식 숲 — `침묵한 정령의 눈물` (권장 이름)

- **대응 결전**: 3막 `침식 정령왕`
- **이름 후보**: (1) 마르지 않는 수액 한 방울 / (2) 썩지 않는 이파리 / **(3) 침묵한 정령의 눈물** ← 권장
- **심기**: 1막 분기 A 메인 9 (동부 첫 진입) **서브 퀘스트**(오래된 정령 사당 정화 또는 숲지기 유품 회수)
- **회수**: 3막 동부 결전 `침식 정령왕` 2페이즈 전환 연출 직후
- **설명 초안**:
  > *"이파리 위에 고인 물방울 하나. 썩지 않고, 마르지 않고, 그저 멈춰 있다. 정령이 울었던 흔적이라 전해지지만, 누구를 위해 울었는지는 아직 밝혀지지 않았다."*
- **테마 연결**: 동부 = 침식/부패/멈춘 시간. 정령왕이 삼키지 못한 마지막 양심의 잔재.
- **회수 대사**: 침식 정령왕 *"…그 눈물이, 아직 남아 있었구나."*

## 서부 황야 — `부서진 풍향계 조각` (권장 이름)

- **대응 결전**: 3막 `황야 폭풍의 군주`
- **이름 후보**: (1) 바람이 새겨진 돌조각 / **(2) 부서진 풍향계 조각** ← 권장 / (3) 모래에 묻힌 깃발 조각
- **심기**: **재설계 중 (사용자 결정: 필드보스 5개 유지)**. 기존 "서부 초입 소형 필드보스 신설 드랍"(안 A) + "메인 10 필드보스 조기 활성화"(안 B) 모두 반려. 대안 4종 중 택일 대기:
  - **(a) 서부 서브 퀘스트 신설** — 동·북과 대칭("풍향계 폐허 조사", 권장)
  - **(b) 특수 지형 인터랙션** — 서부만 대칭 깨고 환경 오브젝트
  - **(c) 메인 9 스토리 이벤트 고정** — 남부와 동일 예외 확장
  - **(d) 기존 잡몹 저확률 드랍** — 재미 체감 낮음, 비권장
- **회수**: 3막 서부 결전 `황야 폭풍의 군주` 최종 페이즈 돌입 연출. 인벤 조각 공명 → 클리어 직후 "완성된 풍향계"로 변환
- **설명 초안**:
  > *"방향이 지워진 낡은 풍향계의 한 조각. 손가락 끝에 대면 아주 희미하게 한 방향으로 떨린다. 그 방향이 어디를 가리키는지는, 아직 알 수 없다."*
- **테마 연결**: 서부 = 방향 상실/폭풍/황야의 침묵. 군주 = 방향 잃은 바람의 화신 → "군주도 아직 제 방향을 모른다"는 미완 훅.
- **회수 대사**: 폭풍의 군주 *"…네 손에 있던 것이, 결국 나를 찾아왔군."*

## 남부 화맥 — `식지 않는 재 한 줌` (권장 이름)

- **대응 결전**: 3막 `라그네스`
- **이름 후보**: **(1) 식지 않는 재 한 줌** ← 권장 (수도 `식어버린 황성 파편`과 "식어버린" ↔ "식지 않는" 언어적 대칭 구조, 4막 최종 보스와의 서사 결속 암시) / (2) 꺼지지 않는 불씨 조각 / (3) 화맥의 낡은 낙인
- **심기**: 1막 분기 A 메인 9 (남부 첫 진입) **메인 퀘스트 말미 이벤트 인벤 지급** — **4테마 중 남부만 메인에 심는 예외** (수도 파편과 대칭 강조)
- **회수**: 3막 남부 결전 `라그네스` 클리어 직후, 라그네스 몸이 재로 흩어지는 연출 중 인벤 재 공명
- **설명 초안**:
  > *"손바닥 위에서 결코 식지 않는 재. 불씨도 아니고 숯도 아니다. 무엇이 이 재를 이토록 오래 뜨겁게 두고 있는지, 지금의 그는 아직 답을 가지고 있지 않다."*
- **테마 연결**: 남부 = 화맥/용광로/꺼지지 않는 분노. 라그네스 = 화맥 그 자체의 의지 → "라그네스가 남긴 재가 아니라, 라그네스를 태우고 있는 재"라는 역설.
- **회수 대사**: 라그네스 *"…그 재를, 아직 들고 있었나. 그럼… 기다린 보람이 있군."*

## 북부 설혼 — `얼어붙은 숨 한 조각` (권장 이름)

- **대응 결전**: 3막 `모르바인`
- **이름 후보**: (1) 녹지 않는 서리 결정 / **(2) 얼어붙은 숨 한 조각** ← 권장 (설혼 "숨마저 얼어붙는 정적"과 모르바인 "침묵의 왕" 성격 일치) / (3) 설혼의 낡은 인장
- **심기**: 1막 분기 A 메인 9 (북부 첫 진입) **서브 퀘스트** (설원 조난자 유품 회수 또는 얼어붙은 사당 조사)
- **회수**: 3막 북부 결전 `모르바인` 2페이즈 종료 후 정적 연출 중
- **설명 초안**:
  > *"누군가가 마지막으로 내쉰 숨이 그대로 얼어붙은 듯한 투명한 조각. 녹지 않는다. 차갑지도 않다. 그저 멈춰 있을 뿐이다. 이것이 누구의 숨이었는지는, 아직 아무도 묻지 않았다."*
- **테마 연결**: 북부 = 설혼/정적/멈춘 숨. 모르바인 = 침묵으로 지배하는 왕 → "모르바인이 삼키지 못한 마지막 숨 한 조각".
- **회수 대사**: 모르바인 *"…그 숨이, 네게로 흘러들었구나. 그렇다면 다음은 네 차례다."*

## 사하르 상인연합 — `이름이 지워진 계약 인장` (권장 이름)

- **대응 결전**: 3막 `사하르 계약군주`
- **이름 후보**: (1) 서명 없는 계약서 한 장 / **(2) 이름이 지워진 계약 인장** ← 권장 (계약군주 "모든 거래에 이름을 새긴다" 성격과 정확히 대응) / (3) 값이 매겨지지 않은 동전
- **심기**: **3막 사하르 개방 초입 씬 카드** (1·2막 심기 불가, 3막 개방이므로). 계약의 전당 NPC "견본 인장" 퀘스트 보상
- **회수**: 3막 사하르 결전 `사하르 계약군주` 클리어 직후, 인장에 플레이어 이름이 새겨지며 회수
- **설명 초안**:
  > *"모든 거래에는 이름이 새겨진다. 그러나 이 인장에는 아직 아무 이름도 없다. 이것이 누구의 계약이 될지는, 인장 자신도 알지 못한다."*
- **테마 연결**: 사하르 = 계약/거래/값의 무게. 계약군주 = 모든 계약의 증인 → "아직 맺어지지 않은 계약" = 플레이어 본인.
- **회수 대사**: 계약군주 *"…드디어, 그 인장에 이름이 새겨지는군. 오래 기다렸다."*

## 아르카논 마법공학 도시 — `울리지 않는 공명 시계` (권장 이름)

- **대응 결전**: 3막 `카르멘`
- **이름 후보**: (1) 작동하지 않는 공명핵 / **(2) 울리지 않는 공명 시계** ← 권장 / (3) 정지한 기어 한 점
- **심기**: **3막 아르카논 개방 초입 씬 카드** (1·2막 심기 불가). 공방 NPC "시제품 테스터" 퀘스트 보상
- **회수**: 3막 아르카논 결전 `카르멘` 클리어 직후, 카르멘 공명이 멈추는 순간 인벤 시계가 처음으로 1회 울리며 회수
- **설명 초안**:
  > *"시계 모양의 공명핵. 태엽도 동력원도 없는데, 손안에서 단 한 번도 울린 적이 없다. 이 시계는 언젠가 단 한 번 울리기 위해 만들어졌다. 그 순간이 언제인지는 설계자도 몰랐다."*
- **테마 연결**: 아르카논 = 공명/마법공학/이론과 실증. 카르멘 = 공명의 정점 마법공학자 → "카르멘의 최후 공명에 맞춰 설계된 단발 기물".
- **회수 대사**: 카르멘 *"…결국, 네 손에서 울리는구나. 내가 틀리지 않았다."*

## 콘텐츠 공수 평가

| 작업 | 공수 | 비고 |
|---|---|---|
| 동·북 서브 심기 | 낮음 | 기존 서브 카드 1~2줄 수정 + 보상 인벤 라인 |
| 서부 필드보스 드랍 | 낮음 | 드랍 테이블 저확률 슬롯 1칸 |
| 남부 메인 심기 | 중간 | 엔딩 이벤트 대사 1~2줄 + 인벤 지급 로직 |
| 사하르·아르카논 3막 초입 | 낮음 | 3막 씬 카드 미확정이라 **지금 미리 반영** 시 0공수 근접 |
| 6개 결전 클리어 연출 | 중간 | 대사 1~2줄 + UI 토스트 |
| 복선 칸 UI | 중간 | 창고 전용 탭 1개 신설, 6+1 슬롯 |
| 수집 칭호 | 낮음 | 기존 칭호 시스템 1종 추가 |

총평: 테마당 2~3줄 대사·이벤트 수준으로 씬 카드 작업 부담 작음. **복선 칸 UI + 회수 트리거**는 공통 시스템 작업으로 분리 권고.
