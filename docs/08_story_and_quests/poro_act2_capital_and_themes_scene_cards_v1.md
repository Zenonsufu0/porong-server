# 포로 서버 2막 — 수도 + 동/서/남/북 4테마 씬 카드 (v1 드래프트)

## 문서 목적
2막 전체 구조 초안(`poro_act2_theme_main_storyline_draft.md`)이 남긴 **나머지 메인 씬 카드 + 서브 씬 카드 + 4테마 확장**을 1장으로 묶은 1차 드래프트. content-designer 작성, 2026-04-19. 세부 대사는 최종판이 아닌 톤 샘플이며, 플래그 키는 구현팀 참조용 예시다.

## 개요
- 2막은 "관측 이상치 확인 → 세력 선택 → 외곽 교차 조사 → 봉인층 하위 개체 격퇴"라는 뼈대를 따른다.
- 수도는 서사 허브 역할을, 4테마는 교차 조사 현장 역할을 맡는다.
- 분기 선택(메인 15)은 각 테마에서 최소 한 번씩 영향 축으로 돌아온다.
- 4막 복선 아이템 6종 중 수도를 제외한 4종(동·서·남·북)은 본 문서에서 회수가 아니라 **흔들리는 재관측**으로 재등장한다. 실제 회수는 3막 결전.

## 목표
- 메인 13·14·16·17·18·19·20의 씬 카드 제시.
- 수도 서브 6종(C~H) 추가, 기존 A·B와 합쳐 서브 8종 체계 마감.
- 동·서·남·북 각 4~5개 메인 씬 카드 + 3~5개 서브 씬 카드 제시.
- 4막 결전 떡밥 연결 지점을 테마별로 명시.
- 세력 선택 영향이 각 테마에서 최소 1회 드러나게 배치.

## 핵심 규칙
- 2막 메인 12·15는 이미 작성된 샘플을 그대로 존중. 본 문서는 12·15 이후와 사이 구간에 집중.
- 등급 드랍 매트릭스를 존중해 2막 보상은 **커먼·레어 중심 + 에픽 일부 + 유니크 드물게 + 레전드리 0** 수준으로 유지.
- T2 재료 제작형 구조를 깨지 않기 위해 2막 메인·서브 보상에 **T2 완제품 직드랍을 넣지 않는다**.
- 각인서는 **2막 결전(메인 20) 보스 직드랍**만 열고, 서브에는 넣지 않는다.
- 4테마 메인 보스 / 정예 설계는 **파티 3명 상한**을 전제로 기믹 분산(원거리 1 + 근거리 1 + 자유 1)을 가정.

## 범위
- 수도 메인 13·14·16·17·18·19·20 (7개).
- 수도 서브 C~H (6개).
- 4테마 각 메인 4~5개 + 서브 3~5개.
- 4막 떡밥 재관측 지점 테마별 1지점씩.

## 비범위
- 3막 세력별 재분기 본문은 다루지 않는다. 2막 종료 시점의 **입구 힌트**만 허용.
- 사하르 상인연합·아르카논 마법공학 도시의 3막 개방 본문은 다루지 않는다.
- 대사 최종 교정, 퀘스트 ID 최종 키, 보상 구체 수치는 추후 확정.
