# 포로 서버 1막 길드 게시판 반복 퀘스트 규격

## 문서 목적
`poro_master_planning.md` 1막 서브 `길드 게시판 의뢰`(3/5/10/20회 반복) 항목의 구체 확장. content-designer 1차 초안 기반.

## 배경
- 수도 길드 의뢰는 메인 6 진입 선행(3회), 분기 B 히든 진입(6회 + 정예·필드보스 3체), 메인 10 게이트(8회)와 연계
- 엘마르(행정 서기관 겸 길드 접수원) 접수 담당
- economy-reviewer 의뢰 반복 20회 보상 곡선 확정 (합계 강화석 44 / 파편 105 / 조각 565 / 골드 3,480)
- 영지 외 블록 설치·필드 채집 제거 정책 준수

## 해금·진입 시점
- **해금 = 메인 3 `정착의 권리`(영지 해금) 이후**
- 근거: 유형 F(영지 납품)가 있어 영지 해금 이전에는 유형 커버리지 빈약. 메인 6 진입 선행(의뢰 3회)과도 타이밍 정합
- 플로우: 메인 3 클리어 → 엘마르가 "길드 접수도 겸하게 됐다" 대사로 게시판 해금

## 운영 제한
| 항목 | 값 |
|---|---|
| 동시 수주 | 최대 **3종** |
| 일일 완료 상한 | **8회** (메인 10 게이트와 일치) |
| 주간 완료 상한 | **40회** |
| 고난도 일일 상한 | **3회** (보상 집중 방지) |
| 게시판 자동 새로고침 | **2시간 주기** |
| 강제 새로고침 | 골드 소량·길드 평판 1회 (일일 3회 제한) |
| 유형 F 고정 슬롯 | 상시 1개 보장 (영지 생활 파이프라인 보호) |

## 적응형 보상 곡선 (Lv3~35)
economy-reviewer 20회 누적 보상(강화석 44 / 파편 105 / 조각 565 / 골드 3,480)을 **평균 Lv 기준 총합**으로 고정, 구성 비율만 동적 재분배:
- **저레벨(Lv3~10)**: 조각·골드 비중 높고, 강화석은 5회차 이후 등장 유지
- **중레벨(Lv11~22)**: 파편 비중 증가, 강화석 등장 빈도 상향
- **고레벨(Lv23~35)**: 조각 포화 방지 위해 `조각 묶음(5개 단위)` 지급, 강화석 난수 상한 +1

## 반복 퀘스트 6유형 카드

### A. 사냥 의뢰
**한 줄 감각** — "외곽에 머리 수가 늘었다. 줄여라."

| 난이도 | 내용 | 1회 보상 | 플레이 시간 |
|---|---|---|---|
| 저 | 수도 외곽 약몹 8마리 | 조각 소량 + 골드 | 5분 |
| 중 | 지정 지역 정예 1체 + 몹 12마리 | 파편 + 조각 + 골드 | 8~10분 |
| 고 | 지정 지역 정예 2체 + 특수 드랍 확인 | 강화석 1 + 파편 + 골드 | 12~15분 |

정책 충돌: 없음.

### B. 배달 의뢰
**한 줄 감각** — "수도는 종이와 상자로 돌아간다."

| 난이도 | 내용 | 1회 보상 | 플레이 시간 |
|---|---|---|---|
| 저 | 수도 내부 NPC → NPC 1구간 | 골드 | 3~5분 |
| 중 | 수도 ↔ 외곽 초소 왕복 | 파편 + 골드 | 7~10분 |
| 고 | 다른 지역 관문(동·서·남·북 중 1) 경계까지 | 강화석 소량 + 파편 + 골드 | 12~15분 |

정책 충돌: 없음. 이동 콘텐츠 지역 순회 유도와 정합.

### C. 호위 의뢰
**한 줄 감각** — "길은 안전해 보일 때 가장 위험하다."

| 난이도 | 내용 | 1회 보상 | 플레이 시간 |
|---|---|---|---|
| 저 | 수도 내부 VIP 동선 호위 (습격 1회) | 조각 | 5~7분 |
| 중 | 외곽 상단 호위 (습격 2회) | 파편 + 조각 | 10~12분 |
| 고 | 관문 간 장거리 호위 (습격 3회 + 매복 정예 1체) | 강화석 + 파편 + 골드 가중 | 15~20분 |

정책 충돌: 없음.

### D. 수색·탐사 의뢰
**한 줄 감각** — "보이지 않는 것을 찾아 오라, 살아서."

| 난이도 | 내용 | 1회 보상 | 플레이 시간 |
|---|---|---|---|
| 저 | 수도 외곽 흔적 3곳 조사 | 조각 | 5분 |
| 중 | 지정 던전 입구 주변 단서 + 약한 몹 접촉 | 파편 + 조각 | 8~12분 |
| 고 | 필드보스 등장 구역 패턴 사전 조사 (미교전 시에도 완료 가능) | 강화석 + 파편 + 보스 조사 보너스 | 15~20분 |

정책 충돌: 없음.

### E. 청소·소탕 의뢰
**한 줄 감각** — "둥지가 커지기 전에 뿌리를 뽑아라."

| 난이도 | 내용 | 1회 보상 | 플레이 시간 |
|---|---|---|---|
| 저 | 외곽 소규모 몹 무리 1회 소탕 | 조각 + 골드 | 5~7분 |
| 중 | 소규모 던전 1층 클리어 | 파편 + 조각 + 골드 | 10~15분 |
| 고 | 소규모 던전 완주 + 정예 1체 추가 처치 | 강화석 1~2 + 파편 + 골드 | 18~22분 |

정책 충돌: 없음.

### F. 영지 채집·납품 의뢰 (영지 전용)
**한 줄 감각** — "영지 텃밭에서 나온 것만 길드도 믿는다."

| 난이도 | 내용 | 1회 보상 | 플레이 시간 |
|---|---|---|---|
| 저 | 영지 올인원 제작대 1차 결과물 납품 (저등급) | 골드 + 조각 | 3~5분 |
| 중 | 영지 제작대 중간 가공품 납품 | 파편 + 골드 | 8~12분 |
| 고 | 영지 제작대 완성급 생활 아이템 + 조리·가공 조합 납품 | 강화석 + 파편 + 생활 숙련 보너스 | 15~20분 |

정책 충돌: 없음. **"필드 채집 제거 → 영지 제작대 일원화"** 정책 강화.

### 유형 제외 검토
- **필드 채집**: 정책 충돌, 유형 F가 대체
- **보고(단순 말걸기)**: 배달 유형에 흡수

## 접수 UX

### 엘마르 행정 탭 연동
- 4카테고리 중 **행정 하위**에 `길드 게시판` 진입 노드
- 별도 UI 창 오픈, 다른 NPC 중복 접수 방지

### 수주 플로우
1. 게시판 클릭 → 유형 필터(전체/사냥/배달/호위/수색/청소/영지)
2. 의뢰 카드 클릭 → 요약 + 예상 시간 + 보상 미리보기
3. `수주` 확정 시 동시 수주 슬롯 체크 (3종 초과 차단)
4. 지도에 목표 위치 마커 자동 표기 (지도 탭 연동)
5. 진행 중 퀘스트 HUD에 슬롯 단위 표기

### 완료 보고 — 하이브리드
- **저·중 난이도** (사냥/배달/청소/영지 납품): **자동 인식** → 게시판 알림 → 월렛 자동 가산
- **고 난이도** (호위/수색/고난도 사냥): **엘마르 대화 보고** 필수 → 보스 조사 플레이버 전달 + 히든 떡밥 트리거 포인트

### 재화 지급
- 골드·강화석·파편·조각: **월렛 자동 가산**
- 생활 숙련 보너스: 영지 시스템 직접 누적, 알림 팝업만
- 귀속/교환 불가 재화는 자동 귀속 태그

## 1막 특화 서사 연출

### 유형별 플레이버 텍스트
| 유형 | 플레이버 |
|---|---|
| 사냥 | "외곽 초소에서 또 보고가 올라왔다. 이번엔 진짜라고 한다." |
| 배달 | "도장이 마르기 전에 도착해야 진짜 공문이다." |
| 호위 | "길드는 안전을 팔고, 우리는 그 값을 받는다." |
| 수색·탐사 | "못 찾으면 못 찾았다고 적어라. 그것도 기록이다." |
| 청소·소탕 | "둥지 하나가 마을 하나를 먹는 건 순식간이다." |
| 영지 납품 | "영지 도장이 찍힌 물건만 믿는 상인들이 늘었다." |

**톤**: 메인 11(수도 체제 균열 암시)과 맞춰 건조·사무적·영웅 서사 아닌 행정·일상 톤.

### 히든 라인 진입 떡밥 (누적 트리거)
- **누적 20회 달성 시점**에 길드 대기실 `기사단 관찰자` NPC 선제 대화:
  > "당신 이름이 게시판 위쪽에 자주 오르더군. 그건 실력 얘기만은 아니야. 누군가가 보고 있다는 뜻이지."
- 1회성, 이후 관찰자는 배경으로만 존재
- 분기 B 히든 진입 선행 플래그와 별개의 **서사 플래그** (이중 유도 구조)

### NPC 반응 대사 예시
- **엘마르** (누적 10회+): *"자네 이름은 이제 도장 없이도 통과시킬 수 있을 것 같군."*
- **카일** (외곽 경계병): *"게시판에 이름 올라간 놈들은 외곽에서 한 번쯤 마주치기 마련이지. 살아서 돌아와."*
- **기사단 관찰자** (20회 이전): 시선만, 대화 불가
- **영지 제작대 관리 NPC** (유형 F): *"길드에 넘어가는 물건이라도, 영지 도장 없인 하나도 못 나가네."*

## 남은 확인 포인트
- 일일 완료 상한 8회가 메인 10 게이트(의뢰 8회)와 정확히 맞물리는데, **게이트 돌파 당일 추가 수주 원하는 유저 동선** 막히지 않는지 플레이테스트 필요
- 유형 F 고난도 보상 `생활 숙련 보너스`가 기존 생활 숙련 곡선과 충돌 여부 (economy-reviewer 교차)
- 기사단 관찰자 선제 접촉(20회)이 분기 B 히든 진입 조건(6회 + 정예·필드보스 3체)보다 늦은 구조의 서사 자연성 재검토
