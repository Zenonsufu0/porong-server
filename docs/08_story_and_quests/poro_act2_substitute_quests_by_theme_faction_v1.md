## 개요
2026-04-19 "파견지 강제성 폐기 + 소급·대체 퀘스트 규칙 신설" 결정에 따른 후속 작업.

**"1막 해당 테마 전무(메인 7 타 테마 선택)"** 케이스를 담당하는 **대체 퀘스트 8종**의 1차 드래프트.

대체 퀘스트는 1막 원본 서사(메인 8·9·10 + 서브 2종)를 **1~2 씬으로 압축 재생**한 뒤, 2막 씬 카드 v1의 저강도 재관측 비트(E1·W1·S1·N1)로 자연스럽게 합류시키는 역할이다.

## 목표
- 1막에서 해당 테마를 전혀 방문하지 않고 2막 메인 16에서 해당 테마를 선택한 플레이어에게, 서사 공백과 보상 공백을 최소 비용으로 메운다.
- 세력 선택(기사단 / 그림자)에 따라 **임무 목표 자체가 달라지는** 분기를 유지한다.
- 1막 복선 아이템을 대체 퀘스트로만 획득한 유저도 3막 결전 회수·7종 컬렉션에서 소외되지 않도록 한다.
- 본편을 정상 진행한 유저 대비 **보상 70~80% 선 + 서사 압축**으로 형평성을 맞춘다.

## 핵심 규칙
- **트리거 공식 (공통)**
  - `quest.theme.<테마>.m9_completed == false`
  - AND `quest.theme.<테마>.m8_completed == false` (부분 진행자는 소급 트리거로 분리 처리)
  - AND `quest.act2_main.m16_theme_choice == <테마>`
  - AND `player.faction == <knights | shadow>` (세력 제안 수락 이후 진입 허용)
- **완료 플래그**: `quest.theme.<테마>.act2_substitute_<knights|shadow>_completed == true`
- **보상 중복 차단**: 1막 본편 메인 8·9·10 및 해당 서브를 하나라도 클리어한 이력이 있으면 본 대체 퀘스트 진입 자체가 차단된다(소급 트리거 쪽으로 분기).
- **보상 스케일**: 1막 메인 8·9·10 합산 보상의 70~80%. 명성 +11%/+3%/+2% 구조는 +8%/+2%/+1%로 치환. 골드·강화석 파편은 75% 감액. 훈장류(본체 판별/가짜 쉼터/과열 내성/냉기 저항)는 I단계 그대로 자동 지급(2막 보정 목적이므로 감액 불가).
- **복선 아이템**: `poro_act1_theme_foreshadowing_items_draft.md`에 정의된 테마별 1종을 대체 퀘스트 완료 시 **자동 1회성** 지급. 3막 결전 회수 대상이므로 지급 보장 필수.
- **T2 직드랍 금지 / 각인서 직드랍 금지** — 2막 메인 20만 허용.
- **파티 3명 상한** — 대체 퀘스트 씬 전투는 솔로 또는 3인 파티 기준으로 난이도 스케일링.

## 1. 동부 침식 숲

### 압축 요약 공통 (기사단·그림자 공통 3~5줄)
- 동부 침식 숲은 "길도 사람도 적도 진짜인지 알 수 없는" 분신·본체 판별의 숲이다.
- 1막 본편에서는 숲지기 브리핑 → 침식 수림 외곽 진입 → 분신 정예 전투 → 초입 필드보스 `썩은 뿌리 감시자` 조우 → 침식 시료 제출이 정규 동선이었다.
- 주요 복선 NPC는 **숲지기**, **약초꾼**, **실종된 순찰대장**이며, 유품 회수와 오염 고목 5그루 표식이 서브 2종의 핵심이었다.
- 4막 복선 아이템은 **`침묵한 정령의 눈물`** — 오래된 정령 사당 정화 서브 보상으로 심기 예정.

### 1A. 동부 × 기사단 — 씬: "뒤늦게 밟는 분신의 땅"
- **트리거**: `quest.theme.east.m8_completed == false` AND `quest.act2_main.m16_theme_choice == east` AND `player.faction == knights`
- **서사 비트**:
  - 동부 경계 비석 앞. 플레이어는 이제야 이 땅을 밟는다. 숲지기는 이미 수도로 송환된 상태고, 그를 대신해 **기사단 임시 파견관**이 초소를 지키고 있다.
  - 파견관이 두꺼운 공식 보고서철을 내민다. "당신이 비운 자리는 숫자로 남아 있습니다. 실종 순찰대 1개 조, 오염 고목 5그루, 침식 시료 미제출." 뒤늦은 조사는 **공식 누락 보고서 보완** 형식으로 이루어진다. 침식 수림 외곽에서 분신/본체 판별 2회, 초입 필드보스 `썩은 뿌리 감시자`의 **잔존 흔적** 조사(이미 정규 플레이어가 처치한 세션이면 저강도 변이체로 대체 스폰), 실종 순찰대장의 유품 회수를 순차 처리한다.
  - 마지막에 사당 터에서 `침묵한 정령의 눈물`이 떨리며 인벤 자동 지급. 파견관은 "정식 기록으로 남깁니다"라는 한 줄로 보고를 마감한다. 2막 E1 저강도 재관측(침식 수림 표층 첫 진입)으로 자동 인계.
- **선택지**: (a) 보고서에 "순찰대장 사인 미상"으로 기재 → 기사단 신뢰 +1% (b) "침식 감염 확증"으로 기재 → 길드 신뢰 +1% (상호배타)
- **보상**: 동부 명성 +8%, 길드 신뢰 +5%, 본체 판별 훈장 I 자동 지급, `침묵한 정령의 눈물` 1개, 침식 수액 중간 분량 75%, 골드 75%.
- **플래그 키**: `quest.theme.east.act2_substitute_knights_completed`

### 1B. 동부 × 그림자 — 씬: "기록되지 않은 숲"
- **트리거**: 상동, `player.faction == shadow`
- **서사 비트**:
  - 수도 빈민가 그림자 연락책이 동부행 표를 건넨다. "공식 실종자 명단에 이름이 없는 순찰병이 두 명 있다. 기사단은 알고도 지웠다." 그림자 임무는 **은밀 조사**다. 숲지기 초소가 아니라 **버려진 밀렵꾼 오두막**이 거점이다.
  - 밀렵꾼 가족이 건넨 쪽지를 따라 침식 수림 외곽을 횡단한다. 분신 본체 판별 2회 구간에서 플레이어는 **분신이 흉내 낸 것이 실종 순찰병의 얼굴**이라는 사실을 목격한다. 오염 고목 한 그루 뿌리에서 이름표가 지워진 인식표 두 개를 회수. 공식 보고에는 절대 오르지 않을 증거가 모인다.
  - 약초꾼(1막 본편 서브 2 NPC)이 오두막에 들러 중화제 시제품을 두고 간다. 떠나기 전 "정령 사당에 들러보라"는 한 마디만 남긴다. 사당 터에서 `침묵한 정령의 눈물`이 인벤 자동 지급. 2막 E1 저강도 재관측에서 **분신이 플레이어를 "숨기는 자"로 인식하는** 미세 변조가 걸리도록 플래그를 남긴다.
- **선택지**: (a) 인식표 두 개를 그림자에 제출 → 그림자 신뢰 +2% (b) 한 개만 제출, 한 개는 본인 보관 → 그림자 신뢰 +1%, 4막 떡밥 플래그 `east_shadow_kept_tag`
- **보상**: 동부 명성 +8%, 그림자 신뢰 +5%, 본체 판별 훈장 I 자동 지급, `침묵한 정령의 눈물` 1개, 왜곡된 포자 75%, 중화제 시제품 2개(본편 3개에서 감액).
- **플래그 키**: `quest.theme.east.act2_substitute_shadow_completed`

## 2. 서부 황야

### 압축 요약 공통
- 서부 황야는 "보이는 길이 진짜 길이 아닌" 가짜 쉼터·이동 압박의 지역이다.
- 1막 본편에서는 보급 감독관 브리핑 → 보급선 파괴지대 → 가짜 쉼터 판별 → 초입 필드보스 `황야 송곳니 군주` 조우 → 폭풍 파편 회수가 정규 동선.
- 주요 복선 NPC는 **보급 감독관**, **사하르 상단 잔존 NPC**, **실종 상단 대장**. 서브 2는 사하르 계약군주 복선을 1~2줄 남긴다.
- 4막 복선 아이템은 **`부서진 풍향계 조각`** — 심기 지점은 재설계 중이나 대체 퀘스트는 완성품 지급 보장.

### 2A. 서부 × 기사단 — 씬: "공식 지도의 빈칸"
- **트리거**: `quest.theme.west.m8_completed == false` AND `quest.act2_main.m16_theme_choice == west` AND `player.faction == knights`
- **서사 비트**:
  - 서부 경계 황야. 보급 감독관은 이미 수도로 귀환했고, 그가 남긴 **공식 지도에는 세 개의 빈칸**이 있다. 기사단 파견관이 "이 빈칸을 메우는 것이 임무"라 말한다. 공식 지도를 완성해야 2막 풍식 협곡 진입이 정당화된다는 논리다.
  - 플레이어는 세 빈칸 좌표를 돌며 가짜 쉼터 3곳을 판별·해체하고, 진짜 쉼터 1곳을 확인해 신호탑 좌표를 공식 지도에 기록한다. `황야 송곳니 군주`의 발자국 흔적을 지도에 추가 표기. 마지막 지점에서 **모래에 반쯤 묻힌 풍향계 조각** 하나를 회수한다.
  - 파견관은 "빈칸이 메워졌습니다. 이제 협곡으로 들어갈 자격이 생겼습니다"라고 공식 보고를 마감. 2막 W1 저강도 재관측(풍식 협곡 입구)으로 자동 인계된다.
- **선택지**: (a) 신호탑 좌표를 공식 지도에 정식 등재 → 기사단 신뢰 +1% (b) "신호탑 위치 유동적" 주석 처리 → 길드 신뢰 +1%
- **보상**: 서부 명성 +8%, 길드 신뢰 +5%, 가짜 쉼터 판별 훈장 I 자동 지급, `부서진 풍향계 조각` 1개, 폭풍 파편 75%, 골드 75%.
- **플래그 키**: `quest.theme.west.act2_substitute_knights_completed`

### 2B. 서부 × 그림자 — 씬: "상단이 삼킨 지도"
- **트리거**: 상동, `player.faction == shadow`
- **서사 비트**:
  - 그림자 연락책은 서부 황야의 공식 지도가 사하르 상단 쪽 정보와 **3곳이나 어긋난다**는 사실을 플레이어에게 흘린다. "기사단이 모르는 쉼터가 있다. 상단은 알고 있고, 말하지 않았다." 임무는 **상단이 숨긴 실종 상단의 마지막 짐**을 찾는 것이다.
  - 플레이어는 보급선 파괴지대를 우회 횡단하며 상단 흔적 3곳을 추적하고, 모래 속 상자에서 상단 일지를 해독한다. 일지에는 **"이름이 지워진 계약 인장"**(사하르 계약군주 복선)이 한 줄 언급되며, 진짜 쉼터 1곳의 위치가 공식 지도와 다른 좌표로 적혀 있다. 풍향계 조각은 그 좌표 근처 모래 밑에서 회수된다.
  - 그림자 연락책은 "기사단 보고서에는 올리지 말라"고 당부한다. 2막 W1 저강도 재관측에서 플레이어는 풍식 협곡을 **기사단 루트와 다른 진입점**으로 들어가게 된다.
- **선택지**: (a) 상단 일지를 그림자에 전량 제출 → 그림자 신뢰 +2% (b) 계약 인장 관련 쪽만 본인 보관 → 그림자 신뢰 +1%, 사하르 신뢰 +1%
- **보상**: 서부 명성 +8%, 그림자 신뢰 +5%, 가짜 쉼터 판별 훈장 I 자동 지급, `부서진 풍향계 조각` 1개, 마른 혈석 75%, 사하르 상단 신뢰 +2%(본편 동일 — 사하르 루트 보존 목적).
- **플래그 키**: `quest.theme.west.act2_substitute_shadow_completed`

## 3. 남부 화맥

### 압축 요약 공통
- 남부 화맥은 "통제되던 화맥이 스스로 깨어나는" 과열·분화·심핵 반응의 지역이다.
- 1막 본편에서는 채굴 감독관 브리핑 → 균열 용암지대 → 과열 바닥/분화선 학습 → 초입 필드보스 `균열 화구 감시자` → 심핵 진동 샘플이 정규 동선.
- 서브 2의 핵심은 **채굴 감독관의 공책** — 태움(길드) / 수도 송부(기사단) 선택지가 있었다.
- 4막 복선 아이템은 **`식지 않는 재 한 줌`** — 남부만 **메인 엔딩 이벤트 지급 예외**이므로 대체 퀘스트도 메인 씬에서 지급 타이밍 유지 필수.

### 3A. 남부 × 기사단 — 씬: "뒤늦은 화맥 보고서"
- **트리거**: `quest.theme.south.m8_completed == false` AND `quest.act2_main.m16_theme_choice == south` AND `player.faction == knights`
- **서사 비트**:

## 4. 북부 설혼

### 압축 요약 공통
- 북부 설혼은 "사람과 길, 기억까지 얼려버리는 백색 장막"의 지역이다.
- 1막 본편에서는 경계 대장 브리핑 → 냉기 확산 지대 → 설혼 안개 체험 → 초입 필드보스 `서리발톱 군주` → 백빙 낙인 유품 회수가 정규 동선.
- 서브 2종의 핵심은 **부상병 3명의 이름 복원**과 **뒤틀린 제단 3개 봉쇄**. 부상병 중 1명은 복원 실패(백빙 낙인 고착)로 남는다.
- 4막 복선 아이템은 **`얼어붙은 숨 한 조각`** — 얼어붙은 사당 조사 서브 보상으로 심기 예정.

### 4A. 북부 × 기사단 — 씬: "복원되지 못한 이름"
- **트리거**: `quest.theme.north.m8_completed == false` AND `quest.act2_main.m16_theme_choice == north` AND `player.faction == knights`

  - 설원 경계 초소. 경계 대장은 이미 교대 근무로 수도 복귀했고, 기사단 파견 **장부 검시관**이 대장 자리를 대신한다. 검시관은 플레이어에게 낡은 장부 세 권을 내민다. "세 명의 이름이 빈칸입니다. 한 명은 이미 늦었고, 두 명은 아직 살릴 수 있을지도 모릅니다."
  - 냉기 확산 지대에서 설혼 안개 1회 체험, 냉기 굴절충 1체 처치, `서리발톱 군주`의 발톱 자국 조사를 압축 수행한다. 초소로 돌아와 부상병 3명과 대화. 장부 기록 2건은 공식 복원 성공, 1명은 **복원 실패**(백빙 낙인 고착, 본편 서브 1 스펙 그대로 유지). 검시관은 실패한 병사의 이름을 공식 장부에 "미확인"으로 남기자고 제안한다.
  - 얼어붙은 사당 터에서 `얼어붙은 숨 한 조각`이 떨리며 인벤 자동 지급. 2막 N1 저강도 재관측(봉인석 평원 초입)으로 인계되며, 검시관은 "우리가 봉인한 게 아니라, 우리가 봉인된 건지 모른다"는 본편 대장 대사를 그대로 읽는다.

## 구현 포인트
- **씬 재사용 자산**: 1막 본편 메인 8·9·10에서 사용한 분신 판별·가짜 쉼터·분화선 회피·설혼 안개 기믹을 **스케일 다운** 버전(체감 시간 1/3, 파밍량 1/3)으로 재사용한다. 신규 기믹 제작 없이 파라미터만 조정.
- **NPC 재사용 vs 신규**: 1막 본편의 숲지기·보급 감독관·채굴 감독관·경계 대장은 모두 "수도로 귀환"으로 처리하고, 기사단 파견관·그림자 연락책을 **테마당 1명씩 신규 배치**한다. 대체 퀘스트 전용 NPC로 모델 2체(기사단 파견관 공용 스킨 + 그림자 연락책 공용 스킨) 설계 권고.
- **복선 아이템 지급 보장**: 본편에서는 서브/메인/드랍 등 심기 방식이 테마별로 다르지만, 대체 퀘스트는 **완료 시점 인벤 자동 지급 단일 방식**으로 통일한다. 3막 결전 회수 트리거는 동일.
- **훈장 일원화**: 본체 판별/가짜 쉼터/과열 내성/냉기 저항 훈장 I은 본편과 동일 스펙 그대로 지급. 2막 표층 보정 목적이므로 감액 불가 원칙 유지.
- **플래그 독립**: `act2_substitute_*_completed` 플래그는 소급 플래그(`act2_substitute_retro_*_completed`, 별도 문서)와 완전 독립. 두 플래그가 동시에 true가 되는 경로는 원천 차단한다.
- **2막 메인 16 진입 게이트**: 대체 퀘스트 미완료 상태에서는 2막 메인 16 선택 확정 직후 강제 삽입되며, 완료 전까지 메인 17로 넘어갈 수 없다. 이탈 방지 설계.
