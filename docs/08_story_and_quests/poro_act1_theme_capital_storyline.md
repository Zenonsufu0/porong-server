# 1막 테마 스토리라인 — 수도 황성

## 개요
수도 황성은 1막의 주 무대이자 동/서/남/북 4개 테마로 분기되는 허브다.
이 문서는 수도 테마에 속한 **공용 허브 메인 1개**와 **서브 스토리라인 2개**를 1차 초안으로 정리한다.
기존 `poro_master_planning.md`의 1막 메인 1~11 흐름을 존중하며, 여기서는 "수도 테마" 관점에서 묶음 단위로 재정리한다.

> 1막 파견 테마 선택지는 **동/서/남/북 4택**이다. 사하르 상인연합과 아르카논 마법공학 도시는 **3막 시점에 개방**되며 1막 선택 대상이 아니다.
>
> 수도에는 파견 열차를 타지 않고 수도 안에 남아 1막을 완주하는 **히든 메인 라인**이 별도로 존재한다. 이 히든 라인의 테마는 **황립기사단 수습 스카우트**다. 수도 길드에서 의뢰를 성실히 쌓은 플레이어를 길드 대기실에 상주하던 기사단 관찰자가 먼저 수습 제의로 말을 걸어온다. 상세는 `act1_capital_main/poro_act1_capital_hidden_mainline.md`를 따른다. 본 문서의 "수도 적응과 파견 자격"은 그 히든 라인의 **선행 공용 허브**로 작동한다.
>
> 내부 수사·파견 보고 누락·명단 대조·회색 기록 추적 같은 "보이지 않는 정보" 컨셉은 **서브 스토리라인 2 `빈민가의 그림자`(빈민가 정보기관 그림자)** 가 이어받는다. 이 서브만 2막 **그림자 협력 제안**의 가중치를 만든다.
>
> 모든 퀘스트의 상태·대사·보상·해금 판정은 **EmpireRPG(Custom Plugin) 단독**이다. 대사·선택지·연출은 **EmpireRPG 커스텀 대화 UI**가 렌더한다. **BetonQuest는 사용하지 않는다.** Citizens는 NPC 배치·외형·클릭 진입점만, MythicMobs는 몹/보스 외형·스폰·스킬 연출만 담당한다.

## 목표
- 수도에서의 1막 메인 흐름을 테마 스토리 단위 1개로 압축한다.
- 수도 허브에서 진행 가능한 대표 서브 스토리 2개를 정의한다.
- 2막 `긴급 호출` 직전까지의 수도 연결선을 명확히 한다.

---

> **세부 씬 카드**: 메인 1~7 + 합류 메인 11의 대사 톤 샘플 · 선택지 · 4막 복선 앵커(메인 2 재가루·메인 5 지도 한 점·메인 11 여섯 번 종) · 플래그 · 커스텀 UI 트리거 키는 `act1_capital_main/poro_act1_main_1_7_11_scene_cards.md` 로 분리 저장.

## 공용 허브 메인: 수도 적응과 파견 자격
### 한 줄 감각
**"낯선 수도에 떨어진 조사자는 제국의 조사 협력자 자격을 얻고, 첫 파견 열차에 오르기 전까지 자신을 증명한다."**

### 진행 단계
1. 수도 중앙 광장 도착, 레오니드와의 첫 대화
2. 수도 외곽 이상 징후 조사 (일반/강화/정예 및 외곽 필드보스)
3. 영지 해금, 첫 채집기 설치, 올인원 제작대 설치
4. 비석 네트워크 개방 + **월드맵 단축키 해금** (기본 키 `M`, 재바인딩 가능. 이후 유저가 원할 때 상시 오픈)
5. T1 8~10강 도달 + 길드 의뢰 3회
6. 테마 선택 보드 진입 — **동/서/남/북 4택**
7. 파견 열차 탑승 준비 — 정거장 문지기 하급 기사 대사 청취 (히든 메인 라인의 진입 떡밥)

### 완료조건
- 전투 레벨 35+
- T1 평균 8~10강
- 수도 명성 15% 이상
- 길드 신뢰 5% 이상
- 선택한 테마 초입 비석 활성화

### 보상
- 수도 명성 누적 +15% 이상
- 길드 신뢰 누적 +5~7%
- 기사단 신뢰 +1%
- T1 강화석/파편 누적, 기본 영지 도구, 월드맵 UI

### 연결
- 이 공용 허브 메인이 끝나면 두 갈래로 분기한다:
  - **기본 동선**: `메인 8. 파견 열차` → 선택한 동/서/남/북 테마 초입 스토리라인
  - **히든 동선(황립기사단 수습 스카우트)**: 수도 길드 의뢰 누적 → 길드 대기실 상주 관찰자(하급 기사) 대화 활성 → 히든 진입 퀘스트 `기사단 수습 시험` → 수도 1막 히든 메인 라인 (`act1_capital_main/poro_act1_capital_hidden_mainline.md`)
- 두 동선 모두 종료 후 `메인 11. ???의 편지`로 관찰자 공간에 연결된다.
- 히든 동선 진입을 놓친 플레이어는 `???의 편지` 수도 복귀 시점에 관찰자 NPC가 먼저 말을 걸어 소급 개방된다.
- 히든 동선은 2막 **황립기사단 수습 제안** 가중에, 서브 2 `빈민가의 그림자` 는 2막 **그림자 협력 제안** 가중에 각각 기여한다. 두 루트는 1막에서 병행 진행 가능.

## 서브 스토리라인 1: 수도 숨은 장소
### 한 줄 감각
**"광장의 뒤편, 아무도 안내하지 않는 장소들이 제국의 진짜 얼굴을 말해준다."**

### 진행 단계
1. 시장 소문 NPC와의 잡담으로 힌트 3개 획득
2. 광장 후면 · 하층 주거구 · 황성 외곽 경계구역 중 히든 포인트 2개 발견
3. 각 포인트에서 간단한 상호작용 (비석 파편, 낡은 현판 등)
4. 접수원 재보고

### 완료조건
- 히든 포인트 3개 중 2개 이상 발견
- 소문 NPC 4명 전원 대화

### 보상
- 수도 명성 +3%
- 히든 발견도 +10%
- 수도 탐험 기록 UI 항목 해금

### 구현 포인트
- 히든 포인트 상호작용은 EmpireRPG 커스텀 대화 UI의 **읽기 전용 트리거(또는 `CustomInteractable`)** 로 처리. BetonQuest 미사용.

## 서브 스토리라인 2: 빈민가의 그림자  *(1막 그림자 루트 단독 트랙 — 빈민가 정보기관 그림자)*
### 한 줄 감각
**"공식 기록이 말하지 않는 것을, 뒷골목 장부가 대신 적는다."**

### 위치와 역할
- 히든 메인 라인이 **황립기사단 수습 스카우트(성실한 공식 루트)** 를 담당하는 동안, 본 서브는 **빈민가 정보기관으로서의 그림자(비공식 정보망)** 를 단독 책임한다.
- 내부 수사·파견 보고 누락·명단 대조·회색 기록 추적 등 "보이지 않는 정보" 컨셉은 히든 메인에서 이관되어 본 서브가 이어받는다.
- 2막 세력 제안 단계에서 **그림자 협력 제안**의 가중치는 본 서브 완주 여부·달성도로만 결정된다.
- 판정·대사·상태 전이 모두 EmpireRPG(Custom Plugin) 단독. 대사·선택지·읽기 전용 GUI는 **커스텀 대화 UI**가 렌더. **BetonQuest는 사용하지 않는다.** Citizens는 NPC 배치만, MythicMobs는 골목 정예 연출만 담당.

### 진행 단계 (인물 분담 매핑 확정 — 권장 레벨 Lv.14~18)
1. 광장에서 하급 기사 **로디크**의 뜬 대화 청취 (자동 트리거, 힌트 메모 획득) — *인물: 로디크*
2. 빈민가 뒷골목(필드 4) 히든 1개 발견 후 암시장 입구에서 **회중** 첫 대면 (메모 제시 분기) — *인물: 회중*
3. 길드 행정 탭에서 **엘마르의 히든 라인** 특수 대화로 "보고하지 않는 편이 나은 이야기" 확인 — *인물: 엘마르 (행정 탭 조건부 분기)*
4. 암시장 주점에서 **회중** 재접선 (회색 대화: 시험 의뢰 수락 시 `회중의 표식` 토큰 지급) — *인물: 회중*
5. 빈민 보호 미니 임무: 골목 부상자 2명 응급 처치 + 정예 `골목 칼잡이 그란` 처치, 회중에게 자동 보고 — *전투, 완료 시 주점 뒷방 좌표 해금*
6. **(내부 수사 이관)** 주점 뒷방에서 **잿빛** 첫 대면. "공식 파견 기록 vs 실제 명단" 커스텀 대화 UI 읽기 전용 모드로 3건 열람, 불일치 2건 식별 — *인물: 잿빛 첫 등장, 회중 배석*
7. **(내부 수사 이관)** 하층 주거구 현장 확인 — 일반 몬스터 15체 처치 중 `회색 장부 단서` 1개 회수 (봉인된 제국 행정부 단서 톤). 잿빛의 귓속말형 보이스오버 오버레이 1회 — *인물: 잿빛 보이스오버*
8. 주점 뒷방 재접선, **잿빛**이 2막 협력 떡밥 발화 + 3지선다 선택지(`act2_invite` 숨김 플래그) — *인물: 회중·잿빛, 선택 결과는 UI 상 명시 없음*

> 단계별 씬·대사 샘플·커스텀 UI 트리거 맵 초안(트리거 키 8종: `dlg.shadow.eavesdrop_roddick` / `dlg.shadow.huijung_first` / `dlg.shadow.elmar_hidden` / `dlg.shadow.huijung_second` / `dlg.shadow.alley_victim_A·B` / `dlg.shadow.elite_gran_intro·death` / `dlg.shadow.jaetbit_first` / `dlg.shadow.field_pickup` / `dlg.shadow.final_backroom`)은 content-designer 8단계 카드 초안으로 확보 상태. 본 문서 외 별도 `sub2_shadow_track_scene_cards.md` 분리 여부는 후속 결정.

### 완료조건 (EmpireRPG 판정)
- 하급 기사 특수 대화 완료
- 빈민가 히든 포인트 1개 이상 발견
- 접수원 특수 대화 완료
- 암시장 안내 NPC 접선 완료
- 빈민 보호 미니 임무 완료 (부상자 2명 전달 + 정예 1체)
- 파견 기록 대조 3건 열람, 불일치 2건 식별
- 하층 주거구 단서 아이템 1개 회수
- 암시장 재접선 보고 완료

### 보상
- 그림자 신뢰 +5~7% (내부 수사 컨셉 이관 반영 상향)
- 수도 명성 +3%
- 암시장 UI 첫 노출 (구매 불가, 프리뷰 전용)
- 소량 골드, 강화석 조각
- EmpireRPG 플래그 `faction_shadow_track=true` (2막 그림자 제안 게이트)
- "회색 장부 단서" 아이템 1개 (2막 그림자 루트 심화 진입 가중)

### 연결
- 본 서브는 **1막 그림자 루트의 유일한 진입 경로**이자 **1막 내부 수사 콘텐츠의 유일한 표현지**다. 히든 메인(기사단 스카우트)과는 **1막 시점엔 플래그 독립** — 병행 진행 가능하며 서로 감점하지 않는다.
- 2막 세력 제안 단계에서 본 서브 완주 플레이어는 **그림자 협력 제안** 확률이 올라간다. 단 **기사단 ↔ 그림자 세력 상호 배타 확정**(`poro_master_planning.md` 세력 구조 참조)에 따라, 본 서브 완주만으로는 "그림자 접촉 경험"일 뿐 2막 기사단 공고문에 **조건부 지원** 가능(단계 2 "의심 평판" 대화 + 단계 5 규율 문답 감점 리스크). 2막 그림자 협력을 **실제로 수락**하는 순간 기사단 루트 잠금.
- 1막 시점에는 암시장을 "보는 것만" 가능하게 두어 세력 미스매치를 방지한다.
- 회색 장부 단서는 2막 수도 심층(폐쇄된 왕도 하수 심연) 진입 대사에서 그림자 루트 전용 대화로 재등장한다.
- **세부 씬·대사·플래그·커스텀 UI 트리거 맵**은 `act1_capital_main/sub2_shadow_track_scene_cards.md` 로 분리 저장되어 있다 (회중 2·4·8, 엘마르 히든 라인 3, 잿빛 6·7·8, 트리거 키 8종 포함).

## 구현 포인트
- `수도 적응과 파견 자격` 공용 허브 메인은 EmpireRPG에서 6~7개 서브이벤트 퀘스트로 분리해 중간 저장 가능하게 한다. 각 서브이벤트 대사·선택지는 **커스텀 대화 UI**가 렌더한다.
- 서브 2개는 모두 공용 허브 메인과 병렬 진행 가능해야 한다.
- 수도 명성/길드 신뢰/기사단 신뢰/그림자 신뢰는 공통 명성 시스템 키(EmpireRPG `faction_trust_*`)로 관리한다.
- 히든 메인(기사단 스카우트)과 서브 2(그림자 정보기관)의 플래그는 서로 감점 없이 독립. 1막 병행 진행을 허용한다.
- 모든 퀘스트 상태·대사·보상·해금 판정은 EmpireRPG(Custom Plugin) 단독. **BetonQuest는 본 1막 수도 콘텐츠에서 사용하지 않는다.**
- Citizens는 NPC 배치·외형·클릭 진입점만, MythicMobs는 몹/보스 외형·스폰·스킬 연출만 담당한다.
- **보조 NPC 통합 확정**: 영지 관리관 / 생활 안내관 / 지도·비석 안내관 / 강화 담당 장인 / 열차 정거장 안내관 5기능은 기존 수도 5인(레오니드·엘마르·베르딕트·세리안·로웬)에 기능 통합하고 신규 생성하지 않는다. 엘마르는 "행정/영지/생활/지도" 4탭 대화 구조(순차 해금), 로웬은 "보급+강화" 2축. 세부 매핑은 `poro_master_planning.md` 1막 핵심 NPC 블록 참조.
- **암시장 접선 NPC와 그림자 정보 수집책은 별도 인물**: 접선책 코드네임 **회중(懷中)** 은 서브 2 단계 2·3·4·8(입장 판정·첫 의뢰·마무리 거래), 수집책 코드네임 **잿빛(灰)** (전직 제국 행정관 설정)은 단계 6·7·8(파견 기록 대조·하층 주거구 단서·2막 협력 떡밥 발화)에 등장. 권한은 분리되어 있고 2막 그림자 협력 제안은 잿빛이 이어받는다.
- **T1↔T2 보정 교환(강화석 5개 = 상위 조각 1개)은 1막 비활성**. 2막 T2 진입 이후 활성화하며, 진입 직후 한시 우대(예: 첫 주 4:1) 이벤트로 1막 잉여 T1 강화석의 자연 소비 유도.
- **`???의 편지` 활성 조건**: 전투 레벨 50 + **전 장비 T1 18강**. 기존 "주력 15강+, 평균 12~15강+" 기준에서 상향됨. 경제 검토 결과 기존 경로만으로는 부족분 강화석 260~290개가 발생하므로 메인 10 완료 시 강화석 80~100개+파편 300개 1회성 지급, 의뢰 반복 보상 곡선 완만 상향 등 튜닝을 선행해야 자율 파밍 버짓(캐주얼 10~18h / 코어 6~10h) 내 흡수 가능.
