# 1막 공용 허브 메인 1~7 + 합류 메인 11 씬 카드

## 문서 목적
`poro_master_planning.md`의 1막 공용 허브 메인 상세와 `poro_act1_theme_capital_storyline.md`의 공용 허브 메인을 보강하는 **세부 씬·대사 톤 샘플·플래그·보상·트리거 키 카드**를 모아둔다. content-designer 1차 초안 기반이며, 대사는 최종판이 아니라 톤 샘플이다.

분기 A(메인 8~10 파견 열차)·분기 B(수도 히든 메인 8~15)·서브 스토리는 **본 문서 범위 외**.

> 판정·대사·읽기 전용 GUI 렌더는 **EmpireRPG 커스텀 대화 UI 단독**. BetonQuest 미사용. Citizens는 NPC 배치·외형·클릭 진입점, MythicMobs는 정예·필드보스 스폰/연출만.

## 문서 범위
- 메인 1. 낯선 수도 (Lv 1~3)

- 메인 11. ???의 편지 (Lv 50, 합류 메인)

## 감정 계단 개요
- 메인 1 도착 → 메인 2 첫 현장·이변 각인 → 메인 3·4 정착·자급 → 메인 5 세계 확장 → 메인 6 자기증명 → 메인 7 방향 선택 → 메인 11 복선 회수·각인 개방
- **4막 결전 `황성 심핵의 재앙체` 복선은 메인 2·5 2지점에만 저강도**로 심고, 메인 11 관찰자 공간에서 "같은 이야기의 세 페이지"로 회수.
- **4막 앵커 연출 — 월드 사운드 대체 매체로 전환 (확정)**: 기존 "광장 종 3번/6번 울림" 연출은 **월드 사운드 전체 제거 결정**에 따라 사운드로는 구현하지 않음. 대체 매체로 **채팅 서사 박스 / UI 화면 효과 / 파티클 / 시각 이펙트**(예: 광장 상공 파문 링 3회·6회 순차 확장) 중 택. 구체 형태는 content-designer + `13_external_plugins_and_custom_ui` 후속 설계.

## 메인 2. 조사자의 자격 (Lv 3~15)

**한 줄 감각** — "증표를 '자격'으로 바꾸는 첫 현장 — 수도의 외피가 밖에서는 벌써 찢어져 있다."

**스토리 비트**
1. 레오니드 브리핑: 외곽 순찰로에서 실종·짐승화 보고가 누적되고 있음. 외곽 경계병에게 신고 후 조사 협력.
2. 외곽 경계병 **카일**이 일반/강화 몬스터 처리 구역을 지정. 정예 한 마리의 행동이 '평범한 짐승과 다르다'는 보고.
3. 정예 처치 중 **희미하게 보이지 않는 맥동**이 한 번 — 전투 후 시체가 재처럼 얇게 균열을 그리며 가라앉는다 (플레이어 로그에만 짧게).
4. 외곽 필드보스 조우 및 처치 — 보스 사체에서 정제되지 않은 금속 파편 회수.
5. 로웬에게 파편 제출 → T1 강화석 조각으로 환산해 지급. 레오니드에게 귀환 보고.

**주요 씬**
- *씬 1. 외곽 브리핑* — 북문 초소 / 카일: "지난 보름 동안 여섯이 돌아오지 못했습니다. 살아서든 죽어서든." / "당신 증표로 이 구역에 들어갈 권리는 생깁니다. 돌아올 권리는 당신이 만드십시오."
- *씬 2. 정예 처치 이후* — 외곽 황무지 경계선 / 카일(원거리 보고): "…사체가 식기 전에 가루가 되는 건 처음 봅니다. 혹시, 당신도 보았습니까?" / "보고서에는 쓰지 않겠습니다. 아직은."
- *씬 3. 로웬 보급* — 수도 병참 보급소 / 로웬: "이 파편은 제련할 가치가 있군. 당신의 첫 강화석이 될 거요." / "강화는 기술이 아니라 습관이오. 작게, 꾸준히, 겁먹지 않고."

**선택지**
- 정예 처치 이후 카일에게 "사체 변화"를 **보고 / 묵인**
  - 보고 → 카일 개인 신뢰 +α, 3막 수도 수사 서브에서 카일이 먼저 말을 걺
  - 묵인 → 즉시 이득 없음. 관찰자 공간(메인 11) 회상 연출에서 해당 순간이 한 번 재조명(서사 미세 영향, 보상 차이 없음)

**완료 조건**
- `quest.act1_main.m2_briefed`
- `quest.act1_main.m2_fieldboss_defeated`
- `quest.act1_main.m2_shard_turned_in`
- `quest.act1_main.m2_cleared`
- (선택) `quest.act1_main.m2_kail_report_true` 또는 `..m2_kail_report_silent`

**보상**
- 수도 명성 +3%
- 기사단 신뢰 +1% (카일 보고 루트 시 +0.5% 추가)
- 길드 신뢰 +1%
- T1 강화석 조각 다수, 첫 강화석 1개, T1 무기 계열 기본 재료
- 조사자 정식 증표 (임시 증표 교체)

**4막 떡밥** — **저강도 1회 심기**. 정예 사체가 재처럼 가라앉는 연출. 4막 `황성 심핵의 재앙체`의 "소모되며 확장되는 피(被)물질" 성질의 가장 이른 싹. 이 시점에는 단순한 이변처럼 처리, 로그에도 해석 없음.

**커스텀 UI 트리거 키**

1. 엘마르 4탭 중 '지도' 탭 개방. 비석 네트워크 개념 설명.
2. 수도 권역 비석 3개 활성화 경로 안내 — 가벼운 도보 순회.
