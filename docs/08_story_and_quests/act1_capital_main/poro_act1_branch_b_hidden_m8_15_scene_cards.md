## 설계 원칙
분기 B는 "성실한 모험가의 기록이 황립기사단이라는 공식 기관의 눈에 닿는" **조용한 인정의 서사**. 분기 A(혈통·의뢰·파티 전투)와 감정 대비 — 분기 B는 **관찰 → 3색 심사(현장·서류·협동) → 면접 → 단독 증명 → 서임 → 합류**의 기사단식 수속 루트.

시즌1 전 구간 "관찰자" 호칭만 사용. 이름 없음.

환경 사운드 없음, 획득·UI 사운드는 유지. 광장 앵커 연출은 파문 링 시각 이펙트로 대체.

**스토리 비트**

## 메인 10. 서류 심사 — 파견 기록 해석

**한 줄 감각** — 검보다 종이가 더 무겁게 느껴지는, 판결 기록관의 조용한 방.

**스토리 비트**
1. 수도 판결기록관 집무실 입장. 베르딕트가 장부 3권 지정
2. 커스텀 대화 UI **읽기 전용 모드** 장부 열람 — 파견 기록·피해 보고·물자 이동표
3. 열람 종료 후 4지선다 1문항 제출 — "파견 기록에서 가장 먼저 의심해야 할 항목"
4. 정답 여부로 베르딕트 촌평 분기. 오답도 진행 가능 (신뢰 소폭 차이)

**주요 씬 — 판결기록관 집무실**
- 베르딕트: "기사단은 검만으로 서지 않아. 기록을 읽지 못하는 기사는 같은 실수를 두 번 한다."
- 베르딕트: "장부 셋일세. 급하게 볼 것은 없어. 모를 땐 모른다고 답하는 것도 심사의 일부야."
- 베르딕트(정답): "좋아. 이건 내가 자네 앞으로 한 줄 적어두겠네."
- 베르딕트(오답): "이번엔 아닐세. 하지만 읽으려 한 흔적은 남았어."

**선택지** — 4지선다 1문항 (정답 후보: "보고 일자 누락")
- 실제 문항은 서류 본문 확정 후 재검토 필요

### 서류 심사 UI UX 초안
- 상단: 장부 제목 + "판결기록관 인장" 장식
- 본문: 4~6문단 텍스트, 2~3장 페이지 전환식 (스크롤 없음)
- 하단 버튼: `[다음 장]` / `[이전 장]` / `[장부를 덮는다]`
- 열람 흐름: 베르딕트 대화 → 장부 목록(3권 완료 아이콘) → 선택 시 읽기 전용 진입 → 3권 모두 완료 후 `[베르딕트에게 답한다]` 활성 → 4지선다

- **오답 시 재도전 불가** (한 번만 제출, 운영 일관성)

## 메인 11. 협동 심사 — 빈 거리의 정예

**한 줄 감각** — 혼자 해오던 방식이 아닌, 기사와 발을 맞추는 낯선 리듬.

**스토리 비트**
1. 수도 빈민 거리 저녁. 협동 담당 시험관 B와 2인 파티
2. 좁은 골목 3구간 정예 2체 + 잡몹 다수. 시험관 참여하되 지휘 없음
3. 시험관 부상 위기 스크립트 — 플레이어 보조 판단 관찰
4. 종료 후 시험관이 처음으로 "동료성" 관점 구두 공유

**주요 씬 — 빈민 거리 막다른 골목**

- 시험관 B(종료): "기사단은 혼자 걷지 않아. 그걸 아는 자가 수습에 어울리네."

**선택지**

## 메인 12. 직속 기사의 면접

**한 줄 감각** — 세 개의 심사가 한 장의 서류로 모이고, 자신의 "색깔"을 처음으로 선언하는 자리.

**스토리 비트**
1. 기사단 본관 면접실. 상급 기사 + 시험관 3인 + 베르딕트(배석). 촌평서·심사 보고 3종 탁자 위
2. 상급 기사가 결과 건조 낭독 — 플레이어 선택 플래그에 따라 한 줄씩 다른 문장 삽입
3. `knight_specialty` 3택 질문
4. 선택에 따라 대응 시험관이 짧게 환영 한 마디 — **2막 재등장 예정**

**주요 씬 — 본관 면접실**
- 상급 기사: "세 심사가 모두 끝났네. 기사단은 세 가지 길 중 하나를 자네에게 맡기려 해."
- 상급 기사: "순찰. 호위. 특수조사. 한 가지를 고르게. 나중이 아니라 지금일세."

**선택지 — specialty 3택 (확정)**

**`knight_specialty = patrol` (순찰)**
- 선택 대사: `"변경을 돌겠습니다. 사람 수보다 길의 수가 많은 곳을요."`
- 시험관 A 응답: "길을 아는 자가 길을 지키네. 변경에서 다시 보세."
- **2막 톤**: 현장 실무형 — 지역 정보·지형·소수 교전 중심. 대사 짧고 담백.

**`knight_specialty = escort` (호위)**
- 선택 대사: `"사람을 지키겠습니다. 그 사람이 누구든."`
- 시험관 B 응답: "옆에 설 줄 아는 자에게 맡겨야 할 임무가 있네. 곧 부르지."
- **2막 톤**: 동료성·파티 리딩형 — 동행 NPC 보호·외교 수행. 대사 가장 따뜻·길다.

**`knight_specialty = investigation` (특수조사)**
- 선택 대사: `"읽어야 할 기록이 많다면, 제가 읽겠습니다."`
- 시험관 C 응답: "읽는 자가 부족하지. 베르딕트 공과 자주 얼굴을 맞대게 될 걸세."
- **2막 톤**: 분석·추적형 — 장부·단서 해석·잠입 외 수사. **서브 2 그림자와 구분: 공식 수사**. 대사 건조·질문형.

**완료 조건**
- `quest.act1_hidden.m12_interview_completed`
- `flag.knight_specialty` ∈ {`patrol`, `escort`, `investigation`}

**보상**
- 기사단 신뢰 +10
- `황립기사단 수습 징표` (미서임 상태, 메인 14에서 정식 교체)
- specialty별 소형 시작 키트 (순찰=지도 조각 / 호위=호신 완장 / 조사=기록 펜)

**UI 트리거 키**
- `dlg.act1.capital.hidden.m12.interview_open`
- `dlg.act1.capital.hidden.m12.specialty_prompt`
- `dlg.act1.capital.hidden.m12.specialty_patrol` / `escort` / `investigation`

## 메인 13. 실전 심사 — 외곽 필드보스 `황성 외곽 잠식 기사`

**한 줄 감각** — 아무도 옆에 없다. 처음으로, 기사단이 아닌 자기 자신에게 증명하는 싸움.

**스토리 비트**
1. 황성 외곽 폐허 지대 진입. 시험관 3인은 결계 바깥 대기, 플레이어 단독
2. 필드보스 `황성 외곽 잠식 기사` **1:1 조우** (분기 A 파티 전투와 대조)
3. 격파 후 잔해에서 이질적 파편 1개 회수 — 대사 강조 없이 획득 로그만
4. 결계 바깥 복귀, 시험관 짧은 고갯짓 후 본관 집결 지시

**주요 씬 — 외곽 폐허 결계 앞**
- 시험관 A: "여기서부터는 우리가 들어가지 못하네. 기사단 규약이야."
- 시험관 B: "돌아오게. 그게 심사 전부일세."
- 시험관 C(복귀 후, 짧게): "...잘 봤네."

**선택지** — 없음 (전투 + 획득 로그만)

**완료 조건**

- **4막 복선 아이템: `식어버린 황성 파편` (1안 권장)**

### 4막 복선 아이템 초안

**이름 후보 (가명)**
1. **`식어버린 황성 파편`** — 담백·장중, 4막 "식어있던 것이 다시 뛰기 시작한다" 반전 훅 강력 (권장)
2. `이름 잃은 박동석` — 문학적, 4막 "심핵" 용어 연결 자연스러움
3. `잠식 기사의 뒤늦은 맥` — 보스명 직결, 4막 연결 덜 시적

**설명 초안 (1안)**
> "황성 외곽 잠식 기사의 잔해에서 회수한 파편. 손에 쥐었을 때 미지근한 온기가 한 번 스쳤다가 곧 사라졌다. 판결기록관 베르딕트에게 문의하면 감정이 가능하다고 적혀 있지만, 지금의 그는 아직 답을 가지고 있지 않다."

- 관찰자(통과): "이제 자네가 수도에 남을 이유는 하나뿐이네. ...그게 뭔지 말하지는 않겠네. 곧 편지 한 통이 올 걸세."
- 관찰자(마지막): "이름을 묻지 말게. 나는 자네가 돌아올 때 여기 있을 걸세."
