## 문서 목적
4막 복선 아이템 심기용 1막 분기 A 동·북 테마 **서브 퀘스트 2종** 세부 설계. content-designer 1차 초안.

## 배경
- 4막 복선 7종 설계 (`poro_act1_theme_foreshadowing_items_draft.md`)
- 동부·북부는 **메인 9 직후 서브 퀘스트 보상 라인**으로 복선 자동 지급
- 대칭 구조: **동부 `젖은 사당` ↔ 북부 `얼어붙은 사당`** — 침식/설혼 온도 대비
- 두 서브 공통 구조: "사당 주변 3지점 해제 + 중간 전투 1회 + 복선 결정 자동 지급"

## 동부 서브 — `젖어가는 사당` (사당 정화 방향 확정)

### 방향 결정 근거
- 동부 테마 정체성 = **침식(조용히 스며드는 부패)**
- 유품 회수는 "이미 끝난 사건 뒷정리" 느낌이라 침식의 진행형 감각과 어긋남
- 사당 정화는 "침식이 현재 진행 중"인 장소에 플레이어가 직접 개입 → 4막 복선 `침묵한 정령의 눈물` 학습 효과
- 북부 조난자 유품 방향과 구조 중복 방지

### 카드

**한 줄 감각** — *침식은 비명을 지르지 않는다. 조용히 젖어들 뿐이다.*

**제목 3안**
1. **`젖어가는 사당`** ← 권장 (4막 복선 `침묵한 정령의 눈물`과 직접 연결)
2. `울지 않는 숲의 제단`
3. `침묵한 돌의 속삭임`

**스토리 비트 (4개)**
1. 테오린이 숲 외곽 사당의 공물이 "마르지 않고 고여 있다"는 이상 보고. 동행 요청
2. 사당 도착. 아일렌이 먼저 와 있고 돌 표면 소리 없는 진동 관측. 정령 반응이 평소와 다르다고 설명
3. 플레이어가 사당 주변 침식 흔적(고인 물·썩은 공물·탁해진 돌) **3지점 정화**. 마지막 지점에서 약한 침식체 1체 튀어나옴
4. 정화 완료 후 사당 중심에 눈물 형태 작은 결정. 에르세아 도착 *"이건 정령이 침식에 저항한 흔적"* → **`침묵한 정령의 눈물 조각` 자동 지급**

**주요 씬 — 사당 앞**
- 아일렌: *"이 돌은 울고 있어. 다만 우리가 그 울음을 들을 귀를 갖지 못한 것뿐이야."*
- 테오린: *"정화가 늦으면 숲 전체가 같은 방식으로 젖는다. 조용히, 아무도 모르게."*

**등장 NPC**
- **테오린** (숲 길 안내자) — 의뢰 제공, 사당 동행
- **아일렌** (정령 조사관) — 현장 해설, 침식 성격 설명
- **에르세아** (침식 기록 보관자) — 퀘스트 마무리, 복선 아이템 수령·기록
- 라비엘(본체 판별 조언자)·니로스(보급관)는 이번 퀘스트 배제 (본체 판별은 4막 후반 스포일러)

**완료 조건**
- `quest.act1_main.m9_east_shrine_3_purified` (3지점 정화)
- `quest.act1_main.m9_east_corruption_defeated` (침식체 1체 처치)
- `quest.act1_main.m9_east_foreshadow_obtained` (복선 아이템 수령)
- `quest.act1_main.m9_east_ercea_reported` (에르세아 보고)

**보상**
- **`침묵한 정령의 눈물 조각`** (4막 복선, 자동 지급, 귀속·파괴 불가)
- T1 성장 재료 중간 묶음 (정령 기운 담긴 재료, 제작대 투입용)
- 소량 경험치 + 지역 평판(동부 침식 숲 +3%)

**플레이 시간** — 15~18분 (이동 포함)

## 북부 서브 — `얼어붙은 한 숨` (얼어붙은 사당 조사 방향 확정)

### 방향 결정 근거
- 북부 테마 정체성 = **설혼(눈에 깃든 혼)**
- 조난자 유품은 생존 드라마 쪽 결, 설혼 신비 톤과 충돌
- 사당 조사 = "사당 자체가 숨 한 번 내쉬고 얼어붙었다"는 설혼적 상상력 직접 체험 → 4막 복선 `얼어붙은 숨 한 조각` 학습 효과
- 동부 `젖은 사당`과 대칭 (`얼어붙은 사당`)
- 조난자 유품 소재는 **길드 게시판 반복 의뢰 유형 D(수색·탐사)** 로 흡수

### 카드

**한 줄 감각** — *숨 한 번이 통째로 얼어붙은 자리, 그 안에 누가 있었다.*

**제목 3안**
1. **`얼어붙은 한 숨`** ← 권장 (4막 복선 `얼어붙은 숨 한 조각`과 직결)
2. `백빙 제단의 멎은 기도`
3. `눈 속에 갇힌 목소리`

**스토리 비트 (4개)**
1. 리안나가 성소에 찾아와 백빙 제단 외곽에서 "공기 자체가 얼어 있는 지점" 발견 보고. 바르트가 확인 요청
2. 플레이어가 제단 외곽 도착. **오스릭의 축복 수여**(진입 조건 서사적 해소)
3. 얼어붙은 공기 기둥 **3개 해제** (공명 돌 순서대로 두드리는 기믹). 세 번째 해제 시 얼어 있던 **설혼체 1체** 깨어나 전투
4. 기둥 중심에 사람 형상 얼음, 그 안에서 숨 한 조각 응결 결정. 리안나가 수거 후 *"네가 지녀야 할 것 같다"* 전달 → **`얼어붙은 숨 한 조각` 자동 지급**

**주요 씬 — 제단 외곽**
- 리안나: *"누군가 마지막 숨을 쉬었고, 그 숨이 여기서 멈췄어. 시간째로."*
- 바르트: *"제단은 듣는 곳이다. 오늘은 네가 대신 들어라."*

**등장 NPC**
- **리안나** (설혼 조사관) — 의뢰 제공, 현장 해설, 복선 아이템 전달
- **바르트** (백빙 제단 연락관) — 진입 서사 승인, 결 대사 담당
- **오스릭** (성소 관리자) — 진입 전 짧은 축복 씬
- 프레드릭(실종 순찰대 기록관)은 조난자 유품 라인용 보존, 에델(방한 보급관)은 본편 진입 전 보급 씬만 짧게 or 생략

**완료 조건**
- `quest.act1_main.m9_north_osric_blessing` (오스릭 축복 수여)
- `quest.act1_main.m9_north_pillars_3_cleared` (공기 기둥 3개 해제)
- `quest.act1_main.m9_north_seolhon_defeated` (설혼체 1체 처치)
- `quest.act1_main.m9_north_foreshadow_obtained` (복선 아이템 수령)

**보상**
- **`얼어붙은 숨 한 조각`** (4막 복선, 자동 지급, 귀속·파괴 불가)
- T1 성장 재료 중간 묶음 (설혼 결정 계열)
- **방한 버프 이용권 1회** (영지 제작대 연계)
- 소량 경험치 + 지역 평판(북부 설혼 +3%)

**플레이 시간** — 15~20분 (진입 축복·기둥 기믹 포함)

**UI 트리거 키**

## 서부 서브 — `풍향계 폐허 조사` (서부 서브 신설 확정, 안 (a) 채택)

### 방향 결정
**사용자 결정: 필드보스 5개 유지 + 서부 서브 신설**. 동·북 대칭 구조 완성 (3테마 서브 + 남부 메인 엔딩 고정 = 비대칭 1테마).

- 서부 테마 정체성 = **폭풍·방향 상실·모래**
- 사당 대신 **풍향계**로 변주 (서부 정체성 반영)
- 공통 골격 "3지점 해제 + 전투 1회 + 복선 자동 지급" 유지, 필드보스 추가 없음

### 카드

**한 줄 감각** — 풍향계가 멈춘 곳에서는 바람도 자기 이름을 잊는다.

**제목 3안**
1. **`풍향계 폐허 조사`** ← 권장 (4막 복선 `부서진 풍향계 조각`과 직결)
2. `멈춘 바람의 삼각점`
3. `사라진 상단의 표지`

**권장 레벨** — Lv.38~42 (메인 9 서부 완료 직후)

**스토리 비트 (4개)**
1. 야라가 제작한 임시 나침반이 특정 바람길에서만 떨림. 과거 기록에 "폐허가 된 풍향계 3기" 좌표. 조사 요청
2. 브렌 동행. 1·2번째 풍향계 복원 (2번째에서 **소규모 잡몹 교전 1회** — 모래바람 정령·반쯤 묻힌 초소병)
3. **삼각 정렬**: 3번째 풍향계 복원 순간 3기 모두 같은 방향 가리킴. 모래 위에 **어느 기록에도 없는 좌표** 1회성 발현
4. 해당 좌표에서 **`부서진 풍향계 조각` 자동 지급**. 도란이 "세 번의 실종이 전부 이 좌표 근처"라고 기록 대조 (3막 떡밥 저강도)

**주요 씬 — 삼각 정렬**
- 위치: 폭풍의 눈 외곽, 풍향계 폐허 3번째 지점
- 연출: 월드 사운드 없음, 채팅 서사 박스 "세 방향의 바람이 한 점으로 모인다" 톤. UI 중앙 나침반 3개 아이콘 동시 고정 파티클
- 브렌: *"바람이 길을 잃은 게 아니라, 누군가 길을 지운 겁니다."*
- 야라: *"이 나침반이 떨린 건 고장이 아니었어요. 기억하고 있었던 거예요."*

**등장 NPC**
- **야라** (나침반 장인) — 의뢰자·임시 나침반 제공
- **브렌** (폭풍 관측자) — 현장 동행, 기압·기류 이상 해설
- **도란** (실종 상단 기록관) — 마무리 기록 대조 (3막 떡밥 저강도)

**완료 조건**
- `quest.act1_main.m9_west_foreshadow_offered` (야라 의뢰 수락)
- `quest.act1_main.m9_west_foreshadow_windvane_1` / `_2` / `_3` (풍향계 3기 복원)
- `quest.act1_main.m9_west_foreshadow_coord_revealed` (삼각 정렬·좌표 발현)
- `quest.act1_main.m9_west_foreshadow_obtained` (복선 아이템 획득, 최종)

**보상**
- **`부서진 풍향계 조각`** (4막 복선, 자동 지급, 귀속·파괴 불가)
- 경험치(Lv40) + 소액 골드

- 서부 명성 +1%
- 야라·브렌·도란 호감도 소폭

**플레이 시간** — 15~20분 (동·북과 동일 범위)

**UI 트리거 키**
- `ui.west.foreshadow.compass_survey_open` (임시 나침반 추적 UI)
- `ui.west.foreshadow.windvane_restore_1` / `_2` / `_3` (풍향계 복원 3단계)
- `ui.west.foreshadow.triangulation_reveal` (삼각 정렬 연출)
- `ui.west.foreshadow.fragment_pickup` (조각 회수)

**특이 사항**
- 필드보스 추가 없음 — 2번째 풍향계에서 **소규모 잡몹 교전 1회만**
