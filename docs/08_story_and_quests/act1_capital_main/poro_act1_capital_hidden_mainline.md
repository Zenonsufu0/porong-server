# 1막 수도 히든 메인 라인 — "묵묵히 쌓은 자에게만 닿는 눈"

## 개요
수도 황성은 1막에서 **공용 허브 메인**(`수도 적응과 파견 자격`)이 끝난 뒤, 동/서/남/북 4테마로 파견 열차를 타고 나가는 것이 기본 동선이다.
본 문서는 이 기본 동선 대신 **수도에 남아 1막을 완주할 수 있는 히든 메인 라인**을 정의한다.

히든 메인 라인의 테마는 **황립기사단 수습 스카우트**다.
수도 길드에서 의뢰를 성실하게 쌓아 올린 플레이어를 길드 대기실에 상주하던 기사단 관찰자가 지켜보다가 먼저 수습 제의를 해오는 톤이다.
내부 수사·파견 보고 누락·회색 기록 추적 같은 "보이지 않는 정보"는 본 라인에서 분리되어 **서브 스토리 2 `빈민가의 그림자`(빈민가 정보기관 그림자)** 가 이어받는다.

> 본 문서는 기존 `poro_act1_theme_capital_storyline.md`의 공용 허브 메인을 대체하지 않는다. 공용 허브 메인이 끝난 뒤 갈라지는 **선택 분기**로 작동한다.
>
> 히든 메인 완주는 2막 **황립기사단 수습 제안**의 가중치로, 서브 2 완주는 2막 **그림자 협력 제안**의 가중치로 각자 작동한다. 두 루트는 1막에서 병행 진행 가능하다.

## 목표
- "성실한 의뢰 수행 → 공식 기관의 관찰 → 스카우트"라는 자연스러운 성장 서사를 1막 안에서 완결시킨다.
- 원정 테마(동/서/남/북)와 구분되는 **수도 안에서의 공식 루트 체험**을 완주 가능하게 한다.
- 1막 메인의 종료 레벨(전투 50, 주력 15강+, `???의 편지` 조건 활성)을 원정 테마와 동일하게 맞춘다.
- 4막 결전 보스 `황성 심핵의 재앙체` 떡밥을 1막에서 최소치만 심어둔다.

## 플러그인 판정 원칙 — Custom Plugin 단독
- 모든 상태·보상·해금·플래그 판정은 **EmpireRPG(Custom Plugin)** 단독이다.
- 모든 대사·선택지·연출 컷은 **EmpireRPG 커스텀 대화 UI**(`13_external_plugins_and_custom_ui` DialogueRegistry / SessionService / TriggerService 3계층)로 처리한다.
- **BetonQuest는 본 라인에서 사용하지 않는다.** 기존 BetonQuest 기반 설계 언급은 전면 제거.
- Citizens는 NPC 배치·외형·클릭 진입점만 담당한다. 상태 저장·보상 지급 금지.
- MythicMobs는 보스/몹 외형·스폰·스킬 연출만. 처치 판정은 EmpireRPG `BossKillListener`가 수신.

## 진입 구조 — 의뢰 누적 → 관찰자 접촉 → 수습 시험

### 단계 A. 공용 허브 메인 완료
- 선행: `수도 적응과 파견 자격` 공용 허브 메인 1~7 완주
- 상태: 메인 7(갈라진 징조) 완료, 파견 열차 정거장 진입 가능

### 단계 B. 수도 길드 의뢰 누적
- EmpireRPG가 플레이어의 길드 의뢰 수행 횟수·품질을 누적 집계 (키: `quest.capital_hidden.guild_request_count`)
- 누적 조건 (초안): 공용 메인 6 완료 후 **수도 길드 의뢰 6회 이상** + **정예/필드보스 합산 3체 이상 처치**
- 기록 소스: 기존 서브 `길드 게시판 의뢰` 수행 누적, 공용 메인 6/10의 의뢰 카운트, 수도 외곽 정리 서브의 전투 카운트

### 단계 C. 기사단 관찰자 NPC 대화 활성
- NPC: 수도 길드 대기실 창가 자리에 상주하는 **황립기사단 관찰자 하급 기사** (Citizens seed `role_type=knight_observer`, `interaction_profile_id=capital_knight_scout`)
- 공용 메인 1~7 기간에는 배경 NPC로만 존재. 대화 시 대기실 톤의 짧은 한 줄만 반환("…여긴 앉기만 하는 자리야. 용무 있으면 접수원한테.")
- 단계 B 조건 충족 순간 EmpireRPG 플래그 `quest.capital_hidden.observer_gated=true` 설정. 관찰자의 커스텀 대화 UI 트리가 확장된다.
- 플레이어 재방문 시 관찰자가 **먼저 말을 건다**:
  - 1차: "…너, 서류 위에 쓰인 이름으로만 봤는데. 요새 현장에선 다른 사람 같더군."
  - 2차: "계속 그렇게 쌓아 올릴 생각이라면, 이건 다른 길이야. 한 번 들어보겠나."

### 단계 D. 히든 진입 퀘스트 — `기사단 수습 시험`
- 관찰자 NPC가 수습 시험을 공식 제안
- 내용: 황립기사단은 "실적 기반 스카우트"를 간헐적으로 운영하며, 관찰자가 눈여겨본 비정규 조사 협력자를 시험 대상에 올린다. 시험은 길드·기사단 공동 관리이며 플레이어의 현장 기록을 근거로 진행된다.
- 완료조건 (EmpireRPG 판정):
  - 수습 시험 개요 대화 완료
  - 시험 서약서 전자 서명(UI 1회 인터랙션)
  - 길드 접수원에게 수습 시험 수주 통지
- 보상: 수도 명성 +2%, 길드 신뢰 +1%, 기사단 신뢰 +2%, EmpireRPG 해금 이벤트 `capital_hidden_main_unlocked` 발생
- 거절 처리: 수락하지 않으면 플래그 유지, 언제든 재수락 가능. 거절 중에도 타 테마로 파견 가능.

### 단계 E. 히든 메인 진입
- 해금 이벤트 `capital_hidden_main_unlocked` 발생 시 수도 길드 게시판에 **"황립기사단 수습 시험"** 탭 추가
- 이 탭에서 아래 수도 메인 8~15를 순차 수주. 수주·진행·완료 판정은 EmpireRPG 단독.

### 놓친 플레이어 보정
- 단계 B 조건을 충족하기 전에 열차에 탑승한 경우, 타 테마 초입 메인 완주 후 **`???의 편지` 수도 복귀 시점**에 관찰자 NPC가 자동으로 먼저 말을 건다(복귀 시점 `quest.capital_hidden.observer_gated` 재평가).
- 이 경우에도 수도 히든 메인 진입 가능. 단, 타 테마 초입 메인 보상을 이미 받은 상태이므로 수도 히든 메인의 **레벨·강화 보조 보상은 감량 또는 치장 대체**, 명성/기사단 신뢰만 가산.

## 히든 메인 라인 상세 — 8단계 (수습 시험 구조)

한 줄 감각: **"묵묵히 쌓은 의뢰 장부 한 페이지가, 기사단 관찰자의 눈에 먼저 닿는다."**

> 모든 단계는 EmpireRPG 퀘스트로 분리 저장되며, 대사·선택지는 커스텀 대화 UI가 렌더링한다. BetonQuest는 사용하지 않는다.

### 수도 메인 8. 수습 추천
- 동선: 수도 길드 대기실 → 기사단 관찰자 NPC → 길드 접수원
- 완료조건: 수습 시험 개요 대화 완료, 시험 서약서 UI 인터랙션, 접수원 통지
- 보상: 수도 명성 +2%, 기사단 신뢰 +2%
- 역할: 히든 진입 판정, EmpireRPG `stage=1 → 2`

### 수도 메인 9. 현장 심사 — 경계 농경지
- 동선: 수도 외곽 성문지대 → 경계 농경지(필드 2) → 시험관(기사단 하급 기사) 합류
- 완료조건:
  - 관찰자 동행 상태에서 일반 15체 처치
  - 강화 개체 3체 처치
  - 균열 경광 1점 해제(상호작용 1회)
- 보상: 수도 명성 +3%, 기사단 신뢰 +3%, 강화석 조각 중간 분량
- 톤: 시험관이 플레이어의 동선·교전 리듬을 기록. 대화는 짧고 공식적.

### 수도 메인 10. 서류 심사 — 파견 기록 해석
- 동선: 수도 기사단 분소 열람실
- 완료조건:
  - 파견 기록 견본 3건 열람 (커스텀 대화 UI의 읽기 전용 모드)
  - 4지선다 서류 문답 1회 (정답/오답 모두 진행 가능, 보상만 차등)
- 보상: 수도 명성 +2%, 길드 신뢰 +1%, 기사단 신뢰 +2%, (정답 시 추가) 기사단 신뢰 +1%
- 톤: 플레이어가 현장형일 뿐 아니라 문서 해독도 최소한 가능한지 확인. 이 단계에서 다뤄지는 파견 기록은 **공식 교보재용 샘플**이며, 실제 누락 사건 추적은 서브 2 `빈민가의 그림자`가 담당한다.

### 수도 메인 11. 협동 심사 — 빈 거리의 정예
- 동선: 하층 주거구(필드 3) 외곽 지정 좌표
- 완료조건:
  - 하급 기사 2인과 3인 파티 편성
  - 정예 1체 처치 (공식 시험 대상 몹, MythicMobs 스폰)
  - 협동 피격 최대치 제한 통과(플레이어 사망 0회)
- 보상: 수도 명성 +3%, 기사단 신뢰 +3%, T1 보조 장비 상자
- 톤: 기사단 협동 프로토콜 체험. 빈민가 핵심부에는 진입하지 않는다.

### 수도 메인 12. 직속 기사의 면접
- 동선: 황성 외곽 경계구역 초입 → 수도 기사단 주둔지 접견실
- 완료조건:
  - 상급 기사 면접 대화 완료
  - 1지망 분야 선택 (**순찰 / 호위 / 특수조사** 중 1택, EmpireRPG `knight_specialty ∈ {patrol, guard, invest}` 기록)
- 보상: 수도 명성 +2%, 기사단 신뢰 +2%, 기사단 공식 명찰(치장 아이템 1종)
- 톤: 선택은 2막 기사단 수습 제안 단계의 대사 톤만 조정. 스탯·진행 감점 없음.

### 수도 메인 13. 실전 심사 — 외곽 필드보스
- 동선: 황성 외곽 경계구역 심부
- 완료조건: 수도 필드보스 `황성 외곽 잠식 기사` 처치 (MythicMobs 스폰, EmpireRPG 리스너 판정)
- 보상:
  - 수도 명성 +5%, 기사단 신뢰 +3%
  - 보스 부산물에서 "황성 심핵" 1차 단서 1건 (4막 결전 떡밥, 1막 내 추가 확장 없음)

### 수도 메인 14. 수습 승인
- 동선: 수도 기사단 주둔지 → 관찰자 NPC 복귀
- 완료조건:
  - 시험관 3인의 총평 대화 완료
  - 황립기사단 공식 수습증 수령(아이템 1종, 치장/증표)
- 보상: 기사단 신뢰 +3%, 수습증 소지 상태 활성화 (2막 황립기사단 수습 제안 게이트)

### 수도 메인 15. 수도에 남는 이유
- 동선: 관찰자 NPC → 레오니드 복귀 보고
- 완료조건:
  - 전투 레벨 50
  - **전 장비 T1 18강** (`???의 편지` 활성 게이트)
  - 수도 필드보스 누적 3종 이상 조우, 2종 이상 처치
  - 길드 의뢰 누적 8회 (단계 B 카운트 포함)
  - `???의 편지` 조건 활성화 (EmpireRPG `letter_unlock` 발행)
- 보상:
  - 수도 테마 명성 +6%
  - 기사단 신뢰 +4%
  - 수도 심층 연결용 전용 조각 1개 (2막 `황성 지하 봉인구` 개방 가중)
  - 튜닝 제안(경제 검토 반영): 강화석 **50~70개 + 파편 200개 1회성 대량 지급** 검토 필요. 전 장비 18강 부족분 완화 용도.

## 범위
- 1막 수도 히든 메인 라인 8단계 (기사단 수습 스카우트 톤)
- 히든 진입 구조(의뢰 누적 → 관찰자 → 수습 시험)
- 공용 허브 메인과의 분기 지점
- Custom Plugin 단독 판정 원칙 문서화(BetonQuest 미사용 명시)

## 비범위
- 빈민가/그림자 접선 및 내부 수사 콘텐츠 (서브 2 `빈민가의 그림자` 참조)
- 대사 전문, 컷신 연출, NPC 이름 최종 확정
- 2막 수도 심층 필드/보스 상세
- 4막 결전 보스의 1막 확장 떡밥

## 구현 포인트
