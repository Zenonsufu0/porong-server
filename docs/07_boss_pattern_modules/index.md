# 07. 보스 패턴 모듈

> **[STATUS: REFERENCE]** — 보스 패턴 모듈 진입점. 공식 보스 기준은 `../06_fields_bosses/CANON.md`가 우선.

## 원칙

보스 패턴은 개별 하드코딩하지 않는다. 패턴 모듈, 보스별 조합, 테마 연출 스킨, 수치 파라미터를 분리한다.

## 문서 구조

| 파일 | 내용 |
|---|---|
| `common_patterns.md` | 공용 패턴 모듈 P-00~P-09, P-11~P-13 스펙 및 파라미터 (P-10 폐기) |
| `field_boss_patterns.md` | 필드보스 5종 패턴 배정 (공용 패턴만 사용) |
| `season_boss_patterns.md` | 시즌보스 6종+최종보스 3종 공용+고유 패턴 배정 (SP-1X~SP-9X) |

## 계층별 재사용 비율

- 필드보스: 공용 100% (고유 패턴 없음)
- 시즌보스: 공용 + 고유 패턴 조합
- 시즌 최종보스: 공용 + 고유 패턴 조합 (고유 비중 최대)

## 보스 전투 타임아웃 (확정 2026-05-15)

| 보스 분류 | 타임아웃 |
|---|---|
| 필드보스 5종 / 시즌보스 1~6 | **15분** |
| 시즌 최종보스 3종 (균열왕·이중체·주시자) | **10분** |

패턴 구현 시 타임아웃 이내에 공략 가능한 HP/패턴 흐름으로 설계한다.
시즌 최종보스는 22강 무기 파티 3인 기준으로 10분 내 클리어 가능해야 하며,
이 기준이 최종보스 HP 역산의 핵심 전제가 된다.
