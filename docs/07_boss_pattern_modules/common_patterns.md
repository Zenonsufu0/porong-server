## P-00 기본 공격

**분류**: 기본 타격  
**개요**: 보스가 전투 중 지속적으로 발동하는 기본 타격. 스킬 쿨타임 사이를 채우며 전투 리듬을 형성. 예고가 짧아 회피보다 생존력(방어/HP)으로 버티는 것이 기본 대응.

| 파라미터 | 기본값 | 조정 범위 |
|---|---|---|
| 피해 (% ATK 대비) | 60% | 40~90% |
| 전방 범위 (블럭) | 2.5 | 2~4 |
| 넉백 | 없음 | 없음/약함 |
| 쿨타임 (초) | 1.8 | 1.0~3.0 |
| 예고 시간 (초) | 0.3 | 0.2~0.5 |

**설계 의도**: 스킬(P-01 이상)과 P-00이 함께 돌아가면 보스가 항상 뭔가를 하는 느낌을 만든다. P-00 피해는 작지만 방치하면 꾸준히 깎임 → 플레이어가 회복 타이밍을 놓치지 않도록 압박.  
**조정 축**: 쿨타임(타격 빈도), 피해량, 근접형/원거리형 전환(원거리 보스는 약한 투사체로 교체).

## P-01 전방 강타

**분류**: 근접 공격  
**개요**: 보스 전방 좁은 범위를 강타. 예비 동작(모션)으로 예고.

| 파라미터 | 기본값 | 조정 범위 |
|---|---|---|
| 피해 (% ATK 대비) | 120% | 80~200% |
| 전방 범위 (블럭) | 3 | 2~5 |
| 넉백 거리 | 1.5 | 0~3 |
| 쿨타임 (초) | 4 | 2~8 |
| 예고 시간 (초) | 0.8 | 0.5~1.2 |

**회피 방법**: 옆으로 1블럭 이동 또는 뒤로 스탭.  
**조정 축**: 피해량, 범위, 쿨타임. 보스 테마에 따라 이펙트(불꽃/독/얼음 등) 교체.

## P-02 직선 돌진

**분류**: 이동+공격  
**개요**: 보스가 전방 직선으로 고속 돌진. 경로 위 유저에게 피해 + 넉백.

| 파라미터 | 기본값 | 조정 범위 |
|---|---|---|
| 피해 | 150% | 100~250% |
| 돌진 거리 (블럭) | 10 | 6~20 |
| 이동 속도 배율 | 3.0× | 2~5× |
| 경로 너비 (블럭) | 2 | 1~4 |
| 쿨타임 (초) | 8 | 5~15 |
| 예고 시간 (초) | 1.5 | 1~2 |

**회피 방법**: 예고선 수직 방향으로 이동.  
**조정 축**: 거리, 속도, 경로 너비. 종료 시 지면 충격 추가 여부.

**분류**: 광역 공격  
**개요**: 보스 중심으로 원형 범위에 폭발. 딜레이 후 발동.

| 파라미터 | 기본값 | 조정 범위 |
|---|---|---|
| 피해 | 130% | 100~200% |
| 반지름 (블럭) | 5 | 3~8 |
| 딜레이 (초) | 1.2 | 0.8~2.0 |
| 근접 면제 여부 | 없음 | 있음/없음 |
| 쿨타임 (초) | 12 | 8~20 |

**회피 방법**: 반지름 밖으로 이탈 또는 보스에 밀착(근접 면제 적용 시).  
**조정 축**: 반지름, 딜레이, 2단 폭발(근접→원형) 여부.

**개요**: 보스 상공에서 여러 투사체가 랜덤 범위에 낙하. 피해 범위가 넓어 이동이 강제됨.

| 낙하 범위 (반지름) | 8 | 5~15 |

| 착탄 범위 (블럭) | 1.5 | 1~3 |

**회피 방법**: 착탄 경고 파티클 위치를 확인하고 이동.  
**조정 축**: 발사 수, 낙하 범위, 착탄 범위, 이동형(유도) vs 고정형.

## P-07 소환체 호출

**분류**: 소환  
**개요**: 보스가 소환체를 필드에 등장시킴. 소환체는 자동 공격하며, 처치하거나 방치하면 추가 효과.

| 파라미터 | 기본값 | 조정 범위 |
|---|---|---|
| 소환 수 | 2 | 1~4 |
| 소환체 HP | 2,000 | 500~10,000 |
| 소환체 공격력 | 40% | 20~80% |
| 방치 시 효과 | 없음 | 폭발/버프/강화 |
| 쿨타임 (초) | 20 | 10~40 |

**처리 방법**: 파티원이 소환체를 나눠 처치하거나, 방치 시 발생하는 효과에 대응.  
**조정 축**: 수, HP, 방치 효과, 보스에게 귀환 여부.

## P-08 페이즈 전환

**분류**: 상태 변환  
**개요**: 특정 HP% 도달 시 보스가 상태 전환. 다음 Phase의 패턴 세트로 전환.

| 파라미터 | 기본값 | 조정 범위 |
|---|---|---|
| 트리거 HP% | 50% | 10~80% |
| 전환 시 무적 여부 | 없음 | 있음/없음 |
| 전환 시 이펙트 | 파티클 | 이펙트 교체 가능 |
| 전환 메시지 | 없음 | 채팅/보스바 |

**용도**: 단독으로 사용하지 않고 P-09와 결합하거나, 새 패턴 세트 활성화 트리거로 사용.  
**조정 축**: 트리거 HP%, 전환 횟수(1~3회).

## P-09 무력화+딜타임

**분류**: 공략 메카닉  
**개요**: 특정 조건(구조물 파괴, 상호작용)을 달성하면 보스 무적이 해제되고 취약 창이 발생.

| 파라미터 | 기본값 | 조정 범위 |
|---|---|---|
| 무적 시간 (초) | 60 | 30~180 |
| 해제 조건 | 구조물 파괴 | 구조물/이동/상호작용 |
| 해제 성공 시 기절 (초) | 3 | 2~8 |
| 취약 창 피해 증폭 | +20% | +15~+50% |
| 실패 시 효과 | 없음 | 분노/피해/강화 |

**용도**: 시즌보스 Phase 전환의 핵심 메카닉. 필드보스에서는 단순화 버전(전환 없는 짧은 기절)으로 적용.  
**조정 축**: 무적 시간, 해제 조건 복잡도, 취약 창 배율.

## P-11 추적 구체

**분류**: 유도 원거리  
**개요**: 보스가 특정 플레이어를 지정해 유도 투사체 발사. 구체는 느리게 날아오지만 진행 방향을 지속 수정. 보스 쪽으로 유도시켜 소멸시키는 것이 최적 대응.

| 파라미터 | 기본값 | 조정 범위 |
|---|---|---|
| 피해 (착탄) | 140% | 100~200% |
| 추적 속도 | 4블럭/s | 3~8블럭/s |
| 추적 지속 (초) | 5 | 3~8 |
| 구체 수 | 1 | 1~2 |
| 쿨타임 (초) | 10 | 6~18 |
| 예고 시간 (초) | 0.8 | 0.5~1.2 |

**회피 방법**: 보스 방향으로 달려 구체를 보스에게 유인 → 구체 보스 충돌 시 소멸 (피해 없음). 또는 구체 소멸 타이밍까지 회피 이동.  
**조정 축**: 추적 속도, 지속 시간, 착탄 폭발 반지름.

## P-12 연속 타격

**분류**: 근접 연타  
**개요**: 보스가 짧은 간격으로 2~4회 빠르게 근접 공격을 연속 발동. 1타당 피해는 작지만 전타를 맞으면 총 피해가 큼.

| 파라미터 | 기본값 | 조정 범위 |
|---|---|---|
| 타격 횟수 | 3 | 2~4 |
| 타당 피해 | 60% | 40~100% |
| 타격 간격 (초) | 0.4 | 0.2~0.6 |
| 범위 (블럭) | 3 | 2~4 |
| 쿨타임 (초) | 8 | 5~14 |
| 예고 시간 (초) | 0.5 | 0.3~0.8 |

**회피 방법**: 첫 예비 동작 확인 즉시 범위 밖으로 이탈. 중간 이탈 시 남은 타 회피 가능.  
**조정 축**: 횟수, 타격 간격, 마지막 타 넉백 추가 여부.

## P-13 낙하 충격

**분류**: 도약+광역  
**개요**: 보스가 공중으로 도약한 뒤 목표 위치에 낙하. 착지 시 범위 광역 피해 + 넉백. 낙하 지점은 파티클로 예고.

| 파라미터 | 기본값 | 조정 범위 |
|---|---|---|
| 피해 (착지 시) | 160% | 120~240% |
| 착지 범위 반지름 (블럭) | 5 | 3~8 |
| 도약~착지 시간 (초) | 1.5 | 1.0~2.5 |
| 낙하 대상 | 최근거리 플레이어 | 랜덤/최근거리 |
| 쿨타임 (초) | 14 | 8~20 |
| 예고 시간 (초) | 1.5 | 1.0~2.0 |

**회피 방법**: 낙하 지점 파티클 확인 후 반지름 밖으로 이탈. 근접 딜러는 낙하 직전 보스에서 멀어지는 것이 핵심.  
**조정 축**: 착지 범위, 착지 후 지면 크랙 장판 잔류 여부, 연속 낙하(2회) 여부.
