## P-01 전방 강타

**분류**: 근접 공격  
**개요**: 보스 전방 좁은 범위를 강타. 예비 동작(모션)으로 예고.

| 파라미터 | 기본값 | 조정 범위 |
|---|---|---|
| 피해 (% ATK 대비) | 120% | 80~200% |
| 전방 범위 (블럭) | 3 | 2~5 |
| 넉백 거리 | 1.5 | 0~3 |
| 쿨타임 (초) | 4 | 2~8 |
| 예고 시간 (초) | 0.8 | 0.5~1.2 |

**회피 방법**: 옆으로 1블럭 이동 또는 뒤로 스탭.  
**조정 축**: 피해량, 범위, 쿨타임. 보스 테마에 따라 이펙트(불꽃/독/얼음 등) 교체.

## P-02 직선 돌진

**분류**: 이동+공격  
**개요**: 보스가 전방 직선으로 고속 돌진. 경로 위 유저에게 피해 + 넉백.

| 파라미터 | 기본값 | 조정 범위 |
|---|---|---|
| 피해 | 150% | 100~250% |
| 돌진 거리 (블럭) | 10 | 6~20 |
| 이동 속도 배율 | 3.0× | 2~5× |
| 경로 너비 (블럭) | 2 | 1~4 |
| 쿨타임 (초) | 8 | 5~15 |
| 예고 시간 (초) | 1.5 | 1~2 |

**회피 방법**: 예고선 수직 방향으로 이동.  
**조정 축**: 거리, 속도, 경로 너비. 종료 시 지면 충격 추가 여부.

---

## P-03 원형 폭발

**분류**: 광역 공격  
**개요**: 보스 중심으로 원형 범위에 폭발. 딜레이 후 발동.

| 파라미터 | 기본값 | 조정 범위 |
|---|---|---|
| 피해 | 130% | 100~200% |
| 반지름 (블럭) | 5 | 3~8 |
| 딜레이 (초) | 1.2 | 0.8~2.0 |
| 근접 면제 여부 | 없음 | 있음/없음 |
| 쿨타임 (초) | 12 | 8~20 |

**회피 방법**: 반지름 밖으로 이탈 또는 보스에 밀착(근접 면제 적용 시).  
**조정 축**: 반지름, 딜레이, 2단 폭발(근접→원형) 여부.

## P-04 부채꼴 휩쓸기

**분류**: 근접 광역  
**개요**: 보스 전방 부채꼴 범위를 휩쓸기. 회피 방향이 명확해 읽기 쉬운 패턴.

| 파라미터 | 기본값 | 조정 범위 |
|---|---|---|
| 피해 | 110% | 80~160% |
| 각도 | 120° | 60~180° |
| 거리 (블럭) | 4 | 3~6 |
| 쿨타임 (초) | 5 | 3~10 |
| 예고 시간 (초) | 1.0 | 0.6~1.5 |

**회피 방법**: 보스 측면 또는 후방으로 이동.  
**조정 축**: 각도, 거리, 연속 횟수(1~3회 연속 발동).

## P-05 투사체 발사

**분류**: 원거리 공격  
**개요**: 보스가 타겟을 향해 투사체 발사. 속도·개수·궤적 조정.

| 파라미터 | 기본값 | 조정 범위 |
|---|---|---|
| 피해 | 100% | 70~150% |
| 투사체 속도 | 12 | 8~20 |
| 발사 수 | 1 | 1~3 |
| 관통 여부 | 없음 | 있음/없음 |
| 쿨타임 (초) | 6 | 3~10 |
| 예고 시간 (초) | 0.6 | 0.4~1.0 |

**회피 방법**: 이동기 또는 횡이동.  
**조정 축**: 속도, 발사 수, 유도 여부, 이펙트(독/불/얼음 등).

## P-06 산개탄

**분류**: 광역 원거리  
**개요**: 보스 상공에서 여러 투사체가 랜덤 범위에 낙하. 피해 범위가 넓어 이동이 강제됨.

| 파라미터 | 기본값 | 조정 범위 |
|---|---|---|
| 피해 (착탄) | 120% | 90~180% |
| 발사 수 | 6 | 3~12 |
| 낙하 범위 (반지름) | 8 | 5~15 |
| 낙하 딜레이 (초) | 1.5 | 1.0~2.5 |
| 착탄 범위 (블럭) | 1.5 | 1~3 |
| 쿨타임 (초) | 15 | 10~25 |

**회피 방법**: 착탄 경고 파티클 위치를 확인하고 이동.  
**조정 축**: 발사 수, 낙하 범위, 착탄 범위, 이동형(유도) vs 고정형.

## P-07 소환체 호출

**분류**: 소환  
**개요**: 보스가 소환체를 필드에 등장시킴. 소환체는 자동 공격하며, 처치하거나 방치하면 추가 효과.

| 파라미터 | 기본값 | 조정 범위 |
|---|---|---|
| 소환 수 | 2 | 1~4 |
| 소환체 HP | 2,000 | 500~10,000 |
| 소환체 공격력 | 40% | 20~80% |
| 방치 시 효과 | 없음 | 폭발/버프/강화 |
| 쿨타임 (초) | 20 | 10~40 |

**처리 방법**: 파티원이 소환체를 나눠 처치하거나, 방치 시 발생하는 효과에 대응.  
**조정 축**: 수, HP, 방치 효과, 보스에게 귀환 여부.

---

## P-08 페이즈 전환

**분류**: 상태 변환  
**개요**: 특정 HP% 도달 시 보스가 상태 전환. 다음 Phase의 패턴 세트로 전환.

| 파라미터 | 기본값 | 조정 범위 |
|---|---|---|
| 트리거 HP% | 50% | 10~80% |
| 전환 시 무적 여부 | 없음 | 있음/없음 |
| 전환 시 이펙트 | 파티클 | 이펙트 교체 가능 |
| 전환 메시지 | 없음 | 채팅/보스바 |

**용도**: 단독으로 사용하지 않고 P-09와 결합하거나, 새 패턴 세트 활성화 트리거로 사용.  
**조정 축**: 트리거 HP%, 전환 횟수(1~3회).

---

## P-09 무력화+딜타임

**분류**: 공략 메카닉  
**개요**: 특정 조건(구조물 파괴, 상호작용)을 달성하면 보스 무적이 해제되고 취약 창이 발생.

| 파라미터 | 기본값 | 조정 범위 |
|---|---|---|
| 무적 시간 (초) | 60 | 30~180 |
| 해제 조건 | 구조물 파괴 | 구조물/이동/상호작용 |
| 해제 성공 시 기절 (초) | 3 | 2~8 |
| 취약 창 피해 증폭 | +20% | +15~+50% |
| 실패 시 효과 | 없음 | 분노/피해/강화 |

**용도**: 시즌보스 Phase 전환의 핵심 메카닉. 필드보스에서는 단순화 버전(전환 없는 짧은 기절)으로 적용.  
**조정 축**: 무적 시간, 해제 조건 복잡도, 취약 창 배율.

## P-10 안전지대

**분류**: 위치 메카닉  
**개요**: 필드에 위험 장판이 생성되며, 유저는 안전 구역에서만 피해 없이 딜 가능.

| 파라미터 | 기본값 | 조정 범위 |
|---|---|---|
| 장판 피해/초 | 80% | 40~200% |
| 장판 크기 (반지름) | 3 | 2~6 |
| 장판 수 | 3 | 1~8 |
| 지속 시간 (초) | 8 | 4~20 |
| 경고 시간 (초) | 1.5 | 1~3 |
| 쿨타임 (초) | 18 | 10~30 |

**회피 방법**: 장판 경고색 확인 후 안전 구역으로 이동하며 딜 지속.  
**조정 축**: 장판 수, 크기, 이동 여부(고정/이동 장판), 주기.
