- 두 개체가 각각 독립적으로 행동하면서 파티를 동시에 압박.
- 한 개체의 HP%가 상대 개체와 25%p 이상 벌어지면 광폭화 발동 (SP-81).
- **동기화 Phase 전환**: 두 개체 모두 전환 HP%에 도달해야 발동. 먼저 도달한 개체는 피해 면역 대기 상태.
- **전투 종료 조건**: Z·S 두 개체 모두 0% 처치 필요. 한 개체만 처치 시 나머지 개체 SP-81 광폭화 + 분노 스택 max 진입.

### Phase 구조

| Phase | 기준 | 특징 |
|---|---|---|
| Phase 1 | 100% ~ 60% | SP-81 광폭화 감시, Z 근접·S 원거리 독립 행동 |
| 전환 #1 | 두 개체 동시 60% 도달 | SP-83 결속 차단, 무적 120s |
| Phase 2 | 60% ~ 30% | S 개체 P-03 원형 폭발 추가, SP-81 지속 |
| 전환 #2 | 두 개체 동시 30% 도달 | SP-84 합체 저지, 무적 120s |
| Phase 3 | 30% ~ 0% | SP-82 쌍 합동 공격 지속, P-00 쿨 단축 |

### 공용 패턴 파라미터 오버라이드

| 패턴 | 코드 | 오버라이드 | 활성 개체 | 활성 Phase |
|---|---|---|---|---|
| 기본 공격 | P-00 | 피해 85%, 쿨 2.0s | Z·S 각각 독립 | Phase 1·2·3 |
| 전방 강타 | P-01 | 피해 155%, 범위 3블럭, 쿨 5s | Z 전용 (근접) | Phase 1·2·3 |

| 무력화+딜타임 | P-09 | 무적 120s × 2회, 기절 4s, 취약 창 +30% | 양 개체 동시 | 전환 #1·#2 |

### 고유 패턴

#### SP-81 쌍둥이 광폭화 (상시 감시)

| 파라미터 | 값 |
|---|---|
| 트리거 조건 | `abs(zombie_hp% - skeleton_hp%) > 25` |
| 광폭화 개체 | HP%가 낮은 쪽 (먼저 깎인 개체) |
| 광폭화 효과 | 공격력 +100%, 이동속도 +50% |
| 지속 조건 | HP% 차이가 25% 이하로 좁혀질 때까지 유지 |
| 알림 | 광폭화 개체 몸 색 변화 + 인게임 경고 메시지 |

**설계 의도**: 파티가 두 개체에 균등하게 딜을 분배하도록 강제. 1인이 한 개체만 공격하면 반드시 광폭화 → 와이프 수준 압박.

#### SP-82 쌍 합동 공격 (Phase 전환 #2)

| 파라미터 | 값 |
|---|---|
| 발동 조건 | 두 개체 동시 HP 30% 동기화 도달 → Phase 전환 #2 이후 Phase 3 전체 지속 |
| 효과 | 두 개체가 파티 동일 대상을 동시 타격하는 연계 공격 |
| 지속 | Phase 3 (30%~0%) 전체 |
| 알림 | 두 개체 눈빛 동기화 → 동시 돌진 예고 |

**설계 의도**: Phase 3에서 두 개체를 물리적으로 분리시켜 각자 상대하는 전략을 강제. 3인 파티는 2:1 또는 1:1:0 분리 배치로 대응.

#### SP-83 결속 차단 (Phase 전환 #1)

**개요**: 무적 전환 시 Z·S 사이에 결속 링크가 형성. 링크가 활성화된 동안 두 개체는 받은 피해의 50%를 상대에게 전이(회복). 파티원이 두 개체 사이 중간 지점에 위치하면 링크를 차단해 피해 전이 차단 + 취약 창 발생.

| 파라미터 | 값 |
|---|---|
| 발동 조건 | 두 개체 동시 HP 60% 동기화 도달 → Phase 전환 #1 |
| 링크 활성 범위 | Z·S 사이 직선 |
| 피해 전이율 | 받은 피해의 50%를 상대 개체가 회복 |
| 차단 조건 | 파티원 1명이 Z·S 사이 중간 4블럭 이내 위치 |
| 차단 지속 | 해당 플레이어가 위치 유지하는 동안 |
| 차단 성공 시 | 두 개체 동시 기절 4초 + 취약 창 +30% |
| 제한 시간 | 120초 |
| 실패 시 | 링크 강화 → 피해 전이율 80%로 증가 + 보스 분노 복귀 |

**인원별 차이**:
- 3인: 1명이 중간 차단, 2명이 딜. SP-81 광폭화와 반대 — 이 구간은 붙어있어야 함
- 2인: 1명 차단 + 1명 딜 (딜 효율 절반)
- 1인: 차단과 딜을 혼자 — 차단 위치에서 원거리 무기로 딜. 근접 무기 불리

**설계 의도**: SP-81(떨어져 있어야 함)과 SP-83(붙어있어야 함)이 교대로 등장 — 파티가 분리/집결을 반복하도록 강제.

---

#### SP-84 합체 저지 (Phase 전환 #2 전용)

**개요**: 무적 전환 시 Z·S가 서로를 향해 천천히 이동하며 합체 시도. 파티원이 두 개체 사이에 위치해 이동을 차단하면서 집중 딜로 해제.

| 파라미터 | 값 |
|---|---|
| 발동 조건 | 두 개체 동시 HP 30% 동기화 도달 |
| 이동 속도 | 1블럭/2초 (멈추지 않음) |
| 합체 조건 | Z·S가 3블럭 이내 접근 → 합체 완료 |
| 차단 방법 | 파티원 1명 이상이 Z·S 사이에 위치하면 이동 차단 |
| 해제 조건 | 차단 유지 중 두 개체 HP 합산 **40%p 추가 감소** (전환 진입 시점 기준) |
| 해제 제한 시간 | 120초 |
| 성공 시 | 두 개체 기절 5초 + 취약 창 +35% |
| 실패 시 (합체 완료) | 융합체 생성 — 클리어 불가 |

**합체 실패 시 — 융합체 (와이프 메카닉)**:
- Z·S가 하나로 합쳐진 융합체 생성, HP = 두 개체 잔여 HP 합산
- HP 재생: 초당 1% (3인 최대 딜로도 압도 불가)
- 4초마다 전장 전체 즉사급 광역 공격
- → 클리어 불가. 10분 타임아웃 또는 전멸로 강제 종료.

**인원별 차이**:
- 3인: 1명 차단 + 2명 딜. 이동하는 두 개체 사이를 유지해야 해 SP-83보다 동적.
- 2인: 1명 차단 + 1명 딜로 40%p 달성 — 매우 빡빡.
- 1인: 차단 + 딜 동시 수행 — 사실상 클리어 불가.

**설계 의도**: Phase 전환 #2 실패 = 즉각 와이프. SP-83(정적 차단)과 달리 두 개체가 이동하므로 차단 위치를 실시간으로 조정해야 한다. 이중체 전투의 최대 긴장 구간.

## 시즌최종보스3 — 진혼의 주시자 (DL-043)

**입장 가능 인원**: 1~3인  
**전투 타이머**: 10분  
**공용 패턴 활용**: P-00, P-03, P-06, P-09  
**고유 패턴**: SP-91, SP-92  
**베이스 몹**: 위더 (별빛 마법사 MythicMobs 소환체 병행)

> **1~2인 입장**: 가능하나 인원 스케일 보정 없음. 3인 기준 설계.

### 패턴 개요

- 위더 본체 + 별빛 마법사(에보커 기반 소환체) 주기적 소환.
- 마법사가 살아있는 동안 위더 받피감이 누적. 4명이면 70%로 사실상 무적.
- 5명 누적 시 별빛 대마법 발동 → 전원 소모(리셋) → 대량 피해.
- 마법사 처리 담당 + 위더 공격 담당으로 역할 분담 필수.

### Phase 구조

| Phase | HP 기준 | 마법사 소환 주기 | 특징 |
|---|---|---|---|
| Phase 1 | 100% ~ 70% | 30초/명 | P-00·P-06 기본 구성 |
| Phase 2 | 70% ~ 35% | 20초/명 | P-03 원형 폭발 추가, 소환 압박 증가 |
| Phase 3 | 35% ~ 0% | 12초/명 | 마법사 관리가 클리어 가부 결정 |

### 공용 패턴 파라미터 오버라이드

| 패턴 | 코드 | 오버라이드 | 활성 Phase |
|---|---|---|---|

### 고유 패턴

#### SP-91 별빛 마법사 소환 (전 Phase)

| 파라미터 | 값 |
|---|---|
| 소환 주기 | Phase 1: 30초/명, Phase 2: 20초/명, Phase 3: 12초/명 |
| 마법사 최대 동시 존재 수 | 5명 (5명 이상 → SP-92 자동 발동) |
| 마법사 HP | 15,000 (공용 단체 처치 가능) |
| 위더 받피감 적용 | 0명=0%, 1명=15%, 2명=30%, 3명=50%, 4명=70%, 5명=SP-92 |
| 마법사 처치 시 | 해당 개체 받피감 스택 즉시 제거 |

**설계 의도**: DPS 압박과 마법사 처리 압박을 동시에 부과. Phase 3에서 소환 주기가 12초로 빨라져 3인이 집중해야 간신히 4명 이하 유지 가능.

#### SP-92 별빛 대마법 (5명 스택 시)

| 파라미터 | 값 |
|---|---|
| 발동 조건 | 마법사 동시 존재 5명 도달 |
| 효과 | 전장 전체 강력한 피해 (파티 전체 HP 60~80% 감소) |
| 처리 | 마법사 5명 전원 소모 (스택 0 리셋) |
| 예고 | 위더 몸 빛 방출 2초 예고 → 대마법 발사 |

**설계 의도**: 대마법을 피하는 게 아니라 마법사를 5명이 쌓이기 전에 처치하는 것이 답. 만약 대마법을 맞으면 파티 전원 HP가 줄어들어 이후 생존이 힘들어짐 → 마법사 관리 실패의 페널티.
