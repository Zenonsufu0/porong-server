### 고유 패턴

#### SP-11 에너지 석판 (Phase 전환 #1 전용)

**개요**: 무적 전환 시 지면에 석판 4개 등장. 파티원이 밟아 에너지를 흡수하면 무적 해제.

| 파라미터 | 값 |
|---|---|
| 석판 수 | 4 |
| 충전 시간/석판 | 6초 (밟고 있어야 함) |
| 총 해제 제한 시간 | 120초 |
| 석판 간격 | 전장 4방향 배치 |
| 방치 시 효과 | 120초 경과 → 보스 분노 복귀 |

**인원별 차이**:
- 3인: 각자 1~2개 담당, 여유 있음
- 2인: 2개씩 순차 처리, 촉박함
- 1인: 4개를 순서대로, 시간 압박 최대 (약 90~100초 소요)

#### SP-12 유도 에너지 구체 (Phase 전환 #2 전용)

**개요**: 무적 전환 시 유도 구체 4개 등장. 근접 공격으로 격파해야 무적 해제.

| 파라미터 | 값 |
|---|---|
| 구체 수 | 4 |
| 구체 HP | 1,000 |
| 유도 속도 | 느림 (2블럭/s) |
| 구체 공격력 | 접촉 시 60%/초 |
| 총 해제 제한 시간 | 120초 |
| 방치 시 효과 | 전장 폭발 → 큰 범위 피해 후 보스 분노 복귀 |

**인원별 차이**:
- 3인: 구체 분담 격파
- 1~2인: 구체 유도 방향을 통제하며 순차 격파

**핵심 메카닉**:
- SP-12에서 구체를 격파하지 않으면 전장 폭발 (즉사 수준). Phase 2 이후 실질 클리어 조건.

| 방치 시 효과 | 분노 상태로 보스 복귀, 공격력 +20% |

**인원별 차이**:
- 3인: 창(근거리)과 석궁(원거리) 조합 시 최적
- 1인 근거리 무기: 반사 피해를 감수하며 처리 (생존 도전 요소)

**회피 방법**: 경고선 수직 방향으로 이탈. P-02보다 범위가 좁아 세로 이동만으로 회피 가능.

#### SP-42 저주 표식+거리 유지 (Phase 전환 #2 전용)

**개요**: 파티원에게 저주 표식 부여. 표식 유저끼리 근접 시 폭발.

| 파라미터 | 값 |
|---|---|
| 표식 부여 대상 | 파티 전원 (3인이면 3개) |
| 폭발 트리거 범위 | 4블럭 이내 접근 |
| 폭발 피해 | 200% |
| 표식 지속 | 쉴드(P-09 연계) 해제까지 |
| 1인 입장 시 | 표식 1개, 이 구간 난이도 감소 |

**인원별 차이**:
- 3인: 서로 거리를 유지하며 쉴드 격파. 밀집 금지.
- 2인: 2인이 서로 반대편에서 딜.
- 1인: 표식 1개만 부여 → 오히려 이 구간이 쉬워지는 역설 구간.

**핵심**: 딜 가능한 위치와 안전한 위치가 겹치는 구간을 빠르게 찾아 이동.

**설계 의도**: Phase 4 딜 사이클 강제. 배리어 → 딜 → 배리어 루프. 느리면 타이머 압박.
