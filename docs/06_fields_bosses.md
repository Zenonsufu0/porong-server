# 06. 필드 / 몹 / 보스

## 필드 5개

| 필드 | 역할 | 필드 재료 | 보스 재료 |
|---|---|---|---|
| 수도 외곽 평원 | 초반 적응 | 야생의 가죽 | 들판 포식자의 송곳니 |
| 폐광 지대 | 강화/광물 | 폐광 철편 | 폐광 골렘의 파편 |
| 오염된 수로 | 포션/저항 | 오염된 점액 | 수로 군주의 점액핵 |
| 무너진 초소 | 장비 파편 | 낡은 기사 문장 | 타락 기사장의 문장 |
| 고대 성벽 잔해 | 고난도/균열 | 균열 파편 | 균열 파수꾼의 결정 |

## 몹 수량

필드별 일반몹 2종, 정예몹 1종, 필드보스 1종.

## 보스

1차는 전부 바닐라 강화형으로 간다. 외부 보스 모델은 2차 확장 후보.

## 필드보스

필드 리젠형. 매칭으로 빼지 않는다. 기여도 기반 보상.

## 시즌보스

입장형/파티형. 시즌보스 1/2/3은 활성 유저 기준 70%/50%/40% 클리어를 목표로 한다. 시즌 최종보스는 10% 목표.
