# 10. 개발 로드맵

> **[STATUS: REFERENCE]** — 개발 Phase 참조. 공식 시즌 방향은 `../final_master_plan.md`가 우선.

> 확정 기준: `final_master_plan.md`의 "개발 로드맵" (2026-05-21)

## Phase 1. 기반

플러그인 정리, DB, 권한, 월드 구조, EmpireRPG 기본 명령어.

## Phase 2. 디스코드 인증

약관 동의, 닉네임 입력, 임시 화이트리스트, 코드 연동, 역할 부여.

## Phase 3. 전투 코어

6무기 클래스, 스킬 입력, 쿨타임, 자기 자원(스택/에너지 등), 직업각인.

## Phase 4. 장비 성장

T1 장비, 강화, 잠재(큐브), 강화 수급 로그, 전승 시스템.

## Phase 5. 영지

영지 저장고, 약초 재배지, 광물 채굴기, 공방 가공기.

## Phase 6. 필드/보스

필드 5개, 일반몹, 정예몹, 드랍 시스템, 필드보스 5개, 시즌보스 3개 + 균열왕, 보스 패턴 모듈.

## Phase 7. 웹/디코봇/통계

필드보스 알림, 유저 조회, 관리자 통계, 경제 로그.

## Phase 8. 치장/리소스팩

치장 소유권, 무기 치장 6종, GUI 아이콘, HUD 완성.
