# 10. 개발 로드맵

## Phase 1. 기반

플러그인 정리, DB, 권한, 월드, EmpireRPG 기본 명령어.

## Phase 2. 디스코드 인증

약관 동의, 닉네임 입력, 임시 화이트리스트, 코드 연동, 역할 부여.

## Phase 3. 전투

6무기 클래스, 스킬 입력, 쿨타임, 자기 자원, 직업각인.

## Phase 4. 장비

T1 장비, 강화, 잠재, 로그.

## Phase 5. 영지/마인팜

영지 저장고, 마력 동력, 광맥 생성기, 자동 생산기, 공방 가공기.

## Phase 6. 필드/보스

필드 5개, 일반몹, 정예몹, 필드보스 5개, 시즌보스 3개.

## Phase 7. 웹/디코봇/통계

필드보스 알림, 유저 조회, 관리자 통계, 경제 로그.

## Phase 8. 컬렉션/치장

모험 도감, 치장 소유권, 무기 치장, GUI/아이콘.
