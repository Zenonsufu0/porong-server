## 0. 공통 전제 재확인

- 각인 구조: 직업각인 1 + 공용각인 3 (직업각인 등급 없음, 공용각인만 5등급 — 커먼/레어/에픽/유니크/레전드리).
- DPS 격차 목표 = 레전드리 3슬롯 vs 커먼 3슬롯 **17.5%** (v1 권장안 확정).
- 피해 증가 카테고리: 일반 / 태그 / 조건부 3축. 태그 최대 2개·합연산 우선.
- 상태 체계: 버프 / 디버프(상태이상·제어·표식류).
- T2 5세트 정체성(황성의위광·청풍의추적·황야의개척·홍염의낙인·설원의침묵) 혼합 금지.

## 1. 직업각인 분기 안2 초안 v2 (특수 효과 분기 매트릭스)

### 1-1. 구조 원칙
- 직업각인 1종당 **2~3개 분기** 토글 선택. 레벨 없음·등급 없음 → DPS는 상수, 분기 차이는 **플레이스타일 축**이 됨.
- 분기 간 DPS 편차 **±2%p 이내** 클램프(수치 우열이 아닌 스타일 분기). 넘어가는 분기는 1막 45일차 튜닝에서 강제 재조정.
- 분기 전환 비용: **자유 토글, 단 전투 중 비활성** + **쿨 60초**(세트·룬과 달리 빠른 전환은 허용하되 전투 스위칭 악용 차단).
- 공용각인 3슬롯과의 상호작용: 분기 효과는 **직업각인 전용 카테고리**(`class_engrave_branch`)에 들어가고, 동명 효과가 있어도 **공용각인 합연산 풀과 독립 계산**한다. 예: 대검 "장악—파쇄 분기"가 파쇄 태그 피해 증가를 주어도 공용각인 정밀·핵심·연계 합연산 풀과 별도.
- 중복 차단 규칙 3조:
  - 동일 태그 이중 증폭 금지(분기·공용·세트 중 **최대 2채널**까지만 동일 태그에 기여).
  - 태그 합연산 풀 편입 시 본 문서 4장 +25%p 캡 공유.
  - 조건부 계수(약점·표식·연계)는 분기에서도 발동률 계수(필드 0.55 / 보스 0.70) 동일 적용.

### 1-6. 권장안
- **초안 v2 수용 — 전사 분기 매트릭스 즉시 상세화, 암살자·마법사는 시즌1 튜닝 이전 1차 수치 확정**.
- 분기 전환 쿨 60초 + 전투 중 비활성은 **기본선**, 시즌1 중반 체감 이상 시 15~30초로 완화 가능.
- 분기 간 DPS 시뮬은 시즌1 35일차 CP3(에픽 활성화율 점검)와 동시에 수행.

### 1-7. 리스크 2
1. **태도 "잔심—냉각" 분기 + 각성의 호흡 + 공용각인 연계 숙련 쿨감 기여 합산이 -20% 클램프 벽에 상시 붙는 현상** — 마법사 "스태프—광역 지배 냉각" 분기도 동일 리스크. 클램프 **-20%는 여전히 유지**하되, 분기 쿨 감소 기여는 합산 편입임을 seed에 명시해 개별 항목 -2s 등 명목값이 실효되지 않는 구간을 테스트 빌드에서 실측.
2. **분기 C(3분기 보유 각인)가 "운영 편의 분기"로 몰려 파밍 효율 스타일이 정답화되는 위험** — 대검 장악·창 압박·마법사 스태프가 해당. 분기 C는 "유틸" 아닌 "다른 플레이 페이스"로 차별화하고, 시즌1 45일차 분기 선택 분포가 40% 초과 시 보정.

## 2. 각성의 호흡 공격 스킬 쿨감 -15% 타이트 캡 재검토

### 2-1. 현재 결정
- 공격 스킬 쿨감 합산 상한 **-20%** (v1 권장: 마법사 과보정 우려로 -15% 타이트 클램프 검토).
- 각성의 호흡 단독 공격 스킬 기여 -0.5%/Lv (5Lv -2.5%).
- 세트·룬·잠재·직업각인 분기 합산 시 실제 도달 가능치: 전사 -10~14% / 암살자 -12~16% / 마법사 -14~18%.

### 2-2. 수치 시나리오 비교

| 캡 | 마법사 실효 DPS 증분 | 딜 각인 5Lv 기대치 대비 비율 | 딜 각인 다양성 영향 | 시즌1 튜닝 여유 |
|---|---|---|---|---|
| -18% | +3.2~3.8% | 80~95% (딜 각인 하한 +4% 근접) | 각성의 호흡이 딜 각인 대체재로 부상 | 좁음 (12→18 튜닝 폭 6p) |
| **-15%** | +2.6~3.1% | 65~78% | 딜 각인 하위(기동 전술 등) 대비 우세, 상위 딜 각인(전투 집중 등) 대비 열세 — 건강 | 적정 (12→15, 15→18 양방향 3~6p 확보) |
| -12% | +2.1~2.5% | 52~63% | 각성의 호흡이 생존 포지션으로 회귀 | 넓음 (12→18 6p 전체 활용) |

### 2-3. 마법사 단일 캡 추가 필요성
- 마법사만 -15%·타 직업 -20% 이원화는 **가능하지만 비권장**. 직업별 개별 캡은 seed·검증·디스코드 `/내각인` 표기 복잡도를 증가시키고, "같은 각인인데 직업마다 다른 수치" UX 혼란 유발.
- 대신 **전 직업 -15% 단일 캡**으로 하향, 공격 쿨감이 필수인 마법사 외 직업(태도·쌍검)에도 동일 가드로 충분. 시뮬상 전사·암살자는 -15% 하한에 닿지 않거나 1~2%p 여유(딜 손실 0.3~0.6%)에 그쳐 영향 미미.

### 2-4. 이동·방어기 쿨감
- 이동·방어기 쿨감 -1%/Lv (5Lv -5%)는 **기존 -20% 유지 권고**(생존 카테고리). 공격 쿨감과 합산하지 않으므로 타이트 캡 하향 영향 없음.

### 2-5. 장단점 표

| 안 | 장점 | 단점 |
|---|---|---|
| -18% 완화 | 마법사 장쿨 핵심기 회전율 여유, 딜 스타일 다양화 | 딜 각인 다양성 붕괴 직전(각성의 호흡 = 준 딜 각인) |
| **-15% 채택** | 각성의 호흡 생존 정체성 보존, 딜 각인 5종 + 운영 3종 다양성 유지, 튜닝 여유 3~6%p 양방향 확보 | 마법사 장쿨 캡 상시 터치 — 일부 "쿨감 빌드" 플레이어가 체감 천장 느낌 |
| -12% 강화 | 각성의 호흡 완전 생존 포지션, 딜 각인 완전 보호 | 쿨감 빌드 자체가 의미 없어짐 — 마법사·태도 "쿨감 한 줄" 플레이스타일 사라짐 |

### 2-6. 권장안
- **전 직업 공격 스킬 쿨감 합산 상한 -15% 단일 캡으로 하향** (기존 -20%에서 -5%p 타이트화).
- 이동·방어기는 별도 풀 유지, 상한 별도 설정 필요 없음.
- 시즌1 튜닝 단계(1막 21일차 / 45일차)에서 **마법사 DPS 실측이 전사 대비 +5%p 이내**면 유지, 초과 시 -13%까지 추가 하향 가능.

### 2-7. 리스크 2
1. **태도·쌍검·마법사 완드 "쿨감 한 줄" 플레이어가 공용각인 연계 숙련·기동 전술과 각성의 호흡을 동시 장착할 때 캡에 상시 닿아 추가 쿨감 각인 장착 의미 상실** — UI에서 "합산 상한 도달" 명시 + 쿨감 합산 바 시각화를 디스코드 `/내각인` 응답에도 포함해 체감 천장을 납득 가능하게 안내.
2. **-15% 하향 시 각성의 호흡을 "자원 회복 +10%"만 보고 장착하는 저Lv 유저와 합산 캡을 상시 터치하는 고Lv 유저의 체감 괴리 확대** — Lv1~3에서는 쿨감 기여가 미미(-0.5~-1.5%)라 캡 체감 없음, Lv4~5 진입 시점에 튜토리얼 토스트 1회 제공 권고.

- 레전드리 드랍 채널: **4막 극상위 보스 전용**, 경매 채널만 유통.
- 45일차 = 시즌1 1.5개월 시점, 선발대 규모 8~25명(활성 500명 서버 기준 2~4%).

### 3-2. 월별 환산
- 45일차 ÷ 1.5 ≈ 30일(1개월) 기준 환산 (선발대가 4막 극상위 도달한 후반 누적치 비례 가정, 도달 시점은 실제 3막 후반~4막 초입 30~40일차).
- **월별 기대치**:

| 시나리오 | 45일차 누적 | 월 평균 | 월간 경매 등록 비율 | 월간 경매 등록 권수 |
|---|---|---|---|---|
| 보수 | 3.9권 | 2.6권 | 70% | 1.8권 |
| **기준** | **10.8권** | **7.2권** | **65%** | **4.7권** |
| 상한 | 28.8권 | 19.2권 | 60% | 11.5권 |

- 경매 등록 비율은 "선발대 자기 소비 비율"을 제외한 값(레전드리 3슬롯 활성화 30권 필요 → 자기 소비 우선).

### 3-3. 월 공급 하한·상한 권장
- **월 공급 하한 = 3권** (2권 이하 시 경매 프리미엄 폭주, 4막 극상위 드랍률 상향 트리거).
- **월 공급 상한 = 12권** (13권 이상 시 레전드리 변별 가치 붕괴, 드랍률 하향 트리거).
- 3~12권 범위 내에서는 자연 변동 허용, CP4(45일차) 누적 10.8권 ±50%(5.4~16.2권) 범위에 부합.

### 3-4. 레전드리/유니크 평균가 비율 10배 초과 방지
- v1 권장 지표(비율 10배 초과 = 재조정 시그널 / 3배 미만 = 변별 붕괴)를 월간 공급 모수와 결합.
- 월 공급 3~7권 구간: 비율 7~10배 예상(레전드리 200만G / 유니크 30만G).
- 월 공급 8~12권 구간: 비율 5~7배로 안정화.
- 월 공급 13권+: 비율 3배 이하로 하락 위험 → 드랍률 30% 하향 자동 트리거.

### 3-5. 담합 방어 실효성
- v1 제안 "동일 계정 주 2회+ 가중치 0.3" 규칙 실효 검증:
  - 월 7권 기준, 1인이 5권 이상 독점하려면 주 1권 이상 낙찰 필요.
  - 주 2회+ 가중치 0.3이면 낙찰 확률 대폭 감소(입찰 보정 계수 0.3배) → 실효적으로 월 4권 이상 독점 차단.
  - 단 담합 파티(2~3인 공모 분산 낙찰) 대응은 추가 필요 → **동일 IP·동일 파티 월 5권+ 추가 가중치 0.4** 보완.

### 3-6. 장단점 표

| 축 | 장점 | 단점 |
|---|---|---|
| 월 하한 3권 | 레전드리 프리미엄 유지, 시즌1 후반 도전 동기 유지 | 선발대 풀 활성화(30권) 불가능 — 시즌1 내내 미완성, 시즌2 이월 |
| 월 상한 12권 | 레전드리 유통 활성화, 경매 건수 확보 | 12권 넘어가면 변별 가치 급락 — 자동 드랍률 조정 필수 |
| 담합 가중치 강화 | 독점 방어, 신규 유저 진입 장벽 완화 | 합법적 고과금 낙찰자도 가중치 페널티 — 약관 동의 명문화 필요 |

### 3-7. 권장안
- **시즌1 월 공급 기준선 5~8권 / 하한 3권 / 상한 12권 가이드 확정**.
- `auction_daily_summary.category_path='engraving/rarity/legendary'` 집계로 일간·주간 모니터링.
- 담합 방어는 v1 규칙 + "동일 IP·동일 파티 월 5권+ 가중치 0.4" 보완.

### 3-8. 리스크 2
1. **선발대 10명 이하 서버(소규모 서버)에서 월 3권 미달 = 경매 미발생** — 드랍률 자동 상향이 있어도 선발대 자체 인원이 적으면 공급 확대 불가. 소규모 서버 전용 "레전드리 각인서 길드 교환소 월 1권" 보조 채널 고려 필요.
2. **상한 12권 돌파 시 자동 드랍률 30% 하향이 선발대 심리적 타격** — "나는 뽑혔는데 다음 사람은 확률이 깎였다" 체감 나쁨. 하향은 보스 재조정 패치 단위로 묶어 적용하고 공지 2주 유예 권고.

## 4. 태그 합연산 +25%p 캡 실측

### 4-1. 태그 시스템 재확인
- 태그 최대 2개(공격 1타당).
- 태그 피해 증가는 **합연산 우선** — 같은 태그에 기여하는 각인·세트·상태 효과를 모두 더해 `1 + Σ` 계산.
- 태그 간 독립 곱연산(태그 A × 태그 B)은 허용, 동일 태그 내 곱연산은 금지.

### 4-2. 빌드 시뮬(태그 "정밀" 기준 최대 투입)

레전드리 3슬롯 + T2 3피스 세트 + 상태 태그 2개 조합에서 "정밀" 태그 피해 증가가 모일 수 있는 최대치를 적산.

| 채널 | 수치(정밀 태그 합연산 풀 기여) | 비고 |
|---|---|---|
| 공용각인 "정밀 숙련" 5Lv 레전드리 | +7.5% × 1.90 = +14.25% | 레전드리 5Lv 배율 1.90 적용 |
| 공용각인 "핵심 증폭" 5Lv 레전드리 (핵심 태그 동시 장착 시 정밀에는 기여 없음) | 0 | 서로 다른 태그라 풀 분리 |
| 공용각인 "연계 숙련" 5Lv 레전드리 (연계 태그) | 0 | 풀 분리 |
| T2 세트 "청풍의추적" 3피스 (정밀 태그 +6% 가정) | +6.0% | 세트 기여 |
| T2 세트 "설원의침묵" 3피스 (정밀/표식 이중 편성 시) | +4.0% | 혼합 세트 편성 |
| 상태 태그 "약점 노출" (디버프) | +3.0% | 상태 기여 |
| 상태 태그 "집중 표식" (버프) | +2.5% | 버프 기여 |
| 직업각인 분기 기여(분기 A 정밀 +6%) — 독립 카테고리 | 독립 | 본 합연산 풀에 들어오지 않음(1-1 규칙) |
| **정밀 태그 합연산 총합** | **+29.75%p** | 캡 +25%p 초과 |

- 단일 캐릭이 정밀 태그만 집중 투자하는 극단 빌드에서 **+29.75%p** 도달 가능. 캡 +25%p는 **4.75%p 초과**되어 실효 잘림.
- 실제 대다수 빌드는 태그 2개 분산(정밀 + 핵심 또는 정밀 + 연계)이므로 각 태그 +18~22%p 수준에 머무름. 캡은 **소수 극단 빌드만 제약**.

### 4-3. 극단 빌드가 캡에 걸리는 조건
- 조건 1: 공용각인 3슬롯 중 2슬롯을 동일 태그(정밀 2·핵심 2 등) 투자 → 희귀 조합.
- 조건 2: T2 세트 혼합 편성(3+3)으로 같은 태그에 세트 기여 2채널 중첩 → 혼합 편성 자체가 드물지만 가능.
- 조건 3: 상태 태그 2개 동시 유지 → 지속시간·쿨다운·조건부 계수 영향 고려 시 실제 도달률 0.6 정도.
- 세 조건 동시 성립은 상위권 정예 유저 일부에서만 발생.

### 4-4. 캡 수치 비교

| 캡 | 도달 조건 | 상위권 제약 정도 | 일반 유저 영향 | 플레이스타일 |
|---|---|---|---|---|
| +20%p | 상위권 빌드 상시 터치 (실효 잘림 매주 경험) | 강함 — 극단 빌드 설계 의욕 저해 | 영향 적음 (일반 빌드는 +18%p 이하) | 태그 집중 빌드 소멸 |
| **+25%p** | 극단 빌드만 +5%p 내외 초과 (월 1~2회 체감) | 적정 — 태그 집중 빌드는 "5%p 포기하고도 재미" | 영향 없음 | 태그 집중 빌드 선택 여지 유지 |
| +30%p | 극단 빌드도 실효 가능 (+29.75%p 그대로 반영) | 약함 — 캡의 의미 거의 없음 | 영향 없음 | 정밀·핵심 극단 몰빵이 수학적 최적 → 빌드 획일화 |

### 4-5. 태그 2개 제한과 캡 상호작용
- 태그 최대 2개 제약이 **캡의 효용을 이미 보호**. 태그 3개 허용이었다면 +25%p로는 부족했을 것.
- 2개 제약 하에서 캡 +25%p는 "태그 다양화(정밀 + 핵심 분산)"와 "태그 집중(정밀 2회)"의 트레이드오프를 만든다 — 집중 빌드는 일부 실효 잘림을 감수하고 단일 태그 방어력 약점(면역 등) 공격에 특화.

### 4-6. 장단점 표

| 캡 | 장점 | 단점 |
|---|---|---|
| +20%p | 상위권 격차 완화, 평균 DPS 평탄화 | 태그 집중 빌드 완전 소멸 — 스타일 다양성 훼손 |
| **+25%p** | 태그 집중 빌드 생존, 상위권 제약 적정(극단 빌드 5%p 잘림), 일반 유저 무영향 | 극단 빌드 설계 시 "잘리는 5%p" 납득 UX 필요 |
| +30%p | 모든 빌드 실효 반영 | 정밀·핵심 극단 몰빵이 수학적 최적 → 빌드 획일화, T2 세트 혼합 편성 폭발 |

### 4-7. 권장안
- **+25%p 캡 유지 확정**.
- UI에 "태그 합연산 현재치 / 캡" 바 표기 권고 (예: 정밀 22.8 / 25.0). Shift+좌클릭 > 전투 > 각인 > 태그 현황 탭 추가.
- 시즌1 45일차 튜닝에서 태그 합연산 평균 실측이 +18%p 미만이면 +22%p로 하향, +22%p 이상이면 +25%p 유지 또는 +27%p 소폭 완화.

### 4-8. 리스크 2
1. **"잘리는 5%p"를 모르고 극단 빌드를 완성한 유저의 패배감** — 각인 Lv4·5 진입 시 UI 경고 토스트 + 디스코드 `/내각인` 응답에 "태그 캡 초과 잘림 -5.0%p" 명시 필수.
2. **직업각인 분기가 독립 카테고리라 본 캡을 우회** — 분기 기여가 태그 풀에 들어오지 않는 규칙(1-1) 덕에 실제 태그 기여는 더 크다. 분기 기여가 시즌1 45일차 시뮬에서 +5%p 초과 시 분기도 태그 풀 편입으로 규칙 전환 검토.

## 5. 오픈 질문 (v2 남김)

1. 직업각인 분기 전환 쿨 60초가 전투 외 상황에서 과도한가(10초·30초 비교 필요).
2. 공격 스킬 쿨감 -15% 캡을 PvP에도 적용할지(현재 PvP는 별도 공식 미정).
3. 레전드리 각인서 소규모 서버(활성 100명 이하) 보조 채널(길드 교환소 월 1권) 도입 여부.

5. 직업각인 분기 기여가 태그 풀에 편입될 경우 본 문서 4-2 시뮬 재계산 필요.
