## 1. 등급 체계 재확인

### 1-1. 현행 공식 = 5단계
- 공식 등급 명칭: **커먼 / 레어 / 에픽 / 유니크 / 레전드리**.
- 성능 계수(DPS 격차 문서 C안 17.5% 확정 기준): ×1.00 / ×1.225 / ×1.45 / ×1.675 / ×1.90.
- 마스터 플래닝·DPS 격차 v1·매트릭스 v1·공용각인 마스터 리스트·드랍 확률 v1 모두 5단계로 정렬되어 있다.

### 1-2. 사용자 요청문의 "6단계(언커먼 포함)" 표기 처리
- 본 문서는 공식 5단계로 전개.
- 6단계 확장(커먼 / 언커먼 / 레어 / 에픽 / 유니크 / 레전드리)은 아래 파급이 모두 발생하므로 **단독 문서 범위 초과**로 판단.
  - DPS 격차 공식(등급당 배율 +0.225)의 구간 재분할 → 5구간 → 6구간 재계산.
  - 경매 시세 구간 재배치(레어 5k~15k, 에픽 40k~120k 등 경계 이동).
  - 마스터 플래닝·DB 스키마(`engraving_grade` enum)·디스코드 봇 `/드랍` 응답 포맷 전부 갱신.
- 오픈 질문 §9-1로 이관.

### 1-3. 본 문서 전제 확정
- 이하 모든 확률·시뮬·DPS 재검증은 **5단계(커먼/레어/에픽/유니크/레전드리)** 전제.
- v1 표의 "유니크" 칸만 수정 대상이며, 레어·에픽·레전드리 칸은 기존 값을 유지·재검증한다.

## 2. 유니크 허용 위치의 기존 기준

### 2-1. 매트릭스 v1의 기조
- `poro_boss_drop_grade_matrix_v1.md` §2-1: **유니크 각인서 = 3막 결전 전용, 레전드리 = 4막 극상위 전용**.
- 1막 시점 유니크·레전드리 드랍 0 명시.
- v1 §1-1: "2막(메인 20)까지 유니크·레전드리 0 유지"는 매트릭스 기조의 자연 확장.

### 2-2. 메인 20에 유니크를 넣을 경우의 충돌
- 메인 20 = 2막 진입 직후 보스. 매트릭스 §2-1의 "유니크 = 3막 결전 전용"과 **정면 충돌**.
- 단, 레전드리가 이미 v1에서 "3막 결전 첫 허용"으로 매트릭스 v2 확장을 전제한 상태이므로, 같은 논리로 **"유니크를 2막 메인 20에 극저확률 선허용"** 역시 매트릭스 v2 확장으로 흡수 가능.
- 흡수 시 매트릭스 v2는 "유니크 = 2막 메인 20에 시드 드랍 + 3막 결전 본격 드랍"으로 재정의되어야 한다.

### 2-3. 3막 결전 유니크 5.0%의 재검증 필요 사유
- v1 확정 5.0%는 "분기 A/B 각 1회 + 주 1회 반복 의뢰 + 45일차 80명 돌파 가정"에서 서버 유통 약 20권을 목표로 한 값.
- 메인 20에 유니크를 추가로 허용하면 유니크 총 공급이 증가 → 3막 결전 5.0%의 "희소성" 의미가 흐려짐.
- 따라서 메인 20 허용 여부는 **3막 결전 유니크 확률 재산정과 한 묶음**으로 결정해야 한다.

## 3. 4안 비교 (A / B / C / D)

### 3-1. A안 — 메인 20 유니크 0.5% + 3막 결전 유니크 5.0% (v1 유지 + 메인 20 시드 추가)

**수치**

| 보스 | 레어 | 에픽 | 유니크 | 레전드리 |
|---|---|---|---|---|
| 메인 20 | 35% | 4.0% | **0.5%** | 0 |
| 3막 결전 A | 60% | 18% | 5.0% | 0.8% |
| 3막 결전 B | 60% | 18% | 5.0% | 0.8% |

**장점**
- 2막 초입에 "유니크라는 새 등급이 열렸다"는 체감 제공. 에픽(4%) → 유니크(0.5%) → 레전드리(0)의 **선명한 등급 계단**.
- 메인 20은 1회성 보스이므로 인당 기대값 0.005권, 45일차 300명 기준 **서버 유통 약 1.5권** — 초저공급으로 3막 결전 5.0%의 희소성을 거의 침식하지 않음.
- 1회성 드랍이라 "반복 파밍 폭주" 리스크 없음.

**단점**
- 0.5%는 1회성 기회라 "보이지 않는 보상" 리스크. 45일차 300명 중 1~2명만 체감.
- 매트릭스 v2 확장이 두 축(유니크 + 레전드리) 모두 필요.

**45일차 공급**: 유니크 메인 20 1.5 + 3막 결전 20 = 약 **21.5권**.

## 4. 권장안 — **A안 (메인 20 유니크 0.5% + 3막 결전 유니크 5.0%)**

### 4-1. 선택 근거
1. **등급 계단이 가장 선명**: 메인 20에서 에픽 4% → 유니크 0.5% → 레전드리 0의 계단이 형성되어 2막 진입 직후 유니크라는 새 등급의 존재를 플레이어에게 **정보로서 각인**시킨다. 체감 확보는 약하지만 "다음 목표(3막 결전)" 인지 효과가 크다.
2. **3막 결전 5.0% 유지로 프리미엄 보호**: 3막 결전이 여전히 주 공급처(20/21.5 ≈ 93%)이므로 결전 보스의 서사·기능 포지션이 흔들리지 않는다.
3. **1회성 기회라 폭주 리스크 최소**: 메인 20 0.5%는 인당 평생 1회. 파밍 악용 불가, 45일차 서버 유통 약 1.5권으로 경매 시세를 왜곡하지 않음.
4. **매트릭스 v2 확장 범위가 관리 가능**: 이미 레전드리(4막→3막 확장)로 매트릭스 v2가 예정되어 있으므로, 유니크(3막→2막 시드)도 같은 문서에서 흡수 가능. 한 번의 갱신으로 처리.
5. **DPS 17.5% 전제와 정합**: 유니크 계수 ×1.675가 그대로 유효. 아래 §6 재검증 참조.

### 4-2. 차점안 — C안 보류 사유
- C안은 2막 유니크 유동성 확보 장점이 있으나 3막 결전 5.0% → 3.5% 하향이 **v1 확정치를 수정**하고 매트릭스 v2 수치 재튜닝 범위를 넓힌다.
- 45일차 유니크 메인 20 4.5권은 1회성 1.5%(A안 대비 3배) 확률이라 "초반 운빨 선발대 독점" 리스크도 A안 대비 크다.
- **45일차 지표 관찰 후 유니크 유동성이 부족하면 C안 또는 C안과 A안의 중간값(메인 20 1.0% + 3막 결전 4.5%)으로 v3 전환 옵션** 제시.

| **3막 결전 A (`황성 봉인 기사단장의 잔영`)** | 100% (확정 1권) | 60% | 18% | 5.0% | 0.8% |
| **3막 결전 B (`이름을 삼킨 계약의 증인`)** | 100% (확정 1권) | 60% | 18% | 5.0% | 0.8% |

> 본 표는 5단계(커먼/레어/에픽/유니크/레전드리) 전제. 6단계(언커먼 포함) 확장은 오픈 질문 §9-1에서 별도 결정.

## 6. 45일차 공급 재시뮬레이션 (A안 기준)

### 6-1. 모집단 (v1 기준 승계)
- 활성 유저 500명 서버.
- 메인 20 돌파자: 보수 250 / 기준 300 / 상한 360.
- 3막 결전 돌파자: 보수 50 / 기준 80 / 상한 120.
- 3막 결전 반복 시도 평균: 4~8회(세력 의뢰 주 1회 × 4~6주).

### 6-2. 등급별 45일차 서버 공급량

| 등급 | 메인 20 경로 | 3막 결전 A·B 합산 | 기타 누적(1~2막) | **총 공급** |
|---|---|---|---|---|
| 커먼 | 300권 확정 | 80권 확정 + 반복 ~300 | ~2,000권 | 약 2,760권 |
| 레어 | 105권 | ~240권 | ~600권 | 약 945권 |
| 에픽 | 12권 | ~72권 | ~25권 | 약 109권 |
| **유니크** | **약 1.5권** (300 × 0.005) | **약 20권** (400 × 0.05) | 0 | **약 21.5권** |
| 레전드리 | 0 | 약 3.2권 (400 × 0.008) | 0 | 약 3.2권 (기준) |

### 6-3. 유니크 상대 희귀도
- 에픽 : 유니크 : 레전드리 = 109 : 21.5 : 3.2 ≈ **34 : 7 : 1**.
- 등급 간 희귀도 비율이 약 5배씩 벌어지는 **지수 계단** 형성. 포로 서버 "완만 기조"와 "선명한 계단" 원칙 양쪽 충족.
- 유니크/레전드리 비율 = 6.7배 → v1 권장 수동 튜닝 트리거(10배 초과 시 하향 / 3배 미만 시 상향) 안전 구간 내.

### 6-4. 선발대 3슬롯 완성 기여
- 유니크 3슬롯 = 3권 필요. 45일차 21.5권은 **서버 7명 수준이 유니크 3슬롯 완성 가능**한 공급량.
- 레전드리 3슬롯 = 3권 필요. 45일차 3.2권(3막)  + 4막 극상위 10.8권(기준 시나리오) = 14권 → **서버 4~5명 수준**.
- 일반 유저 목표선 "유니크 3 + 부분 레전드리": 유니크 3권 확보 비용 × 7명 기준 +a → **일반 유저 상위 10% 정도가 45일차에 도달** 가능. 시즌1 45일차 설계 의도와 정합.

### 6-5. v1 대비 변화
- v1 유니크 20권 → v2 21.5권(+7.5%). 미미한 증가.
- 유니크 메인 20 1.5권이 **2막 경매 유니크 시장의 첫 씨앗** 제공 → 3막 결전 돌파 전에도 유니크 시세가 형성되기 시작.
- 레전드리 공급량은 변화 없음(v1 3.2권 유지).

## 7. DPS 17.5% 격차 재검증

### 7-1. "레전3" 정의 재확인
- DPS 격차 v1 §2-2의 "레전드리 3" = **공용각인 3슬롯 모두 레전드리 5Lv**.
- 유니크가 추가되어도 레전드리 3 정의는 불변. 유니크 3은 여전히 "레전드리 3의 바로 아래 등급 조합"(+14.3%p).

### 7-2. 유니크 확률 변화가 DPS 격차에 미치는 영향
- DPS 격차는 **등급별 계수**에서 결정되며, 드랍 확률 수치와 직접 관계가 없다.
- A안으로 유니크 공급이 20 → 21.5권(+7.5%)로 늘어나는 건 **유니크 3슬롯 완성 접근성**을 소폭 개선할 뿐, 레전드리 3 vs 커먼 3 격차 17.5%는 동일.

### 7-3. 조합 시나리오 재산정 (A안 적용 후)

| 조합 | 45일차 달성 가능 인원 (추정) | DPS 배율 (커먼 3 = 100) |
|---|---|---|
| 커먼 3 | 대다수 | 100.0 |
| 레어 3 | 대다수 | 103.8 |
| 에픽 3 | 상위 40~50% | 107.8 |
| **유니크 3** | **서버 ~7명 (A안 직접 공급 기준)** | **112.0** |
| 유니크 2 + 레전드리 1 | 서버 ~3~4명 | 114.3 |
| 레전드리 3 | 서버 1~2명 | 120.1 |

- 유니크 3슬롯 달성 인원이 v1(약 6~7명) 대비 실질 변화 없음(A안 증분 미미).
- **17.5% 목표 유지**. A안 채택 후에도 재튜닝 불필요.

### 7-4. 마법사 단일 쿨감 캡(-12%) 상호작용
- DPS 격차 v1 §5-3에서 마법사 룬/스태프 격차 +22~24% 지적 → 단일 캡 -12% 선도입 권고.
- 유니크 공급 증가와 마법사 격차는 독립 축이지만, 유니크 3슬롯 마법사 빌드가 실제 등장 가능해지는 시점이 빨라지면 **단일 캡 도입 타이밍도 45일차 이전으로 앞당길 근거**가 된다.
- 권고: 마법사 단일 쿨감 캡 -12%를 **시즌1 30일차 정기 점검 시 선도입**.

## 8. 경매 시세 영향

### 8-1. 유니크가 레어~레전드리 스프레드를 메우는 효과
- v1 예측 시세(DPS 격차 v1 §7-2): 레어 5k~15k · 에픽 40k~120k · 유니크 200k~600k · 레전드리 1,500k~5,000k.
- 에픽~유니크 구간 배율: 200k/120k ≈ 1.67~5배, 유니크~레전드리: 1,500k/600k ≈ 2.5~25배 → **유니크~레전드리 사이 절벽**이 구조적으로 존재.
- A안 채택 후 유니크 공급 +1.5권(메인 20)으로 **2막 경매 유니크 하단 200k → 150k~170k로 -15~-25% 완화**. 유니크~레전드리 절벽 축소.
- 3막 결전 유니크 공급은 v1과 동일 → 유니크 상단(600k)은 유지.

### 8-2. 유니크 평균가 / 레전드리 평균가 비율
- v1 예측 비율: 레전드리/유니크 = 2.5~25배. 트리거 구간(3~10배) 내.
- A안 채택 후 유니크 하단 완화로 비율 중앙값 **3.8배 → 4.2배** 소폭 증가. 여전히 안전 구간.

### 8-3. `auction_engraving_summary` 카드 제안
- 기존 DPS 격차 v1 §7-3에서 제안된 집계 테이블에 **"메인 20 유니크 유통 권수"** 45일차 지표 카드 추가 권고(data-schema 후속).
- 메인 20 유니크 1.5권이 서버에서 **실제로 드랍되는지 드랍 로그로 확인**하는 트리거.

## 9. 오픈 질문

1. **등급 체계 6단계 확장 여부** — 사용자 요청문의 "언커먼"을 공식 등급으로 추가할지 결정. 6단계 전환 시 DPS 격차 문서·마스터 플래닝·매트릭스 모두 재작성 필요.
2. **유니크 각인서의 DPS 기여도(+X%p) 확정** — DPS 격차 v1은 유니크 ×1.675 계수만 가정. 실측 시뮬(카테고리 가중 + realization 0.8) 재검토 필요.
3. **유니크 옵션 3 슬롯 조합 최대 DPS** — 공용각인 마스터 리스트 15종 × 카테고리 가중 기준 유니크 3슬롯 최적해 산출(정밀+핵심+연계 태그 합연산 캡 도달 여부 확인).
4. **마법사 단일 쿨감 캡 -12% 선도입 타이밍** — 시즌1 30일차 정기 점검 vs 45일차 지표 기반.
5. **A안 메인 20 0.5% → C안 1.5% 전환 트리거 조건** — 45일차 유니크 유통 누적이 목표치(21.5권)의 -40% 미만일 때 자동 상향 검토 여부.
6. **매트릭스 v2 갱신 타이밍** — v1 확정 즉시 갱신인지, v2 문서(본 문서) 확정 후 한 번에 갱신인지.
7. **2막 경매 유니크 첫 유통이 선발대 독점으로 흘러가지 않을 담합 방어** — 메인 20 드랍 1.5권은 상위 1~2명이 전담 낙찰할 수 있는 규모, 담합 방어 규칙 강화 여부.

## 10. 장점 / 위험요소 / 조정 제안 요약

### 10-1. 장점
1. 메인 20에 유니크 0.5% 시드를 넣어 **2막 진입 직후 유니크 등급의 존재를 플레이어에게 인지**시킨다.
2. 3막 결전 5.0% 유지로 프리미엄과 서사 포지션 모두 보존.
3. 45일차 유니크 공급 21.5권으로 **서버 상위 7명 수준이 유니크 3슬롯 달성** 가능 — 선발대~일반 상위 중간 타깃 설정 정합.
4. 1회성 1.5% 확률이 아닌 0.5%로 **운빨 선발대 독점 리스크 최소화**.
5. DPS 17.5% 격차 전제 불변 → 재튜닝 없음.

### 10-2. 위험요소
1. **매트릭스 v2 갱신 범위 확장** — 유니크(2막 시드) + 레전드리(3막 첫 허용) 2축 동시 갱신 필요.
2. **메인 20 1.5권의 체감 낮음** — 300명 중 1~2명만 직접 경험. 일반 유저 다수는 여전히 유니크를 "3막 결전 드랍"으로만 인지.
3. **2막 경매 유니크 첫 유통이 선발대 독점 유인** — 공급량 1.5권은 담합 방어 트리거(동일 계정 주 2회+ 가중치 감쇠)가 작동하기 전 독점 가능성.
4. **마법사 단일 빌드 격차 초과가 유니크 공급 증가로 앞당겨짐** — 45일차 전 단일 캡 도입 타이밍 재검토 필요.

### 10-3. 조정 제안
1. **A안 채택** — 메인 20 유니크 0.5%, 3막 결전 A·B 유니크 5.0%·레전드리 0.8%.
2. **매트릭스 v2 작성 시 유니크·레전드리 2축 동시 확장 반영**.
3. **마법사 단일 쿨감 캡 -12%를 시즌1 30일차 선도입**.
4. **2막 유니크 경매 담합 방어 재강화** — v1 §4-2 규칙(주 1회 낙찰 상한)을 **유니크 각인서에도 확대** 적용.
5. **45일차 유니크 유통 카드 신설** — `auction_engraving_summary`에 "메인 20 유니크 권수" 컬럼 추가.
6. **C안 전환 트리거 사전 정의** — 45일차 유니크 유통이 목표치의 -40% 미만이면 C안(메인 20 1.5% + 3막 결전 3.5%)으로 v3 전환.
