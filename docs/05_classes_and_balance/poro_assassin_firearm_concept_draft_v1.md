## 0. 개요

암살자 3무기(단검 · 쌍검 · 총류) 중 **"총류"의 내부 구조**를 정의한다. 총류 장착 시 **권총 6 공용 기반 스킬**과 **저격 3 / 샷건 3 확장 스킬**을 함께 갖는다. 직업각인 선택(저격 분기 = `관측자의 조준선` / 샷건 분기 = `파열의 집행자`)이 확장 3 스킬을 활성화해 두 개의 플레이 스타일을 만든다. 권총 6은 분기와 무관하게 항상 동작하며 **회전 · 기동 · 표식 집행**을 담당한다.

"같은 무기, 전혀 다른 파티 포지션" 구조로 파티 3명 상한 안에서 원거리 단발 폭딜러(저격)와 근접 연속 난사 딜러(샷건) 양 축을 확보한다.

**주의**: 본 문서는 총류 1무기에 한정. 단검(즉발/유지) · 쌍검(고회전/치명누적) 분기 = combat-balance v3 암살자 섹션 참조, 세부 스킬은 별도 content-designer 위임.

## 1. 목표

1. 암살자 총류의 12 스킬 한글명·효과·쿨다운 감각을 1페이지 수준으로 정리한다.
2. 권총 6 공용 기반이 분기 무관 허브 역할을 하도록 배치한다.
3. 저격 · 샷건 두 분기가 **정체성 · 플레이 스타일 · 파티 포지션 전부 상반**되게 설계한다.
4. 키 바인딩 · HUD 3레이어 UI 정합안을 제시한다.
5. 포로 세계관(수도 · 기사단 · 관찰자 · 마법) 안에서 총류 무기의 존재 근거를 확보한다.

## 2. 핵심 규칙 (기존 포로 기준 수용)

- 직업당 12 스킬 원칙 유지.
- HUD 5키 (Q·E·F·우클릭·Shift+우클릭) 중 **Q = 무기 독립 기본 공격** (총류 = "단발 사격").
- 직업각인 2 분기 선택형, ±2%p DPS 편차 클램프.
- 각성의 호흡 공격 쿨감 합산 상한 **-15%** 타이트 캡.
- 태그 최대 2개 · 합연산 +25%p 캡. 피해 증가 카테고리 일반 / 태그 / 조건부 3축.
- 파티 3명 상한. 힐 전담은 포로 기준상 없음. 총류는 힐 전담 아님. 단 샷건 분기에 **소량 자힐/흡혈** 조건부 허용 가능(힐 공백 A+B 조합의 "B 분산"을 훼손하지 않는 수준).
- T2 세트 5종 정체성 유지(황성의위광 · 청풍의추적 · 황야의개척 · 홍염의낙인 · 설원의침묵).
- 분기 전환 = 전투 외 · 쿨 60초 토글.

## 3. 범위

- 권총 6 + 저격 3 + 샷건 3 = 12 스킬 한글명 · 효과 · 쿨다운 감각 (정확 수치는 combat-balance 후속).
- 저격 / 샷건 2 분기의 정체성 · 플레이 스타일 · 파티 포지션.
- 키 바인딩 + HUD 표시 방안.
- 세계관 접점 초안 (기사단 · 관찰자 · 사하르 상인연합 · 아르카논).
- 리소스팩 요구사항 (CustomModelData 예약 구간 제안).

## 4. 비범위

- 정확한 피해 계수 · 쿨다운 초 단위 · 태그 풀 편입 — combat-balance 후속.
- 스킬 핸들러 Java 클래스 · 이벤트 배선 — implementation-reviewer 후속.
- 암살자 단검 · 쌍검 세부 스킬 · 분기 수치 — 본 문서 범위 밖(별도 content-designer 위임 + combat-balance v3 참조).
- T2 세트별 수치 매칭 표 — 6장에서 방향만 제시.

## 5. 총류 무기 정체성 요약

### 5-1. 총류의 공통 톤

- **자원 구조**: "탄창" — 전투 중 보유하는 장전 상태. 기본 사격은 탄창을 소모하지 않지만, **핵심기 · 종결기는 탄 소비**. 재장전(F 또는 특정 스킬) 필요.
- **사거리 구분**: 권총 = 중거리 / 저격 = 장거리 / 샷건 = 근접 (3블록 내 산탄).
- **태그 축**: 권총 = 표식 태그 + 일반 / 저격 = 정밀 태그 + 조건부(약점) / 샷건 = 연계 태그 + 조건부(근접).
- **자원 순환**: 권총 스킬은 **재장전 보조 기믹**을 포함해 저격/샷건 양쪽 모두 지원한다.

### 5-2. 분기 정체성 한 줄 요약

- **저격 분기 (관측자의 조준선)**: 정적 · 준비 · 원거리 · 단발 폭딜. 파티 원거리 역할.
- **샷건 분기 (파열의 집행자)**: 동적 · 근접 · 연속 · 난사. 파티 근접 회전딜러 역할.

## 6. 권총 6 공용 기반 스킬 (분기 무관)

### 6-1. 배분 방식 권장 — 옵션 A (4 능동 + 2 패시브)

HUD 5키 중 **Q = 기본 사격(무기 독립)**이 고정되어 있으므로, 권총 공용 6 스킬 중 4개는 E · F · 우 · Shift+우 슬롯에 배치하고, 나머지 2개는 **패시브(상시 / 자동 발동)** 로 둔다. 이 2개는 분기 확장 3 스킬과 슬롯 충돌하지 않으며, 분기 선택 여부와 무관하게 항상 동작한다.

- **장점**: 분기 활성 시 확장 3 스킬을 E·F·우·Shift+우 중 3칸에 덮어씌울 여유 확보. 권총 능동 4는 "허브", 확장 3은 "특화".
- **단점**: 패시브 2개가 플레이 중 존재감이 옅어질 수 있음 → 시각 피드백(버프 아이콘 · 파티클) 강화로 보완.

**옵션 B(5 능동 + 1 토글) 비교**: 능동 5는 HUD 5키 전부 차지 → 확장 3 활성 시 권총 능동 2개를 비활성화해야 함. 플레이어 체감 "분기 선택 = 권총 일부 상실"로 읽힘 → 권장 X.

**결론**: **옵션 A (4 능동 + 2 패시브)** 를 권장.

### 6-2. 권총 6 스킬 초안

| 번호 | 한글명 | 슬롯 유형 | 효과 개괄 | 쿨다운 감각 |
|---|---|---|---|---|
| P1 | **격발 (Q / 기본 사격)** | Q 기본 | 단발 사격. 탄 소비 없음. 일반 피해. 표식 1스택 부여 확률 25%. | 0.4s GCD |
| P2 | **연쇄 사격** | E / 능동 | 3연발. 3번째 탄은 표식 대상에게 자동 유도(조건부 피해 +8%). | 8s |
| P3 | **회피 롤** | F / 능동 (이동기) | 3블록 전방/후방 롤 회피 + 0.3s 회피 판정 + 롤 종료 시 자동 재장전 1회. 재장전 없이 탄창 1/2 회복. | 10s |
| P4 | **표식 집행** | 우클릭 / 능동 (종결기 후보) | 표식 대상에게 폭발 사격. 표식 스택 수만큼 피해 증폭(최대 3스택, 조건부). | 14s |
| P5 | **속사 자세** | Shift+우클릭 / 능동 (토글 버프) | 4초간 기본 사격 속도 +40% · 이동속도 +10% · 받는 피해 +8%. 지속 중 탄 소비 2배. | 20s |
| P6 | **연화 장전** | 패시브 | 기본 사격 5회 적중 시 다음 핵심기 탄 소비 -1 (최소 0). | 상시 · 내부 쿨 6s |
| P7 | **관성 집중** | 패시브 | 이동 중 사격 시 정확도 +, 정지 중 사격 시 치명 확률 +5%. 두 상태 자동 전환. | 상시 · 전환 쿨 0.5s |

> 번호는 편의상 P1~P7로 표기했으나 스킬 수는 **P1(기본) + P2~P5(능동 4) + P6·P7(패시브 2) = 6개의 "스킬" 이미지**로 집계(Q 기본은 "무기 독립 기본공격"이므로 12 스킬 집계 외로 볼지 내로 볼지 combat-balance와 최종 동기화 필요 — 본 문서는 **P2~P7 = 6 스킬**로 간주).

### 6-3. 권총 6의 역할 요약

- 분기 전환의 **허브**. 저격 선택 시에도 샷건 선택 시에도 "중거리 견제 + 이동기 + 표식 준비"를 항상 제공.
- **회피 롤(P3)** 은 암살자 정체성의 핵심. 저격 분기에서는 "재배치 후 조준", 샷건 분기에서는 "파고들어 난사" 양쪽 다 필수.
- **표식 집행(P4)** 은 권총 단독으로도 종결기 역할 가능 → 분기 미정 초반 레벨 구간에서도 피해 윈도우 확보.

## 7. 저격 분기 (관측자의 조준선) — 확장 3 스킬

### 7-1. 정체성

- **정적 · 준비 · 원거리 · 단발 폭딜**. 긴 준비 시간 후 약점에 고점 단발을 꽂는 플레이 스타일.
- 파티 포지션: **원거리 딜러** (3인 파티의 딜러 슬롯 중 원거리 축). 탱커가 어그로 잡는 동안 후방에서 고점 단발 기여.
- 약점 태그(정밀 태그 + 조건부 약점)를 주력 피해 축으로 삼는다.

### 7-2. 저격 3 스킬 초안

| 번호 | 한글명 | 슬롯 유형 | 효과 개괄 | 쿨다운 감각 |
|---|---|---|---|---|
| S1 | **조준선 전개** | 능동 (선행 차지) | 2초 조준 후 직선 장거리 단발. 조준 중 이동 불가 · 받는 피해 -20%. 적중 시 정밀 태그 피해 +25%p. | 18s |
| S2 | **약점 관측** | 능동 (디버프 부여) | 대상에 "약점 노출" 디버프 부여 (6초). 디버프 대상에게 가하는 모든 피해 +12% (조건부). 파티원 공유. | 22s |
| S3 | **절대 사선** | 능동 (종결기) | 탄창 전량 소모 → 단발 · 초장거리 · 관통. 약점 노출 디버프 대상이면 피해 추가 +30%p (조건부 합산, 태그 캡 별도). | 45s |

### 7-3. 플레이 사이클

1. 권총 P5(속사 자세) · P4(표식) 대신 S1/S2/S3을 슬롯 점유.
2. S2(약점 관측)로 디버프 부여 → S1(조준선 전개) 2초 차지 → 일격.
3. 보스 공개 구간(DPS 체크)에 S3(절대 사선) 맞춰 합계 피해 윈도우 생성.
4. 권총 P3(회피 롤) · P2(연쇄 사격)로 사이사이 생존 + 탄창 보충.

### 7-4. 파티 3명 상한 역할

- 원거리 포지션 **유일**. 탱(전사 대검/창) + 근접 딜(전사 태도 / 마법사 완드 / 암살자 샷건) + 저격 암살자 = 표준 3인.
- S2의 약점 노출 디버프가 **파티 DPS 기여** 역할도 담당 → "이기적 단발 폭딜러"가 아닌 "파티 윈도우 제공자"로 설계.
- 각성의 호흡 -15% 캡 근접도 낮음(장쿨 중심).

## 8. 샷건 분기 (파열의 집행자) — 확장 3 스킬

### 8-1. 정체성

- **동적 · 근접 · 연속 · 난사**. 3블록 내 산탄 · 연속 연타 · 뒤흔들기(넉백/스턴 미세).
- 파티 포지션: **근접 회전딜러**. 전사 태도와 함께 근접 딜 슬롯을 나눠 갖는다.
- 연계 태그(연속 적중 스택)를 주력 피해 축. 조건부(근접 3블록 내) 가산.

### 8-2. 샷건 3 스킬 초안

| 번호 | 한글명 | 슬롯 유형 | 효과 개괄 | 쿨다운 감각 |
|---|---|---|---|---|
| G1 | **근거리 산탄** | 능동 (기본 연타) | 3블록 내 원뿔 산탄. 최대 3체. 연속 적중 시 연계 태그 스택 1 부여(최대 5). | 6s |
| G2 | **파열 돌진** | 능동 (돌격 + 근접 폭발) | 전방 4블록 돌진 후 근접 폭발. 적중 대상 0.5s 경직. 연계 스택 있으면 스택 소모 후 피해 +8%/스택. | 12s |
| G3 | **피 끓는 난사** | 능동 (종결기 · 지속형) | 3초간 초근접 난사(초당 4발). 적중 시 본인 체력 0.5% 흡혈(초당 2% 상한). 지속 중 이동속도 -30% · 방향 전환 가능. | 30s |

### 8-3. 플레이 사이클

1. 권총 P4(표식) · P5(속사 자세) 대신 G1/G2/G3을 슬롯 점유.
2. G2(파열 돌진)으로 근접 진입 → G1(근거리 산탄) 반복해 연계 스택 누적 → G3(피 끓는 난사) 폭발.
3. 권총 P3(회피 롤)로 후퇴 · 재진입 반복. P2(연쇄 사격)은 중거리 견제용으로 간헐 사용.
4. G3의 **소량 흡혈**(초당 2% 상한)이 힐 공백 A+B 조합의 "B 분산" 원칙과 정합 — 샷건 플레이어가 근접 집중 피격 사이클을 자가 관리하도록.

### 8-4. 파티 3명 상한 역할

- 근접 회전딜러. 전사 태도와 근접 딜 슬롯을 공유하되 **태도는 단일 고속 연타, 샷건은 다수 타격 + 흡혈**로 차별화.
- G3 흡혈은 **본인 한정**, 파티 힐 아님 → 촉매서 반려 결정 존중.
- 각성의 호흡 -15% 캡 근접도 **중간** (G1 쿨 6s · G2 12s · 권총 P2 8s 조합 시 쿨감 합산 감시 필요) → combat-balance 시뮬 대상.

## 9. 두 분기의 플레이 스타일 상반 확인

| 축 | 저격 (관측자의 조준선) | 샷건 (파열의 집행자) |
|---|---|---|
| 거리 | 장거리 (10블록+) | 초근접 (3블록 이내) |
| 호흡 | 긴 준비 + 단발 폭딜 | 짧은 사이클 + 연속 난사 |
| 이동 | 정적 (조준 중 이동 불가) | 동적 (돌진 · 회전 · 방향 전환) |
| 탄창 | 단발 집중 · 관통 | 다발 · 넉백 · 흡혈 |
| 태그 | 정밀 + 조건부(약점) | 연계 + 조건부(근접) |
| 파티 역할 | 원거리 딜러 + 파티 윈도우 제공 (약점 노출) | 근접 회전딜러 + 본인 생존 관리 |
| 각성의 호흡 캡 | 낮음 (장쿨) | 중간 (단쿨 다수) |
| DPS 편차 | ±2%p 클램프 (기존 원칙) | ±2%p 클램프 (기존 원칙) |

**충돌 지점 점검**: 두 분기의 파티 포지션(원거리 vs 근접), 호흡(정적 vs 동적), 자원 소비 패턴(단발 vs 연사) 모두 상반. 같은 총류 무기 위에 올라가지만 플레이 감각은 전혀 다른 무기처럼 작동 → 사용자 목표 "완전 상반" 조건 충족.

## 10. 키 바인딩 + HUD 3레이어 정합

### 10-1. 키 바인딩 권장안

권총 6 공용 = **4 능동 (E·F·우·Shift+우) + 2 패시브**. 확장 3 스킬이 활성화될 때 **권총 능동 4 중 3개를 "덮어쓰기" 모드**로 전환.

- **분기 미선택** (초반 레벨 구간): Q 격발 · E 연쇄 사격 · F 회피 롤 · 우 표식 집행 · Shift+우 속사 자세 + 패시브 2 (연화 장전 · 관성 집중)
- **저격 분기 선택**: Q 격발 · E **조준선 전개(S1)** · F 회피 롤 · 우 **약점 관측(S2)** · Shift+우 **절대 사선(S3)** + 패시브 2. **권총 P2·P4·P5 비활성** (표시 회색, 기능 차단).
- **샷건 분기 선택**: Q 격발 · E **근거리 산탄(G1)** · F 회피 롤 · 우 **파열 돌진(G2)** · Shift+우 **피 끓는 난사(G3)** + 패시브 2. **권총 P2·P4·P5 비활성**.

**권장 근거**:
1. **F 회피 롤은 양 분기 공통 유지** — 이동기는 두 플레이 스타일 모두 필수. 롤 종료 시 자동 재장전은 저격(재조준 여유) · 샷건(재진입 여유) 양쪽에 모두 유효.
2. **Q 기본 사격 유지** — HUD 5키 중 Q는 무기 독립 원칙 존중.
3. **패시브 2 상시 유지** — 분기 선택이 "플레이 상실"이 아닌 "플레이 추가" 체감.
4. **권총 P2·P4·P5 비활성 처리** — 12 스킬 원칙 내 "분기 선택으로 9개 활성화" 구조 준수.

### 10-2. 대안 비교

- **대안 A (저격 3이 E·F·우·Shift+우 중 3키 덮어쓰고 권총 1키만 남음)**: F 회피 롤 상실 → 암살자 기동성 핵심 훼손. **반려**.
- **대안 B (권총 4 + 저격 3 중 4 선택)**: 플레이어 자유도 상승하나 UI 복잡도 · 밸런싱 난이도 폭증. 시즌1 범위 초과. **보류**.
- **권장안 (본 10-1)**: F 회피 롤 고정 유지 + 나머지 E·우·Shift+우 3키를 확장 3으로 덮어쓰기. UI 단순 + 정체성 보존.

### 10-3. HUD 3레이어 정합

HUD 3레이어 설계(`poro_custom_hud_3layer_design_draft.md`)의 레이어 2 스킬 바는 **5키 고정 순서(Q·E·F·우·S우)** 다. 권총 6 + 저격 3 + 샷건 3 = 12 스킬이 HUD에 어떻게 표시되는가.

- **방안 A (항상 12개 표시, 비활성 회색)**: 레이어 2 5칸 + 별도 확장 줄 2칸? UI 폭 초과. **반려**.
- **방안 B (활성 5칸 + 패시브 2 레이어 3 버프 편입)**: 레이어 2는 현행 5칸 고정. **패시브 2(연화 장전 · 관성 집중)는 레이어 3(버프 아이콘)에 상시 녹색 테두리로 표시**. 분기 확장 3은 선택 시 E·우·S우 슬롯을 덮어쓰고 아이콘 · 테두리 색상(저격 = 청색 / 샷건 = 적색)으로 분기 표시. **권장**.
- **방안 C (분기별로 권총 6 + 활성 분기 3 = 9개 세로 2열 표시)**: 레이어 2 구조 변경 필요. 시즌1 범위 초과. **보류**.

**권장 결론**: **방안 B**. 레이어 2 = 활성 5키(Q·E·F·우·S우) · 레이어 3 = 패시브 2 + 버프/디버프. 분기 전환 시 E·우·S우 슬롯 아이콘만 교체되어 플레이어가 "분기 전환"을 직관적으로 인식.

CustomModelData 예약 제안:
- 암살자 총류 무기 본체: `30xxxxx` 영역 내 별도 블록 (data-schema 후속).
- 스킬 아이콘: 레이어 2 `3002200 ~ 3002299` 블록 내 암살자 지분 5~12칸 할당. 권총 · 저격 · 샷건 분기별 아이콘 구분.
- 분기 테두리 색상 글리프: 레이어 2 `3002101~3002105` 슬롯 테두리에 분기별 버전(저격 청색 / 샷건 적색) 추가 예약.

## 11. 서사 · 세계관 접점

### 11-1. 총류의 세계관 존재 근거

포로 세계관(수도 · 기사단 · 관찰자 · 마법 · 사하르 상인연합 · 아르카논)은 중세 판타지 톤이 기본이다. 총류는 **"마력 약실(魔力 藥室) · 마법 총기"** 개념으로 편입한다.

- **마력 약실 구조**: 탄환 = 마력 결정 · 약실 = 마력을 운동 에너지로 변환하는 마법 장치. "화약"이 아니라 "마력"이 추진력.
- **제작 주체**: **사하르 상인연합** 공방에서 외부 기술로 유입 또는 재발명. 황성 기사단 공식 무기 체계가 아니므로 **기사단 세력과는 거리감**, **관찰자 세력 · 상인연합 · 그림자 세력과 친화적**.
- **마법 의존도**: 마법 없이는 총기 작동 불가 → 마법사 3무기와 세계관 재료를 공유한다(마력 결정, 마법 부여 기술). 총류 = "기계 + 마법" 혼합.

### 11-2. 세력별 톤

- **기사단(황성)**: 총류를 "명예롭지 않은 무기"로 보는 경향. 암살자는 기사단 공식 직위에 없다 → 기사단 퀘스트 진행 시 약간의 불신 톤.
- **관찰자 세력**: 저격 분기(관측자의 조준선)와 네이밍 직접 연결. 암살자 · 관찰자 세력 연계 서브 퀘스트 여지.
- **사하르 상인연합**: 총기 공급 주 경로. 탄약 · 마력 결정 판매 NPC. 경제 축 연결.
- **아르카논**: 마법 학파 중 총기 약실 기술 연구 분파 소수 존재 → 기술 리서치 퀘스트 여지.
- **그림자 세력**: 샷건 분기(파열의 집행자)와 친화적. 지하 폭력 조직 · 용병 계약 경로에서 샷건 기술 유통.

### 11-3. 암살자 = "계약 집행자" 컨셉

- 암살자는 기사단 공식 직업 아님. **계약 기반 집행자** — 상인연합 · 그림자 세력 · 개인 의뢰의 교차점에 위치.
- 저격 분기 = "관찰 후 정밀 집행" (관찰자 세력 계약 계열)
- 샷건 분기 = "즉흥 파괴 집행" (그림자 세력 · 용병 계약 계열)
- **기사단 퀘스트에서 암살자는 중립 용병 톤**, 상인연합 · 관찰자 · 그림자 퀘스트에서 주 계약자 톤.

9. 레이어 3(버프 아이콘)에 패시브 P6·P7을 상시 녹색 테두리로 표시할 때 10칸 상한 내 여유 확인 (기존 버프 + 세트 버프 + 패시브 2 + 분기 스택 = 혼잡 가능성).
