# 포로 서버 공용각인 3슬롯 실제 목록표 초안

## 문서 목적
이 문서는 이미 정해진 공용각인 3슬롯 구조를 실제 목록 감각으로 정리한 초안이다.

핵심 목표:
- 회복/딜/디버프/생존 등 다양한 조합형 3슬롯 구조를 구체화한다.
- 각인이 세트, 룬, 잠재와 역할이 겹치지 않게 한다.
- 나중에 engraving seed로 옮기기 쉬운 이름과 효과 방향을 만든다.

---

## 1. 추천 목록 (12종 → 15종 확장 확정)

### 기존 12종
- 회복 강화
- 전투 집중
- 약점 추적
- 생존 본능
- 상태 확산
- 정밀 숙련
- 핵심 증폭
- 기동 전술
- 보호 강화
- 연계 숙련
- 자원 효율
- 역전의 의지

### 신규 3종 (combat-balance 확정)
- **각인의 잠식** (디버프 — 지속 피해/도트 강화)
- **제압 숙련** (디버프 — 제어/군중제어 강화)
- **각성의 호흡** (생존 — 유지력/자원·쿨다운 완화)

### 최종 분포 (15종)
- **딜 5종**: 전투 집중 / 약점 추적 / 정밀 숙련 / 핵심 증폭 / 기동 전술
- **생존 4종**: 회복 강화 / 생존 본능 / 보호 강화 / 각성의 호흡
- **디버프 3종**: 상태 확산(부여량) / 각인의 잠식(지속 피해) / 제압 숙련(제어)
- **운영·위기 3종**: 연계 숙련 / 자원 효율 / 역전의 의지

## 3. 신규 3종 설계 포인트

### 각인의 잠식 (지속 피해)
- "상태 확산"(상태이상 부여량)과 "약점 추적"(표식된 대상 즉발 피해 증가) 사이의 빈자리를 채움
- **부여된 디버프의 지속 피해 자체**만 증폭 → 조건부 피해 카테고리로 분류
- **위험요소**: T2 5세트 중 지속 피해·출혈·화상 정체성 세트와 겹치면 조건부 구간 폭주 → **합연산 기준 우선 적용**, 곱연산 금지

### 제압 숙련 (제어)
- 포로 상태 체계(버프 / 디버프(상태이상·제어·표식)) 중 **"제어" 축 각인 부재**를 보강
- 상태 확산(상태이상) + 약점 추적(표식) + 제압 숙련(제어) = 디버프 3축 삼각 구도 완성
- **위험요소**: 창·대검 계열 장악/다운 플레이에서 스턴 체인·경직 누적 악용 가능
- **조정**: 보스(네임드) 대상 제어 지속시간 **+상한 5% 클램프**, PvP는 별도 공식 검토

### 각성의 호흡 (유지력, 스플릿 계수 확정 — 사용자 결정 "전 스킬 확대" + combat-balance DPS 검토 스플릿)
- "자원 효율"(획득량)·"보호 강화"(보호막)·"회복 강화"(회복량)와 차별화된 **"전투 중 유지력"(쿨·자원 순환)** 축
- **적용 범위 확정**: 전 스킬에 재사용 감소 적용, 단 **공격 스킬은 절반 계수(-0.5%/Lv)**, **이동·방어기는 기존 -1%/Lv**. 자원 회복 +2%/Lv 유지.
- **DPS 영향** (combat-balance 시뮬): 전사 +2~3% / 암살자 +2~3% / 마법사 +3~4% (장쿨 핵심기 회전율 기여로 마법사 최대 이득)
- **전역 공격 스킬 쿨감 합산 상한 -20%** 클램프 권고 (세트·룬·잠재·직업각인 합산 시 -30% 초과 시 종결기가 기본기처럼 돌아가는 현상 차단)
- **카테고리 유지**: 스플릿 계수로 공격 기여 완화 → **생존 카테고리 유지 가능**. 순수 전 스킬 -1%/Lv 안은 운영 카테고리 이동 필요했으나 이번 결정으로 회피

## 4. 한 줄 요약

**공용각인은 딜 5 / 생존 4 / 디버프 3 / 운영 3 분포의 15종 조합형 구조로 두고, 레벨당 1.5~2% 완만한 상승과 신규 3종(잠식·제압·호흡)의 역할 분리로 빌드 다양성을 극대화한다.**
