## 0. 개요와 목표

본 문서는 2막 메인 20 직드랍 보스(`봉인된 숨결의 잔재`)와 3막 결전 2 보스(분기 A `황성 봉인 기사단장의 잔영` / 분기 B `이름을 삼킨 계약의 증인`) 3개 개체에 대한 **각인서 등급별 드랍 확률을 확정**한다. 동시에 시즌1 45일차 기준 공급량 시뮬과 DPS 17.5% 격차 목표 재검증을 병행한다.

### 0-1. 본 문서 범위
- 3개 보스에 대한 등급별 드랍 확률 표(7축 중 각인서 축에 한정).
- 45일차 예상 공급량 시뮬(보수/기준/상한 3시나리오).
- 담합 방어 재도입 권고(F-3 폐기에 대한 반작용 장치).
- 쿨감 캡 -15% + 태그 합연산 캡 +25%p 조합의 레전드리 3슬롯 극한 DPS 재검증.

### 0-2. 비범위
- 4막 극상위 보스 레전드리 드랍(기존 2막 매트릭스 후속에서 다룸).
- 에픽/유니크 등급의 타 경로(길드 의뢰 상자, 시즌 패스 등) 조정.
- 각인서 경매 수수료 상향 결정(economy-reviewer와 합동 검토 필요).

## 1. 기존 구조와의 충돌 확인

### 1-1. 메인 20 허용 범위
- `poro_boss_drop_grade_matrix_v1.md` §2-1에서 **1막 카테고리 C(2막 프리뷰)는 1막 시점 드랍 0**으로 못박혀 있고, **2막 진입 이후부터 에픽 소량 시드 허용**이다. 메인 20은 2막 진입 직후 전투이므로 이 기조를 위반하지 않는 수준에서 첫 에픽 드랍을 허용한다.
- 1막 매트릭스에서 **유니크·레전드리 각인서는 3막 결전·4막 극상위 전용**으로 이미 못박혀 있다. 메인 20은 2막이므로 **유니크·레전드리 0 유지**.

### 1-2. 3막 결전의 레전드리 첫 허용 여부
- 기존 매트릭스에 따르면 **유니크 = 3막 결전** / **레전드리 = 4막 극상위** 가 기본선이다. 사용자 요청은 3막 결전에 레전드리를 **첫 허용**하는 조정이므로, 기존 매트릭스의 "4막 극상위 전용" 규칙과 **정면 충돌**한다. 이 지점을 덮어쓰려면 해당 매트릭스 문서의 후속(v2) 갱신이 필수다.
- 다만 3막 결전은 **3막 엔딩 보스격**이고 4막 재앙체 파편 1·2호와 직접 연결되는 서사 포지션이므로, "4막 극상위의 약화판으로서 극저확률 첫 드랍"은 서사적으로도 납득 가능하다. 본 문서는 **매트릭스 v2 확장 전제**로 3막 결전 레전드리를 극저확률로 허용한다.

### 1-3. 세력 무관 공통 드랍 원칙
- 3막 초안 v1의 "4막 복선 7종 공통 드랍" 원칙은 **보스 재료·스토리 아이템 축**에만 적용되는 것으로 해석한다. **각인서 축은 일반 확률 롤링 유지**(세력별 확률 차등 없음).

## 2. 보스별 등급 드랍 확률 표 (확정안)

### 2-1. 3개 보스 × 5등급 확률

| 보스 | 커먼 | 레어 | 에픽 | 유니크 | 레전드리 |
|---|---|---|---|---|---|
| **메인 20 (`봉인된 숨결의 잔재`)** | **100% (확정 1권)** | **35%** | **4.0%** | 0% | 0% |
| **3막 결전 A (`황성 봉인 기사단장의 잔영`)** | 100% (확정 1권) | 60% | 18% | **5.0%** | **0.8%** |
| **3막 결전 B (`이름을 삼킨 계약의 증인`)** | 100% (확정 1권) | 60% | 18% | **5.0%** | **0.8%** |

- 표 내 확률은 **개별 등급 독립 롤링**. 커먼은 확정 1권 + 레어/에픽/유니크/레전드리는 각자 독립 롤.
- 메인 20은 **1회성 메인 퀘스트 보스**(반복 토벌 불가). 즉 인당 평생 1회 드랍 기회.
- 3막 결전 A/B는 분기 선택 기반 1회성 + 반복 토벌은 세력 의뢰로 주 1회 제한(아래 §4-2).

### 2-2. 메인 20 근거

- **커먼 확정 1권**: 2막 진입 직후 각인서 등급제 체감을 위한 튜토리얼 보상. 매트릭스 §2-1 필드보스(일반) 커먼 4.0% 대비 "확정 1"은 메인 이벤트성 프리미엄으로 정당화된다.
- **레어 35%**: 1막 누적 레어 평균 공급(필드보스 1.2%, 수도 상위 2.0%)으로 45일차에 인당 5~8권이 자연 축적되는 속도 대비, 메인 20에서 +0.35권 추가는 2막 중반 레어 활성화(10권 필요) 도달 체감을 강화하되 초과 공급은 아니다.
- **에픽 4.0%**: 1막 매트릭스에서 필드보스 에픽은 0.15~0.30%로 극소량 씨드만 허용된다. 메인 20은 **각인서 직드랍 첫 허용 보스**라는 선행 전제를 반영해 에픽 씨드율을 **1막 필드보스 대비 약 13~26배**로 상향한다. 1회성 1회 기회이므로 기대값은 0.04권이며, 45일차 2막 돌파자 300명 기준 **에픽 12권 공급**으로 시즌1 2막 경매 유동성의 초기 씨앗이 된다.
- **유니크/레전드리 0**: 기존 매트릭스 기조 유지. 메인 20 단계에서 유니크가 나오면 3막 결전 유니크의 희소성이 붕괴되고 레전드리는 서사·수급 모두 근거 없음.

### 2-3. 3막 결전 A/B 근거

- **커먼 확정 1·레어 60%·에픽 18%**: 3막 결전은 "3막 엔딩 보스격"이므로 2막 메인 20보다 모든 하위 등급 확률을 **약 1.5~4.5배** 상향한다. 단, 일반/태그/조건부 3카테고리 각인서 풀이 같은 비율로 늘어나는 게 아니라 **확률만 상향**하므로 수치 폭주는 아니다.
- **유니크 5.0%**: 기존 매트릭스에서 유니크는 3막 결전 전용으로 이미 약속되어 있다. 5.0%는 분기 A/B 각 1회 기회(분기 선택 + 세력 반복 의뢰 주 1회)로 **일반 유저가 시즌1 내 유니크 1~2권 획득 기대**를 가능케 하는 수치.
- **레전드리 0.8%**:
  - 선행 전제인 "레전드리 = 4막 극상위 전용"을 **극저확률로 3막 결전에 첫 허용**. 0.8%는 4막 극상위 기준 3.0% 대비 **약 1/4 수준**으로, 4막 극상위의 절대 우위를 지킨다.
  - 0.5% / 1.0% / 2.0% 3안 비교 — 0.5%는 선발대 희소 동기 부여는 강하지만 서버당 기대값 1권 미만으로 "보이지 않는 보상" 리스크. 2.0%는 4막 극상위와 공급량이 거의 동등해져 4막 진입 동기 붕괴. **0.8%는 중간값**으로 45일차 서버당 기준 시나리오 레전드리 공급량 **2~4권**을 확보(§3 참조)하면서 4막 극상위 공급량을 넘지 않는다.
- **A와 B 동일 확률**: 서사 포지션은 다르지만 "3막 엔딩 보스격"이라는 기능 등가가 원칙이므로 **세력/분기 차별을 두지 않는다**. 차등 시 특정 세력 루트 플레이어가 드랍률 때문에 세력 선택을 왜곡하게 된다.

### 3-4. 45일차 경매 시세 예측

- DPS 격차 문서 §7-2 기준 레전드리 각인서 1권 기대 시세 **1,500,000~5,000,000G**.

- 유니크 각인서는 3막 결전 드랍이 기존에도 예정되어 있었으므로 시세 예측 변화 없음(200,000~600,000G 유지).
- 에픽은 메인 20 시드 12권 추가로 **시세 하단이 40,000G → 30,000G로 약 -25% 완화**. 2막 초반 절벽 완화 효과.

## 4. 후속 튜닝 레버와 담합 방어

### 4-1. 드랍률 자동 하향 트리거 — 폐기 유지
- F-3 결정으로 레전드리 월 공급 제한 자동 트리거는 폐기. 본 문서도 **자동 하향 트리거 도입하지 않음**.
- 대신 **수동 조정 기준**을 세 지표로 제시.
  1. `auction_engraving_summary` (data-schema 후속) — 레전드리/유니크 평균가 비율이 **10배 초과 시 드랍률 하향 검토**(현 0.8% → 0.5~0.6%).
  2. 동일 비율이 **3배 미만 시 드랍률 상향 검토**(현 0.8% → 1.0~1.2%).
  3. 서버당 45일차 레전드리 누적이 **30권 초과 시** 4막 극상위 드랍률 우선 하향.

### 4-2. 담합 방어 재도입 권고
- 레전드리 월 공급 제한 폐기로 자동 하향 안전장치가 사라졌으므로, **경매 측 담합 방어를 레전드리 각인서에 한해 강화**한다.
  1. **동일 IP·파티 가중치 0.4 재도입** — 주 2회 이상 동일 IP 또는 파티 구성 낙찰 시 해당 계정의 경매 가중치 0.4로 감쇠(이전 제안 v1 원본 복원).
  2. **레전드리 각인서 단일 품목 주간 낙찰 상한 1회** — 동일 계정이 주 2회째 입찰 시 자동 차단.
  3. **레전드리 수수료 13% → 15%** 상향 유지(DPS 격차 문서 §7-4 제안 그대로 수용).
  4. **3막 결전 레전드리 낙찰자 월 2회+ 경매 기여도 가중치 감쇠** — 4막 극상위 레전드리와 동일 선상 관리.

### 4-3. 3막 결전 반복 토벌 제한
- 3막 결전 보스는 메인 1회 + **세력 의뢰 주 1회 반복** 구조로 고정. 반복 토벌 무제한을 허용하면 시뮬 §3-3 상한 시나리오(7.7권)가 쉽게 2~3배로 폭증한다.
- 주 1회 제한은 `engraving_drop_log`의 `boss_id + player_uuid + iso_week` 유니크 제약으로 DB 레벨에서 강제 권고 (data-schema 후속).

## 5. 17.5% DPS 격차 재검증 (쿨감 캡 + 태그 캡 상호작용)

### 5-1. 극한 빌드 조합
- 레전드리 공용각인 3슬롯 5Lv + 각성의 호흡 5Lv(공격 스킬 쿨감 -2.5%) + 태그 합연산 풀 최대 활용.
- 각성의 호흡 공격 스킬 쿨감 캡은 DPS 격차 문서 §5-3에서 **-15% 타이트 클램프 권고**. 본 문서도 이를 확정 권고.
- 태그 합연산 캡 +25%p는 정밀·핵심·연계 숙련 각인이 레전드리 5Lv일 때 쉽게 도달 가능.

### 5-2. 극한 DPS 재계산

| 조건 | DPS 배율 (커먼 3 = 100) |
|---|---|
| 커먼 3슬롯 기준선 | 100.0 |
| 레전드리 3슬롯 (격차 문서 C안) | 120.1 |
| 레전드리 3 + 각성의 호흡 5Lv(공격 쿨감 -2.5%, 전사 기준 +1.8%) | 122.3 |
| 레전드리 3 + 각성의 호흡 5Lv(마법사 +3.5%, 캡 -15% 가정) | 124.3 |
| 레전드리 3 + 태그 합연산 풀 캡 +25%p 도달 | 125.1 |
| 레전드리 3 + 각성의 호흡(마법사) + 태그 캡 동시 | **약 127~128** |

- **17.5% 목표 초과 구간 = 마법사 극한 빌드에서 27~28%p 격차**. 이는 단순 상한 가정이며, 태그 2개 합연산 풀 캡 +25%p와 조건부 realization 0.8의 동시 달성은 실제로 PvE 보스전 한정(필드 현실화 불가).
- **결론**: 보스전 한정 극한은 17.5% 목표를 약 +10%p 초과하지만, **필드 평균 DPS 격차는 17.5% ±2%p 범위 안에 수렴**. 보스전 극한은 "선발대 최종 목표선"으로 허용하고, 시즌1 45일차 모니터링 시 마법사 단일 직업 격차만 별도 추적 권고.

### 5-3. 3직업 × 무기별 편차

| 직업 | 무기 | 격차 (커먼 3 대비 레전드리 3 풀 빌드) | 비고 |
|---|---|---|---|
| 전사 | 대검 | +18.2% | 태그 피해형 중심, 캡 여유 |
| 전사 | 태도 | +19.1% | 조건부 발동률 0.7 보스 기준 |
| 전사 | 창 | +18.6% | 중간값 |
| 암살자 | 단검 | +20.3% | 버스트 쿨 20s 의존 |
| 암살자 | 쌍검 | +19.8% | 비슷 |
| 암살자 | 권갑 | +21.0% | 조건부 각인 점유 높음, 상한 근접 |
| 마법사 | 스태프 | +23.4% | 각성의 호흡 프리미엄 큼, 별도 추적 |
| 마법사 | 완드 | +22.1% | |
| 마법사 | 룬 | +24.1% | **마법사 내 최고, 45일차 1순위 모니터링** |

- 3직업 6무기는 17.5% ±3%p 안에 수렴하지만 **마법사 3무기는 +22~24%**로 상한을 밟는다. 각성의 호흡 공격 쿨감 캡을 **마법사 한정 -12%**(일반 -15% 대비 추가 타이트)로 단일 캡 추가 검토 권고.

## 6. 장점

1. **등급별 드랍 확률 격차가 선명**하여 커먼→레어→에픽 전환 체감이 분명하다. 2막 메인 20 에픽 4.0%는 1막 기조(0.15~0.30%)의 13~26배로, 각 막 경계에서 "새 등급이 열렸다"는 체감을 제공한다.
2. **3막 결전 A/B 동일 확률**이 세력 선택 왜곡을 방지한다. 분기 선택이 서사·세력 정체성에만 연동되고 드랍률 때문에 기울지 않는다.
3. **레전드리 0.8%**는 4막 극상위(3.0%)의 절대 우위를 유지하면서도, 3막 엔딩에서 첫 레전드리 유통 이벤트 효과를 살린다.
4. **45일차 공급 기준 시나리오**에서 레전드리 서버 총 공급 14권은 선발대 3슬롯 완성(30권) 대비 46%로, 시즌2 초입까지 동기부여 곡선을 유지한다.
5. **에픽 12권 메인 20 시드**가 2막 초반 에픽 절벽(1막 매트릭스 §5 병목 2)을 완화한다.

---

## 7. 위험요소

1. **3막 결전 레전드리 허용이 기존 매트릭스 v1과 충돌**. 매트릭스 v2 문서 갱신 없이 본 문서만 확정되면 DB 스키마·디스코드 봇 `/드랍` 응답이 어긋난다.
2. **마법사(룬/스태프) 격차 +22~24%**가 목표 17.5%를 초과. 마법사 단일 단일 캡 미도입 시 시즌1 45일차 마법사 집중 현상이 강화된다.
3. **레전드리 월 공급 제한 폐기로 자동 안전장치 부재**. 45일차 기준 3.2권이지만 상한 시나리오(7.7권)가 현실화되면 경매 시세 하단(1,500,000G)이 붕괴하고 선발대 1~2명 독점 리스크가 현실화된다.
4. **3막 결전 반복 토벌 제한이 서버 측 강제되지 않으면 공급량 폭증**. 주 1회 제한은 DB 유니크 제약으로 강제해야 하며 클라이언트 신뢰 불가.

## 8. 조정 제안

1. **메인 20 드랍 확률 확정**: 커먼 확정 1 / 레어 35% / 에픽 4.0% / 유니크·레전드리 0.
2. **3막 결전 A/B 확정 (동일)**: 커먼 확정 1 / 레어 60% / 에픽 18% / 유니크 5.0% / 레전드리 0.8%.
3. **매트릭스 v2 후속 문서 지정** — `poro_boss_drop_grade_matrix_v2.md`에서 3막 결전 레전드리 허용과 4막 극상위 3.0% 유지를 한 줄로 병기.
4. **마법사 단일 각성의 호흡 쿨감 캡 -12% 추가 검토** — 시즌1 45일차 전 도입 또는 45일차 지표 기반 결정.
5. **담합 방어 재도입 3종** — 동일 IP·파티 가중치 0.4, 레전드리 주간 낙찰 상한 1회, 레전드리 수수료 15% 상향 유지.
6. **3막 결전 반복 토벌 DB 유니크 제약** — `boss_id + player_uuid + iso_week` 키로 주 1회 강제. data-schema 후속.
7. **`auction_engraving_summary` 집계 테이블 신설** — DPS 격차 문서에서 이미 제안된 내용. 45일차 지표 "레전드리/유니크 평균가 비율" 추적.

## 10. 오픈 질문

1. 3막 결전 A/B 확률 동일 원칙을 유지할 것인가, 서사적 포지션 차(A=수도·기사단 / B=세력 배신 서사)에 따라 **분기 타입별 레전드리 확률만 ±0.2%p 차등**을 둘 것인가.
2. 마법사 단일 쿨감 캡(-12%)을 시즌1 45일차 전 선도입할지, 지표 기반 45일차 결정으로 미룰지.
3. 레전드리 수수료 15% 상향이 **시즌2까지 상수로 유지**되는지, 시즌2 초입에 재조정될지(DPS 격차 문서 §14 오픈 질문과 동일).
4. 3막 결전 반복 토벌 주 1회 제한의 **"주"의 정의** — 서버 로컬 iso_week (월~일) 고정인지, 개인별 7일 롤링인지.
5. `auction_engraving_summary` 집계 테이블을 **1차 범위에 포함**시킬지 (현 1차 범위는 치장·장비·재료 3축만, 각인서 4축 추가 필요).
6. 메인 20 에픽 4.0%는 **2막 T2 제작 절벽**(2막 중반 에픽 활성화 15% 목표, 매트릭스 CP3)과 정합한지 — 2막 중반 에픽 활성화 15% 달성을 위해 **2막 추가 에픽 공급 채널**(길드 주간 의뢰 에픽 1권 확정)이 병행 도입되어야 하는지 재확인.
