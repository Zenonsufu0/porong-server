## 0. 검토 목표

1. 파티 3명 상한에서 "힐 전담 무기가 없는" 구조가 전투 설계와 보스 기믹에 미치는 영향을 정리한다.
2. 마법사 4번째 무기(촉매서) 경로가 반려된 상태에서, **5가지 대체 옵션(A/B/C/D/E)** 을 기존 포로 기준(T2 5세트·공용각인·DPS 17.5% 격차·각성의 호흡 -15% 타이트 캡·태그 최대 2개·상태 분류 버프/디버프)과 정합 여부 포함해 비교한다.
3. 2막 메인 19·20, 3막 메인 30 A·B 보스 기믹이 **현재 어떤 힐 메커니즘을 전제하는지** 역추적하고, 옵션 채택 시 재조정 필요 여부를 정리한다.
4. 1~2개 옵션 조합 권장안을 제시한다.
5. 리스크 3건과 완화안을 제시한다.

## 0-1. 비범위

- 촉매서·원소 영창구·봉인 장패 등 **4번째 무기 재도입**은 본 문서 범위 밖(2026-04-20 사용자 반려 확정 존중).
- PvP·결투장 힐 공백 — 본 문서는 PvE 파티 전투에만 집중.
- 세트 6번째 신설(힐 세트)은 T2 5세트 정체성 유지 원칙과 정면 충돌하므로 본 문서에서 취급하지 않음.
- 직업각인 분기에 "힐 분기" 편입 여부 — 본 문서는 옵션 B 서술 내에서만 한정 언급, 확정은 후속.

## 1. 현재 구조 요약 — 힐 공백이 생긴 경로

### 1-1. 9무기 역할 분포 (`poro_class_weapon_file_audit_v1.md` 7-4장 근거)

- **전사 (3무기)**: 대검(탱/딜 겸 광역) · 태도(단일 딜) · 창(딜/탱 겸 중거리)
- **암살자 (3무기)**: 단검(단일 폭딜) · 쌍검(지속 고딜) · 총류(근접 폭딜)
- **마법사 (3무기)**: 스태프(광역 딜/제어 준탱) · 완드(단일 중거리 딜) · 룬(지역 장악 딜·디버프)

9무기 중 **힐 전담은 0개**, **힐 기믹이 서브로 붙을 여지 있는 무기는 스태프 1개**(광역 지배+제어, 준탱/서포트 겸).

### 1-2. 기존 힐/회복 축 재고

현재 구조에서 플레이어가 체력을 회복할 수 있는 경로:

- **공용각인 "회복 강화"**: 레벨당 회복 효율 +2%, 5Lv +10% (회복량 증폭만, 회복 소스 아님)
- **공용각인 "생존 본능"**: 최대 체력 +2%/Lv, 5Lv +10% (간접 생존)
- **공용각인 "보호 강화"**: 보호막 효율 +2%/Lv (실드 베이스, 회복 아님)
- **공용각인 "각성의 호흡"**: 피격 시 자원 회복 +2%/Lv — **체력 회복 아님, 자원 순환**
- **회복 물약 3등급** (일반 15% / 고급 25% / 희귀 35%+피감 5%), 쿨 30~40초, 개별 쿨다운
- **음식/만찬**: 최대 체력·회복 효율만 증폭, 힐 소스는 아님 (`poro_food_feast_potion_master_draft.md`)
- **T2 세트 일부**: 흡혈·실드 생성 기믹 가능성 있으나 5세트 정체성 혼합 금지 원칙상 "힐 세트"는 별도 존재하지 않음

→ **결론**: 포로 서버는 **이미 "힐 전담 직업 없음 + 소비품·각인 보조 + 세트 기믹" 혼합 구조**에 가깝다. 파티 3명 상한은 이 구조를 **명시적으로 드러낸** 것뿐, 구조 자체를 새로 강제한 것이 아니다.

### 1-3. 왜 지금 문제로 떠올랐는가

로스트아크식 "파티 4~8명·바드/홀리나이트 1자리"에 익숙한 플레이어 관점에서 3명 상한 + 힐 전담 무기 부재는 **"딜 2 + 탱 1 강제 = 힐은 어디?"** 체감 공백으로 보인다. 그러나 포로 서버 원안(마스터 플래닝 기반 직업 3분류·T2 5세트·각인 구조)은 **"힐 전담 없는 파티 전투"** 를 이미 기본으로 가정하고 있었다.

## 2. 옵션 A — 소비품 의존 강화

### 2-1. 구조
- 힐 전담 없음을 전제로 **물약·만찬·음식을 전투 내 힐 주 공급원으로 상승**.
- 현행 회복 물약 쿨 30~40초를 **쿨 20~25초로 하향**하거나, `RARE+` 등급 신설(50% 회복 + 부가효과) 검토.
- 만찬 효과에 "회복 효율 +6%" 같은 간접 수치 대신 **"파티 전원 주기적 자동 회복 2초당 1%"** 같은 직접 힐 소스 신설 가능성.
- 전투 중 **회복 물약·전투 보조 물약 슬롯 별도 유지** (현 구조 그대로).

### 2-2. 장점
- **구조 변경 최소** — 이미 존재하는 `consumable_items` + `consumable_effects` 테이블 수치 조정과 seed 추가만으로 대응.
- T2 5세트·공용각인·DPS 17.5% 격차에 **전혀 손대지 않음** → 기존 확정 기준 완전 보존.
- 구현 복잡도 최저(약 0.5~1주).
- 포로 서버 경제 축 중 하나인 **연금술·요리 생활 스킬 수요를 명확하게 상향**시킴. economy-reviewer 관점에서 반길 만한 구조.

### 2-3. 위험요소
- **악용 조합**: 회복 물약 쿨 하향 시 고랭커가 전투 내내 물약 회전으로 "자힐 사이클" 완성 → 파티원 간 회복 능력 격차가 **물약 재고 격차**로 전환. 소비품 빈부 격차 발생.
- **플레이스타일 붕괴**: 전투가 "물약 타이밍 게임"으로 수렴. 보스 기믹 회피·패턴 숙련의 가치가 상대적으로 저하.
- **보스 설계 영향**: 2막 메인 20·3막 메인 30 기믹이 현재 "준비된 파티의 안정성" 기대치로 설계되어 있음. 물약 쿨 단축은 **DPS 시간 압력(엔레이지 타이머)을 재조정** 필요. 보스 HP 5~8% 하향이 동반되어야 함.
- **경제 영향**: 연금술 생활 스킬이 힐 공급의 단일 병목이 되어 **가격 담합·재고 편중 위험**. 영지 연금 Lv 요구치에서 소외된 신규 유저 이탈.
- **DPS 17.5% 격차와 연결**: 소비품이 힐을 전담하면 공용각인 "회복 강화"가 사실상 사용되지 않아 15종 중 1종이 죽는 결과.

### 2-4. 조정 제안
- 회복 물약 쿨 하향은 **30→25초** 수준의 **소폭 조정만**, RARE는 40→35초 유지.
- 만찬 "파티 자동 회복" 소스 신설은 **2초당 0.5%** 수준의 매우 보수적 수치, 만찬 1종만 한정(수도 만찬 1종).
- 회복 물약 보관 상한(인벤 20개 → 30개) 소폭 상향으로 재고 압박 완화.

## 3. 옵션 B — 직업별 부분 힐 기믹 분산

### 3-1. 구조
- 전사 창 / 마법사 스태프 / 암살자 단검(또는 쌍검)의 스킬 5키 중 **1개에 소규모 자힐 or 파티 힐 태그**를 추가.
- 예시:
  - **전사 창 "압박 전개"**: 압박 스택 5 소비 시 **자힐 3% + 파티 반경 3블록 내 힐 1.5%** 1회 (쿨 25s).
  - **마법사 스태프 "냉기 장막"**: 제어 장막 지속 중 **영역 내 파티원 2초당 0.8% 힐** (지속 6s, 쿨 30s).
  - **암살자 단검 "그림자 이동"**: 무적 + 후방 이동 시 **자힐 2%** 1회 (쿨 20s).
- 힐량은 전투 체감 기준 **"큰 실수 1회는 못 버티는 정도"** — 보조 축, 전담 대체 아님.

### 3-2. 장점
- **힐 공백을 "분산형"으로 해소** — 파티 3명 전원이 각자 자기 몫의 작은 힐을 가짐.
- 직업각인 분기(`poro_engraving_dps_gap_review_v2_followup.md` 1-2장)에 **"힐 분기" 옵션 추가 경로** 열림 — 예: 창 "압박" 분기 C = "회복 분기"(압박 스택 소비 시 자힐 증폭 +50%). 무기 정체성 훼손 없이 **분기로만 선택**.
- 3인 파티에서 **플레이스타일 강요 없음** — 전담 힐러 강요 없이 "각자 관리"라는 포로 특유의 색 강화.

### 3-3. 위험요소
- **전사 창·대검 기존 각인 정체성 훼손 위험**: 창은 "압박·표식·중거리 제어", 대검은 "균열 소모·단타 광역"으로 정체성 확정됨. 힐 태그 추가가 **무기 정체성 혼탁화** 가능성. CLAUDE.md 원칙 "대검/태도/창의 기존 각인 정체성을 존중한다" 직접 충돌 우려.
- **수치 폭주**: 자힐 3% × 3명 × 쿨 25s 회전 시 파티 유효 힐 = 초당 **약 0.4% HP** 지속 공급. 현재 보스 DPS 설계와 겹쳐 보면 **엔레이지 타이머 시 체감 난이도 10~15% 하락**.
- **태그 최대 2개 원칙과 충돌 가능성**: "힐 태그"를 신규 태그로 추가 시 태그 풀 확장 필요. 기존 "정밀 / 핵심 / 연계 / 광역 / 표식 / 상태이상 / 조건부 사거리" 등과 별도 "회복 태그" 편입이면 합연산 캡(+25%p) 별도 설정 필요.
- **악용 조합**: 창·스태프·단검 3명 파티 구성 시 자힐 분산이 **실질 "저해상도 힐러 1명분"** 으로 뭉침 → 탱 없는 3딜 파티가 생존 가능해져 탱 포지션 의미 약화.
- **암살자 "표식 집행·고회전" 정체성**과 힐 기믹은 톤 충돌(암살자는 살상에 충실해야 함). 암살자 쪽 부여는 보류 권장.

### 3-4. 조정 제안
- 전사 창·마법사 스태프 2무기에만 자힐 기믹 추가, 암살자는 제외.
- 힐량은 현재 수치(3%·1.5%)의 절반 수준으로 시작 — **자힐 1.5% / 파티 힐 0.8%**. 30일차 실측 후 상향 여부 결정.
- 태그 신설 대신 **"조건부" 카테고리 내부 효과**로 편입 → 태그 풀 확장 없음, 합연산 캡 재설계 불필요.
- 직업각인 분기 C(창)·분기 C(스태프)에 "회복 분기" 옵션 추가 → ±2%p DPS 편차 클램프 안에서 운영.

## 4. 옵션 C — "힐 없는 전투" 설계

### 4-1. 구조
- 힐을 **원천적으로 대체**: 딜링 즉시 보상 기반(흡혈·실드 생성·회피 보상)이 자힐 대체.
- T2 세트 5종 중 1~2개에 **"전투 즉시 보상" 기믹**을 소폭 강화:
  - **홍염의낙인**: 지속 피해 누적 시 **최대 누적량의 3%만큼 자힐** 1회/8초.
  - **청풍의추적**: 회피 성공 시 **다음 공격 피해 +8% + 자힐 1%**.
- 공용각인 "보호 강화"(실드 효율)의 역할 상승 → 딜 스킬 일부가 보호막 생성 기믹 동반.
- 힐 없이도 보스 기믹을 클리어 가능하도록 **보스 데미지 패턴을 "예측 가능 + 회피 가능"으로 재설계** (대형 피해는 반드시 전조 1.5s 이상).

### 4-2. 장점
- **"스타일 붕괴 없이 힐 공백 해소"** — 포로 서버의 "준비된 파티의 안정성" 철학과 정합.
- T2 세트 정체성 **훼손 없이 강화만** (홍염·청풍의 기존 키워드 내).
- DPS 17.5% 격차와 직접 충돌하지 않음 — 세트 기믹은 세트 편성 선택으로 귀결.
- 공용각인 "보호 강화" 1종의 실효가 올라가 **15종 내 딜-생존 균형 개선**.

### 4-3. 위험요소
- **보스 설계 전면 재검토 필요** — 2막 메인 20·3막 메인 30의 "맵 붕괴 영역·정적의 일격" 기믹이 힐 전제라면 회피 가능한 전조 패턴으로 재설계. 보스 설계 수작업 비용 매우 큼 (2~3주).
- **신규 유저 진입 장벽 상승** — 회피·포지셔닝 숙련이 힐로 커버되지 않아 **기믹 숙련 실패 = 즉시 데스**로 연결. 일반 유저와 상위권 격차 확대.
- **악용 조합**: 흡혈 세트(홍염) + 지속 피해 각인(잠식) + 지속 피해 태그 무기(룬) 조합 시 **딜링만으로 풀HP 복구** → "불멸 빌드" 수렴 위험. 합연산 캡으로 묶여 있으나 흡혈 수치가 세트 기믹으로 들어가면 **캡 밖**에서 곱연산 우회 가능.
- **성장 곡선 불균형** — T2 세트 완성 전(T1 구간)에는 대체 힐 수단 부족 → T1 플레이 체감 악화.

### 4-4. 조정 제안
- 세트 기믹 흡혈은 **피해량의 3% 상한 + 초당 누적 최대 2% HP** 하드 캡.
- T1 구간 힐 공백은 **옵션 A(소비품) 소폭 병행**으로 완화 — T1 회복 물약 쿨 25s, T2 이후 세트 기믹으로 자연 전환.
- 보스 재설계는 **2막 메인 20 / 3막 메인 30 2종만 우선**, 정예·필드보스는 현행 유지.
- 흡혈·실드 기믹은 합연산 풀 편입을 명시 — 태그 +25%p 캡과 별도 "자힐 합연산 캡 +8%" 신설.

## 5. 옵션 D — 아이템·각인 슬롯 힐 패시브

### 5-1. 구조
- 공용각인 15종 중 **"회복 강화"·"각성의 호흡"·"보호 강화" 3종의 힐 지향성을 강화**.
- 예시: "회복 강화"에 **"2초당 체력 0.5% 자동 회복"** 추가, 5Lv +2.5% 풀 HP/초 (합산 시 초당 1.25%).
- 플레이어가 **공용각인 3슬롯 중 1~2슬롯을 힐 위주로 편성**하면 파티 전체 생존 안정화.
- DPS 17.5% 격차는 **"힐 각인 슬롯 vs 딜 각인 슬롯"의 선택 비용**으로 자연 발생.

### 5-2. 장점
- 기존 공용각인 15종 구조 완전 보존 — 신규 각인 추가 없음.
- "힐 빌드 vs 딜 빌드" 선택의 무게가 **각인 3슬롯 내부에서 완결** → 시스템 단순화.
- 일반 유저도 레어·에픽 등급 "회복 강화"로 접근 가능 — 상위권 레전드리 격차 완화.

### 5-3. 위험요소
- **각인 쏠림**: 파티 3명 중 최소 1명이 "회복 강화"에 1슬롯 투자 강제 → **실질적으로 "힐러 포지션"이 각인 슬롯으로 이동**. "힐 전담 없음" 원칙의 우회 변형.
- **수치 폭주**: 자동 회복 0.5%/2s × 5Lv × 레전드리 1.90배 = 초당 약 0.475% HP. 3명 × 초당 1.4% 이상 — 보스 DPS 시뮬 상 엔레이지 타이머 의미 상실 구간 발생.
- **딜 각인 다양성 위축**: v1 DPS 격차 리뷰(권장안 17.5%) 기준으로 이미 "딜 각인 5종 vs 생존 4종 vs 디버프 3종 vs 운영 3종"의 균형이 잡혀 있음. 이 옵션은 생존 쪽을 더 키워 **딜 각인 선택 다양성 저하**.
- **플레이스타일 붕괴**: 각인이 힐·딜·디버프 모두를 떠안으면 "각인은 곧 역할"로 축소되어 **무기·세트의 정체성 역할이 희석**.

### 5-4. 조정 제안
- "회복 강화"에 자동 회복 추가는 **2초당 0.3% (5Lv 1.5% 레전드리 배율 후 초당 0.6%)** 수준의 매우 낮은 수치.
- 3슬롯 중 "회복 강화 × 2" 같은 중복 스택은 1슬롯만 유효 처리 — 쏠림 방지.
- 시즌1 45일차 튜닝에서 자동 회복 수치 -30% 하향 트리거 검토.

## 8. 권장안 — A + B 조합

### 8-1. 근거

1. **옵션 A (소비품 의존 소폭 강화)** — 구조 변경 최소, 기존 포로 기준 완전 보존, 구현 복잡도 최저. 파티 3명 상한에서 "기본 힐 공급원"으로 소비품 위치를 명확화.
2. **옵션 B (직업별 부분 힐 기믹 분산, 2무기만)** — 전사 창·마법사 스태프 2무기에만 **소규모** 자힐/파티힐 기믹을 조건부 카테고리로 부여. 태그 풀 확장 불필요, DPS 17.5% 격차 영향 미미.

### 8-2. 왜 이 조합인가

- **A 단독은 "힐의 전투 내 능동성"이 부족**. 물약 회전만으로는 보스 기믹 중 즉시 대응이 어려움. B의 소규모 자힐이 "예측 가능한 작은 힐 여유"를 더해 전투 호흡 안정화.
- **B 단독은 T1 구간·소비품 수요 편중** 문제. A가 소비품 축을 병행 지탱해 T1 초반 생존 체감 개선.
- **C(힐 없는 전투)는 보스 재설계 부담 과도** — 2막 메인 20·3막 메인 30은 이미 v1 씬 카드로 상세 설계됨. 3~4주 재설계는 시즌1 일정상 비현실적.
- **D(각인 슬롯 힐)는 각인 쏠림 + "힐 각인 장착자 = 힐러"로 귀결** — 3명 상한 힐 공백의 본질적 해소가 아닌 우회.
- **E(스태프 보강)는 사용자가 반려한 촉매서 경로의 **변형**에 불과** — 힐 전담 강요 리스크를 마법사 3무기 내부로 이동시킬 뿐, 본질 해결 아님. 2026-04-20 사용자 결정 정신과 불일치.

### 8-3. A+B 조합 구체 수치

- **A 부분**:
  - 회복 물약 쿨 30→25초 (일반·고급), RARE는 40→35초.
  - 만찬 1종 신설 "제국 수비 만찬" 내부에 "파티 2초당 0.5% HP 자동 회복" 추가 (45분 지속).
  - 인벤 회복 물약 보관 상한 20→30.
- **B 부분**:
  - **전사 창 "압박 전개"**: 압박 스택 5 소비 시 자힐 1.5% + 주변 3블록 내 파티 힐 0.8% 1회, 쿨 30초.
  - **마법사 스태프 "냉기 장막"**: 제어 장막 영역 내 파티 2초당 0.5% 회복, 지속 4s, 쿨 30초.
  - 두 무기 모두 **조건부 카테고리**로 편입, 태그 풀 확장 없음.
  - 직업각인 분기 C에 "회복 분기" 추가(창·스태프) — 선택 시 해당 기믹 효과 +50% 증폭, 대신 분기 A/B 대비 공격 DPS -2%.

### 8-4. 기존 결정과의 정합 체크

- T2 세트 5종 정체성: **영향 없음** ✅
- 공용각인 15종: **영향 없음** (회복 강화 실효 유지) ✅
- 직업각인 분기 ±2%p 클램프: **준수** (회복 분기 -2% DPS) ✅
- 각성의 호흡 -15% 타이트 캡: **영향 없음** ✅
- 태그 최대 2개·합연산 +25%p 캡: **영향 없음** (조건부 편입) ✅
- 상태 체계 버프/디버프: **영향 없음** (자힐은 버프도 디버프도 아닌 HP 직접 조작) ✅
- 대검·태도·창 정체성: **창 "압박" 분기 추가, 기존 정체성 유지** ✅

  - 페이즈 1: 원거리 견제 + 바닥 패턴 (원거리 1명 지속 요구)
  - 페이즈 2: 근접 강타 + **맵 붕괴 영역** (근접 역할이 버프 획득 순환)
  - 페이즈 3: **첨탑 공명** (3인 순환 포인트 3개 · 번갈아 포인트 고정)
- **힐 전제**: 페이즈 2 "맵 붕괴 영역"은 회피 실패 시 큰 피해, 페이즈 3 "순환 포인트 고정"은 고정 중 피격 허용 전제. 현 설계는 **"물약 + 소규모 자힐로 버티는 예측 가능 패턴"** 전제에 가깝다.
- **A+B 채택 시 재조정 필요 여부**: **부분 필요 (경미)**.
  - 페이즈 2 맵 붕괴 영역 피해는 현행 유지 (회복 물약 쿨 25s로 대응 가능).
  - 페이즈 3 포인트 고정 중 피해는 **1초당 2.5% HP → 1초당 2.0% HP로 소폭 하향** 권고. 스태프 "냉기 장막" 자힐 0.5%와 창 "압박" 자힐 1.5%로 순환 가능하도록 튜닝.
  - 각인서 첫 직드랍 구조는 유지.

### 9-3. 3막 메인 30 — 세력별 결전 2종

#### A 루트 `황성 봉인 기사단장의 잔영`
- **현행 기믹**: 3인 순환 포인트(방패 지점 3개) + 텔레온 NPC 1명 대체 가능 + 페이즈 3 "정적의 일격 — 침묵 중 3인 중 1명만 타격".
- **힐 전제**: "정적의 일격" 페이즈는 **1명 피격 집중** 구조 — 집중 피격자가 **죽지 않고 견디는 힐 사이클** 전제.
- **A+B 채택 시 재조정 필요 여부**: **필요 (중간)**.
  - 집중 피격자 전용 생존 윈도우를 위해 **정적의 일격 피해량 -10%** 하향 검토.
  - 또는 **집중 피격 시 다른 2명이 자힐 기믹으로 "방패" 지점에 서면 피격자 피해 -15%** 신설(옵션 B 자힐 기믹과 연계).

#### B 루트 `이름을 삼킨 계약의 증인`
- **현행 기믹**: 3인 이름 새김 포인트(증언대 3개) + 12번 NPC 1명 대체 가능 + 페이즈 3 "이름 반환 일격 — 3인 중 1명이 본명을 외치면 타격 확정".
- **힐 전제**: "이름 새김 포인트"는 위치 고정 + 피해 감수 전제. A 루트와 유사.
- **A+B 채택 시 재조정 필요 여부**: **필요 (중간)**. A 루트와 동일 수준으로 포인트 고정 중 피해 -10% 또는 자힐 연계 튜닝.

### 9-4. 종합 정합 결론

| 보스 | A+B 채택 시 재조정 필요 | 작업량 |
|---|---|---|
| 2막 메인 19 | 없음 (일회성 버프 부여권 유지) | 0 |
| 2막 메인 20 페이즈 3 | 포인트 고정 중 피해 -0.5% 하향 | 소 |
| 3막 메인 30 A · B | 페이즈 3 피해 -10% + 자힐 연계 | 중 |

**종합**: 보스 기믹 전면 재설계 필요 없음. 페이즈 3 수치 튜닝 수준으로 대응 가능. 옵션 C 대비 작업량 1/5 이하.

## 10. 리스크 3건 + 완화안

### 10-1. 리스크 1 — A의 "물약 재고 격차 + 연금술 담합"

- **내용**: 회복 물약 쿨 25s 체제에서 고랭커·연금 고Lv 유저가 물약 회전을 독식. 영지 연금 Lv 7~15 도달 안 한 신규 유저는 **회복 물약 재고 부족**으로 파티 참여 제한.
- **완화안**:
  - 회복 물약 **커먼 등급 NPC 구매** 허용 (가격 1,500G, 연금 Lv 불필요). T2 진입 전까지는 골드로 커버 가능하게.
  - 영지 연금 Lv 7 도달을 2막 초반 자연 페이싱과 일치시켜 Lv 미달 유저 수 최소화.
  - 45일차 모니터링: 회복 물약 경매 시세 3,000G 초과 시 NPC 구매 가격 1,200G로 하향 트리거.

### 10-2. 리스크 2 — B의 "창·스태프 1자리 강제 현상"

- **내용**: 옵션 B 자힐 기믹이 창·스태프 2무기에 집중되면 3인 파티에서 "창 1 + 스태프 1 + 자유 1"이 표준 편성으로 굳어짐. 전사 태도·대검, 암살자 3무기, 마법사 완드·룬의 파티 선호도 하락.
- **완화안**:
  - 창·스태프 자힐 효과량은 "파티 생존 필수"가 아닌 "있으면 편함" 수준으로 엄격 제한 — 초당 0.4% HP 이하 유지.
  - 시즌1 45일차 파티 편성 통계 수집: "창·스태프 동시 편성률"이 60% 초과 시 자힐 수치 -30% 하향.
  - 다른 무기에도 **각자 다른 축의 생존 기믹 1개씩** 소폭 배분 고려 (예: 태도 "잔심 해제 시 0.5s 무적", 단검 "그림자 이동 시 자힐 1%", 룬 "룬 해제 시 파티 실드 2%"). 시즌1 튜닝에서 필요 시 반영.

### 10-3. 리스크 3 — 3막 메인 30 페이즈 3 피해 -10% 하향이 "레전드리 각인서 허용" 결전의 난이도를 체감상 약화

- **내용**: 3막 메인 30은 레전드리 각인서 첫 드랍 허용 결전. 피해 -10%가 "최종 결전의 무게"를 약화시키는 체감 반발 우려. 특히 기사단 루트 "정적의 일격" 긴장감 훼손.
- **완화안**:
  - 피해 -10%는 **"집중 피격자 전용"만** — 다른 2명 피해는 현행 유지.
  - 대신 **페이즈 3 지속시간을 +5s 연장** → 집중 피격 사이클이 **한 번 더 돌아옴**으로써 누적 피해는 동일. 체감상 "더 긴 결전"으로 무게 유지.
  - B 루트는 "이름 반환 일격" 타이밍 판정을 **0.3s 더 좁힘** (1.0s → 0.7s)으로 난이도 압박 전환.
