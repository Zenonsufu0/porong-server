# 포로 서버 이용 약관 v1 초안

## 문서 목적
"타 유저 조회 전역 거부" → "닉네임·진행도 공개 + 약관 동의 기반" 정책 번복에 따른 약관 동의 시스템 설계. feature-planner 1차 초안. **법무 검토 이전 게임 기획자·운영자 관점 초안**.

## 배경 (사용자 결정)
- 닉네임·플레이 기록 공개를 전제로 한 서버 운영
- 디스코드 공지·차원도서관 타인 조회·시즌 Top 10 공개 등 소셜·경쟁 축 복원
- 플레이어 동의 근거 확보 필수

## 약관 7항목 초안

### (a) 서비스 이용 동의
포로 서버는 마인크래프트 기반 커스텀 RPG 서버입니다. 접속한 플레이어는 서버 규칙·운영진 안내·서버 내 공지를 준수할 의무가 있습니다. 계정은 플레이어의 마인크래프트 UUID를 기준으로 식별되며, 본 계정의 모든 플레이 기록·자산·진행도의 관리 책임은 계정 소유자에게 있습니다. 운영진은 서버의 지속적 운영, 시즌 전환, 콘텐츠 업데이트, 점검, 긴급 종료 등을 수행할 권한을 가집니다.

### (b) 공정한 플레이 동의
포로 서버에서는 클라이언트 해킹·매크로 자동화·악성 모드·듀프(아이템 복제)·버그 악용·불법 거래(현거래)·타인 계정 도용 등 모든 어뷰징 행위가 금지됩니다. 위반 사실이 확인되거나 합리적으로 의심되는 경우 운영진은 조사 진행 중 임시 정지를 포함한 제재를 적용할 수 있습니다. 공정한 경쟁 환경 유지는 모든 플레이어의 공통 책임입니다.

### (c) 개인 식별 정보 공개 동의 (핵심)
플레이어의 인게임 닉네임·소속 길드·보스 처치 기록·경매 참여·낙찰 내역·시즌 랭킹·복선 회수 현황·주요 업적 달성 여부 등은 서버 내 공개 공간 및 **연동된 디스코드 채널·웹 페이지·차원도서관 조회 명령**을 통해 다른 플레이어에게 공개될 수 있습니다. 본 동의는 포로 서버의 소셜·경쟁 축(랭킹·명예의 전당·공개 공지)과 직결되므로, **거부 시 서버 이용이 불가**합니다. 단, 실명·이메일·전화번호 등 현실 개인정보는 수집·공개 대상이 아닙니다.

### (d) 채팅·커뮤니케이션 동의
서버 내 채팅·디스코드 연동 채널에서는 욕설·비하·혐오 표현·스팸·광고·타인의 현실 개인정보 노출 행위가 금지됩니다. 채팅 기록은 운영·분쟁 중재 목적으로 일정 기간 보관될 수 있으며 신고 접수 시 운영진이 조회할 수 있습니다. 반복 위반 시 채팅 제한·접속 정지 등의 조치가 적용될 수 있습니다.

### (e) 콘텐츠 소유권
플레이어가 서버 내에서 제작한 영지 구조물·캐릭터 진행도·인벤토리·경매 기록 등은 해당 플레이어의 플레이 결과물로 존중됩니다. 다만 이 결과물은 포로 서버라는 운영 환경 안에서만 의미를 가지며, 서버 외부로의 데이터 이전·상업적 이용·제3자 제공은 운영진 허가 없이 불가합니다. 시즌 종료·월드 리셋 시 일부 데이터는 초기화될 수 있으며, 이 경우 사전 공지를 원칙으로 합니다.

### (f) 운영 권한
운영진(GM)은 규칙 위반 조사·버그 상황 복구·콘텐츠 정상화·분쟁 중재 등을 위해 플레이어 캐릭터 상태 조회·인벤토리 확인·일시적 접속 제한·계정 정지·자산 회수 등의 조치를 수행할 수 있습니다. 운영진의 조치는 서버 규칙과 약관에 근거해 이루어지며, 명백한 오판이 확인되면 복구 절차를 제공합니다. 운영 판단은 서버 전체의 건전성을 최우선 기준으로 합니다.

### (g) 약관 변경 통지
본 약관은 서버 운영 정책·콘텐츠 구조·외부 연동 범위 변화에 따라 개정될 수 있습니다. 오타 수정·표현 정리 등 **마이너 개정**은 공지만으로 효력이 발생합니다. 개인정보 공개 범위·제재 기준·운영 권한 변경 등 **메이저 개정**의 경우 접속 시 재동의 절차를 거치며, 재동의 거부 시 접속이 제한됩니다. 모든 개정 이력은 버전 단위로 보존됩니다.

### (+) 미성년자 이용 시 보호자 동의
**만 19세 미만** 이용자의 경우, 본 약관에 동의하기 전 **부모 또는 법정 보호자의 동의**가 필요합니다(2026-04-19 사용자 결정, 만 14세 기준에서 상향 확정). 보호자는 자녀의 서버 이용·닉네임 공개·플레이 기록 공개에 대해 이해한 뒤 동의해야 하며, 이용 중 언제든지 동의를 철회하고 계정 삭제를 요청할 수 있습니다. 서버는 보호자 동의 절차를 완료하지 않은 미성년자의 일부 기능(공개 랭킹 노출·경매 참여 등) 이용을 제한할 수 있습니다. Discord ToS는 만 13세 이상 가입을 요구하므로, 디스코드 계정 보유 자체는 13세 이상 자인 효력이 있으나 포로 서버 내 추가 동의 라인은 만 19세 기준으로 관리합니다.

### (h) 개인정보 보관·파기 정책
서버 이용에 필요한 최소 식별 정보(UUID·디스코드 ID·닉네임·약관 동의 로그·IP 해시)는 **관련 법령이 정하는 보관 기간**을 준수하여 저장합니다. **시즌 전환·시즌2 이관 시점**에 운영·법적 보관 목적상 불필요한 정보는 파기하며, 시즌2 운영에 필요한 정보만 선별 이관합니다. 파기 대상·보관 항목·보관 기간 구체표는 법무 검토 후 부속 문서로 공개 예정입니다. 이용자는 자신의 정보 열람·수정·파기 요청을 운영진에 제출할 수 있습니다.

## UX 플로우 (2026-04-19 갱신 — 디스코드 선행 통합 + 봇 자동화 + 별도 월드)

> 약관 수집은 **디스코드 봇 임베드 단일 경로**로 통합. 체험 모드는 **별도 월드**로 구현한 "디스코드 연동 전 서버 소개 공간". 본 문서의 인게임 동의 UI는 비디스코드 경로(2차 웹 약관) 대비용으로 코드만 유지한다.

### 최초 접속 — 체험 월드 → 디스코드 봇 약관 → 메인 월드
1. 마크 최초 접속 시 **체험 월드(`experience_world`, 동일 Paper 인스턴스 내부 분리 월드)** 자동 스폰 — 세계관 연출·서버 소개·디스코드 초대 링크 NPC 배치
2. 유저가 디스코드 서버 입장 즉시 **봇이 자동으로 약관 임베드 노출**(웰컴 채널 또는 DM) → 동의 버튼 클릭
3. 동의 시 봇이 **디스코드 "동의자" 역할 자동 부여**(서버 내부 채널 접근 권한 개방) + **마크 `player_whitelist` 자동 인서트**(pending) 파이프라인 자동화
4. 유저가 체험 월드에서 `/confirm <OTP>` 입력 → `player_discord_link.is_verified=true` 확정 → 메인 월드로 `player.teleport()` 이관
5. 체험 월드는 "소개 공간"이므로 전투·경매·랭킹·영지 등 서버 핵심 기능은 WorldGuard·플러그인 훅으로 차단

### 디스코드 동의 모달 구성
- 상단: 약관 버전 (예: "포로 서버 이용 약관 v1.0")
- 본문: 임베드 + 외부 정적 링크(전문 표시), "아래까지 읽었음" 버튼 클릭 타임스탬프 기록
- 하단: 동의 / 거부 버튼
- 미성년자 체크박스: "본인은 만 19세 미만이며 보호자 동의를 받았습니다"

### 인게임 동의 화면 (비디스코드 경로 대비용, 기본 미노출)
- 2차 웹 약관 경로 도입 전까지는 코드 경로만 유지
- 기본 플로우에서는 호출되지 않음 — 디스코드 경로 실패·비상 복구 시 수동 활성화

### 동의 거부 시 처리
- 디스코드 모달에서 거부 선택 시 체험 모드 유지, 메인 서버 진입 차단
- 재동의 쿨다운 없음 — 재시도 자유

### 재동의 규칙 (2026-04-19 갱신 — 디스코드 단일 경로 통합 + 72h 유예)
- **마이너 개정**: 디스코드 공지·리마인드 1회, 자동 동의 처리
- **메이저 개정 (v1.1 등)**: **디스코드 봇 임베드 재동의 필수**. 운영 공지 선행 후 **재동의 유예 72h** 적용 — 유예 중에는 메인 월드 접속 허용하되 재동의 리마인드 노출. **72h 초과 후에도 재동의 미완 시 체험 월드로 강등**(메인 월드 접근 차단, 재동의 완료 시 즉시 복귀).
- 개정 시 MAJOR/MINOR 분류 명시 공지
- **타이머 충돌 우선순위 (2026-04-19 확정)**: 디스코드 강퇴 48h 유예와 v1.1 재동의 72h 유예가 동시 발동하면 **48h 강퇴 우선 적용**. 강퇴 타이머 소진 시 화이트리스트 제거·체험 월드 진입 모두 차단. 재동의 72h는 강퇴 미발동 유저에 한해서만 적용.

## 약관-기능 연동 규칙

### 미동의 차단 기능
- **접속 자체**: 최초 동의 거부 시 서버 입장 불가
- **디스코드 봇 조회**: 대상 플레이어가 v1 미동의 시 봇이 "비공개 플레이어" 응답
- **타 플레이어 조회**(`/차원도서관 <닉네임>`): 미동의 시 검색 결과 미노출
- **랭킹·명예의 전당**: 미동의 시 "익명" 처리
- **경매 참여**: 낙찰 기록 공개되므로 미동의 시 입찰 불가
- **공개 채팅·디스코드 연동**: 미동의 시 채팅 송출 제한

> 원칙: "미동의 = 접속 불가"이므로 위 규칙은 **메이저 개정 재동의 대기 중** 상태에 주로 적용

### 약관 동의 플래그 (플래그 저장소 v0.1 활용)
- `player.agreement.v1_agreed` = true/false
- `player.agreement.latest_version` = "v1.0"
- `player.agreement.is_minor` = true/false  *(기준: 만 19세 미만)*
- `player.agreement.guardian_consent` = true/false

### 미성년자 보호 플로우
- 디스코드 동의 모달에 연령(생년월일) 입력 + 만 19세 미만 체크박스 제공
- "만 19세 미만" 선택 시 보호자 동의 확인 문구·체크박스 추가 노출
- 미성년자 플래그 true 계정은 선택적 공개 기능 제한 가능
- 보호자 동의 철회 요청은 운영진 문의 채널(디스코드 티켓) 수동 처리
