# 포로 서버 오픈 질문 통합 레지스트리 v1

## 문서 목적
`docs/` 전체에 분산된 모든 `오픈 질문` / `남은 확인 포인트` 섹션을 카테고리별로 중앙화. 의사결정이 필요한 항목을 한 장소에서 조회·우선순위·해결 상태 추적.

## 운용 규칙
- 각 질문은 **주제 카테고리 → 하위 항목** 구조.
- 출처 파일 경로와 원문 유지. 결정이 내려지면 마스터플래닝 변경 섹션에 반영하고 본 레지스트리에서 **취소선 처리 후 결정 요약 1줄 남김**(히스토리 보존).
- 신규 오픈 질문은 해당 문서에도 기록하되, 본 레지스트리에 **동시 반영**하는 것을 원칙으로 함.
- 우선순위: **🔴 블로킹**(타 시스템 선행 조건) / **🟠 주요**(기능 완성도) / **🟡 일반**(디테일·튜닝).
