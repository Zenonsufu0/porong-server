# 05. 영지 / 농사 시스템 — CANON

> **[STATUS: CANON]** — 개인 영지, 저장고, 생산 시설, 공방 가공의 공식 기준.

## 공식 기준

| 항목 | 기준 |
|---|---|
| 사용자 노출 용어 | 개인 영지 |
| 외부 플러그인 | IridiumSkyblock은 섬 생성/보호/방문 껍데기만 담당 |
| EmpireRPG 책임 | 저장고, 시설, 가공, 작위, 생산 로직 |
| 시설 3종 | 약초 재배지, 광물 채굴기, 공방 가공기 |
| 공방 가공기 | 1대 = 대기 슬롯 3개, 동일 레시피 최대 8세트 |
| 공방 제작 대상 | 강화 흔적 3종 (별의 흔적 / 달의 흔적 / 태양의 흔적) 포함. 레시피 미확정 — 사용자 확정 필요. |
| 물류 | 호퍼/케이블 없음. GUI 기반 수동 조작 |
| 폐지 확정 | 마력 시스템, 발전기, 자동채굴기 |

## 충돌 처리

다른 문서에 마력 잔량, 발전기, 마력 소비 기계, 마력 과부하, 자동채굴기 기반 흐름이 남아 있으면 폐지된 구버전 설계로 본다.

영지 GUI 문서가 위 구버전 흐름을 전제하면 해당 GUI 문서를 현재 기준으로 재작성하거나 archive로 이동한다. CANON 기준을 GUI 문서에 맞춰 되돌리지 않는다.
