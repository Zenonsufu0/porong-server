# 05. 개인 영지 / 마인팜

## 핵심 구조

IridiumSkyblock은 개인 영지 생성과 보호만 맡고, 영지 성장과 생산 시스템은 EmpireRPG가 담당한다.

## 영지 저장고

DB 기반 가상 저장고. 커스텀 생산기 생산물은 바로 저장고로 들어간다.

출금 기본안:

- 좌클릭: 64개 출금
- 우클릭: 1개 출금
- 전체 입금 버튼 제공

## 마력

영지 전체 합산 마력 구조.

마력 과부하 발생 시 모든 마력 소비 기계를 정지하고, 경고음/화면 경고/GUI 빨간 표시를 띄운다.

## 기계

> **2026-05-19 확정: 자동채굴기(광맥 생성기/광물 추출기) 전면 제거. 영지는 농사 전용.**

- 자동 재배기
- 공방 가공기
- 영지 저장고

## 바닐라 농사/채광

초기에는 인벤토리 획득. 편의 권한 해금 후 영지 저장고 자동 입금 가능.

## 자동 심기

영지 레벨 해금 후, 골드/재료를 지불해 권한을 구매해야 사용할 수 있다.

## 공방 가공기

올인원 단일 가공기. 제작 시간과 대기열 제한이 있다.
