# 포로 서버 최종 남은 결정 사항

세부 수치와 제작식은 서브에이전트/분석 스킬로 검토한다.

## 운영 결정

1. 실제 오픈일
2. 45일 종료 예정일
3. 종료 후 연장 여부 판단 기준
4. 시즌 종료 후 성장 데이터 초기화 여부
5. 치장/칭호/컬렉션 기록 보존 여부

## 디스코드 결정

1. 임시 화이트리스트 만료 기간: 3일 또는 7일
2. 닉네임 오입력 수정 가능 횟수
3. 보스 알림 기본값: ON/OFF
4. 인증 유저 표시 칭호 최종명
5. 채널명 최종 표기

## 콘텐츠 결정

1. 시즌 최종보스를 오픈 시 포함할지, 시즌 중 업데이트로 둘지
2. 모험 도감/컬렉션북 중 최종 UI 명칭
3. 마력 동력/전력 용어 병기 여부
4. 영지 등급 명칭 최종 확정
5. 커피값 후원 안내를 서버 오픈 때부터 둘지, 나중에 둘지
