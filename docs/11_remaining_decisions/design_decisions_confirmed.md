## 2. 크리티컬 확률

**기본 크리티컬 확률: 10%**

- 장비/각인/잠재 버프 전 기본값 10%
- combat_balance_v1.md의 피해 공식에 반영:
  ```
  실제 피해 = 기본 피해 × (크리티컬 발생 시 × 크리티컬 배율, 아닐 시 × 1.0)
  크리티컬 배율은 별도 확정 필요 (기본값 1.5 제안)
  ```

## 3. T1 장비 지급 방식

**첫 접속 시 T1 장비 5슬롯 일괄 지급**

| 항목 | 내용 |
|---|---|
| 지급 시점 | 서버 최초 접속 |
| 지급 부위 | 무기 1개 (선택한 무기 종류) + 투구 + 상의 + 하의 + 신발 |
| 지급 등급 | T1 기본 등급 (0강, 잠재 없음) |
| 제한 | 1계정 1회 지급 |

**T1 장비 추가 획득 방법:**

| 방법 | 내용 |
|---|---|
| 영지 등급작 | 영지 공방 가공기에서 재료 + 기존 T1 소모 → 동급 또는 상위 품질 T1 제작 |
| 드랍/구매 | **1차 시즌 없음** (T1 파편 드랍 삭제, 드랍 테이블 업데이트 필요) |

> "T1 제작은 영지에서 하는 등급작으로만 한함"  
> = T1 장비를 전투 드랍으로 추가 수급하는 경로를 막고, 영지 활동과 강화에 집중  
> 기존 `drop_tables_v1.md`의 T1 장비 파편 드랍 → **삭제/대체 필요**

## 4. 큐브 수급 목표 (재설계)

**목표: 하루 30~70개 큐브 수급 가능**

기존 에이전트 설계(5~10조각/일)에서 대폭 상향. 큐브를 희귀 성장 재화가 아닌 자유로운 잠재 재롤 소모품으로 설계.

| 등급 | 드랍 경로 | 하루 수급량 목표 |
|---|---|---|
| 하급 큐브 | 고대 성벽 잔해 일반몹 직드랍 (10~15%), 전 필드 정예몹 (5~20%), 필드보스 S/A | 20~50개 |
| 중급 큐브 | 중후반 필드 정예몹 저확률, 시즌보스 1~2 보상 | 5~15개 |
| 상급 큐브 | 시즌보스 3/최종보스 전용 | 0~5개 |

> **큐브는 완성품 직드랍 구조** (조각 조합 방식 삭제 또는 선택 경로로만 유지)  
> 잠재능력 재롤이 자유로우므로 진짜 희귀한 것은 "유니크 옵션 조합 확률"

## 5. 45일 시즌 강화 목표 (전체 인원 30명 기준)

| 지표 | 수치 |
|---|---|
| 전체 예상 인원 | 30명 |
| 무기 20강 도달 목표 인원 | 4~5명 (13~17%) |
| 무기 18강 도달 목표 인원 | 약 8~12명 (25~40%) |
| 무기 15강 도달 목표 인원 | 약 15~20명 (50~65%) |

**현재 CSV 강화 성공률 기준 검증:**

| 강화 | 성공률 | 기대 시도 | 해당 단계 골드 기대값 |
|---|---|---|---|
| 18강 | 30% | 3.3회 | 17,160G |
| 19강 | 22% | 4.5회 | 29,250G |
| 20강 | 15% | 6.7회 | 54,940G |

→ 무기 1슬롯 20강 기대 총 비용: **약 80,000~100,000G** (1~19강 비용 포함)  
→ 하루 4h 플레이어 시즌 수입 추정 약 288,000~360,000G  
→ 무기 1슬롯에 시즌 수입의 25~35% 투자 시 20강 달성 가능 → **상위 4~5명 달성 가능한 설계로 확인**

> 5슬롯 전체 20강은 사실상 불가. 무기 20강이 시즌 최고 목표.
