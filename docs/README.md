# 포로 서버 문서 폴더 구조 v3

## 폴더 설명
- `00_index_and_execution`: 인덱스, Codex 실행 순서, 프롬프트/구현 가이드
- `01_boss_design_core`: 보스 설계 핵심, 패턴 스크립트, 체력/피해/딜타임
- `02_boss_rewards_ui_logs`: 데카, 결과창, 기여도, 보상, 상징 보상, reward seed
- `03_life_system_core`: 생활 구조, 영지, 제작식, 드랍, 가격, 경제 검토
- `04_consumables_and_alchemy`: 음식/만찬/물약 전용 문서, effect seed
- `05_classes_and_balance`: 전사/암살자/마법사 수치 초안, 밸런스
- `06_operations_api_dashboard`: 운영/DB/API/대시보드/관리자 명령
- `07_checklists_and_pending`: 체크리스트와 미확정 목록
