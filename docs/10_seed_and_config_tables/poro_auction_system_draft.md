# 포로 서버 파티 보스 드랍 경매 시스템 초안

## 문서 목적
`poro_master_planning.md` 각인서 드랍 체계 + 경매·보조 수급 경로 확장. 사용자 결정(파티 내 골드 입찰 + 수수료 10% 서버 회수 + 나머지 비낙찰자 균등 분배 + 7일 평균 시세) 기반 + economy-reviewer 1차 상세 설계.

## 사용자 확정 규칙
- 보스 클리어 후 **경매 대상 아이템(`auction_required` 플래그, 에픽 등급 이상 및 각인서·잠재재료·T2 핵심 제작재료) 드랍 시** 경매 개시
- 파티원 **골드 입찰** → 최고 입찰자 낙찰
- **7일 평균 시세 표시** (입찰 참고)
- 낙찰가에서 **수수료 10% 서버 회수** → 나머지를 **비낙찰자에게 균등 분배**

## 배경 (기존 확정)
- 각인서 드랍: 일반·고급 개인보상 / 희귀 경매+개인보상 소확률 / 영웅·전설 경매 전용
- 골드 월렛 통합 저장, 거래·우편·선물 시점 단방향 실체화
- 보조 수급: 월간 입찰 실패 보상 / 길드 기여도 교환소 / 시즌 패스 / 상위→하위 분해 일방향

## 1. 경매 UX·플로우

### 경매 개시
- **트리거**: 보스 체력 0 + 드랍 확정 + **3초 지연** 강제 팝업 (전투 종료 이펙트·보상 모션 후)
- **판정**: 아이템 마스터 `auction_required` 플래그 기반 + 등급 보조 (**에픽** 이상 + 각인서·잠재재료·T2 핵심 제작재료는 플래그 true 고정)
- **시간**: 기본 **60초** + 종료 10초 내 입찰 시 **10초 연장(최대 3회)** (스나이프 방지)
- **동시 다중 경매**: 순차 진행 기본, 아이템 등급 내림차순

### 입찰 UI 구성
- 상단: 아이템 아이콘·이름·등급
- 중단: **7일 평균 시세(절삭평균+중앙값)**, 최저 입찰가, 현재 최고 입찰가(**금액만 익명**)
- 하단: 내 입찰액 입력창, 남은 시간
- **현재 최고 입찰자 닉네임은 경매 종료 후 공개** (진행 중 익명 → 담합·눈치 압박 차단)

### 입찰 규칙
- **최소 단위**: 7일 시세의 5% 또는 **100골드 중 큰 값** (시세 없으면 100골드 고정)
- **입찰 수정**: **상향만 가능**, 취소·하향 불가, 종료 5초 전 수정 잠금
- **하한 입찰가**: 7일 시세 × **30%** (담합 방어, 미형성 구간은 기준시세 × 30%)
- **최소 입찰 간격**: 2초 (봇·매크로 방지)

### 낙찰·지급
- **월렛 홀드 방식**: 입찰 시점 월렛 예약 차감, 상향 시 차액만 추가 홀드, 종료 후 낙찰자 외 즉시 해제
- **아이템 지급**: **경매 전용 우편함 7일 보관** (인벤 가득 방어) + 골드 분배는 월렛 즉시 반영
- **T2 대량 스택**: 우편 일괄 수령

### 골드 분배 계산
```
수수료 = floor(낙찰가 × 0.10)
분배풀 = 낙찰가 - 수수료
인당 = floor(분배풀 / (파티원수 - 1))
잔여 = 분배풀 - 인당 × (파티원수 - 1) → 서버 회수 또는 분배풀 선두 귀속
```

| 파티 | 낙찰가 | 수수료 | 분배풀 | 인당 | 비고 |
|---|---|---|---|---|---|
| 4명 | 10,000 | 1,000 | 9,000 | 3명 × 3,000 | |
| 3명 | 10,000 | 1,000 | 9,000 | 2명 × 4,500 | |
| 2명 | 10,000 | 1,000 | 9,000 | 1명 × 9,000 | **담합 위험 최고** |
| 1명(솔로) | — | — | — | — | **경매 미발생, 개인 보상 전환** |
| 8명 | 10,000 | 1,000 | 9,000 | 7명 × 1,285 + 잔여 5골드 | |

## 2. 일주일 평균 시세 시스템

### 계산 방식
- **상·하위 10% 절삭 평균** (메인, UI 주 표시) + **중앙값** (보조 괄호 표시)
- 이상치 방어 + 분포 편향 인지 동시 제공

### 데이터 소스
- **전 서버 낙찰 기록 기본**, 아이템 단위 집계
- 지역 귀속 아이템(예: 서부 전용 각인서)은 `price_scope = region` 자동 적용
- 아이템 마스터 `price_scope` 컬럼 — global / region / tier

### 갱신 주기
- **1시간 단위 집계 잡** + 낙찰 시 즉시 카운트 증분
- 집계 테이블에 누적 합·개수만 유지, 평균은 조회 시 계산

### UI 배치
- **입찰창 내 표시 = 메인**
- **월렛 UI 하위 탭 "시세 대시보드" = 보조**

### 최초 출시 시세 부재
- 아이템별 **운영 기준 시세 사전 세팅** + UI에 **"시세 형성 중 (기준 시세 표기)"**
- 전환 조건: 낙찰 건수 30+ AND 경과 72시간+ → 실측 시세로 자동 전환

### 시세 조작 방지
- 동일 파티 구성(60%+ 겹침) 반복 낙찰은 **집계 제외**
- 시세 대비 **300% 초과 낙찰은 이상치**로 절삭 평균 모수 제외 (낙찰 자체는 유효)
- 동일 계정 주간 동일 아이템 **5회+ 낙찰은 가중치 0.5** 적용

## 3. 수수료 정책

### 등급별 차등 요율 (사용자 수용 확정, 등급 명칭 통합)
- **에픽 7% / 유니크 10% / 레전드리 13%**
- 기존 "희귀/영웅/전설" 용어는 "커먼/레어/에픽/유니크/레전드리" 5등급으로 통합 재매핑
- 저가 마찰 완화 + 고가 싱크 강화

### 수수료 회수 용도 3분할
- **40% → 월간 입찰 실패 보상 재원** (기존 보조 수급 경로와 직결)
- **40% → 소각** (시즌 골드 순증 억제)
- **20% → 길드 기여도 교환소 재원 보조**
- 비율은 시즌별 운영 상수로 조정 가능

### 특수 파티
- 길드 공성·레이드: **3%p 할인** (희귀 4% / 영웅 7% / 전설 10%). 전면 면제는 길드 편중 유발로 비권장.

## 4. 악용·편법 방지

### 담합 낙찰
- **하한 입찰가 = 7일 시세 × 30%** (미형성 시 기준시세 × 30%) — 하한 미만 입찰 불가
- 동일 파티 반복 + 시세 대비 40% 이하 낙찰 패턴 자동 알림

### 낙찰 후 골드 부족
- **월렛 홀드(예약 차감)** 방식으로 원천 차단 — 입찰 시점 잔고 부족이면 입찰 불가

### 고의 유찰
- **전원 미입찰 시 아이템 소멸 + 보스 재도전 기여도 전환**
- 최소 입찰가 이상 입찰자 1명이라도 있으면 해당 유저 단독 낙찰 (수수료 면제 또는 절반은 정책 선택)

### 봇·매크로
- 최소 입찰 간격 **2초**
- 종료 10초 이내 입찰 시 **자동 10초 연장** (스나이프 방지)
- CAPTCHA 비권장(UX 저해), 비정상 패턴 후처리 기반

## 6. 월간 입찰 실패 보상 (BP 시스템) 연동

### 참여 포인트 환산
```
BP = floor(입찰액 / 기준시세) × 등급가중
```
- 등급가중: 희귀 1.0 / 영웅 1.5 / 전설 2.0
- 기준시세 30% 미만 입찰 BP 0 (스팸 방어)
- 상향 수정은 **최종액 기준 1회만 인정**
- **낙찰 실패**: BP × 1.0 / **낙찰 성공**: BP × 0.3 (역차별 방지)

### 월간 상한·이월
- 유저당 월 BP 누적 상한 **500** (시즌 튜닝 대상)
- 미사용 BP 50% 감쇠 후 다음 달 이월
- 월 1회 교환소 오픈 — 각인서·잠재재료·골드 선택 교환

### 재원 연계
- 수수료 회수 40%가 실패 보상 재원으로 직접 투입
- 재원 부족 시 교환 레이트 자동 하향 조정(운영 상수 노출)

## 7. 운영 체크포인트
- 시즌 초 2주간 낙찰가 분포·유찰률·담합 의심 패턴 일간 모니터링
- 수수료 회수 / 시즌 골드 순증 비율이 **3% 미만이면 싱크 기능 약함** → 요율·용도 재검토
- 일반 유저 vs 상위권 월평균 참여·낙찰·BP 격차 **2배 초과** 시 하한 비율 조정
- 시세 조작 자동 탐지 로그 주간 리포트
- 경매 대상 아이템(`auction_required` 플래그) 판정 콘텐츠 추가 시 QA 체크리스트 포함

## 8. 오픈 질문
- 파티 최대 인원 상한 (분배·UI 설계 영향)
- 경매 전용 우편함을 기존 우편 분리 vs 플래그 식별
- 수수료 3분할 비율(40/40/20) 운영 상수 vs 하드코딩
- 하한 미만 단독 입찰 시 **수수료 면제 / 절반** 정책 선택
- BP 월간 상한 500·등급가중 수치 시즌1 시뮬레이션
