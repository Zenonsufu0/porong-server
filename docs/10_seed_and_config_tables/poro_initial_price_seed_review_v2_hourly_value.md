## 1. 시간당 가치 역산 — 핵심 수식

### 1-1. 정의

- **GPH (Gold Per Hour)** = 1시간 사냥으로 플레이어가 얻는 **골드 환산 기대값**.
  - 순수 골드 드랍 + 보스 고유 재료의 경매 환산 기대값 + T1 재화(조각·파편·강화석) 환산 기대값 합산.
  - 치장·각인서는 자율 시장 원칙에 따라 **역산 기준 GPH에 포함하지 않는다**(변동성 과대, 시세 관측치 우선).
- **MPH_x (Material Per Hour, 재화 x)** = 1시간 사냥으로 얻는 해당 재화 x의 **기대 수량**.

### 1-2. 1차 수식 — NPC 시드 상한

```
npc_seed_price(x) = GPH / MPH_x × k_category
```

- `GPH` : 해당 시점(막·파밍 구간)의 대표 GPH 기댓값.
- `MPH_x` : 같은 구간에서 재화 x의 시간당 공급 기댓값.
- `k_category` : 카테고리 계수. 기초 소비품 0.9~1.0, T1 강화 재화 0.7~0.85, 치장 시드 기준선 3~5, 강화 보조 물약은 제작 원가 변형식 사용.
- 원칙: **1시간 사냥으로 얻는 기대 골드 ≥ 같은 1시간에 NPC에서 살 수 있는 동량 재료 가치**. 시간당 기회비용 기반 자연 상한.

### 1-3. 2차 수식 — 랜덤박스 가격 역산 (v1 수식 유지, 명시)

```
box_price = Σ(rarity_expected_value × probability) × discount_factor
// discount_factor = 0.80~0.90  (v1 0.70~0.80에서 +0.10 상향 — 시간당 가치 역산 맥락에서 유저 유인 소폭 축소)
```

### 1-4. 대표 GPH 기댓값 (1차 추정, 1막 기준)

보스 드랍 매트릭스 v1 §2-2 카테고리 A 드랍량과 §3 1막 평균 플레이 기간을 결합한 1차 추정.

| 구간 | 일반 유저 GPH | 코어 유저 GPH | 비고 |
|---|---:|---:|---|
| 1막 초반 (메인 1~5) | 1,200 | 1,800 | 필드 몹 위주, 필드보스 주 1~2킬 |
| 1막 중반 (메인 6~10) | 1,800 | 2,800 | 필드보스 정규 루트 진입 |
| 1막 후반 (메인 11~15) | 2,400 | 4,000 | 수도 상위 필드보스 + 영지 부수입 |
| 2막 초입 | 3,500 | 6,000 | v2 후속 재추정 대상 (본 문서 추정 범위 밖) |

> 본 문서 가격 시드는 **1막 중반(GPH ≈ 1,800)을 기준선**으로 삼는다. 시즌1 오픈 시점 대다수 유저가 머무를 구간.

### 3-3. (c) 영지 NPC 청사진·블록 — 진행도 게이트 (역산 미적용)

v1 §3-2 그대로 유지. **아르카논 접촉 이후 해금 원칙과 막별 해금 시점이 가격 축을 지배하므로 GPH 역산을 적용하지 않는다**. 청사진 가격은 해당 막 평균 보유 골드 대비 상대치로 설정.

- 역산 충돌 지점: Lv2→Lv3 청사진 22,000G를 1막 중반 GPH 1,800으로 역산하면 "약 12시간 누적 골드"인데, 이는 영지 루트 파밍 + 메인 퀘스트 보상 합산 예상치와 부합. 다만 **역산보다는 "해당 막 누적 골드 30~40%"를 기준**으로 삼는 것이 일관적.
- 따라서 (c)는 본 v2 범위 밖으로 분리, v1 §3-2 가격 표를 그대로 승계.

- **장점**: 희귀 이상의 초기 진입 문턱 완화. 자율 시장에 가격 발견 여지 충분히 열어둠.
- **위험요소**: 최초 시드가가 과도하게 낮아 경매 평균가가 시드가를 크게 상회하면 "시드가 기준선으로서의 정보 기능" 상실. 시드가가 아닌 확정 구매 축이 상한 기능을 포기한다는 점을 운영 공지로 명시 필요.
- **조정 제안**: 30일차 영웅·전설 해금 시점 +7일 누적 경매 평균가가 최초 시드가의 150%를 초과하면 최초 시드가 소폭 상향(시즌2 재검토).

### 3-5. (e) 랜덤박스 — 역산 적용 (가격만)

## 4. 45일차 자동 조정 루프 찬반 + 권장안

### 4-1. 이탈 감지 지표 정의

시간당 가치 기대값과 실제 경매 평균가 이탈을 `auction_category_summary`·`auction_daily_summary`에서 계산.

- **지표 D1**: `observed_hourly_value = auction_daily_summary.avg_price × MPH_x_observed` (품목별, 일 단위).
- **지표 D2**: `deviation = (observed_hourly_value - expected_hourly_value) / expected_hourly_value`.
- **트리거**: `|deviation| > 0.30` 이 **7일 이상 지속** 시 해당 품목 NPC 시드가 조정 후보.

### 4-2. 자동 조정 루프 — 찬반

| 축 | 도입 (완전 자동) | 미도입 (수동) | 권장 (반자동 — 알람 + 수동 승인) |
|---|---|---|---|
| **반응 속도** | 빠름 | 느림 | 중간 |
| **시세 왜곡 리스크** | 중~높음 — 조정이 시장에 2차 신호로 작용 | 낮음 | 낮음 — 승인 단계 있음 |
| **운영 부하** | 낮음 | 높음 | 중간 |
| **자율 시장 원칙 부합도** | 낮음 — 운영 자동 개입이 잦음 | 높음 | 높음 |
| **버그·오탐 리스크** | 높음 — 자동 조정 오작동 시 연쇄 | 낮음 | 낮음 |

### 4-3. 권장안 — **반자동 (알람 + 수동 승인)**

- 이탈 >±30% 지속 7일 감지 시 **디스코드 운영 채널 알람 + 조정 제안 수치 자동 생성**.
- 실제 NPC 시드가 변경은 **운영자 수동 승인** 후 `/shop reload` 핫리로드.
- 자동 조정은 v1 리스크 R1(시장 왜곡) 재점화 가능성이 있으므로 **시즌1에서는 도입하지 않는다**. 시즌2 이후 실측 데이터 3개월치 확보 후 재검토.
- **조정 제안 수치 자동 생성 공식**: `suggested_price = current_price × (1 + deviation × 0.5)` — 편차의 절반만 반영(과보정 방지).

## 5. 리스크 4개 + 완화안

### 5-1. 리스크 R1 — 시즌1 초기 파밍 속도 추정 오차에 GPH 기준선이 민감

- 증상: 1막 초반 7일차 실측 GPH가 추정치(1,200) 대비 ±30% 이탈. 모든 카테고리 시드가 기준선이 흔들림.
- 영향: (a), (b), (e), (f) 전체 재계산 필요. 오픈 초기 혼란.
- 완화안:
  - **7일차 GPH 실측 체크포인트** (`wallet_log` 시간당 집계)를 필수 게이트로 설정.
  - 7일차 GPH 실측치가 추정과 ±20% 이내면 v2 그대로 유지, 초과 시 `k_category` 계수만 조정하고 가격 표는 22일차까지 보류.
  - 시즌1 오픈 전 **Closed Beta 3일간 GPH 예비 관측** 권장.

### 5-2. 리스크 R2 — 카테고리 (c) 진행도 게이트와 역산 충돌 가능성

- 증상: Lv4·Lv5 청사진 진행도 게이트 가격(65,000G·150,000G)이 해당 막 GPH 역산 기준치와 2배 이상 괴리.
- 영향: "(c)만 역산 예외"가 유저에게 비일관 신호로 읽힘. 청사진 판매 부진 또는 골드 인플레 압력.
- 완화안:
  - **공지 톤 분리**: (c)는 "막별 진척 게이트"로 명시, 역산이 아닌 진행도 기반임을 공식화.
  - 트리거 T5(v1 §6-5 청사진 판매 부진) 유지 — 월 구매 건수 < 예상치 50%면 -20% 임시 인하.
  - Lv2 청사진은 메인 5·6 보상 지급 병행 경로 검토(v1 §3-2 조정 제안 유지).

### 5-3. 리스크 R3 — 강화 보조 물약 제작 원가 변동 시 역산 재계산 주기 미정

- 증상: 약초·마력 결정 등 재료 시세가 영지 설비 Lv 업에 따라 급변해도 NPC 가격은 정적.
- 영향: 연금 직군 수익성 왜곡, (f) 전반의 "제작 원가 × 마진" 식의 기초가 무너짐.
- 완화안:
  - **재계산 주기 22일차·45일차 2회 고정**(§3-6 조정 제안).
  - 이탈 감지 반자동 루프(§4-3)에 (f)를 포함.
  - 영지 Lv 업 이벤트 시 -7일 ~ +7일 구간 수동 보정 허용.

### 5-4. 리스크 R4 — 치장 최초 시드가 대폭 하향에 따른 수집 수요 조기 포화 (신규)

- 증상: 희귀·영웅·전설 확정 구매가가 v1 대비 40~50% 하향돼 시즌 초반 2~3주 내 대량 구매 발생, 자율 시장 치장 경매 수요 고사.
- 영향: 치장 경매 유동성 저하, 수집 플레이어의 "뽑기 체감" 약화.
- 완화안:
  - **영웅·전설 확정 구매 30일차 해금** 유지(v1 §3-3 조정 제안).
  - **일일/주간 개인 구매 상한** 도입 검토 — 영웅 치장 주 1개, 전설 치장 월 1개.
  - CP5(v1 §6) 치장 경매 시세 ±30% 변동폭 감시 주기를 시즌 초 14일간 ±20%로 강화.
