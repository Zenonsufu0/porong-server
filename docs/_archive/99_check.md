### 1.2 문제: 23강 달성 필요 강화석 vs 서버 전체 공급량

**23강 달성에 필요한 강화석 (0→23강, 기댓값 기준):**

```
구간        | 기대 소모 (경제 문서 역산)
0→22강      | ~40,885개  (economy_numbers_v2 §0 표기)
22→23강     | 기댓값 500회 × 72개 = 36,000개
0→23강 합계 | ~76,885개  (무기 1슬롯 단독)
```

**서버 30명 기준 45일 총 강화석 공급 추정:**

전제: 2h 유저 12명, 3h 유저 13명, 6h+ 유저 5명

```
2h 유저: 26개/일 × 12명 × 45일 = 14,040개
3h 유저: 58개/일 × 13명 × 45일 = 33,930개
6h+ 유저: 124개/일 × 5명 × 45일 = 27,900개
서버 합계: 약 75,870개
```

**전 유저 5슬롯 강화 소비 (선발대 제외):**

```
2h 유저 12명 × 5슬롯 15강 1,116개 = 13,392개
3h 유저 13명 × 5슬롯 18강 4,506개 = 58,578개
소계: 71,970개

선발대(6h+) 5슬롯 18강 소비: 5명 × 4,506개 = 22,530개
→ 전 유저 합계 소비: 94,500개
→ 공급(75,870개) - 전 유저 18강 소비(94,500개) = -18,630개 (공급 부족)
```

**결론:** 서버 전체 강화석 공급이 전 유저의 5슬롯 18강 소비조차 감당하기 어렵다. 선발대가 시장에서 추가 매입하여 23강을 달성하는 것은 현재 공급 수준에서 수학적으로 불가능하다.

### 3.1 역전 구조 설계

- 1~5강: 2,000~8,000G (낮지 않음, 초반에도 골드 소비감 있음)
- 10강: 피크 20,000G
- 15강: 12,000G (하락 시작)
- 18강: 5,000G
- 21강 이상: 1,500G (낮게 → 강화석이 실질 제한 요소)

**역전 구조 목적:** 초반 유저는 골드가 부족해서 1~10강 진입을 느끼고, 후반 유저는 강화석이 부족해서 21강+ 진입이 막힌다. 골드를 많이 모은 선발대가 강화석을 사는 것이지, 골드 비용 때문에 막히는 것이 아니어야 한다.

이를 위해 5슬롯 15강 목표 소모를 1,700개가 아닌 1,200개 수준으로 낮추거나, 일일 수급을 50개/일(2h 기준)로 더 높여야 한다.

## 5. 제안 21~25강 성공률

### 5.1 목표 조건

- 조건 A: 서버 상위 1~3명이 45일 내 23강 달성 가능
- 조건 B: 0→23강 필요 강화석이 선발대 자체 수급 + 시장 매입으로 달성 가능한 범위
- 조건 C: 24~25강은 45일 내 달성이 사실상 불가능한 이론치로 유지

### 5.2 선발대 매입 가능 강화석 추정

선발대(6h+, 5명) 45일 자체 수급: 246개/일 × 45일 = 11,070개/명
선발대 5슬롯 18강 소비: 약 4,506개(기존 소모 곡선) 또는 6,550개(제안 소모 곡선)

```
자체 수급 11,070개 - 5슬롯 18강 소비 6,550개 = 잔여 4,520개/명

시장에서 추가 매입: 2h/3h 유저 잉여를 가정.
  - 2h 유저 12명 × 380개 = 4,560개 (30일에 15강 달성 가정)
  - 3h 유저 13명 × 460개 = 5,980개 (38일에 18강 달성 가정)
  - 시장 공급 합계: 약 10,540개

선발대 1명 총 강화석 (자체 + 시장 매입 5명 나눔):
  4,520 + 10,540 / 3 (상위 3명 집중 매입) = 4,520 + 3,513 = 약 8,033개
```

**23강 달성 필요 강화석 역산:**

선발대 최상위 1명이 약 8,000개를 확보 가능하다면, 0→23강에 8,000개로 달성 가능한 성공률은?

```
0→20강 소모 (제안 파편 곡선): 약 14,560개 → 이미 8,000개를 초과한다.

문제: 제안 파편 소모 곡선이 너무 높아 8,000개로는 0→20강도 불가능하다.
```

**현실적 판단:** 선발대 1명이 45일 내 확보 가능한 강화석은 자체 수급 11,070개 + 시장 매입 3,500개 = 14,570개 수준이 현실적 상한이다.

이 범위 안에서 0→23강을 달성하려면 소모 곡선과 성공률을 함께 조정해야 한다.

### 5.3 균형점 도출

**접근법:** 제안 파편 소모 곡선 유지 시, 0→23강 목표를 "선발대 자체 수급으로는 불가, 시장 협력으로는 가능"으로 설정.

즉, 0→23강 기대 강화석 = 선발대 1명 45일 수급(11,070개) × 3명 = 33,210개 수준.
이를 선발대 3명이 협력(각자 11,070개, 서로 공유 아닌 개인 수급)하면 각 11,070개.

단일 선발대 1명 기준 23강 달성 가능 수준 = 자체 수급 11,070 + 시장 매입 3,500 = 14,570개.

**목표:** 0→23강 기대 강화석 ≈ 12,000~18,000개.

이를 위한 파편 소모와 성공률 재조합:

```
0→20강 목표 소모: 약 5,000~7,000개
20→21강: 약 1,500~2,000개
21→22강: 약 2,500~3,500개
22→23강: 약 3,500~5,000개
합계: 약 12,500~17,500개
```

**파편 소모를 줄이되 성공률도 조정하는 중간 경로:**

| 강화 | 무기 파편(시도당) | 제안 성공률 | 기대 시도 | 기대 파편 소모 |
|---|---:|---:|---:|---:|
| 17강 | 40개 | 10% | 10.0 | 400개 |
| 18강 | 50개 | 5% | 20.0 | 1,000개 |
| 19강 | 62개 | 2% | 50.0 | 3,100개 |

| 22강 | 118개 | 1.0% | 100.0 | 11,800개 |
| 23강 | 147개 | 0.5% | 200.0 | 29,400개 |

0→20강: 약 5,000개(1~16강) + 400 + 1,000 + 3,100 + 7,700 = 17,200개

이미 17,200개로 선발대 자체 수급 상한(11,070개)을 초과한다.

**결론:** 파편 소모 곡선을 현재(economy_numbers_v2 §0)보다 높이는 방향은 23강 달성을 불가능하게 만든다. 성공률 상향이 반드시 선행되어야 한다.

### 6.3 자연 시세 추정

선발대 소비 (영지 550,000G + 5슬롯 18강 골드 약 1,458,000G + 편의 425,000G): 약 2,433,000G
선발대 잉여 골드: 약 3,399,000G

선발대가 22강 도전에 사용 가능한 골드: 약 3,400,000G
매입 가능 강화석 수: 시장 공급 12,820개

이론 최대 시세: 3,400,000G ÷ 12,820개 = **약 265G/개**

그러나 선발대가 강화석 매입에 전액을 쓰지는 않으므로, 실제 시세는 이보다 낮게 형성된다. 현실적 추정:

| 조건 | 추정 시세 |
|---|---|
| 공급 < 수요 (기본 구조) | 100~200G/개 |
| 서버 초반 파편 희소 시 (1~15일차) | 200~300G/개 |
| 서버 중반 파편 안정 후 (25~35일차) | 80~150G/개 |
| 서버 후반 선발대 수요 집중 (40~45일차) | 150~250G/개 |

**판단:** 자연 시세 100~200G/개는 라이트 유저에게 충분한 판매 동기(30G/개 이상 기준 충족)를 제공한다. 오히려 너무 높아질 위험이 있다.

**경고:** 시세가 250G/개를 초과하면 "강화석=골드"가 되어 강화석이 사실상 화폐로 작동한다. 이 경우 운영 개입이 필요하다.

## 8. 보조재 충돌 검토

### 8.1 보조재 B (×1.35, 12~16강)

역전 설계와 충돌하지 않는다. 12~16강은 골드 비용이 하락하는 구간이며, 보조재 B는 강화석 소모를 줄이는 효과이므로 "강화석 절약 vs 골드 지출" 선택지로 자연스럽게 작동한다.

### 8.2 보조재 C (×1.50, 17강+)

역전 설계와 **부분 충돌**이 있다. 17강 이후 골드 비용이 낮아지는 구간에서 보조재 C 제작에 골드가 들어가는 구조(기본형 2,500G/개 × 30회 = 75,000G)는 후반 골드 소비를 의도치 않게 높인다.

단, 이는 선발대 구간에서만 해당되며, 선발대의 골드 수입이 충분하므로 심각한 충돌은 아니다. 다만 보조재 C 대체형(균열 파편 30개 + 4,000G)은 골드가 더 많이 드는 경로인데, 이를 주력 루트로 권장하는 것은 역전 설계와 충돌한다. 기본형(핵심 재료 1개 = 5회 분량)을 주력으로, 대체형은 재료 부족 시 보조 루트로 재정립 권장.

### 항목 1. 문서 불일치 감지 및 판정

#### 1-A. 선발대 6h+ 강화석 일일 생산량: 124개 vs 246개

**충돌 위치:**
- `enhancement_droprate_v1.md §5.1`: 정예몹 40마리 × 2.50개 + 필드보스 2.0회 × 12개 = **124개/일**
- `economy_numbers_v2.md §5 (신규 재계산 섹션)`: 정예몹 40마리 × 4.0개 + 일반몹 160마리 × 0.15개 + 필드보스 2.5회 × 25개 = **246개/일**

**계산 검증:**

| 수급원 | enhancement_droprate_v1 기준 | economy_numbers_v2 기준 |
|---|---:|---:|
| 정예몹 기댓값/마리 | 2.50개 (필드5 현행) | 4.00개 (A 방향 적용 후) |
| 정예몹 40마리 | 100개 | 160개 |
| 일반몹 160마리 | 0개 (없음 원칙) | 24개 (0.15개 × 160마리) |
| 필드보스 보상 | 2.0회 × 12개 = 24개 | 2.5회 × 25개 = 62개 |
| 합계 | 124개 | 246개 |

**판정:** economy_numbers_v2.md §5의 246개는 **A 방향 적용 후** 수치이고, enhancement_droprate_v1.md §5.1의 124개는 **A 방향 적용 전** 현행 수치다. 두 수치 모두 각자의 전제 조건 하에서 정확하다. 불일치가 아니라 **"기준 시점" 차이**다.

올바른 값: **A 방향 결정 전 = 124개, A 방향 결정 후 = 246개** (양쪽 모두 유효, 전제 명시 필요)

문서 수정 방향: `enhancement_droprate_v1.md §5.1`에 "A 방향 적용 전 수치" 주석 추가, 또는 §5.1을 A 방향 기준으로 재계산한 별도 표로 교체.

#### 1-B. 일반몹 강화석 드랍: "없음" vs 3~6%

**충돌 위치:**
- `enhancement_droprate_v1.md §2.1`: 일반몹 강화석 파편 드랍 **없음** (모든 필드 공통, 2026-05-15 확정)
- `drop_tables_v1.md §1`: 폐광 지대 3%, 오염된 수로 4%, 무너진 초소 5%, 고대 성벽 잔해 6% 드랍

**판정:**
두 문서의 작성 시점이 다르고, 이번 A 방향 결정으로 인해 방향이 바뀌었다. 1차 검토 보고서 §7 "중요" 항목에서도 이미 충돌이 기록되었다.

- `enhancement_droprate_v1.md §2.1`의 "없음"은 더 최신 확정값으로 기재되어 있으나, A 방향 결정은 이 원칙을 수정하는 방향이다.
- `drop_tables_v1.md §1`의 수치는 사실상 A 방향의 일반몹 측 수급 설계와 일치한다.

**올바른 값 판정: A 방향 채택 시 drop_tables_v1.md §1의 수치가 의도된 설계값이다.**
`enhancement_droprate_v1.md §2.1`의 "없음" 원칙을 "필드2~5 소량 허용"으로 개정해야 한다.

단, A 방향이 정예몹 대폭 상향이 주목적이므로 일반몹 드랍 허용 범위를 drop_tables_v1.md §1보다 소폭 상향 조정(필드2 5%, 필드3 7%, 필드4 10%, 필드5 15%)하는 것이 1차 검토 §4.3 제안과도 일치한다.

#### 1-C. 큐브 시스템: 단일 큐브 vs 3등급 분리

**충돌 위치:**
- `economy_numbers_v2.md §0`: 잠재 큐브 1종 (커먼 40% / 레어 35% / 에픽 20% / 유니크 5%)
- `drop_tables_v1.md §5`: 하급(커먼~레어) / 중급(레어~에픽) / 상급(에픽~유니크) 3등급 분리

**판정:**
`drop_tables_v1.md §6 "에이전트 간 조정 필요 사항"` 자체에 이 충돌이 이미 기록되어 있다. 해당 섹션의 조정 방향 컬럼: "조각 10개 공통 + 추가 재료로 등급 결정하는 통합안 권장".

**올바른 값 판정: drop_tables_v1.md §5의 3등급 분리 방식이 더 정교한 최신 설계다.**
economy_numbers_v2.md §0의 단일 큐브 확률표는 하급 큐브의 확률표로 재해석하거나, 3등급 분리 방식을 채택하는 방향으로 통일해야 한다.

큐브 1개 = 조각 10개 + 등급별 추가 재료 구조는 drop_tables_v1.md §5가 확정안.

#### 1-D. 시즌보스 주간 캡: 기존 "제거 결정" vs drop_tables_v1.md §4 "주간 첫 클리어/반복 보상" 유지

**충돌 위치:**
- 이전 결정 기록: 시즌보스 주간 캡 제거 방향
- `drop_tables_v1.md §4`: 주간 첫 클리어 = 풀 보상, 주간 2회차 이후 = 반복 보상(감소), 매주 월요일 초기화

**판정:**
drop_tables_v1.md §4는 "주간 캡 완전 제거"가 아니라 "주간 첫 클리어 풀 보상 + 이후 반복 보상 감소"로, 사실상 소프트 캡 구조다. 이는 "캡 제거"와 양립 가능하다. 반복 보상이 0이 아니라 감소된 보상을 계속 제공하는 구조이기 때문이다.

**판정: 두 설계는 충돌하지 않는다.** "주간 캡 제거"는 "반복 참여 자체를 막지 않는다"는 의미로, drop_tables_v1.md §4의 "2회차 이후 감소 보상"과 조화된다. 추가 참여는 언제든 가능하되 보상이 줄어드는 구조가 최종안.

### 항목 2. A 방향: 필드4~5 드랍률 재설계 — 22강 시도 8~10회/일 달성 가능성 계산

#### 2-B. 현행 선발대(6h+) 수급: 124개/일

- 합계: 124개

부족분: 512개 - 124개 = **388개/일 부족** (8회 기준)

#### 2-C. A 방향 최대 상향 시 달성 가능 수급 계산

**정예몹 기댓값 상향 한계 분석:**

정예몹 처치 40마리로 512개를 달성하려면: 512 / 40 = 12.8개/마리 필요

현실적으로 구현 가능한 정예몹 기댓값 최대치: 분포 0% / 1% / 5% / 20% / 35% / 39% (6개) 기준

기댓값 = 0×0 + 1×0.01 + 2×0.05 + 3×0.20 + 4×0.35 + 5×0.39 = 0 + 0.01 + 0.10 + 0.60 + 1.40 + 1.95 = **4.06개/마리**

분포를 더 올릴 경우 (최대값 6개, 분포 조정): 6개 고정 = 기댓값 6.0개/마리 (비현실적, 게임 밸런스 파괴)

현실적 상한: **필드5 기댓값 4.0~5.0개/마리**

**일반몹 추가 드랍 계산 (6h+ 기준, 160마리/일):**

| 일반몹 드랍 확률 | 추가 수급 |
|---:|---:|
| 15% × 160마리 | 24개 |
| 20% × 160마리 | 32개 |
| 30% × 160마리 | 48개 |

#### 2-D. 결론: 8~10회/일 달성 가능 여부

**판정: 위험 — 8~10회/일은 현실적으로 달성 불가능하다.**

22강 시도당 소모 64개(현행 값) 기준으로, 정예몹 기댓값을 물리적 한계(6개/마리)까지 올려도 375개 수준이다. 이는 5.9회/일에 해당한다.

8회 달성을 위해서는 512개/일이 필요하며 이를 위해선 정예몹 기댓값이 약 10개/마리가 되어야 한다. 이는 설계 불가능한 수준이다.

**현실적으로 달성 가능한 최대 시도 횟수:**

| 적용 방향 | 예상 22강 시도/일 |
|---|---:|
| 현행 (A 방향 미적용) | 약 1~2회 |
| A 방향 기본 (정예몹 4.0개, 일반몹 15%, 필드보스 상향) | 약 3~4회 |
| A 방향 강화 (정예몹 5.0개, 일반몹 20%, 필드보스 최대 상향) | 약 4~5회 |

**대안 권고:**

22강 시도 8~10회/일 목표는 "22강 소모 파편(64개)" 기준에서 온 것이므로, 방향을 두 가지로 분리해야 한다.

방향 A-1: 드랍률 상향 (현실 한계: 5~6회/일)
- 정예몹 필드5 기댓값을 4.0개로 확정 (현행 2.5에서 +60%)
- 일반몹 필드5 15~20% 드랍 허용
- 균열 파수꾼 기본 보상 20~30개로 상향

방향 A-2: 22강 시도당 소모 파편 수 하향 조정 (병행 필수)
- 현행 22강 파편 소모 64개를 40~48개로 하향
- 이 경우 A 방향 기본(246개/일) 기준으로도 246/48 = **5.1회/일** 달성

방향 A-1 + A-2 동시 적용 시:
- A 방향 강화 + 22강 소모 40개: 319개 / 40개 = **8.0회/일** 달성 가능

그러나 22강 파편 소모 하향은 economy_numbers_v2.md §0 확정값을 건드리는 것이므로 별도 결정이 필요하다.

**권장안: A 방향 기본 드랍 상향 + 22강 파편 소모 48개로 소폭 하향 조합**
- 결과: 246개 / 48개 = **5.1회/일** (타협안), 강화 A-2 조합 시 **6.6회/일**

- 하드 유저: 유니크 2~3개 기대 (상급 큐브 15~20개 기준)
- 선발대: 유니크 4~7개 (상급 큐브 30~45개 기준)

#### 5-D. 큐브 사용 골드 5,000G/회 총 부담 vs 골드 수입

| 유저 유형 | 큐브 사용 횟수 (45일) | 큐브 골드 소모 | 45일 수입 대비 비율 |
|---|---:|---:|---:|
| 라이트 (2h) | ~45회 | 225,000G | ~15% |
| 하드 (3h) | ~67회 | 335,000G | ~10% |
| 선발대 (6h+) | ~108회 | 540,000G | ~9% |

**판정: 주의 — 라이트 유저의 큐브 골드 부담이 수입의 15%로 다소 높다.** 단, 라이트 유저는 전체 조각을 큐브로 전환하지 않고 일부를 경매소에 판매할 수 있다. 큐브 조각 경매 시세가 형성되면 골드 부담은 줄어든다. 라이트 유저가 큐브 조각을 팔아 골드 수입을 보충하는 구조가 자연스럽게 성립한다.

### 항목 6. 종합 건강 진단

#### 6-A. 라이트 유저 (2h): 45일 내 15강 달성 가능한가?

**전제:** A 방향 적용 후 일일 수급 38개, 5슬롯 15강 소모 1,116개

```
1,116개 / 38개 = 29.4일 → 약 30일에 달성
```

**판정: 정상** — 45일 내에 여유 있게 달성 가능하다. 30일 달성 후 15일의 잉여 기간에 강화석을 시장에 공급하거나 16~17강 추가 강화를 시도할 수 있다.

단, enhancement_droprate_v1.md §5.2 기준(26개/일)에서는 43일이 필요해 타이트하다. A 방향 적용이 라이트 유저 경험에 중요하다.

#### 6-B. 하드 유저 (3h): 45일 내 18강 가능한가?

**전제:** A 방향 적용 후 일일 수급 92개, 5슬롯 18강 소모 4,506개

```
4,506개 / 92개 = 48.9일 → 약 49일 필요 (45일 초과)
```

순수 자체 수급만으로는 45일 내 달성 불가. 그러나:
- 천장 시스템 활용 시 실질 소모 감소 효과
- 시장 매입 병행 시 달성 가능

```
부족량: 4,506 - (92 × 45) = 4,506 - 4,140 = 366개
366개 / 시장 시세 100G = 36,600G 투자로 해소 가능
하드 유저 잉여 골드 1,459,000G의 2.5% 수준
```

**판정: 정상** — 소량의 시장 매입(366개, 36,000G 수준)으로 달성 가능하다. 시장 형성이 전제되어야 한다.

#### 6-C. 선발대 (6h+): A 방향 적용 시 22강 도전이 현실적인가?

항목 4-D 분석 결과:
- 22강 달성 필요 강화석: ~40,885개
- 선발대 자체 수급 45일: 11,070개
- 시장 매입 가능: ~1,426개
- 총 확보 가능: ~12,496개
- 달성 가능 구간: **0→18강 + 18→20강 일부** (20강 기대 소모: 8,352개, 12,496개로 달성 가능)

**판정: 주의** — 22강은 현행 소모 곡선(64개/시도, 0.3% 성공률) 기준으로 45일 내 단독 달성이 불가능하다. 그러나 1차 검토 보고서 §5.4의 균형안(21강 성공률 2%, 22강 성공률 1%) 적용 시 22강 기대 소모가 약 22,152개로 감소하고 이 경우 자체 수급 11,070개 + 시장 매입 ~4,000개 수준으로 도달 가능성이 생긴다.

결론: A 방향 드랍 상향 + 1차 검토 §5.4 성공률 상향 동시 적용 시 선발대 상위 3~5명의 22강 달성이 현실적인 목표가 된다.

#### 6-D. 골드 과잉/부족 구간

| 구간 | 상태 | 내용 |
|---|---|---|
| 초반 1~10일 | 과잉 우려 없음 | 강화 골드 소모로 흡수 |
| 중반 11~30일 | 정상 | 강화 + 영지 작위 병행 소모 |
| 후반 31~45일 | 과잉 주의 | 하드/선발대 잉여 골드 증가. 강화석 시세 상승 시 자연 흡수. 미흡할 경우 추가 골드 싱크 필요 |
| 선발대 전체 | 잉여 대규모 | 강화석 시장 매입에 투입되지 않으면 골드 화폐 가치 하락 |

**판정: 주의** — 후반 골드 과잉 구간이 존재한다. 경매소와 영지 편의 해금이 흡수해야 한다.

#### 6-E. 강화석 공급 과잉/부족 구간

| 구간 | 상태 | 내용 |
|---|---|---|
| 초반 1~15일 | 부족 | 유저 전투력 미성장으로 파밍 효율 낮음 |
| 중반 16~30일 | 정상 | 강화 목표에 근접하여 수급과 소모가 균형 |
| 후반 31~45일 | 선발대 부족 / 라이트 잉여 | 라이트 유저 15강 주차 후 잉여 발생. 선발대 20강+ 도전으로 수요 폭증 |
| 서버 전체 | 부족 기조 | 선발대 22강 목표 대비 공급 절대량 부족 |

**판정: 정상** — 부족 기조는 의도된 설계다. 강화석이 시장 재화로 기능하고 경매소 시세가 유지되어야 한다. 단, 선발대가 강화석을 살 시장이 형성되어야 하므로 라이트 유저의 잉여 발생 구조 확보가 관건이다.
