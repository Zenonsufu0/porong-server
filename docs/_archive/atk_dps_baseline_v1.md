## 개요

이 문서는 세 가지 항목을 다룬다.

1. 강화 마일스톤 옵션 (5강/10강/15강/20강)
2. 각인별 DPS 기여율 추정 (6무기 × 공격형 각인)
3. 시즌보스별 입장 시점 실 ATK 및 단인 DPS 역산

전제:
- T1 무기 기본 ATK 80, 강화 공식 `총 ATK = 80 × (1 + 0.04 × 강화단계)`
- 피해 공식: `유효 방어율 = DEF / (DEF + 200)`, `기본 피해 = ATK × (1 - 유효 방어율)`
- 잠재능력은 이 문서에서 산정하지 않는다. 잠재능력 포함 시 여유 폭은 각 표의 빈칸으로 표시한다.

## 1. 강화 마일스톤 옵션

### 결정: 마일스톤 보너스 없음

강화 단계별 ATK 선형 증가로만 구성한다. 5강/10강/15강/20강 달성 시 별도 보너스 옵션을 부여하지 않는다.

### 결정 근거

**45일 시즌 서버 특성**

강화 보너스가 존재하면 5강/10강/15강 각 구간마다 "티어 점프"가 발생한다. 45일 시즌에서 유저는 강화 구간을 빠르게 통과하고, 마일스톤마다 수치가 급변하면 보스 난이도 조정 기준이 복잡해진다. 선형 증가는 DPS 예측이 용이하고 보스 HP 역산이 단순하다.

**수치 폭주 방지**

ATK% 마일스톤 보너스가 각인 수치와 합산될 경우, 20강 도달 유저의 유효 DPS가 0강 대비 2배 이상으로 벌어질 수 있다. 현재 각인 수치(최대 +60%)만으로도 강화 배율(0강 대비 20강 +80% ATK)과 복합 시 충분히 격차가 발생한다.

**성장 곡선 자연스러움**

강화 성공률 테이블(v1 확정)이 이미 높은 구간(15강 이상)에서 급격한 난이도 상승을 제공한다. 마일스톤 보너스를 추가하면 강화 목표가 "티어 달성"으로 좁아지고, 도달 전후의 체감 격차가 불합리하게 커진다. 선형 증가는 매 강화가 동등한 가치를 가지도록 유지한다.

**구현 복잡도**

마일스톤 보너스는 PoroRPG 아이템 인스턴스에 조건부 스탯 레이어를 추가해야 한다. 1차 시즌 구현 일정을 고려하면 불필요한 복잡도다.

### 강화 ATK 참조 표

| 강화 단계 | 총 ATK | 마일스톤 보너스 |
|---:|---:|---|
| 0강 | 80 | — |
| 5강 | 96 | 없음 |
| 10강 | 112 | 없음 |
| 15강 | 128 | 없음 |
| 18강 | 138 | — |
| 20강 | 144 | 없음 |

## 2. 각인별 DPS 기여율 추정

### 추정 방법

각 무기의 공격형 각인 1개를 선택해 기여율을 추정한다.

기여율 추정 공식:

```
각인 기여율 = 각인 수치 × 해당 스킬의 사이클 내 피해 비중 × 실전 발동률
```

피해 비중 가정 (스킬 4종 기준):
- 좌클릭(기본기): 사이클 내 피해의 약 45~55%
- 이동기: 사이클 내 피해의 약 5~10% (이동/회피 목적 위주)
- 특수기: 사이클 내 피해의 약 15~20%
- 핵심기: 사이클 내 피해의 약 25~35%

실전 발동률 가정:
- 조건부 각인(피격 후, 이동 후, 연속 명중)은 보스 이동 패턴을 고려해 0.65~0.80 적용
- 자원 만충 소모형 각인은 사이클당 1회 확정 발동으로 1.00 적용
- 누적 중첩형 각인은 평균 중첩 비율 적용 (최대 중첩의 약 70~80%)

### 공격형 각인 선택 기준

버프형/방어형 각인(성벽수호자, 기계궁수 DEF 파트)은 DPS 기여가 없거나 간접적이므로 제외하고, 각 무기의 공격 우선 각인을 선택한다.

| 무기 | 선택 각인 | 선택 이유 |
|---|---|---|
| 검 | 결투검 | 좌클릭 연속 명중 기반 피해 증가, 직접 DPS 기여 |
| 망치 | 파쇄자 | 충격 만충 소모 시 특수기 +30~40%, 직접 피해 증가 |
| 창 | 관통창 | 압박 5단계 핵심기 +35%, 최대 자원 소모형 |
| 석궁 | 추적자 | 이동 후 조준 ON 효과 80% 적용으로 이동 패턴에서도 화력 유지 |
| 낫 | 사신 | 흐름 만충 폭발 +50%, 최대 피해 집중형 |
| 스태프 | 화염술사 | 마력 3단계 이상 핵심기 +25%, 5단계 시 범위 확장 |

### 각인별 DPS 기여율 계산

**검 — 결투검**

- 사이클: 25초, 좌클릭 0.8초 기준 사이클 내 최대 약 31회 좌클릭
- 실전: 보스 이동/회피로 연속 명중 중단 빈도 고려, 평균 약 2.3중첩 유지
- 각인 수치: 1중첩 +20%, 2중첩 +40%, 3중첩 +60% → 평균 적용 수치 ≈ +46%
- 좌클릭 피해 비중: 50%
- 실전 발동률: 0.75 (이동 초기화 발생 고려)
- **기여율 = 0.46 × 0.50 × 0.75 ≈ 17%**
- 최대값 참고 (3중첩 완전 유지 시): 0.60 × 0.50 × 1.00 = 30%

**망치 — 파쇄자**

- 사이클: 28초, 충격 만충(좌클릭+특수기 조합) 후 소모 1회
- 각인 수치: 충격 만충 소모 시 특수기 +30%, 보스 추가 +10% → 합계 +40%
- 특수기 피해 비중: 18%
- 실전 발동률: 0.90 (자원 만충 후 소모형, 발동 조건이 자기 자원 기반으로 안정적)
- **기여율 = 0.40 × 0.18 × 0.90 ≈ 6%**
- 주의: 망치의 DPS 특성은 단발 피해보다 기절 제어로 파티 전체 딜타임 창출에 있음. 파티 DPS 기여 맥락에서는 간접 효과가 크다.

**창 — 관통창**

- 사이클: 28초, 압박 5단계 달성 후 핵심기 1회 (관통 판정 포함)
- 각인 수치: 핵심기 +35%
- 핵심기 피해 비중: 30%
- 실전 발동률: 0.95 (자원 5단계 만충은 사이클마다 안정적으로 달성)
- **기여율 = 0.35 × 0.30 × 0.95 ≈ 10%**
- 추가: 관통 판정(다음 대상까지 50% 피해)은 보스 단일 대상 전투에서 적용 빈도 낮음. 다중 보스 등장 패턴에서는 추가 DPS 발생.

**석궁 — 추적자**

- 사이클: 23초, 이동기(5초) 후 1초 내 발사로 조준 ON 80% 효과 적용
- 기존 조준 ON 효과: 피해 +25%, 치명타율 +15%
- 추적자 적용 시: 조준 ON 80% → 피해 +20%, 치명타율 +12%
- 좌클릭 사이클 내 비중: 50% (조준 해제 상태 발사 포함)
- 이동기 후 조준 ON 80% 구간 발동: 사이클당 약 3~4회 발사에 적용 (이동기 직후 집중)
- 해당 구간 피해 비중: 25%
- 실전 발동률: 0.70 (이동기 후 1초 창이 좁음)
- **기여율 = (0.20 × 0.25 × 0.70) + (이동 중 페널티 제거 효과) ≈ 3.5% + 2% ≈ 5.5% → 약 6%**
- 참고: 추적자의 핵심 가치는 이동 패턴 보스에서 화력 손실 최소화에 있음. 정적 보스 기준으로는 기계궁수(최대 +25%) 대비 수치가 낮다.

**낫 — 사신**

- 사이클: 25초, 흐름 3단계 충전 주기: 이동기(5초) × 3회 = 약 15초 충전
- 사이클당 만충 폭발 0.8회 (충전 15초 + 쿨타임 16초 오버랩 고려)
- 각인 수치: 만충 폭발 +50%
- 만충 폭발 피해 비중: 핵심기 기준 약 28% (낫은 핵심기가 자원 소모형 폭발)
- 실전 발동률: 0.80 (이동기 후 2초 내 명중 조건, W-2 조정 기준)
- **기여율 = 0.50 × 0.28 × 0.80 ≈ 11%**
- 폭발 후 이동 속도 +15% (2초): 다음 이동기 쿨타임과 연계 시 충전 속도 향상. 간접 DPS 기여 소폭 있음.

**스태프 — 화염술사**

- 사이클: 31초, 마력 5단계 충전(좌클릭 5회 × 0.9초 ≈ 4.5초) 후 핵심기
- 각인 수치: 마력 3단계 이상 핵심기 +25%, 5단계 시 범위 +2블럭
- 핵심기 피해 비중: 32% (스태프는 핵심기 비중이 높고 만충 효과 +45% 내장)
- 실전 발동률: 0.90 (마력 3단계 이상 도달은 매 사이클 달성 가능)
- **기여율 = 0.25 × 0.32 × 0.90 ≈ 7%**
- 5단계 범위 확장은 보스 단일 대상에서 DPS 수치 기여 없음. 다중 대상 패턴 발생 시 유효.

### 각인별 DPS 기여율 요약 표

| 무기 | 공격형 각인 | 각인 핵심 수치 | 적용 스킬 비중 | 실전 발동률 | DPS 기여율 추정 | 비고 |
|---|---|---|---:|---:|---:|---|

## 3. 시즌보스별 입장 시점 실 ATK 및 단인 DPS 역산

### 전제 조건

**유효 방어율 계산:**

```
보스1 DEF 150: 150 / (150 + 200) = 150 / 350 ≈ 0.4286 → 관통율 57.14%
보스2 DEF 200: 200 / (200 + 200) = 200 / 400 = 0.5000 → 관통율 50.00%
보스3 DEF 250: 250 / (250 + 200) = 250 / 450 ≈ 0.5556 → 관통율 44.44%
최종보스 DEF 280: 280 / (280 + 200) = 280 / 480 ≈ 0.5833 → 관통율 41.67%
```

**강화 단계별 현실 진행 기준 (combat_balance_v2.md 강화 성공률 테이블 기반):**
- 18강 도달 기대 시도 약 36회, 19~20강 추가 약 27회
- 일 2~3h 기준 18강 목표 30~35일차

**입장 시점별 강화 단계 추정:**

| 시점 | 일차 | 50%+ 유저 (중위권) | 상위 20% 유저 | 근거 |
|---|---:|---|---|---|
| 보스1 | ~10일 | 8강~12강 | 13강~15강 | 강화 초반 성공률 높음, 강화석 수급이 진입 병목 |
| 보스2 | ~20일 | 12강~15강 | 16강~18강 | 15강 이후 성공률 하락 시작, 활성 파밍 여부로 격차 |
| 보스3 | ~30일 | 15강~17강 | 18강~19강 | 18강이 현실적 장벽 구간, 상위권만 초과 |
| 최종보스 | ~38일 | 17강~18강 | 19강~20강 | 시즌 종반, 재화 충분 누적, 20강은 소수 |

**스킬 피해 배율 기준 (평균 사이클 가중치):**

```
좌클릭 배율 1.0 기준, 1사이클 평균 가중 배율:
- 좌클릭 비중 50%, 배율 1.0 → 기여 0.50
- 이동기 비중 8%,  배율 1.2 → 기여 0.096
- 특수기 비중 17%, 배율 1.5 → 기여 0.255
- 핵심기 비중 25%, 배율 2.0 → 기여 0.500
- 합계 = 1.351

사이클 평균 스킬 배율 ≈ 1.35
```

이는 평균 스킬 배율을 대표하는 보수적 추정값이다. 무기별 핵심기 배율은 확정 수치가 없으므로, 위 가중치로 통합하여 단인 DPS를 산출한다.

**단인 분당 히트 수 추정 (좌클릭 기준):**

```
1사이클 내 히트 구성 (검 기준 사이클 25초):
- 좌클릭: 25초 / 0.8초 = 약 31회 → 실전 약 20~22회 (이동/패턴 회피 반영)
- 이동기: 1회
- 특수기: 1회
- 핵심기: 1회
- 총 히트: 약 24~25회 / 25초 = 약 1회/초

분당 히트: 약 55~60회 (무기별 사이클 차이 고려, 평균 57회)
```

**DPS 계산 공식:**

```
기본 피해 (1회) = ATK × 관통율
스킬 가중 피해 (1회) = 기본 피해 × 사이클 평균 스킬 배율(1.35)
분당 DPS = 스킬 가중 피해 × 분당 히트 수(57) × 각인 기여율
```

## 4. 리스크 및 주의사항

### R-1. DEF 증가에 따른 단인 DPS 감소 추세가 과도할 수 있음

중위권 기준으로 보스1(5,762) → 최종보스(4,733)로 약 18% DPS 감소가 발생한다. 강화 진행(ATK 증가)이 DEF 증가를 충분히 상쇄하지 못하면 후반 보스일수록 체감 화력이 약해지는 문제가 생긴다.

조정 포인트:
- 보스3과 최종보스의 DEF를 5~10 낮추거나
- 취약 창 딜타임에서 DEF를 임시 감소시키는 설계로 보완

### R-2. 각인 기여율 무기 간 편차

결투검(+17%) vs 파쇄자(+6%), 추적자(+6%)의 격차가 크다. 수치상 결투검이 2.8배 높다. 그러나 망치의 파쇄자는 기절 제어로 딜타임을 창출하고, 추적자는 이동 강제 패턴에서 손실 방지가 핵심이므로 단순 수치 비교는 과도한 판단이다.

실서버 로그에서 무기별 평균 DPS를 관측한 후, 특정 무기가 다른 무기 대비 20% 이상 높으면 각인 수치 또는 스킬 배율 조정을 검토한다.

### R-3. 잠재능력 포함 시 수치 폭주 가능성

현재 잠재능력 수치는 미확정이다. 잠재능력이 ATK% 또는 스킬 피해%를 직접 올리면, 각인 기여율(최대 +17~30%)과 합산 시 단인 DPS가 각인 미포함 기준 대비 50% 이상 증가할 수 있다. 잠재능력 설계 완료 후 이 문서의 최종보스 DPS 수치를 재검증해야 한다.

구체적 위험:
- 잠재능력 ATK+20% 가정 시, 최종보스 상위권 단인 DPS: 4,940 × 1.20 = 5,928
- 결투검 최대 +30% 조합 시: 5,928 × 1.30 → 7,706/분까지 상승 가능
- 최종보스 HP 역산이 이 수치를 기준으로 맞춰지지 않으면 파티 3인 TTK가 의도보다 크게 짧아진다

### R-4. 사이클 평균 스킬 배율(1.35) 가정의 불확실성

핵심기/특수기의 실제 스킬 배율이 확정되지 않았다. 현재 1.35는 구조적 가중치 기반 추정이다. 실서버 구현 후 스킬 배율 확정 시 이 문서 전체를 재계산해야 한다.

확인 필요: 각 무기 4개 스킬의 배율 확정값 → `combat_balance_v2.md`에 추가 기록 권장.

### R-5. 강화 단계 입장 시점 추정의 불확실성

강화 단계는 일 2~3h 플레이와 강화석 수급 속도를 기준으로 추정했다. 강화석 드랍률(W-5 미결)이 확정되면 재추정이 필요하다.

보수적 시나리오: 강화석 수급이 계획보다 빡빡하면 보스1 입장 시 중위권이 8강 이하가 될 수 있다. 이 경우 보스1 DPS가 약 4,500/분 수준으로 내려가며, 현재 필드보스 HP 기준(90,000~110,000)과 비교 시 파티 3인 TTK가 15분을 초과할 수 있다.

### R-6. 분당 히트 수 가정(57회)

실전 보스 패턴 회피, 넉백, 이동 등으로 실제 히트 수는 변동이 크다. 보스 이동이 잦은 보스3/최종보스에서는 낫(이동기 후 2초 내 명중 조건)과 석궁(조준 ON 조건)의 실전 히트 비율이 추정보다 낮을 수 있다.
