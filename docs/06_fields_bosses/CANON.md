# 06. 필드 / 보스 / 드랍 — CANON

> **[STATUS: CANON]** — 필드 5개, 필드보스, 시즌보스, 드랍 구조, 기여도 보상의 공식 기준.

## 공식 기준

| 항목 | 기준 |
|---|---|
| 필드 수 | 5개 |
| 필드 구성 | 일반몹 2종 + 정예몹 1종 + 필드보스 1종 |
| 필드 재료 | `전장의 파편` 1종 통합 |
| 강화석 | DB 가상 재화로 처치 시 직접 적립. 강화석 파편 및 실물 아이템 드랍 없음 |
| 필드보스 | 5종, 30분 정시 스폰, 5분 미처치 소멸 |
| 시즌보스 | 시즌보스 3종 + 균열왕 |
| 시즌보스 입장 | 주간 입장 제한 없음 |
| 시즌보스 보상 리셋 | 매주 월요일 00:00. 주간 첫 클리어 풀 보상, 이후 재도전 감소 보상. 균열왕의 심장은 매 클리어 드랍 허용 (DL-037) |
| 균열왕 최소 입장 인원 | 제한 없음. 솔로/소수 입장 허용. 입장 시 클리어 가능성 경고 메시지만 출력 (DL-038) |
| 보상 기여도 | 총 피해량 3% 이상 |
| 원샷 방지 | 단일 타격 85% 클램프, 후속 공격 0 피해 |
| 외형 | 1차 시즌은 바닐라 강화형. ModelEngine/BetterModel은 2차 확장 |

## 충돌 처리

필드별 개별 재료, 강화석 파편, 시즌보스 주간 입장 제한, 고퀄 외부 모델 전제는 1차 시즌 공식 기준이 아니다.

시즌보스 보상 제한이 필요한 경우에도 "주간 입장 제한 없음" 기준과 분리해 별도 결정 로그로 확정해야 한다. 확정 전 주간 첫 클리어/반복 보상 구조는 구버전 설계로 본다.
