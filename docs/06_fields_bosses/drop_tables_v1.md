## 설계 전제 (확정 변경사항)

- **T1 메인 장비 수급:** 서버 최초 접속 시 5슬롯 일괄 지급. 이 장비만 강화 가능.
- **장비 완제품 드랍: 없음.** 전투 드롭으로 완성된 장비 획득 불가.
- **장비의 흔적 드랍: 신규 추가** — 정예몹 이상에서 랜덤 슬롯의 흔적(무기/투구/상의/하의/신발) 드롭. 등급·보조스텟이 내재됨. 장착 가능, 강화 불가, 전승 소스로 사용 가능. (상세: `item_grade_substat_v1.md`)
- **강화석 직접 드랍으로 전환 (파편 시스템 폐지):** 강화석 파편(stone_fragment) 시스템 완전 삭제. 강화석(mat_stone_enhance)을 몹 처치 시 직접 드랍. (2026-05-20 확정)
- **강화석 경제 의도:** 초반/약한 유저는 강화석 잉여·골드 부족 → 경매소 판매. 선발대는 골드 잉여·강화석 부족(17강+) → 경매소 구매. (`economy_numbers_v2.md §12` 참조)
- 드랍 확률 = 1회 처치당 해당 아이템 드랍 확률. 수량 = 드랍 시 지급 수량.
- **처치 속도 기준:** 일반몹 45마리/분, 정예몹 30마리/분 (별도 구역). 파밍 배분: 일반몹 50분·정예몹 40분·보스 10분 (2h 기준).

> **주의:**
> 이 문서의 골드 드랍 수치(필드1 일반몹 2~5G 등)는 economy_numbers_v2.md 확정값(필드1 4~8G)과 다름. economy_numbers_v2.md §0이 우선함.

| 조건 | 보상 |
|---|---|
| 기여도 미충족 | 보상 없음 |
| 기여도 충족 | 기본 보상 (확정) + 희귀 보상 (개인 독립 확률, 미지급 가능) |

- **기본 보상**: 기여도 충족 모든 유저에게 동일하게 확정 지급

- **기본 기여도 기준**: **총 피해량 3% 이상** (확정 2026-05-14)
- 상위 필드일수록 기본 보상 수량 증가 + 희귀 보상 풀 상향

## 4. 시즌보스 보상 테이블

### 인원별 드랍 배율 적용 규칙 (2026-05-16 확정)

3인 기준(1.00×)이 기본. 1인(0.55×) / 2인(0.78×) 입장 시 아래 규칙으로 배율 적용.

**방법 B — 소모성만 배율, 희소재 보호**

| 아이템 분류 | 1인 (0.55×) | 2인 (0.78×) | 비고 |
|---|---|---|---|
| 강화석 | 3인 수량 × 0.55 (올림) | 3인 수량 × 0.78 (올림) | 소모성 재료 |
| 강화석 완성품 | 3인 수량 × 0.55 (올림) | 3인 수량 × 0.78 (올림) | 소모성 재료 |
| 큐브 조각 | 3인 수량 × 0.55 (올림) | 3인 수량 × 0.78 (올림) | 소모성 재료 |
| 시즌보스 핵심 재료 | **3인과 동일** | **3인과 동일** | 희소재 보호 |
| 장비의 흔적 | **3인과 동일** | **3인과 동일** | 희소재 보호 |
| 치장 제작 파편 | **3인과 동일** | **3인과 동일** | 희소재 보호 |
| 시즌 칭호 재료 | **3인과 동일** | **3인과 동일** | 희소재 보호 |
