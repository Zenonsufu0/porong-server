## 설계 전제 (확정 변경사항)

- **T1 장비 수급:** 서버 최초 접속 시 5슬롯 일괄 지급. 전투 드랍으로 추가 수급 불가 (영지 등급작 전용)
- **T1 장비 파편 드랍: 전면 삭제** (아래 모든 테이블에서 T1 장비 파편 항목 제거됨)
- **큐브: 완성품 직드랍 구조** (조각 방식 삭제. 하급/중급/상급 큐브 완성품이 직접 드랍)
- **큐브 등급 확률:** growth_cube_grade_table.csv 참조 (하급: COMMON 75% / 중급: COMMON 35% RARE 45% / 상급: RARE 25% EPIC 50% UNIQUE 25%)
- **큐브 등업:** growth_cube_upgrade_table.csv 참조 (하급 5개 → 중급 60%, 중급 5개 → 상급 35%)
- 드랍 확률 = 1회 처치당 해당 아이템 드랍 확률. 수량 = 드랍 시 지급 수량
- **하루 큐브 수급 목표: 30~70개 (하급 20~50 + 중급 5~15 + 상급 0~5)**
