> `field_boss_stats_v1.md`(보스 방어 측: HP/DEF)의 **공격 측 대칭 문서**.

> 작성일: 2026-06-01

> 이전 상태: 정본 부재 — 몹 ATK는 MythicMobs 설정의 임의값(HP 670 시절 튜닝). DL-115에서 방어구 HP를 670→100으로 낮추면서 "몹 ATK 정본표 부재(바닐라 데미지)"가 명시적 부채로 남았다.

## 0. 왜 필요한가 (배경)

DL-115에서 0강 풀세트 HP를 **670 → 100**으로 스케일 다운했다. 그런데 몹/보스가 플레이어를 때리는 경로는 **바닐라 데미지**(`PlayerDefenseListener`가 DEF 경감만 얹음)라, 몹 공격력은 HP 670 시절 임의값 그대로였다.

그 결과 현행 설정에는 **내부 모순**이 존재한다:

- **보스 바닐라 평타(`Damage:` 속성)는 과도** — F5 보스 `Damage:120`은 18강 플레이어(HP 158)에게 약 83(52%), 사실상 즉사.
- **같은 보스의 스킬 패턴 데미지(`damage{a=8~22}`)는 과소** — F5 원형폭발 `a=22`는 약 16(10%)로 위협이 안 됨.

둘 다 HP 670 기준으로 따로 튜닝됐기 때문이다. 본 문서는 두 경로를 **추천 강화 풀세트 최대 HP의 비율(%) 기준**으로 일관되게 재도출한다.

## 1. 설계 전제

### 1.1 플레이어 생존 기준 (베이스 방어구만)

`combat_balance_v2.md §1` 선형 공식 기준. **인내 스탯·받피감 잠재·포션은 제외한 보수적 바닥값**이다(실제 유저는 이보다 더 잘 버틴다).

```
풀세트 최대 HP(n강)  = 100 + 3.2n          (기본 20 + 방어구 80×(1+0.04n))
풀세트 DEF(n강)      = 52 × (1 + 0.04n)
유효 방어율 mit(n)   = DEF / (DEF + 200)     (§2 피해 공식 공유 상수)
관통율(1−mit)        = 200 / (DEF + 200)
```

### 1.2 필드별 추천 강화 / 생존 파라미터

추천 강화 구간은 `field_boss_stats_v1.md §1.3` 기준.

| F1 수도 외곽 평원 | 5강 | 116 | 62.4 | 23.8% | 0.762 |

## 2. 티어별 피해 비율 룰 (핵심 정본)

몹/보스 공격을 5개 티어로 나누고, 각 티어의 **유효 피해 = 추천 강화 풀세트 최대 HP의 고정 비율**로 정의한다.

> **설계 철학 (2026-06-01 조정):** **필드 콘텐츠(몹+필드보스)는 가볍게, 시즌보스는 무겁게.** 필드는 오픈월드·상시 리젠·사냥의 연장이라 "죽을 걱정"이 곧 유저 부담이다. 무엇보다 **회복약은 공방 제한 제작 소모재**라, 필드에서 포션을 쓰게 만들면 정작 시즌보스에 쓸 포션이 마른다. 따라서 **필드 전체는 포션 없이도 버티는 가벼움**으로, **포션 관리·회피 실력은 시즌보스 전용 긴장**으로 분리한다.
>
> 같은 P-코드 패턴이라도 **필드보스와 시즌보스는 배율을 분리**한다(필드보스 강 24% ↔ 시즌보스 강 40%). 일반/정예몹과 필드보스는 한 묶음의 "필드 난이도", 시즌보스는 별도의 "도전 난이도".

| 티어 | 유효 피해(추천 HP 대비) | 명목 피격 횟수(HTK) | 설계 의도 |
|---|---:|---:|---|
| 일반몹 평타 | **2.5%** | ~40타 | 사냥 거의 무피해. 군집이어도 부담 미미 |
| 정예몹 평타 | **6%** | ~17타 | 일반몹보다 확실히 위협, 그러나 사냥 흐름 유지 |
| 필드보스 기본공격(P_00) | **8%** | ~12타 | 가벼운 칩 데미지 |
| 필드보스 일반 패턴(강타·돌진·휩쓸기·투사체) | **16%** | ~6타 | 맞아도 여유, 포션 거의 불필요 |
| 필드보스 강 패턴(원형폭발·낙하·집중탄) | **24%** | ~4타 | 가장 센 패턴도 4~5대 버팀. 사냥의 연장 |
| 시즌보스 기본공격 | **12%** | ~8타 | 상시 압박. 붙어 있으면 꾸준히 깎임 |
| 시즌보스 일반 패턴 | **24%** | ~4타 | 맞으면 아프다. 회피 가능 |
| 시즌보스 강 패턴 | **40%** | ~2.5타 | "반드시 피해야 하는" 텔레그래프. 3타 사망, 포션 1회로 겨우 상쇄 |

도출식:

```
몹 ATK(raw) = (티어 비율 × 추천 강화 풀세트 HP) / 관통율
            = (티어 비율 × HP) / (1 − mit)
```

`raw`는 **DEF 경감 적용 전** 값 = MythicMobs `Damage:` 속성 / 스킬 `damage{a}`에 그대로 넣는 값이다(`PlayerDefenseListener`가 받는 쪽에서 `×(1−mit)` 처리).

### 2.1 설계 정합성 검증

- **원샷 방지(CANON 85% 클램프):** 최고 티어인 시즌보스 강 패턴 40% ≪ 85%. 모든 티어가 클램프 한참 아래(저강화 유저가 상위 필드 난입 시에도 단일타 비치명).
- **포션 정합(`combat_balance_v2 §11`):** 하급 치유 포션 = 최대 HP 30% 회복 / 보스전 3회. **필드(몹+필드보스)는 포션 없이도 버티도록** 설계 — 회복약은 시즌보스용으로 아낀다. 시즌보스 강 패턴(40%)만 포션 1회(30%)로 겨우 상쇄하는 "줄타기"가 되고, 연달아 맞으면 포션으로도 못 따라가 회피가 핵심.
- **군집 정합:** 동적 필드 스폰이 일반몹 3~4마리를 붙여도 라운드당 7.5~10%(2.5%×3~4)로 부담이 거의 없다. 사냥 쾌적 의도대로. 정예몹(6%)이 섞여야 비로소 체감 위협.
- **보수성:** 인내 투자(0.4 DEF/pt)·받피감 잠재·풀강 포션을 더하면 실제 생존력은 표보다 높다. 본 표는 **하한**이다.

| 필드 | 일반몹 평타 | 정예몹 평타 |
|---|---:|---:|
| F1 | **4** | **9** |
| F2 | **5** | **11** |
| F3 | **5** | **12** |
| F4 | **5** | **13** |

> 필드별 일반몹 2종은 동일 ATK를 공유한다(타입 차이는 HP/이속/특수효과로 표현). 정예몹은 일반몹의 약 2.4배(2.5%↔6%) — 일반몹은 사실상 무피해, 정예가 유일한 사냥 위협 요소.

### 3.2 필드보스 (기본공격 / 일반 패턴 / 강 패턴)

필드보스 배율: 기본공격 8% / 일반 패턴 16% / 강 패턴 24% (필드 가볍게).

## 5. 시즌보스 ATK (잠정 — combat-balance 후속)

시즌보스→플레이어 ATK도 정본 부재. 동일 §2 룰을 적용하되, **추천 강화는 `season_boss_stats_v1.md` 기준**(인내·잠재 미반영 바닥값). 시즌보스는 일반/정예 티어 없이 보스 3티어만 사용한다.

| 보스 | 추천 강화 | 풀세트 HP | 관통율 | 기본공격 | 일반 패턴 | 강 패턴 |
|---|---:|---:|---:|---:|---:|---:|
| 시즌보스 1 | 10강 | 132 | 0.733 | **22** | **43** | **72** |
| 시즌보스 2 | 12강 | 138 | 0.722 | **23** | **46** | **77** |
| 시즌보스 3 | 14강 | 145 | 0.711 | **24** | **49** | **81** |
| 시즌보스 4 | 16강 | 151 | 0.701 | **26** | **52** | **86** |
| 시즌보스 5 | 18강 | 158 | 0.691 | **27** | **55** | **91** |
| 시즌보스 6 | 21강 | 167 | 0.676 | **30** | **59** | **99** |
| 시즌최종 3종 | 22강+ | 170 | 0.672 | **30** | **61** | **101** |

시즌보스 배율: 기본공격 12% / 일반 24% / **강 40%** — 필드보스(8/16/24%)보다 무거운 **도전 난이도**. 회피·포션 관리가 핵심.

> **잠정 사유:** 시즌보스 생존은 인내 투자·받피감 잠재·포션 의존도가 필드보다 훨씬 크다(선발대 22강 전제). 본 표는 베이스 방어구 바닥값이라 실제보다 보수적이다. 강 패턴(72~101)은 추천 강화 풀세트 HP의 40%로, 회피 필수 기믹. 텔레그래프가 특히 긴 최종보스 일부 패턴은 45% 이상도 검토 대상.
> **광폭화 보정:** 타락한 이중체 광폭화(ATK +100%) 등 메카닉 배율은 본 표 값에 곱연산으로 적용(`season_boss_patterns.md`).

## 6. 적용 매핑 (MythicMobs 구현 참조)

정본값을 실제 설정에 반영할 때의 매핑. **server/plugins/MythicMobs 적용은 별도 배포 단계(사용자 승인 필요).**

| 정본 티어 | MythicMobs 반영 위치 |
|---|---|
| 일반/정예 평타 | `mobs/field_*.yml` 각 몹 `Damage:` 속성 |
| 보스 기본공격 | 보스 `Damage:` 속성 + `P_00_BasicAttack`/`*_BasicStrike` 스킬 `damage{a}` |
| 보스 일반 패턴 | `*_Charge`/`*_Sweep`/`*_Projectile` 등 스킬 `damage{a}` (`skills/FieldBossSkills.yml`) |
| 보스 강 패턴 | `*_RadialBlast`/`P_13_DropImpact`/`*_ScatterShot` 집중탄 스킬 `damage{a}` |

> `FieldSpawnService`(동적 스폰)는 몹 ATK를 별도 스케일링하지 않으므로 MythicMobs 설정값이 곧 정본 적용 지점이다.
> `PlayerDefenseListener`가 받는 쪽에서 `DEF/(DEF+200)` 경감을 자동 적용하므로, **스킬/속성 값에는 raw(경감 전) 값을 그대로 넣는다.**
