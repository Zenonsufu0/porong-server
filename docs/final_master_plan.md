## 5. 전투 시스템

### 5.1 입력 방식

무기를 들고 있을 때만 스킬 입력 활성화. 무기 없을 시 기본 마인크래프트 조작.

| 입력 | 역할 |
|---|---|
| 좌클릭 (LC) | 기본기 / 주력기 (슬롯 1) |
| 우클릭 (RC) | 이동기 / 보조기 (슬롯 2) |
| Shift + 우클릭 (SRC) | 특수기 / 제어기 / 자기 강화 (슬롯 3) |
| F (오프핸드) | 핵심기 (슬롯 4) |

> **발동 조건 (구현 사양):**  
> - LC (근거리 — 검/도끼/창/낫): `PlayerAttackEntityEvent`만 처리. 허공 좌클릭(`LEFT_CLICK_AIR`) 차단.  
> - LC (원거리 — 석궁/스태프): `LEFT_CLICK_AIR` + `PlayerAttackEntityEvent` 모두 처리 (논타겟). LC 스킬 자체가 기본 공격 역할.  
> - RC / SRC: `RIGHT_CLICK_AIR` + `RIGHT_CLICK_BLOCK` 모두 처리 (논타겟).  
> - F: `PlayerSwapHandItemsEvent` — 항상 논타겟.

### 5.2 설계 금지 항목 (1차)

플레이어 지속 장판 / 적 표식 / 받는 피해 증가 / 방어 감소 / 파티 전체 피해 증가 / 복잡한 스택 폭발

### 5.3 각인 구조 (공통)

- **소모형 (A각인):** F 발동 시 자원 전체 소모. 만충 시 최대 스케일링.
- **유지형 (B각인):** 자원 최대 6스택 (무기 무관 공통). F 소모 없음. 스택당 선형 피해 증가. 6스택 달성 시 특수 효과 활성.

**공용각인:** 1차 시즌 제외.

### 5.4 무기 6종 및 스킬 상세

검 / 도끼 / 창 / 석궁 / 낫 / 스태프. 각 무기 스킬 4개 (LC/RC/SRC/F), A/B 각인 2종.

→ 스킬 상세 (계수·쿨타임·자원·구현 메모): `docs/04_combat_weapon_skills/weapon_skills_v1.md` 참조  
→ 무기별 베이스 아이템·시작 장비 정의: `docs/04_combat_weapon_skills/CANON.md` 참조

### 5.5 장비 아이템 포맷 & 잠금 규칙

이름·로어 템플릿, PDC 키, 강화 잠금 규칙, 바닐라 인챈트 정책.

→ 상세: `docs/01_plugin_architecture/implementation_reference.md` 참조

### 5.6 시작 지급 장비

0강 무기 (클래스 선택) + 0강 투구·흉갑·바지·신발 + 시작 골드 500G.

→ 장비 정의 상세: `docs/04_combat_weapon_skills/CANON.md` 참조

### 5.7 피해 계산 공식

```
최종 피해 = 기본 공격력 × 스킬 계수
           × (1 + 강화 보너스) × (1 + 각인 보너스)
           × 상태 배율 × 치명타 배율(조건부)
```

## 6. 튜토리얼 & 핫바

### 6.1 최초 접속 플로우

```
1. 첫 접속 감지 (tutorial_complete 플래그 없음)
   → 튜토리얼 맵 스폰 지점 강제 텔포
   → 이동·전투·인벤토리 잠금 (튜토리얼 종료 전까지)

2. 튜토리얼 안내 GUI (슬라이드형, 뒤로가기 없음)
   [1/5] 포로 서버 소개 — 45일 시즌 서버, 장비 성장·영지 운영
   [2/5] 전투 시스템 — 무기 6종, 스킬 4개, 강화·잠재능력
   [3/5] 영지 시스템 — 약초 재배지·광물 채굴기·공방 가공기, 작위 성장
   [4/5] 필드 & 보스 — 필드 파밍, 필드보스, 시즌보스, 균열왕
   [5/5] 준비 완료 — [이동하기] 버튼 클릭 시 다음 단계 진행

3. 무기 선택 GUI 오픈 (54칸, 뒤로가기 없음)
   → 무기 6종 아이콘 표시 (검사/궁수/법사/격투가/창기사/음유시인)
   → 클릭 → 재확인 → 클래스 확정

4. 기본 장비 지급
   → 0강 무기 (선택 클래스) → 핫바 슬롯 0 자동 배치
   → 0강 투구·흉갑·바지·신발 → 캐릭터 장비 슬롯 자동 장착
   → 핫바 슬롯 7: 나침반 아이콘 (탐험)
   → 핫바 슬롯 8: 메뉴 아이콘 (메뉴)
   → 시작 골드 500G 지급

5. 영지 자동 생성 → 영지 스폰 텔포
   → tutorial_complete = true 저장
   → 이동·전투·인벤토리 잠금 해제
```

## 7. 장비 성장

### 7.1 T1 단일 체계

1차 시즌은 T1 장비만 사용. 제외: T1.5, T2, 세트 장비, 세트 효과, 공용각인.

### 7.2 잠재능력 등급

**커먼 / 레어 / 에픽 / 유니크 / 레전더리 — 모두 1차 구현.**  
**장비 슬롯: 무기 / 투구 / 상의 / 하의 / 신발 (5종). 악세서리 없음 (1차 제외).**

**큐브 작동 방식 (확정 2026-05-21):** 큐브 1회 = 5,000G. 전 라인 재롤 + 등업 시도. 큐브 촉매 폐지.  
**메모리얼 시스템:** 큐브 결과를 현재 옵션 vs 새 옵션 중 선택 가능. 등업 발생 시 필수 선택 (재롤 불가).

등업 확률: 커먼→레어 30% / 레어→에픽 6% / 에픽→유니크 1.8% / 유니크→레전더리 0.3%

→ 라인별 등급 구조·이탈 확률·잠재 옵션 풀 전체: `docs/02_database_api_stats/potential_options_v1.md` 참조

### 7.3 강화 테이블 (T1, 파괴/강등 없음)

1강 2,000G → 점진 증가 → 10강 20,000G → 21~25강 27,000G 고정.  
방어구 강화석 = ceil(무기 강화석 ÷ 1.5).  
오픈 후 골드 부족 시 몹 드랍 상향 조정 (강화 수치 불변).

→ 1~25강 전체 비용표 (강화석 수량 포함): `docs/02_database_api_stats/economy_numbers_v2.md` 참조

## 8. 개인 영지

### 8.1 구조 원칙

```
IridiumSkyblock = 섬 생성/보호/방문 껍데기
EmpireRPG = 저장고/기계/가공/작위
```

유저에게 "스카이블럭" 표현 노출 금지. **개인 영지**로 표현.

### 8.2 영지 작위 (8단계)

개척지(시작) → 기사령 → 준남작령 → 남작령 → 자작령 → 백작령 → 후작령 → 공작령.  
작위별 구매제. 저장고 용량·시설 슬롯 수 단계적 확장.

→ 작위별 업그레이드 비용·재료·저장고 용량·시설 슬롯·XZ 한도: `docs/05_island_farm_system/CANON.md` 참조

### 8.3 영지 시설 (3종)

영지 설정 GUI 슬롯 배정 방식 (물리 블럭 설치 없음). 작위별 슬롯 수 내 자유 배정.

| 시설 | 역할 | 생산 주기 | 최대 레벨 |
|---|---|---|---|
| 약초 재배지 | 제국 약초 생산 | 20분 | 3 |
| 광물 채굴기 | 마도철 원석 생산 | 20분 | 3 |
| 공방 가공기 | 레시피 대기열 제작 | 즉시(대기열) | 1 |

공방 가공기 1대 = 대기 슬롯 3개 (동일 레시피 최대 8세트).

→ 레벨별 산출량·레어 보너스: `docs/05_island_farm_system/CANON.md` 참조

### 8.4 영지 저장고

DB 기반 가상 저장고 (물리 상자 없음).

| 조작 | 동작 |
|---|---|
| 좌클릭 | 64개 출금 |
| 우클릭 | 1개 출금 |
| 전체 입금 버튼 | 인벤토리 전체 입금 |
| 자동 입금 | 편의 해금 후 자동 저장 |

### 8.5 공방 가공기 탭 (확정)

| 탭 | 주요 결과물 |
|---|---|
| 영지 제작 | 농부의 정수 / 광부의 정수 / 자연의 정수 / 원시 정수 / 미감정 흔적 / 고대흔적 |
| 제련 | 마도합금 |
| 정제 | 정제 약초 |
| 연금술 (치료) | 치료 포션 소/중/대 |
| 연금술 (부스트) | 골드·강화·경험치 부스트 포션 |
| 요리 | 전사/학살자/암살자/사냥꾼의 만찬 |

**삭제된 탭:** 연료, 큐브 촉매, 치장/영지, 보스 보상권

→ 제작 체인 레시피 전체 (CSV 확정): `docs/05_island_farm_system/workshop_crafting_spec.md` 참조

### 8.6 바닐라 농사/채광

기본: 인벤토리 획득. 편의 해금 후: 영지 저장고 자동 입금. 자동 심기: 작위 취득 후 골드/재료로 권한 구매.

### 8.7 기본 광물 생성기

울타리 + 빈칸 + 물 구조 (플레이어 직접 설치). 물이 울타리에 닿을 때마다 작위별 확률로 광물 1개 생성.  
자작령 이상: 다이아몬드·에메랄드 생성 시 보너스 드랍 (행운 레벨 비례).

## 9. 필드 & 드랍

### 9.1 필드 5개

| 필드 | 역할 |
|---|---|
| 수도 외곽 평원 | 초반 적응 |
| 폐광 지대 | 강화/광물 |
| 오염된 수로 | 포션/저항 |
| 무너진 초소 | 장비 숙련 |
| 고대 성벽 잔해 | 고난도/균열 |

구성: 일반몹 2종 + 정예몹 1종 + 필드보스 1종.

### 9.2 드랍 구조 (확정 2026-05-19)

**공통 재료: `전장의 파편` (`mat_field_shard`) 1종으로 통합** (필드별 개별 재료 없음)

| 몹 종류 | 드랍 |
|---|---|
| 일반몹 | 골드 + 전장의 파편 + 강화석 (낮은 확률) |
| 정예몹 | 골드 + 전장의 파편 + 강화석 + 큐브 조각 |
| 필드보스 | 전장의 파편 + 강화석 + 고대흔적(`ancient_trace_*`) + 장비의 흔적(`equip_trace_*`) |

> **강화석 직접 드랍 확정 (M-5):** 강화석 파편(`mat_stone_fragment`) 시스템 폐지. 강화석(`mat_stone_enhance`) 직접 드랍.

### 9.3 아이템 레벨(IL) 시스템 (확정 2026-05-14)

**IL 공식:** 강화 단계 1당 IL +5 / 평균 IL = 5슬롯 평균 강화 단계 × 5

| 필드 | 적정 평균 IL | 경고 임계 |
|---|---|---|
| 수도 외곽 평원 | IL 0~35 | IL 65 이상 (평균 13강+) |
| 폐광 지대 | IL 30~55 | IL 75 이상 (평균 15강+) |
| 오염된 수로 | IL 50~75 | IL 85 이상 (평균 17강+) |
| 무너진 초소 | IL 65~85 | IL 90 이상 (평균 18강+) |
| 고대 성벽 잔해 | IL 80~100 | 없음 |

경고 3회 누적 → 임시 화이트리스트 제외 (운영자 수동 해제). 일반몹/정예몹 30마리마다 경고 1회.

### 9.4 원샷 방지 (확정 2026-05-14)

```
단일 타격 피해 ≥ 필드보스 최대 HP × 85%
  → 해당 타격을 85%로 클램프
  → 트리거한 플레이어의 후속 공격: 0 피해 (해당 전투 종료까지)
  → 클램프된 85%는 기여도 산정에 정상 반영
```

### 9.5 기여도 보상 (확정 2026-05-14)

기준: **총 피해량 3% 이상** = 기여도 충족.
- 충족: 기본 보상 (확정, 동일 지급) + 희귀 보상 (개인 독립 확률)
- 미충족: 보상 없음

### 9.6 잠수 방지

분당 10마리 미만 처치 시 해당 유저를 스폰 계산에서 제외.  
필드 5분 이상 위치 변화 없으면 메인 스폰으로 강제 텔포.

## 14. GUI & 리소스팩

### 14.0 공통 재화 소모 규칙 (전역 정책)

모든 시스템(강화·큐브·공방·상점 등)에서 재료 소모 시 아래 순서를 따른다.

1. **영지 창고** 잔량 먼저 차감
2. 창고 잔량이 부족하면 **플레이어 인벤토리**에서 추가 차감
3. 합산 후에도 부족하면 소모 거부 → `§c재료가 부족합니다. (창고 + 인벤토리 합산)` 출력, 동작 중단

> 골드 소모는 플레이어 골드 잔액에서만 차감 (창고 개념 없음).

### 14.1 GUI 계층 및 플로우

**진입:** 나침반 우클릭 (`PlayerInteractEvent`) 또는 `/메뉴` 커맨드 → 메인 GUI 오픈

```
메인 GUI (54슬롯) ← menu_main.png
  ├── 장비 GUI (54슬롯) ← menu_equipment.png
  │       강화 / 잠재 / 각인 / 캐릭터 / 전승
  │       → 각 하위 GUI (바닐라 아이콘)
  ├── 영지 GUI (54슬롯) ← menu_territory.png
  │       영지이동 / 영지상태 / 창고 / 공방 / 작물관리 / 상점 / 경매장 / 영지설정
  │       → 각 하위 GUI (바닐라 아이콘)
  ├── 보스 GUI (54슬롯) ← menu_boss.png
  │       파티생성 / 파티목록 / 보스정보 / 클리어기록
  │       → 각 하위 GUI (바닐라 아이콘)
  └── 필드 GUI (바닐라 아이콘)
          필드 6개 + 훈련장 아이콘 나열 → 클릭 시 해당 필드 이동
```

공통 규칙: 빈 슬롯 = 검은 유리판. ESC 또는 뒤로가기 슬롯 → 상위 GUI 복귀.

→ 각 GUI 슬롯 매핑 상세 (54슬롯 레이아웃 전체): `docs/08_resourcepack_pipeline/` 참조

### 14.5 리소스팩 제작 우선순위
