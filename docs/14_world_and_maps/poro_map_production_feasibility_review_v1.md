## 개요

포로 서버 시즌1은 **수도 황성 + 1막 4테마(동/서/남/북) + 3막 사하르 상인연합 + 3막 아르카논 마법공학 도시** 7개 지역 + **체험 월드 B안** + **영지 월드(보이드+스키매틱 7×7 슬롯)** 으로 구성된다. 본 문서는 "각 지역의 세계관 톤·핵심 랜드마크를 실제 Minecraft 월드로 빌드할 때 필요한 규모·도구·인력·기간이 현실적인지" 판정하고, 시즌1 오픈 일정과 정합하는 옵션을 제시한다.

영지 월드는 이미 `poro_three_world_technical_breakdown_draft.md`에서 **FAWE + 스키매틱 스탬핑** 방식이 확정되어 본 검토의 대상이 아니다. 본 문서는 **수도 + 1막 4테마 + 3막 2개 지역 + 체험 월드** 6~7개 커스텀 맵 구역의 제작 가능성만 다룬다.

## 목표

1. 각 지역의 **필요 규모**(청크 단위)와 **핵심 랜드마크**를 정리해 빌드 작업량을 가시화한다.
2. 제작 **도구·인력·기간 옵션 4종**(A/B/C/D)을 비교해 시즌1 오픈 일정과 맞는 선택지를 제시한다.
3. **1차 공개 최소 맵 범위**와 **2차·3차 확장 로드맵**을 단계화한다.
4. **세계관 톤**(컬러 팔레트·블록 팔레트·조명·파티클)을 맵 빌드에 어떻게 투영할지 가이드한다.
5. **리소스팩 의존·Citizens·FAWE와의 상호 의존**을 리스크 관점에서 정리한다.

## 핵심 규칙

- 영지 월드 스키매틱(Lv1~5 각 1장)은 본 문서 범위 밖. 별도 에픽.
- 3막 개방 지역(사하르/아르카논)은 **1막/2막 시점에는 원거리 실루엣/소문 수준만 노출**, 실제 빌드 공개는 3막 업데이트 시점으로 분리 가능.
- 체험 월드는 B안(수도 인트로 축소판)으로 확정되어 있으며, 본 문서는 **체험 월드 = 수도 일부 자산 재활용 가능**이라는 구조적 이점을 반영한다.
- **리소스팩 미적용 유저는 체험 월드 자동 강등**이 이미 확정되어 있으므로, 메인·영지 월드의 **핵심 랜드마크는 리소스팩 전제 연출을 상당히 사용해도 된다**. 단 체험 월드 자체는 바닐라 블록만으로 실루엣 식별이 가능해야 한다(기존 체험 월드 설계 원칙 유지).
- 수도 외곽 필드보스 5종 전용 공간은 수도 반경 제한(`poro_master_planning.md`) 안에서 해결한다.
- 본 문서에서 인력/기간 수치는 공개 Minecraft 빌드 커뮤니티의 **일반 벤치마크**(유료 에셋 마켓플레이스 평균, 공개 빌드팀 공개 일정)에 기반한 **추정치**다. 실제 포로 서버 건축 인력 규모는 **사용자 확인 필요**.

## 구현 포인트

### 1. 지역별 요구 맵 규모·요건 정리

각 지역의 **플레이 영역**(청크 단위)·**핵심 랜드마크**·**NPC 배치 요구 수**·**필드보스 전용 공간**을 정리한다. 규모는 마스터 플래닝의 "테마 공통 구조(필드 5, 필드보스 5, 최상위 보스 2, 심층 필드 2, 심층보스 2, 결전보스 1, 극상위 보스 1)"를 기준으로 역산한다.

#### 1.1 수도 황성 (허브 + 1막 히든 메인)

- **플레이 영역**: 성벽 내부 약 24×24 청크(384×384 블록) + 외곽 필드보스 링 반경 40청크.
- **핵심 랜드마크**: 황성 본궁, 빈민가, 외곽 성문 4개(동/서/남/북 파견 열차 플랫폼), 기사단 분소, 길드 본부 대기실, 관측소, 광장.
- **NPC 배치 요구 수**: 대략 40~60명 (길드 NPC, 기사단 NPC, 상점·생활, 히든 메인 관찰자, 서브 퀘스트 NPC 등).
- **필드보스 전용 공간**: 수도 외곽에 5개 스폰 포인트 (반경 분산, 겹침 방지).
- **지형 생성 방식 권장**: **플랫 월드 베이스 + 수동 빌드 + 스키매틱**. 상징성·품질이 가장 중요한 지역이므로 자동 생성에 의존하지 않는다.
- **빌드 품질 민감도**: 매우 높음. 유저가 가장 오래 체류하는 허브.

#### 1.2 동부 침식 숲 (물기·식물 잠식)

- **플레이 영역**: 약 32×32 청크 (필드 5 + 심층 필드 2 포함).
- **핵심 랜드마크**: 침식 숲 입구 관문, 분신 판별 사당, 젖어가는 사당, 침식된 마을 폐허, 심층 뿌리굴.
- **NPC 배치 요구 수**: 약 20~30명.
- **필드보스 전용 공간**: 필드보스 5 + 최상위 2 + 심층 2 + 결전 1 + 극상위 1 = **11개 보스 스폰 영역**.
- **지형 생성 방식 권장**: **WorldPainter 또는 Arceon으로 지형 베이스 생성 + 핵심 랜드마크 수동 빌드 + 스키매틱 배치**. 자연 지형 비중이 크므로 도구 활용 효율이 높다.
- **톤 키워드**: 짙은 녹청·습기·넝쿨·고인 물 반사·안개.

#### 1.3 서부 황야 (건조·바람·폐허)

- **플레이 영역**: 약 32×32 청크.
- **핵심 랜드마크**: 가짜 쉼터, 풍향계 폐허, 사하르 상단 전초 캠프(사하르 본부는 3막 개방 전까지 실루엣만), 황야 모래폭풍 존.
- **NPC 배치 요구 수**: 약 20~25명.
- **필드보스 전용 공간**: 11개 (동부와 동일 구조).
- **지형 생성 방식 권장**: **WorldPainter 지형 베이스 + 폐허 스키매틱 세트 + 풍향계 수동 빌드**. 황야는 반복 지형 비중이 크고 폐허 패턴 재사용 가능.
- **톤 키워드**: 연황·모래·바람 파티클·뜨거운 햇빛·낡은 석재.

#### 1.4 남부 화맥 (화염·내열)

- **플레이 영역**: 약 32×32 청크.
- **핵심 랜드마크**: 용암 균열 대협곡, 과열 광산 입구, 태움 제단, 식지 않는 재 평원, 심층 화맥 용광로.
- **NPC 배치 요구 수**: 약 15~20명 (험지라서 상주 NPC 적음).
- **필드보스 전용 공간**: 11개.
- **지형 생성 방식 권장**: **WorldPainter 지형 + 용암 흐름 수동 교정 + 핵심 랜드마크 수동**. 용암/네더라이트 조합 연출은 바닐라 자동 생성 한계가 있어 수동 교정이 필요하다.
- **톤 키워드**: 주황·적흑·용암 발광·재 파티클·뜨거운 공기 왜곡.

#### 1.5 북부 설혼 (냉기·언데드)

- **플레이 영역**: 약 32×32 청크.
- **핵심 랜드마크**: 부상병 야영지, 얼어붙은 한 숨 호수, 설원 폐허, 한파의 성벽, 심층 냉기 심연.
- **NPC 배치 요구 수**: 약 15~20명.
- **필드보스 전용 공간**: 11개.
- **지형 생성 방식 권장**: **WorldPainter 지형(설원 바이옴 커스텀) + 얼음 구조물 스키매틱**.
- **톤 키워드**: 창백한 청백·성에 파티클·푸른 발광·무채색.

#### 1.6 사하르 상인연합 (3막 개방)

- **플레이 영역**: 약 20×20 청크 (교역 도시 중심).
- **핵심 랜드마크**: 사하르 본부 캐러밴 광장, 교역로 관문, 환영의 시장, 지하 금고.
- **NPC 배치 요구 수**: 약 30~40명 (상인 밀집).
- **필드보스 전용 공간**: 3막 기준 11개 스폰 영역.
- **지형 생성 방식 권장**: **수동 빌드 중심**(상업 도시 밀도가 높아 자동 생성 부적합) + 사막 지형 베이스는 서부와 공유 가능.
- **톤 키워드**: 금·청록·천막·향신료 연기.

#### 1.7 아르카논 마법공학 도시 (3막 개방)

- **플레이 영역**: 약 20×20 청크 (첨탑 봉인층은 수직 확장).
- **핵심 랜드마크**: 중앙 첨탑(Y 축 220까지 수직 활용), 마법공학 광장, 봉인층 던전, 부양 플랫폼.
- **NPC 배치 요구 수**: 약 25~35명.
- **필드보스 전용 공간**: 11개 (수직 배치 허용).
- **지형 생성 방식 권장**: **수동 빌드 + 부양 플랫폼 스키매틱**. 수직 구조와 커스텀 블록 밀도가 가장 높아 자동 생성 불가.
- **톤 키워드**: 남보라·마법 발광·부양 파티클·크리스탈·기계적 선형 구조.

#### 1.8 요구 규모 요약

총합 기준으로 보면 **수도 1 + 1막 4테마 + 3막 2지역 = 7개 대규모 맵 구역 + 체험 월드 1**이며, 합산 플레이 영역은 대략 **170~200청크² 상당의 빌드 면적**에 이른다. 순수 바닐라 수동 빌드만으로는 1인 건축가가 수년 단위, 4~5인 건축팀이어도 1년 이상 소요되는 규모다. 따라서 **도구 병용이 사실상 필수**다.

### 2. 제작 인력·도구 실측 판정

#### 2.1 현재 리소스 상황 (확인 필요 항목 포함)

- **건축가 수**: **사용자 확인 필요**. 본 문서는 "0~1인 + Claude Code 에이전트 보조" 시나리오부터 "3~5인 빌드팀" 시나리오까지 분기로 다룬다.
- **사용 가능 도구**: Minecraft 바닐라 + WorldEdit/FAWE(3월드 에픽에 이미 포함, 도입 확정) + 리소스팩(진행 중) + Citizens NPC 배치. WorldPainter·Arceon 같은 외부 맵 툴은 **도입 여부 사용자 확인 필요**.
- **참고 월드 활용 여부**: 공개 Minecraft 빌드 마켓플레이스(예: Planet Minecraft, BuildersRefuge 스키매틱 공유, 유료 마켓) 활용 여부 **사용자 확인 필요**. 유료 에셋 구매 예산 유무도 확인 필요.
- **제작 기간 벤치마크 (공개 자료 기반 추정)**:
  - 1인 건축가 기준 "수도 황성 + 빈민가 + 외곽" 상징적 허브 완성 = **약 3~6개월**.
  - 1인 건축가 기준 자연 테마 1개(32×32 청크) 완성 = **약 2~4개월**(지형 툴 활용 시 1.5~2.5개월).
  - 3~5인 빌드팀 기준 허브 + 4테마 완성 = **약 4~6개월**.
  - 순수 수동 빌드 + 커스텀 블록 밀도 높은 도시(아르카논·사하르) = 1인 기준 **3~5개월 추가**.

#### 2.2 지형 생성 방식 3종 비교

- **바닐라 biome + 커스텀 structure**: 넓은 영역 커버 용이, 그러나 포로 서버 고유 톤 반영 어려움. 필드보스 존·랜드마크가 자동 생성 위에 충돌.
- **WorldPainter / Arceon 등 맵 툴**: 자연 테마(동/서/남/북)에 최적. 지형 베이스 생성 + 수동 교정 하이브리드가 현실적.
- **플랫 월드 + 스키매틱**: 수도·사하르·아르카논처럼 도시·구조물 밀도가 높은 지역에 최적. 체험 월드가 이미 이 방식.

**권장**: 지역별 혼용. 수도/사하르/아르카논은 플랫+스키매틱, 1막 4테마는 WorldPainter 기반 하이브리드.

### 3. 옵션 A/B/C/D 비교

#### 옵션 A — 전부 커스텀 빌드 (순수 수동 + FAWE만 보조)

- **장점**: 최대 품질·세계관 톤 일관성·유저 몰입감 최상.
- **단점**: 인력·기간 부담 최대. 소수 인력 상황에서는 시즌1 오픈 자체가 지연.
- **예상 기간**: 3~5인 빌드팀 기준 **6개월~1년**, 1인 기준 **1.5~2년 이상**.
- **필요 인력**: 숙련 건축가 **최소 3인 이상** 권장.
- **시즌1 오픈 정합성**: 현실적으로 **부정합**(일정 초과 위험 큼).
- **리스크**: 품질 기대치는 최상이지만 기간 리스크가 크고 중간 마일스톤에서 피드백 루프가 길다.

#### 옵션 B — 커스텀 빌드 + 공개/유료 에셋 혼합

- **장점**: 품질 유지하면서 기간 단축. 반복 지형(폐허·마을·자연 지형)은 공개 에셋으로 커버, 상징적 랜드마크만 수동 빌드.
- **단점**: 에셋 톤 통일 작업 필요(리페인트·블록 교체). 라이선스 검토 필요.
- **예상 기간**: 2~3인 빌드팀 기준 **3~6개월**.
- **필요 인력**: 숙련 건축가 **2~3인** + 에셋 큐레이터 1인.
- **시즌1 오픈 정합성**: 중간 규모 팀이면 **정합 가능**.
- **리스크**: 에셋 라이선스·상업 이용 조건 확인 누락, 톤 불일치.

#### 옵션 C — 프로시저럴 생성 + 스키매틱 중점

- **장점**: 최단 기간. FAWE·커스텀 biome·WorldPainter로 지형 대량 생성, 핵심 랜드마크만 수동.
- **단점**: 허브(수도)·도시(사하르/아르카논)는 프로시저럴이 약함. 품질 편차 큼.
- **예상 기간**: 1~2인 체제로도 **2~3개월** 가능 (단, 수도/사하르/아르카논 수동 부분은 별도 산정).
- **필요 인력**: 지형 툴 숙련자 1인 + 수동 빌드 1~2인.
- **시즌1 오픈 정합성**: 소수 인력 상황에서도 **정합 가능**.
- **리스크**: 유저 체감 품질 리스크. 1막 4테마가 "생성된 필드" 느낌으로 보이면 몰입 저하.

#### 옵션 D — 체험 월드 + 수도만 1차 공개, 나머지 점진 확장

- **장점**: 가장 안전하고 현실적. 1차 공개 범위가 명확하고 유저 피드백을 맵 확장에 반영 가능.
- **단점**: 시즌1 콘텐츠 전체(1막 4테마)가 오픈 시점에 없음. 오픈 직후 플레이 타임 짧음.
- **예상 기간**: 수도 + 체험 월드만 **1~2개월**, 이후 1막 4테마 **각 1~2개월 단위로 순차 추가**.
- **필요 인력**: 1~2인으로도 1차 공개 가능.
- **시즌1 오픈 정합성**: **최고로 정합**. 단 오픈 스코프 축소 합의 필요.
- **리스크**: 오픈 당시 "콘텐츠 부족" 지적 가능. 대응책으로 **1막 메인 1~5는 수도 내부에서 완결**되도록 설계(마스터 플래닝의 수도 히든 메인 라인과 일치).

### 4. 권장안

**권장 옵션: D (점진 확장) + C 혼합**. 즉 **1차 공개는 수도 + 체험 월드로 최소 스코프 확보**, **2차 확장에서 1막 4테마를 C 방식(프로시저럴 + 핵심 랜드마크 수동) 으로 순차 오픈**, **3차 확장에서 3막 사하르/아르카논을 B 방식(수동 + 공개 에셋 혼합)** 으로 마감한다.

근거:
- 포로 서버 현재 인력 상황이 "사용자 본인 + Claude Code 에이전트"로 추정되며(**사용자 확인 필요**), 순수 옵션 A는 시즌1 오픈 일정과 정합하지 않을 가능성이 크다.
- 마스터 플래닝에 따르면 **3막에 사하르/아르카논이 개방**되므로, 본 지역 2개는 1차 오픈 시점에 불필요하다. 이 구조적 여유가 옵션 D를 자연스럽게 만든다.
- 1막 메인 1~5가 수도 내부에서 진행될 수 있으므로(수도 히든 메인 라인 존재), 수도 + 체험 월드만 완성되면 최소 오픈 가능한 플레이 루프가 성립한다.
- FAWE·리소스팩·Citizens NPC는 이미 도입 중이며, 옵션 C의 프로시저럴 지형 + 핵심 랜드마크 수동 방식과 자연스럽게 결합된다.

### 5. 단계별 로드맵

#### 5.1 1차 공개 (시즌1 오픈 시점) — 최소 범위

- **범위**: **체험 월드 + 수도 황성 + 수도 외곽 필드보스 5종 존**.
- **목표 기간**: 1~2개월 (인력 1~2인 기준).
- **제작 방식**: 플랫 월드 + 수동 빌드 + 스키매틱.
- **오픈 시 커버 가능한 콘텐츠**: 체험 월드 온보딩 → 수도 스폰 → 1막 메인 1~5(수도 내부 완결) → 수도 히든 메인 라인 초입 → 수도 외곽 필드보스 5종.
- **리소스팩 의존**: 수도 커스텀 UI·랜드마크 CustomModelData 필수. 리소스팩 미적용자는 체험 월드 강등(기존 확정).

#### 5.2 2차 확장 (시즌1 중반, 오픈 후 1~3개월)

- **범위**: 1막 4테마 중 **우선 1~2개 테마 선행 오픈**. 권장 순서: **동부 → 서부 → 남부 → 북부** (톤 대비·난이도 곡선 기준, 사용자 재검토 가능).
- **목표 기간**: 테마당 1~2개월.
- **제작 방식**: WorldPainter 기반 지형 + 핵심 랜드마크 수동 빌드 + FAWE 스키매틱 배치.
- **오픈 시 커버 가능한 콘텐츠**: 해당 테마의 필드 5 + 필드보스 5 + 최상위 보스 2.

#### 5.3 3차 확장 (시즌1 후반 / 3막 진입)

- **범위**: 1막 나머지 테마 완결 + **사하르 상인연합 + 아르카논 마법공학 도시**.
- **목표 기간**: 사하르/아르카논 각 2~3개월, 잔여 1막 테마 1~2개월.
- **제작 방식**: 사하르/아르카논은 수동 빌드 + 공개/유료 에셋 혼합(옵션 B).
- **리소스팩 업데이트 필수**: 사하르/아르카논 고유 커스텀 블록 추가.

### 6. 세계관 톤 유지 가이드

지역별 **컬러 팔레트·블록 팔레트·조명·BGM·파티클 연출 표준**을 정의한다. 월드 앰비언트 BGM은 기존 확정(월드 사운드 제거 a안)에 따라 **재생하지 않으므로**, 톤 전달은 **시각(블록·조명·파티클)** 중심이다.

#### 6.1 수도 황성

- **컬러 팔레트**: 금·상아·군청.
- **바닐라 블록 추천**: 광택 안산암, 칼사이트, 석영, 네더 금, 청금석 블록(포인트), 군청색 양털(깃발).
- **조명**: 랜턴 + 빛 블록(light block) 조합, `doDaylightCycle=true` 유지로 낮/밤 연출.
- **파티클**: 광장 파문 링(이펙트로 BGM 대체 용도, 기존 확정 연동).
- **리소스팩 의존**: 커스텀 UI, 상징 깃발 텍스처.

#### 6.2 동부 침식 숲

- **컬러 팔레트**: 짙은 녹청·회청·흑녹.
- **바닐라 블록 추천**: 이끼 블록, 진달래 이파리, 머드, 점액 블록(고인 물 질감), 습지 나무, 어두운 떡갈나무.
- **조명**: 광원 최소화·인광 버섯(glow berries)·횃불 지양.
- **파티클**: 스포어(mangrove propagule 파티클), 물방울, 옅은 안개.
- **리소스팩 의존**: 사당 외형 커스텀 모델.

#### 6.3 서부 황야

- **컬러 팔레트**: 연황·적갈·흐린 청동.
- **바닐라 블록 추천**: 사암, 적사암, 매끄러운 사암, 테라코타(연황·오렌지), 잘린 구리, 떨어지는 모래.
- **조명**: 강한 햇빛 유지, 그림자 강조.
- **파티클**: 모래 입자(falling sand), 바람 효과(스킨·깃발 흔들림).
- **리소스팩 의존**: 풍향계 커스텀 모델, 사하르 상단 깃발 텍스처.

#### 6.4 남부 화맥

- **컬러 팔레트**: 주황·적흑·탄소.
- **바닐라 블록 추천**: 마그마 블록, 현무암, 검게 그을린 나무, 네더랙, 블랙스톤, 안산암(식은 재).
- **조명**: 용암 발광 + 네더 포탈 글로우 활용.
- **파티클**: 재(ash) 파티클(소울 샌드 밸리 재활용), 불꽃, 열 왜곡(텍스처 시프트).
- **리소스팩 의존**: 태움 제단 커스텀 모델.

#### 6.5 북부 설혼

- **컬러 팔레트**: 창백한 청백·무채색 회색.
- **바닐라 블록 추천**: 얼음·파란 얼음·눈 블록·가문비나무·포웨더러드 블록·다이아몬드(포인트 광원).
- **조명**: 청색 빛 블록 + 분수(light-level 11).
- **파티클**: 성에, 차가운 호흡 입김(snowflake 파티클).
- **리소스팩 의존**: 언데드 몹 외형, 얼어붙은 성벽 텍스처.

#### 6.6 사하르 상인연합

- **컬러 팔레트**: 금·청록·진홍.
- **바닐라 블록 추천**: 사암·구리·청록 양털·빨간 양털·잿빛 벽돌·등불.
- **조명**: 등불 밀집, 천막 아래 그림자.
- **파티클**: 향신료 연기(캠프파이어 smoke), 동전 반짝임(electric 파티클 연출).
- **리소스팩 의존**: 캐러밴 수레 커스텀 모델, 화폐·향신료 아이템.

#### 6.7 아르카논 마법공학 도시

- **컬러 팔레트**: 남보라·청백·은회색.
- **바닐라 블록 추천**: 광택 딥슬레이트, 자수정 블록, 엔드 스톤 벽돌, 바다 등불(광원), 청금석, 다듬은 철.
- **조명**: 자수정 발광 + 엔드 로드 + 바다 등불.
- **파티클**: 마법 부여 오라(enchantment), 엔드 입자, 부양 반짝임(end_rod).
- **리소스팩 의존**: 첨탑 첨두·부양 플랫폼 커스텀 블록, 봉인층 GUI.

### 7. 리스크

#### 7.1 품질·기간 트레이드오프

- 옵션 D 권장안은 1차 공개 시점에 수도만 완성된다. "월드가 좁다"는 유저 피드백이 나올 수 있음. **대응**: 1막 메인 1~5가 수도 내부에서 완결되고, 히든 메인 라인이 존재함을 오픈 전 공지로 선제 커뮤니케이션.
- 옵션 C 프로시저럴 지형은 품질 편차가 크다. **대응**: 필드보스 스폰 존·랜드마크 주변 300블록 반경은 **무조건 수동 교정** 원칙을 세운다.

#### 7.2 인력 부족 시 대처

- 사용자 본인 + Claude Code 에이전트만으로 전체 맵 완성은 **비현실적**. **대응**: (1) 공개 빌드 마켓플레이스 유료 에셋 구매 예산 확보, (2) 빌드 외주 1~2인 고용, (3) 스코프 축소 옵션 D 고수.
- 에셋 사용 시 라이선스 확인 필수(상업 서버 이용 가능 여부).

#### 7.3 리소스팩 미적용 유저 대응

- 기존 확정으로 **리소스팩 미적용 유저는 체험 월드 자동 강등**. 따라서 메인/영지 월드의 커스텀 블록 의존은 허용된다.
- **그러나 체험 월드 자체는 바닐라 블록만으로 실루엣 식별 가능해야 한다**(기존 체험 월드 설계 원칙). 1차 공개 체험 월드 빌드 시 "리소스팩 없이도 수도/관찰자/세계수가 알아볼 수 있는지" 검증 테스트 필수.

#### 7.4 Citizens NPC·FAWE·리소스팩 상호 의존

- NPC 배치는 빌드 완성 후에 확정되어야 한다. 따라서 맵 빌드 마일스톤과 NPC 배치 마일스톤의 직렬화 필요.
- FAWE 스키매틱은 영지 월드에서 이미 사용 중. 메인 월드 랜드마크 배치에도 FAWE 재활용 가능.
- 리소스팩 CustomModelData 번호 체계는 이미 초안 확정. 맵 빌드 과정에서 필요한 신규 커스텀 블록은 **전역 레지스트리 PR**을 통해 충돌 방지.

#### 7.5 시즌1 오픈 일정과의 정합성 (오픈 질문)
