# 포로 서버 스모크 테스트 시나리오 초안

## 목적
핵심 플레이 루프가 실제로 끊김 없이 이어지는지 빠르게 검증하는 시나리오 모음.

## 시나리오 1. 신규 유저 1막 진입
1. 캐릭터 생성
2. 수도 도착
3. 레오니드 대화
4. 수도 외곽 조사
5. 외곽 필드보스 처치
6. 영지 개방
7. 비석 개방
8. 첫 테마 선택
9. 열차 파견
10. 첫 테마 초입 도착

성공 기준:
- 메인라인이 끊기지 않을 것
- 퀘스트 HUD가 정상 표시될 것
- 명성 %가 누적될 것

## 시나리오 2. 1막 성장 루프
1. 길드 게시판 의뢰 수행
2. 필드 사냥
3. 골드/강화석 수급
4. T1 강화
5. 영지 채집기 설치
6. 제작대 설치 및 기초 제작

성공 기준:
- T1 18강까지 자연스럽게 유도될 것
- 필드 사냥/강화/서브퀘스트가 연결될 것

## 시나리오 3. 2막 진입
1. 전투 레벨 50 달성
2. ???의 편지 획득
3. 관찰자 공간 이동
4. 각인 선택
5. 수도 복귀
6. 길드 긴급 호출
7. 균열의 그림자 입장

성공 기준:
- 1막과 2막 전환이 자연스러울 것
- T2 진입 장비/각인 시스템이 정상 열릴 것

## 시나리오 4. 세력 제안 확인
1. 테마 표층 진행
2. 최상위 보스 클리어
3. 길드 제안 확인
4. 자작령/기사단/그림자 제안 확인

성공 기준:
- 제안은 발생하되 영구 고정은 아닐 것
- 전환 가능 구조가 유지될 것

## 시나리오 5. 3막 심층 루프
1. 심층 개방
2. 심층 필드 진입
3. 심층 균열 반복
4. 심층보스 1~2 공략
5. 결전 준비
6. 결전보스 공략

성공 기준:
- 심층 필드 2개, 심층보스 2개, 결전보스 1개 구조가 자연스럽게 이어질 것
- 드랍/성장/세력 신뢰가 함께 누적될 것
