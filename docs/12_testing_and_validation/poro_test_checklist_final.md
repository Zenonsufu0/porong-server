# 포로 서버 테스트 체크리스트 최종 초안

## 목적
이 문서는 포로 서버 시즌1 구현 후 검증해야 하는 핵심 테스트 항목을 정리한 체크리스트다.

## A. 서버 기동 / 기본 환경
- 플러그인 정상 로드
- DB 연결 정상
- 설정 파일 로드 정상
- 필수 월드/지역 로드 정상
- 기본 명령어 응답 정상

## B. 계정 / 캐릭터 / 직업
- 첫 접속 시 초기화 정상
- 직업 선택 정상
- 초기 장비 지급 정상
- 전투 레벨 저장/복구 정상
- 각인 선택 저장/복구 정상

## C. 1막 메인라인
- 수도 도착 연출 정상
- 레오니드 대화 정상
- 수도 외곽 조사 퀘스트 정상
- 수도 외곽 필드보스 처치 카운트 정상
- 영지 개방 정상
- 비석/월드맵 개방 정상
- 첫 테마 선택 정상
- 열차 파견 연출 정상
- 레벨 50 달성 시 ???의 편지 지급 정상
- 관찰자 공간 이동 정상
- 각인 개방 정상

## D. 장비 / 강화 / 잠재 / 제작
- T1 장비 장착/해제 정상
- T1 강화 수치 반영 정상
- 강화석 조각/파편/강화석 소모 정상
- T2 장비 지급/제작 진입 정상
- 상위 강화석 조각/파편/상위 강화석 소모 정상
- 강화석 -> 상위 강화석 조각 교환 정상
- 잠재 표시/UI 반영 정상
- 영지 올인원 제작대 카테고리 진입 정상

## E. 퀘스트 / HUD
- 메인 퀘스트 진행도 정상
- 서브 퀘스트 진행도 정상
- 현재 추적 퀘스트 HUD 표시 정상
- HUD ON/OFF 설정 정상
- 퀘스트 완료 후 다음 단계 자동 갱신 정상

## F. 명성 / 세력 신뢰
- 테마 명성 % 증가 정상
- 길드 신뢰 % 증가 정상
- 귀족 신뢰 % 증가 정상
- 황립기사단 신뢰 % 증가 정상
- 그림자 신뢰 % 증가 정상
- 세력 전환 시 진행도 보존 정상

## G. 보스 / 균열 / 심층
- 필드보스 스폰/처치 정상
- 최상위 보스 개방 정상
- 균열의 그림자 진입/보상 정상
- 심층 필드 진입 조건 정상
- 심층보스 2종 개방 정상
- 결전보스 개방 정상
- 극상위 입장 조건 검증 정상

## H. 경제 / 드랍
- 골드 드랍 정상
- 강화석 드랍 정상
- 테마 부산물 드랍 정상
- 퀘스트 전용 드랍 정상
- 그림자/기사단/귀족 전용 드랍 정상
- 영지 생산물 획득 정상

## I. 최종 확인
- 메인라인 진행 막힘 없음
- 반복 루프 보상 과도/부족 여부 점검
- UI 겹침 여부 점검
- 세이브/재접속 후 상태 유지 점검
