## 0. 문서 목적

매트릭스 v1 확정 후 남은 **서사·파밍 동선·변환 레이트** 관련 3건의 결정을, 포로 서버 기준(1막=T1 18강 게이트, T2=재료 제작형, 상업화는 치장/편의 중심, 완만 기조)을 존중하며 **장단점 표 + 권장안 + 리스크 + 45일차 대시보드 지표 영향**으로 정리한다. 매트릭스 v1의 유효 셀 구조, 레전드리 공급 곡선(기준 10.8권), 치장·각인서 채널 분리 원칙은 그대로 승계한다.

## 1. 자문 1 — 카테고리 C 1막 공개 톤

### 1-1. 배경 정리

카테고리 C는 1막 시점 **도전 불가·관찰 가능**의 2막 게이트 프리뷰. 매트릭스 v1에서 1막 드랍은 "단서 아이템 1셀"만 허용. 질문은 **이 존재를 1막 유저에게 얼마나 명시적으로 알릴지**.

### 1-2. 장단점 비교 표

| 축 | 노출 A (명시적 예고 보스) | 노출 B (은폐) | 노출 C (중간/부분 노출) |
|---|---|---|---|
| **서사 몰입** | 강 — "도전 불가" UI와 서사 떡밥이 2막 티저 역할 | 약 — 2막 진입 시 처음 등장, 신선도 높음 | 중 — 수도 히든 메인 완주자 등 소수만 알아챔 |
| **파밍 기대감** | 강 — 2막 진입 동기 명확 | 약 — 1막 종반 동기가 "메인 진행" 외에 약함 | 중 — 발견 유저에게만 강하게 작용 |
| **스포일러 경계** | 위험 — 2막 서사 일부가 1막 단계에서 노출됨 | 안전 — 2막 정체 보존 | 중 — 연출 씬만 제한 노출 |
| **드랍 체감 괴리** | 낮음 — "도전 불가" UI가 명시적이라 "싸웠는데 보상 없음" 민원 차단 | 중간 — 추후 2막에서 "이게 1막에도 있었냐" 역혼란 | 낮음 — 소수 노출이라 민원 집계량 적음 |
| **운영/QA 부하** | 중 — 안내 UI·대사·단서 아이템 루프 필요 | 저 — 1막 범위에서 카테고리 C 콘텐츠 최소 | 중 — 씬 카드 몇 개 제작, 조건 분기 로직 |
| **디스코드 봇 `/드랍 <보스명>` 처리** | 명료 — "도전 불가/예고 보스" 상태 플래그로 응답 | 단순 — 1막 범위 응답에서 생략 | 까다로움 — 노출 조건이 유저별로 달라져 응답 분기 필요 |

### 1-3. 권장안 — **노출 C (중간/부분 노출)**

- 매트릭스 v1이 "1막 노출 2체"로 이미 **전원 공개(노출 A)도 아니고 완전 은폐(노출 B)도 아닌** 중간 상태를 전제한다. 이 구조를 형식화한다.
- 2체 중 **1체는 메인 퀘스트 씬으로 누구나 조우**(서사 떡밥 + "봉인됨/침식이 아직 덜 깨어남" 안내), **1체는 수도 히든 메인/테마 탐색 루트에서 소수만 조우**(진짜 프리뷰).
- "도전 불가" 처리: 상호작용 시 **결투 진입 차단 + 전용 대사 + 단서 아이템 자동 지급**(스토리 전용, 월렛 카테고리 귀속). 이 단서 아이템은 2막 해금 시 "2막 첫 대면 가속 효과"(1회성 버프 또는 서사 우대 플래그)로만 소비.
- 디스코드 봇에서 카테고리 C는 1막 기간 `preview=true` 플래그로 분리 응답 — `/드랍` 명령 시 "1막 프리뷰 · 드랍 없음 · 2막 해금 예정" 한 줄로 고정.

### 1-4. 리스크 / 45일차 지표 영향

- **리스크**: 히든 루트 조우 유저와 일반 진행 유저 간 "정보 격차"가 커뮤니티 불만으로 번질 수 있음. 매트릭스 v1 §7-3(T2 시드 누수)와 결이 다른 **정보 자산 격차**로 분류.
- **45일차 대시보드 지표 영향**:
  - `auction_daily_summary`·`auction_category_summary`에는 직접 영향 없음(C 카테고리 1막 드랍 아이템이 월렛 귀속이므로 경매 미유통).
  - 별도 **`boss_encounter_log`(카테고리=C, contact_type=preview)** 집계가 있으면 1막 후반 "예고 보스 조우율"을 CP2(21일차)·CP4(45일차)에 체크. 히든 루트 조우율이 활성 유저의 **5~15%** 범위면 적정, 2% 미만이면 너무 숨긴 것·30% 초과면 너무 드러난 것으로 판단.
  - `engraving_drop_log` 등 기존 집계에는 영향 0. 매트릭스 v1 표 2-1~2-6 수치 수정 불요.

## 2. 자문 2 — T2 상위 조각 시드의 지방 테마 확대 여부

### 2-1. 배경 정리

매트릭스 v1 §2-6은 "수도 상위 필드보스 2~3종 한정, 0.5~1개/킬, 월렛 귀속 권고". §5-3은 병목으로 "지방 테마 루트 유저 박탈감"을 명시. §7-3은 과잉 리스크로 "경매 누수 시 2막 초반 T2 2~3부위 조기 제작"을 경고. 질문은 **시드 위치를 수도 한정으로 둘지, 4테마로 확대할지**.

### 2-2. 장단점 비교 표

| 축 | 수도 한정 유지 | 4테마 확대 |
|---|---|---|
| **파밍 동선 자유도** | 낮음 — 1막 후반 수도 집중 강제, 지방 루트 체류 유인 감소 | 높음 — 테마 선호별 플레이 유지, 지방 완주 보상 정합 |
| **박탈감** | 지방 테마 고정 유저 불만 가능 | 완화 — 어느 테마든 T2 시드 경로 보유 |
| **2막 절벽 보존** | 강 — 수도 한정이라 총 공급량 제한 쉬움 | 약 — 4테마 합산 공급이 의도치 않게 누적될 수 있음 |
| **시즌1 45일차 공급량** | 보수적, 예측 편차 작음 | 2~3배 증가 여지, 일일 상한/드랍률 재튜닝 필요 |
| **경매 시세 영향** | 수도 시드는 월렛 귀속으로 경매 0 — 시세 영향 없음 | 마찬가지로 월렛 귀속 유지 시 시세 영향 없음. 단 **비귀속으로 바꿀 경우** 시세 급변 |
| **서사 정합** | 수도 히든 메인 완주자 우대 톤과 부합 | 4테마 균등 완주자 우대 톤과 부합. 둘 다 유효 |
| **운영/QA** | 단순 — 드랍 테이블 2~3개만 관리 | 복잡 — 테마별 5~8개 필드보스 드랍 테이블 편집 |

### 2-3. 권장안 — **4테마 확대 (단, "소량·월렛 귀속·테마당 보스 1종 한정"의 3중 제약)**

매트릭스 v1 §5-3 병목을 정면으로 해결하고, §7-3 리스크는 "귀속 플래그 + 수량 클램프"로 별도 차단하는 접근이 더 정합하다. 구체안.

- 수도 상위 필드보스 2~3종은 **0.5~1개/킬** 유지(매트릭스 v1 그대로).
- 4지방 각 테마에서 **1종 필드보스만** 시드 허용. 드랍량은 **0.2~0.4개/킬**로 수도 대비 약 40% 수준.
- **모든 시드는 월렛 귀속(거래 불가)** 플래그 고정 — 매트릭스 v1 §10-3 조정 제안 승계.
- 4테마 합산 1막 후반 공급량이 2막 초반 T2 제작 수요의 **10~15% 이하**에 머물도록 CP6(45일차) 측정으로 클램프. 초과 시 지방 테마 드랍량을 0.1~0.2로 축소.

### 2-4. 수치 시뮬 (참고)

- 1막 후반 14일 × 일 활성 3~6시간 × 시간당 수도 상위 보스 킬 ~1회 → **수도 경로만으로 인당 누적 조각 ~12~25개**.
- 4테마 확대 시 지방 경로까지 병행하는 유저는 **+6~12개 추가**. 일반 유저 상한선은 인당 누적 ~35개 선. 2막 T2 1부위 제작 요구량이 **상위 조각 40~60개**로 설계되면 1막 시드만으로는 여전히 1부위 미완성, 2막 초반 절벽 유지.
- **비확대 시나리오** 인당 12~25개는 2막 T2 제작 1부위 절반~2/3 선행으로 2막 초반 유저에게 "절반 완성" 체감 → 동기 부여와 박탈감의 경계에서 동기 쪽이 약간 우세.
- **확대 시나리오** 인당 20~35개는 "1부위 거의 완성" 체감으로 2막 초반 동기 강화, 단 월렛 귀속이라 경매 누수는 0. 매트릭스 v1 §7-3 리스크는 수량이 아닌 **귀속 플래그로 이미 차단**되므로 확대와 무관.

### 2-5. 리스크 / 45일차 지표 영향

- **리스크**: 4테마 드랍 테이블 편집·QA 부담. 드랍량 클램프 튜닝이 CP2(21일차)까지는 대체로 확인 불가(수도 보스 킬 숫자 대비 지방 보스 킬 숫자가 초반 상대적으로 적음).
- **45일차 대시보드 지표 영향**:
  - `auction_daily_summary`·`auction_cosmetic_summary`: 직접 영향 없음(월렛 귀속 전제).
  - `wallet_log` + `crafting_log` 조인으로 CP6 "T2 상위 조각 1막 시드 기여율"을 측정(매트릭스 v1 §11 CP6 재사용). 목표선은 **"2막 T2 제작 수요의 10~15% 이하"**로 수정(v1은 10%). 지방 확대에 맞춰 상한을 5%p 열어주는 조정.
  - `boss_drop_log`: 테마×보스×품목 디멘션에 **지방 보스 T2 시드 행** 신설. CP1(7일차)에 드랍률 실측 ±10% 관측.
  - 비선발대 유저 인당 누적 T2 상위 조각 수량을 **CP3(35일차)** 에 신규 카드로 추가 — **20개 이상 누적 유저 비율**이 전체의 40~60%면 정상, 70% 초과면 드랍량 축소 트리거.

## 3. 자문 3 — 1막 한시 4:1 상위 변환 우대 이벤트

### 3-1. 배경 정리

매트릭스 v1 §13-3은 기본 5:1 등급 변환 레이트를 전제로 1막 커먼 과공급 구간(§5-1)에서 **한시 4:1 우대**를 검토 과제로 남김. 질문은 **도입 / 미도입 / 일일 상한 변형** 중 선택.

### 3-2. 장단점 비교 표

| 축 | 도입 (단순 4:1 7일) | 미도입 (5:1 고정) | 변형 (4:1 + 일일 상한) |
|---|---|---|---|
| **커먼 과공급 해소** | 강 — 7일간 가파른 소비 | 약 — 자연 소진 대기 | 중 — 상한 내에서 꾸준히 해소 |
| **레전드리 공급 가속** | 1막 시점 유니크·레전드리 드랍 0이므로 **직접 가속 없음** | 동일 | 동일 |
| **에픽 공급 가속** | 중 — 커먼→레어→에픽 2단 변환으로 에픽 접근 가속(1막 에픽 절벽 완화, §5-2와 부합) | 없음 | 약~중 — 상한만큼 완만 가속 |
| **인플레이션 리스크** | 중 — 에픽 각인서 경매 시세 하락 가능, 치장/장비와 다른 축이라 월렛 인플레는 약함 | 없음 | 낮음 — 상한으로 예측 가능 |
| **체감(플레이어)** | 강 — "이벤트 기간"이라는 명확한 플레이 동기 | 중립 | 중 — 매일 소비 루틴 유발(일일 퀘스트형) |
| **운영 복잡도** | 낮음 — 기간 플래그 1개 | 낮음 | 중 — 유저별 일일 카운터 + 리셋 |
| **2막 진입 절벽 완화** | 중 — §5-2 대안으로 기능 | 없음 | 중 |
| **담합·시세 조작 리스크** | 중 — 4:1 이벤트 직전 커먼 사재기·이벤트 기간 덤핑 가능 | 없음 | 낮음 — 일일 상한이 사재기 이득 축소 |

### 3-3. 권장안 — **변형 (4:1 + 일일 상한, 1막 한시 9~11일)**

- 4:1 우대는 §5-1 병목(커먼 과공급)과 §5-2 병목(1막 에픽 절벽 완화)을 **동시에** 해결할 수 있는 유일한 축이므로 **완전 미도입은 기회 손실**.
- 그러나 단순 4:1 7일은 매트릭스 v1 §7(경매 인플레이션)·§9-1(실측 없는 고정값)과 결이 충돌. **일일 상한이 실측 데이터 수집 창도 겸함**.
- 구체안:
  - **기간**: 1막 중앙 22일차 시작, **9~11일간**(CP2 21일차 체크포인트 직후 시작 → CP3 35일차 직전 종료).
  - **레이트**: 커먼→레어 4:1, 레어→에픽 4:1. 에픽→유니크는 1막에 유니크 드랍이 0이므로 **변환 불허**(시즌1 후반에 별도 검토).
  - **일일 상한**: 유저당 일일 **커먼→레어 20회·레어→에픽 10회** (누적 커먼 80권/레어 40권 소비). 매트릭스 v1 §3 추정 기준 일반 유저 인당 커먼 주간 축적 20~40권 대비 과소하지 않고, 상위권 과잉 소비는 차단.
  - **사재기 방지**: 이벤트 시작 48h 전 커먼 경매 거래 **주간 개인 상한 10권** 한시 적용(매트릭스 v1 §10-5 치장 랜덤박스 상한 원리 차용).

### 3-4. 리스크 / 45일차 지표 영향

- **리스크**:
  - 이벤트 종료 직후 에픽 각인서 경매 낙찰가 일시 하락 가능성. `auction_daily_summary.avg_price`의 7일 이동평균이 이벤트 직후 ±30%를 넘기면 CP5 알람 발생 전제로 운영 대응 준비.
  - 일일 상한 UI·카운터 버그 시 특정 유저 변환 실패 민원. 이벤트 기간 DB `conversion_daily_counter` 테이블 무결성 점검 필수.
  - 1막 한시라는 단서가 **시즌 기본 레이트 = 4:1**로 오해될 여지. 공지 톤에 "1막 한시 우대, 기본 5:1로 복귀" 명시.
- **45일차 대시보드 지표 영향**:
  - `engraving_inventory` 등급 분포 — CP2(21일차) 대비 CP3(35일차) 커먼 감소율 **15~30%**, 에픽 증가율 **8~15%**가 정상 범위. 이를 벗어나면 다음 막 이벤트 설계 시 상한 튜닝.
  - `engraving_activation_log` — CP3 35일차 에픽 활성화 인원 비율 **15% 이상**(매트릭스 v1 §11 CP3 기존 목표)을 소폭 상향해 **18~22%**로 조정. 4:1 이벤트 수혜를 반영.
  - `auction_category_summary`(에픽 각인서 카테고리) — `avg_price` 7일 이동평균 변동폭 ±30% 상한 유지. 이벤트 종료 직후 2주간 자동 알람 감도 상향(±20%).
  - 신규 집계 뷰 `conversion_event_summary` 권장 — 일자별 변환 건수·유저 수·상한 도달율. CP4(45일차) 리포트에 1줄 포함.

## 4. 3건 권장안 종합 + 상호 정합 체크

1. **자문 1 — 중간 노출(C)**: 서사 떡밥과 스포일러 보호의 균형. 단서 아이템은 월렛 귀속이라 경매 시세·레전드리 공급 곡선에 영향 0.
2. **자문 2 — 4테마 확대 + 월렛 귀속 + 테마당 1종**: 지방 박탈감 해소. 경매 누수 차단은 귀속 플래그로 별도 유지. CP6 측정 상한 10% → 15%로 5%p 여유.
3. **자문 3 — 4:1 + 일일 상한 + 9~11일**: 커먼 과공급·에픽 절벽 동시 해소. CP3 에픽 활성화 목표 15% → 18~22% 소폭 상향.

### 4-1. 상호 정합

- 3건 모두 **매트릭스 v1 경제 축(경매 인플레이션)**에 영향이 미미하거나 상한 장치로 통제 가능. 레전드리 45일차 공급 곡선(기준 10.8권)에는 **직접 영향 없음** — 3건 전부 1막 범위이고 레전드리 드랍은 4막 극상위 전용이라 독립.
- 자문 2 확대와 자문 3 이벤트가 동시 진행되면 "2막 진입 체감"이 과하게 완만해질 수 있으므로 자문 3 종료(33일차 전후) 이후 자문 2 지방 시드 드랍량을 CP3 기준 재확인.
- 자문 1의 히든 경로 조우 유저와 자문 2의 지방 테마 시드 유저가 **상이한 코호트일 수 있음** — 서로 다른 파밍 동선에 보상이 골고루 분산되는 효과는 긍정적.
