# 포로 서버 4테마+수도+사하르+아르카논 필드 몬스터 & 보상 테이블 존재 여부 검토 + 누락분 보강 제안 v1

> **상태**: 2026-04-19 content-designer 1차 드래프트
> 선행 결정 — (1) 생활은 필드 채집 없음, 영지 전용. 필드는 **전투·퀘스트·드랍 전용**. (2) T2 재료 제작형, 강화 25강/파괴 없음. (3) 각인서 5등급제·레전드리 월 공급 제한 폐기(F-3). (4) 1막은 카테고리 A 필드보스·B 정예 중심, 일반 몬스터는 드랍 파이프라인의 저층을 담당.
> 범위 — 시즌1 1막 기준 현존 테이블 인벤토리 + 누락 영역 식별 + 테이블 스키마 보강 제안. 2막 표층·심층·3막 개방 지역은 참조 범위까지.

---

## 1. 개요

본 문서는 "테마별 필드 몬스터 및 보상 테이블이 실제로 존재하는가"라는 질문에 대해 `docs/01_boss_design_core` · `docs/08_story_and_quests` · `docs/10_seed_and_config_tables` · `custom-plugins/empire-rpg/src/main/resources/seeds` 전역 실측 인벤토리를 정리하고, 공백 영역에 대한 **테이블 스키마 제안**과 **테마별 생태계 차별화 가이드**, **45일차 보스 드랍 매트릭스 정합성 체크**를 한 묶음으로 제공한다.

결론 요약: **보스 축(카테고리 A/B/C)은 `poro_boss_drop_grade_matrix_v1.md`로 정규화돼 있으나**, 보스 아래 저층(일반 몬스터·강화 개체·정예 3단의 "몬스터 마스터 + 스폰 테이블 + 드랍 테이블")은 **`poro_master_planning.md` 본문의 불릿 요약 수준**에 머물러 있고, 쿼리·시뮬·DB 이행에 바로 꽂을 수 있는 **정규 마스터 테이블은 부재** 상태다.

---

## 2. 목표

1. 현존 필드 몬스터 관련 테이블·서술 자료를 **품목별 실측 리스트**로 확정한다.
2. "일반 몬스터·강화 개체·정예 3단 × 7테마(수도+동서남북+사하르+아르카논)" 범위에서 **누락 영역**을 명시한다.
3. 생활 필드 채집 폐기 결정에 맞춰 **필드 드랍 품목에서 생활 원재료를 제외하는 원칙**을 명문화한다.
4. 테마별 생태계 차별화(동=분신/침식, 서=가짜 쉼터/도적, 남=과열/균열, 북=냉기/봉인)가 몬스터 아키타입·태그·드랍 품목에 반영되게 **테이블 필드 구조**를 제안한다.
5. 신 테이블이 1막 45일차 지표·공급량 예측과 충돌하지 않는지 **정합성 체크리스트**를 제시한다.

---

## 3. 핵심 규칙

### 3-1. 필드 드랍 원칙 (생활 채집 폐기 반영)

- **필드 = 전투·퀘스트·드랍 전용**. 필드 리젠/채집 부담 제거 원칙을 드랍 품목 축에도 그대로 반영한다.
- 필드 드랍 품목 허용 카테고리(보스 매트릭스와 동일 축 7종 유지): 골드 / T1 강화 재화(월렛) / T1 장비(실물) / 보스 고유 재료 / 각인서(2막+) / 치장 / T2 상위 조각·파편(1막 후반 수도 상위 한정).
- **제외 원칙 — 생활 원재료는 필드 드랍에서 금지**: 광석(철·구리·은·금·미스릴), 원목, 약초, 곡물, 마력 원석 등 **영지 공장 1단계 Extractor 산출물에 해당하는 원재료는 필드 몬스터 드랍 품목에서 전면 배제**한다. 이 원재료는 **영지 채집기만의 공급원**으로 단일화해 공장 경제를 보호한다.
- **경계선**: "보스 고유 재료"(예: 침식 수액·화맥 코어·봉인 잔흔)와 "지역 특산 희귀 재료"(퀘스트 단서·치유 부산물 등)는 **생활 원재료가 아닌 전투·서사 재료**로 분류. 영지 공장으로 가공되지 않고 T2 제작·퀘스트·교환소 소비에 들어간다.
- **소비품 재료(연금/요리) 취급**: `poro_master_planning.md`의 월렛 실물 유지 경계선에 따라 필드에서는 드랍하지 않는다. 연금/요리 재료 공급은 영지 약초/곡물 채집기 전담.

### 3-2. 몬스터 3단 구조 유지

기존 테마별 서술(`poro_master_planning.md`)이 **일반 / 강화 개체 / 정예 / 필드보스 4단**으로 정리돼 있으므로, 테이블 설계도 동일한 단을 유지한다. 필드보스 이하 3단(일반·강화·정예)을 본 문서가 다루고 필드보스 이상은 `poro_boss_drop_grade_matrix_v1.md` 관할이다.

### 3-3. 태그 원칙 준수

몬스터에도 공격·상태 판정과 연동되는 태그는 **최대 2개**로 제한한다. 플레이어 피해 증가 카테고리(일반/태그/조건부)와 매핑해, 예를 들어 "침식 + 식물" 2태그를 가진 동부 몬스터는 "식물 태그 피해 +x%" 각인·세트와 직접 상호작용한다.

---

## 4. 범위

- **대상 테마**: 수도 황성 + 동부 침식 숲 + 서부 황야 + 남부 화맥 + 북부 설혼 + 사하르 상인연합 + 아르카논 마법공학 도시 (총 7개).
- **대상 층위**: 일반 몬스터 / 강화 개체 / 정예 몬스터 3단. 필드보스 이상은 기존 보스 매트릭스 v1로 참조.
- **대상 품목 축**: 골드 / T1 강화 재화(월렛) / T1 장비 / 보스·지역 재료 / 각인서(2막+) / 치장 / T2 시드(1막 후반 한정). **생활 원재료 제외**.
- **대상 행위**: (a) 현존 자료 실측 인벤토리, (b) 누락 영역 식별, (c) 보강 테이블 스키마, (d) 테마별 생태계 차별화 가이드, (e) 보스 매트릭스 정합성 체크.

---

## 5. 비범위

- **필드보스 5종·최상위·심층·결전·극상위 드랍 매트릭스 재설계** — 기존 `poro_boss_drop_grade_matrix_v1.md`가 담당.
- **실제 스폰 좌표·밀도 튜닝·리젠 시간 숫자 확정** — MythicMobs 스폰 프로파일 구현 단계에서 작성.
- **각 몬스터의 스킬·AI 패턴 상세 설계** — combat-balance 후속.
- **영지 채집기 산출물 종류·레시피** — `poro_factory_estate_draft.md` 관할.
- **경매·월렛·아이템 실체화 파이프라인 구현 세부** — data-schema 후속.

### 6-2. 누락 영역 식별

현존 자료를 기준으로 다음이 **명백하게 누락** 상태다.

1. **몬스터 마스터 정규 테이블 전면 부재**. 7테마 × 3단(일반/강화/정예) 서술형 이름 리스트는 있으나, `monster_id`·`theme`·`level_band`·`archetype`·`tag` 필드를 가진 정규 테이블이 없다. 최소 약 80~110종 규모(테마 당 일반 4~5 + 강화 3 + 정예 3 + 3막 2테마 포함)로 추정.
2. **스폰 테이블 부재**. 테마 × 필드 5종 × 지형 타입(숲·평원·동굴·사당·유적 등) × 몬스터 구성비가 어디에도 정의돼 있지 않다.
3. **일반 몬스터·강화 개체·정예 드랍 확률 정규화 부재**. 마스터 플래닝 본문은 "골드 소량 / 강화석 조각 낮은 확률 / 침식 수액 등" 수준의 **언어적 서술**만 있어 DB 이행 불가.
4. **3막 개방 테마(사하르·아르카논) 몬스터 생태계 테이블 미작성**. 이름 목록과 대표 드랍 한 줄 요약만 마스터 플래닝 본문에 들어가 있다.
5. **태그 축 미정**. 침식·폭풍·화염·냉기·기계·도적 등 테마 고유 태그를 몬스터 단위에 붙일지, 필드 단위에 붙일지 규칙 자체가 부재.
6. **보스 매트릭스와 일반 몬스터 드랍의 연속성 미정**. 2-2 강화 재화 드랍 표(조각 0.3 / 0.03 / 0 등)가 보스 매트릭스 v1에 들어가 있으나, 이 수치가 어느 몬스터에 붙는지 `monster_id` 수준 매핑이 없다.
7. **각인서 드랍 권한 분리 미정**. "각인서는 2막 최상위 보스부터"가 확정이라 **1막 일반/강화/정예는 각인서 드랍 0권**이어야 하는데, 이 0권 명시가 일반 몬스터 테이블에는 아직 없다.
8. **플러그인 리소스 시드 부재**. `custom-plugins/empire-rpg/src/main/resources/seeds/` 경로에는 생활 스킬 테이블만 존재. 몬스터·스폰·드랍 시드 CSV가 전혀 없다.

### 6-3. 보강 테이블 스키마 (3테이블 세트)

아래 3개 테이블을 **정규 마스터 세트**로 제안한다. `poro_boss_drop_grade_matrix_v1.md`와 축(등급 5단·드랍 7품목) 정합 유지.

**(A) `monster_master`** — 몬스터 단위 정체성

- `monster_id` (PK, string, 예: `east_mob_common_01`)
- `display_name` (string, 한국어)
- `theme` (enum: capital / east / west / south / north / sahar / arcanon)
- `tier` (enum: common / elite_low / elite_high) — 일반 / 강화 개체 / 정예 3단
- `level_band` (enum: low / mid / high) — 1막 초입 / 1막 중반 / 1막 후반·2막 접경
- `archetype` (enum: melee / ranged / magic / stealth / summoner)
- `tag_primary` (enum: 테마별 고유 태그 1)
- `tag_secondary` (nullable enum: 태그 최대 2 원칙)
- `size_class` (enum: small / medium / large) — 시각/히트박스 힌트
- `faction_note` (nullable string: 도적/침식/기계 등 서사 계열)
- `spec_ref` (nullable string: MythicMobs 스펙 ID 링크, 후속 연결용)

**(B) `monster_spawn_table`** — 필드 구성비

- `spawn_id` (PK)
- `theme` (enum)
- `field_code` (string, 예: `east_field_01_forest_entrance`)
- `location_type` (enum: forest / plain / ravine / cave / shrine / ruin / wasteland / glacier / bazaar / lab)
- `monster_id` (FK → `monster_master`)
- `spawn_weight` (int, 상대비중 1~100)
- `density_class` (enum: sparse / normal / dense)
- `level_cap_hint` (int, nullable — 레벨 감쇠 가이드)
- `spawn_condition` (nullable string: 낮/밤·메인 진행도 플래그 등)

**(C) `monster_drop_table`** — 드랍 (보스 매트릭스와 동일 7품목·5등급 축)

- `drop_id` (PK)
- `monster_id` (FK)
- `drop_category` (enum: gold / t1_shard / t1_fragment / t1_stone / t1_equipment / boss_material / engraving_book / cosmetic / t2_seed / region_material)
- `carrier` (enum: WALLET / ITEM)
- `rarity` (enum: common / rare / epic / unique / legendary, nullable — 각인서·치장에 주로 사용)
- `drop_rate_pct` (float, 0~100)
- `drop_quantity_min` / `drop_quantity_max` (int)
- `is_life_material_blocked` (bool, **기본 true** — 생활 원재료 차단 플래그)
- `note` (nullable string)

> 스키마 공통 메타: `created_at` / `updated_at` / `source_doc` (원안 출처 문서 경로). 셀 0% 항목은 기록하지 않고, **0% 가정이 의도된 경우(예: 1막 일반 몬스터 각인서 0권)는 `drop_rate_pct=0` 명시행으로 남긴다** — 매트릭스 누락 방지용.

### 6-4. 1막 3단 기준 드랍 가이드 (보스 매트릭스 2-2 반영)

마스터 플래닝 본문 서술을 보스 매트릭스 표 2-2(강화 재화 드랍)와 정합시킨 **1막 3단 드랍 가이드**.

| 몬스터 단 | 골드(킬당) | T1 조각 | T1 파편 | T1 강화석 | T1 장비 | 각인서 | 치장 | 지역 재료 |
|---|---|---|---|---|---|---|---|---|
| 일반 몬스터 (common) | 소량 | 0.3 | 0.03 | 0 | 0% | **0%** | 커먼 <0.3% | 테마 저등급 파편 저확률 |
| 강화 개체 (elite_low) | 보통 | 1.0 | 0.15 | 0.02 | 0% | **0%** | 커먼 0.5% | 테마 중등 파편 보통 |
| 정예 (elite_high) | 중간 | 2.5 | 0.5 | 0.08 | 0.3%(방어구 저확률) | **0%** | 커먼 1% / 레어 <0.1% | 테마 샘플 확정(퀘스트용) |

### 6-5. 테마별 생태계 차별화 가이드

각 테마의 서사 정체성이 몬스터 아키타입·태그·드랍에 일관되게 반영되도록 **테마 고유 키워드**와 **추천 태그 풀**을 정리한다.

- **수도 황성** (범용 + 변이 톤): 일반 `균열쥐/들개/까마귀/들쥐/식충` 유지. 아키타입은 melee 6 · ranged 2 · magic 1 · stealth 1 비중. 태그 풀 = `urban / crack(균열) / vermin`. 드랍은 거친 광석·낡은 천조각·오염 혈흔 대신 **"균열 파편·잠식 금속편"** 등 전투·서사 재료 전환(생활 원재료 배제 원칙 반영). 외곽 필드보스 5종 구조는 기존 유지.
- **동부 침식 숲** (분신·환영): 아키타입 melee 4 · ranged 2 · magic 2 · stealth 2. 태그 풀 = `corruption(침식) / spirit(정령) / plant / phantom(분신)`. **기믹 키**: 분신형 정예(본체 판별), 환영 무리(HP 공유), 침식 수액 피해 지면. 드랍 축은 `침식 수액 / 왜곡된 포자 / 정령편` — 약초성 섬유는 "섬유 채집"이 아니라 "정령의 부산물"로 서사 재해석해 생활 원재료 분리.
- **서부 황야** (가짜 쉼터·도적·야수): 아키타입 melee 5 · ranged 4 · stealth 2. 태그 풀 = `storm(폭풍) / bandit(도적) / beast / illusion(가짜 쉼터)`. **기믹 키**: 가짜 쉼터 매복, 폭풍 파편 이동 저해 지면, 도적 무리 수적 압박. 드랍 축은 `폭풍 파편 / 마른 혈석 / 붕괴 잔해핵` + 사하르 상단 떡밥용 `실종 상단 서신 단편` (1회성, 서사 회수용).
- **남부 화맥** (과열·태움): 아키타입 melee 4 · magic 4 · summoner 2. 태그 풀 = `ignite(화염) / crack(균열) / overheat / beast`. **기믹 키**: 과열 바닥/분화선, 피격 시 화상 스택 부여(상태이상 학습 루프), 용암등 환경 상호작용. 드랍 축은 `화맥 잔재 / 붉은 핵편 / 과열 결정 / 화맥 코어`. "거친 광물·재가루"는 생활 원재료 혼동 소지가 있어 **"열핵 가루"로 명칭 전환** 권장.
- **북부 설혼** (냉기·언데드/봉인): 아키타입 melee 3 · ranged 2 · magic 3 · summoner 2 · stealth 1. 태그 풀 = `frost(냉기) / undead(언데드) / seal(봉인) / beast`. **기믹 키**: 냉기 스택 누적 후 빙결, 설혼 안개 시야 축소, 봉인석 상호작용. 드랍 축은 `냉기 결정 / 설원 털가죽 / 깨진 봉인석 조각 / 봉인 잔흔`. "설원 털가죽"은 가죽 제작 연동 오해 소지 — **"설원 잔흔"으로 전투·서사 재료화** 권장.
- **사하르 상인연합** (3막): 아키타입 stealth 3 · ranged 3 · melee 3 · magic 1. 태그 풀 = `mirage(환영) / contract(계약) / bandit / desert`. **기믹 키**: 환영 도플갱어, 계약 낙인 디버프, 모래유리 지면. 드랍 축은 `미라지 모래편 / 계약 파편 / 유리핵 조각`. "건조 약초"는 **생활 원재료 혼동 위험 — 전면 삭제 또는 `미라지 수액`으로 대체**.
- **아르카논 마법공학 도시** (3막): 아키타입 ranged 4 · magic 4 · summoner 2. 태그 풀 = `arcane(마력) / machine / overload(과부하) / echo(공명)`. **기믹 키**: 마력 과부하 자폭 드론, 공명 연쇄(인접 적 버프), 회로 패널 상호작용. 드랍 축은 `공명 파편 / 마력 회로 조각 / 공명 핵편 / 폭주 코어`. "깨진 금속편"은 생활 원재료 철·구리와 혼동 — **"파손 합금편" 등 비금속 계열 재료명 권장**.

### 6-6. 45일차 지표 정합성 체크

`poro_boss_drop_grade_matrix_v1.md`의 45일차 지표·공급량 추정과 본 문서의 신규 필드 드랍 가이드가 충돌하지 않는지 점검.

- **T1 강화 재화**: 보스 매트릭스 표 2-2가 이미 "일반/강화/정예/필드보스" 4단 수치를 포함하고 있어, 본 문서의 3단 가이드는 표 2-2 상위 3행을 그대로 옮긴 형태다. **충돌 없음**. 45일차 예상 자율 파밍 버짓(-80~-250석 의도적 부족)은 유지된다.
- **각인서 공급**: 1막 전 구간 일반/강화/정예 3단 각인서 드랍 0% 명시로 **2막 최상위 보스부터 공급** 원칙과 정합. 레전드리 월 공급 제한 폐기(F-3)가 있어도 1막 일반 몬스터는 어차피 0%이므로 **공급 하한에 추가 유입 없음** — 보스 매트릭스 3-2 추정(기준 시나리오 10.8권)에 영향 없음.
- **치장 드랍**: 일반 몬스터 커먼 치장 <0.3%는 보스 매트릭스 2-5 채널 분배의 "필드보스 직접 드랍 40%·보상 상자 25%·랜덤박스 20%·확정구매 15%"와 **병렬 채널**. 수치상 일반 몬스터 치장 드랍을 하루 500킬 가정 시 1.5개/일 수준이라, 경매 시세에 가산되는 커먼 공급은 주당 10개 내외 — 3단계 시나리오 표의 ±30% 경매 가격 변동 알람 범위 내. **채널 4에 "5. 일반 몬스터 직접 드랍(커먼 한정)"을 추가**해 매트릭스 2-5를 5채널로 확장 권장.
- **T1 장비 실물**: 정예 0.3% 수치는 표 2-3 "정예 0.3%"와 그대로 일치. **충돌 없음**.
- **생활 원재료 제외 원칙**: 필드 드랍에서 원목·광석·약초·곡물·마력 원석이 빠지므로 영지 채집기 생산량 곡선(Lv1=1.00 기준 ~ Lv5=2.6)과 **경쟁 재화 없음**. 공장 경제 보호 정합.
- **레전드리 각인서**: 1막 필드 몬스터는 드랍 0% 고정이므로 F-3(월 공급 제한 폐기)과 **직접 충돌 없음**. 필드 드랍이 "레전드리 자연 공급 하한"에 추가되지 않는다(4막 극상위 보스 전용 유지).

## 7. 오픈 질문

1. **태그 축 부여 단위**: 태그를 `monster_master`에 붙일지, `monster_spawn_table` 필드 단위에 붙일지(같은 몬스터가 다른 필드에서 다른 태그를 가질 수 있게 할지) 규칙이 아직 없다. 포로 피해 증가 카테고리(일반/태그/조건부)와 맞물리므로 combat-balance 협의 필요.
2. **몬스터 종 수 총량**: 테마 당 "일반 4~5 + 강화 3 + 정예 3" = 테마당 10~11종 × 7테마 = 약 70~77종 + 3막 2테마 보강 = 약 90종 전후. 이 규모가 MythicMobs 스펙 관리 부담·컨텐츠 체감에 적정한지 content 측 2차 조율.
3. **테마별 고유 태그 풀 확정**: 6-5의 제안 태그(`corruption`·`storm`·`ignite`·`frost`·`mirage`·`arcane` 등)가 직업각인/공용각인 "태그 피해 +x%" 축과 1:1 매핑 가능한지 combat-balance 검증.
4. **일반 몬스터 치장 드랍 유지 여부**: F-CP3 결과에 따라 일반 몬스터 커먼 치장 드랍을 0%로 덮을지 검토. 경매 시장 초기 공급 다양성 vs 시세 붕괴 리스크 트레이드오프.
5. **지역 재료 명칭 정리**: 마스터 플래닝 본문의 "거친 광석·낡은 천조각·설원 털가죽·건조 약초·깨진 금속편" 등 **생활 원재료로 오인될 수 있는 명칭**을 전투·서사 재료로 리네임해야 하는지(예: "설원 털가죽" → "설원 잔흔") 사용자 확인 필요.
6. **3막 사하르·아르카논 일반 몬스터 테이블 착수 시점**: 1막/2막 안정화 후 3막 개방 2주 전 착수로 충분한지, 1막 구축과 함께 선행 시드로 만들어둘지.
7. **보스 매트릭스 v2와의 병합 시점**: 본 문서(일반/강화/정예 3단)를 보스 매트릭스 v2에 흡수해 "전 층위 통합 매트릭스"로 만들지, 별도 문서로 유지할지.
8. **수도 외곽 블록 제한과의 정합**: 메인 3 이전 수도 반경 제한 하에서 외곽 필드보스 5종 구역 일반 몬스터 드랍 표본이 **과대 표집**되지 않는지(F-CP2 쏠림 임계 재설정 필요 여부).
