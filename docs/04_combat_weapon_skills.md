# 04. 전투 / 무기 / 스킬

## 입력 방식

무기를 들고 있을 때만 스킬 입력이 활성화된다.

| 입력 | 역할 |
|---|---|
| 좌클릭 | 기본기/주력기 |
| 우클릭 | 이동기/보조기 |
| Shift+우클릭 | 특수기/제어기/자기 강화 |
| F | 핵심기 |

## 금지 항목

- 플레이어 지속 장판
- 적 표식
- 받는 피해 증가
- 방어 감소
- 파티 전체 피해 증가
- 복잡한 표식 폭발

## 무기 클래스

| 무기 | 자원 | 각인 A | 각인 B |
|---|---|---|---|
| 검 | 검세 3 | 수호검 | 결투검 |
| 망치 | 충격 3 | 파쇄자 | 성벽수호자 |
| 창 | 압박 5 | 관통창 | 질풍창 |
| 석궁 | 조준 1 | 추적자 | 기계궁수 |
| 낫 | 그림자 흐름 3 | 사신 | 암영무희 |
| 스태프 | 마력 충전 5 | 화염술사 | 비전술사 |

## 공용각인

1차 시즌에서는 공용각인을 제외한다.
