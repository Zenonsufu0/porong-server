# New Greatsword Variant

Use this skill when the user wants to start a new greatsword attempt.

## Goal
Create one new variant folder under:
`assets/source/items/weapons/01_greatsword/`

## Folder structure
- `<variant>/design.md`
- `<variant>/task.md`
- `<variant>/refs/`
- `<variant>/source/`
- `<variant>/exports/`

## Rules
- Use lowercase English and underscores.
- Use numeric prefixes when possible.
- Do not create fake `.bbmodel` files unless explicitly asked.
- Report the created path and suggest the working `.bbmodel` filename.
